JavaScript is required by the InfoHub

The InfoHub uses JavaScript to provide access to your installed Mentor Graphics documentation. If you are seeing this Error tab, then JavaScript is currently disabled in your browser and needs to be enabled to continue.

To fix this issue: enable JavaScript in your browser, refer to Browser Settings.

Internet Explorer Active Content Error

Microsoft has made a change to the default settings for Internet Explorer in Windows XP SP2 that disables JavaScript from running for locally installed HTML content and thus preventing the InfoHub from functioning.

To fix this issue: select Tools > Internet Options menu item and then select the Advanced tab. Scroll down to the Security section, select "Allow active content to run in files on My Computer". This is not an issue for Mozilla or Netscape users.
