Displaying Defect Parts Location

Defect parts information is a combination of detected lines from each inspection algorithm, capturing the details of locations inspected by an inspection machine. Defect parts are drawn as closed contour paths near to defect regions. These lines are described as the offset from the Defect Reference Pixel and are detected by inspection machines using algorithms. During inspection time, all necessary information is captured in a file called <IR_NAME>_Defects.ini. When loading an inspection report, if this optional file is present, Calibre DefectReview reads it and populates the information in the IMU. You can then display defect parts information in the IMU.

Note:

This is supported for LRF inspections only.

Procedure

Click the Defect Contour icon in the IMU window toolbar.

The IMU displays all defect parts in red as shown in the following figure. It is displayed for all images.
