Using General Defect Operations

This section describes many of the general operations that can be performed on any defect (isolated or clustered) in the Defect Map window.

Procedure

  1. In the Defect Map window, right-click a defect to invoke a popup menu.
  2. Select an item from the popup menu. There are a number of defect-specific operations available.
    Table 1. General Defect Operations Menu Items

    Menu Item

    Description

    Delete

    Delete a defect or a set of defects selected with rubber band mode (see “Using Rubber Band Selection Mode”).

    When you delete a defect, a message box appears prompting you to confirm the deletion. For example, if you select defect ID 147 of inspection 1 for deletion, then the following message is issued:

    DefectId(s) 1_147 will be removed permanently from the list

    Classify

    Classify a defect or set of defects as specific classification types. You can classify a defect as a specific type. To classify a set of defects, first select them using RubberBand Selection.

    Track Defects

    Launch Defect Progress Tracking using the defect’s attribute such as Cd Var, Size, Area, Reflectance, Reflectance Residue, Transmittance, or Transmittance Residue.

    Generate Layout Images

    Generate layout images for a defect or set of defects.

    Once layout images are generated, images can be viewed in the Image Measurement Unit window. The image names are prefixed with the word “Layout”. You can generate layout images only if the layout data for the reticle has been set up.

    The details for setting up layout data are described in “Layout Import”.

    Certain operations are expanded as follows:

    • For the Classify operation:

      1. Click Classify. A cascading menu with all defect types and corresponding defect codes appears as shown in Figure 1.

      2. Click the desired classification type.

        Figure 1. Extended Menu for Classify

    • For the Track Defects operation:

      1. To track defects across inspections from a mask, right-click on a defect in the Defect Map to invoke the popup menu.

      2. Select Track Defects.

      3. Select an Attribute: Cd Var, Defect Area, Defect Size, Reflectance, Reflectance Residue, Transmittance, or Transmittance Residue.

        Figure 2. Track Defects Selection

        The Defect Progress Tracking dialog appears.

        Figure 3. Defect Progress Tracking in an Inspection

      4. Specify an Upper or Lower Control Limit or both (if any) and click Execute.

        For details on Defect Progress Tracking (DPT), please refer “Defect Progress Tracking (DPT)”.
