Must Catch Defects File and Capture Rate

You can use a Must Catch Defects file to find the capture rate.

Refer to “Specifying a Must Catch Defects File” for details on specifying a Must Catch Defects file.

When a Must Catch Defects file is specified, a complete grid is computed using the must catch defect locations.

Figure 1. Detection Line Using a Must Catch Defects File

In the bottom two rows of this example graph, the cells with white background are the dummy grid cells added for forming a complete grid using the Must Catch Defects file.

Locations in the Must Catch file can be deleted using the Delete option in the right click menu of the detection line graph. You can delete a row, a column, or a cell. The defects after deletion are shown with a zero capture rate as illustrated in Figure 2.
