Saving a Correlation List

You can save the contents of the Correlation List into CSV format only from the ALL folder

Procedure

  1. In the Calibre DefectReview window, select Utilities > Save Correlation List. A file chooser dialog appears
  2. Enter the name of the CSV file in the dialog box.
  3. Click Save.
  4. The CSV file contains all the columns displayed in the Correlation List.
