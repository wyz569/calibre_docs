Configuring Image Type Names in the Grid Display

You can configure image type names in the image grid display. Once you have changed the name, all modalities are updated to display the new name.

Procedure

  1. Open dat-ini.xml in a text editor and search for the node ImageTypeFilter.
    Figure 1. ImageTypeFilter Node

  2. In the ImageTypeFilter node, change the displayValue to your new image type name.
    Note:

    You cannot modify the node name (for example, defect_image in the previous figure).
