Using the Ruler

Ruler is provided in Image Measurement to measure the distance between two points on the images.

Procedure

  1. In the IMU window toolbar, click the Ruler button.
    Figure 1. Ruler Button

  2. Left-click in any image to select a starting point for measurement.
  3. Drag the cursor to the end measurement point, then release. A line is visible between the start and end points until the mouse is released.

    The Ruler displays the distance measured between the start point and the end point, and information about the ruler is displayed in the status bar of the IMU window (see Figure 2).

    Refer to the status bar description on “The Image Measurement Unit Window” for more details.
