Creating a Database User

You must create a user to run the MySQL database.

Procedure

  1. Login to the MySQL Database Server.
    mysql -h <host> -u <root_user> -p <root_password>
  2. To create a database user, enter the following command from the MySql prompt:
    CREATE USER 'dbuser'@'localhost' IDENTIFIED BY 'WelCome!23';

    If you attempt to create a new database user with the following:

    username: dbuser
password: WelCome!23

    This user already exists in Calibre DefectReview by default and does not need to be created. You can otherwise create a database username of your choice, but remember to create a similar profile in Calibre DefectReview (see “User Management” for details).
