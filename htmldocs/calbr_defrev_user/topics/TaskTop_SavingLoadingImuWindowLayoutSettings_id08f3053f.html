Saving and Loading IMU Window Layout Settings

The layout that is configured for each of the display units in the IMU can be saved.

Procedure

  1. To save the layout, select Dock Layout > Save in the menu bar of the Calibre DefectReview window.
  2. To reload the layout, select Dock Layout > Load in the menu bar of the Calibre DefectReview window.
  3. For more details on saving and loading the layout of Calibre DefectReview refer to “Dock Layout”.
