Reversing Labels

You can reverse the order of defect type labels or defect size labels on detection line graph.

Prerequisites

  • A detection line graph should be generated as described in “Generating a Detection Line Plot”.

Procedure

  1. Right-click in the detection line graph. A menu appears.
    Figure 1. Reverse Labels

  2. Click one of the following options:

    • Reverse Defect Type Labels — the defect type labels are reversed as shown in the following figure:

      Figure 2. Reverse Defect Type Labels

      • Reverse Defect Size Labels — the defect size labels are reversed as shown in the following figure:

        Figure 3. Reverse Defect Size Labels
