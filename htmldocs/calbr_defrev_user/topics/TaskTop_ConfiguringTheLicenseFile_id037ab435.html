Configuring the License File

Prior to installing any node-locked or floating license (.lic) file, you must first enter the PC host names in the license file.

Procedure

  1. Open the Siemens EDA license file using any text editor.
  2. Check the MAC address provided in the license.lic file as shown in the following figure:
    Figure 1. MAC Address Information

  3. Open a command window in the PC corresponding to the MAC address.
  4. Enter the following command:
    ipconfig/all
  5. Get the PC’s host name for the specified MAC address as shown in the following figure:
    Figure 2. ipconfig/all Command Information

  6. In the license file, replace the text “put_server_name_here” with the host name of the PC. For the above case, it will be pc65 as shown in the following figure:
    Figure 3. Replace Host Name

  7. Save the license file.
  8. If there are multiple instances of “put_server_name_here” text in your license file, repeat steps 2 through 7.
