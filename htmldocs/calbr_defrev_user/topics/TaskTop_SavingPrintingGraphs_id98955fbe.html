Saving and Printing Graphs

You can save or print the graph of an analysis result into an image file in PNG format.

Procedure

  1. Saving a Graph: Click the Graph View button.
    Figure 1. Graph View Save and Print

  2. Click Save. A File dialog box appears.
  3. Enter the name of the file and click Save.
  4. Printing a Graph: Click the Graph View button.
  5. Click Print to print a graph.
