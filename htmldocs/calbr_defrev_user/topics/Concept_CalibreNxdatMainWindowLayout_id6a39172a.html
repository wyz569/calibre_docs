Calibre DefectReview Main Window Layout

The Calibre DefectReview interface is organized into several key functional regions.

Figure 1 shows the layout of Calibre DefectReview.

Figure 1. Calibre DefectReview Main Window

Table 1 describes each functional area of the Calibre DefectReview main window.

Table 1. Calibre DefectReview Layout Descriptions

Region

Description

Defect Map (1)

Provides a graphical view of the defects as well as a convenient spatial way to navigate defects. Refer to “Defect Map” for details.

Defect List (2)

Provides a table view of the defect properties. The Defect List enables you to navigate defects based upon their sorted properties. Refer to “Defect List” for details.

Defect Selection (3)

Provides options to select a subset of defects present in the input inspection file. Refer to “Defect Selection” for details.

Defect Classification (4)

Provides an interface for classifying defects. Refer to “Defect Classification” for details.

Image Measurement (5)

Displays defects and provides utilities for analyzing images associated with a defect. Refer to “Image Measurement” for details.

Repeatability Analysis (6)

Provides an interface to analyze multiple inspections together and perform repeatability analysis on these inspections. Refer to “Repeatability Analysis” for details.

When multiple inspection files are loaded into Calibre DefectReview, the main window appears. However, for a single file, a single tab or folder is opened instead. Refer to “Understanding Folders” for more information on folders. For multiple files, multiple folders are opened.

Refer to “Multiple Inspection Operations” for details on analyzing the results of multiple inspections.

Refer to “Calibre DefectReview GUI Quick Reference” for a summary of all the menu and toolbar options in the main Calibre DefectReview window.
