Performing Blank Transfer Analysis

You can use the Defect Management window to find whether a defect from blank inspection is transferred to patterned inspection at later stages of an inspection.

Procedure

In the Defect Management window, select the Blank Transfer Analysis query from Select Query options.

A defect D1 from blank inspection B1 is considered to be transferred to defect D2 in a pattern inspection P1 if:

  • Both defects are within the search radius specified.

  • D1 is detected in a blank inspection before D2 is detected in a patterned inspection (D1's inspection start time is equal to or less than D2's inspection start time).

Both D1 and D2 should belong to the same mask (the mask unique ID should be the same).
