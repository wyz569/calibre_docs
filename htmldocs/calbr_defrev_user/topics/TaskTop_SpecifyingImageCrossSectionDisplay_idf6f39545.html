Specifying the Image or Cross Section Display

You can specify the type of the image or cross section results to be displayed in any display unit through a menu available from the header of the individual display units.

Procedure

  1. Right-click on the header of any display unit and a menu appears.
    Figure 1. Display Unit Right-Click Display Unit Header Menu

  2. Select an operation to perform from the right-click menu.
    Table 1. Display Unit Header Menu Operations

    Option

    Description

    Defect Images

    Lists available defect or reference to display. The defect and reference images include:

    • Defect

    • Reference

    • Reflected Defect

    • Reflected Reference

    See “Defect and Reference Images” for further information on defect and reference images.

    Functional Images

    Lists available functional images to display. These include:

    • Diff Image

    • Avg Image

    • Sum Image

    • Blinking

    • Custom

    See “Functional Images” for further information

    ZI Images

    Lists available zoomed-in images to display. These include:

    • Interpolated Defect Image

    • Interpolated Reference Image

    • Zoomed Interpolated Defect Image

    • Zoomed Interpolated Reference Image

    • Defect Threshold Image

    • Reference Threshold Image

    • Defect Edge Image

    • Reference Edge Image

    See “ZI Images” for further information

    Other Images

    Lists any external images. See “Other Images” for further information.

    Cross Section

    Displays a CrossSection window graph. See “CrossSection Graph” for further information

    Remove

    Removes the selected item from the display unit. If the item is removed, then the display unit is shown as empty.
