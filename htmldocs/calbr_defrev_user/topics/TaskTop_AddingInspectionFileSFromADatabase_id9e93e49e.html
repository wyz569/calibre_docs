Adding Inspection File(s) From a Database

You can use Calibre DefectReview to open multiple inspection files from the database.

Prerequisites

  • An inspection file must already be loaded into Calibre DefectReview

  • For database prerequisites, refer to “Defect Database”

Procedure

  1. In the Calibre DefectReview main window, display the Select Inspection(s) dialog box by performing either of the following:

    • Select File > Open Database Inspection(s) > Open Database Inspection(s).

      Figure 1. Open Database Inspection(s) Menu

    • Click the Open Database Inspection(s) icon in the tool bar as shown in the following figure.

      Figure 2. Open Database Inspection(s) Toolbar Icon

  2. A Select Inspection(s) dialog box appears. The inspections stored in the database are displayed in the result table.
    Figure 3. Select Inspection(s) Dialog Box

  3. In the Select Inspection(s) dialog box:

    • To select a single file, click the filename.

    • To select multiple files:

      • Click on one file, then press the Ctrl key and click on other desired files.

      • To open all files in a given folder, click on one file and then use Ctrl-a to select all the files.

      • You can also use the Shift key and select a range of files.

  4. Click Open.

Results

You can filter the results as described in the Results section of “Opening Inspection File(s) from a Database”
