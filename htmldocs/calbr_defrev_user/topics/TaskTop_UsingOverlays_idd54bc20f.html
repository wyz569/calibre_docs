Using Overlays

You can overlay the die cycle or the reticle image (or both) on the Defect Map.

Procedure

  1. To overlay a die cycle, click the Overlay Die Cycle button (the “D” iconTable 1). The same button can be toggled to remove the overlay. The die cycle is shown as blue colored rectangles on the Defect Map.
  2. You can distinguish the defect die for any defect by left-clicking on any defect.

Results

The defect die is shown in red. Refer to Figure 2 for details.
