Editing Defect Comments

You can edit comments of a single defect or set of defects after selecting the defects from the Defect List or the Defect Map.

Procedure

  1. In the Defect Classification window, click the Comments button. The presence of comments is indicated by green colored text in the Comments button. If there are no comments, the text is grey. The Defect Comments dialog box appears.
  2. In the Defect Comments dialog box, enter comments in the space provided, then click OK.
