Adding or Removing Defects on the Grid

You can add or remove defects from the grid displayed in the Defect Map window.

Prerequisites

  • The grid must be detected. From the Calibre DefectReview menu bar, select Utilities > Auto Grid Detection > X-direction or Y-Direction to detect and display the grid used for the inspection in either X- or Y-direction, respectively. Refer to “Auto Grid Detection” for details.

Procedure

  1. In the Defect Map window, right-click a defect to invoke the popup menu.
  2. Select an item from the popup menu.
    Table 1. Menu Items for Adding or Removing Defects on the Grid

    Menu Item

    Description

    Add to Grid

    Add defects to the grid.

    If the defect is successfully added, then the color code of the defect changes to yellow.

    If the defect is not successfully added, then the following message is issued:

    The defect <defect_num> cannot be part of a grid

    Remove From Grid

    Remove a defect from any grid. The color code of the defect changes to a non-grid defect.

    Add New Grid Line

    Add a new grid line on any defect. A new line is displayed on the Defect Map window and the color code of the associated defect changes to the grid defect color.
