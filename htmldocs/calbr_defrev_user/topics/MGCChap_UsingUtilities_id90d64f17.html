  • Inspection File Export
  • Deleting Unselected Defects
  • Unfiltering All Filtered Defects
  • Calibre MDPAutoClassify
  • Calibre DefectClassify
  • Copy Auto Defect Classifications
  • Calibre DefectClassify Report
