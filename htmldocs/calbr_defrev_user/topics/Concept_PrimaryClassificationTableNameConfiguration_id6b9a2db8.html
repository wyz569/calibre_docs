Primary Classification Table Name Configuration

You can specify the name displayed for the Primary Classification table in the dat-ini.xml file, using the tableName attribute in the classificationTable node.
Figure 1. tableName Attribute

The tableName attribute represents the Primary Classification table name.

For example, the Primary Classification table name needs to be changed to “1st Tier Classification”. Enter the string “1st Tier Classification” as the tableName value in the dat‑ini.xml file.

Figure 2. tableName Set to New Name

The new name is displayed in the Defect Classification window as shown in Figure 3.

Figure 3. New Name Displayed in the Defect Classification Window

Note:

If you change the Primary Classification table name in Defect Classification window through the dat-ini.xml file, the name also changes in the Repeatability Analysis window and in the Classify right-click menu in the Defect Map.
