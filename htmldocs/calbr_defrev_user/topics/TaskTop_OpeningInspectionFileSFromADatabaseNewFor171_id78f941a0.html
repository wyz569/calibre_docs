Opening Inspection File(s) from a Database

You can open inspection files stored in the database from the File menu of Calibre DefectReview.

Prerequisites

  • For database prerequisites, refer to“Defect Database” .

Procedure

  1. In the Calibre DefectReview main window, display the Select Inspection(s) dialog box by performing either of the following:

    • Select File > Open Database Inspection(s) > Open Database Inspection(s).

      Figure 1. Open Database Inspection(s) Menu Option

    • Click the Open Database Inspection(s) icon in the tool bar as shown in the following figure.

      Figure 2. Open Database Inspection(s) Icon

    Additionally, you can open a database inspection file in place (where the newly-opened file replaces the currently open file while preserving the GUI window layout) using one of the following methods:

    • Select File > Open Database Inspection(s) > Open Database Inspection(s) Inplace.

    • Click the Open Database Inspection(s) In Place icon in the toolbar.

      Figure 3. Open Database Inspection(s) In Place Icon

  2. The Select Inspection(s) dialog box appears. The inspections stored in the database are displayed in the result table. The results are sorted based on the descending order of Inspection Start Time.
    Figure 4. Select Inspection(s)

  3. In the Select Inspection(s) dialog box:

    • To select a single file, click the filename.

    • To select multiple files:

      • Click on one file, then press the Ctrl key and click on other desired files.

      • To open all files in a given folder, click on one file and then use Ctrl-a to select all the files.

      • You can also use the Shift key and select a range of files.

  4. Click Open.

Results

The results in the Select Inspection(s) can be narrowed down by applying various filter options available in the dialog box as show in the following figure.

Figure 5. Filters in Select Inspection(s)

  • Mask Unique Id — To filter the results based on mask unique ID, provide substring of mask unique ID in the Mask Unique Id input box as shown in following figure and press Enter. Inspections with a mask unique ID starting with Mask-1 are displayed in the table. To remove the filter, remove the string from the Mask Unique Id input box and press Enter.

    Figure 6. Mask Unique ID Filter

  • Inspection Unique Id — To filter the results based on an inspection unique ID, provide the substring of the unique ID in the Inspection Unique Id input box as shown in the following figure and press Enter. Inspections with an inspection unique ID starting with Demo-1 are displayed in the table. To remove the filter, remove the string from Inspection Unique Id input box and press Enter.

    Figure 7. Inspection Unique ID Filter

  • Inspection Type — To filter the results based on inspection type, select the type of file from the drop-down box as shown in the following figure. Inspections with a selected type (in this example, LDF) are displayed in the table. To remove the filter, select the empty row from drop-down box

    Figure 8. Inspection Type Filter

  • Select Date Range — To filter the results based on inspection start time, check Select Date Range and specify the date time range as shown in the following figure. Inspections with a start time in between the given range are displayed in the table. To remove the filter, uncheck Select Date Range.

    Figure 9. Select Date Range Filter
