How Do I Increase the Size of Images?

The initial image size in the Calibre DefectReview IMU is set in the Calibration Parameter Editor.

Procedure

  1. In the Calibre DefectReview main window, select Utilities > Calibration Parameter Editor.
  2. In the Calibration Parameter Editor dialog box, set the Image Scale Factor. By default, this is 197.
