Deleting a Defect from the Grid

You can delete defects from the grid by selecting Delete from the right-click menu in a detection line graph.

Prerequisites

  • A detection line graph should be generated as described in “Generating a Detection Line Plot”.

Procedure

  1. Right-click on the detection line graph. A menu appears.
  2. Select Delete from the menu. There are four different sub-options available: Row, Column, Cell, and Selected. The Selected option deletes selected defects from grid point.
    Note:

    If the RubberBand mode is enabled, then the Selected option is enabled and Row, Column, Cell options are deactivated.

    Figure 1. Delete Options

  3. If the Must Catch Defects option is not specified in the Parameters dialog box (refer to “Specifying a Must Catch Defects File” for details), the defects are also deleted from grid points in the Defect Map.
  4. If the Must Catch Defects option is specified, grid detection is automatically performed after the deletion.
