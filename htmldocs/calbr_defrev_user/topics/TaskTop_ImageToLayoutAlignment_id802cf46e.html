Image to Layout Alignment

Once you have successfully loaded a wafer file, you can align SEM images with layout images. This is performed with the Calibre DefectClassify tool.

Procedure

There are two ways to launch Calibre DefectClassify:

  • In the Calibre DefectReview window, select Utilities > Image to Layout Alignment.

  • Click the Image to Layout Alignment icon in the Calibre DefectReview toolbar.

    Figure 1. Image to Layout Alignment Icon

Refer to the Calibre DefectClassify User’s Manual for full details.
