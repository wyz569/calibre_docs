Deleting a User

You can delete an existing user account from Calibre DefectReview.

Procedure

  1. In the Calibre DefectReview menu bar, select Utilities > Manage Users. The Manage Users dialog box appears with a list of users.
  2. Right-click the user to be deleted. A popup menu appears with a Delete User item (see Figure 1).
    Figure 1. Delete User Menu Item

  3. Click Delete User. A message appears, prompting you to confirm the deletion.
  4. Click Yes to delete the user. Once the account is deleted, the user is not able to access Calibre DefectReview upon the next log in or when importing or adding a file in the active session.
