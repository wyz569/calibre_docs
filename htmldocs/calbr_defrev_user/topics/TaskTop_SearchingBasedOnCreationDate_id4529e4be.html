Searching Based on Creation Date

You can set the search utility to search based on the file creation date.

Procedure

  1. In the Search and Open Inspection(s) dialog box, select the Creation Date check box.
  2. To set the start and end dates, for each entry, click the down arrow and a calendar appears. Select the desired dates as shown in Figure 1.
    Figure 1. Creation Date Option

  3. Note the following:

    • The system creation date (returned by operating system) is used to search the results.

    • By default, the End Date is the current date. If the End Date is not entered, all files with creation dates between the starting date and the current date are selected.
