Showing Capture Rate

Show Capture Rate mode allows you to find the number of inspections in which the defects have been captured for a particular defect location.

Prerequisites

A folder must be open.

Procedure

  1. Right-click in an empty area of the Defect Map to invoke a popup menu.
  2. Select Show Capture Rate. A check mark indicates the option is enabled.

Results

The defects in the Defect Map are displayed as shown in Figure 1. The defect colors temporarily change to reflect what is shown in the Capture Rate Color Legend dialog box. When the Capture Rate Color Legend dialog box is closed, the defects return to their original colors.

Figure 1. Defect Map With Show Capture Rate

Examples

In the following example, there are three inspection files and Show Capture Rate is selected. A Capture Rate Color Legend dialog box appears as shown in Figure 2.

Figure 2. Capture Rate Color Legend Dialog Box

As shown in Figure 2:

  • Defects captured in only one inspection file are highlighted in black.

  • Defects captured in two inspection files are highlighted in red.

  • Defects captured in all the three inspection files are highlighted in green.
