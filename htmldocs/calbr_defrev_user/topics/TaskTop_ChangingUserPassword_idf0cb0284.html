Changing a User Password

You can change a Calibre DefectReview user password in the Utilities menu.

Procedure

  1. In the Calibre DefectReview menu bar, select Utilities > Manage Users. The Manage Users dialog box appears with a list of users.
  2. Right-click on a user. A popup menu appears with Change Password.
  3. Click Change Password. The Edit User Information dialog box appears as shown in the Figure 1.
  4. In the Edit User Information dialog box, enter the new password twice, then click OK.
