Performing Defect Source Analysis

You can use the Defect Management window to find the defect and the inspection where the defect was first detected.

Procedure

Select Defect Source Analysis query from Select Query options.

A defect D1 is considered to be the source of another defect D2 if:

  • D1 and D2 are both available within a search radius specified.

  • D1 is detected before D2 (D1's inspection start time is equal to or less than D2's inspection start time).

Both D1 and D2 should belong to same mask (the mask unique ID should be same).
