Detection Lines

Detection line analysis helps verify the detection capability of a mask inspection system on test masks. Each square (a grid point in the test mask) in the mosaic graph shows the capture rate of a given defect of a particular size. The row in the mosaic represents different defect types and the column represents defect sizes.
