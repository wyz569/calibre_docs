Saving Tables

You can save the table(s) of a single analysis or all analysis results into a CSV file.

Procedure

  1. Select the analysis type in the Analysis Types area that you want to save results.
  2. Click Table View.
  3. Click Save (to save a single analysis table) or Save All (to save all analysis tables). A File dialog box appears.
    Figure 1. Table View Save and Save All

  4. Enter the name of the file and click Save.
