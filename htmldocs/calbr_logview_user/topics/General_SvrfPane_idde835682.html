SVRF Pane

To access: SVRF tab

The SVRF pane displays the contents of the SVRF rule file used for the Calibre job. The information displayed in this pane is extracted from the “STANDARD VERIFICATION RULE FILE COMPILATION MODULE.”

Figure 1. SVRF Pane

You can export the SVRF section to a file, using the File > Export menu option or from the command line. Both a CSV parameter file and an SVRF file are exported.
