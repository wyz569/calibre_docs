PEX XACT Noxref

Parasitic extraction

Replaces the noxref prefix for nets or the noxref suffix for instances in the extracted netlist with a user-defined string. Used only in Calibre xACT.

Usage

PEX XACT NOXREF {INSTANCE_SUFFIX  string | NET_PREFIX string |
INSTANCE_SUFFIX string NET_PREFIX string }

Arguments

INSTANCE_SUFFIX string

A keyword set that specifies to replace the noxref suffix on instances with a user-defined string in the extracted netlist. The string may only contain alpha-numeric characters and ‘_’, ‘-’, ‘$’, or ‘%’.

NET_PREFIX string

A keyword set that specifies to replace the noxref_ prefix on nets with a user-defined string in the extracted netlist. The string may only contain alpha-numeric characters and ‘_’, ‘-’, ‘$’, or ‘%’. The string specified with the NET_PREFIX keyword cannot start with a number.

Description

An optional statement used to replace the noxref_ prefix on nets or the _noxref suffix on instances in the extracted netlist with a user-defined string. This statement can only be specified once in your rule file. If this statement is not specified, the default prefix for nets is noxref_ and the default suffix for instances is _noxref. May be used with the SPEF, DSPF, HSPICE, and CalibreView netlist formats.

Examples

Given the following DSPF netlist output sample:

...
X61_noxref VDD noxref_35 VDD ppnode 1=2e-08 nfin=2 as=2.22e-15 ps=1.36e-07
...

Example 1

Include the following statement to replace the noxref prefix on nets with the string xnet: 

PEX XACT NOXREF NET_PREFIX xnet

For example:

X66_noxref VDD xnet35 VDD ppnode 1=2e-08 nfin=2 as=2.22e-15 ps=1.36e-07
...

Example 2

Include the following statement to replace the noxref suffix on instances with the string xinst: 

PEX XACT NOXREF INSTANCE_SUFFIX xinst

For example:

X66xinst VDD noxref_35 VDD ppnode 1=2e-08 nfin=2 as=2.22e-15 ps=1.36e-07
...

Example 3

Include the following statement to replace the noxref suffix on instances with the string refx% and the noxref prefix on nets with the string pre_$x:

PEX XACT NOXREF INSTANCE_SUFFIX refx% NET_PREFIX pre_$x

For example:

X66refx% VDD pre_$x35 VDD ppnode 1=2e-08 nfin=2 as=2.22e-15 ps=1.36e-07
...
