tvf::set_traceback_line

Application: compile-time and runtime.

Used to trace SVRF syntax errors at a specified line in the generated SVRF rule file to line numbers in your TVF rule file.

Usage

tvf::set_traceback_line line_number

Arguments

line_number

A required argument that specifies the line number in the generated SVRF rule file, usually reported by an SVRF compiler message.

Return Values

None.

Description

Used to trace SVRF syntax errors at line_number to line numbers in your TVF rule file.

Examples

For example in compile-time TVF, if the SVRF compiler reports this error: 

ERROR: Error INP3 on line 52 of SVRF generated from TVF

You add this line to your TVF code:

tvf::set_traceback_line 52; # trace line 52

This instructs the Tcl preprocessor to issue a trace message when the output line number is equal to 52. When you run Calibre again on the TVF file that contains this command, the TVF line number is reported as shown here:

ERROR: Error TVF1 on line 79 of rules.tvf - TVF preprocessor error:
Traceback forced at SVRF output line number 52

This tells you the error is on line 79 of the TVF file.
