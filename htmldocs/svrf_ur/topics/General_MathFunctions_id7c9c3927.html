Supported Math Functions in DFM Expressions

Certain math functions are supported in DFM Expressions. Math functions take zero or more numeric input arguments and produce a numeric value. There is no compile-time or runtime exception checking on the arguments of these functions, and unreasonable or undefined values may result from certain arguments.

In the table, x and y are numeric arguments and can include the numeric variables and numeric-valued sub-expressions. Unless otherwise noted, the argument and return values are double-precision floating-point numbers.

Table 4. Rounding and Remainder Functions

Function

Definition

CEIL(x)

Returns the smallest integer greater than or equal to x. In other words, x rounded up to the nearest integer.

CEIL(x, y)

Returns x rounded up to the nearest integer multiple of y (on a grid with step y, returns the smallest grid value greater than or equal to x).

FMOD(x, y)

Returns the remainder of x/y, where y must be non-zero.

Specifically, the function is calculated as follows:

FMOD(x, y) = x - TRUNCATE(x/y)*y

where TRUNCATE rounds towards zero.

FLOOR(x)

Returns the largest integer value that is less than or equal to x. In other words, x rounded down to the nearest integer.

FLOOR(x,y)

Returns x rounded down to the nearest integer multiple of y (on a grid with step y, returns the largest grid value less than or equal to x).

REMAINDER(x, y)

Returns the remainder of x/y calculated as follows:

REMAINDER(x, y) = x - RINT(x/y)*y

where RINT rounds to the nearest integer; if the argument is midway between two integers, RINT rounds to the nearest even integer. The argument y must be non-zero.

RINT(x)

Returns x rounded to the nearest integer; if x is midway between two integers, RINT rounds to the nearest even integer according to the IEEE 754 standard. Examples: RINT(1.1) returns 1; RINT(3.5) returns 4; RINT(-0.5) returns 0.

ROUND(x)

Returns x rounded to the nearest integer. If x is midway between two integers, rounds away from 0. Examples: ROUND(2.5) returns 3; ROUND(-2.5) returns -3.

ROUND(x, y)

Returns x rounded to the nearest integer multiple of y. If x is midway between two integer multiples of y, rounds away from 0. If y is not an integer, the result is a floating point value.

Examples: ROUND(2.5, 2) returns 2, ROUND(3, 2) returns 4, ROUND(1.21, 0.1) returns 1.2.

TRUNCate(x)

Returns x with the digits to the right of the decimal point removed; in other words, rounds toward zero. Examples: TRUNC(1.1) returns 1; TRUNC(1.8) returns 1; TRUNC(‑1.8) returns ‑1.
