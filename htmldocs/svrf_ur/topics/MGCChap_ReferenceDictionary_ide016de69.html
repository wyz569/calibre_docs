Reference Dictionary A - F

This chapter contains descriptions for the statements and operations that can be used in a rule file. These appear in alphabetical order and this chapter contains those starting with letters A through F.

Also see:

  • “Reference Dictionary G - O”

  • “Reference Dictionary P - Z”

For a categorized listing of statements and operations by application, see “Summary of Rule File Elements” or refer to the sections listed in the following table:

Table 1. SVRF Statements and Operations by Category

Category and Link to Summary Section

Summary

Layer Operations

Operations that create a derived layer.

Connectivity Extraction Statements

Statements that define the electrical connectivity and net names of a design.

Device Recognition Statement

Statement that defines devices for recognition and LVS comparison.

Mask Data Preparation Statements

Statements that pertain to database fracturing, which occurs prior to mask fabrication.

Post-Design Correction Statements

Statements that are used for optical and process correction. Most commands are documented in their corresponding RET product manuals.

Specification Statements

Specification statements that describe the working environment for applications using the rule file. They control how data are processed.
