equal

Effective property computation language function. Not used in a TVF Function.

Tests if all instances in the input group return the same value for an expression.

Usage

equal(vector_expression)

Arguments

vector_expression

A required expression. It may be numeric- or string-valued. See “Effective Property Computation Vector Functions”.

Return Values

Integer.

Description

Evaluates vector_expression across all instances in the input group. If all such evaluations result in the same value, the function returns 1. If at least one of the evaluations of vector_expression differs from the others, the function returns 0.

When a string vector_expression is given, that expression is compared for case-insensitive equality across all instances in the input group by default. Case sensitivity during equality comparison is controlled by the setting of LVS Compare Case.

Examples

LVS REDUCE type(subtype) PARALLEL [
    EFFECTIVE xprop
    EFFECTIVE STRING ypropStr
    all_xprop_plus_one_are_the_same = EQUAL(xprop+1)
    all_ypropStr_are_equal = EQUAL(ypropStr)
...
]
