Unsupported Functionality Errors and Warnings

This section describes runtime messages related to unsupported functionality.
Table 1. Unsupported Functionality Errors and Warnings

Explanation

The following errors are reported by flat Calibre applications:

ERROR: EXTENT CELL … WITH MATCH unsupported in flat CALIBRE applications.

The WITH MATCH option of the Extent Cell operation is only supported in hierarchical Calibre applications.

ERROR: INSIDE CELL … WITH MATCH unsupported in flat CALIBRE applications.

The WITH MATCH option of the (Not) Inside Cell operation is only supported in hierarchical Calibre applications.

The following errors are reported by Calibre nmDRC:

ERROR: CBLOCK OASIS DRC results database not supported with PIPE.

The CBLOCK keyword is not supported with PIPE in DRC Results Database.

ERROR: STRICT OASIS DRC results database not supported with PIPE.

The STRICT keyword is not supported with PIPE in DRC Results Database.

ERROR: ASCII DRC results database not supported with PIPE.

ASCII format is not supported with PIPE in DRC Results Database.

ERROR: Binary DRC results database not supported with PIPE.

The BINARY keyword is not supported with PIPE in DRC Results Database.

The following warnings are reported by Calibre applications:

WARNING: PINS operation unsupported in Calibre applications; output layer will be empty.

This operation is only applicable to ICverify.

WARNING: PORTS operation unsupported in Calibre applications; output layer will be empty.

This operation is only applicable to ICverify.

WARNING: TOPEX operation unsupported in Calibre applications; output layer will be empty.

This operation is only applicable to ICverify.

The following warnings are reported by flat Calibre applications:

WARNING: EXTENT CELL … OVER unsupported in flat CALIBRE applications. UNDER option ignored.

The OVER option of the Extent Cell operation is ignored in flat applications.

WARNING: EXTENT CELL … UNDER unsupported in flat CALIBRE applications. UNDER option ignored.

The UNDER option of the Extent Cell operation is ignored in flat applications.

WARNING: EXTENT CELL ... RING unsupported in flat CALIBRE applications. RING option ignored.

The RING option of the Extent Cell operation is ignored in flat applications.

WARNING: DRC CHECK MAP AREF specification for DRC RuleCheck <name> not supported in CALIBRE DRC-F.

The AREF keyword is specified in a DRC Check Map specification statement for a nmDRC rule check in Calibre nmDRC-F. AREF output for arrayed rectangles is only supported in Calibre nmDRC-H. For Calibre nmDRC-F, rectangles are output without AREF structure. The name of the affected nmDRC rule check appears in the message.

WARNING: DRC CHECK MAP TEXTTAG specification for DRC RuleCheck <name> not supported in CALIBRE DRC-F.

The TEXTTAG keyword is specified in a DRC Check Map specification statement for a DRC rule check in flat nmDRC. TEXTTAG output is only supported in Calibre nmDRC-H and is otherwise ignored.

WARNING: DRC CHECK MAP AUTOREF specification for DRC RuleCheck <name> not supported in CALIBRE DRC-F.

The AUTOREF keyword is specified in a DRC Check Map specification statement for a DRC rule check in flat nmDRC. AUTOREF output is only supported in Calibre nmDRC-H and is otherwise ignored.

WARNING: DRC CHECK MAP PROPERTIES specification for DRC RuleCheck <name> not supported in CALIBRE DRC-F.

The PROPERTIES keyword is specified in a DRC Check Map specification statement for a DRC check in Calibre nmDRC. PROPERTIES output is only supported in Calibre nmDRC-H. For flat operation, PROPERTIES is ignored. The name of the affected DRC check is placed into the message.

WARNING: INSIDE CELL ... WITH LAYER HIER unsupported in flat CALIBRE applications. HIER option ignored.

The WITH LAYER HIER option of [NOT] Inside Cell is ignored in flat applications.

The following warnings are reported by Calibre nmDRC applications and by the ICrules CHEck DRc command:

WARNING: PATHCHK operation unsupported in DRC applications; output layer will be empty.

This operation is not supported in nmDRC applications. Pathchk is used only for ERC checks in LVS connectivity extraction.

WARNING: DEVICE LAYER operation unsupported in DRC applications; output layer will be empty.

This operation is not supported in nmDRC applications. The Device Layer operation is used only in Calibre nmLVS/Calibre nmLVS-H, Calibre YieldAnalyzer, and ICtrace.

The following warnings are reported by non-Calibre nmDRC applications:

WARNING: RDB or RDB ONLY option for NET AREA RATIO only supported in DRC applications.

A Net Area Ratio operation with RDB [ONLY] specified is required as part of the flow in a non-Calibre nmDRC application. These applications do not support the RDB [ONLY] option. The RDB [ONLY] option is ignored (but the geometric output is generated as usual).

WARNING: RDB or RDB ONLY option for DENSITY only supported in DRC applications.

A Density operation with RDB [ONLY] specified is required as part of the flow in a non-Calibre nmDRC application. These applications do not support the RDB [ONLY] option. The RDB [ONLY] option is ignored (but the geometric output is generated as usual).

WARNING: PRINT or PRINT ONLY option for DENSITY only supported in DRC applications.

A Density operation with PRINT [ONLY] specified is required as part of the flow in a non-Calibre nmDRC application. These applications do not support the PRINT [ONLY] option. The PRINT [ONLY] option is ignored (but the geometric output is generated as usual).

WARNING: NET AREA RATIO PRINT operation only supported in DRC applications.

A Net Area Ratio Print operation is required as part of the flow in a non-Calibre nmDRC application. These applications do not support the Net Area Ratio Print operation. An empty layer is output.

WARNING: DFM ANALYZE operation only supported in Calibre DRC applications; output layer will be empty.

A DFM Analyze operation is required as part of the flow in a non-Calibre nmDRC application. The output layer will be empty.

WARNING: DFM MEASURE operation only supported in Calibre DRC applications; output layer will be empty.

A DFM Measure operation is required as part of the flow in a non-Calibre nmDRC application. The output layer will be empty.

WARNING: DFM TRANSITION operation only supported in Calibre DRC applications; output layer will be empty.

A DFM Transition operation is required as part of the flow in a non-Calibre nmDRC application. The output layer will be empty.

The following warnings are reported by ICverify applications:

WARNING: INSIDE CELL operation unsupported in ICverify applications; output layer(s) will be empty.

An inside cell operation is required in an ICverify application. This operation is only supported in Calibre applications. All output layers are empty.

WARNING: NOT INSIDE CELL operation unsupported in ICverify applications; output layer(s) will be copies.

Issued when an Not Inside Cell operation is required in an ICverify application. This operation is only supported in Calibre applications. All output layers will be copies of the input layer.

WARNING: EXTENT CELL operation unsupported in ICverify applications; output layer will be empty.

An Extent Cell operation is required in a flat application. This operation is only supported in Calibre applications. The output layer will be empty.

WARNING: EXTENT DRAWN operation unsupported in ICverify applications; output layer will be empty.

WARNING: OPCLINEEND operation unsupported in ICverify applications; output layer will be empty.

WARNING: OPCSBAR operation unsupported in ICverify applications; output layer will be empty.

WARNING: LITHO operation unsupported in ICverify applications; output layer will be empty.

WARNING: FRACTURE operation unsupported in ICverify applications; output layer will be empty.

A fracture operation is required as part of the flow in an ICverify application. This operation is only supported in Calibre applications. The output layer will be empty.

WARNING: MDP operation unsupported in ICverify applications; output layer will be empty.

An MDP operation is required as part of the flow in an ICverify application. This operation is only supported in Calibre applications. The output layer will be empty.

WARNING: TDDRC operation unsupported in ICverify applications; output layer will be empty.

A TDDRC operation is required as part of the flow in an ICverify application. This operation is only supported in Calibre applications. The output layer will be empty.

WARNING: DFM ANALYZE operation unsupported in ICverify applications; output layer will be empty.

A DFM Analyze operation is required as part of the flow in an ICverify application. This operation is only supported in Calibre applications. The output layer will be empty.
