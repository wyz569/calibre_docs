VELEMENT

Vector function for DFM expressions

Returns the specified element of a numeric vector. Supported for the DFM Property family of operations.

Usage

VELEMENT(vector, tuple_index[, slice_index])

Arguments

vector

A required numeric vector.

tuple_index

A required argument that specifies the tuple index of the element to return. Indices are zero-based. The tuple index can be thought of as the row in a 2D array.

slice_index

An optional argument that specifies the slice index of the element to return. Indices are zero based. The slice index can be thought of as the column in a 2D array. If omitted, slice_index defaults to 0.

Return Values

Number.

Description

Returns the specified vector element.

A runtime error occurs if the vector is empty or missing, or if tuple_index or slice_index is out of range. You can protect against evaluation failure with expression chains. See “Expressions that Do Not Evaluate Successfully” and “Expression Chains”.

Vectors are described in “Vector DFM Properties”.

Examples

This example gets the vector element with tuple index 2 and slice index 5 (row 2, column 5). An expression chain is used to assign a default value in case of expression failure. The choice of default value depends on your application. 

num_layer = DFM PROPERTY vector_layer
				[ "+vec" = VPROPERTY(vector_layer, vector1)]
				[ v_2_5 =  VELEMENT(VPROPERTY_REF("+vec"), 2, 5 )] 
						[v_2_5 = -1]  // expression chain assigns default
