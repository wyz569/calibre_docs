Vector Function Summary

Several functions allow you to create, manipulate, and access vector data. The vector functions are supported only in DFM expressions in the DFM Property operation.

See “Vector DFM Properties” for a description of the vector notation. The topic also describes how to access individual vector elements using a user-defined function created by a DFM Function statement.

The vector functions are summarized in these two tables:

  • General vector functions — Table 1

  • Sorting and selection functions — Table 2

  • Functions for device property computation applications — Table 3

See “Property Access Function Summary” for a general description of the functions ending in PROPERTY and PROPERTY_REF.

Table 1. General Vector Function Summary Table

Function

Description

CONCAT

Concatenates the input vectors in order from left to right.

NETVPROPERTY

Accesses DFM properties of type vnetID (vector of net IDs) from objects on the input layer.

NETVPROPERTY_REF

References vector net number properties from a property definition in the same operation.

NETVELEMENT

Returns the specified element of a netID vector.

NETVTUPLE

Creates a vector of netID values. It is most useful in DFM Property Select Secondary, where it creates one tuple for each cluster that is grouped with the select geometry.

SVELEMENT

Returns the specified element in a vector of strings.

SVPROPERTY

Accesses DFM properties that are vectors of strings from objects on the input layer.

SVPROPERTY_REF

References vector string properties from a property definition in the same operation.

SVTUPLE

Creates a vector of string type properties. It is most useful in DFM Property Select Secondary, where it creates one tuple for each cluster that is grouped with the select geometry.

VBYNETMAX

VBYNETMIN

Returns the net ID occurring most or least often in a vector of net IDs. Additionally can determine the maximum or minimum numeric value in one vector and return the corresponding net ID from another vector. The third variant computes sums of numeric values in one vector corresponding to net IDs in another vector. The net ID corresponding to the greatest sum is returned.

VECTOR

Creates a vector of numeric type properties.

VELEMENT

Returns the specified element of a numeric vector.

VMAX

VMIN

Return maximum or minimum values from vector slices.

VNETID

Creates a vector of netID values.

VNETIDMAX

VNETIDMIN

Determines the tuple index of a maximum value of a specified slice in one vector and uses that index to return a corresponding element (a net ID) in another slice of another vector.

VNETSUM

Returns the sum of numeric values in a slice (column) of one vector corresponding to net IDs in a slice of another vector, where the net IDs match a specified net ID. The simpler form of the function is a shortened version of the longer one and assumes slice index values of 0.

VPROPERTY

Accesses vector type DFM properties from objects on the input layer.

VPROPERTY_REF

References vector numeric properties from a property definition in the same operation.

VSTRING

Creates a vector of string type properties.

VTUPLE

Creates a vector of numeric type properties. It is most useful in DFM Property Select Secondary, where it creates one tuple for each cluster that is grouped with the select geometry.
Table 2. Sorting and Selection Vector Functions Summary Table

Function

Description

FIND_SORTED

Given a sorted numeric, string, or netID vector, finds the first match to the specified tuple search argument and returns the tuple index. Tuple indexes start at 0. Supported in the DFM Property family of operations.

PRESORTED

Given an input property vector of sorted netID values, returns a vector that is sorted at the hierarchy level in which the expression evaluation takes place.

SORT

Performs a lexicographic (dictionary) sort on the input vector. Supported in the DFM Property family of operations.

SORT_INDEX

Returns a one dimensional vector of index values. The index values map the results of the function SORT() on the input vector back to the input vector indices.

SORT_UNIQUE

Performs a lexicographic, or dictionary, sort of the input vector and then removes duplicate tuples. Supported in the DFM Property family of operations.

SORT_UNIQUE_INDEX

Returns a one dimensional vector of index values. The index values map the results of the function SORT_UNIQUE() on the input vector back to the input vector indices.

UNIQUE

Removes equal adjacent tuples from the input vector. Supported in the DFM Property family of operations.

UNIQUE_COUNT

Returns the number of unique elements in a vector. Supported in the DFM Property family of operations.

UNIQUE_INDEX

Returns a one dimensional vector of index values. The index values map the results of the function UNIQUE() on the input vector back to the input vector indices.
Table 3. Vector Functions for Device Property Computations Summary Table

Function

Description

NORMALIZE_XY

Acts as a transformation function that shifts the first elements of input vector tuples by a fixed amount.

RANGE_XXY

Filters the output from SELECT_MERGE_XXY or SORT_MERGE_XXY by limiting the data to a specified interval.

SELECT_MERGE_XXY

Operates on an arbitrary number of number-vector pairs.

SORT_MERGE_XXY

Returns a vector that contains an array of measurement data sorted by interval values.
