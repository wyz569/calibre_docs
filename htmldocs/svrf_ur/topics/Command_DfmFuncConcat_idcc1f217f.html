CONCAT

Vector function for DFM expressions

Concatenates input vectors into a vector of vectors. Valid only in a DFM Property operation.

Usage

CONCAT(vector1, vector2 [, vector3 …])

Arguments

vectorN

A required vector argument. At least two vectors must be specified. Vectors must be of the same type (numeric, string, or netID) and have the same tuple size.

Return Values

A vector.

Description

Concatenates a set of input vectors into a vector of vectors. For additional information about vectors, see “Vector DFM Properties”.

Examples

This DFM Property expression:

[vec1 = CONCAT(VECTOR(0,1,1.5),VECTOR(2,3,0.5))]

produces a vector of this form in a text database, where the <> notation indicates a tuple:

vec1 <0 1 1.5> <2 3 0.5>
