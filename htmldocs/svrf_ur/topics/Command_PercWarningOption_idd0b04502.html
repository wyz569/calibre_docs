PERC Warning Option

Specification statement

Controls unused net reporting.

Usage

PERC WARNING OPTION {INCLUDE_UNUSED_NET_CONTEXT | SKIP_UNUSED_NET}

Arguments

INCLUDE_UNUSED_NET_CONTEXT

A keyword that includes contextual information in the output results for unused nets.

SKIP_UNUSED_NET

A keyword that suppresses warnings about unused nets. This keyword overrides INCLUDE_UNUSED_NET_CONTEXT if both are active in the rules.

Description

Controls warnings and information given about unused nets. This statement may be specified more than once. Unused nets are those referenced in the PERC Load INIT proc but are not found during the run.

By default, unused net names are reported with this warning:

WARNING: Unused net name specified in the INIT proc: "net_name"

When INCLUDE_UNUSED_NET_CONTEXT is used, ‘in cell "cell_name"’ is appended to the previous message when the parent cell name can be determined. Additionally, either one or the other of the following messages is appended to the previous warning when applicable:

Net removed by netlist transformation(s) in cell "cell_name"

Net reduced by netlist transformation(s); surviving net is "net_name" [in cell "cell_name"]

INCLUDE_UNUSED_NET_CONTEXT also provides information required by the perc::get_surviving_net command.

See also PERC Report Option.

Examples

This example shows a typical usage of specification statements associated with PERC Warning Option.

PERC REPORT "perc.rep"
PERC LOAD perc.func INIT init_1 SELECT rule.1 rule.2
PERC REPORT OPTION print_tvf_warning
PERC WARNING OPTION skip_unused_net
