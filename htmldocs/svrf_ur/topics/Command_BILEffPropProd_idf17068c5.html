prod

Effective property computation language function. Not used in a TVF Function block.

Returns the product of expression values from all instances in the input group.

Usage

prod(vector_expression)

Arguments

vector_expression

A required expression. Only numeric expressions are allowed. See “Numeric vector_expression Argument”.

Return Values

Number.

Examples

The following example:

QQ = PROD ( W/L+1 )

returns the value of this over all instances in the input group:

(W1/L1+1) * (W2/L2+1) * ... * (Wn/Ln+1)

See also “Effective Property Computation in Standard Form”.
