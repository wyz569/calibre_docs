PEX Inductance Minlength

Parasitic extraction

Specifies the minimum length that a path needs to exceed in order to be considered by the inductance field solver. Used only in Calibre xL and the Calibre xACT inductance extraction flow.

Usage

PEX INDUCTANCE MINLENGTH value

Arguments

value

A required parameter that specifies the minimum path length in microns. The default is 100 microns if this statement is not specified.

Description

An optional statement used to specify the minimum length that a net path must exceed in order to be considered by the inductance field solver. Only signal interconnect paths that measure a total length larger than the given value between a driver and a receiver point are considered by the inductance field solver. A driver/receiver point could be a device pin or a text port.

In Calibre xL, paths with length less than or equal to value are assigned a per unit length parasitic inductance specified with the PEX Inductance Self statement.

In Calibre xACT, any signal path that is less than the specified minimum length value is ignored and does not have parasitic inductance extracted.

Examples

Example 1

The following statement explicitly instructs the inductance tool to extract inductance for all paths, no matter what their lengths are.

PEX INDUCTANCE MINLENGTH 0

Example 2

The following statement instructs the inductance field solver to extract inductances for paths measuring more than 20 microns between driver and receiver end points.

PEX INDUCTANCE MINLENGTH 20
