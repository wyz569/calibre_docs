ERC Check Text

Specification statement

Specifies the quantity and type of comments to appear in ERC results databases.

Usage

ERC CHECK TEXT {{[COMMENTS | ALL] [RFI]} | NONE}

Arguments

COMMENTS

An optional keyword that instructs the tool to output the rule check comments to the ERC results database. This is the default behavior along with RFI.

ALL

An optional keyword that instructs the tool to output the entire text of the rule check to the ERC results database.

RFI

An optional keyword that instructs the tool to output the rule file title and pathname to the ERC results database. When the optional keyword COMMENTS is included (the default), the tool also outputs the rule file title and pathname to the ERC results database. When the optional keyword ALL is included, the tool also outputs the rule file title and pathname to the ERC results database.

NONE

An optional keyword that instructs the tool to not output any rule check text or rule file information to the ERC results database.

Description

Specifies the amount and type of text to appear in ERC results databases.

You can specify this statement once in your rule file.

Text consists of at sign (@) comments, rule file information, and rule check text. This statement allows you to include some or all of the rule check text in the ERC results database.

See also ERC Results Database.

Examples

Example 1

// Default setting.
ERC CHECK TEXT COMMENTS RFI

Example 2

// Place all ERC text in the ERC results DB.
ERC CHECK TEXT ALL

Example 3

// No ERC text in the results DB.
ERC CHECK TEXT NONE
