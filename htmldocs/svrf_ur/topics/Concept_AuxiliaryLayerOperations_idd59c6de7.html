Auxiliary Layer Operations

Any layer operation that is not a dimensional check layer operation is an auxiliary operation. The auxiliary layer operations generally use some type of Boolean or topological property to analyze edge or polygon relationships.

The operations are listed in several tables. The operations that appear in parentheses, for example (Not Touch), reflect the syntax of the logical negation of the corresponding statement in the table cell. The behaviors that occur in parentheses in the Description cells of the table reflect what these logical negations do. These operations can be used by all Calibre and ICverify applications except where noted.

Calibre RET and Litho commands are listed in the section “Post-Design Correction Statements”.

Superscripts after the operation name indicate the following: c denotes layer constructor,  s denotes layer selector (also node-preserving),  cs denotes the operation can fulfill either function.

Refer to the Calibre Administrator’s Guide for complete information about licensing.

The auxiliary layer operations are divided into polygon-directed (Table 1), edge-directed (Table 2), error-directed (Table 3), and miscellaneous (Table 4) operations. Some operations appear in more than one category.

The operations in the following table produce derived polygon layers.

Table 1. Layer Operations: Polygon-Directed Auxiliary Operations

Operation

Description

ANDc

Boolean operation that selects the polygon area from the input layers intersected by multiple polygonal areas.

Areas

(Not Area)s

Selects polygons that meet an area constraint.

Cuts

(Not Cut)s

Selects polygons that (do not) share some, but not all, of their area with polygons from a second layer.

Deanglec

Constructs a derived polygon layer by replacing skew edges with sequences of orthogonal or 45-degree edges.

Densityc

Constructs a derived polygon layer by measuring the density of an input layer within a user-specified window over a specified region of the design.

Density Convolvec

Constructs a representation of the input layer that shows density values after applying convolution algorithms.

Device Layerc

Constructs a layer that contains seed shapes from all device instances with the specified type and model name. Seed shapes are those from device_layer arguments in Device operations. Used only in LVS applications.

DFM Analyzec

Identifies areas on a layout where manufacturing yield can potentially be improved. Uses a Calibre YieldAnalyzer license.

DFM CAF

Calculates a numeric critical area value on one or more input layers and outputs a layer containing a DFM property named CAF. Uses a Calibre YieldAnalyzer license.

DFM Circle Analyzec

Characterizes how close to circular the polygons on the input layer are. By default the output layer is a copy of the input layer, with properties attached to each polygon.

DFM Classifyc

Groups geometries on a layer by one or more specified DFM properties and selects a representative geometry from each group.

DFM Contourc

Generates a derived polygon layer defined by the contour specifications in DFM Spec Contour. A set of output layers provide a contour map of a DFM property value or layer density. Uses a Calibre YieldEnhancer or Calibre YieldAnalyzer license.

DFM Copys

Copies layers of any type to a new layer, and allows selection of specific cells to copy.

DFM Create Layer Polygonsc

Creates polygons on a new layer in a DFM database using Calibre YieldServer. The polygons are generated by a dfm::new_layer command in a Calibre YieldServer script.

DFM Critical Areac

Identifies yield limiting conditions. Uses a Calibre YieldAnalyzer license.

DFM Densityc

Measures the density of the input layer(s) within a user-specified window. The density value is attached as a DFM Property to an output polygon for each window where the density value meets a specified constraint. A user-defined expression may be specified instead of the default density calculation. Uses a Calibre YieldEnhancer license.

DFM DV Override

Collects DFM property data from input layers and generates an output layer with net ID and voltage information. The output layer is used by DFM DV with the OVERRIDE keyword to override certain voltage values, typically for analog applications.

DFM DV Analyzec

Writes layer data to an ASCII format results database (RDB). The results database is organized to assist in analyzing results from a DFM DV operation.

DFM Expand Edgec

Expands edges based on DFM properties attached to the input layer. Uses a Calibre YieldEnhancer license by default.

DFM Expand Enclosurec

Generates regions of expanded enclosure for vias. Uses a Calibre YieldEnhancer license.

DFM Fillc

Creates a derived polygon layer containing fill polygons generated according to a fill specification. Uses a Calibre YieldEnhancer license.

DFM GCA

Outputs a derived polygon layer that represents areas susceptible to producing open or short circuits in the presence of random particles of a given radius. Requires a Calibre YieldAnalyzer product license.

DFM Growc

Expands polygons uniformly. Uses a Calibre YieldEnhancer license by default.

DFM Histogram

Analyzes the distribution of DFM Property values and computes a histogram. Requires a Calibre YieldAnalyzer product license.

DFM Jogcleanc

Creates an output layer from which jogs have been removed based on specified criteria. Uses a Calibre YieldEnhancer license.

DFM Measurec

Performs single-layer, table-based width and spacing checks. This operation is polygon directed if you use the REGION keyword. You can optionally output the results to a results database (RDB) for statistical analysis. Uses a Calibre YieldAnalyzer license.

DFM NARACs

Converts a layer containing DFM properties into a Net Area Ratio Accumulate (NARAC) layer.

DFM Optimizec

Generates an optimized fill layer for variable-sized rectangular fill shapes. The optimization criteria is defined by DFM Spec Optimize and DFM Spec Optimize Data. Uses a Calibre YieldEnhancer license.

DFM ORc

Performs an OR operation on the input polygon layers. DFM properties on the input layers are included in the output layer. Properties are merged if conflicting property values exist on overlapping polygons from different layers. Optional keywords specify the method for merging properties.

DFM Partitionc

Creates a polygon layer which represents the input layer broken into rectangles and triangles, meeting specific criteria for minimum size, separation, and so forth. Uses a Calibre YieldEnhancer license.

DFM Pattern Analyzec

Analyzes the symmetry of target layer shapes at marked locations. Keywords define the content of the output layer.

DFM Pattern Capturec

Captures patterns based on specified lithographic hotspots or based on a specified mask layer.

DFM Pattern Classifyc

Creates an output layer by classifying patterns based on specified seed layers.

DFM Propertyc

Associates user-defined properties to individual objects on the input layer and creates a derived layer containing only those objects that meet the specified criteria.

DFM Property Mergec

Manipulates special unmerged polygon layers created with either DFM Text, DFM Expand Edge UNMERGED, DFM Copy REGION UNMERGED, or DFM Read.

DFM Property Netc

Creates an empty polygon layer containing DFM properties from an input layer having connectivity information or DFM properties.

DFM Property Select Secondaryc

Given a primary and secondary layer inputs, clusters each geometry on the primary layer with zero or more geometries on the secondary layer(s) and writes out geometries from the selected secondary layer for which all user-defined property expressions evaluate successfully.

DFM Property Singletonc

Creates an empty polygon layer containing user-defined DFM properties calculated from one or more input layers.

DFM RDBc

Writes original or derived layer data to results database (ASCII by default). Behaves like a layer constructor operation. Uses a Calibre YieldEnhancer product license in Syntax 4 only.

DFM RDB ASCIIc

Writes original or derived layer data to a custom-formatted text file. Requires a Calibre YieldEnhancer license.

DFM RDB DEFc

Backannotates objects (such as VIA, NET, and FILL enhancement shapes) from Calibre layer operations to a new incremental or full DEF layout. Uses a Calibre RealTime Digital license.

DFM Readc

Accesses layout properties in the input database as DFM properties.

DFM Rectanglesc

Generates an output layer consisting of rectangles with the specified dimensions and spacing, where these parameters can be provided by explicit numeric values or by DFM properties attached to the input layer. The rectangles are generated within the polygons on an input layer. Uses a Calibre YieldEnhancer license.

DFM Redundant Viasc

Identifies redundant or non-redundant polygons on a layout and produces a derived polygon layer containing the identified polygons. Used with DFM Spec Via Redundancy. Requires a Calibre YieldAnalyzer license.

DFM Remove Edgec

Removes edges shorter than a specified value from the input polygons and outputs the resulting polygons. Uses a Calibre YieldEnhancer license.

DFM Select By Net

Given two input layers with connectivity information, outputs polygons from the first layer that meet an area constraint. The second input layer may be a net-indexed layer produced by DFM Property Net. The area constraint is calculated for polygons on the same node.

DFM Shiftc

Shifts the input layer based on DFM properties attached to the input layer or explicit numeric values. Uses a Calibre YieldEnhancer license by default.

DFM Sizec

Oversizes or undersizes the polygons on the input layer by the amount specified. Uses a Calibre YieldEnhancer license by default.

DFM Stampc

Produces an output layer that is a copy of the input layer, preserving any attached DFM properties.

DFM Stripec

Generates an output layer with stripes of fixed width. The stripes are placed within the shapes on a polygon layer. Uses a Calibre YieldEnhancer license.

DFM Texts

Converts text objects to DFM properties.

DFM Transformc

Transforms geometries on an input layer. Requires a Calibre YieldEnhancer license.

DFM Transitionc

Adds metal and vias to interconnect transitions. Requires a Calibre YieldEnhancer product license.

DFM Via Shiftc

Improves via location and metal coverage, reducing spacing violations. Requires a Calibre YieldEnhancer product license.

Donuts

(Not Donut)s

Selects polygons from the input layer that (do not) have interior cycles.

Encloses

(Not Enclose)s

Selects all polygons that (do not) completely enclose any polygon from a second layer.

Enclose Rectangles

(Not Enclose Rectangle)s

Selects all polygons that can (not) enclose a rectangle of the specified dimensions.

Expand Edgec

Constructs a derived polygon layer by converting edges on the input layer into rectangles according to the expansion arguments.

Expand Textc

Constructs a derived polygon layer consisting of merged squares centered on the positions of text objects having a text name.

Extentc

Constructs a derived polygon layer consisting of one rectangle that equals the database extent read in at runtime, or that represents the minimum bounding box of all shapes on the input layer.

Extent Cellc

Constructs a derived polygon layer consisting of rectangles that represent the extents of cells in the given list. Used only in Calibre.

Extent Drawnc

Constructs a derived polygon layer consisting of the extent of shapes (not text) on the specified input layers.

Extentsc

Constructs a derived polygon layer consisting of the merged, minimum bounding boxes of each polygon on layer.

Growc

Constructs a derived polygon layer where outward expansion of polygons from the input layer occurs in specified directions.

Holesc

Constructs a derived polygon layer of polygons that fit inside of holes in polygons from the input layer.

Insides

(Not Inside)s

Selects all polygons that (do not) share all of their area with a polygon from a second layer.

Inside Cellc

(Not Inside Cell)c

Selects all polygons that are (not) inside the sub-hierarchy of the specified cells. Used only in Calibre.

Interacts

(Not Interact)s

Selects all polygons that (do not) share points with polygons from a second layer.

Magnifyc

Constructs a derived layer by magnifying the polygons on the input layer by the amount specified in the numeric argument.

Nets

(Not Net)s

Connectivity-related operation that selects all layer1 polygons that are (not) on the specified nets.

Net Areas

Connectivity-related operation that selects all polygons from a layer that lie on an electrical node such that the total area of polygons on the same node satisfies the given constraint.

Net Area Ratiocs

Connectivity-related operation that selects polygons that lie on an electrical node and result in an expression value that meets the constraint.

Net Area Ratio Accumulatec

Connectivity-related operation that outputs polygons when the constraint is satisfied. The value of the operation’s expression is attached to the output layer.

Net Area Ratio Coupledc

Connectivity-related operation that allows computation of a coupled area between input layers. This can be useful when doing antenna checks involving coupled gates.

Net Interactc

Connectivity-related operation that outputs polygons that interact with the specified number of nodes associated with polygons from a second layer.

NOTc

Two-layer Boolean operation that selects polygon areas not common to polygons from a second layer. Node-preserving layer constructor.

ORc

One- or two-layer Boolean operation that merges intersecting polygons. For two-layer operations, OR finds the union of polygons on input layers. The one-layer version is rarely useful because all applications using SVRF pre-merge data by layer.

Ornetc

Connectivity-related operation that combines (without fully merging) all overlapping input polygons on the same net. Node-preserving layer constructor. Performed flat in hierarchical applications.

Outsides

(Not Outside)s

Selects all layer1 polygons that (do not) lie completely outside layer2 polygons.

Pathchkc

Constructs a derived layer consisting of layout nets that do or do not have a path to power, ground, or labeled nets, or that satisfy a combination of these conditions. Used during circuit extraction in Calibre nmLVS/Calibre nmLVS-H and ICtrace Mask mode.

Perimeter s

Selects polygons having a specified perimeter.

Pinsc

ICverify operation that generates a derived polygon layer consisting of all pin shapes on the specified original layer.

Portsc

ICverify operation that generates a derived polygon layer consisting of all port shapes on the specified original layer. The port shapes are selected (only) from the top-level cell template.

Pushc

For hierarchical applications, constructs a derived layer by pushing the input layer to the lowest possible level (that is, away from the top-level cell) of the hierarchy. In flat applications, performs like Copy.

Rectangles

(Not Rectangle)s

Selects polygons that are (not) rectangles, with optionally specified dimensions.

Rectanglesc

Constructs a derived polygon layer consisting of arrays of rectangles with specified dimensions and spacing. Used for planarization.

Rotatec

Constructs a derived polygon layer by rotating all polygons on the input by layer by the given angle. Performed flat in hierarchical applications.

Shiftc

Constructs a derived layer by translating polygons in the x- and y-directions.

Shrinkc

Constructs a derived layer by contracting polygons in the specified directions.

Sizec

Constructs a derived layer by expanding or contracting polygons by the amount specified.

Snapc

Constructs a derived layer by placing vertices on the input layer on the grid specified by the resolution argument.

Stampc

Connectivity-related layer operation that constructs a layer containing polygons that are overlapped by polygons from a second layer and transfers the nodal information from the second layer to the output.

Topexc

ICverify operation that constructs a derived polygon layer consisting of all shapes on the specified original layer that are from the top-level cell template and have external aspect.

Touchs

(Not Touch)s

Selects polygons that are (not) outside of all polygons on a second layer and (do not) share a coincident edge or portion thereof with a polygon on the second layer.

Vertexs

Selects polygons having a specified vertex count.

With Edges

(Not With Edge)s

Selects polygons that (do not) share an edge or edge segment on a second layer.

With Neighbors

(Not With Neighbor)s

Selects orthogonal rectangles that (do not) have the specified number of orthogonal rectangles within the specified distance.

With Texts

(Not With Text)s

Selects polygons that (do not) intersect the position of the specified text object.

With Widthc

(Not With Width)c

Constructs a layer containing polygons (not) meeting the specified width constraint.

XORc

One- or two-layer Boolean operation that constructs a derived polygon layer of regions contained by exactly one polygon.

Operation

Description

Selects edges whose smaller angle magnitude with the x-axis of the coordinate system satisfies (does not satisfy) the given constraint.

Performs single-layer, table-based width and spacing checks. This operation is edge-directed if you use the [ ] operators. You can optionally output the results to a results database (RDB) for statistical analysis. Requires a Calibre YieldAnalyzer license.

Performs a Boolean OR operation on edges from the input layers. This operation accepts more input and enables more control over the output the OR Edge or DFM Copy. Uses a Calibre YieldEnhancer license in most applications.

DFM Shift Edgec

DFM Stampc

Outside Edges

Path Lengths

Touch Outside Edges

The operations in the following table produce derived error layers that are non node-preserving.

Table 3. Layer Operations: Error-Directed Auxiliary Operations

Operation

Description

DFM Classifys

Groups geometries on a layer by one or more specified DFM properties and selects a representative geometry from each group.

DFM DV

Performs a voltage-dependent External operation on the input layer(s). Edge pairs that satisfy both the external constraint and the voltage constraint are output. Voltages are determined by properties attached to a net-indexed input layer.

DFM Shift Edgec

Shifts edges based on DFM properties attached to the input layer. Requires a Calibre YieldEnhancer product license.

DFM Spacec

Measures edge-to-edge distances for large (>0.5 um) distance constraints. Performs measurements and produces results that are similar to the DRC operations External, Internal, and Enclosure.

Drawn Acute

Selects acute angles on any original geometry read from the layout database.

Drawn Angled

Selects edges that are not orthogonal to the database axes on any original data read from the layout database.

Drawn Offgrid

Selects off-grid vertices on any original geometry read from the layout database.

Drawn Skew

Selects skew edges (edges which are non-orthogonal and do not have a slope of ±1) on any original geometry read from the layout database.

Offgridc

Selects off-grid vertices on the input layer.

Offgrid Directionalc

Checks input edges for their locations based upon an enclosing layer, a direction of measurement, a specified grid, and, optionally, the direction edges face.

The operations in the following table do not specifically belong in any of the preceding tables.

Table 4. Layer Operations: Miscellaneous Auxiliary Operations

Operation

Description

Copys

Selects a layer and generates a copy of it.

DFM Mergec

Generates a merged copy of the input layer. DFM Merge preserves hierarchy better than the Merge operation. For flat applications, the operation is functionally equivalent to Copy.

Inductance MIChecks

Selects geometries whose estimated mutual inductance violates a constraint.
