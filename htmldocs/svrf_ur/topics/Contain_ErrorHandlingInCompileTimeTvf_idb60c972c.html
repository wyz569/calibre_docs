Error Handling in Compile-Time TVF

The command line option ‑tvf_instrument enables extra error reporting in compile-time TVF runs. If you want to only see error reporting for an error at a specific line, you can use the tvf::set_traceback_line command. The tvf::insert_tvf_line_info command can also be used.
