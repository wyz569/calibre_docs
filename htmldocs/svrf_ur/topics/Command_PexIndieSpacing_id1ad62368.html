PEX Indie Spacing

Parasitic extraction

Controls whether spacing for in-die variation is based on edges or density.

Usage

PEX INDIE SPACING {EDGE_BASED | AVERAGE}

Arguments

EDGE_BASED

Use the spacing for the side of the polygon to calculate the bias for its edge. This is the default.

AVERAGE

Use a single spacing value to calculate the bias for both edges of the polygon.

Description

Defines a way to calculate bias for a polygon as a function of its spacing. This is a foundry predefined statement.

For more information on in-die variation, see “Handling Parasitic On-Chip Variation” in the Calibre xRC User’s Manual.
