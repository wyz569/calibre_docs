Output File Open Warnings

This section describes runtime warnings related to opening output files.
Table 1. Output File Open Warnings

Explanation

The following warnings are reported by Calibre nmDRC applications and by the ICrules CHEck DRc command:

WARNING: Cannot open summary report file <filename> for output.

The specified summary report file cannot be opened for write access. The Calibre nmDRC run will continue as if a summary report file was not specified.

WARNING: Cannot open database output file <filename> for write access.

Reported when the database is being written to binary polygon files but Calibre nmDRC is unable to open the output binary polygon file <filename> for write access.

WARNING: Cannot open NET AREA RATIO PRINT file <filename> for output.

The file name argument (<filename>) of the net area ratio print operation cannot be opened for write access. No printing occurs in this event.

WARNING: Cannot open DENSITY PRINT file <filename> for output.

The file name argument (<filename>) of the print or print only keyword in the density operation cannot be opened for write access. No printing occurs in this event.

WARNING: Cannot open NET AREA RATIO RDB file <filename> for output.

The file name argument (<filename>) of the RDB or RDB ONLY keyword in the Net Area Ratio operation cannot be opened for write access. No RDB output occurs in this event.

WARNING: Cannot open DENSITY RDB file <filename> for output.

The file name argument (<filename>) of the RDB or RDB ONLY keyword in the Density operation cannot be opened for write access. No RDB output occurs in this event.

WARNING: Cannot open DFM ANALYZE RDB file <filename> for output.

The file name argument (<filename>) of the RDB or RDB ONLY keyword in the DFM Analyze operation cannot be opened for write access. No RDB output occurs in this event.

WARNING: Cannot open DFM MEASURE RDB file <filename> for output.

The file name argument (<filename>) of the RDB or RDB ONLY keyword in the DFM Measure operation cannot be opened for write access. No RDB output occurs in this event.

WARNING: Cannot open DFM TRANSITION RDB file <filename> for output.

The file name argument (<filename>) of the RDB or RDB ONLY keyword in the DFM Transition operation cannot be opened for write access. No RDB output occurs in this event.

WARNING: Cannot open LAYOUT INPUT EXCEPTION RDB file <filename> for output.

The filename argument (<filename>) of the Layout Input Exception RDB specification statement cannot be opened for write access. No RDB output occurs in this event.

The following is reported by flat Calibre applications:

WARNING: Cannot open database input file <filename> for read access.

The layout database format is Binary or ASCII and the specified input polygon file could not be opened for read access. The run continues as if the file were empty.

The following exception is reported by Calibre nmDRC applications. Severity is controlled by Layout Input Exception Severity.

EXCEPTION: Unable to use directory <name> for disk based storage.

Disk (file) based geometry storage is to be used but Calibre nmDRC has a problem creating the specified directory or opening a file in this directory with read/write permission.

Exception name: LAYER_DIRECTORY
