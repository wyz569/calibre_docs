Rules for Writing DFM Expressions

When constructing DFM Expressions, you must follow some basic rules to ensure the expression works as intended.

  • Expressions must be enclosed in brackets ([ ]). In the case of DFM Property, the property name assignment must also be within the brackets. In the case of DFM Function, the function name is also within the brackets.

  • You can use parentheses to establish precedence in an expression. Whatever is inside the parentheses has the highest precedence in a calculation.

  • DFM Expressions within DFM Analyze, the DFM Property family of operations, and DFM Function can contain Conditional Expressions.

    When DFM Expressions contain nested conditional expressions, place the nested expression in its own set of parentheses to avoid any ambiguity about operator precedence.

  • Expression Chains can be used in DFM Property and its related operations. An expression chain provides an alternate property definition if an expression fails to evaluate.

  • The measurement functions and other built-in functions allowed in a given expression are determined by the operation in which the expression appears. Refer to “Measurement Functions” and “Reference for DFM Functions” for a more information.

  • Expressions can contain variables, which are expanded on evaluation. These variables must be defined using the Variable specification statement.

Note:

An expression that fails to evaluate does not satisfy any constraint. See “Expressions that Do Not Evaluate Successfully”.

See “DFM Expressions” for a list of the elements that can be used in a DFM expression.

Example 1

DFM ANALYZE metal poly WINDOW 100 STEP 50 
			[ AREA(metal) / ( !AREA(poly) * AREA() ) ] <= 0.25

If poly is present in the data capture window, the denominator of the expression is zero and the evaluation fails, which means the data capture window is not output. If poly is not in the data capture window, the expression is the ratio of metal area in the data capture window to the window area itself. If that ratio is less than or equal to 0.25, then the data capture window is output.

Example 2

In the following example, an SVRF variable is used along with the EW measurement function to return the separation distance between edges for MET_S edge pairs, divided by 0.5:

VARIABLE MAX_S 0.5
MET_S = EXTERNAL MET <= MAX_S
MET_PROP = DFM PROPERTY MET_S [ S = EW(MET_S)/MAX_S ]
