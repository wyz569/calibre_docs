Command Line Arguments Passed to Compile-Time TVF

A user-supplied command line argument can be passed into a compile-time TVF script by using the -tvfarg command line option. The command line itself can be accessed with the global variables calibre_argc, calibre_argv, and calibre_argv0.

User supplied command line arguments

A user-supplied command line argument is specified with the ‑tvfarg option. For example:

calibre -drc -hier -tvfarg my_argument rules

Note that the argument (my_argument) cannot contain spaces. In the compile-time TVF script, my_argument can be accessed as follows:

set my_tvf_arg [tvf::get_tvf_arg]

If no -tvfarg option is given on the command line, tvf::get_tvf_arg returns an empty string.

Global variables for command line arguments

The global variables calibre_argc, calibre_argv, and calibre_argv0 store values from the Calibre invocation line. See “Global Variables in TVF for Calibre Command Line Arguments” for details.
