Source System

Specification statement

Specifies the source database type for Calibre nmLVS, Calibre nmLVS-H, Calibre PERC, and Calibre xRC.

Usage

SOURCE SYSTEM {CNET | SPICE}

Arguments

CNET

Keyword that specifies that the source is a compiled netlist (Cnet) database.

SPICE

Keyword that specifies that the source is a SPICE, SPICE 2, HSPICE, or CDL netlist, or a Verilog netlist converted into extended SPICE form.

Description

Specifies the source (schematic) database type for Calibre applications. This statement is required for Calibre applications that execute LVS and can be specified once in your rule file. Must be used with a Source Path statement.

You can specify source database information in Calibre Interactive - nmLVS. See “Setting Up a Calibre Interactive nmLVS Run” in the Calibre Interactive (New GUI) User’s Manual.

See “SPICE Format” in the Calibre Verification User’s Manual for more information about how Calibre interprets SPICE netlists.

See also Source Primary.

Examples

SOURCE SYSTEM SPICE
SOURCE PATH "/home/source_netlists/chip.spi"
SOURCE PRIMARY "top"
