PERC Specification Statements

This section lists the specification statements for Calibre PERC. Most LVS specification statements also affect Calibre PERC.
Table 1. Specification Statements: PERC

Statement

Description

DFM Database

Specifies the path to the DFM database directory to be created. Used in LDL applications.

DFM YS Autostart

Allows Tcl code within a rule file to be executed automatically after a Calibre DFM run or prior to a Calibre YieldServer run. Used in LDL applications.

LVS Compare Case Strict

Controls enforcement of SPICE netlist case-sensitivity settings in the rule file.

LVS Spice Port Annotations

Provides access to comment-coded port annotations in a SPICE netlist. Although in the LVS family, this statement exists primarily for Calibre PERC applications.

Mask SVDB Directory

Creates a Standard Verification Database Directory (SVDB) in Calibre nmLVS and Calibre PERC applications.

PERC Constraints Path

Specifies constraint parameters for a Calibre PERC run.

PERC Group

Specifies a group name for a set of rule check procedures.

PERC LDL CD Constraint

Specifies the LDL CD current density constraint reporting limit.

PERC LDL Distribute Probe

Specifies that source and sink probe points can be automatically distributed to layers other than the original pin layer in order to give more accurate resistance path calculations.

PERC LDL Enable Cell Based Flow

Specifies whether to enable the GDS (likewise, OASIS) cell-based flow in Calibre PERC LDL runs.

PERC LDL Layout Reduce Top Layers

Specifies top-level metal routing layers to optimize LDL layout processing. This statement should be used in any LDL rule file.

PERC LDL Preserve Trivial Pins

Specifies that trivial pins are preserved in LDL runs.

PERC LDL Probe Path

Specifies the pathname of a probe mapping file, which defines probe locations for an LDL CD or P2P run.

PERC LDL Waiver Path

Specifies to use automatic waivers during a Calibre PERC LDL DRC run.

PERC Load

Specifies the initialization procedures and rule checks to execute during a Calibre PERC run.

PERC Netlist

Specifies the input netlist for Calibre PERC runs.

PERC Pattern Path

Specifies a Calibre PERC pattern file.

PERC Property

Specifies properties made available for Calibre PERC, even if the properties are not otherwise needed for LVS.

PERC Report

Specifies the pathname of the Calibre PERC report file.

PERC Report Maximum

Specifies the maximum number of reported items in the Calibre PERC report file.

PERC Report Option

Specifies the level of detail in the PERC Report.

PERC Report Placement List Maximum

Specifies the maximum number of elements in placement lists in the PERC Report.

PERC Summary Report

Specifies the pathname of the Calibre PERC summary report file.

PERC UPF Path

Specifies the pathname of a Unified Power Format file.

PERC Voltage Trace Distance Enable

Specifies whether distance accounting is performed for voltage tracing.

PERC Waiver Comment

Specifies that a comment string, along with PERC Load statement information, is written into the header section of a waiver description file generated by Calibre RVE for PERC.

PERC Waiver Path

Specifies to use waivers during the Calibre PERC run.

PERC Warning Option

Controls unused net reporting.

PEX Resistance Open Layer

Specifies layers to be treated as infinite resistance layers.

TVF Function

Defines a runtime block of Tcl code. Calibre PERC rule checks appear in this block.
