Length

Layer operation

Selects edges with a length that satisfies the given constraint.

Usage

LENGTH layer constraint

Arguments

layer

An original layer or layer set, or a derived polygon or edge layer.

constraint

A required constraint listed in the “Constraint Notation” column of Table 1 in the “Constraints” topic. The constraint must contain non-negative real numbers, which are in user units.

Description

Selects all edges of layer having length that conforms to the constraint. See also Not Length, Path Length, and Unit Length.
