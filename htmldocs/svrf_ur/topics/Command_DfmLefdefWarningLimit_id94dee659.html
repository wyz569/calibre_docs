DFM LEFDEF Warning Limit

Specification statement

Specifies the maximum number of warning messages generated for a single message type. Used only with LEF/DEF input in flows that generate a DFM database (DFMDB).

Usage

DFM LEFDEF WARNING LIMIT number

Arguments

number

A required integer that specifies the maximum number of messages generated for a single warning message type. The number is a positive integer.

Description

An optional statement that specifies the maximum number of times a warning of the same type appears in the transcript when using LEF/DEF input. The warning suppression applies to the portion of the run that builds the DFM database, up until the Calibre generation of the database from LEF/DEF is completed. If this statement is not specified, then there is no limit to the number of times the same warning message type appears. Use this statement only with the Layout System LEFDEF statement in PEX flows and in flows that generate a DFM database (DFMDB).

Examples

The following is a sample from a transcript generated by a Calibre xACT run with LEF/DEF input where the “Skipping Component...” warning message is repeated 40 times:

...
--- HDB GENERATION
WARNING: Skipping Component 'p40' as it is not connected.
WARNING: Skipping Component 'p39' as it is not connected.
WARNING: Skipping Component 'p38' as it is not connected.
WARNING: Skipping Component 'p37' as it is not connected.
WARNING: Skipping Component 'p36' as it is not connected.
...
WARNING: Skipping Component 'p05' as it is not connected.
WARNING: Skipping Component 'p04' as it is not connected.
WARNING: Skipping Component 'p03' as it is not connected.
WARNING: Skipping Component 'p02' as it is not connected.
WARNING: Skipping Component 'p01' as it is not connected.
WARNING: Net gnd in top has no DEF instance connectivity and will not be netlisted
...

To limit the generation of this warning type to only 4 times include the following statement in your rule file:

DFM LEFDEF LIMIT WARNING 4

With this statement specified, the resulting sample transcript contents are: 

...
--- HDB GENERATION
WARNING: Skipping Component 'p40' as it is not connected.
WARNING: Skipping Component 'p39' as it is not connected.
WARNING: Skipping Component 'p38' as it is not connected.
WARNING: Skipping Component 'p37' as it is not connected.
WARNING: Additional warnings and notes of the above type, if any, will be suppressed. 
WARNING: Net gnd in top has no DEF instance connectivity and will not be netlisted
...
