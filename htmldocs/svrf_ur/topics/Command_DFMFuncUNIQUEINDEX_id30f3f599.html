UNIQUE_INDEX

Vector function for DFM expressions

Returns a one dimensional vector of index values. The index values map the results of the function UNIQUE() on the input vector back to the input vector indices. Supported for the DFM Property family of operations.

Usage

UNIQUE_INDEX(vector)

Arguments

vector

A required vector argument, which may be a numeric, netID, or string vector.

Return Values

A numeric vector with tuple size 1.

Description

UNIQUE_INDEX maps the results of applying UNIQUE() to the input vector back to the original vector indices. The result of UNIQUE_INDEX(vector) is a vector with tuple size 1 and the same tuple count as the result of UNIQUE(vector). Each value in the UNIQUE_INDEX output is a zero-based index.

UNIQUE_INDEX is similar in concept to SORT_INDEX. See that command reference for discussion and an example.
