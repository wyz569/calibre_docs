PEX Specification Statements

This section lists the PEX specification statements. These statements are used in parasitic extraction applications.
Table 1. Specification Statements: PEX

Statement

Description

Capacitance Order

Determines the vertical order of layers for capacitance calculations from bottom to top.

PEX 3DIC Coupling

Specifies whether to extract the TSV-to-TSV coupling for a digital or analog circuit. This statement cannot be used without the PEX Netlist 3DIC statement. Use only in CalibreÂ xACT.

PEX 3DReference

Specifies the optional parameters to be used by the capacitance field solver.

PEX 3DReference

Specifies the optional parameters to be used by the capacitance field solver.

PEX BA Mapfile

Specifies a file containing the device and pin name mappings used to backannotate the extracted parasitics to a pre-layout simulation netlist.

PEX Bias Preferred Direction

Specifies whether the preferred or non-preferred table is used when scaling the design in the horizontal or vertical direction during extraction.

PEX Bulk Model

Specifies whether or not PEX Extract Bulk ... Ideal statement is applied during extraction. Used only in Calibre xACT and Calibre xACT 3D.

PEX Capacitance Parameters

Specifies the width and/or spacing capacitance parameters used by Calibre xRC during extraction.

PEX CMP Mode

Specifies the type of CMP mode used to handle layer thickness variation during parasitic extraction.

PEX Contact Capacitance

Switches on or off the calculations for contact capacitance.

PEX Corner

Specifies the names of the foundry-supplied process corners and is generated by the calibration process using the xCalibrate rule file generator.

PEX Corner Custom

Optional statement that specifies variations on foundry-supplied process corners. The rules calculating parasitic capacitance and resistance must include functions for process variation.

PEX DEF Extract Blockages

Controls whether Calibre xACT considers placement and routing blockage shapes during coupling capacitance extraction.

PEX DEF Map

Maps a DEF layer name to a layer name used in the SVRF rule file.

PEX DEF Port Map

Maps a DEF layer port name to a layer name used in the SVRF rule file. Used only in CalibreÂ xACT.

PEX Density Actual

Specifies whether or not to use actual width instead of an estimated actual density in the density computations for a specified layer.

PEX Density Context

Specifies the default fill density applied outside the cell boundary for the design block.

PEX Density Estimate

Specifies an estimated value of the local density of a layer in the layout.

PEX Density Mode

Specifies the boundary window handling mode for density computation.

PEX Density Outside

Specifies the pattern density of the virtual surrounding geometries for the design block.

PEX Density Window

Specifies the window size, in microns, for density computation.

PEX Elayer

Sets the geometries of the specified layers to be physically equivalent during extraction calculations on those layers.

PEX Exception Severity

Specifies an exception type and how the exception is handled when encountered during extraction through a user-selected severity value.

PEX Extract Bulk ... Ideal

Specifies how device bulk pins are connected in the extracted netlist. Used only in Calibre xACT and CalibreÂ xACT 3D.

PEX Extract Cgate

Specifies how to netlist the gate coupling capacitance for transistors. Used only in CalibreÂ xACT or the Calibre xACT 3D direct netlisting flow.

PEX Extract Enclosure

Specifies the orientation of the overlap width and length of asymmetric via layer pads. The orientation controls which via_resistance or via_resistance2 table values are used during extraction.

PEX Extract Exclude

Specifies to exclude listed net names from extraction results.

PEX Extract Floating Nets

Specifies how floating nets like metal fill are handled during extraction.

PEX Extract Include

Specifies to limit extraction to the listed net names.

PEX Extract Remove PDB

Specifies whether or not to remove an existing PDB before creating the PDB in the PDB generation step.

PEX Extract Rgate

Specifies how to calculate the gate resistance of a transistor.

PEX Extract Temperature

Specifies a temperature dependency for the extracted resistance.

PEX Fieldsolver Boundary

Specifies the boundary conditions used by the capacitance solver.

PEX Fieldsolver Cell_array

Specifies the boundary conditions used by the capacitance solver and grounds all the conductors in the image.

PEX Fieldsolver Endcap Spacing

Specifies a virtual spacing between the source/drain boundary and an endcap for field solver device extraction.

PEX Fieldsolver Environment

Specifies one or more environment variables to be passed to and used by the Calibre xACT 3D tool.

PEX Fieldsolver Mode

Specifies the capacitance extraction accuracy mode used by the Calibre xACT 3D tool.

PEX Fill Model

Specifies overrides of foundry fill model settings for a specified layer.

PEX Fin

Specifies one or more layers that contain fin geometries for FinFET modeling. Used only in the Calibre xACT and Calibre xACT 3D direct netlisting flows.

PEX Fracture_2D

Specifies to apply 2-D fracturing to a specified marker region when performing point-to-point resistance extraction. Used in Calibre PERC and Calibre xRC.

PEX Fracture Frequency

Specifies the fracture frequency needed for node insertion.

PEX Ground Layer

Specifies layers with shapes for ground regions.

PEX Ideal Xcell

Specifies whether the Calibre xRC tool should output the intentional devices inside of the xcells to the netlist during the formatting stage.

PEX Ignore Capacitance

Specifies to ignore certain types of capacitance between specific layers.

PEX Ignore Capacitance Vianame

Specifies to ignore capacitance for the specified LVS via layers during extraction.

PEX Ignore Inductance

Used to ignore any inductance for the specified layer or connections during extraction.

PEX Ignore Resistance

Used to ignore parasitic resistance for the specified layer and via connections during extraction.

PEX Ignore Resistance Marker

Used to ignore parasitic resistance between one or more specified layers and a marker layer during extraction.

PEX Indie Spacing

Controls whether spacing for in-die variation is based on edges or density.

PEX LEF Extract Cell Obstructions

Controls whether obstruction geometries are treated as layout objects and included as ground when calculating parasitic capacitance.

PEX Magnify

Supports optical shrink for in-die variation processes.

PEX Map

Specifies the mapping between a calibrated layer and one or more physical layers.

PEX Map Required

Specifies a required physical layer name that must be mapped to a calibrated layer with PEX Map.

PEX Marker

Specifies how marker layers, defined with the PEX Map statement, are applied during extraction.

PEX Netlist

Generates a netlist with parasitic elements and places it in the file specified.

PEX Netlist 3DIC

Specifies to replace a resistor on a specified layer with a subcircuit model in the output netlist.

PEX Netlist ADMS

Generates a netlist for the extracted block in an ADMS flow and stores it at the location specified.

PEX Netlist Auto_LPE

This statement controls whether or not to include the LPE properties in the CalibreView netlist.

PEX Netlist Capacitance Unit

Specifies a common unit of capacitance to apply to all parasitic netlists and reports.

PEX Netlist Character Map

Specifies to replace characters in net names.

PEX Netlist Comment

Appends a user-specified comment string to the end of the netlist header information.

PEX Netlist Connection Section

Specifies to include or omit *|I and *|P from DSPF and SPEF netlists.

PEX Netlist Create Smashed Device Names

PEX Netlist Device Resistance Model

Specifies which model to use for parasitic resistors.

PEX Netlist EMIR_RES

Specifies to netlist the dimensions of the resistance geometries inside of PCDEF cells and in layers ignored with PEX Ignore statements without extracting the resistance values. Used only in Calibre xACT and Calibre xACT 3D.

PEX Netlist Escape Characters

Specifies escape character strings used in the netlist output file.

PEX Netlist Export Ports

Specifies whether or not the formatter creates ports for internal nets.

PEX Netlist Filter

Specifies the removal of device instances from the extracted netlist.

PEX Netlist Filter Unused Option

Filters out unused device instances during the formatter stage based upon component type in either layout or source.

PEX Netlist Global Nets

PEX Netlist Groundlayer

PEX Netlist Linewrap

PEX Netlist LPE Ignore Idealnet

PEX Netlist LPE Using Extmode

Instructs the netlist formatter to obtain LPE values from the mapfile based on the command line extraction mode.

PEX Netlist Mosfet Property

PEX Netlist Mutual Resistance

Specifies whether or not the Calibre xL tool uses current-controlled current sources to model mutual resistance.

PEX Netlist Node Prefix

Specifies the delimiting character for replicated layout instances in the netlist.

PEX Netlist Short Toplevel Ports

PEX Netlist Simple

PEX Netlist Smashed_Device Delimiter

Specifies the delimiting character for smashed or flattened layout instances in the layout netlist.

PEX Netlist Subnode Section

Specifies to include or omit *|S from DSPF and SPEF netlists.

PEX Netlist Unshort Device Pins

PEX Netlist Uppercase

PEX Netlist Via Device

PEX Netlist Virtual Connect

PEX Pin Order

PEX Plug Capacitance

Specifies whether or not via capacitance between ground and other layers is extracted. Use only with Calibre xACT and Calibre xACT 3D direct netlisting flows.

PEX Probe File

PEX Reduce Analog

PEX Reduce TICER

PEX Reduce Via Resistance
