DRC Maximum Results NAR

Specification statement

Specifies a default value to be used whenever the MAXIMUM keyword is not explicitly specified for the RDB [ONLY] parameter in the Net Area Ratio operation.

Usage

DRC MAXIMUM RESULTS NAR {number | ALL}

Arguments

number

Non-negative integer that specifies the maximum number of geometric objects to output for the associated layer in any single DRC rule check in the ASCII DRC results database.

ALL

Keyword that specifies the maximum result count is 264.

Description

Specifies a default value to be used whenever the MAXIMUM keyword is not explicitly specified for the RDB [ONLY] parameter in the Net Area Ratio operation. This statement is extremely useful in newer nodes when the antennas are being debugged and there is the potential for very large Net Area Ratio RDB files.

Examples

Example 1

Limit the number of geometric objects to output for each layer to 50 per NAR RDB rule check.

DRC MAXIMUM RESULTS NAR 50

Example 2

Specify a limit of 264 for the number of geometric objects output to the RDB for each layer. 

DRC MAXIMUM RESULTS NAR ALL
