DRC Results Database Libname

Specification statement

Specifies the LIBNAME record of GDSII results databases. Used only in Calibre nmDRC and Calibre nmDRC-H.

Usage

DRC RESULTS DATABASE LIBNAME name

Arguments

name

The value of the DRC Results Database GDSII LIBNAME record.

Description

Specifies the value of the LIBNAME record in GDSII-type results databases produced by Calibre nmDRC applications. The default LIBNAME value is drc.db.

This statement may be specified at most once. The LIBNAME specification is a global results database attribute and it applies to all GDSII-type results databases created during a Calibre nmDRC run.

See also DRC Check Map.

Examples

DRC RESULTS DATABASE LIBNAME "design_mask"
// assigns design_mask as the global LIBNAME to 
// GDSII-type DRC results databases
