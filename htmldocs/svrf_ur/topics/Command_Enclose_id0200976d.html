Enclose

Layer operation

Specifies all polygons that completely enclose any polygon from a second layer.

Usage

ENCLOSE layer1 layer2 [constraint [BY NET] [EVEN | ODD]]

Arguments

layer1

A required argument specifying an original layer or layer set or a derived polygon layer. The operation selects polygons from this layer that meet the conditions of the operation.

layer2

A required argument specifying an original layer or layer set or a derived polygon layer.

constraint [BY NET] [EVEN | ODD]

An optional argument, where the constraint specifies the number of layer2 polygons or nets that are enclosed by a layer1 polygon. The constraint must be met in order for the layer1 polygon to be selected.

The constraint is one of those listed in the “Constraint Notation” column of Table 1 in the “Constraints” topic. The constraint must contain non‑negative integers. A constraint of == 0 does not produce any output; use Not Enclose instead.

  • BY NET — Specifies that constraint applies to the number of distinct nets in the set of layer2 polygons that are enclosed by a layer1 polygon. Connectivity is required on layer2.

  • EVEN | ODD — Specifies that a layer1 polygon is selected if the number of layer2 polygons that meet the constraint is an even or odd number, respectively. For example, the following operation selects polygons from layer1 that enclose four, six, or eight polygons on layer2:

    layer1 ENCLOSE layer2 >3 <9 EVEN

    It is not useful to specify these keywords if the specified constraint does not allow for the number of polygons to meet the even or odd criterion, such as with == constraints.

Description

Selects layer1 polygons that contain any layer2 polygons as a subset (that is, a logical OR of a layer1 and layer2 polygon(s) yields exactly the layer1 polygon). Edges of layer1 and layer2 polygons may be coincident and still satisfy the enclose condition. The optional constraint limits the selection of layer1 polygons according to the number of layer2 polygons that are enclosed. If either input layer is empty, there is no output.

See also Not Enclose, Enclose Rectangle, Enclosure, Rectangle Enclosure, Inside, and Interact.
