PEX Extract Floating Nets Reduced Depth

Parasitic extraction

Specifies whether or not to treat floating nets as floating when calculating their effects on signal nets and not include the floating nets in the netlist. Used only in Calibre xACT and Calibre xACT 3D.

Usage

PEX EXTRACT FLOATING NETS REDUCED DEPTH
{MAX [THRESHOLD value] | MIN}

Arguments

MAX

A required keyword used to specify that floating nets such as metal fill should be treated as floating when calculating their effects on signal nets. The floating nets are not netlisted, only their effects are included on the signal nets. This is the default.

THRESHOLD value

Optional keyword set used to specify a capacitance threshold value. The value must be specified in units of femtofarads. Capacitance detected below this threshold is ignored. The default value is 1e-4 femtofarads if this keyword set is not specified. Can only be specified with MAX.

MIN

A required keyword used specify a lower accuracy mode where floating nets are treated as grounded during extraction and not listed in the netlist.

Description

An optional statement that specifies whether or not floating nets such as metal fill should be treated as floating when calculating their effects on signal nets. By default, the floating nets are not netlisted, only their effects are included on the signal nets. Use this statement to capture the coupling effects for floating nets including metal in designs where signal lines are spaced far apart. If this statement is specified with the MIN keyword, then floating nets are treated as grounded during extraction and not listed in the netlist.

When this statement is specified for lumped or distributed RCC extraction, the netlist size may increase, sometimes significantly, due to the introduction of new coupling capacitors. If the netlist is too large to simulate, reduce the number of parasitic capacitors with the PEX Reduce CC statement. If this statement is specified with selected net extraction, the floating nets around the selected nets must also be selected or they are assumed to be grounded.

Examples

Example 1

This example shows how applying PEX Extract Floating Nets Reduced Depth MAX controls extracting metal fill.

Figure 1 shows a simplified layout for metal fill between two nets.

Figure 1. Simplified Layout for Metal Fill Between Two Nets

For this example, C1 = 15 ff, C2 = 25 ff, C3 = 0.001 ff, and C4 = 0.001 ff which results in Ceff = 0.0005 ff. To extract floating net capacitance that is larger than 0.0004 ff, specify the statement: 

PEX EXTRACT FLOATING NETS REDUCED DEPTH MAX THRESHOLD 4e-4

In this case, Ceff, as shown in Figure 2, appears in the resulting network.

Figure 2. Depth MAX THRESHOLD 4e-4

If the THRESHOLD is set to 6e-4:

PEX EXTRACT FLOATING NETS REDUCED DEPTH MAX THRESHOLD 6e-4

then Ceff is not extracted as shown in Figure 3 because it is smaller than the specified threshold.

Figure 3. Depth MAX THRESHOLD 6e-4

Example 2

To extract a smaller netlist with reduced accuracy specify the following statement:

PEX EXTRACT FLOATING NETS REDUCED DEPTH MIN
