LVS Apply Unused Filter

Specification statement

Specifies to apply an unused device filter to a set of cells specified in a list.

Usage

LVS APPLY UNUSED FILTER filter_name CELL LIST list_name

Arguments

filter_name

A required filter name specified in an LVS Configure Unused Filter statement.

CELL LIST list_name

A required parameter set that references a list_name specified in an LVS Cell List statement.

Description

Specifies to apply the filter_name to the cells associated with the list_name. This statement, together with LVS Configure Unused Filter and LVS Cell List, are used for cell-specific unused device filtering.

Multiple LVS Apply Unused Filter statements applied to the same list_name are not allowed.

The cells in the list_name are hcells and conform to LVS Cell List conditions. In order to apply a cell-specific LAYOUT filter (from LVS Configure Unused Filter) to a layout hcell, it is necessary to apply it to the source cell that corresponds to the layout cell.

Examples

In this example, my_filter is applied to cell A, which is a layout cell. LVS Filter Unused Option K is used as the filtering condition. Note that list1 references cell B, which is a source cell. This is typical of how LVS Cell List functions.

HCELL A B
LVS CONFIGURE UNUSED FILTER my_filter K LAYOUT
LVS APPLY UNUSED FILTER my_filter CELL LIST list1
LVS CELL LIST list1 B
