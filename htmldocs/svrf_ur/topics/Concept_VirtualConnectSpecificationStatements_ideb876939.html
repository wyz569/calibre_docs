Virtual Connect Specification Statements

This section summarizes the Virtual Connect specification statements. These form a single net from two or more (disjoint) nets that either share the same (case insensitive) name, or a portion of a name when a colon (:) or semicolon (;) is present. Used by all Calibre and ICverify applications except where noted.
Table 1. Specification Statements: Virtual Connect

Statement

Description

Virtual Connect Box Colon

Specifies virtual connections for identical net names containing a colon (:) in LVS Box cells. Does not apply to LVS Box BLACK. Used only in Calibre nmLVS-H and Calibre xRC.

Virtual Connect Box Name

Specifies virtual connections for matching net names in LVS Box cells. Does not apply to LVS Box BLACK. Used only in Calibre nmLVS-H and Calibre xRC.

Virtual Connect Colon

Specifies identical net names containing a colon (:) to be virtually connected.

Virtual Connect Depth

Specifies the hierarchical depth at which virtual connections are made.

Virtual Connect Incremental

Allows incremental virtual connections within all Connect zones in Calibre nmDRC-H flows.

Virtual Connect Name

Specifies matching net names to be virtually connected.

Virtual Connect Report

Directs all Calibre and ICverify applications to report a warning (Calibre nmDRC applications) or a note (other applications) when a virtual connection is made by the connectivity extractor.

Virtual Connect Report Maximum

Specifies the maximum number of virtual connect messages issued when virtual connect reporting is enabled.

Virtual Connect Semicolon As Colon

Specifies identical net names containing a semicolon (;) to be virtually connected.
