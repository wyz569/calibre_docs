SVRF Version Statement Errors

This section describes compilation errors from the SVRF Version statement.
Table 1. SVRF Version Statement

Error #

Explanation

VER1

The parameter of a SVRF Version specification statement is not recognizable as a valid Calibre version string. The string is not of the form vDDDDPD_DD… DPDD…D where v is literal, D is any decimal digit, and P is a decimal point.

VER2

The Calibre version predates that specified in an SVRF Version specification statement.
