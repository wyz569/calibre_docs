tvf::do_echo_svrf

Application: compile-time and runtime

Writes a string to the output location specified by the tvf::echo_svrf command.

Usage

tvf::do_echo_svrf string

Arguments

string

A required argument that specifies the string to output.

The location of the output is specified with the tvf::echo_svrf command.

Return Values

None.

Examples

Example 1: Runtime TVF

This runtime example echoes a string to stdout. 

TVF FUNCTION my_function [/*
tvf::echo_svrf 1 
tvf::do_echo_svrf "entering my_function"
# function code ...
tvf::echo_svrf 0
*/]

Example 2: Compile-Time TVF

This compile-time example echoes a string to the output file specified with tvf::echo_svrf.

#!tvf

set file_id [open "svrf_tvfecho.output" w]
tvf::echo_svrf 1 $file_id
tvf::do_echo_svrf "// echo tvf to svrf"

# TVF file ...
tvf::echo_svrf 0
