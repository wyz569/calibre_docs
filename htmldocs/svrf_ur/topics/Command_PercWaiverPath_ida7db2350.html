PERC Waiver Path

Specification statement

Specifies to use topological waivers during the Calibre PERC run.

Usage

PERC WAIVER PATH filename [ABORT UNWAIVABLE]

Arguments

filename

A required pathname of a waiver description file.

The filename parameter can contain environment variables. See “Environment Variables in Pathname Parameters”.

ABORT UNWAIVABLE

An optional keyword that specifies a run will abort if a PERC Load SELECTTYPE UNWAIVABLE statement is specified, and the tool attempts to waive results for rule checks in that statement.

Description

Specifies that the Calibre PERC run should use topological waivers. A Calibre Auto-Waivers license is required to use waivers. See the Calibre Administrator’s Guide for licensing information. This statement may be specified only once in the rule file.

A result is waived if it meets the conditions of at least one waiver statement in the waiver description file. The waiver description file can be generated with Calibre RVE or written manually using a text editor. See “Topology Waiver Description File Format” in the Calibre PERC User’s Manual for details.

Calibre PERC topological waivers are discussed in detail under “Topology Waiver Application” in the Calibre PERC User’s Manual. The format of the waiver data in the PERC Report is discussed under “Waived Topology Result Report” in the same manual.

To apply geometry waivers in an LDL flow, see PERC LDL Waiver Path.

Examples

PERC REPORT "perc.rep"
PERC LOAD perc.func INIT init_1 SELECT rule.1 rule.2
PERC WAIVER PATH "PERC.waivers"
