tvf::COPY

Application: compile-time only.

Identical to the SVRF Copy operation.

Usage

tvf::COPY layer

Arguments

layer

A required argument that specifies the name of the layer to copy.

Return Values

None.

Description

The tvf::COPY command is identical to the SVRF Copy operation.

Although most layer operations are not mapped to TVF commands, Copy is supported for historical reasons. Layer assignment and output is generally done with the tvf::SETLAYER and tvf::OUTLAYER command.
