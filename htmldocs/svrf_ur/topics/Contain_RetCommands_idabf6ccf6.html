RET … Commands

The SVRF commands beginning with “RET” are used in resolution enhancement and post-design correction. Most RET commands are described in detail in the appropriate user or reference manual.
  • RET Cellarray
  • RET DPSTITCH
  • RET FLARE_CONVOLVE
  • RET [EUV] MBSRAF
  • RET NMDPC
  • RET PXOPC
  • RET Refine
  • RET SBAR
