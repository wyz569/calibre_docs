DFM Transform

Layer operation

License: Calibre YieldEnhancer

Performs geometric transformations on an input layer.

Usage

DFM TRANSFORM layer [ROTATION angle [CENTER x_center y_center]]
[FLIPX y_offset] [FLIPY x_offset]

Arguments

layer

A required polygon layer.

ROTATION angle

An optional keyword set that specifies the angle of rotation in degrees. Positive values define a counter-clockwise rotation; negative values define a clockwise rotation.

CENTER x_center y_center

An optional keyword set that defines the center of rotation in database units. If CENTER is not specified, the database origin is the center of rotation. This keyword can be used only in conjunction with ROTATION.

FLIPX y_offset

An optional keyword set that reflects the geometric coordinates across a horizontal line specified by y_offset, which is specified in user units. If y_offset is 0, coordinates are reflected across the x-axis. The offset value is measured from the origin (0,0).

FLIPY x_offset

An optional keyword set that reflects the geometric coordinates across a vertical line specified by x_offset, which is specified in user units. If x_offset is 0, coordinates are reflected across the y-axis. The offset value is measured from the origin (0,0).

Description

Transforms geometries on an input layer and produces an output layer containing the transformed geometries. Transformations include rotation about a specified point of origin and reflection across the x‑ and y‑axes with a specified offset. The input layer is flattened.

Note:

This operation is intended to be used in special cases such as metal fill replication or for other DFM enhancements on layouts containing several identical parts. It is not intended to be used for regular layer derivations, because it creates flat output layers.

Examples

Example 1

m1_flipx = DFM TRANSFORM M1_fill FLIPX 0
// flips geometries on layer M1_fill across the x-axis.

Example 2

m1_rot90 = DFM TRANSFORM M1_fill ROTATION 90 CENTER 0.2 0.1
// rotates geometries on layer M1_fill 90 degrees counterclockwise
// about the point x = 0.2, y = 0.1.
