PROPERTY

LVS Property Initialize computation language function.

Defines numeric properties to be initialized.

Usage

PROPerty prop_name1[, prop_name2 …]

Arguments

prop_name1

A required numeric device property name. If more than one is specified, then names are separated by a comma (,) followed by whitespace. Property-name/SPICE-parameter combinations such as instpar(w) are not accepted. To handle such combinations use the LVS Property Map specification statement.

Return Values

Number.

Description

This statement is used to declare the names of the generated numeric properties for which values should be assigned to the input instance. The PROPerty statement precedes the optional DEBUG and WARN statements. The PROPerty statement is required.

Property-name/SPICE-parameter combinations such as instpar(w) are not accepted. These parameters typically appear in Pyxis designs. To handle such combinations use the LVS Property Map specification statement.

You should choose property names that are SPICE compliant. See “SPICE Syntax Conventions” in the Calibre Verification User’s Manual for additional information. The rule file compiler allows you to specify property names that are not SPICE compliant, so care should be taken in choosing names.
