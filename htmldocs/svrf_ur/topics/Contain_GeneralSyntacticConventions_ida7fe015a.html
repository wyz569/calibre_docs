General Syntactic Conventions

At the lowest level, all elements in the rule file (except pre-processor directives, macros, and Include statements, which are processed in the first phase of compilation) are either keywords, numbers, string constants, or names. The topics in this section discuss the details for command parsing, numeric expressions and constraints, and variable usage.
