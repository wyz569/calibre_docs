Not Donut

Layer operation

Selects polygons from the input layer that do not have interior cycles (or holes).

Usage

NOT DONUT layer [constraint]

Arguments

layer

A required original layer, layer set, or a derived polygon layer.

constraint

An optional constraint that specifies a condition on the number of interior cycles. A polygon on layer is selected if the number of interior cycles does not meet the constraint. The constraint is one of those listed in the “Constraint Notation” column of Table 1 in the “Constraints” topic. The constraint must contain non-negative integers.

Description

Selects all layer polygons that do not have interior cycles (or holes). If a constraint is used, polygons that do not meet the constraint are selected. This is the complement of the Donut operation. See the Donut reference page for a description of cycles.
