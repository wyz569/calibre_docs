EWP

DFM measurement function

Returns the minimum distance between an edge pair on the input layer if the edges are parallel or fails to evaluate if they are not parallel.

Usage

EWP(layer)

Arguments

layer

A required error layer with one or two edges per cluster.

Return Values

Floating-point number in user units.

Description

Returns the minimum separation distance between pairs of edges in an edge cluster that are parallel to each other. (The edges do not have to project onto each other.) Evaluation fails for non-parallel pairs and no value is returned.



Because evaluation fails when EWP finds non-parallel edge pairs, EWP can prevent result output. Where this is a concern, provide a default value for the function in an expression.

Examples

An expression chain in a DFM Property operation provides a default value in case of evaluation failure. In the following example, the property ewp_prop is assigned the value -1 if EWP fails to evaluate. 

layer1_errors = EXT layer1 < 10 ABUT < 90 SINGULAR
layer1_prop =  DFM PROPERTY layer1_errors 
                 [ewp_prop = EWP(layer1_errors)]
                 [ewp_prop = -1] // default in case EWP fails to evaluate
