max

Effective property computation language function. Not used in a TVF Function block.

Returns the maximum value for an expression from all instances in the input group.

Usage

max(vector_expression)

Arguments

vector_expression

A required expression. It may be numeric- or string-valued. See “Effective Property Computation Vector Functions”.

Return Values

Number.

Description

Evaluates vector_expression across all instances in the input group and returns the maximum expression value found. Either numeric or string arguments are allowed.

Recall that a string-type vector_expression argument may only consist of a single string property name. For string-type vector_expression arguments, the function selects the lexicographically largest string value found for the specified property in the input group (the group of devices being reduced together).

Case sensitivity for comparisons is determined by the VALUES setting in the LVS Compare Case specification statement.

Examples

The following example:

EFFECTIVE L
L = MAX( L )

returns the largest Li found in the input group (Li represents the value of L for the ith instance in the group). Largest is defined numerically if L is numeric, or lexicographically if L is string-type.
