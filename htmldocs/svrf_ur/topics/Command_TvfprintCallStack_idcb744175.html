tvf::print_call_stack

Application: runtime only.

Prints a trace of all TVF layer operation calls during runtime.

Usage

tvf::print_call_stack { 0 | 1 }

Arguments

0 | 1

A required argument choice that specifies whether to generate output (1) or not (0).

Return Values

None.

Description

This command specifies whether to print a trace of all TVF layer operation calls during runtime. This can be useful during debugging, but can generate many lines of output. Executing “tvf::print_call_stack 0” after a given number of loop iterations may help to reduce unwanted trace output.

The output looks like this: 

--- TVF:1 tvf::SETLAYER one_not_two = @0 NOT @1
Traceback forced at TVF layer operation number 0
2: traceLayerOperation {tvf::SETLAYER one_not_two = @0 NOT @1}
1: tvf::SETLAYER one_not_two = {@0 NOT @1}
--- TVF:2 Calling Drc_new_edge_collection 1 keep
Traceback forced at TVF layer operation number 0
7: traceLayerOperation Calling Drc_new_edge_collection 1 keep
6: getNewLayer type_1
5: NOT @0 @1
4: LeafLayerOperation { @0 NOT @1}
3: parseImplicitLayerDefinition { @0 NOT @1} false top-level
2: tvf::parseLayerDefinition { @0 NOT @1}
1: tvf::SETLAYER one_not_two = {@0 NOT @1}

Examples

tvf::print_call_stack 1; # trace all layer operation calls
