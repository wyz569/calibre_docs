DRC_NE

Function for DFM expressions

Performs a not equal to (!=) comparison with a built-in tolerance of 10e‑8.

Usage

DRC_NE(a, b)

Arguments

a

A required numeric value, or an expression that evaluates to a numeric value. The expression can include DFM functions.

b

A required numeric value, or an expression that evaluates to a numeric value. The expression can include DFM functions.

Return Values

Floating-point value.

Description

Performs a not equal to (!=) comparison with a built-in tolerance of 10‑8. The comparison is suitable for use when comparing geometric measurements that may be affected by grid-snapping.

DRC_NE returns 1.0 if a and b are not equal within an absolute or relative tolerance of 10e‑8, and returns 0.0 otherwise.

See “DRC Constraint Comparison Functions” for a general discussion of the DRC comparison functions, including a discussion of relative and absolute comparison. DRC_NE provides the same comparison behavior within a property definition as the != constraint in a layer operation.

See “Equality Tests and Comparisons for Floating-Point Values” for relevant discussions of floating-point arithmetic.
