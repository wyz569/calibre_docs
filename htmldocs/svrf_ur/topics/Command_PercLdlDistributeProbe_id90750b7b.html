PERC LDL Distribute Probe

Specification statement

Specifies that source and sink probe points can be automatically distributed to layers other than the original pin layer in order to give more accurate resistance path calculations. This applies to LDL CD and P2P checks.

Usage

PERC LDL DISTRIBUTE PROBE {ALL | layer}

Arguments

ALL

A keyword that specifies that any parasitic resistance layer can be the target of probe point distribution.

layer

A name of a parasitic resistance layer.

Description

In certain design processes, using a single source or sink probe point on a given layer can lead to overly pessimistic resistance path values in current density and point-to-point resistance checks. Specifying PERC LDL Distribute Probe allows a probe point that would ordinarily be at a single location on one layer to be split into several probe points on some other layer. Use of this statement is typically based upon a foundry’s process recommendation.

For a situation where this statement applies, assume a well region is surrounded by a tap ring. Calculating the resistive path to one point that intersects the well can lead to an inaccurate path resistance. If the probe point can be distributed to several points on the tap ring, then the resistive path can be calculated more accurately.



Probe points can only be distributed to layers for which parasitic resistance is calculated. Specifying ALL means probe points can be moved to any resistance layer. Specifying a layer explicitly means probe points can be distributed only to that layer. This statement can be used more than once to specify multiple target layers for probe point distribution.

Examples

// enable moving of probe points to ntaps
PERC LDL DISTRIBUTE PROBE ndiff
