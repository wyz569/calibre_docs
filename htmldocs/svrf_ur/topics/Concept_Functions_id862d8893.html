Functions

ICverify application functions referred to in the SVRF Manual start with an optional “$” character and use italic font to indicate an argument that is replaced by the user.

Siemens EDA application functions are structured like this example:

[$]function_name(arg_1, arg_2, arg_3)

The following documentation conventions apply to functions, as demonstrated by the preceding example:

  • The name of the function may begin with a dollar sign ($). Some functions have no leading dollar sign.

  • Italicized arguments are replaced by the user. Non-italicized arguments are literal.
