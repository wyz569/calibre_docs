DRC Unselect Check

Specification statement

Specifies rule check statements or rule check groups to be excluded by Calibre nmDRC.

Usage

DRC UNSELECT CHECK rule_check [rule_check …]

Arguments

rule_check

A required argument that specifies a rule check name or group. You can specify this argument any number of times for each statement.

Description

Assists in rule check selection for the Calibre nmDRC application. By default, the tool selects all Calibre nmDRC rule checks for execution when the rule file is compiled. This statement modifies this selection process according to the following sequence:

  1. The tool selects all rule checks for execution if there are no DRC Select Check specification statements in the rule file. Otherwise, only those rule checks specified in DRC Select Check statements are selected.

  2. The tool unselects all rule checks specified in DRC Unselect Check specification statements in the rule file, if specified.

This statement can be useful for temporarily skipping a rule check that requires much time or memory to complete. The selected and unselected Calibre nmDRC rule checks are shown in the transcript after rule file compilation in a Calibre run. The Calibre nmDRC summary report lists unselected checks as “NOT EXECUTED”.

You can specify which checks to run in Calibre Interactive nmDRC. See “Specifying the Checks to Execute in a Calibre Interactive nmDRC Run” in the Calibre Interactive (New GUI) User’s Manual.

Implications for Mask Results Output

The rule checks that are active for a given run comprise the check set. If you change the check set by using DRC Unselect Check or similar statements, this can change the layers that are read in from the layout database and the layer derivations that are executed for the run. Layers (both original and derived) that are not needed for the check set are not processed. This can affect which layers are output, the extents of cells, and the hierarchy of any mask (geometric) results database.

See “Rule Check Selection and Mask Results Databases” in the Calibre Verification User’s Manual for additional information.

See also Group, DRC Recon Remove Check, and DRC Unselect Check By Layer.

Examples

DRC UNSELECT CHECK contact_1a contact_1b

Alternatively, you can do this:

GROUP contact_checks contact_?
DRC UNSELECT CHECK contact_checks
