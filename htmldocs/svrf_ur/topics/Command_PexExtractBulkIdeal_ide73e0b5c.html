PEX Extract Bulk ... Ideal

Parasitic extraction

An optional statement that controls how device bulk pins are connected in the extracted netlist. Used only in Calibre xACT and Calibre xACT 3D.

Usage

PEX EXTRACT BULK {‘:’pin_name  | layer} [{‘:’pin_name  | layer}...] IDEAL

Arguments

‘:’pin_name

Specifies the name of a device bulk pin. A colon ( : ) must precede the pin_name. This specified bulk pin will be connected to the port node instead of the substrate plug node. The same operation is applied to all the bulk layers that pin_name is connected to.

layer

Specifies an LVS layer name. All device bulk pins common to the specified LVS layer are connected to the port node instead of the substrate plug node.

ALL
Note:

As of the 2020.4 release, the ALL keyword has been deprecated. If the statement PEX EXTRACT BULK ALL IDEAL is specified, then a warning is generated and the statement is ignored.

Specifies that device bulk pins on all layers are connected to the port node instead of the substrate plug node.

Description

The PEX Extract Bulk IDEAL statement is an optional statement that controls how bulk pins are connected in the extracted netlist. This statement disconnects the device bulk pin from the substrate tap in the RC network and attaches it to an ideal ground net name. Use this statement when the design is sensitive to bulk or ground coupling. At least one bulk pin name or bulk layer name must be specified. This statement can be specified more than once or you can specify multiple space-separated bulk pin names and bulk layer names in one statement.

Examples

Example 1

Figure 1 shows a resistor network where the M1 device bulk pin is connected to a specific substrate tap R3 with net M1:b.

Figure 1. Bulk Pin Plug Node Connection

Here is a sample from the extracted netlist that shows the net M1:b connection:

...
XM1 M1:d M1:g M1:s M1:b pch_ulv_mac L=8e-09 nfin=3 ppitch=1 AD=3.74e-15
...

To disconnect the M1 bulk pin from the substrate tap node and connect it to an ideal net on LVS layer psub, include the following statement in your rule file:

PEX EXTRACT BULK psub IDEAL

Figure 2 shows the resulting resistor network where the bulk pin is now connected to ideal net VSS.

Figure 2. Bulk Pin Ideal Net Connection

The resulting sample from the extracted netlist is:

...
XM1 M1:d M1:g M1:s VSS pch_ulv_mac L=8e-09 nfin=3 ppitch=1 AD=3.74e-15
...

Example 2

To disconnect bulk pins, :b, :dnw, and :sub, and all bulk pins on LVS layers psub and nxwell, include the following statement in your rule file:

PEX EXTRACT BULK :b psub :dnw :sub nxwell IDEAL
