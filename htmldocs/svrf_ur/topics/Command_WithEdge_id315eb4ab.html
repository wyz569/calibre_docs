With Edge

Layer operation

Selects polygons on the input layer that share an edge or edge segment on a second layer.

Usage

WITH EDGE layer1 layer2 [constraint]

Arguments

layer1

A required original layer or layer set, or a derived polygon layer.

layer2

A required derived edge layer.

constraint

An optional constraint listed in the “Constraint Notation” column of Table 1 in the “Constraints” topic. Specifies the number of edges or edge segments a layer1 polygon must have on layer2 to be selected by the With Edge operation. The constraint must contain non-negative integers. While a constraint value of 0 is allowed, it will always return an empty layer.

Description

Selects all layer1 polygons that share edges or edge segments on layer2. The constraint limits selection of polygons from layer1 according to the number of edges or segments that the polygon has in layer2. See also Not With Edge.

Note:

When using a constraint, be aware that collinear edge segments on a single layer are counted individually and not as a single edge. Refer to “Edge Breaking” in the Calibre Verification User’s Manual.

Examples

Example 1

The With Edge operation shown in the following figure selects all met1 polygons that have edges longer than three units.

Figure 1. With Edge

Example 2

Find all metal polygons having at least one edge longer than 100 microns.

x = metal LENGTH > 100
long_metal_polygons = metal WITH EDGE x

Example 3

Find all metal polygons having a width less than 3 microns.

x = INTERNAL [metal] < 3
thin_metal_polygons = metal WITH EDGE x

Example 4

Find all contacts that have inside coincident edges with metal.

y = contact COINCIDENT INSIDE EDGE metal
contact_touch_inside = contact WITH EDGE y

Example 5

Find contacts with exactly two edges on layer y.

z = contact WITH EDGE y == 2
