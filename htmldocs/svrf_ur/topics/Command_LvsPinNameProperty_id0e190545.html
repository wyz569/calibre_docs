LVS Pin Name Property

Specification statement

Specifies prioritized list of zero or more ICtrace property names to be used in determining pin names for source instance pins. Used only in ICtrace.

Usage

LVS PIN NAME PROPERTY name [name …]

Arguments

name

A required name of a property. You can specify name any number of times in this statement. The name can be a string variable.

Description

Specifies a prioritized list of property names to be used in determining pin names for source instance pins in ICtrace. This statement can be specified any number of times. The default is no statement, but “pin” is always implied as the last property name in the list.

In Pyxis Layout, this statement sets the value of application variable lvs_pin_name_property.

Examples

/* Use this property name for determining instance pin names in the source. */
LVS PIN NAME PROPERTY phy_pin
