tvf_string_function

Device TVF data access function.

Provides access to a string value returned by a Tcl procedure defined within a TVF Function definition.

Usage

TVF_STRing_FUNction::tvf_func::tcl_proc( parameter [, parameter …] )

Arguments

tvf_func

Required name of a TVF Function.

tcl_proc

Required name of a Tcl procedure in the tvf_func block.

parameter

Required name of a parameter that is passed to the tcl_proc. More than one parameter is specified in a comma-delimited list.

Return Values

A string value from the tcl_proc.
