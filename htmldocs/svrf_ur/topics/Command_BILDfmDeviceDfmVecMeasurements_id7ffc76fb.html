device::dfm_vec_measurements

Device DFM Property data retrieval function.

Converts DFM-style rectilinear measurement vectors into Device enclosure-vector-style width and length measurements in database units.

Usage

device::dfm_vec_measurements object

Arguments

object

A required dfm_vector_value object.

Return Values

Vector of numbers.

Description

The device::dfm_vec_measurements function is used in conjunction with the TVF Functional Interface for Device Property Calculation. It must be used in a TVF Function block. The device::scaled_dfm_vec_measurements description contains a complete explanation of device::dfm_vec_measurements behavior and related functions.

The enclosure_* functions of the DEVice statement’s built-in language return values in user units, while DFM Property measurement functions return them in database units. In the latter case, dfm::scaled_dfm_vec_measurements can be more convenient because it can scale the measurements to user units.

The device::dfm_vec_measurements function is used when there is only one polygon from which measurement values need to be retrieved and that interacts with a seed shape. The device::dfm_vec_arr_measurements function is used when there is more than one polygon interacting with the seed shape.

Examples

See “Examples” under device::enclosure_measurements and device::enclosure_parallel_measurements.
