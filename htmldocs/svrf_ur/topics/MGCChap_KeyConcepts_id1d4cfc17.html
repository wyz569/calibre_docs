Key Concepts

This chapter covers the basic concepts needed to write and use a SVRF rule file for the Calibre and ICverify products.

“Preface” summarizes the products referred to in this manual and provides syntax and other documentation conventions.

When there is no need to distinguish between applications in Calibre, the term Calibre is used. When there is no need to distinguish between Calibre nmDRC, Calibre nmDRC-H, and ICrules, the term Calibre nmDRC or DRC is used. Calibre nmLVS, Calibre nmLVS-H, and ICtrace are handled in a similar manner.
