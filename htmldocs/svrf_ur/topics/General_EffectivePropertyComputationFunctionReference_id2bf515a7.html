Effective Property Computation Function Reference

A variety of built-in functions facilitate device reduction. You can use these functions within numeric, string, or complex expressions. Numeric results can be used as an argument to a relational operator.

The following table describes effective property computation functions.

Table 1. Effective Property Computation Built-In Functions

Function

Description

EFFECTIVE

Defines numeric properties to be calculated.

EFFECTIVE STRING

Defines string properties for evaluation during reduction.

Debugging functions

DEBUG

Enables debug monitoring during effective property computation.

WARN

Issues a warning to the transcript when the effective property computation program attempts an impossible calculation.

Standard form functions

count

Returns the count of instances in an input group.

equal

Tests if all instances in the input group return the same value for an expression.

max

Returns the maximum value for an expression from all instances in the input group.

min

Returns the minimum value for an expression from all instances in the input group.

prod

Returns a product of expression values from all instances in the input group.

sum

Returns a sum of expression values from all instances in the input group.

Runtime TVF functions

These functions are used in TVF Function blocks. Refer to “Effective Property Computation Functions for TVF”.

foreach_instance

Iterates over all instances in the input group and executes a Tcl script.

set_effective_property

Assigns an effective property value and returns it to the calling environment.

is_missing

Tests whether a property is missing from the input netlist. This function has limited use and is typically not needed.

is_unknown

Tests whether a property is assigned the UNKNOWN value. This function has limited use and is typically not needed.
