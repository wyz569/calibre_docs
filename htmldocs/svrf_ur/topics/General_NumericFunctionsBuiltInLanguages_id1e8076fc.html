Numeric Functions for Built-In Languages

The following are the numeric functions available in the built-in languages.
Table 1. Numeric Functions for Built-in Languages

Function

Description

Absolute value

 
ABS(numeric-expression)

Returns the absolute value of the specified numeric expression.

Exponent

 
EXP(numeric-expression)

Returns the value of e (Napierâ€™s constant, the base of natural logarithms) raised to the power of the numeric expression.

Logarithm

 
LOG(numeric-expression)

Returns the natural logarithm of the specified numeric expression. The value of the numeric expression must be greater than zero.

Power

 
POW(numeric-expression, numeric-expression)

Returns the value of the first expression raised to the power of the second expression. If the first expression is zero, the second must be positive. If the first expression is negative, the second expression must be an integer.

Square root

 
SQRT(numeric-expression)

Returns the positive square root of the specified numeric expression. The value of the numeric expression must be non-negative.

Truncate

 
TRUNC(numeric-expression)

Returns the result of truncating the fractional part of the specified numeric expression. Truncation is toward zero. The absolute value of the numeric expression must not exceed 2,147,483,647.
