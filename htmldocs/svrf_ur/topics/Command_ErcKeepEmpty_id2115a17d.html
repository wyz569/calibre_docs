ERC Keep Empty

Specification statement

Specifies whether empty rule checks (rule checks with zero results) are retained in ERC results.

Usage

ERC KEEP EMPTY {YES | NO}

Arguments

YES

A keyword that instructs the tool to retain rule checks containing zero results in ERC execution and to write them to the ERC results database. This is the default if you do not include this statement in the rule file.

NO

A keyword that instructs the tool not to output rule checks containing zero results to the ERC results database.

Description

Specifies whether rule checks with zero results are retained in ERC runs.

You can specify this statement once in your rule file.

This statement applies only to an ERC Results Database.

Examples

// default. keep empty checks.
ERC KEEP EMPTY YES
