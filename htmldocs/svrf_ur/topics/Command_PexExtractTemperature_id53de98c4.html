PEX Extract Temperature

Parasitic extraction

Specifies a temperature dependency for the extracted resistance.

Usage

Syntax 1, for Calibre xRC and Calibre xACT 3D:

PEX EXTRACT TEMPERATURE T [RANGE T1 T2 T3]

Syntax 2, for the Calibre xACT digital flow:

PEX EXTRACT TEMPERATURE T1 [T2 … Tn]

Arguments

T

Required floating-point number specifying the circuit temperature in degrees Celsius for resistance extraction. The default value is 25 degrees.

RANGE T1 T2 T3

Optional keyword and three floating-point numbers specifying the three temperatures used in calculating TC1 and TC2 for PEX Reduce ROnly and PEX Reduce Digital TICER functionality. This functionality is enabled only when the PEX Netlist statement specifies the TCOEFF option. Default values for RANGE are -55, 27, and 155.

T1 [T2 … Tn]

Required floating-point number specifying a single temperature for resistance extraction or a series of temperatures to be used in multi-temperature extraction within the Calibre xACT digital flow. Specify temperature in degrees Celsius. The default value is 25 degrees.

Description

The PEX Extract Temperature statement allows you to modify the resistance extracted during the PDB step in the formatter by specifying the temperature dependence that can affect resistance. This statement is used in conjunction with the PEX Resistance Parameters statement. This statement should be specified once. If this statement is specified more than once in your rule file, including before DRC ICSTATION YES, the first occurrence is the one that is used; any other occurrences that follow are ignored. (Note that DRC ICSTATION YES should only appear in control files generated by Calibre Interactive. It should not appear in any other rule files.)

The temperature values are applied based on the following standard SPICE equation. (Note that TC1 and TC2 are defined for each layer with PEX Resistance Parameters):

     R = R * (1 + TC1 * (T - Tnom) + TC2 * (T - Tnom) ^2)

The equation is applied during the formatting of the netlist, so you can modify temperature settings and reformat the netlist accordingly. Both the circuit temperature (T ) and the nominal temperature (Tnom) are written to the netlist, so that you can see what modifications were made to the parasitic components.

If PEX Extract Temperature is used but TC1 and TC2 are not specified for a layer, the tool issues an error message and exits.

The Calibre xACT digital flow supports extraction of multiple process and temperature corners in a single extraction run. Specify PEX Extract Temperature with multiple temperatures to enable concurrent extraction of multiple temperature corners. For more information on how to extract multiple process corners and temperature corners in a single run, see “Netlisting Multiple Corners and Multiple Temperatures” in the Calibre xACT User’s Manual.

Examples

Example 1

The following statements specify a circuit temperature of 35 degrees Celsius: 

PEX EXTRACT TEMPERATURE 35 
RESISTANCE SHEET M2 [.03 0 [0.0028 0.0035]]
RESISTANCE SHEET M1 [.02 0 [0.0027 0.0039]]

Example 2

The following statement specifies the three temperatures used in calculating TC1 and TC2: 

PEX EXTRACT TEMPERATURE 25 RANGE -55 25 155

Example 3

Include the following statement to specify multiple temperature corners for Calibre xACT digital flow runs: 

PEX EXTRACT TEMPERATURE 25 120

This statement creates two netlists, one for each of the two temperatures.
