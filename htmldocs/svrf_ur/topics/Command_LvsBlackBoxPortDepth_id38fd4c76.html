LVS Black Box Port Depth

Specification statement

Defines the hierarchical levels from which ports are recognized in LVS Box BLACK cells.

Usage

LVS BLACK BOX PORT DEPTH {PRIMARY | ALL | number}

Arguments

PRIMARY

A keyword that specifies black box ports are recognized at the top level of box cells only. This is the default.

ALL

A keyword that specifies black box port polygons are recognized at all levels of box cells’ hierarchy.

number

A keyword that specifies black box port polygons are recognized within box cells down to the hierarchical level specified by the number. The PRIMARY keyword is equivalent to 0. Values other than 0 are deprecated.

Description

Specifies the levels of hierarchy from which ports are recognized within LVS Box BLACK cells. Black box port layers are defined by the LVS Black Box Port statement.

When a setting other than PRIMARY or 0 is used (numeric values other than 0 are deprecated), port polygons are recognized inside black box cells down to the hierarchical level specified in this statement. The text objects that label ports must exist at the primary level of the black box cell in order for port names corresponding to lower-level port polygons to appear in the extracted netlist. The text objects must intersect lower-level port polygons when viewed flat.

Examples

In this example, black box port polygons are read from all levels of the cellA hierarchy. Polygons on layer m1 in cell placements within cellA are recognized as ports when they intersect text objects on layer txt at the primary level of cellA. The intersection is from the flat point of view.

LAYER m1 0 // lower-level black box port polygons on this layer
LAYER v1 1
LAYER m2 2 // primary-level black box port polygons on this layer
LAYER v2 3
LAYER m3 4 // connections from black box cell to cells higher up
LAYER txt 5

TEXT LAYER txt // port text objects at primary level of black box cell

ATTACH txt m1
ATTACH txt m2

CONNECT m1 m2 BY v1
CONNECT m2 m3 BY v2

LVS BOX BLACK cellA // requires LVS BLACK BOX PORT statement
LVS BLACK BOX PORT m1 txt m2 // black box lower-level ports
LVS BLACK BOX PORT m2 txt m3 // black box primary-level ports

// ports taken from all levels of black box cell hierarchy
LVS BLACK BOX PORT DEPTH ALL
