LVS Abort On ERC Error

Specification statement

Specifies whether LVS terminates when ABORT LVS keywords are specified in ERC Select Check statements. Used only in Calibre nmLVS/Calibre nmLVS-H and Calibre PERC.

Usage

LVS ABORT ON ERC ERROR {YES | NO}

Arguments

YES

A keyword that allows ABORT LVS keywords specified in ERC Select Check statements to terminate the run before LVS comparison. This is the default behavior if this statement is not specified in your rule file.

NO

A keyword that does not allow ABORT LVS keywords specified in ERC Select Check statements to terminate the run before LVS comparison.

Description

Controls LVS comparison termination when ABORT LVS keywords are specified in ERC Select Check statements.

If YES is specified, then circuit extraction occurs as usual, and warnings are written in the run transcript for each selected ERC check that has non-empty results:

WARNING: Results generated by an ERC check with ABORT LVS (check <name>) - aborting.

Circuit extraction then terminates with this transcript error:

ERROR: Results generated by one or more ERC checks with ABORT LVS - aborting.

When circuit extraction terminates, the extracted layout netlist contains a comment that causes any future LVS comparison to abort when the netlist is used.

If hyperscaling is used (-hyper cmp command line option), then the tool does not terminate until after the source netlist is read.

If NO is specified, LVS ignores any ABORT LVS keywords used in ERC Select Check statements. No special messages are written to the transcript if NO is specified. The NO keyword is useful for quickly disabling all ABORT LVS options instead of removing them from each ERC Select Check statement in your rule file.

This statement may be specified once in your rule file.

Examples

With these statements:

LVS EXECUTE ERC YES
ERC SELECT CHECK ABORT LVS check_power check_ground 
// Observe ABORT LVS keywords in ERC Select Check statements
LVS ABORT ON ERC ERROR YES

These messages can appear in the run transcript for a combined circuit extraction and LVS comparison run:

WARNING: Results generated by an ERC check with ABORT LVS (check check_power) - aborting.
 
WARNING: Results generated by an ERC check with ABORT LVS (check check_ground) - aborting.
 
ERROR: Results generated by one or more ERC checks with ABORT LVS - aborting.

Check the ERC Results Database and ERC Summary Report for details on ERC results.
