Device Annotation Built-In Language

The Device annotation built-in language is useful for MOS device well proximity and shallow trench isolation (STI) stress effect property calculations and similar applications. It is primarily used to extract physical parameters for the newer BSIM4 models.

The Device Layer operation supports DFM Property device annotations through the ANNOTATE keyword. A user-defined program is used for adding measurements like stress effects that have the potential to disrupt the hierarchy of more exotic devices.

The Device statement supports device annotations through its device_annotation_program block. This block is used for better performance of the Pushdown Separated Properties (PDSP) flow.

The annotation program uses a built-in language similar to the Device Property Computation Built-In Language. The device annotation built-in language has all the features of the built-in languages described in “Common Features in the Built-In Languages” and “Numeric Functions for Built-In Languages”. It also employs the “TVF Functional Interface for Device Property Calculation”.
