PEX Qualification Mode

Parasitic extraction

Controls the PEX tool invocations allowed during extraction.

Usage

PEX QUALIFICATION MODE {XACT | XRC}

Arguments

XACT

Required keyword that specifies to allow only the following Calibre xACT invocations:

  • calibre -xact

  • calibre -xact -3d

XRC

Required keyword that specifies to allow only the following Calibre xRC invocations:

  • calibre -xrc -phdb

  • calibre -xrc -phdb

  • calibre -xrc -fmt

  • calibre -xact -phdb

  • calibre -xact -pdb -3d

  • calibre -xact -fmt

  • calibre -xact (for LEF/DEF flows only)

Description

An optional statement used to control which extraction tool, Calibre xRC or Calibre xACT, can be run. This is a foundry predefined statement found in the foundry-provided calibrated rule files. If this statement is not specified then no checks are made and any tool flow can be run.

If this statement is in your rule file and the invocation you have specified is not allowed, then the tool stops with an error.

Examples

Example 1

If the following statement is in your calibrated rule file:

PEX QUALIFICATION MODE XACT

and the Calibre xACT 3D PDB extraction flow runs the command line:

calibre -xact -3d -pdb -rcc rules.svrf

extraction stops with the error:

ERROR: This rule deck is only supported using the Calibre xACT 3D Direct Netlisting Flow

Example 2

If the following statement is in your calibrated rule file:

PEX QUALIFICATION MODE XRC

and the Calibre xACT 3D direct netlisting flow is run with the command line:

calibre -xact -3d -rcc rules.svrf

extraction stops with the error:

ERROR: This rule deck is only supported using the Calibre xACT 3D PDB flow

Example 3

If the following statement is in your calibrated rule file:

PEX QUALIFICATION MODE XACT

and Calibre xRC is run with the command line:

calibre -xrc -pdb -rc rules.svrf

extraction stops with the error:

ERROR: This rule deck is only supported for  Calibre xACT flows

Example 4

If the following statement is in your calibrated rule file:

PEX QUALIFICATION MODE XRC

and the Calibre xACT direct netlisting flow is run with the command line:

calibre -xact -rcc rules.svrf

extraction stops with the error:

ERROR: This rule deck is only supported for PDB based flows
