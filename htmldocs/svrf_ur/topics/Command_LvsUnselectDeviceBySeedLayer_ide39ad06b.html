LVS Unselect Device By Seed Layer

Specification statement

Causes device recognition not to use Device statements having specified seed layers.

Usage

LVS UNSELECT DEVICE BY SEED LAYER layer [layer …]

Arguments

layer

A required name of a Device seed shape layer. More than one layer may be specified.

Description

Configures device recognition such that Device statements having the specified seed layers are disabled for device extraction and netlisting. By default, all Device statements are enabled.

This statement is the logical complement of LVS Select Device By Seed Layer. See that reference page for a complete discussion of device seed layer selection behavior.

Examples

In this example, the first two Device statements are enabled for recognition, while the third is disabled:

DEVICE M(ngen) gate  gate(g) sd(s) sd(d) <ngen>
DEVICE M       gate ngate(g) sd(s) sd(d) pwell(b)
DEVICE MN     ngate ngate(g) sd(s) sd(d) pwell(b)

LVS UNSELECT DEVICE BY SEED LAYER ngate
