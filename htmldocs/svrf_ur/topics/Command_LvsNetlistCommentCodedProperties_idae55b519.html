LVS Netlist Comment Coded Properties

Specification statement

Specifies whether the SPICE netlister should use the comment-coded parameters $A and $P (for capacitors) or $W and $L (for bipolar transistors). Used only in Calibre nmLVS and Calibre PERC.

Usage

LVS NETLIST COMMENT CODED PROPERTIES {YES | NO}

Arguments

YES

A keyword that indicates the SPICE netlister should use the following comment-coded properties:

  • For capacitors (element C and equivalent devices), property A is netlisted as $A and property P is netlisted as $P.

  • For bipolar transistors (element Q and equivalent devices), property L is netlisted as $L and property W is netlisted as $W.

This is the default.

NO

A keyword that causes capacitors and bipolar transistors to be netlisted with regular device parameters. The device parameters A and P for capacitors, and parameters L and W for bipolar transistors, are not coded as comments.

Description

Specifies whether the SPICE netlister and the Query Server should use the comment-coded parameters for capacitors and bipolar transistors. This statement only affects how such properties get netlisted, not whether such properties are calculated. Properties A and P for capacitors are not calculated by default, so they would have to appear in a user-defined property computation program. Similarly, the L and W properties for bipolar devices are not calculated by default.

Calibre-specific device annotations, such as $X, $Y, $T, and $D, are always netlisted as comments.

Examples

Assume a Device statement like this:

DEV C cap_layer anode(POS) cathode(NEG)
    [
      property C, A, P
      CA = 1E-02 // area cap
      CP = 1E-10 // perim cap
      CCA = CA * (unit_cap() / (unit_len() * unit_len()))
      CCP = CP * (unit_cap() / unit_len())
      A = area(cap_layer)
      P = perim(cap_layer)
      C = CCA * A + CCP * P
    ]

This could extract the following device instance when LVS Netlist Comment Coded Properties YES is used:

C0 1 10 8.26001e-13 $A=8.26001e-11 $P=3.6358e-05 $X=-22206 $Y=62402 $D=1
