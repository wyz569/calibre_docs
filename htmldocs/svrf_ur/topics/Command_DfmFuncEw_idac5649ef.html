EW

DFM measurement function

Returns the minimum distance (Edge Width) between each edge pair on the input layer.

Usage

EW(layer)

Arguments

layer

A required error layer with one or two edges per cluster.

Return Values

Floating-point number in user units.

Description

Returns the minimum separation distance between pairs of edges in an edge cluster. The function returns 0 for single edges and intersecting edges.
