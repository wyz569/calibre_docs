PEX Report Coupling Capacitance

Parasitic extraction

Generates a coupling capacitance-to-total capacitance report and stores it at the location specified.

Usage

PEX REPORT COUPLING CAPACITANCE filename
[NUMBER count] [SPLIT_NET] [THRESHOLD value] [LAYOUT | SOURCE]

Arguments

filename

A required output file name for the report.

The filename parameter can contain environment variables. For information regarding the use of environment variables in the filename parameter, refer to “Environment Variables in Pathname Parameters”.

NUMBER count

An optional keyword and parameter that specify the maximum number of coupled capacitors to include in the report. The default value for count is 1000. The maximum allowed value for count is 2000000000.

SPLIT_NET

An optional keyword that controls the output format of the report. When SPLIT_NET is used, the ratio of coupling capacitance for Net B is shown on a separate line.

THRESHOLD value

An optional keyword and parameter that specifies the threshold for coupling capacitance shown in the report. Capacitance values below this threshold are not shown. For Calibre xRC, the units of value are scaled by the units specified with the Unit Capacitance statement. For Calibre xACT, the units of value are taken as farads.

LAYOUT

An optional keyword that specifies using net names from the layout in the report. This is the default behavior for non-ASIC extraction.

SOURCE

An optional keyword that specifies using net names from the source netlist in the report. For ASIC extraction, this is the default behavior.

Description

This statement generates a report of the ratio of coupling capacitors and total net capacitance between pairs of nets in a layout. Net names may come from the layout or the source netlist.

Since Calibre xRC generates the report during the formatter stage, this statement does not need to be specified in the PDB step. If the SOURCE keyword is specified but no source names are found for the nets, then the nets are reported using the layout name prefixed with the string “layout_”.

The report shows the ratio of coupling capacitance between any two nets related to the total net capacitance for each net. For nets A and B, the ratios are:

     Percentage A = [CCAB / CA_total] X 100

     Percentage B = [CCAB / CB_total] X 100

where:

  • Percentage A is the amount of coupling capacitance between nets A and B based on the total intrinsic and coupled capacitances on net A.

  • Percentage B is the amount of coupling capacitance between nets A and B based on the total intrinsic and coupled capacitances on net B.

  • CCAB is the coupling capacitance between nets A and B.

  • CA_total is the total parasitic capacitance, coupling plus intrinsic, on net A.

  • CB_total is the total parasitic capacitance, coupling plus intrinsic, on net B.

For Calibre xRC, the Unit Capacitance statement changes the units reported for the values in the Coupling Cap column in the report. For Calibre xACT, the Unit Capacitance statement does not change the units reported for the values in the Coupling Cap column in the report.

Examples

Example 1

In this example, the report generated by Calibre xACT includes net-to-net coupling capacitors in between pairs of nets in the circuit:

PEX REPORT COUPLING CAPACITANCE "coupledCaps.rep"

The output generated from this statement is:

Ratio from A(%)    Ratio from B(%)  Coupling Cap(F)   NetA  NetB
      14.1766         3.19351        2.627139e-15      7     VSS
      8.58376         2.21703        1.823839e-15      8     VSS
      6.81333         7.81192        1.447666e-15      8     7

Line one of the report shows about 2.6 femtofarads of coupled capacitance between net 7 and VSS. The contribution of this capacitance to the total parasitic capacitance on 7 is about 14%. Line one of the report also shows that the reported coupling capacitance between 7 and VSS accounts for about 3.2% of the total capacitance on VSS.

Example 2

In this example, the report generated by Calibre xACT includes coupling capacitors on Net A that have a value greater than 1.5 femtofarads: 

PEX REPORT COUPLING CAPACITANCE "splitNets.rep" SPLIT_NET 
THRESHOLD 1.5e-15

The output generated from this statement is:

Ratio from A(%)   Coupling Cap(F)   NetA  NetB
      14.1766      2.627139e-15      7     VSS 
      8.58376      1.823839e-15      8     VSS 
      3.19351      2.627139e-15      VSS   7 
      2.21703      1.823839e-15      VSS   8

The ratios of coupled capacitance reported between net 7 and VSS are split into two separate lines in the report shown on line one and line three.

Example 3

To generate a report using Calibre xRC, that only includes coupling capacitors that have a value greater than 2 femtofarads, do not include the magnitude (e-15) for the THRESHOLD keyword value parameter. For Calibre xRC, the magnitude for value is determined by the Unit Capacitance statement setting. For example: 

PEX REPORT COUPLING CAPACITANCE "xrc_cc.rep" THRESHOLD 2
UNIT CAPACITANCE fF

The Calibre xRC coupling capacitance report generated from these statements looks something like:

Ratio from A(%)    Ratio from B(%)  Coupling Cap(F)   NetA  NetB
      47.96           19.0173        2.478252e-15      M4D   M3A
      38.8656         19.6887        2.008315e-15      M4D   M2D
      29.4781         5.60833        2.080800e-15      M5D   M1_C
...

To generate a similar report using Calibre xACT, the threshold value parameter must include the units of magnitude (e-15):

PEX REPORT COUPLING CAPACITANCE "xact_cc.rep" THRESHOLD 2e-15

The Unit Capacitance statement had no effect on the coupling capacitance report generated by Calibre xACT.
