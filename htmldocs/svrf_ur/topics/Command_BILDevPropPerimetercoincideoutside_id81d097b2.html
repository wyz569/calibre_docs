perimeter_coincide_outside

Device property computation language function.

Returns the total length of the parts of perimeters on the first pin or layer that coincide with the perimeter of the second pin or layer and where the coincident edges are adjacent to the exteriors of shapes on the first pin or layer.

Usage

PERIMeter_COincide_OUTside(pin_or_layer, pin_or_layer)

Arguments

pin_or_layer

A required pin name or device layer as described in “Pin and Layer References”. Two arguments of this type must be specified separated by a comma and a space.

Return Values

Floating-point number in meters.

Description

Returns portion 2b in Figure 1 of the perimeter function.
