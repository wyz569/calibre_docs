EW Group: Edge Separation

Returns the minimum separation distance between edges (or, the Edge Width). To determine minimum separation, the tool finds a pair of points, one on each edge, such that no other point pair is closer together. Function variants return the x- and y-axis projection, constrain one edge to be horizontal or vertical, or require the edges to be parallel in order for a measurement to be taken.

EW

Returns the minimum distance between edges.


EWP

Returns the minimum distance between edges for parallel edge pairs.


EWX, EWY

Return the minimum distance between edges if the separation runs parallel to the desired axis, or 0 otherwise.


EWXP, EWYP

Return the projection length on the desired axis of the minimum separation between edges.


See “Measurement Functions” for information on using the DFM measurement functions, including cell context dependencies and per-shape evaluation.
