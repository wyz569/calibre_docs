NETNAME

Net based function for DFM expressions

Returns a string containing the net name for the specified netID. Supported in the DFM Property and DFM Property Select Secondary operations.

Usage

NETNAME(netID)

Arguments

netID

A required netID.

The netID can be determined with functions such as NETID or NETPROPERTY operating on an input layer in the DFM Property operation.

Return Values

A string.

Description

Returns a string containing the net name for the specified netID. An empty string is returned if a net name is not associated with the net. (NETNAME returns the same string that is reported by the DN property in a results database (RDB) created by DFM RDB with the NODAL keyword.)

Net names are determined within the cell where the property expression is evaluated. (See the discussion with NETID, GLOBALNETID, and “Cell-Context-Dependent Property Functions Results in Hierarchical Batch Runs”.)

Examples

This example finds the net name at the local cell level. 

LAYER layer1 1
LAYER txt 4

TEXT DEPTH PRIMARY
TEXT LAYER txt 
PORT LAYER TEXT  txt  
ATTACH txt layer1 MASK
CONNECT layer1

net_name_layer1 = DFM Property layer1 
	[ netid1 = NETID(layer1)]
   [ netname1 = NETNAME(NETID(layer1))]
