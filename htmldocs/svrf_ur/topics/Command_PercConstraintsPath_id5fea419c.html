PERC Constraints Path

Specification statement

Specifies constraint parameters for a Calibre PERC run.

Usage

PERC CONSTRAINTS PATH filename

Arguments

filename

A required pathname of an XML constraints file.

The filename parameter can contain environment variables. See “Environment Variables in Pathname Parameters”.

Description

Specifies that a Calibre PERC rule file can retrieve constraint parameters from the specified filename. The format and usage criteria for the file are discussed under “XML Constraints File” in the Calibre PERC User’s Manual. This statement may be specified once.

A constraints file can be accessed from any rule file. This obviates any need for copying constraint parameters from one rule file to another, thus simplifying rule file maintenance.

Examples

PERC REPORT "perc.rep"
PERC LOAD perc.func INIT init_1 SELECT rule.1 rule.2
PERC CONSTRAINTS PATH "constraints.xml"
