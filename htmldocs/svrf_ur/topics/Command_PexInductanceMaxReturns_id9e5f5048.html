PEX Inductance Max Returns

Parasitic extraction

Specifies the maximum number of return paths used for inductance calculations. Used only in Calibre xL.

Usage

PEX INDUCTANCE MAX RETURNS number

Arguments

number

A required parameter that specifies the maximum number of return paths used to calculate loop inductance. The value must be a positive integer.

Description

An optional statement that specifies the maximum number of return paths used for inductance calculations. The Calibre xL tool determines the optimal number of return paths when this statement is not included in the rule file.

CAUTION:

Using this statement may introduce significant accuracy errors if the specified number of return paths is not adequate, or increase extraction times if an unnecessarily large number of return paths are set.

Examples

The following statement instructs the Calibre xL tool to limit the maximum number of return paths for inductance extraction to 10.

PEX INDUCTANCE MAX RETURNS 10

If less than 10 return paths are available in the design for all signal nets to be extracted, then all of the return paths are used.
