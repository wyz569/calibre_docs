EFFECTIVE STRING

Effective property computation language function.

Defines string properties for evaluation during reduction.

Usage

EFFECTIVE STRing property_name [, property_name …]

Arguments

property_name

A required string property name for reduction. More than one property_name may be specified in a comma-delimited list.

Return Values

None.

Description

The semantics of this statement are nearly identical to the EFFECTIVE statement, with the main difference being EFFECTIVE STRing takes string property inputs rather than numeric.

See “Limitations on String Properties” for information about string property restrictions.
