PERC LDL Waiver Path

Specification statement

Specifies to use automatic waivers during a Calibre PERC LDL run.

Usage

PERC LDL WAIVER PATH filename

Arguments

filename

A required pathname of a waiver setup file.

The filename parameter can contain environment variables. See “Environment Variables in Pathname Parameters”.

Description

Specifies that a Calibre PERC LDL run should use automatic waivers for geometry results. Without this statement, geometric waivers are not applied. A Calibre Auto-Waivers license is required to use waivers. See the Calibre Administrator’s Guide for licensing information. This statement may be specified only once in the rule file.

The waiver setup file format is discussed under “Waiver Setup File” in the Calibre Auto-Waivers User’s and Reference Manual. Note that a waiver setup file is different from a topology waiver description file specified by PERC Waiver Path.

Examples

PERC REPORT "perc.rep"
// waive LDL resistance errors
PERC LDL WAIVER PATH "LDL_waivers.setup"
DFM DATABASE "dfmdb" OVERWRITE REVISIONS [DEVICES PINLOC]

PERC LOAD perc.func INIT init_1 SELECT rule.1 rule.2
DFM YS AUTOSTART execute_perc_res_checks extract_resistance

PERC LDL LAYOUT REDUCE TOP LAYERS m4 m5 m6 m7 m8
