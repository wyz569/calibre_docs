Not Enclose

Layer operation

Selects all polygons that do not completely enclose any polygon from a second layer. Optional constraints modify the selection criteria.

Usage

NOT ENCLOSE layer1 layer2 [constraint [BY NET] [EVEN | ODD]]

Arguments

layer1

A required original layer or layer set, or a derived polygon layer. The operation selects polygons from this layer that meet the conditions of the operation.

layer2

A required original layer or layer set, or a derived polygon layer.

constraint [BY NET] [EVEN | ODD]

An optional argument that places conditions on the number of layer2 polygons that are enclosed by the layer1 polygon. If the number of layer2 polygons or nets that are enclosed by the layer1 polygon does not satisfy the constraint, the layer1 polygon is selected. A layer1 polygon that encloses zero layer2 polygons is always selected.

The constraint is one of those listed in the “Constraint Notation” column of Table 1 in the “Constraints” topic. The constraint must contain non-negative integers. Not Enclose with a constraint of == 0 selects all layer1 polygons.

  • BY NET — Specifies that the constraint applies to the number of distinct nets in the set of layer2 polygons that are enclosed by a layer1 polygon. Connectivity is required on layer2.

  • EVEN | ODD — Specifies that the number of layer2 polygons or nets that are enclosed by the layer1 polygon is even or odd, respectively. The layer1 polygon is selected if the constraint is not met. For example, if a constraint of “> 1 EVEN” is specified, a layer1 polygon that encloses three layer2 polygons is selected, while a layer1 polygon that encloses four layer2 polygons is not selected.

Description

Selects all layer1 polygons that contain no layer2 polygons as a subset (that is, a logical OR of a layer1 and layer2 polygon(s) does not yield exactly the layer1 polygon). This is the complement of the Enclose operation. It can also select all layer1 polygons that do not completely enclose a specified number of layer2 polygons, as indicated by a constraint. If layer2 is empty, the Not Enclose operation generates output equivalent to the polygon data on layer1.

Not Enclose selects polygons that do not satisfy an Enclose operation with the same arguments. If a constraint is specified, the constraint and optional keywords specify conditions for the complementary Enclose operation. Polygons on layer1 that do not satisfy the conditions as applied to the complementary Enclose operation are selected.

See also Not Enclose Rectangle, Not Inside, and Not Interact.
