PERC Voltage Trace Distance Enable

Specification statement

Specifies whether distance processing is performed for voltage tracing.

Usage

PERC VOLTAGE TRACE DISTANCE ENABLE {NO | YES}

Arguments

NO

A keyword that specifies voltage trace distance processing is disabled. This is the default.

YES

A keyword that specifies voltage trace distance processing is enabled. This keyword is required when perc::trace_path is used. This keyword should not be specified when perc::trace_path is not actively used in the rule file.

Description

Specifies whether voltage trace distance processing occurs. By default, it is disabled. This statement may be specified only once in the rule file.

The YES keyword is required when perc::trace_path is used, otherwise a runtime error is given. YES should not be specified in other circumstances because this adds runtime and memory overhead for no benefit.

Examples

// do not enable distance processing unless perc::trace_path is used
#UNDEFINE trace_path
#IFDEF trace_path
PERC VOLTAGE TRACE DISTANCE ENABLE YES
#ENDIF
