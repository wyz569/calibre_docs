Miscellaneous Warnings

This section describes miscellaneous runtime warnings.
Table 1. Miscellaneous Warnings

Explanation

The following warnings are reported by Calibre nmDRC and by the ICrules CHEck DRc command:

WARNING: Unable to use directory <name> for disk-based layers. Using memory-based layers.

Disk file based layers are to be used but Calibre nmDRC has a problem creating the specified directory or opening a file in this directory with read/write permission. The run continues with virtual memory based layers used instead.

The following warnings are reported by the ICrules CHEck DRc command:

WARNING: PINS from layer <layer> not read since the hierarchical mode is FLAT.

A pins operation is required by the check set and the hierarchical mode of execution is FLAT. The operation generates an empty layer in this event.

WARNING: DRC RuleCheck <name> is prohibited in TOP mode and will not be executed.

Reported for each selected Calibre nmDRC rule check in the rule check group drc_top_prohibited_checks when the hierarchical mode of the check is TOP.

WARNING: DRC RuleCheck <name> is prohibited in PEEKED mode and will not be executed.

Reported for each selected Calibre nmDRC rule check in the rule check group drc_peeked_prohibited_checks when the hierarchical mode of the check is PEEKED.

WARNING: DRC RuleCheck <name> is prohibited in FLAT mode and will not be executed.

Reported for each selected Calibre nmDRC rule check in the rule check group drc_flat_prohibited_checks when the hierarchical mode of the check is FLAT.

WARNING: DRC interrupt acknowledged; will conclude DRC.

Reported when CTRL-S is pressed during execution of CHEck DRc.

WARNING: <name> would not be processed.

Reported for every excluded cell name parameter to CHEck DRc which would not have been processed anyway. The most common cause of this warning is a typographic error.
