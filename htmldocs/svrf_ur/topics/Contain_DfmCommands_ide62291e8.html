DFM … Commands

The commands in this section begin with “DFM.”

The chapter “Summary of Rule File Elements” categorizes the statements and operations by application. See the sections listed in the following table for summaries of statements and operations relevant to DFM applications:

Summary Section

Description

Layer Operations

Operations that create a derived layer. Many layer operations are used in both DRC and DFM applications.

DFM Specification Statements

Specification statements that apply to Calibre DFM applications (Calibre YieldAnalyzer and Calibre YieldEnhancer).

Post-Design Correction Statements

A few operations starting with “DFM” are documented with the post-design applications, primarily for double- and multi-patterning.
