PEX Inductance Specification Statements

This section lists the PEX inductance specification statements. These statements are used by Calibre xL to extract parasitic inductance.
Table 1. Specification Statements: PEX Inductance

Statement

Description

PEX Generate Driver File Tag

Instructs the Calibre xRC tool to generate a driver and receiver file.

PEX Ground

Specifies one or more ground net names to consider during inductance extraction.

PEX Inductance Differential Pair

Specifies two nets operating as a differential pair.

PEX Inductance Driver File

Specifies the location of the driver and receiver file.

PEX Inductance Driver Summary

Specifies whether or not to output driver summary information to the transcript.

PEX Inductance Extract Layers

Specifies to process only geometries on the specified layers during inductance extraction.

PEX Inductance Filter

Specifies whether or not to filter paths for inductance extraction.

PEX Inductance Frequency

Specifies the global maximum operating frequency for inductance extraction for all signal nets in the design.

PEX Inductance ... Frequency

Specifies the nets for which the frequency is different from the global maximum operating frequency.

PEX Inductance Max Returns

Specifies the maximum number of return paths used for inductance calculations.

PEX Inductance MICHECK Constraint

Controls the output of the mutual coupling.

PEX Inductance Minlength

Specifies the minimum path length used to accurately compute self inductance.

PEX Inductance Mode

Specifies the inductance extraction mode used by CalibreÂ xL. By default, Calibre xL runs in loop inductance extraction mode if this statement is not specified.

PEX Inductance Range

Specifies how far to search from a given net segment for the return paths to include in self impedance calculations.

PEX Inductance Returnpath

Specifies the type of nets (ground or power) that may be considered as candidates for return path in inductance extraction.

PEX Inductance Same Net Mutual

Enables the extraction of the mutual inductance between wire segments within the same net, such as in stacked metal configuration.

PEX Inductance Self

Specifies the self inductance value to use on wire segments that are filtered.

PEX Inductance Switch Time

Specifies the global switch time to use for all nets in the design.

PEX Inductance ... Switch Time

Specifies the nets for which the switch time is different from the global switch time.

PEX Inductance Victim

Specifies a victim net name.

PEX Inductance Victim Path

Specifies the victim path associated with a victim net.

PEX Netlist Mutual Resistance

Specifies whether or not to use current-controlled voltage sources to model mutual resistance.

PEX Power

Specifies one or more power net names to consider during inductance extraction.

Unit Inductance

Relates the preferred user-defined unit of inductance to the fixed standard unit of inductance, which is picohenries, for via and bonding wire inductance calculations.
