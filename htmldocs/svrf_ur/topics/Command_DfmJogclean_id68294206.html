DFM Jogclean

Layer Operation

License: Calibre YieldEnhancer

Creates an output layer from which jogs have been removed based on specified criteria.

Usage

DFM JOGCLEAN layer JOGX jogx_val JOGY jogy_val [ADJACENT]

Arguments

layer

A required original or derived polygon layer.

JOGX jogx_val JOGY jogy_val

A required argument set that specifies constraints for jog removal. A jog is defined as an edge having one convex endpoint and one concave endpoint.

  • JOGX jogx_val — Specifies jog removal in the horizontal direction. Any jog parallel to the x-axis and with a length less than jogx_val is removed.

  • JOGY jogy_val — Specifies jog removal in the vertical direction. Any jog parallel to the y-axis and with a length less than jogy_val is removed.

All arguments are required. To specify jog removal in only one direction, specify the other constraint value as 0. For example, to remove jogs less than 2 user units in the x‑direction only, specify “JOGX 2 JOGY 0”.

ADJACENT

An optional keyword that specifies the adjacent edge must also meet the jog removal criteria in order for the jog to be removed. An example is shown in the following figure.

Example 1

This example demonstrates jog removal in only one direction, that is, one of JOGX or JOGY is specified as 0.

The following command removes jogs parallel to the x-axis that are less than 3 user units in length:

DFM JOGCLEAN M1 JOGX 3.0 JOGY 0
Figure 3. DFM JOGCLEAN With JOGX Only

The following command removes jogs parallel to the y-axis that are less than 3 user units in length:

DFM JOGCLEAN M1 JOGX 0 JOGY 3.0
Figure 4. DFM JOGCLEAN With JOGY Only

Example 2

This removes jogs less than 1.3 user units in length in both directions. 

DFM JOGCLEAN M1 JOGX 1.3 JOGY 1.3
Figure 5. DFM JOGCLEAN With JOGX and JOGY

Example 3

This example demonstrates the ADJACENT option. 

DFM JOGCLEAN M1 JOGX 2.0 JOGY 2.0 ADJACENT

When ADJACENT is specified, a set of perpendicular edges must meet the jog value constraints in both the x and y directions for the associated jog to be removed. The following figure shows the result.
