DFM Function Reference

DFM functions are used within DFM expressions to create data used in classifying or producing layout objects. The functions are used in the DFM Property family of operations, DFM Analyze, DFM Function, and DFM Spec Fill Optimizer.

DFM Expressions are constructed using math functions and operators, constants, and DFM functions (numeric, string, net-based, or vector; built-in or user-defined). The expressions can be arbitrarily complex and can reference values generated by other expressions in appropriate contexts.

There are many built-in DFM functions; these are called by specifying the function name with appropriate arguments. There is a large set of Measurement Functions that return measurements for layer objects, and these are among the most frequently used. You can also create user-defined functions using the DFM Function statement.

This chapter provides the necessary reference material to use the DFM functions in your DFM expressions.
