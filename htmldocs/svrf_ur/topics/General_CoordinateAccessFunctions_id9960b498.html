Coordinate Access Function Summary

The coordinate access functions measure objects on the input layer and return a coordinate value. The functions are supported in DFM expressions in the DFM Property operation.

The section “EC Group: Projection Endpoints” has important general information about the behavior of the coordinate access functions. All of the endpoint measurement functions are based on the mutual edge projection EC. They return an endpoint coordinate of the EC segment as measured along the axis parallel to EC.

Table 1. Coordinate Access Function Summary

Function

Summary

ECBEGIN

Returns the coordinate of the “beginning” endpoint of the EC segment, where the beginning endpoint is identified in the originating cell space and is the endpoint with the lesser coordinate value in the originating cell space (ECBEGIN = ECMIN in originating cell space). The returned value is measured the local cell space, which may be different than the originating cell space if promotion occurs.

ECEND

Returns the coordinate of the “ending” endpoint of the EC segment, where the ending endpoint is identified in the originating cell space and is the endpoint with the greater coordinate value in the originating cell space (ECEND = ECMAX in originating cell space). The returned value is measured the local cell space, which may be different than the originating cell space if promotion occurs.

ECMAX

Returns the larger value of the two endpoint coordinates in the context of the local cell space.

ECMIN

Returns the smaller value of the two endpoint coordinates in the context of the local cell space.
