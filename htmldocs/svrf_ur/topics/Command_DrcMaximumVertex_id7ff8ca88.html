DRC Maximum Vertex

Specification statement

Specifies the maximum vertex count of any polygon result written to the DRC results database. Used only in Calibre nmDRC and Calibre nmDRC-H.

Usage

DRC MAXIMUM VERTEX {number | ALL}

Arguments

number

Non-negative integer that specifies the maximum vertex count of any result polygon that Calibre nmDRC writes to the results database. The value must be an integer greater than or equal to 4. If you do not include this statement in your rule file, the default value of number is 4096.

ALL

Keyword that specifies the maximum count as 232 for OASIS and 8190 for GDSII.

Description

Specifies the maximum vertex count of any polygon result to be written to the DRC Results Database by Calibre nmDRC/Calibre nmDRC-H. This statement can be specified only once in a rule file.

Calibre nmDRC segments result polygons whose vertex count exceeds the specified value and writes the segmented polygons to the results database. The segmented polygons have a vertex count less than or equal to that specified (or the default if unspecified).

If polygon segmentation occurs, Calibre nmDRC reports a warning once per rule check; the warning indicates the number of polygons segmented like this:

WARNING: <number> polygon result(s) segmented in DRC RuleCheck <name>.
Note:

By default, Calibre can read in OASIS polygons having up to 232 vertices, which is also the maximum number it can output. However, you should avoid having input polygons of greater than 1E06 vertices as this can degrade performance.

Calibre can output and read GDS polygons with a maximum of 8190 vertices. If more than 8190 vertices are specified for any GDSII-type DRC results database, then Calibre resets this value to 8190 and generates a warning.

If your results database output is ASCII, you should not use a maximum vertex count that exceeds 4096. The ASCII database format uses this as an upper limit for vertex counts.

For more information on this topic, see “Polygon Segmentation” in the Calibre Verification User’s Manual.

Examples

DRC MAXIMUM VERTEX 4096 //default setting
