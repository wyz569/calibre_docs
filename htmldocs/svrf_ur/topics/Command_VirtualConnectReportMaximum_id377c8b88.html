Virtual Connect Report Maximum

Specification statement

Specifies the maximum number of virtual connect messages issued when virtual connect reporting is enabled.

Usage

VIRTUAL CONNECT REPORT MAXIMUM {ALL | number}

Arguments

ALL

A keyword that specifies to report all virtual connections. This is the default if this statement is not included in the rule file.

number

A positive integer that specifies the maximum number of virtual connect messages reported during the run.

Description

Instructs the Calibre tool to limit the number of virtual connect messages reported during the run.

This statement affects the number of NOTE or WARNING messages issued in Calibre nmDRC when Virtual Connect Report YES or LVS Report Option V is specified. This statement also affects the number of WARNING messages when the Virtual Connect Report YES UNSATISFIED is specified.

Examples

VIRTUAL CONNECT REPORT YES
// Report only the first 99 virtual connections
VIRTUAL CONNECT REPORT MAXIMUM 99
