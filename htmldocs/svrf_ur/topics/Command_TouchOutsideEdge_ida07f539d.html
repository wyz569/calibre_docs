Touch Outside Edge

Layer operation

Selects complete edges that coincide with an edge from a polygon on a second layer having the opposite facing exterior side.

Usage

TOUCH OUTside EDGE layer1 layer2 [ENDPOINT {ALSO | ONLY}]

Arguments

layer1

A required original layer or layer set, or a derived polygon or edge layer.

layer2

A required original layer or layer set, or a derived polygon or edge layer.

ENDPOINT {ALSO | ONLY}

Optional keyword group that specifies edges are selected from layer1 that are collinear with layer2 edges, have opposite facing orientation from layer2 edges, and interact only at their end points. ENDPOINT ALSO specifies to select such layer1 edges in addition to the default output from this operation. ENDPOINT ONLY specifies to select only layer1 edges that have this end point behavior; no other edges are output.

Description

Selects all complete layer1 edges that are coincident with layer2 edges, and where the polygons do not share internal area. Intersection at end points does not constitute touching by default. This is similar to the Coincident Outside Edge operation; however Touch Outside Edge selects only complete edges.

For input edges oriented at integral multiples of 45 degrees with respect to the coordinate axes, the overlap between the edges must be at least 1 dbu to satisfy the touching condition. Input edges not in that set (these are called skew edges) having a non-zero overlap satisfy the touching condition.

The ENDPOINT ALSO and ENDPOINT ONLY options change the default behavior and allow selection of layer1 edges that touch layer2 edges only at their end points.

See also Touch Edge, Touch Inside Edge, and Not Touch Outside Edge.
