Historical Well Proximity Calculation Methods

This section is provided for historical completeness to show methods used for well proximity calculations. These methods may appear in various process rule files but are not considered the most efficient or current. They are presented to the interested user so that they can be understood when encountered.
Note:

Currently, the recommended interface for well proximity and strained silicon measurement is the Device Annotation Built-In Language. That functionality should be used instead of others documented in this section for best performance.
