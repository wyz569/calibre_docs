Expand Text

Layer operation

Constructs a derived polygon layer consisting of merged squares centered on the positions of text objects having the specified text name.

Usage

EXPAND TEXT text_name [text_layer] BY number
[PRIMARY ONLY] [CASE SENSITIVE]

Arguments

text_name

A required name of a text object. Can be a string variable (see Variable). The text_name parameter can contain one or more question mark (?) wildcard characters, where the (?) matches zero or more characters.

text_layer

An optional original layer that contains the text_name. Used to prevent ambiguity in selecting of text objects having the same name but appearing on different layers.

BY number

A required keyword followed by a positive, floating-point number, interpreted in user units, that specifies the size of marker squares.

PRIMARY ONLY

An optional keyword that specifies only top-cell text is used in the operation. Text objects in a placed cell of the Layout Place Cell statement with one placed cell parameter are treated as top-level text objects.

CASE SENSITIVE

An optional keyword that specifies text_name parameters must match by case in order to be expanded.

Description

Creates a derived polygon layer consisting of merged squares centered on the positions of text objects having the specified text_name. The squares have edge length of number.

The parameter order must be observed to avoid ambiguity. The text_name is case-insensitive by default.

The Expand Text operation is a non-node-preserving layer constructor.

As a performance optimization, the Calibre database constructor only reads in text objects that are actually needed for the run. If you require all text objects to be read for applications like Calibre PERC LDL that perform DFM Database post-processing, then using “?” for the text_name parameter is useful. Be aware that this can increase memory consumption.

See also Expand Cell and Expand Cell Text.

Examples

x = EXPAND TEXT VDD text_layer BY 0.2 
//place 0.2 x 0.2 markers on layer x for VDD text locations on text_layer
