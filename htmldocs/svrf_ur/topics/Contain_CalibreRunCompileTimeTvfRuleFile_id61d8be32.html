Calibre Run with Compile-Time TVF Rule File

A Calibre run with a compile-time TVF rule file is started in the same manner as with a SVRF rule file. The opening line in the compile-time TVF rule file instructs Calibre to use the TVF preprocessor. Additional command line options are available to pass arguments to the TVF script, enable additional error reporting, or write the generated SVRF code to a file.

A compile-time TVF file can be specified in Calibre Interactive just like a SVRF rule file.
