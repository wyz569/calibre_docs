LVS Reduce Semi Series MOS

Specification statement

Specifies whether to reduce MOS devices connected in series including those with bypass nets.

Usage

LVS REDUCE SEMI SERIES MOS {NO | YES}

Arguments

NO

Keyword that instructs LVS not to reduce semi series MOS transistors. This is the default behavior if you do not include this statement in the rule file.

YES

Keyword that instructs LVS to reduce semi series MOS transistors.

Description

Specifies whether to reduce MOS devices connected in series, including those with bypass nets. The default is NO. You can specify this statement only once in your rule file. This statement is independent of the LVS Reduce Series MOS specification statement.

Initial correspondence points prevent device reduction transformations at the top level of the design.

For more information, refer to “Semi-Series MOS Transistor Reduction” in the Calibre Verification User’s Manual.

See also Device and LVS Device Type.

Examples

// default setting.
// L and W properties are assumed to be used in the Device statements.
LVS REDUCE SEMI SERIES MOS NO
