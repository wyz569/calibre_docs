tvf::GET_LAYER_NAME

Application: runtime only.

Returns the layer name for the specified layer handle. The layer name is more useful in debugging statement than the layer handle.

Usage

tvf::GET_LAYER_NAME $layer

Arguments

$layer

A required argument that specifies the layer handle.

Return Values

The layer name.

Examples

TVF FUNCTION example_function [/*
   # create the input layer handle 
   tvf::GET_LAYER_ARGS layer1 
   # get the layer name from the layer handle, for later error reporting
   set name_1 [tvf::get_layer_name $layer1]

   # ... rest of TVF Function
}
*/]
