tvf::COMMENT

Application: compile-time only.

Specifies a single-line user-supplied comment within a rule check (a rule check comment). Identical to the tvf::@ command for rule comments.

Usage

tvf::COMMENT comment

Arguments

comment

A required argument that specifies the comment string. There must be whitespace between the command name and the comment string.

The comment string is evaluated by the Tcl interpreter. If the comment string contains brackets ([…]), then the contents of the brackets are executed as a command.

Return Values

None.

Examples

tvf::RULECHECK VIA_SIZE {
   tvf::COMMENT Exact via size is 0.36 by 0.36
   tvf::OUTLAYER "NOT RECTANGLE via == 0.36 BY == 0.36"
}
