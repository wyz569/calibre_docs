DFM Operation Support for Built-In DFM Functions and Expression Constructs

Many built-in DFM functions are available for use in DFM expressions. The availability of these functions depends on the operation in which the DFM expression appear. The availability of some expression constructs, such as expression chains and conditional expression, also depends on the DFM operation.

DFM expressions and the built-in DFM functions are used in the following operations:

  • DFM Property operations with geometric output

    • DFM Property

    • DFM Property Merge

    • DFM Property Select Secondary

  • DFM Property operations with no geometric output

    • DFM Property Net

    • DFM Property Singleton

  • DFM Function

  • DFM Spec Fill Optimizer

  • DFM Analyze
