Post-Design Correction Statements

This section summarizes the statements used for optical and process correction. Most commands are documented in their corresponding RET product manuals (Calibre OPCpro, Calibre nmOPC, Calibre OPCverify, and so on).
Table 1. Post-Design Correction Statements

Operation or Statement

Description

DFM Balance

A Calibre Multi-Patterning operation used to output a balanced mask. Used with DFM Spec Balance.

DFM Cluster

A Calibre Multi-Patterning operation that selects or generates separators or polygons based on geometric and graph-based properties of connected components.

DFM DP

A Calibre Double Patterning operation that splits the target layer into two masks and generates error layers.

DFM MP

A Calibre Multi-Patterning operation that splits the target layer into three or four masks and generates error layers.

DFM MPSTITCH

A Calibre Multi-Patterning operation that generates multi-pattern stitches. Requires DFM SPEC MPSTITCH for constraint specification.

DFM Spec Balance

A Calibre Multi-Patterning statement that specifies multi-pattern mask balancing criterion.

DFM SPEC MPSTITCH

A Calibre Multi-Patterning statement that specifies stitching values and constraints for DFM MPSTITCH.

Litho DenseOPC

Specifies to run Calibre nmOPC and to place the results on a layer you specify.

Additional variants are Litho CellArray DenseOPC (for CellArray OPC), Litho EUV DenseOPC (for EUV OPC), and Litho ML DenseOPC (for machine learning OPC).

Litho File

Used to specify an inline file.

Litho nmBIAS

Calibre RET layer operation that adjusts target layer edges for optical correction. Requires a Calibre OPCpro or Calibre TDopc product license.

Litho OPC

Specifies to run Calibre OPCpro and to place the results on a layer you specify.

Litho OPCverify

Specifies to run Calibre OPCverify and to place the results on a layer you specify.

Litho ORC

Specifies to run Calibre ORC and to place the results on a layer you specify.

Litho Printimage

Specifies to run Calibre PRINTimage and to place the results on a layer you specify.

OPCBIAS

Calibre RET layer operation that performs variable isolated/dense line and via biasing. It oversizes or undersizes objects by moving individual edges pre-defined amounts. Requires a Calibre OPCpro or Calibre TDopc product license.

OPCLINEEND

Calibre RET layer operation that performs variable line-end extension, from simple end extension to hammerhead creation. Requires a Calibre OPCpro or Calibre TDopc license.

OPCSBAR

Calibre RET layer operation that creates scattering bars for optical correction. Requires a Calibre OPCsbar license.

RET DPSTITCH

Calibre RET operation used to derive layers. Requires a Calibre Multi-Patterning product license.

RET FLARE_CONVOLVE

Generates flare maps based on a flare model for the EUV flow.

RET [EUV] MBSRAF

Calibre nmSRAF operation that generates SRAFs based on model-based templates.

RET NMDPC

Calibre RET operation - splits via layers by pitch or edge a layer into two masks. Requires a Calibre Multi-Patterning product license.

RET Optimize_EUV_HDB

A Calibre nmOPC operation that finds regions with similar shadow biases and flares. Requires a Calibre EUV product license.

RET PXOPC

Calibre RET operation that performs pixel-based OPC and optimized SRAF insertion. Requires a Calibre pxOPC product license.

RET REFINE

Calibre RET operation for enhancing marked regions to improve printability after initial OPC. Required licenses depend on the particular tools called by a setup file.

RET SBAR

Calibre nmSRAF operation that generates SRAFs based on dimensional templates.
