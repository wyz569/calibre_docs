Global Variables in Compile-Time TVF

Global variables allow you to share variables between Tcl interpreters, encrypted and unencrypted. Compile-time TVF includes several commands for handling global variables.

The following commands are available; see the command reference page for details:

tvf::exists_global_variable

tvf::set_global_variable

tvf::get_global_variable

tvf::unset_global_variable
