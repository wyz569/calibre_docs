Drawn Offgrid

Layer operation

Selects off-grid vertices.

Usage

DRAWN OFFGRID

Arguments

None.

Description

Selects off-grid vertices on original shapes read from the layout database during a Calibre nmDRC-style application run. The Calibre nmDRC system uses the Resolution or Layer Resolution specification statement in the rule file to determine what constitutes an off-grid vertex. This operation produces a derived error layer indicating the off-grid vertices in the layout database. Each element of the derived layer is a cluster of two edges; the two edges are adjacent edges of the original geometry whose common endpoint is the off-grid vertex. The output edge cluster is trimmed to a distance of one user unit from the location of the vertex.

This operation only reads original layers required by the check set. The check set does not necessarily require the entire set of original layers referenced in the rule file, as only the layers actually used in layer operations are read in.

This operation only acts on unmerged original data and does not use an input layer parameter. The Drawn Offgrid operation is error-directed, which means it cannot generally be used to derive layers. See “Derived Error Layers” in the Calibre Verification User’s Manual.
