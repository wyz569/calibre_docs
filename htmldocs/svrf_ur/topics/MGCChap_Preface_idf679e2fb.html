Preface

This manual describes Standard Verification Rule Format (SVRF) syntax and command descriptions used with Calibre® and ICverify™ products.

Some new features may not be described in this manual. For latest functionality, refer to the Calibre Release Notes.

The following table lists the products that are referred to in this manual. Refer to the Calibre Administrator's Guide for complete information about licensing and software installation.

Table 1. Products Discussed in the SVRF Manual

IC Verification

The verification tools provide design rule checks (DRC), equation based DRC, layout versus schematic comparison (LVS), and parasitic extraction.

Electrical rule checking (ERC), short isolation, and soft connection checks can be performed as a part of Calibre nmLVS connectivity extraction.

Calibre PERC is a circuit reliability verification platform for validating source and layout designs using advanced ERC, Electrostatic Discharge (ESD), Electrical Overstress (EOS), voltage, and cell-based (LEF/DEF) checks. The Logic Driven Layout (LDL) interface allows current density, point-to-point resistance, and topology-aware DRC checks of a layout.

The ICrules and ICtrace products provide flat Calibre nmDRC and flat Calibre nmLVS as part of ICverify.
