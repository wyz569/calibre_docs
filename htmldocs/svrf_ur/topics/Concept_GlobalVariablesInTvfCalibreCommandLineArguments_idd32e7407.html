Global Variables in TVF for Calibre Command Line Arguments

TVF supports global variables that store values from the Calibre invocation line, analogous to argc and argv. These global variables are available for both compile-time and runtime TVF.
Table 1. Global Variables in TVF for Calibre Command Line Arguments

Variable

Definition

calibre_argc

The count of Calibre invocation arguments. The calibre command itself is not counted.

calibre_argv

The list of Calibre invocation arguments. The calibre command itself is not listed.

calibre_argv0

The calibre command as it is specified on the command line.

Example

The following lines can be added to runtime or compile-time TVF code:

puts "calibre command: $calibre_argv0"
puts "calibre arguments: $calibre_argv"
