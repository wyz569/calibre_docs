Other Specification Statements

This section lists the remaining specification statements that are not listed in the previous tables.
Table 1. Specification Statements: Other

Statement

Description

Include

Includes another file into the rule file in which this statement appears.

Net Area Ratio Print

Prints the input net area ratio accumulation (NARAC) information to a specified file.

SVRF DBdiff Options

Specifies DBdiff command-line options for controlling FastÂ XOR runs.

SVRF Error

Issues a compiler error if this statement is encountered during rule file compilation.

SVRF Message

Issues a message in the run transcript when this statement is encountered during rule file compilation.

SVRF Property

Defines a name in the rule file corresponding to an OASIS property in the layout.

SVRF Version

Defines the earliest Calibre version that is permitted to compile the rule file.

Text Print Maximum

Controls the number of text objects and ports printed in the transcript of a Calibre run. Not used in ICverify.

Title

Assigns a title to the rule file. The rule file title can be attached to DRC results in the Calibre nmDRC results database. It also appears in summary reports generated by the Calibre nmDRC application.

Variable

Defines the value of a variable in the rule file.
