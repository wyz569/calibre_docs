Touch

Layer operation

Selects polygons that are outside of all polygons on a second layer and share a coincident edge or portion thereof with a polygon on the second layer.

Usage

TOUCH layer1 layer2 [constraint [BY NET] [EVEN | ODD]]

Arguments

layer1

A required argument specifying an original layer or layer set, or a derived polygon layer. The operation selects polygons from this layer that meet the conditions of the operation.

layer2

A required argument specifying an original layer or layer set, or a derived polygon layer.

constraint [BY NET] [EVEN | ODD]

An optional argument, where the constraint specifies the number of layer2 polygons or nets that touch a layer1 polygon. The constraint must be met in order for the layer1 polygon to be selected.

The constraint is one of those listed in the “Constraint Notation” column of Table 1 in the “Constraints” topic. The constraint must contain non‑negative integers. The constraint == 0 does not produce any output; use Not Touch instead.

Optional keyword choices:

  • BY NET — Specifies that constraint applies to the number of distinct nets in the set of layer2 polygons that touch a layer1 polygon. Connectivity is required on layer2.

  • EVEN | ODD — Specifies that a layer1 polygon is selected if the number of layer2 polygons that meet the constraint is an even or odd number, respectively. For example, the following operation selects polygons from layer1 that touch four, six, or eight polygons on layer2:

    layer1 TOUCH layer2 >3 <9 EVEN

    It is not useful to specify these keywords if the specified constraint does not allow for the number of polygons to meet the even or odd criterion, such as with == constraints.

Description

Selects layer1 polygons that lie completely outside all layer2 polygons, but share a coincident edge, or segment, with a polygon from layer2. The optional constraint limits the selection of layer1 polygons according to the number of layer2 polygons that are touched. If either input layer is empty, there is no output.

See also Not Touch and Touch Edge.

Examples

This example shows two Touch operations. The operation on the left selects all layer1 polygons that lie completely outside all layer2 polygons but touch at least one layer2 polygon.
