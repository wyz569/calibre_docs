Miscellaneous Functions for DFM Expressions

Several built-in DFM functions do not belong to any particular category and are summarized here.
Table 1. Miscellaneous Function Summary

Function

Summary

MATCH

Determines whether or not a string matches a pattern and returns true or false (1 or 0).

NARAC

Accesses values computed by Net Area Ratio operations with the ACCUMULATE keyword specified.

PRECISION

Returns the value of the Precision specification statement in the SVRF rule file. However, if the function is used within a DFM Property operation with the DBU keyword specified, the function returns 1.0.

RANDOM

Returns a random number between 0 and 1.
