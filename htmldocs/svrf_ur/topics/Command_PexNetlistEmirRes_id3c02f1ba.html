PEX Netlist EMIR_RES

Parasitic extraction

Specifies to include the PCDEF and PEX ignore internal resistance dimensions in the extracted DSPF netlist for the EM/IR analysis flow. Used only in Calibre xACT and Calibre xACT 3D.

Usage

PEX NETLIST EMIR_RES {ALL | layername [layername...]}

Arguments

ALL

A required parameter that specifies to include parasitic resistance width and length information for polygons on all layers within PCDEF cells or specified in PEX Ignore statements in the extracted netlist.

layername [layername...]

A required parameter that specifies one or more original or derived layers. The parasitic resistance width and length information within PCDEF cells or specified in PEX Ignore statements for the specified layername(s) is included in the extracted netlist. The layername may contain one or more wildcard characters (*), where the wildcard matches zero or more characters. When using a wildcard, be sure to enclose layername in quotes.

Description

An optional statement used to netlist the width and length properties of parasitic resistors on geometries inside of PCDEF cells or in layers ignored with PEX Ignore statements. The resistors have a nominal value of 1e-6 ohms and are netlisted at the top level. Use this statement to generate a netlist suitable for use in the EM/IR analysis flow where length and width values are required for all net geometries to calculate accurate Blech length. This statement can only be used with the DSPF netlist format and may only be specified once. This statement does not affect capacitance extraction.

Examples

Example 1

When the following PEX Xcell statement is used to designate all cells that start with the name “mycell” as pcells, the 5 ohm resistor shown inside of mycell_123 in Figure 1 is not netlisted.

PEX XCELL “mycell*” PCDEF INTRINSIC IDEAL
Figure 1. PEX Netlist EMIR_RES Example When PCELLS are Used

Only the 10 ohm resistor appears in the netlist; for example:

...
R123 VSS!:175 VSS!:189 10 $w=5.01e-07 $l=7.0e-06 $lvl=4 $llx=7.239 $lly=-2.231 $urx=7.74 $ury=4.845
...

To enable Blech length calculations for EM/IR analysis, include the width and length values of the net geometries inside the mycell* PCDEF cells by specifying the following statement:

PEX NETLIST EMIR_RES ALL

The width and length information along with a 1e-06 resistor is added to the netlist for the geometries inside the PCDEF cell; for example:

...
R121 VSS!:174 VSS!:175 1e-06 $w=4.99748e-07 $l=3.5e-06 $lvl=4 $llx=7.239 $lly=4.845 $urx=7.74 $ury=4.945
R123 VSS!:175 VSS!:189 10 $w=5.01e-07 $l=7.0e-06 $lvl=4 $llx=7.239 $lly=-2.231 $urx=7.74 $ury=4.845
...

Example 2

The following statements specify to ignore the 5 ohm resistor within the mymarker layer region as shown in Figure 2. 

M1_device = M1 AND mymarker
PEX IGNORE RESISTANCE M1_device
Figure 2. PEX Netlist EMIR_RES Example When PEX Ignore Resistance is Specified

Only the 10 ohm resistor is netlisted; for example:

...
R123 VSS!:175 VSS!:189 10 $w=5.01e-07 $l=7.0e-06 $lvl=4 $llx=7.239 $lly=-2.231 $urx=7.74 $ury=4.845
...

To include the parasitic resistor width and length information, including a 1e-6 ohm resistor, for net geometries on M1 within the marker region, specify the following statement:

PEX NETLIST EMIR_RES ALL

The resulting sample output is:

...
R121 VSS!:174 VSS!:175 1e-06 $w=4.99748e-07 $l=3.5e-06 $lvl=4 $llx=7.239 $lly=4.845 $urx=7.74 $ury=4.945
R123 VSS!:175 VSS!:189 10 $w=5.01e-07 $l=7.0e-06 $lvl=4 $llx=7.239 $lly=-2.231 $urx=7.74 $ury=4.845
...

Example 3

The following statements specify to ignore the 5 ohm resistor and 3 ohm resistor within the mymarker layer region as shown in Figure 3. 

M1_device = M1 AND mymarker
J2_device = J2 AND mymarker
PEX IGNORE RESISTANCE M1_device
PEX IGNORE RESISTANCE J2_device
Figure 3. PEX Netlist EMIR_RES layername Example When PEX Ignore Resistance is Specified

In the extracted netlist, level 10 ($lvl=10) is layer J2 and level 4 ($lvl=4) is layer M1. For the M1 layer only the 10 ohm resistor is netlisted and for the J2 layer only the 6 ohm resistor is netlisted; for example:

...
R1 neta:1 neta:2 6 $w=5 $l=7.0e-6 $lvl=10 $llx=8 $lly=7 $urx=9 $ury=8
...
R123 VSS!:175 VSS!:189 10 $w=5.01e-07 $l=7.0e-06 $lvl=4 $llx=7.239 $lly=-2.231 $urx=7.74 $ury=4.845
...

To include the width and length information of resistance net geometries for all layers whose name begins with ‘J’, specify the following statement:

PEX NETLIST EMIR_RES "J*"

The resulting netlist contains the ignored resistor (R2) for J2 layer, but not the ignored resistor for the M1 layer:

...
R1 neta:1 neta:2 6 $w=5 $l=7.0e-6 $lvl=10 $llx=8 $lly=7 $urx=9 $ury=8
R2 neta:2 neta:3 1e-06 $w=5 $l=3.5e-6 $lvl=10 $llx=8 $lly=7 $urx=9 $ury=8
...
R123 VSS!:175 VSS!:189 10 $w=5.01e-07 $l=7.0e-06 $lvl=4 $llx=7.239 $lly=-2.231 $urx=7.74 $ury=4.845
...
