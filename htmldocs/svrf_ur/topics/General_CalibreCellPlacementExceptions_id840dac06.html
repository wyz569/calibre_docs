Calibre Cell and Placement Exceptions

This section describes runtime exceptions related to cells and placements.
Table 1. Calibre Cell and Placement Exceptions

Comment

Explanation

Warnings preceded by “†” can have their severities controlled by Layout Input Exception Severity. Warnings preceded by “*” are fatal if Layout Error On Input YES is specified.

The following exceptions are reported by all Calibre applications:

†

EXCEPTION: Excluded cell name <name> not within the input layout database.

 

A cell name in a rule file Exclude Cell specification statement is not located in the input layout database.

Exception name: EXCLUDE_CELL_NAME

†

EXCEPTION: Cell name parameter <name> for EXTENT CELL operation not located.

 

A cell name parameter for an Extent Cell operation is not located in the input layout database.

Exception name: EXTENT_CELL

†

EXCEPTION: Cell name parameter <name> for [NOT] INSIDE CELL operation not located.

 

A cell name parameter for a [NOT] Inside Cell operation is not located in the input layout database.

Exception name: INSIDE_CELL

†*

EXCEPTION: Cell <name> is referenced but not defined.

 

A placement of a cell without a corresponding structure record is encountered.

Exception name: MISSING_REFERENCE

†*

EXCEPTION: Another cell record encountered for cell <name>.

 

A second (etc.) structure record was encountered for a cell.

Exception name: DUPLICATE_CELL

†

EXCEPTION: Primary cell <name> selected by wildcard.

 

The top-level cell was selected by wildcard; that is, its name matched the Layout Primary or Layout Primary2 specification statement parameter (which contained one or more “*” wildcard characters) without the match being literal.

Exception name: TOP_CELL_CHOSEN

†*

EXCEPTION: Absolute angle in placement of cell <name> within cell <name> not supported.

 

A placement has an absolute angle.

Exception name: PLACEMENT_ANGLE_ABSOLUTE

†*

EXCEPTION: Absolute magnification in placement of cell <name> within cell <name> not supported.

 

A placement has an absolute magnification.

Exception name: PLACEMENT_MAGNIFICATION_ABSOLUTE

†*

EXCEPTION: Non-positive magnification in placement of cell <name> not supported.

 

A placement has a non-positive magnification.

Exception name: PLACEMENT_MAGNIFICATION_NONPOSITIVE
