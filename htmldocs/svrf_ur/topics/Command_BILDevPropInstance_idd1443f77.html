instance

Device property computation language function

Returns the instance number of the current device instance.

Usage

INSTance()

Arguments

None.

Return Values

Instance number of the current device instance.

Description

Each flat extracted device has a non-negative instance ID number that is unique over all devices of all types.

In hierarchical LVS, instance IDs are not unique but are generated on a per-cell basis. As this is inconvenient for unique instance identification, this command is best used in flat debugging.

Instance numbers from this function are not valid if you use LVS Push Devices YES.

Examples

In this Device statement, the instance() function is used to assign a property value:

device mp pgate pgate(g) psd(s) psd(d) nwell(b) [
 
  property W,L,I
  I = inst()
...

An extracted netlist then shows the instance numbers as property “i”:

.SUBCKT nand GND PWR IN1 OUT IN2
** N=7 EP=5 IP=0 FDC=4
M0 OUT IN1 PWR PWR p L=1.75e-06 W=2e-05 i=2 $X=5125 $Y=51000 $D=1
M1 PWR IN2 OUT PWR p L=1.75e-06 W=2e-05 i=3 $X=13125 $Y=51000 $D=1
M2 7 IN1 GND GND n L=1.25e-06 W=1.5e-05 i=1 $X=5375 $Y=14000 $D=0
M3 OUT IN2 7 GND n L=1.25e-06 W=1.5e-05 i=0 $X=13375 $Y=14000 $D=0
.ENDS
