EWX

DFM measurement function

Returns the minimum distance between an edge pair on the input layer if the separation is parallel to the x-axis or 0 otherwise.

Usage

EWX(layer)

Arguments

layer

A required error layer with one or two edges per cluster.

Return Values

Floating-point number in user units.

Description

Returns the minimum separation distance pairs of edges in an edge cluster only if that separation is parallel to the x‑axis. The function returns 0 for single edges and intersecting edges, or if the minimum separation does not run parallel to the x-axis.



Measurements are taken in the context of the cell in which the property expression is evaluated, which may not be the same cell in which the geometries originated. See “Cell-Context-Dependent Property Functions Results in Hierarchical Batch Runs”. The return value of functions that depend on the coordinate axes depends on the cell context; these functions do not provide predictable results in hierarchical batch runs. For DFM Property operations, the GLOBALXY keyword can be used to provide axis-aligned measurements that are meaningful in the global frame of reference. See “GLOBALXY” for details.
