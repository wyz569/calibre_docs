pin_net

Device property computation and device annotation language function

Returns the net number of a specified pin. In hierarchical applications, this function returns node number local to the cell, which is a limitation.

Usage

PIN_NET(“pin_name”)

Arguments

pin_name

A required name of a device pin.

Return Values

Integer.

Examples

This is a test of a drain pin’s node ID to see if it matches the VDD net ID:

if ( pin_net(D) == named_net( "VDD" ) )
    // case 1 branch
else
    // case 2 branch

This is only useful in flat LVS.
