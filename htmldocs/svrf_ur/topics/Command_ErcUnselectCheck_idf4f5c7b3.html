ERC Unselect Check

Specification statement

Specifies rule check statements and rule check groups not to be executed by the ERC process.

Usage

ERC UNSELECT CHECK rule_check [rule_check …]

Arguments

rule_check

A required name of a rule check or rule check group from the rule file. You can specify rule_check any number of times in one statement.

Description

Disables ERC rule check statements and rule check groups specified in this statement during LVS.

You can specify this statement any number of times in your rule file.

The following algorithm describes how LVS applications (Calibre nmLVS and mask-mode ICtrace) select rule checks:

  1. Unselect all rule checks.

  2. Select rule checks specified in ERC Select Check statements.

  3. Unselect rule checks specified in ERC Unselect Check statements.

You can specify ERC options from the LVS Options pane in Calibre Interactive—nmLVS. See “Executing Electrical Rule Check (ERC) Operations in Calibre Interactive nmLVS” in the Calibre Interactive (New GUI) User’s Manual.

See also Group and LVS Execute ERC.

Examples

LVS EXECUTE ERC YES
GROUP ALL_CHECKS PWR1 PWR2 PWR3 PWR4
ERC SELECT CHECK ALL_CHECKS
ERC UNSELECT CHECK PWR1 PWR2 
// Exclude PWR1 and PWR2 ERC checks in the run
