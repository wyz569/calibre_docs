PEX Netlist LPE Using Extmode

Parasitic extraction

Instructs the netlist formatter to obtain LPE values from the mapfile based on the command line extraction mode.

Usage

PEX NETLIST LPE USING EXTMODE {NO | YES}

Arguments

NO

Required keyword that instructs the netlist formatter to obtain LPE values using the net parasitic models. This is the default behavior when you do not include this statement in the rule file.

YES

Required keyword that instructs the netlist formatter to obtain LPE values from the mapfile based on extraction mode, such as -r, -c, -rc, -rcc, and so forth.

Description

This statement instructs the netlist formatter to obtain LPE values from the mapfile based on the command line extraction mode, such as -r, -c, -rc, -rcc, and so forth specified by the formatter except when -all is used. If the -all extraction mode is specified by the formatter, then the extraction mode specified in the pdb step is used. If the -netmodel option is specified in the formatter step, then the LPE values follow the net models found in the net model file specified with PEX Netlist Select File, whether or not this statement is specified with the YES or NO keyword.

Examples

Example 1

To control which LPE values are used based on the formatting extraction mode, include the following statements in your rule file:

PEX NETLIST LPE USING EXTMODE YES
PEX BA MAPFILE mymapfile

The LPE values written to the netlist are obtained from the mapfile based on extraction mode. For example, if the following LPE exist in mymapfile:

%Models
N N LPE fred NORC 0 C 100 R 200 RC 300
P P LPE NORC 0 C 1 R 2 RC 3

And extraction mode is -r, then the values used for resistance will be 200 for models named fred and 2 for models with no name.

Example 2

If your netlist format is CALIBREVIEW and you want to use the LPE values from the mapfile, include the following statements in your rule file:

PEX NETLIST LPE USING EXTMODE YES
PEX BA MAPFILE mymapfile CALIBREVIEW

Example 3

To control which LPE values are used based on the formatting extraction mode, include the following statements in your rule file:

PEX NETLIST LPE USING EXTMODE YES
PEX BA MAPFILE mymapfile

If -rcc is specified for the pdb stage and -c is specified for the formatting stage: 

calibre -xrc -pdb -rcc rules.svrf
calibre -xrc -fmt -c rules.svrf

then -c extraction mode is used to determine which LPE values are written to the netlist.
