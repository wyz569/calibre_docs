TVF Errors

This section describes compilation errors related to TVF.
Table 1. TVF Errors

Error #

Explanation

TRT1

A TVF Function name is required but is not specified.

TRT2

A call is made to a TVF Function that does not exist.

TRT3

The TVF Function definition included either the EDGE or ERROR option in a case in which the options are not allowed. The EDGE and ERROR options are only used for TVF Function definitions used with the TVF layer operation.

TRT4

The TVF Function block did not compile.

TRT5

A TVF_NUMeric_FUNction call lacks an argument.

TVF1

Reports Tcl related errors from the TVF pre-processor when the -tvf_instrument command-line option is used.

TVF2

Reports Tcl related errors from the TVF pre-processor.

TVF3

A Tcl proc contains a syntax error or an unknown command name is used.

TVF4

A call to a Tcl proc has failed. Check the proc name and construction.

TVF5

A Tcl interpreter could not be created. This error occurs when the TCL library initialization failed.
