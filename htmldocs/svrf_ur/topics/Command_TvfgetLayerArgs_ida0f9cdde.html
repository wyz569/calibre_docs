tvf::GET_LAYER_ARGS

Application: runtime only.

Imports the layers passed to a TVF Function by a TVF layer operation, and sets up the layer handles. This command must be used once in a TVF Function statement that is called by a TVF layer operation.

Usage

tvf::GET_LAYER_ARGS layer1 [layerN …] [args]

Arguments

layer1 [layerN …]

A required argument that specifies the imported layer arguments.

The number of imported layer arguments must match the number of layer arguments passed in by the TVF layer operation unless the optional args argument is used.

args

An optional argument, where “args” is a literal name used to reference a list of layer arguments passed in by a TVF layer operation. The args argument includes layer arguments not matched to the layer1 … layerN list. If args is specified, it must be the final argument in the command. See the “Examples” section.

Return Values

None.

Description

Imports the layers passed to a TVF Function by a TVF layer operation, and sets up the layer handles $layer1, $layer2, and so on.

This command must be used once in a TVF Function statement that is called by a TVF layer operation. See “TVF Layer Operation”.

Examples

The following example shows how to use the “args” argument for tvf::GET_LAYER_ARGS to pass a list of two or more layers to the TVF Function. The function handles a layer list of unknown length and copies out the last layer as the rule check output. 

TVF FUNCTION my_list_func [/*
tvf::GET_LAYER_ARGS L1 L2 args
   set num_args  [llength $args]
   puts "length of args $num_args"
   if {$num_args > 0} {
      set last [expr {$num_args -1}]
      foreach lay $args {
      # print layer names for the arg layers
         puts "layer [tvf::GET_LAYER_NAME $lay]"
      }
      # copy out last layer
      puts "copied last layer: [tvf::GET_LAYER_NAME [lindex $args $last]]"
      tvf::OUTLAYER "copy [lindex $args $last]"
   } else {
      # do not have the args argument, copy out second layer
      puts "copied last layer: [tvf::GET_LAYER_NAME $L2]"
      tvf::OUTLAYER "copy $L2"
   }*/] 

rule_list { TVF my_list_func poly oxide via metal1 metal2 }

This produces the following output in the transcript: 

length of args 3
layer via
layer metal1
layer metal2
copied last layer metal2
