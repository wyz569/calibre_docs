string_compare

Trace Property computation language function.

Compares two strings to determine if they are identical.

Usage

STRING_COMPare(“strexpr1”, “strexpr2”)

Arguments

strexprN

A required string expression. Each string should be enclosed in quotation marks. The arguments can be either string constants or string property values returned by layout_string_value or source_string_value. The comma is literal.

Return Values

Integer.

Description

Compares the input strings strexpr1 and strexpr2 and returns the values given in the following table.

Table 1. STRING_COMPare Return Values

Return Value

Condition

0

strexpr1 and strexpr2 are equivalent.

-1

strexpr1 is lexicographically less than strexpr2.

1

strexpr1 is lexicographically greater than strexpr2.

Local string variables are not supported and cannot be given as arguments.

Case sensitivity of comparison is controlled by the current setting of the LVS Compare Case specification statement.

The behavior of this function is similar to the C strcmp() function with the exception of case sensitivity handling.
