PEX3

A layer parameter to a Capacitance Crossover operation or the first or INSIDE OF layer parameter to a Capacitance Intrinsic operation or any layer parameter to a Capacitance Nearbody operation with INSIDE OF specified does not appear as a parameter to a Capacitance Order operation. The name of the erroneous layer appears in the error message.

PEX4

A base layer in a Capacitance Intrinsic operation cannot appear as:

1. Any layer in a Capacitance Connection or resistance extraction operation.

2. Any layer in a Capacitance Crossover or Capacitance Nearbody operation.

3. The first layer in a Capacitance Intrinsic operation.

The name of the erroneous layer appears in the error message.

PEX5

This error has six possible sources:

1. The layer parameter to a Resistance Sheet operation does not appear as a connector layer in any Connect BY operation.

2. The layer parameter to a Resistance Sheet operation appears as a contact layer in a Connect BY operation.

3. The first layer parameter to a Resistance or Capacitance Connection operation does not appear as the first connector layer in any Connect BY operation.

4. The second layer parameter to a Resistance or Capacitance Connection operation does not appear as a connector layer in a Connect BY operation where the first layer parameter is the first connector layer.

5. The layer parameter to a Resistance or Capacitance Connection operation appears as a contact layer in some Connect BY operations.

6. The layer parameter in the Capacitance Order statement includes contact or via layers.

The name of the erroneous layer appears in the error message.

PEX6

A parasitic extraction operation is missing its property specification. The name of the erroneous operation appears in the message.

PEX7

A parasitic extraction operation has a superfluous property specification ( [ ).

PEX8

A layer in a Capacitance Intrinsic or Capacitance Crossover operation is not properly ordered within the Capacitance Order operation. Specifically (where L1 < L2 means that L1 is to the left of L2 in the Capacitance Order operation, and L1 > L2 means that L1 is to the right of L2 in the Capacitance Order operation) any of the following can cause this error:

  1. Capacitance Intrinsic Plate A [B] INSIDE OF C, where A > C.

  2. Capacitance Intrinsic [PLATE | FRINGE] A B INSIDE OF C, where if B is in the capacitance order operation, B > C.

  3. Capacitance Intrinsic Fringe A [B] INSIDE OF C D where A < C or A > D.

  4. Capacitance Crossover [PLATE] A B INSIDE OF C, where C < A and C > B, or where C < B and C > A.

  5. Capacitance Crossover [PLATE] A B INSIDE OF C D where C > A, or where C > B, or where D < A, or where D < B.

  6. Capacitance Crossover Fringe A B INSIDE OF C, where neither C < A nor C > A.

  7. Capacitance Crossover Fringe A B INSIDE OF C D where C > A or D < A.

  8. Capacitance Nearbody A INSIDE OF C where neither C < A nor C > A.

  9. Capacitance Nearbody A INSIDE OF C D where C > A or D < A.

  10. Capacitance Nearbody A B INSIDE OF C, where C < A and C > B, or where C < B and C > A.

  11. Capacitance Nearbody A B INSIDE OF C D where C > A, or where C > B, or where D < A, or where D < B.
