LENGTH

DFM measurement function

Returns the length of each edge on the input layer.

Usage

LENGTH(layer)

Arguments

layer

A required derived edge layer.

Return Values

Floating-point number in user units.

Description

Returns the length of each edge on the layer.



Examples

Refer to “Verifying Contact Run Length for Current Crowding” in Calibre Solutions for Physical Verification for an example of LENGTH usage.
