PEX Reduce Rgate

Parasitic extraction

Controls whether or not resistance for gate and delta nodes is reduced by TICER or other reduction options during extraction. Used only in Calibre xACT and Calibre xACT 3D.

Usage

PEX REDUCE RGATE { YES | NO }

Arguments

YES

A keyword that instructs the tool to reduce resistance for gate and delta nodes during extraction.

NO

A keyword that instructs the tool not to reduce resistance for gate and delta nodes during extraction. This is the default behavior if you do not include this statement in your rule file.

Description

An optional statement that controls whether or not resistance for gate and delta nodes is reduced by TICER or other reduction options during extraction. Use this statement to control reduction of gate resistance generated by PEX Extract Rgate.

When PEX Reduce Rgate is set to NO, gate resistance is not reduced by PEX Reduce Analog, PEX Reduce Digital, PEX Reduce ROnly, and PEX Reduce TICER statements. When PEX Reduce Rgate YES is specified, the gate-related resistor networks are treated by reduction as regular parasitic resistors and may be subject to reduction.

Examples

Example 1

To protect gate resistors from being reduced, include the following statement in your rule file:

PEX REDUCE RGATE NO

Example 2

To allow the reduction of gate resistors in the extracted netlist, include the following statement in your rule file:

PEX REDUCE RGATE YES
