LVS Property Initialize Function Reference

The following summarizes the functions available in the LVS Property Initialize language.
Table 1. LVS Property Initialize Built-In Language Functions

Function

Definition

PROPERTY

Defines numeric properties to be initialized.

Debugging functions

DEBUG

Enables debug monitoring during LVS Property Initialize computation.

WARN

Issues a warning to the transcript each time the LVS Property Initialize program attempts an impossible computation.

Property Initialization Functions

input_numeric_value

Returns property values in the input database.

is_missing

Determines whether a property is missing on an instance.
