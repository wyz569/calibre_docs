PRECISION

Function for DFM expressions

Returns the value of the Precision specification statement in the SVRF rule file.

Usage

PRECISION()

Arguments

None.

Return Values

Positive floating-point number.

Description

The function returns the value of the Precision specification used in the SVRF rule file. However, if the function is used within a DFM Property operation with the DBU keyword specified, the function returns 1.0.
