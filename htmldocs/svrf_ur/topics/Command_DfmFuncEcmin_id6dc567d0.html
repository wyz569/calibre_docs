ECMIN

DFM measurement function

Returns the smaller value of the two endpoint coordinates of the EC segment, where EC is the mutual edge projection length of two edges or the length of a single edge. Supported only for DFM Property operations.

Usage

ECMIN(layer)

Arguments

layer

A required layer name. The layer may be an edge layer or a derived error layer (edge cluster) with one or two edges per cluster.

If layer is a secondary layer, ECMIN must be used within a per-shape function (MIN, MAX, SUM, PRODUCT, or VECTOR).

Return Values

Coordinate in the local cell space.

Description

Returns the smaller value of the two endpoint coordinates of EC (mutual edge projection), measured along the axis parallel to EC. ECMIN is always less than or equal to ECMAX. The value is returned in the local cell space.

See “EC Group: Projection Endpoints” for a complete description of the endpoint calculation. The endpoint functions are intended for use with Manhattan edges or with edge pairs with at least one edge parallel to an axis.



The coordinate value is returned in the local cell space. The local coordinate system may not be the same as the cell the geometries originated in if promotion occurs, causing the return value to depend on the cell context. See “Cell-Context-Dependent Property Functions Results in Hierarchical Batch Runs” and Figure 2 in the topic “EC Group: Projection Endpoints.”
