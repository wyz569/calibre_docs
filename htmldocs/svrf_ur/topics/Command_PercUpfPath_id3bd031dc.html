PERC UPF Path

Specification statement

Specifies the pathname of a Unified Power Format file.

Usage

PERC UPF PATH filename

Arguments

filename

A required pathname of a Uniform Power Format (UPF) 2.0 file.

The filename parameter can contain environment variables. See “Environment Variables in Pathname Parameters”.

Description

This statement specifies the pathname of a UPF 2.0 file. The format of the file follows the IEEE Std 1801™-2009 document. When this statement is used, Calibre PERC reads the contents of the filename and makes them available to the perc::get_upf_data or perc::get_upf_pst_data commands in the rule file. Merging of power state tables in the UPF file is supported through the perc::merge_upf_pst command.

This statement may appear only once in a rule file. If this statement is not used, UPF is not used during the run.

The UPF commands and options supported by Calibre PERC are shown in the following table. With the exception of set_domain_supply_net, all of the supported command settings are accessed in a rule file using perc::get_upf_pst_data.

After reading the input filename, Calibre PERC writes an output UPF file that contains the recognized commands. The output file name is the PERC Report File name with an additional suffix “.upf”. The output file is divided into supported commands (as in Table 1) followed by unsupported commands.

There are two levels of error checking:

  • Calibre PERC checks the UPF syntax and aborts the run immediately if the UPF file has syntax errors.

  • After the design data has been loaded, Calibre PERC checks the UPF semantics and indicates the UPF file is invalid if the file does not match the design. An invalid UPF file does not abort the entire run, but the rule checks that need access to UPF data will fail.

Calibre PERC outputs UPF errors to the PERC Report.

Examples

In this example, UPF commands in power_defs.upf are available for use during the run.

PERC UPF PATH "power_defs.upf"
PERC LOAD power_functions INIT power_init SELECT check1 check2
PERC NETLIST SOURCE
PERC REPORT perc.report
