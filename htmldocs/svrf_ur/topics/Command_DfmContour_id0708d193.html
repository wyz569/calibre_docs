DFM Contour

Layer operation

License: Calibre YieldEnhancer. Substitute: Calibre YieldAnalyzer

Generates a derived polygon layer defined by the contour specifications in DFM Spec Contour. A set of output layers provide a contour map of a DFM property value or layer density.

Usage

DFM CONTOUR “spec_name” output_name

Arguments

spec_name

A required argument specifying the name of a contour specification to use when generating the contour output. The contour specification is defined by a DFM Spec Contour statement.

output_name

A required argument specifying the contour output to generate. The outputs are defined with the OUTPUT keyword in the corresponding DFM Spec Contour statement.

Description

Generates a contour map that can be used to visualize property values across a whole design without the need to calculate the property value at a high resolution.

The output layer of DFM Contour is the contour map of regions that meet the constraint for that output_name. DFM Contour operations referencing the same spec_name are executed concurrently.

For complete details, see the discussion and examples provided with DFM Spec Contour.
