DFM Merge

Layer operation

License: Calibre nmDRC-H

Generates a merged copy of the input layer.

Usage

DFM MERGE layer

Arguments

layer

A required original layer or derived polygon layer.

Description

DFM Merge performs a partial flattening of hierarchical data, resulting in a merged copy of the input layer. For flat data, DFM Merge is functionally equivalent to the Copy operation. DFM Merge preserves hierarchy better than the Merge operation.

During DFM Merge, polygon data is promoted up the hierarchy until the entire polygon is unified and promoted into the highest common cell. Individual data are never promoted over a hierarchical boundary. These resulting layers are referred to by the Calibre transcript as fully‑merged layers and annotated as HIER‑FMF.

DFM Merge has limited uses. Although the result of a DFM Merge operation is still a layer in hierarchical form (versus flat), a considerable amount of partial flattening can occur in generating the layer.

Examples
