tvf::unblock_transcript

Application: compile-time only.

Ends transcript blocking set with tvf::block_transcript. Output to the transcript and to the rule file specified with the -E command line option is resumed.

Usage

tvf::unblock_transcript

Arguments

None.

Return Values

None.
