Rule File Variables in Runtime TVF

You can access variables defined with the SVRF Variable statement using the tvf::svrf_var statements. You can also make use of global Tcl variables and the initialization section of the TVF layer operation.

Accessing SVRF Variable Values

Variables can be defined in a rule file using the SVRF Variable statement. Rule file variables are accessed in runtime TVF using the tvf::svrf_var function.

See the tvf::svrf_var command reference page for syntax and an example.

Global Tcl Variables and the TVF Layer Initialization Section

You can accomplish the same thing with the added flexibility of reassigning variable values using global Tcl variables and the initialization section of the TVF Layer Operation, as shown here:

	tvf::GET_LAYER_ARGS L1 L2
	global globalVar1
	set dist $globalVar1
	tvf::SETLAYER ext1 = "EXTERNAL $L1 $L2 <= $dist REGION OPPOSITE"

*/]

layer1 = TVF myTVFscript A B [
	global globalVar1
	set globalVar1 0.1

	global globalVar1
	set globalVar1 0.2
]
