Layout Placement Multiple

Specification statement

Specifies a value that is used to detect whether placements are offgrid.

Usage

LAYOUT PLACEMENT MULTIPLE value

Arguments

value

A required positive integer constant dimensioned in database units.

Description

Detects whether placements are offgrid by dividing the X or Y coordinates of placements by the specified value. This statement is used in conjunction with Layout Input Exception Severity PLACEMENT_MULTIPLE to determine how to handle a placement problem when the results are not an integer. This statement may be specified only once and may not be used with Layout Magnify or with the MAG argument for the Layout Path[2] statements.

Examples

LAYOUT PRECISION 10000
PRECISION 2000 // precision is reduced
LAYOUT PLACEMENT MULTIPLE 2
LAYOUT INPUT EXCEPTION SEVERITY PLACEMENT_MULTIPLE 1
