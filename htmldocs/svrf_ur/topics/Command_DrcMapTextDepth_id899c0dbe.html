DRC Map Text Depth

Specification statement

Specifies the hierarchical depth from which to read text objects for the DRC Map Text YES specification statement. Used only in Calibre nmDRC-H.

Usage

DRC MAP TEXT DEPTH {ALL | PRIMARY | depth}

Arguments

ALL

Keyword that instructs Calibre nmDRC-H to read text objects from all levels of layout hierarchy. This is the default behavior.

PRIMARY

Keyword that instructs Calibre nmDRC-H to read text objects from the top-level cell only. This is equivalent to a depth of 0.

depth

Non-negative integer, variable, or numeric expression, which instructs Calibre nmDRC-H to read text objects down to the hierarchical level of depth. The top level is 0.

Description

Controls the depth for reading text objects for the DRC Map Text YES specification statement.

See also DRC Map Text Layer.

Examples

Example 1

Basic syntax to map all text objects for output from the top two levels of hierarchy.

DRC MAP TEXT DEPTH 1 // read text objects from top two levels
DRC MAP TEXT YES
DRC RESULTS DATABASE cell.gds GDSII
DRC MAXIMUM RESULTS ALL

Example 2

Using an environment variable.

VARIABLE level ENVIRONMENT
DRC MAP TEXT DEPTH level
DRC MAP TEXT YES
DRC RESULTS DATABASE cell.gds GDSII
DRC MAXIMUM RESULTS ALL
