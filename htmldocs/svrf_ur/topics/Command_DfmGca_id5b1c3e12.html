DFM GCA

Layer operation

License: Calibre YieldAnalyzer

Outputs a derived polygon layer that represents areas susceptible to producing open or short circuits in the presence of random particles of a given radius.

Usage

For Outputting Areas Susceptible to Short Circuits

DFM GCA input_layer [input_layerN …]
SHORT
R defect_radius
[D depth]
[EXCLUDE FAULTS BETWEEN layer1 '&' layer2 …]
[INSIDE OF {EXTENT | x1 y1 x2 y2 | LAYER extent_layer [extent_layerN …]}]
[METRIC {SQUARE | OCTAGONAL}]
[POLYGONAL]

For Outputting Areas Susceptible to Open Circuits

DFM GCA input_layer [input_layerN …]
OPEN
R defect_radius
[D depth]
[INSIDE OF{EXTENT | x1 y1 x2 y2 | LAYER extent_layer [extent_layerN …]}]
[METRIC {SQUARE | OCTAGONAL}]
[PORTSLAYER port_layer1 [GROUPED BY group_layer1] ...]

For Outputting Areas Susceptible to Open Circuits Excluding Short Circuit Areas

DFM GCA input_layer [input_layerN …]
OPEN EXCLUDING SHORT
R defect_radius
[D depth]
[INSIDE OF{EXTENT | x1 y1 x2 y2 | LAYER extent_layer [extent_layerN …]}]
[METRIC {SQUARE | OCTAGONAL}]

For Outputting Areas Susceptible to Short Circuits Excluding Open Circuit Areas

DFM GCA input_layer [input_layerN …]
SHORT EXCLUDING OPEN
R defect_radius
[D depth]
[INSIDE OF{EXTENT | x1 y1 x2 y2 | LAYER extent_layer [extent_layerN …]}]
[METRIC {SQUARE | OCTAGONAL}]

For Outputting Areas Using Previously Defined Specifications

DFM GCA
SPEC spec_name fault_name
R defect_radius
[D depth]
[INSIDE OF{EXTENT | x1 y1 x2 y2 | LAYER extent_layer [extent_layerN …]}]
[METRIC {SQUARE | OCTAGONAL}]

Arguments

input_layer [input_layerN …]

A required argument that specifies one or more original or derived polygon input layers. Multiple input_layers are allowed only if none of the shapes on one specified input_layer touch or overlap any of the shapes on another specified input_layer. All input layers must appear in Connect statements.

Application Keywords for Short and Open Critical Area Isolation

These keywords specify the type of failure for which critical areas are output. Either SHORT, OPEN, SHORT EXCLUDING OPEN, OPEN EXCLUDING SHORT, or SPEC keywords must be specified for the DFM GCA command.

SHORT [{EXCLUDE FAULTS BETWEEN layer1 '&' layer2} …] [POLYGONAL]

A keyword, with exclusive options, that outputs short defect types between different nets on the input layer. For more information regarding SHORT and its options, see DFM CAF.

OPEN [PORTSLAYER port_layer [GROUPED BY group_layer]]

A keyword, with exclusive options, that outputs open defect types for nets on the input_layer. For more information regarding OPEN and its options, see DFM CAF.

SHORT EXCLUDING OPEN

A keyword that outputs short circuit defect types between different nets on the input layer, and subtracts the region of overlap outputted by the implicit SHORT and OPEN keywords from the total critical area.

OPEN EXCLUDING SHORT

A keyword that outputs open circuit defect types for nets on the input layer, and subtracts the region of overlap outputted by the implicit SHORT and OPEN keywords from the total critical area.

SPEC spec_name fault_name

A keyword that outputs areas of defect types with definitions previously established by DFM Spec Random Spot Fault. The spec_name specifies the DFM Spec Random Spot Fault definition and the fault_name specifies the fault name of interest within the specified definition. For more information regarding specifications and command requirements, see DFM Spec Random Spot Fault.

General Keywords

R defect_radius

A required keyword that specifies one defect_radius value in user units. For more information, see the DFM CAF command.

D depth

An optional keyword and argument that specifies the minimal overlap depth of a defect edge with a layout shape for a short or open to be considered a fault.

For more information see the DFM Critical Area command.

INSIDE OF {EXTENT | x1 y1 x2 y2 | LAYER extent_layer [extent_layerN …]}

An optional keyword and argument set that defines the analysis region. By default, the analysis region is the layout extent. You can define a different analysis region by either specifying coordinates (x1 y1 x2 y2) in user units for the lower-left and upper-right corners of a rectangular data capture extent, or by specifying one or more extent_layers with the LAYER keyword. LAYER specifies that the analysis region is the area within the specified extent_layers.

For more information, see the DFM Critical Area command.

METRIC {SQUARE | OCTAGONAL}

An optional keyword set that specifies the approximate shape to use for the defect. The default is SQUARE.

Description

Outputs a derived polygon layer that represents areas susceptible to producing open or short circuits in the presence of random particles of a given radius. This operation produces flat output and is intended primarily for visualization of the DFM CAF command. DFM GCA is not intended for use in production flows.

Environment Variables

Three environment variables are provided to control the behavior of the DFM GCA command:

  • CALIBRE_DFM_CAA_MAX_EDGES_PER_TILE — Specifies the upper bound for the partition size as the average flat edge count per tile. The default value is 1000000. If this variable is set to a non-positive value, then every cell is processed as one tile. In order to enable the multi-threaded or Calibre MTflex parallelism of the DFM CAF command, this variable must be set to a value respective to the data extent.

  • CALIBRE_DFM_CAA_MIN_EDGES_PER_TILE — Specifies the lower bound for the partition size as the average flat edge count per tile. The default value is 10000. Low scalability of parallel processing may result when sparse layers are processed in fewer tiles. In this case, the operation uses tiles of smaller size, but not less than the specified lower bound, so that all threads are utilized.

  • CALIBRE_DFM_OPEN_GCA_DEANGLING_PRECISION — Specifies the precision in user units of the grid‑aligned critical area for faults found with the OPEN keyword. The default is 0.01. In general, smaller values can impact performance on non-rectilinear designs. See “Deangle” for grid‑alignment behavior.

Examples

Example 1 — Area-Based Short Output

//Outputs geometric critical area-based short circuit for layer M1
// with a defect radius of 0.2.
M1_SHORT_GCA = DFM GCA M1 SHORT R 0.2
