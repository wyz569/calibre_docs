Resistance Connection

Parasitic extraction

Defines the proportionality constants for computing the resistance of current flowing between two conductor layers.

Usage

Syntax 1

RESISTANCE CONNECTION layer1 layer2 [VIANAME via_layer] ‘[’n1 n2‘]’

Syntax 2

RESISTANCE CONNECTION VIANAME via_layer ‘[’n1 n2‘]’

Arguments

layer1

A required original layer or a derived polygon layer.

layer2

A required original layer or a derived polygon layer.

VIANAME via_layer

A keyword set that specifies a via layer. This keyword set is optional for Syntax 1 and required for Syntax 2.

‘[’n1 n2‘]’

A pair of required strings that must be enclosed in brackets ( [ ] ).

The strings are described as follows:

  • n1 — A non-negative real number, which is resistance multiplied by area (ohms * microns squared), that specifies the inverse area proportionality constant.

  • n2 — A non-negative real number that specifies the inverse edge proportionality constant. Typically, n2 is 0.

For more information about units of resistance, refer to the Unit Resistance statement.

Description

Defines the proportionality constants for computing the resistance of current flowing between two conductor layers. The resistance is based on the area and edge effects of the connection between layer1 and layer2 in the presence of a contact layer or via. Use this statement only when via resistance is not modeled in the calibrated rule files. If via resistance is modeled in the calibrated rule files, then this statement is ignored.

Input layers must appear in a Connect BY statement for the given connectivity mode and in the same layer order. The resistance value is based on the area and edge effects of the connection common to both layer1 and layer2. Either or both of these effects can be modeled by specifying a non-zero number value.

In Syntax 1, use the VIANAME via_layer keyword set to specify a particular via layer between a certain pair of conductor layers, layer1 and layer2. When this keyword is specified without layer1 and layer2 as shown in Syntax 2, the via resistance is changed for all vias of the specified via_layer type.

You can define the layer1 and layer2 proportionality constants (n1 and n2) with the Resistance Connection operation only once and in the given order of layer1 and layer2.

Examples

Example 1

CONNECT metal poly BY contact
RESISTANCE CONNECTION metal poly [0.05  0.1]
// The next line will give a PEX5 error because of layer order:
//RESISTANCE CONNECTION poly metal [0.05  0.1]

Example 2

Consider the following rule file statements, in which two layers are directly connected:

CONNECT layer1 layer2
RESISTANCE CONNECTION layer1 layer2 [0.83 0]

In this case, the common area is large, as shown in the following figure.

Figure 1. Example Two-Layer Resistance Connection

Example 3

Consider the following rule file statements, in which two layers are connected through a third layer, usually a via:

CONNECT layer1 layer2 BY via 
RESISTANCE CONNECTION layer1 layer2 [0.83 0]

The common area is only that which is common to all involved layers, as shown in the following figure.

Figure 2. Example Connect By Resistance Connection

Example 4

Consider the following rule file statements, where there are two different via layer types used to connect layer1 and layer2. 

CONNECT layer1 layer2 BY via0 
CONNECT layer1 layer2 BY via1

Here is a sample taken from an extracted netlist that shows the foundry calculated resistance for a via0 via, R31, as 14 ohms:

...
R31 4:12 4:46 14
R58 4:46 4:45 16.0031
...
Figure 3. Example Resistance Connection With VIANAME

To change the extracted resistance (R31) for the via0 via between layer1 and layer2, to 15 ohms use the following statement:

RESISTANCE CONNECTION layer1 layer2 VIANAME via0 [0.01536 0]

Here is the resulting sample taken from an extracted netlist with this statement specified:

...
R31 4:12 4:46 15
R58 4:46 4:45 16.0031
...

Example 5

Use Syntax 2 to change the resistance for all via0 layer type vias that exist between any two layers. For example:

RESISTANCE CONNECTION VIANAME via0 [0.01536 0]

The extracted resistance for vias of the layer type via0 would only be changed.
