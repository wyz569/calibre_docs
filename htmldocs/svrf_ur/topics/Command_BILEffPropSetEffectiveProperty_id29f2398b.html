set_effective_property

Effective property computation language TVF function.

Assigns an effective property value and returns it to the calling environment. This command must be used in a Tcl proc to assign effective property values declared in a TVF_NUMeric_FUNction command.

Usage

$property set_effective_property numeric_expression

Arguments

$property

A required Tcl variable call to a property value declared in a tvf_numeric_function command.

numeric_expression

A required Tcl expression that evaluates to a number.

Return Values

Number.

Examples

The following code shows the primary elements involved in calculating the effective resistance of series resistors. The LVS Reduce Series Resistors specification statement declares the property r as the one to be reduced. The TVF_NUMeric_FUNction command calls for the property r to be calculated in the set_r_props_series proc. The property is calculated and then assigned by set_effective_property. The return command is simply to satisfy internal requirements of TVF_NUMeric_FUNction and plays no role in property values.

LVS REDUCE SERIES RESISTORS yes
[
  EFFECTIVE r
  tvf_numeric_function::tvf_redux::set_r_props_series( r )
]
TVF FUNCTION tvf_redux [/*
  proc set_r_props_series { r } {
    set rsum 0
    foreach_instance {
      set rsum [ expr { $rsum + [$r] } ]
    }
    $r set_effective_property $rsum
    return 0.0
  } 
*/]
