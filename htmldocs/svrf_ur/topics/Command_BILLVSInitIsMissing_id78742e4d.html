is_missing

LVS Property Initialize computation language function.

Determines whether a property is missing on an instance.

Usage

IS_MISSING(numeric_expression)

Arguments

numeric_expression

A required numeric expression. Numeric expressions are defined in “Property Computation Language Characteristics”; see the discussion of assignment operators.

Return Values

Integer.

Description

This function allows user programs to avoid overwriting property values that are actually present in input data.

If the value of the given numeric_expression is the special “missing” value from an input_numeric_value function, then IS_MISSING( ) returns 1; otherwise, the function returns 0.
