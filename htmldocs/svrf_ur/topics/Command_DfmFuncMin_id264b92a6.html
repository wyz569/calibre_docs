MIN

Per-shape function for DFM expressions

Computes the minimum of an expression over a set of layer objects. Valid in DFM Property and DFM Analyze.

Usage

MIN(arg_expr[,default])

MIN::FAILARG(arg_expr[,default])

Arguments

arg_expr

A required argument that specifies the expression to evaluate. The expression must include at least one layer-dependent measurement function. The expression can reference only one layer. More than one measurement function may be used in the expression, but the same layer must be referenced in each measurement function. The arg_expr must evaluate to a number.

The arg_expr cannot contain another per-shape function—per-shape functions cannot be nested.

default

An optional argument that specifies a default numeric value if the set of shapes the per-shape function operates on is empty or if the expression evaluation fails for all shapes.

The default can be an expression that evaluates to a numeric value. It can contain any function valid in a DFM expression, including per-shape functions. If a default is needed but a default expression fails to evaluate, then the call to the per-shape function also fails to evaluate. However, if the per-shape function evaluates without using a default, then the default expression is not evaluated.

MIN::FAILARG

An alternate syntax form that specifies the per-shape function fails to evaluate if arg_expr fails to evaluate for any shape. By default, a shape is dropped from the calculation if evaluation fails, and the evaluation continues for the remaining shapes.

Return Values

Floating-point number in user units.

Description

Returns the minimum of arg_expr evaluated on each shape, where the set of shapes depends on how the function is used in the DFM Property or DFM Analyze operation. Per-shape evaluation applies to the secondary layers in DFM Property and to any layer in DFM Analyze; without a per-shape function the default behavior for those layers is to sum each measurement over all shapes before evaluating the overall expression.

The topic “Per-Shape Evaluation for DFM Measurement Functions” includes the following information:

  • Usage rules

  • Evaluation details

  • Measurement function availability

  • Examples using default and ::FAILARG

Examples

The following statement computes the minimum and maximum area of polygons on layer B intersecting each polygon on layer A:

DFM PROPERTY A B 
    [ min_B = MIN( AREA(B) ) ] 
    [ max_B = MAX( AREA(B) ) ]
