PEX Resistance Open Layer

Specification statement

Specifies layers to be treated as infinite resistance layers. Used by Calibre PERC LDL applications.

Usage

PEX RESISTANCE OPEN LAYER layer [layer ...]

Arguments

layer

A required parasitic resistance layer. More than one layer may be specified.

Description

Specifies layers to be treated as infinite resistance layers. When this statement is used, the specified layers are omitted from parasitic extraction and treated as if they do not exist in the layout. This leaves any connections the specified layers might make as open circuits in the parasitic network.

This statement can be useful in certain cases where substrate layers should be omitted from the parasitic resistance network. This can omit substrate pins from intentional devices, which is sometimes desirable.

Examples

Example 1

psub = (EXTENT) NOT NWELL
...
// omit psub as a resistance layer
PEX RESISTANCE OPEN LAYER psub

Example 2

The conductor layers, M1, M2, and M3 connected with via layers V1 and V2, produce the resistive network shown in Figure 1.

CONNECT M1 M2 BY V1
CONNECT M2 M3 BY V2
Figure 1. Resistor Network Before

To completely ignore M2 during extraction as shown in Figure 2, specify the following statement in your rule file:

PEX RESISTANCE OPEN LAYER M2
Figure 2. Resistor Network After
