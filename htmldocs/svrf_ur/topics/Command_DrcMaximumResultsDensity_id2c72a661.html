DRC Maximum Results Density

Specification statement

Specifies a default value for the MAXIMUM keyword in the Density operation.

Usage

DRC MAXIMUM RESULTS DENSITY {number | ALL}

Arguments

number | ALL

A required argument or keyword choice that specifies a default value for the MAXIMUM keyword in the Density operation. MAXIMUM is used with the RDB keyword to limit the number of results that are output to the ASCII results database generated by the Density operation.

  • number — Specifies a non‑negative integer for the number of results to output.

  • ALL — Specifies that all results are output.

Description

The DRC Maximum Results Density command specifies a default value to use for the MAXIMUM keyword in the Density operation. The default value is used whenever MAXIMUM is not specified. This command does not apply in cases for which the MAXIMUM keyword in the Density operation cannot be used, such as when the COMBINE or MAGNITUDE keywords are specified.

The default is unlimited in hierarchical and flat applications when Density MAXIMUM or DRC Maximum Results Density is not specified.

Examples

This example specifies that the maximum number of results written to the density.rdb file is 500. 

DRC MAXIMUM RESULTS DENSITY 500
density_rule { 
     DENSITY layer1 > 0.5 WINDOW 0.1 RDB density.rdb 
}
