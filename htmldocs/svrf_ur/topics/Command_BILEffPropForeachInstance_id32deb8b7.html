foreach_instance

Effective property computation language TVF function.

Iterates over all instances in the input group and executes a Tcl script. This function must appear in a Tcl procedure in a TVF Function block.

Usage

foreach_instance { Tcl_script }

Arguments

Tcl_script

A required Tcl script that performs calculations on the instance input group. The Tcl_script is typically enclosed in braces ({ }) to prevent premature evaluation of variables. The Tcl_script contains calls to numeric TVF objects (see “tvf_numeric_function”) for calculating various property values.

Return Values

None.

Examples

The following block iterates over all device instances passed in and calculates the value rsum. The $R numeric TVF object evaluates to the current instance’s R property value, which is added to $rsum. 

foreach_instance {
  set rsum [ expr { $rsum + [$R] } ]
}
