ANGLE

DFM measurement function

Returns the angle between two edges in an edge cluster. Supported only for DFM Property operations.

Usage

ANGLE(layer)

Arguments

layer

A required error layer with exactly two edges per cluster.

Return Values

Floating-point number in radians.

Description

Returns the angle (in radians) between sets of two edges in an edge cluster. The result is converted to the range [0, π].



ANGLE evaluation fails if either input edge is a point. Evaluation failure can be protected against by using an expression chain; see “Expression Chains”.

The ANGLE function must be used within the scope of a per-shape function. See “Per-Shape Evaluation for DFM Measurement Functions” for details.

Examples

In the following DFM Property operation, the input layer is expected to consist of two-edge clusters, such as are produced with dimensional check operations. ANGLE must always be within a per-shape function, so it is within MIN(). An expression chain assigns a default value of 1000 in case of evaluation failure. 

DFM PROPERTY meas_lay
		[ "angle" = MIN(ANGLE(meas_lay) * 180/PI()) ]
			[ "angle" = 9999] // default value in case of evaluation failure

Other per-shape functions like VECTOR or MAX could be used here and the measured values would not differ. The output of VECTOR is of vector type, which can be used in downstream vector functions, while MIN and MAX output scalar numbers.
