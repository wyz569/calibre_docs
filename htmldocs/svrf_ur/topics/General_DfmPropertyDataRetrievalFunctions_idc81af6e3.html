DFM Property Data Retrieval Functions

The Device Layer ANNOTATE operation and the Device statement’s device_annotation_program support data access functions that are used in conjunction with the DFM Property operation to retrieve values for use within device property computations. This capability is part of the TVF functional interface for device property computation. For vector functions, there are corresponding functions for converting DFM vector properties into Device enclosure function measurement format.

The DFM Property data retrieval functions are these:

dfm_numeric_value

dfm_string_value

dfm_numeric_array

dfm_vector_value

dfm_vector_array

These functions are used in conjunction with dfm_vector_array and dfm_vector_value to convert the vector objects they return to Device enclosure-style measurements.

device::dfm_vec_arr_measurements

device::scaled_dfm_vec_arr_measurements

device::dfm_vec_measurements

device::scaled_dfm_vec_measurements

These functions are also applicable to the “Historical Well Proximity Calculation Methods”.
