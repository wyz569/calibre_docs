PEX Resistance Parameters ... RSH

Parasitic extraction

Specifies via layer specific settings used to extract lateral resistance for via bars.

Usage

PEX RESISTANCE PARAMETERS layer1 layer2 RSH value MAXLENGTH length MAXAREA area

Arguments

layer1 layer2

A required parameter set used to specify a via between layer1 and layer2.

RSH value

A required keyword set that specifies the sheet resistance for lateral extracted resistors along the via perpendicular to the via layer thickness.

MAXLENGTH length

A required keyword set that defines the distance used to control the distribution of reduced vias. The distance between the distributed vias will not exceed this limit. The length argument is a non-negative real number in units set by Unit Length.

MAXAREA area

A required keyword set that specifies an area threshold for large area via connections. Via connections with areas greater than this value are modeled as a set of distributed connectors.

Description

This optional statement specifies via layer specific settings used to extract lateral resistance for via bars. During extraction the via bar is segmented according to the MAXLENGTH and MAXAREA values, where each segment will be assigned a lateral resistor according to the RSH value and segment dimensions. Figure 1 shows the lateral extracted resistors.

In addition to the lateral extracted resistors, vertical resistors are still extracted using the standard via model. This statement may be specified any number of times. If this statement is not specified, the tool uses the standard via model and lateral resistors across the via layer plane will not be extracted.

Figure 1. Lateral Extracted Resistors

Examples

Include the following statement in your rule file to extract lateral resistance for a via bar between layers metal1 and metal2:

PEX RESISTANCE PARAMETERS metal1 metal2 RSH 0.03 MAXLENGTH 0.5 
	MAXAREA 0.25

In this example, a via layer is defined between metal1 and metal2. Lateral resistors are extracted where sheet resistance is 0.03 ohms per square. The threshold for x or y dimensions of the via bounding box is 0.5 microns. The area threshold for breaking up the via into smaller sections is 0.25 square microns (MAXLENGTH squared).
