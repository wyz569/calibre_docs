Not With Edge

Layer operation

Selects polygons that do not have an edge or edge segment on a second layer.

Usage

NOT WITH EDGE layer1 layer2 [constraint]

Arguments

layer1

A required original layer or layer set, or a derived polygon layer.

layer2

A required derived edge layer.

constraint

An optional constraint listed in the “Constraint Notation” column of Table 1 in the “Constraints” topic. Specifies the number of edges or edge segments a layer1 polygon must not have on layer2 to be selected by the operation. The constraint must contain non-negative integers. While a constraint value of 0 is legal, it will always return an empty layer.

Description

Selects all polygons on layer1 that share no edges on layer2. This is the complement of the With Edge operation. If layer2 is empty, the Not With Edge operation generates output equivalent to the polygon data on layer1.

Note:

When using a constraint, be aware that collinear edge segments are counted individually and not as a single edge. Refer to “Edge Breaking” in the Calibre Verification User’s Manual.
