tvf::get_tvf_arg

Application: compile-time only.

Gets the argument specified with the -tvfarg command line option.

Usage

tvf::get_tvf_arg

Arguments

None.

Return Values

Returns the argument specified with the -tvfarg command line option. If no -tvfarg option is given on the command line, the command returns an empty string.

Examples

Suppose the following command line invocation is used:

calibre -tvfarg my_argument

where my_argument is the value being passed to the TVF script. Note that my_argument cannot contain any spaces.

The value my_argument is accessed in compile-time TVF code with this command:

set tvf_arg [tvf::get_tvf_arg]
