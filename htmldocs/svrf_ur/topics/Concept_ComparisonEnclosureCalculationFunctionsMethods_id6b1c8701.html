Comparison of Enclosure Calculation Functions and Methods

The enclosure functions, the DFM Property data retrieval functions, and the Device TVF functions are useful for MOS device well proximity computation and similar applications. They are primarily used to extract physical parameters for BSIM4 device models.

The enclosure functions consist of the following:

  • device::enclosure_measurements, device::enclosure_parallel_measurements, and device::enclosure_perpendicular_measurements functions — Used only in Calibre and requires a Calibre Advanced Device Properties (ADP) product license. This function set is similar to the ENClosure_* functions, but has better performance in most cases.

  • DFM Property Data Retrieval Functions — Used only in Calibre and requires a Calibre Advanced Device Properties (ADP) product license.

    For information about using the DFM Property operation for device enclosure calculations, see “Well Enclosure or Stress Effect Calculations with DFM Property”.

  • enclosure_parallel and enclosure_perpendicular functions — Used only in Calibre and requires a Calibre Advanced Device Properties (ADP) product license. These are older functions that do not perform as well as device::enclosure_measurements for well proximity applications.

  • enclosure_vector function — Used only in Calibre and requires a Calibre nmLVS product license. For well proximity applications, this operation has poorer performance than those listed previously.
