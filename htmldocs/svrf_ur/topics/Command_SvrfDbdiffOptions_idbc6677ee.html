SVRF DBdiff Options

Specification statement

Specifies DBdiff command-line options for controlling FastXOR runs.

Usage

SVRF DBDIFF OPTIONS “dbdiff_option  [dbdiff_option ]…”

Arguments

“dbdiff_option”

A quoted string that includes one or more DBdiff command line options. The specified options are used during FastXOR runs.

Description

This specification statement is equivalent to setting the CALIBRE_FX_DB_DIFF_OPTIONS environment variable or changing the values in the fastxor.template file. The option list must be quoted and the delimiter between multiple options is the space character.

The entire list of options is printed for this command:

dbdiff -help

See “FastXOR Environment Variables” and “Layout Comparison Command Line Syntax” in the Calibre Layout Comparison and Translation Guide for more details.

This command takes precedence over the fastxor.template file and the CALIBRE_FX_DB_DIFF_OPTIONS, in that order.

CAUTION:

The options in SVRF DBdiff Options are not used in a FastXOR run started from Calibre Interactive. When using Calibre Interactive, the DBdiff options must be set in the GUI or with CALIBRE_FX_DB_DIFF_OPTIONS.

Examples

SVRF DBDIFF OPTIONS "-report diff.report –transform R90"
