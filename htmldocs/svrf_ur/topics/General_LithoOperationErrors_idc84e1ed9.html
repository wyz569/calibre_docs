Litho Operation Errors

This section describes compilation errors from Litho operations.
Table 1. Litho Operation Errors

Error #

Explanation

LTH1

The Litho operation is missing a required secondary keyword.

LTH2

The number of input layers specified for the Litho operation with the PSMGATE keyword was other than two or three.

LTH3

The keyword FILE cannot be specified in a Litho operation when the keyword PSMGATE is specified.

LTH4

The keyword FILE must be specified in a Litho operation when the keyword ORC, OPC, DENSEOPC, OPCVERIFY, MPC, MPCVERIFY, PSMGATE, or PRINTIMAGE is specified.

LTH5

There must be exactly two or three input layers for a Litho operation with PSMGATE specified.

LTH6

The FILE parameter in a Litho or Fracture operation is not followed by a filename or an inlined file.

LTH7

The Fracture or MDPmerge operation is missing a required secondary keyword.

LTH8

The FILE parameter is missing for RET, Fracture, or MDP operation.

LTH9

The FILE parameter in a RET, Fracture, or MDP operation is not followed by a filename or an inlined file.

LTH10

The FILE parameter for a LITHO operation has a syntax or semantic error.

LTH11

The FILE parameter for a Fracture operation has a syntax or semantic error.

LTH12

The FILE parameter for an MDP or RET operation has a syntax or semantic error.

LTH13

Either SMALLSIDEOUTTRAP or SPLITCD must be specified in an MDPstat operation.

LTH14

No keyword has been specified in an MDP or RET operation.

LTH15

The file parameter for a Density Convolve operation has a syntax or semantic error.

LTH16

A litho operation has API specified without OPC, ORC, PRINTIMAGE, DENSEOPC, MPC, or MPCVERIFY.

LTH20

EUV functionality is only available in conjunction with Calibre nmOPC or Calibre OPCverify runs. The SVRF statement must be either “LITHO EUV DENSEOPC” or “LITHO EUV OPCVERIFY”.

LTH22

EUV functionality is only available in conjunction with Calibre pxOPC, Calibre LPE, or Calibre mbSRAF runs. The SVRF statement must be either “RET EUV PXOPC”, “RET EUV REFINE”, or “RET EUV MBSRAF”.

LTH23

The particular combination of operations in the rule file requires it be run in either ULTRAflex or TURBOflex mode. See “Which Construction Method to Use” in the Calibre Post-Tapeout Flow User’s Manual for information on ULTRAflex.

LTH24

The MAP argument is required for DENSEOPC operations. Without MAP, the results are not returned to the main Calibre run.

LTH25

A litho operation other than DENSEOPC includes the ML keyword.
