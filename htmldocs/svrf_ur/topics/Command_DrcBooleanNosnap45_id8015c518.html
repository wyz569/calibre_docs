DRC Boolean Nosnap45

Specification statement

Specifies how merged shapes with 45-degree edges snap to the grid. Used only in Calibre nmDRC-H.

Usage

DRC BOOLEAN NOSNAP45 {NO | YES}

Arguments

NO

Keyword that specifies an offgrid intersection point of 45‑degree edges is snapped to the grid, resulting in a merged shape with edges that are slightly off 45‑degrees. This is the default behavior when you do not include this statement in the rule file.

YES

Keyword that specifies to preserve 45‑degree edges in a merged shape when the edges intersect at an offgrid point. A 1 dbu edge is inserted at the intersection of the 45-degree edges instead of snapping the intersection point to grid.

Description

Specifies how merged shapes with 45‑degree edges that intersect at an offgrid point are snapped to the grid. Calibre generally only operates on merged data and merging is performed automatically. In hierarchical applications, merging is per-cell, not necessarily across the hierarchy.
