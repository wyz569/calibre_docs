DFM Expressions

DFM Expressions are used to calculate DFM properties, define an analysis or density function, or specify a user-defined function. They can be used to calculate scores, statistics, and measurement values based on layer data. DFM Expressions are used in DFM Analyze, the DFM Property operation family, DFM Function, and DFM Spec Fill Optimizer.

Operation or Statement

Usage

DFM Analyze

Supports one expression per operation, including a constraint. The expression defines the analysis to perform and must return a numeric value.

DFM Property and related operations

Supports multiple property definitions per operation, including an optional constraint for each property definition. Expression chains are supported. Constraints can be negated using “!”.

DFM Spec Fill Optimizer

Supports one optimization expression per operation, which typically calculates a density. Constraints may also be written as expressions; see the operation for details. The expression must return a numeric value.

DFM Function

Supports one expression, which defines the user-defined function.

DFM Expressions are formed with the following elements:

  • Math Operators (+, -, !, ~, ^, *, /, +, -)

  • Numeric values

  • Strings, when used in the DFM Property family of operations

  • Vectors, when used in the DFM Property family of operations

  • Variables defined with the Variable statement

  • Conditional Expressions when used in the DFM Property family of operations, DFM Analyze, and DFM Function

  • Expression Chains, when used in the DFM Property family of operations

  • Supported functions:

    • Supported Math Functions in DFM Expressions, such as sine (SIN), cosine (COS), square root (SQRT), and other math functions.

    • Measurement Functions and other built-in DFM functions defined in the Reference for DFM Functions. The validity of a function is determined by the operation it is used in.

    • User-defined functions created with DFM Function.

DFM Expressions are enclosed in brackets ( [ ] ). In DFM Property and its related operations, the property name and assignment operator (=) are included within the brackets. In DFM Function the function name is included within the brackets. The following topics provide detailed information on writing DFM expressions.
