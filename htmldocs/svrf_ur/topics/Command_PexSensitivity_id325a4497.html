PEX Sensitivity

Parasitic extraction

Specifies settings for netlist generation with process variation sensitivity information.

Usage

PEX SENSITIVITY { ON | { [REFERENCE center] [TEMPERATURE]
VARIATION layer [TYPE parameter [NAME name]
      [{PERCENT value} | {DELTA diff}] [MASK id]]
[VARIATION layer [TYPE parameter [NAME name]
      [{PERCENT value} | {DELTA diff}] [MASK id]]]…}
}

Arguments

ON

A required keyword that specifies to run sensitivity extraction for all default variations.

REFERENCE center

An optional keyword set that specifies the foundry-supplied process corner to use as the center for the sensitivity calculations. The default value for center is the typical process corner.

TEMPERATURE

An optional keyword that specifies sensitivity extraction using temperature effects. This keyword should only be specified once.

VARIATION layer

A required keyword set that specifies layers used for sensitivity variations. More than one VARIATION layer set may be specified. May not be used with ON.

TYPE parameter

An optional keyword set that specifies the type of variation used during sensitivity extraction. One or more of the following parameter choices must be specified:

  • DIELECTRIC_CONSTANT

  • ILD — inter-layer dielectric thickness variation

  • MASK_OFFSET_X

  • MASK_OFFSET_Y

  • THICKNESS

  • WIDTH

If not defined, the default types THICKNESS, WIDTH, and ILD are applied to the listed layers.

NAME name

An optional keyword set that specifies the name of the process variation parameter. If not defined, a default name is derived from the layer name and type.

PERCENT value

An optional keyword set that specifies the coefficient of variation for the variation parameter. If neither PERCENT or DELTA keywords are specified, a default value of 0.1 (10%) is used. May not be used with the DELTA keyword.

DELTA diff

An optional keyword set that specifies the absolute adjustment to be applied to the variation parameter when computing the sensitivity coefficients. If not defined, a name is derived from the layer name and type. May not be used with the PERCENT keyword or the ILD variation type.

MASK id

An optional keyword set that specifies the mask id. May only be used with the MASK_OFFSET_X or MASK_OFFSET_Y variation types. If not defined, the default for id is 1.

Description

Specifies settings for generating a netlist with process variation sensitivity information using one of the process corners as a reference. Process variations include metal thickness, line width, and inter-layer dielectric (ILD) thickness. A sensitivity netlist is generated when the -corner option is not supplied during formatting.

PEX Sensitivity supports generating sensitivity SPEF (S-SPEF) and ELDO netlists with sensitivity coefficients for process variation parameters. The value of a parasitic device corresponds to a set of process variation parameter values.

When the TEMPERATURE keyword is used, temperature sensitivity coefficients are calculated during the formatting step (-xrc -fmt) from TC1 and TC2 which are specified in the PEX Extract Temperature statement.

Examples

Example 1

This example uses the typical corner as reference and defines variations in metal 1 to metal 3:

PEX SENSITIVITY VARIATION metal1 VARIATION metal2 VARIATION metal3

Example 2

This example runs sensitivity extraction for all default variations:

PEX SENSITIVITY ON

Example 3

The following statement specifies four process variation parameters on two layers centered on the fastRC process corner:

PEX SENSITIVITY REFERENCE fastRC
   VARIATION m1 NAME m1thick TYPE THICKNESS DELTA 0.02
   VARIATION m1 NAME m1width TYPE WIDTH PERCENT 0.002
   VARIATION m2 NAME m2thick TYPE THICKNESS DELTA 0.05
   VARIATION m2 NAME m2width TYPE WIDTH PERCENT 0.003

Example 4

The following statement specifies a mask offset sensitivity setup:

PEX SENSITIVITY REFERENCE typical
   VARIATION m1 
      TYPE MASK_OFFSET_X // PERCENT 0.1 is assumed by default
      TYPE MASK_OFFSET_Y DELTA 0.002
      TYPE MASK_OFFSET_X MASK 2 PERCENT 0.05
      TYPE MASK_OFFSET_Y MASK 2 DELTA 0.004
