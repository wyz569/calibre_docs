LVS Netlist Unnamed Isolated Nets

Specification statement

Controls preservation of unnamed, isolated, layout nets in the extracted layout SPICE netlist. Used only in Calibre nmLVS-H and Calibre PERC.

Usage

LVS NETLIST UNNAMED ISOLATED NETS {NO | YES}

Arguments

NO

Keyword that does not preserve unnamed, isolated nets in the extracted layout netlist. This is the default behavior if you do not include this statement in your rule file.

YES

Keyword that preserves unnamed, isolated nets in the extracted layout netlist.

Description

Controls preservation of unnamed, isolated, layout nets in the extracted layout SPICE netlist.

If NO is specified, or the statement is not used, only named, isolated, layout nets are written to the netlist. If YES is specified, unnamed isolated nets are also written to the netlist. Isolated nets are identified in the netlist by this control statement:

*.CALIBRE ISOLATED NETS: <netID> ...

The YES option can cause a large increase in the size of the extracted netlist, especially for designs containing fill shapes.

See “Isolated Nets” in the Calibre Verification User’s Manual for more information.

Also see LVS Preserve Floating Top Nets.

Examples

// Do not preserve unnamed isolated nets
LVS NETLIST UNNAMED ISOLATED NETS NO
