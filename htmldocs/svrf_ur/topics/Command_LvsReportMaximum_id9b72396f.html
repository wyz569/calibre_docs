LVS Report Maximum

Specification statement

Specifies an upper limit on the number of items that LVS prints out in various sections of its report. Also used by Calibre PERC.

Usage

LVS REPORT MAXIMUM {number | ALL}

Arguments

number

Positive integer that specifies an upper limit on the number of items printed in the LVS Report. The default is 50 if you do not include this statement in the rule file.

ALL

Keyword that specifies no upper limit on the number of items printed in the LVS report.

Description

Specifies an upper limit on the number of items that appear in various sections of the LVS report. This limit affects the maximum number of discrepancies listed, the maximum number of items listed for each discrepancy, and the maximum number of items in various other lists printed by LVS, such as initial correspondence points and ambiguous devices or nets that are resolved arbitrarily. If the number of items for a given list is larger than the specified limit, the list is terminated.

LVS usually lists the most informative (and most drastic) discrepancies first. Fixing the discrepancies that appear early in the list often clears later discrepancies. Therefore, using the default setting of 50, or a lower number, is most beneficial because such settings make the report easier to read.

Setting number to a high value or using ALL is rarely useful. However, if you want to see a complete listing of things like initial correspondence points or ambiguous nets and devices, then using settings such as these can be useful. If your LVS run is incorrect, having a high value can make the report very long and difficult to use.

This statement can be specified only once in your rule file.

You can specify the LVS Report maximum discrepancy count in Calibre Interactive—nmLVS. See “Specifying Calibre nmLVS Report Options” in the Calibre Interactive User’s Manual.

In Calibre PERC, this statement is overridden by the PERC Report Maximum setting.

In Pyxis Layout, this statement sets the value of application variable lvs_report_list_limit.

Examples

// Report a maximum of 25 discrepancies.
LVS REPORT MAXIMUM 25
