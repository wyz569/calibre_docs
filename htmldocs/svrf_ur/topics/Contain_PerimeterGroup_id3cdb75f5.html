PERIMETER Group

Returns a sum of the edge lengths for each polygon on the input layer. Applicable to original or derived polygon layers.

PERIMETER

Returns a sum of edge lengths for each polygon.


PERIMETERX, PERIMETERY

Returns a sum of edge lengths parallel to the desired axis for each polygon.


PERIMETERXP, PERIMETERYP

Returns a sum of edge projections onto the desired axis for each polygon.


See “Measurement Functions” for information on using the DFM measurement functions, including cell context dependencies and per-shape evaluation.
