Layer Functions in Runtime TVF

Several layer functions are provided for runtime TVF operation. The functions are summarized here. They perform layer derivation, layer output, get layer extent, get the layer name, and other functions.

The following table summarizes the layer functions that can appear in a runtime TVF function block. See the description in the “TVF Command Reference” for a complete description.

Table 1. Runtime TVF Layer Functions

Function

Description

tvf::DELETE_LAYER $layer

Deletes the specified layer.

tvf::GET_LAYER_ARGS layer1 [layerN …] [args]

Imports the layers passed to a TVF Function by a TVF layer operation, and sets up the layer handles. This command must be used once in a TVF Function statement that is called by a TVF layer operation.

tvf::GET_LAYER_EXTENT $layer

Returns the bounding box of the extent of the layer as a list {x1 y1 x2 y2}.

tvf::GET_LAYER_NAME $layer

Returns the layer name for the layer handle $layer. See “Runtime TVF Example” for usage.

tvf::IS_LAYER_EMPTY $layer

Used to test for an empty layer. Returns true (1) if $layer is empty.

tvf::OUTLAYER arguments

Executes the arguments (which form a layer operation) and returns the layer to the TVF layer operation that called the TVF Function. This command must be used once and only once in the TVF Function called by a TVF layer operation, and the output layer type must match the output type specified for the TVF Function.

See “Layer Operations in Runtime TVF” for restrictions on the allowed layer operations.

tvf::SETLAYER layer_name = arguments

Executes the arguments (which form a layer operation) and stores the output in the layer_name handle. Used for layer derivation.

See “Layer Operations in Runtime TVF” for restrictions on the allowed layer operations.

tvf::svrf_var variable_name

Gets the value of a variable defined with the Variable statement in the SVRF rule file.

tvf::sys_var variable

Accesses the value of a system variable in runtime TVF.

A TVF Function that is called by a TVF layer operation statement must have exactly one tvf::GET_LAYER_ARGS function and one tvf::OUTLAYER function. The other functions in Table 1 can be used as needed. The tvf::SETLAYER command is frequently used to derive intermediate layers.
