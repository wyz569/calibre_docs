PEX Extract Remove PDB

Parasitic extraction

Specifies whether or not to remove an existing PDB before creating the PDB in the PDB generation step. Used only in Calibre xRC and Calibre xACT 3D PDB flow.

Usage

PEX EXTRACT REMOVE PDB {NO | YES}

Arguments

NO

Keyword that instructs the tool not to remove an existing PDB if one exists in the PDB generation step. This is the default behavior when the statement is not included.

YES

Keyword that instructs the tool to remove an existing PDB if one exists before generating the PDB in the PDB generation step.

Description

An optional statement used to remove an existing PDB if one exists before generating the PDB in the PDB generation step. Do not use this statement if you want to incrementally add to the PDB rather than regenerate the PDB every time the PDB generation step is run.

Examples

To create a new PDB each time the PDB generation step is run, include the following statement in your rule file:

PEX EXTRACT REMOVE PDB YES
