SVRF FDI Options

Specification statement

Specifies FDI options for controlling the read-in of third-party databases by Calibre tools that support them.

Usage

SVRF FDI OPTIONS “fdi_option  [fdi_option ]…”

Arguments

“fdi_option”

A quoted string that includes one or more FDI argument sets.

Description

This specification statement is equivalent to setting the MGC_CALIBRE_DB_READ_OPTIONS environment variable. The environment variable settings take precedence over options defined using this statement. A list of options for batch use is shown under Layout System.

The option list must be quoted and the delimiter between multiple options is the space character.

The following options may not be used:

  • -design

  • -def

  • -lef

  • -outFile

  • -system

CAUTION:

The options in SVRF FDI Options are not used in a run started from Calibre Interactive. When using Calibre Interactive, the FDI options must be set in the GUI or with MGC_CALIBRE_DB_READ_OPTIONS.

Examples

SVRF FDI OPTIONS "-annotateNets PROPERTY 10"
