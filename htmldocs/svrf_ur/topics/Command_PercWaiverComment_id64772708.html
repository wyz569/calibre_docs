PERC Waiver Comment

Specification statement

Specifies that a comment string, along with PERC Load statement information, is written into the header section of a waiver description file generated by Calibre RVE for PERC.

Usage

PERC WAIVER COMMENT “comment”

Arguments

“comment”

A required, quoted, character string that is used as an administrative comment. Variable resolution is supported if a character sequence within the string constant is preceded by the “^” character.

Description

Specifies an administrative comment appears in the header section of a waiver description file generated by Calibre RVE for PERC. This statement may be specified any number of times and each comment is applied to waivers.

When this statement is present in the rule file, the waiver description file generated by Calibre RVE contains this extra information in the header section:

// PERC LOAD STATEMENTS:
      //    PERC LOAD <args1 ...>
      //    PERC LOAD <args2 ...>
      //    ...      
// PERC WAIVER COMMENTS:
      //    PERC WAIVER COMMENT <message1>
      //    PERC WAIVER COMMENT <message2>
      //    ...

See “Topology Waiver Description File Format” in the Calibre PERC User’s Manual for more information about waiver description files.

Variable Resolution

Any non-empty sequence of characters within the string constant preceded by the caret (^) character and terminated by a space, caret (^), backslash (\), or the end of the string is interpreted as the name parameter in a Variable statement. The variable is resolved and its value is substituted in the message for its variable name.

If a variable is defined as an array of values, such as this:

VARIABLE "scale" "um" "x1000 dbu"

only the first value of the array is accessed by ^.

The caret (^) character may be escaped (treated as literal) by preceding it with two backslashes (\\).

The following conditions report a compiler error:

  • The variable is not resolvable and reports compiler error RES8.

  • The variable is not resolvable as a numeric or string type and reports compiler error RES7.

  • The variable is not resolvable as a string array and reports compiler error RES8.

Examples

// write administrative comment into waiver description file
PERC WAIVER COMMENT "Foundry supported waivers for 45nm process"
