Calibre nmDRC-H Warnings

This section describes runtime messages from the Calibre nmDRC-H application.
Table 1. Calibre nmDRC-H Warnings

Explanation

These warnings are reported by Calibre nmDRC-H:

WARNING: Creating a FLAT COPY of layer <layer>.

Reported whenever a temporary flat copy of a layer is created in Calibre nmDRC-H.

WARNING: Layer <layer> CONVERTED to an exclusive FLAT instantiation.

Reported whenever a layer is converted from an exclusive hierarchical instantiation or dual instantiation to an exclusive flat instantiation in Calibre nmDRC-H.

WARNING: Layer <layer> CONVERTED to a dual FLAT/hierarchical instantiation.

Reported whenever a layer is converted from an exclusive hierarchical or flat instantiation to a dual instantiation in Calibre nmDRC-H.

WARNING: Layer <layer> CONVERTED to an exclusive HIERARCHICAL instantiation.

Reported whenever a layer is converted from a dual instantiation or exclusive flat instantiation to an exclusive hierarchical instantiation in Calibre nmDRC‑H.

WARNING: Layer <layer> CREATED with an exclusive FLAT instantiation.

Reported whenever a layer is created with an exclusive flat instantiation in Calibre nmDRC-H.

WARNING: Layer <layer> CREATED with a dual FLAT/hierarchical instantiation.

Reported whenever a layer is created with a dual instantiation in Calibre nmDRC-H.

WARNING: SPACE filter not supported - both SPACE and NOTCH checked.

A derived edge layer with an exclusive flat instantiation is input to a one-layer External operation (with one input layer) with a SPACE filter specified in Calibre nmDRC-H. Or a layer with an exclusive flat instantiation is input to a one-layer External operation (with two input layers) with a SPACE filter specified in Calibre nmDRC-H. Both NOTCH and SPACE are checked in that event.

WARNING: NOTCH filter not supported - both SPACE and NOTCH checked.

A derived edge layer with an exclusive flat instantiation is input to a one-layer External operation (with one input layer) with a NOTCH filter specified in Calibre nmDRC-H. Or a layer with an exclusive flat instantiation is input to a one-layer External operation (with two input layers) with a NOTCH filter specified in Calibre nmDRC-H. Both NOTCH and SPACE are checked in that event.

WARNING: One-layer INTERNAL check performed without regard to polygon numbers.

A derived edge layer with an exclusive flat instantiation is input to a one-layer Internal operation (with one input layer) in Calibre nmDRC-H. Or a layer with an exclusive flat instantiation is input to a one-layer Internal operation (with two input layers) in Calibre nmDRC-H. The polygon numbers are not available on the input layer, so checking is performed without regard to polygon numbers in that event. When this occurs, there is a potential increase in runtime.

WARNING: PROJECTING constraint only supported with PARALLEL ONLY - setting PARALLEL ONLY.

A dimensional check operation has a constrained PROJECTING filter in Calibre nmDRC-H but the PARALLEL ONLY filter is not also specified. The operation is executed as if PARALLEL ONLY was specified.

WARNING: Output operation <name> abbreviated due to high probability of exceeding maximum result limit.

An External, Enclosure, or Internal output-only operation has been prematurely completed due to the generation of such a large number of DRC results that there is a high probability of exceeding the maximum result limit for this operation. The name of the operation appears in the message.

WARNING: Hierarchical OFFGRID checking with unequal X and Y resolutions checks rotated/reflected placements flat.

Offgrid checking for the Flag Offgrid specification statement or the Drawn Offgrid or Offgrid operation is processing a resolution with unequal x- and y-values. In this case, rotated and/or reflected placements will be checked flat, which may be slow.

WARNING: OFFGRID placement of <pcell> in <cell> at <point> will require FLAT offgrid/SNAP processing for all placements of this cell.

An offgrid placement is encountered during offgrid checking for the Flag Offgrid specification statement or the Drawn Offgrid or Offgrid operation, or during snapping for the Snap Offgrid specification statement or the snap operation. This will cause all placements of this cell to be processed flat, which may be slow. The name <pcell> of the placement, its containing cell name <cell>, and the offgrid basepoint <point> (in <cell> coordinate space) appears in the message.

WARNING: HCELLs in a hierarchical DRC run may have a significant negative performance impact.

Reported whenever Hcells are encountered in the layout during the DRC run.
