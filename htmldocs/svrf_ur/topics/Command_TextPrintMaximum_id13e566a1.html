Text Print Maximum

Specification statement

Controls the number of text objects and ports printed in the transcript of a Calibre run. Used only by Calibre.

Usage

TEXT PRINT MAXIMUM {ALL | number}

Arguments

ALL

A keyword that specifies that all text objects and ports are printed to the transcript.

number

A positive integer that specifies the specific number of text objects and ports printed to the transcript. The default is 1000 if you do not specify this statement in your rule file.

Description

Controls the maximum number of text and port objects printed in the transcript of a Calibre run. This statement may be specified once in your rule file.

This statement outputs the connectivity extraction text (Text Layer or Text), [Not] With Text, Expand Text, Device TEXT MODEL LAYER and TEXT PROPERTY LAYER, DFM Text, and Port Layer Text objects to the transcript. Recognized text objects are reported in the CALIBRE LAYOUT DATA INPUT MODULE section of the transcript.

Because the number of text objects can be excessive in certain cases, a default value of 1000 is used to limit the number of text objects printed in each text reporting section. A warning is issued whenever the number of objects printed is less than the total number which could have been printed.

Examples

TEXT PRINT MAXIMUM 1000
