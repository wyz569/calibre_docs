input_numeric_value

LVS Property Initialize computation language function.

Returns property values in the input database.

Usage

INPut_NUMeric_VALue(property_name)

Arguments

property_name

A required numeric property name declared in the PROPERTY statement.

Return Values

Number.

Description

This function allows user programs to access the values of properties present in the input database. The return value of this function is the related property value present for the current instance in the input database.

If no such property value exists in the input database for the current instance, this function returns the “missing” value instead. The “missing” value is defined internally within the language and may not be accessed directly by user programs; however, presence of the “missing” value may be detected by use of the is_missing function.
