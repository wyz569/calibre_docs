device::dfm_vec_arr_measurements

Device DFM Property data retrieval function.

Converts DFM-style rectilinear measurement array vectors into Device enclosure-vector-style width and length measurements in database units.

Usage

device::dfm_vec_arr_measurements object polygon_index

Arguments

object

A required dfm_vector_array object.

polygon_index

A required positive integer that represents the index of the polygon from which to access measurement values.

Return Values

Vector of numbers.

Description

The device::dfm_vec_arr_measurements function is used in conjunction with the TVF Functional Interface for Device Property Calculation. It must be used in a TVF Function block. The device::scaled_dfm_vec_measurements description contains a complete explanation of device::dfm_vec_arr_measurements behavior and related functions.

The enclosure_* functions of the DEVice statement’s built-in language return values in user units, while DFM Property measurement functions return them in database units. In the latter case, device::scaled_dfm_vec_arr_measurements can be more convenient because it can scale the measurements to user units.

The device::dfm_vec_arr_measurements function is used when there may be more than one polygon from which measurement values need to be retrieved and that interacts with a seed shape.

Examples

See “Example 1” under dfm_vector_array. It can be modified not to require scaled measurements (which it currently uses) by removing the dependency on UNIT_LENGTH().
