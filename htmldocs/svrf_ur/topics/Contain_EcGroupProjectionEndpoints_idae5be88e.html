EC Group: Projection Endpoints

Returns endpoint coordinates of the EC segment, where EC is the projection of one edge upon another. The functions apply to error layers (edge clusters) with one or two edges per cluster and edge layers. The functions are only supported in DFM Property operations.

When used with a secondary layer, these functions must be used within a per-shape function (MIN, MAX, SUM, PRODUCT, or VECTOR).

All of the endpoint measurement functions are based on the mutual edge projection EC. They return an endpoint coordinate of the EC segment as measured along the axis parallel to EC.

Function

Summary

ECMIN

Returns the smaller value of the two endpoint coordinates in the context of the local cell space.

ECMAX

Returns the larger value of the two endpoint coordinates in the context of the local cell space.

ECBEGIN

Returns the coordinate of the “beginning” endpoint of the EC segment, where the beginning endpoint is identified in the originating cell space and is the endpoint with the lesser coordinate value in the originating cell space (ECBEGIN = ECMIN in originating cell space). The returned value is measured the local cell space, which may be different than the originating cell space if promotion occurs.

ECEND

Returns the coordinate of the “ending” endpoint of the EC segment, where the ending endpoint is identified in the originating cell space and is the endpoint with the greater coordinate value in the originating cell space (ECEND = ECMAX in originating cell space). The returned value is measured the local cell space, which may be different than the originating cell space if promotion occurs.

The statement ECMAX >= ECMIN is always true.

The endpoint functions are intended for use with Manhattan edges or with edge pairs with at least one edge parallel to an axis. The behavior with angled edges is described at the end of this topic, but may be considered non-intuitive.

See NORMALIZE_XY for an example using ECBEGIN and ECEND. Refer to “Finding Polygon Extents for Special Configurations” in Calibre Solutions for Physical Verification for an example of ECMAX and ECMIN usage.

Coordinate Space Considerations. The value returned by the endpoint functions is given in the local coordinate space. The local coordinate system may not be the same as the cell the geometries originated in if promotion occurs, causing the return value to depend on the cell context.

As an example, consider the following figure, where cell B includes a flipped placement of cell A. When the calculation occurs in the cell A coordinate system, ECBEGIN = ECMIN and ECEND = ECMAX. If promotion occurs so that the calculation occurs in cell B, ECBEGIN = ECMAX and ECEND = ECMIN. This is because ECMIN and ECMAX are defined to be the minimum and maximum value coordinates in the local cell space, respectively, while ECBEGIN and ECEND are associated with a particular endpoint determined in the originating cell.

Operation with Two Angled Edges. As stated previously, the endpoint calculation functions are intended for use when at least one edge of a pair is parallel to an axis. However, the functions still return a value if both edges are angled and non-parallel—the value is calculated as follows:

1. Determine the EC segment as described in “EC”.

2. Rotate the EC segment about the origin by the minimum angle needed to make it parallel to an axis.

3. Calculate the coordinate point by measuring along the axis the EC segment is parallel to.



Note that depending on the edge chosen by the operation for the EC segment, EC and its endpoint coordinate values can vary significantly for two angled non-parallel edges. If the edges are angled but parallel to each other, the EC length does not vary by edge chosen; only the endpoint coordinate value does.

Operation with One Angled Edge For single angled edges, a perpendicular line is drawn from the edge or its continuation to the coordinate origin (0,0). The point where the perpendicular intersects the edge (or its continuation) is d=0. The coordinate d is measured along the edge; ECMIN is the lesser of the coordinates d corresponding to the end points of the edge; ECMAX is the greater.
