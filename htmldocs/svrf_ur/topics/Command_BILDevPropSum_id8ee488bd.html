sum

Device property computation language function.

Returns the sum of vector elements.

Usage

SUM(vector_expression)

Arguments

vector_expression

A required arithmetic expression that contains one or more vector selection expressions, each returning a simple array of numbers. In addition, vector_expression may contain numeric constants, arithmetic operators, numeric functions, and so forth.

Return Values

Number.

Description

Applies the specified arithmetic expression to each element of the operand array or arrays and returns the sum of the results. This function can be used to sum elements of vectors that are output by the enclosure_parallel, enclosure_perpendicular, and enclosure_vector functions.

For example, if S is the result of an enclosure function call, then the expression

SUM(S::w)

returns the sum of all individual “w” values from S. Similarly, the expression

SUM(S::a - 3 * S::b + 7)

returns the sum of all (ai - 3 * bi + 7) values from S, where i is a running index over S.

Restrictions

Several restrictions apply to the vector_expression:

  • The vector_expression must contain at least one vector selection expression.

  • If vector_expression contains multiple vector selection expressions, they must all be from the same vector-typed variable.

  • The vector_expression must not contain another SUM function call (although it may contain a property or local variable that is generated by another SUM function).

Examples

Example 1

device mp seed poly psd psd nwell <active> [
    property w
    s = enclosure_perpendicular( seed, active, psd, 3 )
    w = sum( s::w )
]

The property w contains the total width of the seed shape across the active layer.

Example 2

device mp seed poly psd psd nwell <active> [
    property w, len, saeff, sbeff
    w = perim_co( psd, seed ) / 2
    len = area( seed ) / w
    s = enclosure_perpendicular( seed, active, psd, 3 )
    saeff = w / sum( s::w / ( s::a + 0.5 * len ) ) - 0.5 * len
    sbeff = w / sum( s::w / ( s::b + 0.5 * len ) ) - 0.5 * len
]

The property saeff is calculated by summing the results of the expression for each corresponding value of w and a in the results vector s:

     s::w / (s::a + 0.5 * len)

then performing the remaining calculation:

     saeff = w / <result_of_sum> - 0.5 * len

Property sbeff is computed in a similar way.
