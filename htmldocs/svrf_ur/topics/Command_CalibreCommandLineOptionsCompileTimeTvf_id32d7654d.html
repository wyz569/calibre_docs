Calibre Command Line Options for Compile-Time TVF

The Calibre command line has several options for use with compile-time TVF rule files.

Usage

calibre {calibre_options | -E svrf_output_from_tvf |
calibre_options -E svrf_output_from_tvf}
[ -tvfarg tvf_argument ] [-tvf_instrument]Â  tvf_rule_file

Arguments

calibre_options

The Calibre command line options for the run. This parameter is required in order to run Calibre.

-E svrf_output_from_tvf

Parameter set that causes the Tcl preprocessor to save the generated SVRF code to a file named svrf_output_from_tvf.

If this option is specified without the calibre_options parameter, then Calibre runs in preprocessor only mode to generate the SVRF code and write it to file. A Calibre run is not started.

-tvfarg tvf_argument

An optional parameter set used to pass in a user-defined argument to the TVF script, where tvf_argument is a valid string with no spaces. The value of tvf_argument is accessed with the tvf::get_tvf_arg command.

-tvf_instrument

An optional parameter that causes Calibre to run with extra error reporting.

tvf_rule_file

A required parameter specifying the path to the compile-time TVF rule file.

Description

When Calibre executes, it interprets the TVF rule file and generates an internal SVRF output. The SVRF output is then fed to the SVRF compiler, one line at a time. Upon successful compilation of the SVRF output, the Calibre job is run.

Transcript Output

The TVF rule file is written to the transcript, followed by the Calibre execution information. The translated SVRF code is not written to the transcript by default. You can search for the opening line (#!tvf) to find the start of the TVF rule file.

Examples

Example 1

This example starts a standard Calibre nmDRC run, where the file tvf_drc_rules is a compile-time TVF script. 

calibre -drc -hier tvf_drc_rules

Example 2

This example starts a standard Calibre nmDRC run and saves the generated SVRF code to the file svrf_from_tvf_drc_rules. 

calibre -drc -hier -E svrf_from_tvf_drc_rules tvf_drc_rules

Example 3

This example saves the generated SVRF code to the file svrf_from_tvf_drc_rules. Calibre is not run.

calibre -E svrf_from_tvf_drc_rules tvf_drc_rules
