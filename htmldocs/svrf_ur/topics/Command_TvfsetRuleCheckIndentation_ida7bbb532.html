tvf::set_rule_check_indentation

Application: compile-time, pre-processor mode

Controls indentation of each rule check printed in the output SVRF file (the default is a tab). The indentation parameter is a string that is prepended to each line. The effects of this command are only seen in the SVRF file saved with the -E command line option.

Usage

tvf::set_rule_check_indentation indentation

Arguments

indentation

A required argument that specifies the indentation string. If the indentation is an empty string (“”), rule check lines are not indented.

Return Values

None.
