PERC Report

Specification statement

Specifies the pathname of the Calibre PERC report file.

Usage

PERC REPORT filename

Arguments

filename

A required filename.

The filename parameter can contain environment variables. See “Environment Variables in Pathname Parameters”.

Description

Specifies the Calibre PERC report filename. This statement must be specified once in the rule file when using Calibre PERC for rule checking.

See “Calibre PERC Reports” in the Calibre PERC User’s Manual for details on the PERC Report.

For information about viewing Calibre PERC results in Calibre RVE, see “Using Calibre RVE for PERC” in the Calibre RVE User’s Manual.

Case-sensitive matching and reporting is controlled by the Source Case, Layout Case, and LVS Compare Case statements. If you want names to be treated as case-sensitive, all three statements must be enabled.

See also PERC Load, PERC Report Maximum, PERC Report Option, and PERC Report Placement List Maximum, and PERC Summary Report.

Examples

In this example, perc.rep contains all the results from rule.1 and rule.2.

PERC REPORT "perc.rep"
PERC LOAD perc.func INIT init_1 SELECT rule.1 rule.2
