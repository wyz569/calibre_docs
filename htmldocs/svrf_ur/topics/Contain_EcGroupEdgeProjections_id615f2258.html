EC Group: Edge Projections

The EC functions return the length of one error edge projected onto another (or, the Edge Concurrency). Function variants return the x- and y-axis projection, or constrain one edge to be horizontal or vertical in order for a measurement to be taken.

EC

Projection is performed for error edge clusters of one or two edges as follows:

  • If one edge is horizontal or vertical but the other is not, the projection is done onto that edge.

  • For parallel edges, EC is the projection of either edge onto the other.

  • For non-Manhattan, non-parallel edges, the projection is done onto the edge closest to horizontal or vertical (see figure at right).

  • For single-edge input, EC returns the edge length.


ECX, ECY

Returns EC, but at least one edge must be parallel to the specified axis or the function returns 0.


ECXP, ECYP

Returns the length of the projection of EC onto the specified axis. EC is found as described previously.
