Unit Specification Statements

This section lists the Unit specification statements. These specify the preferred units used in interpreting appropriate numeric values for capacitance, length, resistance, and time.

The statements in Table 1 apply only to Calibre nmLVS and PEX applications.

Table 1. Specification Statements: Unit

Statement

Description

Unit Capacitance

Provides the capacitance scale factor for device calculations.

Unit Inductance

Provides the inductance scale factor for device calculations.

Unit Length

Provides the length scale factor for device calculations. This statement has no effect on the rule file precision or upon layer operation measurement constraints.

Unit Resistance

Provides the resistance scale factor for device calculations.
