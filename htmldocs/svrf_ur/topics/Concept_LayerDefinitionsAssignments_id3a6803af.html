Layer Definitions and Assignments

The tvf::SETLAYER command is used to assign derived layers to a new layer name.

In SVRF, layer definitions assign names to derived layers using this syntax:

layer_name = layer_operation

For example:

gate = poly AND diff

Layers defined in this way have user-specified names, so this syntax cannot be duplicated in TVF as a command. Instead, this is handled in TVF with the tvf::SETLAYER command:

    tvf::SETLAYER layer_name = arguments

where arguments contains the layer operation.

For example:

tvf::SETLAYER gate = poly AND diff

or 

tvf::SETLAYER lout = ((L1 AND L2) OR "($L3 AND $L4)") AND {(L5 OR L6)}

Quotation marks (“ ”) enable for variable substitution in a list of arguments, whereas braces ({ }) pass their contents as a list with no variable substitution.

The layer operations specified to SETLAYER use the same syntax as corresponding SVRF layer operations. Layer operations are passed as arguments to the SETLAYER command, and are then passed to the SVRF compiler. The return value from SETLAYER command is the name of the layer.

Query for Derived Layers and Layer Expressions

The tvf::SETLAYER command keeps track of layers defined globally as well as locally within a tvf::RULECHECK command. The ‑getnames, ‑local, ‑global, and ‑getexpr switches to the tvf::SETLAYER command query for layer names and layer expressions.

See the tvf::SETLAYER command for details and examples.

Example: Layer Derivation with SETLAYER in Compile-Time TVF

This example shows how SVRF syntax can be translated to TVF for layer derivation.

SVRF: 

VARIABLE grid .23 
CR0 = OPC OR NOOPC 
CR18 = SIZE CR0 BY grid 
//ADDED BIAS 
CLTEST = EXTERNAL CR18 CR0 < .70 OPPOSITE EXTENDED 1.1 REGION CENTERLINE
CX1 = SIZE CR18 BY 2*grid 
CC1 = CX1 NOT CLTEST

TVF:

set grid .23 
tvf::SETLAYER CR0 = "OPC OR NOOPC" 
tvf::SETLAYER CR18 = "SIZE CR0 BY $grid" 
tvf::// ADDED BIAS 
tvf::SETLAYER CLTEST = "EXTERNAL CR0 CR18 < .70 OPPOSITE EXTENDED 1.1 \
		REGION CENTERLINE" 
#use of line continuation character
# double quotes enable variable substitution
tvf::SETLAYER CX1 = "SIZE CR18 BY [expr {2 * $grid}]" 
tvf::SETLAYER CC1 = "CX1 NOT CLTEST"
