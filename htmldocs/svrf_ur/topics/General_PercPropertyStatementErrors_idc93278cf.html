PERC Property Statement Errors

This section describes compilation errors from the PERC Property statement.
Table 1. PERC Property Statement Errors

Error #

Explanation

LPP1

Missing or invalid device element name.

LPP2

Improperly specified device model name.

LPP3

Missing or invalid property name.

LPP4

A PERC Property statement is specified more than once.

LPP5

A PERC Property statement specifies an undefined DEVice element.

LPP6

A PERC Property statement specifies a property that is not computed for all corresponding DEVice statements.

LPP7

A property has inconsistent type (numeric or string) in all corresponding Device statements.

LPP8

TOLERANCE cannot be specified for a STRING property.

LPP9

TOLERANCE specification is missing a property name.

LPP10

TOLERANCE specification is missing a value.

LPP11

TOLERANCE value cannot be negative.

LPP12

TOLERANCE specification missing ].

LPP13

TOLERANCE specification has bad syntax.

LPP14

Duplicate property name.

LPP15

Something follows a TOLERANCE specification.
