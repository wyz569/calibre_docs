LVS Spice Exclude Cell

Specification statement

Specifies to ignore calls to a subcircuit in a SPICE netlist. Used by Calibre nmLVS, Calibre nmLVS-H, Calibre 3DSTACK, Calibre PERC, Calibre xACT, Calibre xRC, and Calibre xL.

Usage

LVS SPICE EXCLUDE CELL[S] [LAYOUT | SOURCE | LAYOUT SOURCE] cell_name [cell_name …]

Arguments

cell_name

A required name of a SPICE subcircuit. More than one name may be specified. This argument must appear last in the statement.

A cell_name parameter may contain one or more asterisk (*) characters; the * character is a wildcard that matches zero or more characters. A string that contains a * character must be enclosed in quotation marks because the * is a reserved symbol.

LAYOUT

An optional keyword that specifies the statement applies to the layout netlist.

SOURCE

An optional keyword that specifies the statement applies to the source netlist.

Note:

The combination LAYOUT SOURCE is used by default if neither keyword is specified.

Description

This statement affects Calibre applications that read SPICE netlists. Any call to a subcircuit having a name that matches any cell_name, either explicitly or by wildcard pattern, is ignored for the netlist being parsed (SOURCE, LAYOUT, or both). Any subcircuit definition matching a cell_name specification is still parsed.

If a string or string array variable reference is given for a cell_name, the variable is expanded, and then each of its strings is treated as a literal name or wildcard pattern.

This statement may be specified more than once, and the conditions specified by all occurrences of this statement are aggregated for the run.

Case sensitivity applied to name matching is controlled by Layout Case and Source Case specification statements for the layout and source designs, respectively.

Hierarchical *.CONNECT (or *.J) statements that reference nets extending through cell instances ignored due to this statement are ignored for such instances, and a warning like this is issued:

Warning: Hierarchical connect involving call to excluded subcircuit "<cell>" ignored in file "<name>" at line <N>.

See also LVS Box, LVS Filter, LVS Exclude Hcell, and LVS Spice Cull Primitive Subcircuits.

Examples

# exclude all PAD cell calls from the source netlist during processing
LVS SPICE EXCLUDE CELL SOURCE "PAD*"
