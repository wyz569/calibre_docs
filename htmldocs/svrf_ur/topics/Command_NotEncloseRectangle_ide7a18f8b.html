Not Enclose Rectangle

Layer operation

Selects all polygons that cannot enclose a rectangle of the specified dimensions.

Usage

NOT ENCLOSE RECTANGLE layer width length [ORTHOGONAL ONLY [FIXED]]

Arguments

layer

A required original or derived polygon layer.

width

A required positive floating-point number interpreted in user units.

length

A required positive floating-point number interpreted in user units.

ORTHOGONAL ONLY [FIXED]

An optional keyword that specifies the enclosed rectangle must have sides that are parallel (respectively) to the coordinate axes of the database. A polygon on layer is selected if it cannot enclose a rectangle of the given width and length and with sides that are parallel to the coordinate axes of the database.

  • FIXED — Specifies that the width and length are measured along the x-axis and y‑axis, respectively, and cannot be interchanged.

Description

Selects all polygons on the specified layer that cannot enclose a rectangle of the specified width and length. This is the complement of the Enclose Rectangle operation. An enclosed rectangle may have one or more edges that are coincident with enclosing polygon.

Not Enclose Rectangle selects polygons that do not satisfy an Enclose Rectangle operation with the same arguments. If optional keywords are specified, the keywords specify conditions for the complementary Enclose Rectangle operation. Polygons on layer that do not satisfy the conditions as applied to the complementary Enclose Rectangle operation are selected.

The tool considers only enclosed rectangles that have sides parallel (respectively) to the coordinate axes or that have sides at a 45-degree angle to the coordinate axes. If ORTHOGONAL ONLY is specified, the enclosed rectangle must have sides parallel to the coordinate axes. The Not Enclose Rectangle operation returns layer polygons that cannot enclose a rectangle of the specified dimensions and orientation.

The tool may align width in any direction: along either the x- or y-axis, or either the plus- or minus- 45-degree angle. To align width and length with the x-axis and y-axis respectively, specify FIXED with ORTHOGONAL ONLY. By default, the operation behaves exactly the same if width and length are interchanged. For example, the following two operations produce equivalent results:

NOT ENCLOSE RECTANGLE poly 8 2
// this produces the same results: 
NOT ENCLOSE RECTANGLE poly 2 8

Performance is faster with ORTHOGONAL ONLY.

See also Not Enclose.

Examples

// select all polygons on metal that cannot enclose a 2 x 4 rectangle.
// The 2 x 4 rectangle may be orthogonal to the database axes 
// or oriented at 45 degrees to the database axes.
rule { NOT ENCLOSE RECTANGLE metal 2 4 }
