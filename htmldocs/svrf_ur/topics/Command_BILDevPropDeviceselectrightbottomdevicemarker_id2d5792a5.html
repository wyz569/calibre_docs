device::select_right_bottom_device_marker

Device TVF layer generation function. Used only in a compile-time TVF rule file.

Returns a derived polygon layer containing marker shapes coincident with the bottom-right corners of device seed shapes (where “bottom” and “right” can change due to cell placement transformations, but is always consistent with a certain corner). Input layers must be rectangular with edges parallel to the database axes. Pin shapes must be coincident with edges of seed shapes on each side of a seed shape. Devices that do not conform to these criteria cause the function to have empty output. This function is always called in the context of a tvf::SETLAYER command.

Usage

device::select_right_bottom_device_marker seed_layer pin_layer marker_width [‑help | ?]

Arguments

seed_layer

A required argument that specifies a device seed layer. Usually this is the gate layer of a MOS transistor.

pin_layer

A required argument set that defines a pin layer having shapes that touch the seed_layer. Usually this is a source/drain layer of a MOS transistor.

marker_width

A required argument that specifies a floating-point number in user units of length. This is the output width of marker polygons. This value must be less than one half the length of the minimum seed shape edge or no output occurs.

-help | ?

Prints the usage message for this function in a Tcl shell.

Return Values

Derived polygon layer.
