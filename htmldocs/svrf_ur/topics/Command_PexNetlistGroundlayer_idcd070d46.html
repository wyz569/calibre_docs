PEX Netlist Groundlayer

Parasitic extraction

Specifies to map the coupling to ground layer geometries that do not have a net name to a user specified net in the generated netlist. Use only with Calibre xACT.

Usage

PEX NETLIST GROUNDLAYER   layer gnd_net [layer gnd_net …]

Arguments

layer gnd_net

A required parameter set that specifies an LVS layer name or number and an associated ground net name to be used when netlisting coupled and intrinsic capacitors connected to the LVS layer. More than one layer gnd_net pair can be specified.

Description

An optional statement used to map the coupling to unnetted ground layer geometries to a user specified net in the generated netlist. This statement takes precedence over the PEX Netlist statement’s GROUND ground_name keyword set.

Examples

To add a ground net named GND1 to the netlist that was not in the calibrated rule file and assign it to LVS layer TOP_GROUND, include the following statement in your PEX rule file:

PEX NETLIST GROUNDLAYER TOP_GROUND GND1
