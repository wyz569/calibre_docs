DRC Specification Statements

This section lists the Calibre nmDRC specification statements. This a category of specification statements used for setting or modifying operating parameters for the Calibre nmDRC system.

The statements in Table 1 are used by all Calibre nmDRC applications except where noted.

Table 1. Specification Statements: DRC

Statement

Description

DRC Auto Error

Specifies whether to automatically produce error-directed output for dimensional rule checks by ignoring the REGION keyword.

DRC Boolean Nosnap45

Specifies how grid snapping is performed for merged layers having 45-degree edges.

DRC Cell Name

Specifies whether to append the cell name associated with each DRC result to its signature line in an ASCII DRC results database. Calibre nmDRC-H only.

DRC Cell Text

Specifies whether connectivity extraction text is read throughout the hierarchy of the input layout database. Calibre nmDRC-H only.

DRC Check Map

Specifies how DRC outputs the results for a specified rule check. Used for producing GDSII, OASIS, or ASCII results databases.

DRC Check Text

Specifies the DRC rule check text mapping mode for Calibre nmDRC execution. In ICverify, sets the initial results database text mapping mode default for the $check_drc() function.

DRC Exclude False

Minimizes the possibility of false edge measurement errors in hierarchical dimensional check operations in Calibre nmDRC‑H runs. Use of the YES option is strongly discouraged in a production rule file.

DRC Extend Trivial Edge

Specifies how trivial edge measurement results are extended in dimensional check operations.

DRC Incremental Connect

Enables incremental connectivity extraction for antenna checking and related applications.

DRC Incremental Connect Warning

Controls the generation of connectivity extraction warnings for [Not] Net operations that appear in a DRC Incremental Connect YES flow.

DRC Keep Empty

Specifies whether empty rule checks are retained in Calibre nmDRC execution. In ICverify, sets the initial default to control writing rule checks that contain zero results to the results database for the $check_drc() function.

DRC Magnify Density

Specifies the default magnification factor for Density RDB databases.

DRC Magnify NAR

Specifies the default magnification factor for Net Area Ratio RDB databases.

DRC Magnify Results

Specifies that the DRC results database is magnified by the given value as it is being output by Calibre nmDRC.

DRC Map Text

Specifies to transfer text objects in the input layout database to a mask results database. Used only with Calibre nmDRC‑H.

DRC Map Text Depth

Specifies the hierarchical depth from which to read text objects for the DRC Map Text YES specification statement.

DRC Map Text Layer

Specifies layers from which to map text objects to the mask Calibre nmDRC‑H results database.

DRC Maximum Cell Name Length

Specifies the maximum length of cell (and placement) names in GDSII and OASIS DRC results databases. Used only with Calibre nmDRC-H.

DRC Maximum Results

Specifies the Calibre nmDRC rule check maximum result count for Calibre nmDRC execution. In ICverify, sets the initial default maximum number of results that the $check_drc() function can place in the results database for each rule check statement.

DRC Maximum Results Density

Specifies a default value for the MAXIMUM keyword in the Density operation.

DRC Maximum Results NAR

Specifies a default value to be used whenever the MAXIMUM keyword is not explicitly specified for the RDB [ONLY] parameter in the Net Area Ratio statement.

DRC Maximum Unattached Label Warnings

Specifies the maximum number of warnings for unattached labels issued by the connectivity extraction module in DRC.

DRC Maximum Vertex

Specifies the maximum vertex count of any polygon result written to the DRC results database. Used only in Calibre nmDRC/Calibre nmDRC-H.

DRC Print Area

Computes the flat area for each specified layer and prints it to the transcript when the specified layers are generated. Used only in Calibre nmDRC/Calibre nmDRC-H.

DRC Print Extent

Computes the lower-left and upper-right corners of the specified layer and prints it to the transcript when the specified layers are generated. Used only in Calibre nmDRC/Calibre nmDRC-H.

DRC Print Perimeter

Computes the flat perimeter for each specified layer and prints it to the transcript when the specified layers are generated. Used only in Calibre nmDRC/Calibre nmDRC-H.

DRC Recon Add Check

Specifies rule check statements or rule check groups to be executed in a Calibre nmDRC Reconnaissance run.

DRC Recon Add Check By Layer

Selects rule check statements to be executed in a Calibre nmDRC Reconnaissance run based on original layer names.

DRC Recon Remove Check

Specifies rule check statements or rule check groups not to be executed in a Calibre nmDRC Reconnaissance run.

DRC Recon Remove Check By Layer

Selects rule check statements that are not to be executed in a Calibre nmDRC Reconnaissance run based on original layer names.

DRC Results Database

Specifies the filename and type of the Calibre nmDRC results database. Used only in Calibre nmDRC/Calibre nmDRC-H.

DRC Results Database Libname

Specifies the LIBNAME record of GDSII results databases.

DRC Results Database Precision

Specifies the precision value in the Calibre nmDRC results database. Has no effect on coordinate values output to the database.

DRC Select Check

Specifies rule checks or groups to be executed.

DRC Select Check By Layer

Specifies rule checks to be executed by layer names.

DRC Summary Report

Specifies the DRC summary report filename and writing method. In ICverify, sets the initial summary report filename and file opening mode for the $check_drc() function.

DRC Terminate Executive Check

Specifies the name of a RuleCheck causing the processing of subsequent RuleCheck operations to be expedited by producing no output.

DRC Tolerance Factor

Specifies a tolerance factor for dimensional check measurement constraints (External, Internal, and Enclosure).

DRC Tolerance Factor NAR

Specifies the tolerance for comparing Net Area Ratio computed values to the constraint values.

DRC Unselect Check

Specifies rule checks or groups to be excluded from the run.

DRC Unselect Check By Layer

Specifies rule checks to be excluded from a run, based upon layer names.

Group

Names a collection of rule check statements.
