count

Effective property computation language function.

Returns the count of instances in an input group.

Usage

count()

Arguments

None.

Return Values

Integer.

Examples

The count() function is useful for calculating averages.

[ EFFECTIVE r, avg
avg = sum(r) / count()
...
]
