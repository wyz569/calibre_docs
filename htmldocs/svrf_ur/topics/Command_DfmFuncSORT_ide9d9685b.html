SORT

Vector function for DFM expressions

Performs a lexicographic (dictionary) sort on the input vector. Supported for the DFM Property family of operations.

Usage

SORT(vector)

Arguments

vector

A required vector argument, which may be a numeric, netID, or string vector.

Return Values

A vector of the same type as the input.

Examples

This example forms a vector of areas, a two-tuple vector of perimeters and areas, and a vector of strings. All vectors are sorted. 

layer metal1  0     
layer metal2  1   

exampleA = DFM PROPERTY metal1 metal2 OVERLAP
  [ areavec = VECTOR(AREA(metal2))]
  [ areasort = SORT(VPROPERTY_REF(areavec))]
  [ perim_area_vec = VECTOR(PERIM(metal2), AREA(metal2))]
  [ perim_area_sort = SORT(VPROPERTY_REF(perim_area_vec))]
  [ string_vec = CONCAT(VSTRING("Vector"), VSTRING("string:"), 
		VSTRING("x"), VSTRING("a"), VSTRING("10"), VSTRING("1"))]
  [ string_sort = SORT(SVPROPERTY_REF(string_vec))]

dfm_out {
   DFM RDB exampleA "sort.rdb"  ALL CELLS MAXIMUM ALL checkname "%_l_"
}

The following vectors are attached to layer exampleA:

areavec: <0.04> <0.02> <0.03> <0.04> <0.02>
areasort: <0.02> <0.02> <0.03> <0.04> <0.04>
perim_area_vec: <0.8 0.04> <0.6 0.02> <0.8 0.03> <0.8 0.04> <0.6 0.02>
perim_area_sort: <0.6 0.02> <0.6 0.02> <0.8 0.03> <0.8 0.04> <0.8 0.04>
string_vec: <"Vector"> <"string:"> <"x"> <"a"> <"10"> <"1">
strsort: <"1"> <"10"> <"Vector"> <"a"> <"string:"> <"x">
