tvf::svrf_var

Application: runtime only

Gets the value of a variable defined with the Variable statement in the SVRF rule file. This statement gives the runtime TVF Function block access to variables defined with the Variable statement.

Usage

tvf::svrf_var variable_name

Arguments

variable_name

A required argument that specifies a name defined with the Variable statement in the SVRF rule file.

Return Values

The variable value.

Examples

VARIABLE myVar1 0.1

TVF FUNCTION myTVFfunc [/*
   tvf::GET_LAYER_ARGS L1 L2
   set dist [tvf::svrf_var myVar1]
   tvf::OUTLAYER "EXTERNAL $L1 $L2 <= $dist REGION OPPOSITE"
*/]
layer1 = TVF myTVFfunc A B

Also see “Example 3” with the Variable statement for information on passing array variables.
