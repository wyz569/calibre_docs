PEX Inductance Differential Pair

Parasitic extraction

Specifies two nets operating as a differential pair. Used only in Calibre xL.

Usage

PEX INDUCTANCE DIFFERENTIAL PAIR [LAYOUTNAMES | SOURCENAMES]
net_name_1 net_name_2

Arguments

LAYOUTNAMES

An optional keyword that specifies the net names are derived from the layout. This is the default.

SOURCENAMES

An optional keyword that specifies the net names are derived from the schematic. Ensure the SVRF rule file contains a valid Source Path statement and a valid Source Primary statement identifying the primary cell when specifying this option.

net_name_1

A required name identifying one of the differential pair nets.

net_name_2

A required name identifying the second differential pair net.

Description

Specifies two nets that operate under a differential signaling mode. This statement can be defined more than once to specify multiple differential pairs.

For inductance extraction calculations, a differential pair forms a current loop where one of the nets is the return path of the other net in the specified pair. This statement generates partial self and mutual impedances for the specified nets of the differential pair. This statement should only be used for nets that are connected in a differential signaling mode.

If PEX Inductance Returnpath NONE statement is specified in the rule file, then only self and mutual impedance for nets specified with this statement are computed and netlisted.

Examples

Example 1

The following statement instructs the Calibre xL tool to calculate self and mutual impedance between nets IN_P and IN_M.

PEX INDUCTANCE DIFFERENTIAL PAIR SOURCENAMES IN_P IN_M

Example 2

Specify the following statements to extract the partial self and mutual impedance for net pairs P6 and P11, and P9 and P7:

PEX INDUCTANCE DIFFERENTIAL PAIR P6 P11
PEX INDUCTANCE DIFFERENTIAL PAIR P9 P7
PEX INDUCTANCE RETURNPATH NONE

The presence of the PEX Inductance Return Path NONE statement means only the self and mutual impedances for net pairs P6 and P11 and P9 and P7 are netlisted.
