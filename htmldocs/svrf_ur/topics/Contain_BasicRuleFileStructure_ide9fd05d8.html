Basic Rule File Structure

The statements and operations that drive the Calibre tool are stored in a text file. You prepare this file, called the rule file, using any text editor. When the rule file is loaded, it is checked for syntax and other errors, and compiled. Because the compiler processes a rule file as a single entity, the order in which the elements are presented is generally not important. However, there are certain situations where elements are order-dependent.

See “Summary of Rule File Elements” for a categorized listing of the available SVRF operations and specification statements.
