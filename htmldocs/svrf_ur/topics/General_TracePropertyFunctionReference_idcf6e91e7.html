Trace Property Function Reference

The following summarizes the functions available in the Trace Property language.
Table 1. Trace Property Built-In Language Functions

Function

Definition

PROPERTY

Defines numeric properties to be compared in layout and source.

PROPERTY STRING

Defines string properties to be compared in layout and source.

Debugging functions

DEBUG

Enables debug monitoring during Trace Property computation.

WARN

Issues a warning to the transcript when the Trace Property program attempts an impossible computation.

Property Retrieval Functions

layout_numeric_value

Returns a numeric property value from a layout instance.

source_numeric_value

Returns a numeric property value from a source instance.

layout_string_value

Returns a string property value from a layout instance.

source_string_value

Returns a string property value from a source instance.

String-Related Function

string_compare

Compares two strings to determine if they are identical.

Discrepancy Reporting Functions

report_message

Generates a Trace Property discrepancy message in the LVS Report.

report_numeric_discrepancy

Generates a Trace Property numeric discrepancy in the LVS Report with the error value given as a percentage.

report_numeric_value

Generates a Trace Property numeric discrepancy message in the LVS Report.

report_string_discrepancy

Generates a Trace Property string discrepancy in the LVS Report.
