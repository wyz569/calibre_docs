Measurement Function Summary

This section provides a summary table for DFM measurement functions.
Table 1. DFM Measurement Function Summary

Function or Group Name

Description

ANGLE

Returns the angle between two edges in an edge cluster.

AREA

Returns the area of polygons on the input layer.

COUNT

Returns the number of objects on the input layer.

EC Group: Edge Projections

Returns the length of one error edge projected onto another (or, the Edge Concurrency). For single error edges, the length is returned.

EC Group: Projection Endpoints

Returns endpoint coordinates of the EC segment, where EC is the projection of one edge upon another.

EW Group: Edge Separation

Returns the minimum separation distance between edges (or, the Edge Width).

LENGTH Group

Returns a length measurement for edges on the input layer.

PERIMETER Group

Returns a sum of the edge lengths for each polygon on the input layer.
