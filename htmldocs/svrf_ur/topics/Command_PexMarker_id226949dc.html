PEX Marker

Parasitic extraction

Specifies how marker layers, defined with the PEX Map statement, are applied during extraction.

Usage

PEX MARKER [calibrated_layer_name] [ANCHOR | MASK | EXTRACT | NONE]

Arguments

calibrated_layer_name

An optional parameter that specifies the name of the calibrated layer to be marked. If a calibrated layer is not specified, the statement applies to all layers.

ANCHOR

An optional keyword that specifies the marker shape does not have to cover the whole geometry.

MASK

An optional keyword that specifies the marker shape must cover the geometries belonging to each mask. Marker shapes can be overlapping (stitched), defining portions of the original layout belonging to each mask.

EXTRACT

An optional keyword that instructs the tool to extract mask layers.

NONE

An optional keyword that instructs the tool not to apply marker layers. Turns off double patterning for the specified calibrated layer. This is the default.

Description

This optional statement specifies how marker layers, defined with the PEX Map statement, are applied during extraction. Use this statement to control double patterning behavior. This statement may be specified once per calibrated layer. If a calibrated layer is not specified, the statement applies to all layers. PEX Marker statements with specific layer names override the all layer default.

The PEX Map statement must define the marker layers for each of the exposure masks. If PEX Marker statement is not specified, double patterning is turned off for all defined marker layers.

Examples

Example 1

To specify that marker layers must cover geometries belonging to each mask for calibrated layer M1, include the following statements in the rule file:

PEX MARKER M1 MASK
PEX MAP M1 metal1 metal1a metal1b A(1) B(2)

Layers A(1) and B(2) represent the mask layers associated with the calibrated layer. Layer A represents mask 1 and layer B represents mask 2.

Example 2

To extract mask layers M1_A(1) and M1_B(2) include the following statements in the rule file: 

PEX MARKER M1 EXTRACT
PEX MAP M1 M1_A(1) 
PEX MAP M1 M1_B(2)

Example 3

To turn off double patterning for all layers include the following statement in the rule file: 

PEX MARKER NONE
