Ports

Layer operation

Generates a derived polygon layer consisting of all port shapes on the specified layer. Used only in Pyxis Layout.

Usage

PORTS layer

Arguments

layer

A required original layer or layer set.

Description

Generates a derived polygon layer consisting of all the port shapes on layer. Selects the port shapes only from the top-level cell of the active window.

Examples

x = PORTS metal1 // all ports on metal1
