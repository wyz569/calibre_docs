is_unknown

Effective property computation language TVF function.

Tests whether a property is assigned the UNKNOWN value. This function has limited use and is typically not needed.

Usage

$property is_unknown

Arguments

$property

A required Tcl variable call to a property value.

Return Values

Integer.

Description

Tests whether the property is assigned an UNKNOWN value (occurs when the property is not in the input netlist and a calculation is attempted with the property). It returns a 1 if the property is unknown and 0 otherwise. This function is used to avoid errors in the processing of an unknown property in a TVF Function proc. It is used in a foreach_instance block and is frequently used in conjunction with is_missing.
