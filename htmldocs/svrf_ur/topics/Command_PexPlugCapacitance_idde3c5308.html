PEX Plug Capacitance

Parasitic extraction

Specifies whether or not via capacitance between ground and other layers is extracted. Use only with Calibre xACT and Calibre xACT 3D direct netlisting flows.

Usage

PEX PLUG CAPACITANCE {YES | NO}

Arguments

YES

A required keyword that specifies to include the effects of vias between ground and other layers in capacitance calculations. This is the default.

NO

A required keyword that disables the extraction of via capacitance between ground and other layers.

Description

Use this optional statement to turn off calculations for all via capacitance between ground and other layers. This statement should only be used when PEX Ground Layer or PEX Map statements are specified for base layer mapping. PLUGs are vias to ground or base layers. If there are no PEX Ground layers or base layer mappings, then PLUGs are not created and this statement does nothing.

Examples

Base layers can be mapped using either the PEX Map statement:

PEX MAP substrate nxwell
PEX MAP substrate psub

or the PEX Ground Layer statement:

PEX GROUND LAYER nxwell psub

To make the default behavior explicit in your rule file, include the following statement:

PEX PLUG CAPACITANCE YES

To exclude capacitance for PLUG vias from extraction, use the following statement:

PEX PLUG CAPACITANCE NO
