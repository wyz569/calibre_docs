PERC LDL Layout Reduce Top Layers

Specification statement

Specifies top-level metal routing layers to optimize LDL layout processing.

Usage

PERC LDL LAYOUT REDUCE TOP LAYERS layer [layer …]

Arguments

layer

A required top-level metal routing layer. At least one layer must be specified. The list of layers should include all metal routing layers that constitute the top-level supply grid.

Description

Specifies top-level metal routing layers for LDL runs. Specifying these layers optimizes layout processing. Using this statement dramatically improves performance, so it is highly recommended for nearly any LDL rule file.

Note:

This statement is ignored for Layout System LEFDEF.

The top-level routing supply grid consists of low-resistance metal layers. These layers typically have no direct connections to devices. All such layers should be specified. Failure to provide a complete set of supply grid layers can lead to disjoint paths in results.

This statement may be specified multiple times in a rule file. The list of layers is aggregated across all occurrences of this statement. The one exception is in a Calibre Interactive generated control (or header) file where statements in the control file take precedence over statements in the Include file.

Examples

# metal routing layers of the top-level supply grid
PERC LDL LAYOUT REDUCE TOP LAYERS m4 m5 m6 m7 m8
