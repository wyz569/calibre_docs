PEX Reduce Mincap

Parasitic extraction

Specifies to combine or remove parasitic capacitors below a threshold value to reduce netlist size.

Usage

PEX REDUCE MINCAP {COMBINE threshold} | {REMOVE {threshold | NO}}

Arguments

COMBINE

A required keyword that causes capacitors with a value below threshold to be added together and reported as a single parasitic capacitor.

REMOVE

A required keyword that causes capacitors with value below threshold to be removed from a net.

threshold

A required non-negative, floating point value greater than 0 that specifies a capacitor threshold. The value is in user-defined capacitance units specified by the Unit Capacitance statement. If you do not specify Unit Capacitance, then picofarads is used. A threshold cannot be used with the NO keyword.

NO

A required parameter that specifies to turn off the default reduction of small coupling capacitance between pairs of nets that are less that 2e-19f. This keyword cannot be used with threshold.

Description

Combines or removes extracted capacitors based on a user-defined threshold value that is greater than 0. This command can be used to reduce the size of the netlist by reducing the number of extracted capacitors. Combining and removing capacitors take place during the formatting step of extraction.

This reduction technique can be used when the netlist includes parasitic self or mutual inductance as extracted by Calibre xL or the Calibre xACT inductance flow.

Combining Capacitors

When you specify a threshold value for PEX Reduce Mincap COMBINE, the extracted intrinsic capacitors that fall below this value are combined with the intrinsic capacitor of the nearest available node with an intrinsic capacitor.

For a net pair with extracted coupled capacitance, coupled capacitors less than the threshold value will be added to the nearest coupled capacitor on the same net pair.

For both intrinsic and coupled capacitors, if there is only a single combined capacitor left and the value falls below the threshold, then the capacitor is kept as is.

Removing Capacitors

The results of using PEX Reduce Mincap REMOVE are the same for intrinsic and coupled capacitors.

If the value of a capacitor is less than the user-defined threshold, the capacitor will not be included in the netlist during formatting. If the NO keyword is specified, then capacitance reduction is turned off.

SVRF Statement Precedence

Reduction statements are processed in this order:

  • If both PEX Reduce Coupled and PEX Reduce Mincap are in the rule file, PEX Reduce Coupled is ignored.

  • PEX Reduce TICER runs.

  • PEX Reduce Mincap COMBINE runs.

  • PEX Reduce Mincap REMOVE runs.

Examples

Assuming Unit Capacitance is specified as picofarads, to combine extracted capacitors falling below10 fF, use:

PEX REDUCE MINCAP COMBINE 0.01

To remove capacitors on nets that fall below 20 fF, use:

PEX REDUCE MINCAP REMOVE 0.02

To turn reduction of capacitance that is less than 2e19f, use:

PEX REDUCE MINCAP REMOVE NO
