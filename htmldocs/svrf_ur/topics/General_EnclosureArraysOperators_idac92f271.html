Enclosure Arrays and Operators

The ENClosure family of built-in functions returns results in arrays. The arrays require special operators to access values.

Depending on the array type, you can access the values from within the Device property computation program itself or by using the tvf_numeric_function as follows:

  • enclosure_parallel, enclosure_perpendicular, and enclosure_vector

    • Enclosure array — Structure for accessing the output from within the Device property computation program.

    • TVF Object for Enclosure Array — Using the TVF interface to access the enclosure array.

  • enclosure_parallel_MULtifinger and enclosure_perpendicular_MULtifinger

    • TVF Object for Multifinger Enclosure Array — Using the TVF interface to access the multifinger enclosure array. This is the only access method for the multifinger array.

The results of ENClosure_PARallel, ENClosure_PERpendicular, and ENClosure_VECtor are enclosure arrays. The enclosure array is a vector of triplets. Each triplet has the form (a,b,w). Individual components of those triplets can be retrieved with the vector selection operator (::) from within the device computation program. For example, if S is the result of ENClosure_PARallel or ENClosure_PERpendicular, then:

     S::a — Returns a vector of numbers containing all “a” values in S.

     S::b — Returns a vector of numbers containing all “b” values in S.

     S::w — Returns a vector of numbers containing all “w” values in S.

Each of these vector selection expressions returns a value that is a simple array of numbers. These expressions can be used in functions such as sum. The names a, b, and w are case-insensitive; for example, both S::a and S::A can be used.

The enclosure array can be passed to a Tcl process if desired. This is done through tvf_numeric_function. The enclosure array is handled as a TVF Object for Enclosure Array for non-multifinger functions. For the multifinger functions, use the TVF Object for Multifinger Enclosure Array.
