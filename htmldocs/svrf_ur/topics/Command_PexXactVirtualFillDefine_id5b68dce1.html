PEX XACT Virtual Fill ... Define

Parasitic extraction

Defines fill parameters for virtual metal fill in the Calibre xACT digital flow.

Usage

PEX XACT VIRTUAL FILL name DEFINE x1 y1 FILLSEPARATION x2 y2
OFFSET x3 y3 METALSEPARATION x4 y4

Arguments

name

A required argument specifying a name for the virtual fill specification.

x1 y1

A required parameter set that specifies the dimensions of the fill shape.

FILLSEPARATION x2 y2

A required keyword parameter set that specifies the spacing between adjacent fill shapes.

OFFSET x3 y3

A required keyword parameter set that specifies the x and y fill-to-fill offset.

METALSEPARATION x4 y4

A required keyword parameter set that specifies the x and y metal-to-fill separation.

Description

An optional statement used to define fill parameters for virtual metal fill in the Calibre xACT digital flow. Use this statement with the PEX XACT Virtual Fill ... Use statement to model virtual fill geometries on particular layers during extraction.

The xy measurements shown in Figure 1 describe the statement parameters.

Figure 1. Virtual Fill Parameters

This statement may only be used in the Calibre xACT digital flow. If used with Calibre xRC or Calibre xACT 3D, this statement has no effect and a warning is generated.

Examples

The following statements instruct the Calibre xACT tool to define a virtual fill shape and use it for LEFDEF layers M1 and M2.

PEX XACT VIRTUAL FILL filltype1 DEFINE 0.25 0.25 FILLSEPARATION 0.05 0.05 OFFSET 0.075 0.075 METALSEPARATION 0.25 0.35
PEX XACT VIRTUAL FILL M1 USE filltype1 H
PEX XACT VIRTUAL FILL M2 USE filltype1 V
