tvf::exists_global_variable

Application: compile-time only.

Checks for the existence of a global variable. Global variables allow you to share variables between Tcl interpreters, encrypted and unencrypted. Returns 1 if the variable exists, 0 otherwise.

Usage

tvf::exists_global_variable variable_name [(index1[, index2]…)]

Arguments

variable_name

A required argument that specifies the name of the global variable.

(index1[, index2]…)

An optional argument that specifies index values. If the optional index arguments are provided, variable_name is assumed to be the name of an array and the index determines the element in the array. Index values may be specified without the parentheses and commas.

Return Values

Returns 1 if the variable exists, 0 otherwise.

Examples

This example tests for existence of the global variable my_global:

puts "my_global existence is [tvf::exists_global_variable my_global]"

This example tests for existence of an element in a two-dimensional array:

puts "element 0 0 exists: [tvf::exists_global_variable my_g_array(0, 0)]"

This example is equivalent to the preceding example, but does not use parentheses and commas when specifying the index values for the two-dimensional array:

puts "element 0 0 exists: [tvf::exists_global_variable my_g_array 0 0]"
