Summary Table of Compile-Time TVF Elements and Commands

This section summarizes the elements and commands that can appear in a compile-time TVF file. The opening line must be the first line in the file but all other elements may be in any order.
Table 1. Elements of Compile-Time TVF File

Element

Summary

Opening line

This must be the first line in a compile-time TVF file:

#!tvf

Standard Tcl constructs

Standard Tcl constructs, such as loops, conditional statements, lists, variables, and Tcl comments, may be used.

Verbatim SVRF code

Passes SVRF code directly to the SVRF compiler without change.

Verbatim SVRF Code in Compile-Time TVF

tvf::VERBATIM

SVRF statements mapped to TVF

Most SVRF specification statements are mapped to TVF commands.

SVRF Statements Mapped to Compile-Time TVF

SVRF Statements Not Mapped to TVF

Comment characters

tvf:://

tvf::@

tvf::COMMENT

Layer definitions and assignments

Layer definitions and assignments are handled with the tvf::SETLAYER command. Command options allow you to get global and local layers defined with the SETLAYER command, and to get the layer derivation expression.

Layer Definitions and Assignments

tvf::SETLAYER

Rule checks

The SVRF rule check syntax is handled with the tvf::RULECHECK command.

Rule Checks in Compile-Time TVF

tvf::RULECHECK

tvf::SETLAYER

tvf::OUTLAYER

tvf::COPY

Global variables

You can get and set global variables in compile-time TVF scripts.

Global Variables in Compile-Time TVF

tvf::exists_global_variable

tvf::get_global_variable

tvf::set_global_variable

tvf::unset_global_variable

Command line arguments

You can get a user-defined command line argument and the arguments for the Calibre invocation line.

Command Line Arguments Passed to Compile-Time TVF

Global Variables in TVF for Calibre Command Line Arguments

tvf::get_tvf_arg

calibre_argc, calibre_argv, and calibre_argv0

Command for error reporting

Error Handling in Compile-Time TVF

tvf::set_traceback_line

Commands for transcript control

You can echo the translated SVRF code to the transcript.

Transcript Output with Compile-Time TVF

tvf::echo_svrf

tvf::block_transcript

tvf::is_transcript_blocked

tvf::unblock_transcript

Commands for formatting the output SVRF file

You can save the generated SVRF code to a file. Several commands are available for formatting the output file.

See “Saving the Generated SVRF Code to File” in “Calibre Command Line Options for Compile-Time TVF”.

tvf::echo_to_svrf_file

tvf::set_rule_check_indentation

tvf::set_disclaimer

Command to get Calibre version

tvf::get_calibre_version
