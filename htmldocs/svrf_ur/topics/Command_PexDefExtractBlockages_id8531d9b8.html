PEX DEF Extract Blockages

Parasitic extraction

Controls whether placement and routing blockage shapes are considered during coupling capacitance extraction in the Calibre xACT digital flow.

Usage

PEX DEF EXTRACT BLOCKAGES {NO | YES}

Arguments

NO

A required keyword that instructs the tool not to extract coupling to placement and routing blockage shapes. This is the default behavior if you do not specify this statement.

YES

A required keyword that instructs the tool to extract coupling to placement and routing blockage shapes as defined in the BLOCKAGES section in the DEF file.

Description

Controls whether the tool includes coupling capacitance to placement and routing blockage shapes as defined in the BLOCKAGES section of the DEF file during extraction. DEF files use blockages to prevent placement and routing in certain areas of the layout. By default, Calibre does not include coupling to placement and routing blockage shapes in extraction.

Examples

To extract coupling to placement and routing blockage shapes, specify the following statement in your rule file:

PEX DEF EXTRACT BLOCKAGES YES
