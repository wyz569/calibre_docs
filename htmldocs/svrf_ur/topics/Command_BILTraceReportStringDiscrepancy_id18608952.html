report_string_discrepancy

Trace Property computation language function.

Generates a Trace Property string discrepancy in the LVS Report.

Usage

REPort_STRing_DISCrepancy(prop_name)

Arguments

prop_name

A required name of a string-valued property declared in a PROPERTY STRING statement.

Return Values

None.

Examples

See “Sample Trace Property Program”.
