Mask1D Dialog Box

To access: In the Recipe tab, select Add > define_mask 1D.

Use this dialog box to set up a 1-dimensional (linear) mask.

Figure 1. Mask1D Dialog Box

Objects

Table 1. Mask1D Dialog Box Contents

Field

Description

+

–

Toggles the dialog box to show or hide additional fields.

Del

Add

Update

Clear

Buttons that act on rows selected in the table.

  • Del removes the row entirely.

  • Clear removes the data in the rows.

  • Update updates density based on length, space, and width.

Name

(Required) The name to use in other command’s Mask field. Valid names may include letters, numbers, and underscores.

Table data

The values to use in creating a mask. See “define_mask 1D” for a detailed description.

Other

Use for entering additional command parameters manually.

Usage Notes

No fields in the mask dialog boxes may take symbolic names or mathematical expressions.

This command dialog box is preserved for backwards compatibility. Generally two-dimensional masks are required to fit the large number of test structures on a chip.
