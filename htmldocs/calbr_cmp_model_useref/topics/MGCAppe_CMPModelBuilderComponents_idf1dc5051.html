CMP Model Builder Components

There are many components of a CMP model and it is imperative to know how they fit together. Some files are required inputs for an optimization run.

Other files are used primarily for storing temporary information for optimizations. The following components are the major components of CMP modeling.

  • CMP Models
  • Layouts
  • Measured Data Files (MD Files)
  • Mask 1D and 2D
  • Snapshots
  • Grid Files
  • Grid Frames (Pixels)
  • Optimized CMP Model File
