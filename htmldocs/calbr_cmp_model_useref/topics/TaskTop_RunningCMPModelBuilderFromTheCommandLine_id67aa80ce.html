Running CMP Model Builder from the Command Line

The CMP Model Builder batch method is suitable for running several slightly different simulations after the initial setup.

Prerequisites

  • Set the CALIBRE_HOME environment variable. See “Setting the CALIBRE_HOME Environment Variable” in the Calibre Administrator’s Guide for details.

  • Process recipe file.

  • Measured data file.

Procedure

  1. Create a Tcl script that calls the cmpoptimize command. See “Examples” in the cmpoptimize reference.
  2. Run the script by calling Calibre WORKbench:
    calibrewb optimize_script.tcl
