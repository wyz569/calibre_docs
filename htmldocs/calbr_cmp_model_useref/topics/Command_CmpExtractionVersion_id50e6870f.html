CMP_EXTRACTION_VERSION

Sets the extraction version.

Usage

setenv CMP_EXTRACTION_VERSION {1 | 2 | 3}

Arguments

1

Specifies to run the original CMP extraction version.

2

Specifies to run a newer version for extraction. This version uses weighted averaging of intersecting geometries and sets extra to 0 or 100 for field or array regions, respectively.

3

Specifies to run the current extraction version. This version improves area density extraction for dummy fill shapes in negative extraction, and for cheesing-like structures during positive extraction. No change to width and space data. It is the default.

Description

This high-level variable controls which form of extraction is used. It was introduced in Calibre v2010.4. In Calibre v2013.2, the default extraction version changed from 2 to 3. Version 3, the newest, is recommended over version 2.

This variable was initially called CALIBRE_MGC_CMP_EXTRACTION_VERSION.
