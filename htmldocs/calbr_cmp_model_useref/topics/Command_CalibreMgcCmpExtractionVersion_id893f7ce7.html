CALIBRE_MGC_CMP_EXTRACTION_VERSION

Controls which form of extraction is used.

Usage

setenv CALIBRE_MGC_CMP_EXTRACTION_VERSION 1

Arguments

1

The default value of the required argument. This setting runs the current version of the software.

Description

This high-level variable controls which form of extraction is used. It was introduced in Calibre v2010.1.

Note:

This variable has been replaced with CMP_EXTRACTION_VERSION. Please update any scripts to use the new name.
