Optimize Tab

To access: From Calibre WORKbench, Tools > CMP Model Builder, click the Optimize tab.

Use this interface to adjust optimization runs and view the results. If you click the + button in the upper right corner of the window, additional controls are displayed.

Optimization controls, only visible when the run is set to O (Optimization).

Search Front/Newton/Gradient/Full — Algorithm to use for optimization. The default is Front. See “Usage Notes” for more description.

Parameter ordering — Use the Up and Down buttons to move selected parameters in the table.

Parameter — The parameter name as it appears in the recipe, stripped of the $ character.

Best — The recommended setting post-optimization.

The parameters listed first are optimized first and given more weight for the final solution.

Usage Notes

Best Practices

To ensure reasonable time to completion, limit the number of parameters per optimization run to 10 or fewer. If you have more than 10 parameters you want to optimize together, you can optimize parameters with the largest effect first, then load their log files and optimize secondary parameters.

Optimization Methods

The optimization algorithms are the same as those used for Calibre WORKbench.

  • Front optimization is the default. It uses a global search optimization method that is less dependent on a starting point than the Gradient and Newton options, which is why it is the recommended method. Front optimization performs search iterations based on the explore or runtime options, and uses up to 16 CPUs in -turbo mode.

  • Newton optimization uses a quasi-Newton optimization method, in which the second derivatives are evaluated in terms of the sequence of gradients from successive iterations using an update formula for the Hessian matrix.

  • Gradient optimization uses a simple conjugate gradient descent method. It is faster than the other methods but often gives different results.

  • Full optimization causes a full factorial search over all combinations of variables. It is the slowest of the optimization methods.
