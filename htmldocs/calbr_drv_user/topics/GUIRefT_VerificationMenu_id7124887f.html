Verification Menu

Contains commands used to access Calibre .

Description

The Verification menu shown in Figure 1 provides access to several Calibre GUI interfaces for verification tools, such as Calibre Interactive, Calibre RVE, Calibre Pattern Matching, and Calibre RealTime. Refer to for descriptions of these fields.

Figure 1. Verification Menu

Fields

Table 1. Verification Menu

Field

Keyboard Shortcut

Description

Run nmDRC

  

Invokes Calibre Interactive for initiating a Calibre nmDRC run. Refer to “Using Calibre Interactive to Perform DRC” in the Calibre Interactive User’s Manual for more information.

Run DFM

  

Invokes Calibre Interactive for initiating a Calibre DFM run. Refer to “Using Calibre Interactive to Perform DFM” in the Calibre Interactive User’s Manual for more information.

Run nmLVS

  

Invokes Calibre Interactive for initiating a Calibre LVS run. Refer to “Using Calibre Interactive to Perform LVS” in the Calibre Interactive User’s Manual for more information.

Run PERC

  

Invokes Calibre Interactive for initiating a Calibre PERC run. Refer to “Using Calibre Interactive to Perform PERC” in the Calibre Interactive User’s Manual for more information.

Run PEX

  

Invokes Calibre Interactive for initiating a Calibre PEX run. Refer to “Using Calibre Interactive to Perform Parasitic Extraction” in the Calibre Interactive User’s Manual for more information.

Run xACT

  

Invokes Calibre Interactive for initiating a Calibre xACT run. Refer to “Using Calibre Interactive xACT” in the Calibre Interactive User’s Manual for more information.

Run 3DSTACK

  

Invokes Calibre Interactive for initiating a Calibre 3DSTACK run. Refer to “Getting Started With Calibre 3DSTACK “in the Calibre 3DSTACK User’s Manual for more information.

Start RVE

Invokes the Calibre RVE tool selection box. Refer to “Getting Started with Calibre RVE” in the Calibre RVE User’s Manual for information on using Calibre RVE.

RVE/CI Setup

  

Invokes the RVE/CI Setup dialog box. Refer to “RVE/CI Setup Dialog Box” for information on using this dialog box.

RealTime

  

Provides options for running Calibre RealTime with Calibre DESIGNrev. Options include:

  • Run DRC

  • Options — Displays the Calibre RealTime-Options dialog box used to specify configuration options for running Calibre RealTime.

Refer to the Calibre RealTime Custom User’s Manual for more information

Pattern Matching

  

Provides options for invoking Calibre Pattern Matching from Calibre DESIGNrev. Options include:

  • Run Capture — Invokes the Capture Patterns GUI.

  • Run Match — Invokes the Run Matching GUI.

  • Open GUI — Invokes the Calibre Pattern Matching GUI.

Refer to the Calibre Pattern Matching User’s Manual for more information.
