PATH_ENDSEGMENT_SHORT

Input for: readerprefs file (see readerprefs File Format)

The endsegment of a type-0 GDSII path or flush-ended OASIS path is less than 1/2 the path width.

Usage

PATH_ENDSEGMENT_SHORT {0 | 1 | 2 | 3 | 4}

Arguments

0

Quietly expand the path, even if a small notch develops.

Note:

HC mode only supports severity 0 behavior.

1

Warn and expand the path, even if a small notch develops (default).

2

Error.

3

Quietly extend the short endsegment to avoid a notch prior to expanding the path.

4

Warn and extend the short endsegment to avoid a notch prior to expanding the path.

Examples

# Reader Preferences File 
exceptionSeverity PATH_ENDSEGMENT_SHORT 4
