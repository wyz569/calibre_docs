Loading a Calibre Results Database into Calibre RVE

Use the procedure to load the results from Calibre verification runs when they are saved as an ASCII results database. This is done automatically if you ran Calibre from Calibre Interactive and selected Show results in RVE.

Prerequisites

  • Calibre DESIGNrev is running in interactive GUI mode and a layout is open.

  • You have the verification results for the layout saved in ASCII RDB format.

Procedure

  1. Choose Verification > Start RVE.

    This displays the Calibre RVE dialog box.

  2. Select the appropriate database type.
  3. Click the Browse button to open the file browser, then select the path to the Calibre ASCII database corresponding to the database you loaded into Calibre DESIGNrev.
  4. Click Open.

    The system opens the results database in the Calibre RVE application, allowing you to step through results and view them in the Calibre DESIGNrev revision application.
