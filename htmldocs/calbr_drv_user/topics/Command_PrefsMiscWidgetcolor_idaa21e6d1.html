prefs_misc_widgetColor

Input for: preferences file (see preferences File Format)

Sets a color scheme for the GUI appearance.

Usage

prefs_misc_widgetColor {grey | desktop}

Arguments

grey | desktop

A required value used to set a color scheme for the GUI appearance. Valid values are:

  • grey — The GUI uses a grey (monochrome) color scheme (default).

  • desktop — The GUI use the desktop color scheme.

Description

This preference can also be set in the Session tab of the Preferences dialog box by selecting the “Grey” or “Follow desktop color” radio button.

Examples

prefs_misc_widgetColor desktop
