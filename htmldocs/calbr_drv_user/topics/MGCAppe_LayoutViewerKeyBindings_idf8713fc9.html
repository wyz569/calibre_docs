Layout Viewer Key Bindings

A set of default key bindings is provided with the layout viewers. When creating custom key bindings, you should be careful not to overwrite the default key bindings.

The “bindKey Function Targets” column in Table 1 lists the specific key function targets used to implement the default Calibre DESIGNrev key bindings.

You can use the $cwb bindKey command to create custom key or mouse wheel bindings.
