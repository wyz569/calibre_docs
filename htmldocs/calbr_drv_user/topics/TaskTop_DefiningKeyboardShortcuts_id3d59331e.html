Defining Keyboard Shortcuts

The layout viewers allow you to define your own keyboard shortcuts for a wide range of functions. These range from clearing rulers and accessing menu items to hiding layers or displaying dialog boxes.

For a complete list of the functions you can map to these keys, refer to the $cwb bindKey command description in the Calibre DESIGNrev Reference Manual.

The Shift function keys are reserved for customer use, and are guaranteed to not be assigned bindings within Calibre DESIGNrev. Any other keys can have bindings assigned by Calibre DESIGNrev.

Procedure

  1. Create a keyprefs file in your layout viewer configuration directory.

    You can copy the $HOME/.calibrewb_workspace/keyprefs.template file (a template file provided by Siemens EDA) and rename it to keyprefs to use as a starting point.

  2. Edit the keyprefs file and use the bindkey command to create your own set of unique keyboard shortcuts.
    Note:

    Alternatively, you can type the layout viewer batch command $cwb bindkey in the layout viewer console window to create a temporary keyboard shortcut. Using this method, the keyboard shortcut is not saved in the keyprefs file and is not restored in subsequent sessions. Usage information for the batch commands can be found in the “Layout Viewer Batch Commands and Object Methods” chapter in the Calibre DESIGNrev Reference Manual.
