prefs_view_layerColorDrawMode

Input for: preferences file (see preferences File Format)

Specifies the display of layer colors.

Usage

prefs_view_layerColorDrawMode {overwrite | blend}

Arguments

overwrite | blend

A required argument used to specify the display of layer colors. Valid values are:

  • overwrite — The color of objects on the highest level overwrites object colors on lower levels (default).

  • blend — The color of objects on all overlapping layers are blended.

Description

This preference can also be set by choosing the Layer > Overwrite Colors check box to overwrite layer colors or choosing the Layer > Blend Colors check box to blend layer colors.

Examples

prefs_view_layerColorDrawMode blend
