POLYGON_AREA_ZERO

Input for: readerprefs file (see readerprefs File Format)

A zero-area boundary exists in a GDSII-type database or a zero-area polygon exists in an OASIS database.

Usage

POLYGON_AREA_ZERO {0 | 1 | 2 | 3}

Note:

In HC mode, the polygon is silently read into the database.

Arguments

0

Ignore the object.

1

Warn and ignore the object.

2

Error.

3

Display the path (default).

Examples

# Reader Preferences File 
exceptionSeverity POLYGON_AREA_ZERO 2
