prefs_text_drawScaled

Input for: preferences file (see preferences File Format)

Toggles between drawing scaled text or fitted height text.

Usage

prefs_text_drawScaled {0 | 1}

Arguments

0 | 1

A required value used to toggle between drawing scaled text or fitted height text. Valid values include:

  • 0 — Draw fixed (no text attributes) height text.

  • 1 — Draw scaled height text (default).

Description

This preference can also be set in the Objects tab of the Preferences dialog box by clicking the “Fixed (no text attributes)” or “Scaled” radio button in the Text field.

Examples

prefs_text_drawScaled 0
