Choose Layout Files Dialog Box

To access: File > Open Layout Files

Use the Choose Layout Files dialog box to browse to and select a layout to open and for specifying options that control what is loaded with the layout. This dialog box follows the conventions of a standard file and directory navigation tool, but also includes some advanced features specific to the layout viewer.
