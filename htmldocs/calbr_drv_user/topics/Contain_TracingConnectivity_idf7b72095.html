Tracing Connectivity

A net is an electronically interconnected set of objects in an electronic design. The layout viewer allows you to highlight a net based on a property name. You can also highlight a set of paths and polygons as a connected net. It extracts the connectivity based on the intersection of polygons and paths on layers in the layout.
Note:

Net extraction is not supported for overlays. An overlay is a set of superimposed yet unmerged layouts used for visual comparison.
