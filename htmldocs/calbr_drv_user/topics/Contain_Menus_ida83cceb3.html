Menus

The Calibre DESIGNrev layout viewer menus provide access to the application commands.

Since Calibre DESIGNrev is the foundation for Calibre MDPview, Calibre WORKbench and Calibre LITHOview, some menu options and toolbar buttons are available in some viewers but not in others.

The application commands are organized into the menus shown in Figure Figure 1.
