prefs_view_clipColor

Input for: preferences file (see preferences File Format)

Specifies the color of the incremental loading background.

Usage

prefs_view_clipColor xwin_color_name

Arguments

xwin_color_name

A required argument that specifies the color of the incremental loading background. You must specify a valid xwin_color name or a Hex color code. Refer to “Layout Viewer Color Reference” for a list of valid names. When this preference is not specified, the default is gray40 (#666666).

Description

This preference can also be set in the View tab of the Preferences dialog box by entering a value in the “Incremental load background” text box. You can optionally click the color button to display the Choose Incremental Load Background Color dialog box.

Examples

prefs_view_clipColor #aaaaaa
