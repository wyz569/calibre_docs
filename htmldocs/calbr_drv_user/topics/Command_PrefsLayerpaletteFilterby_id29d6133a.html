prefs_layerpalette_filterBy

Input for: preferences file (see preferences File Format)

Controls the display of the Filters panel.

Usage

prefs_layerpalette_filterBy {none | visible}

Arguments

none | visible

A required argument that controls the display of the Filters panel. Valid values are:

  • none — Hides the Filters panel.

  • visible — Displays the Filters panel (default).

Description

This preference can also be set in the Misc tab of the Preferences dialog box by enabling or disabling the “Always show filter panel” check box.

Examples

prefs_layerpalette_filterBy none
