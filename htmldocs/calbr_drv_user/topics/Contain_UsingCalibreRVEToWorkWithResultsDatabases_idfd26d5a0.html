Using Calibre RVE to Work with Results Databases

When used in conjunction with Calibre DESIGNrev, Calibre RVE makes it possible for you to visually inspect the results databases output by DRC, LVS, or yield analysis.

In most cases you configure Calibre Interactive to load the results directly into Calibre RVE as soon as the run completes. However, if you do not, there are options for loading the data into Calibre RVE after the run.
