XNAME_UNSUPPORTED

Input for: readerprefs file (see readerprefs File Format)

An XNAME record (not supported) has been encountered.

Usage

XNAME_UNSUPPORTED {0 | 1 | 2}

Arguments

0

Quietly process the placement as normal (default).

Note:

HC mode only supports severity 0 behavior.

1

Warn and process the placement as normal.

2

Fatal read error.

Examples

# Reader Preferences File 
exceptionSeverity XNAME_UNSUPPORTED 1
