Configuration File Reference

Any time you configure the layout viewer, the tool saves your setting in one of several configuration files. While you can use the GUI to configure some aspects of the tool, you also have the option of editing the configuration files. Some aspects of the layout viewer behavior can only be modified by editing one of the configuration files.
  • keyprefs File Format
  • layerprops File Format
  • preferences File Format
  • readerprefs File Format
  • wbinit.tcl File Format
  • wbposttkinit.tcl File Format
