prefs_view_refOriginMarker

Input for: preferences file (see preferences File Format)

Specifies the type of marker to draw at the selected reference’s origin.

Usage

prefs_view_refOriginMarker {none | square | cross}

Arguments

none | square | cross

A required argument used to specify the type of marker to draw at the selected reference’s origin. Valid values are:

  • none — No marker (default).

  • square — Draw a square.

  • cross — Draw a cross.

Description

This preference can also be set in the View tab of the Preferences dialog box by selecting the desired option from the “Reference origin markers” dropdown list.

Examples

prefs_view_refOriginMarker Square
