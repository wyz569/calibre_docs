No Wires In Loaded Design

A source file known to contain wires erases the wires when the file is loaded into Calibre DESIGNrev.

Causes

Solution

The loading option that converts paths to polygons has been set.

  1. Choose File > Open Layout Files.

  2. Click the Options tab.

  3. Ensure the “Convert Paths to Polygons” check box is unchecked before loading the file.
