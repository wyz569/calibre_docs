prefs_save_keepEmptyCells

Input for: preferences file (see preferences File Format)

Specifies how to handle empty cells and references when exporting a layout.

Usage

prefs_save_keepEmptyCells {0 | 1 | 2}

Arguments

0 | 1 | 2

A required value used to specify how to handle empty cells and references when exporting a layout. Valid values are:

  • 0 — Do not write empty cells, but write their references.

  • 1 — Write all cells and references (default).

  • 2 — Do not write empty cells or their references.

Description

This preference can also be set in the Export Layout dialog box by selecting the “Write all cells and references”, “Do not write empty cells, but write their references”, or “Do not write empty cells of their references” radio button.

Examples

prefs_save_keepEmptyCells 0
