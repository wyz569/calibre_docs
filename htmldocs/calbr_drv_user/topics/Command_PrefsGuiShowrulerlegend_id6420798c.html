prefs_gui_showRulerLegend

Input for: preferences file (see preferences File Format)

Enables or disables the display of the ruler legend in the layout viewing area.

Usage

prefs_gui_showRulerLegend {0 | 2}

Arguments

0 | 2

Required argument used to enable or disable the display of the ruler legend in the layout viewing area. Valid values are:

  • 0 — Hides the memory ruler legend in the layout viewing area (default).

  • 2 — Shows the memory ruler legend in the layout viewing area.

Description

This preference can also be set in the Grid tab of the Preferences dialog box by enabling or disabling the “Show Ruler Legend” check box.

Examples

prefs_gui_showRulerLegend 2
