Adding Custom Fill Patterns

You can customize the layout viewer with your own fill patterns.

Procedure

  1. Create one image file for each custom fill pattern. The image file must be in the X bitmap format (.xbm). There are no size restrictions, as the pattern repeats when tiled across the area to be filled.
  2. Save the fill pattern image file to an accessible directory.
  3. Assign the fill pattern to the layer of your choice in the layer properties file.
