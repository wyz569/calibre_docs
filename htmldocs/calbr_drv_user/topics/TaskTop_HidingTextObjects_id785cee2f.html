Hiding Text Objects

Text in areas of dense features and multiple layers can hinder you from selecting the correct shape.

Prerequisites

  • Calibre DESIGNrev is running in interactive GUI mode and a layout is open.

Procedure

You can toggle between the following options to hide text:

If you want to...

Do the following:

Hide all text items

Choose View > Hide Text.

Hide text items on a lower level

Choose View > Hide Lower Level Text.
