PATH_COINCIDENT

Input for: readerprefs file (see readerprefs File Format)

A path has no length.

Usage

PATH_COINCIDENT {0 | 1 | 2 | 3 | 4}

Arguments

0

Quietly ignore the path.

1

Quietly process the path (default).

Note:

HC mode only supports severity 1 behavior.

2

Warn and ignore the path.

3

Warn and process the path.

4

Fatal error.

Examples

# Reader Preferences File 
exceptionSeverity PATH_COINCIDENT 2
