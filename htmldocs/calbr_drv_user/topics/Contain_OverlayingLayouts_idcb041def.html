Overlaying Layouts

Overlaying refers to the process of superimposing layouts (without concatenating the data) for the purpose of visual comparison. Calibre DESIGNrev provides the ability to create, edit, and work with overlays. The user interface for overlays is slightly different than that for viewing individual layout files.
