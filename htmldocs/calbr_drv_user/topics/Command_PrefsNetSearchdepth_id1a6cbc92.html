prefs_net_searchDepth

Input for: preferences file (see preferences File Format)

Sets a depth for searching for nets.

Usage

prefs_net_searchDepth {top | all}

Arguments

top | all

A required value specifying the depth for searching for nets in the hierarchy. Valid values include:

  • Top only — Searches the top level only (default).

  • All levels — Searches all levels.

Description

This preference can also be set in the Nets tab of the Preferences dialog box by selecting the “Top only” or “All levels” radio button.

Examples

prefs_net_searchDepth all
