No Grid Display

The grid is on, but grid points are not visible.

Causes

Solution

  • The grid is set to None in the Preferences dialog box (Grid tab).

  • The grid is set to larger than is visible in the zoomed area.

  • The grid is set to the same color as the background.

Change the settings in the Grid tab of the Preferences dialog box.

The grid is not displayed when it becomes so dense it obscures the layout. If you zoom to a point where grid lines or points abut one another, the grid disappears. This feature ensures the grid does not obscure the design.

None.
