prefs_edit_showPopupSelect

Input for: preferences file (see preferences File Format)

Enables or disables the display of information about a selected object.

Usage

prefs_edit_showPopupSelect {0 | 1}

Arguments

0 | 1

A required argument used to enable or disable the display of information about a selected object. If enabled and multiple objects are selected, information is displayed about the object the cursor is currently in. Valid values are:

  • 0 — Disables the display of information about a selected object (default).

  • 1 — Enables the display of information about a selected object.

Description

This preference can also be set in the Objects tab of the Preferences dialog box by enabling or disabling the “Show object info popup window” check box.

Examples

prefs_edit_showPopupSelect 1
