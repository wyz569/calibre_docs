prefs_grid_autoColorEnabled

Input for: preferences file (see preferences File Format)

Specifies whether the grid color is auto-generated or a user-specified custom color.

Usage

prefs_grid_autoColorEnabled {0 | 1}

Arguments

0 | 1

Required arguments used to specify whether the grid color is automatically generated or a custom color specified by the user. Valid values are:

  • 0 — The grid color is a custom color specified by the user.

  • 1 — The grid color is automatically generated (default).

Description

This preference can also be set in the Grid tab of the Preferences dialog box using the Auto or Custom radio buttons in the Color field.

Examples

prefs_grid_autoColorEnabled 0
