Copying Cells to a Different Layer

You can select and copy existing objects to a new layer.

Prerequisites

  • You have a layout open in Calibre DESIGNrev that contains one or more cells.

Procedure

  1. In the Cells Browser, select the cell you want to copy.
  2. Choose Layer > Copy.

    This displays the Copy dialog box.

    Figure 1. Copy Current Cell Dialog Box

    If you want to...

    Do the following:

    Copy the current cell at the current view depth

    1. Set the view depth by clicking the Depth values in the status bar.



      This displays the View Depth dialog box.

    2. Enter the start and end depths and click OK.

    3. In the Copy dialog box, specify an Input Layer (the input layer must be an existing layer).

    4. Specify an Output Layer (the output layer can be an existing layer or a layer that does not exist).

    5. Select the “Current Cell” option.

    6. Select the “Follow view depth” option.

    Copy the current cell and all levels of the hierarchy

    1. In the Copy dialog box, specify an Input Layer (the input layer must be an existing layer).

    2. Specify an Output Layer (the output layer can be an existing layer or a layer that does not exist).

    3. Select the “Current Cell” option.

    4. Select the “All levels” option.

    Copy a selected area of the current cell

    1. In the Copy dialog box, specify an Input Layer (the input layer must be an existing layer).

    2. Specify an Output Layer (the output layer can be an existing layer or a layer that does not exist).

    3. Select the “Current Cell” option.

    4. Select the desired view depth option:

      • Follow view depth — Copies the levels of the hierarchy based on the start and end view depth of the current cell

      • All levels — Copies all levels of the hierarchy of the current cell

    5. In the Layout Viewing area, use the left mouse button to draw a rectangle around the area to copy.

    6. In the Copy dialog box, select the Selection rectangle check box and choose the selection criteria:

      • Contain — Copies objects that are completely inside the selection rectangle

      • Touch — Copies objects that touch the selection rectangle

      • Clip — Copies objects that are based on the clip created by the selection rectangle
  3. Click Run.

    The current cell is copied from the input layer to the output layer. The objects in the cell that are copied are based on the view depth and selection rectangle settings.
