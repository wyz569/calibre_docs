AREF_ANGLED

Input for: readerprefs file (see readerprefs File Format)

Controls handling of AREF rotation that is not a multiple of 90 degrees.

Usage

AREF_ANGLED {0 | 1 | 2}

Arguments

0

Quietly process the AREF placement as normal (default).

Note:

HC mode only supports severity 0 behavior.

1

Warn and process the AREF placement as normal.

2

Error.

Examples

# Reader Preferences File 
exceptionSeverity AREF_ANGLED 2
