prefs_ruler_direction

Input for: preferences file (see preferences File Format)

Sets the allowed snapping direction used for drawing rulers.

Usage

prefs_ruler_direction {a | m | 45}

Arguments

a | m | 45

A required value that specifies the snapping direction used for drawing rulers. Valid values are:

  • a — Sets no restriction (default).

  • m — Restricts movement to manhattan (90-degree) angles.

  • 45 — Restricts movement to 45-degree angles.

Description

This preference can also be set in the Rulers tab of the Preferences dialog box by choosing the Arbitrary, Manhattan, or 45-deg radio button.

Examples

prefs_ruler_direction m
