Creating an Array of Cell References

A cell reference array is a collection of cell references arranged in a two-dimensional array. Cell reference arrays are treated as a single object and manipulated using any function that applies to cell references.

The properties you can edit are all of the cell reference properties, which impact each cell reference in the array, plus the dimensions of the array (expressed in terms of number of references along the x and y axis), and the spacing of the references in each direction.

Prerequisites

  • Calibre DESIGNrev is running in interactive GUI mode and a layout is open.

Procedure

  1. In the Cells Browser, select the cell containing the array.
  2. Set the viewer depth to 0. The individual cell references appear outlined.
  3. Select the array cell. The application highlights the entire array.
  4. Display the Object Properties dialog box using one of the following methods:

    • Click the right mouse button to display the Array of Cell menu and choose Properties.

    • Choose Object > Properties.

  5. To modify the orientation of the individual cell references, edit any of the following:

    • Magnification

    • Angle of rotation

    • Mirror

  6. To modify the origin of the array, edit the X or Y coordinates.
  7. To modify the size of the array (the number of references), edit the values in the Columns and Rows fields located in the Array section of the Object Properties dialog box.
  8. To modify the spacing between references, edit the values in the dx and dy text boxes. These values can be negative to reflect relative positions below or to the left of the array origin.
  9. Click Apply.
