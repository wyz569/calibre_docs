Specification of an Empty Layer

An empty layer can be specified in the rule file.

The following example shows how to specify a layer that is empty:

LAYER empty 999 // no data on this layer
LAYER MAP 999 DATATYPE >=0 1000 // all polygon data on 999 goes to 1000
LAYER MAP 999 TEXTTYPE >=0 1000 // all text on 999 goes to 1000
LAYER from_999 1000 // everything from 999 is mapped here.

The layer 999 contains no data and may be used for suitable purposes.
