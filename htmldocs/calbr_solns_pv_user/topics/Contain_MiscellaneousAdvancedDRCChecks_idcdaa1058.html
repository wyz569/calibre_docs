Miscellaneous Advanced DRC Checks

This section includes miscellaneous advanced DRC checks.
Table 1. Miscellaneous DRC Check Solutions

Name

Description

SVRF Elements

Checking Dangling Wires

Identifies sections of metal that connect to one or more vias at one end, and are unconnected at the other end.

External, DFM Space, DFM Copy, XOR, Convex Edge, Internal, NOT, DFM Property, Expand Edge, Interact, DFM RDB

Finding Circular Polygons

Finds polygons that are approximately circular using two different methods.

Inside Cell, DFM Circle Analyze, DFM Property, DFM RDB

Finding Length and Width Edges of Rectangles

Identifies length and width edges of rectangular polygons that are not square.

Length, Rectangle, DFM Property

Finding Pad Edges Closest to the Chip Edge

Identifies edges on pads that are closest to the edge of the chip.

Enclosure, Expand Edge, DFM Property, DFM RDB

Finding Row and Column Counts in an Array

Finds the number of rows and columns in an array of shapes.

Convex Edge, DFM Copy, DFM Property, DFM RDB, Grow, Extents, External, NOT

Finding Polygon Extents for Special Configurations

Determines polygon extents for cases where extents overlap. An unmerged polygon layer containing the polygon extents is also generated.

DFM Property, DFM Property Select Secondary, DFM RDB

Finding Unique Violations

Finds unique violations from a set of DRC violations. The rule check outputs only one error marker from a set of redundant violations. The redundant violations occur when more than one spacing violation occurs between the same two polygons. This is useful for reducing the number of DRC results.

External, Internal, DFM Classify, DFM Property, DFM Space, DFM RDB

Resizing Polygons by a Percentage of the Area

Resizes polygons by a percentage of their area.

DFM Property, DFM Size, DFM RDB
