Advanced DRC Solutions

Advanced DRC solutions use DFM properties to classify shapes in the design. Many of the solutions use equation-based DRC (eqDRC), which is a set of SVRF statements that extend the capabilities of Calibre nmDRC. Using eqDRC, you can analyze complex interactions that are difficult or impossible to verify using traditional DRC methods. The eqDRC toolset includes the ability to manipulate DFM properties, and to write complex equations using DFM Expressions.
