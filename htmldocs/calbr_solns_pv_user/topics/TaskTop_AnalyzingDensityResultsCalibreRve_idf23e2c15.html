Analyzing Density Results with Calibre RVE

Visualizing Density results using Calibre RVE is important for interpreting what they mean and how to fix them. Results visualization also assists you in writing better Density checks by providing properties that can be sorted, highlighted in color maps and histograms, and displayed in the layout itself.

Prerequisites

  • Calibre nmDRC and Calibre RVE licenses.

  • DRC rule file and geometric layout.

  • Basic understanding of the Density layer operation.

  • Familiarity with using Calibre RVE. See “Using Calibre RVE” in the Calibre RVE User’s Manual.

  • Integration between your layout viewer and Calibre RVE. See “Connection to a Design Tool” in the Calibre RVE User’s Manual.

Procedure

  1. Open your rule file and add the RDB option to a Density layer operation. For example:
    filldensity3 = DENSITY fill keepout < d 
  [(AREA(fill)/!AREA(keepout))/AREA()] WINDOW w STEP s 
  RDB filldensity3.rdb

filldensity3 {
@ Density fill< ^d with function (AREA(fill)/!AREA(keepout))/AREA()
COPY filldensity3
}

    The layer operation’s constraint, window, and step parameters should reflect something useful for your process. The RDB file filldensity3.rdb is used for detailed results analysis in Calibre RVE.

  2. Run DRC in the usual way for your flow using the rule from Step 1. Use the DRC Select Check statement in the rule file as necessary to limit results output.
  3. If it is not already open, start your layout viewer and open your layout.
  4. Start Calibre RVE in the usual way for your flow and open your DRC Results Database. The results for the filldensity3 rule check shown in Step 1 might appear as follows in Calibre RVE:


  5. Highlight the filldensity3 results by selecting H on your keyboard. The results could appear like this:


    The results from the DRC Results database show merged regions of low density. While informative, this may not be enough detail for you to decide what the results mean. The following steps show how to get more detail.

  6. In Calibre RVE, select Setup > Options. In the Options tab, ensure the following settings are enabled:

    DRC Browser — Properties: Show calculated properties; Side RDBs: Density: Show As Check

    Highlighting — Layout Highlighting: Highlight Text Size: 0.2

    DRC/DFM Highlighting: Show Check Names, Show Result IDs, Show Properties: ALL.

  7. Select Apply to save your settings and Close to dismiss the Options tab.
  8. Ensure the View > Check Text Pane option is selected. Calibre RVE then displays the Density RDB files as a DENSITY_RDBS rule check with RDB files listed as links in the check text pane:


  9. Select the filldensity3.rdb link. The RDB results file opens in a new tab.


    The results table shows the following columns by default. Areas are in square um.

    # — Result number.

    #PA — Result polygon area.

    #PP — Result polygon perimeter in um.

    DA — Density Area property. This is the area of the data capture window. Maximum area is the Density WINDOW value squared.

    DA fill — Area of layer “fill” in the data capture window.

    DA keepout — Area of layer “keepout” in the data capture window.

    DV — Value of the Density operation expression, which is compared to the operation’s constraint.

    Coordinates — x and y coordinates of the vertices.

    Vertices — Vertex count.

  10. Select a result from the table and press H on your keyboard. Adjust the visibility of layers as necessary so you can see the properties associated with the result.


    The annotations appear because of the selections made in Step 6.

  11. Right-click the DV column header in the Calibre RVE results table and select Histogram DV. This opens a histogram of the density operation values that meet the operation’s constraint.

    Typically, the closer to 0 the DV value becomes, the more severe the problem.



  12. Right-click in the histogram plot area and select Show Range Controls.


    The control panel appears at the bottom of the histogram pane. These controls allow you to adjust the appearance of the histogram. In this case, the color order is inverted:



    Unselect Show Range Controls from the right-click menu to dismiss the control panel.

  13. Right-click in the histogram plot area as in the previous step and select Show Color Map. The layout highlights with all density results from the RDB with colors assigned from the histogram plot.


    Turning off highlighting of Check Names, Result Numbers, and Properties from Step 6 can give a clearer layout presentation when overlapping windows are displayed.

  14. Double-click any histogram bar and the corresponding results highlight in your viewer with a color that matches the bar.
  15. Right-click any histogram bar and a menu of options displays from which you can select.

Results

You can use these features to determine how to fix density issues in your design. You can also use the features to help you to adjust your Density layer operations to produce more meaningful results.
