Density Operation Categories

This section presents typical Density checks.
Table 1. Density Operation Solutions

Name

Description

Finding Density Outside of a Cell or Layer

Calculate densities of a layer while avoiding a cell or another layer. This is particularly useful when you have placed IP blocks that you want to avoid when performing density checks at a higher level.

Finding Density Inside of a Layer

Calculate densities of one layer that is inside of another layer.

Finding Density Inside of a Layer and Outside of a Cell

This is a combination of the two preceding methods.

Reducing RDB Data Size with Density COMBINE

Reduce RDB data size by using the COMBINE option to merge results selectively.

Using INSIDE OF LAYER Options

Discusses the differences in the INSIDE OF LAYER options. In general, INSIDE OF LAYER should be avoided in favor of the preceding solutions in this table.
