DRC HTML Report for Calibre PERC LDL CD

The following example shows a DRC HTML report created for a Calibre PERC LDL CD results database.

The report creates a histogram and colormap of the ERROR(%) property for one check. The report also includes a table of results grouped by check name, then sorted by the ERROR(%) value, with only the results with the three highest values of ERROR(%) shown for each check. A portion of the report is shown in Figure 1.

Figure 1. DRC HTML Report for PERC LDL CD

The following sections describe the command line and input files used to create the report. In contrast to Calibre RVE for PERC, you must process the RDB created by the PERC LDL run; you cannot process the DFM database produced by the run.

Command Line

calibre -rve -drc -input simple_batch_path.txt -report rpt_perc_cd.ini
	‑outputdir batch_out

Input Files

simple_batch_path.txt — A DRC HTML Report Database Map File. This file specifies the DRC Results Database, the layout database, and a Calibre DESIGNrev layer properties file, as shown here: 

simple.perc.report.cd.rdb,,simple.calibre.db,simple.calibre.db.layerprops

rpt_perc_cd.ini — The DRC HTML Report Configuration File. The text for this file is described in the section “Configuration File”.

simple.perc.report.cd.rdb — A DRC Summary Report created by the Calibre run that created the results database.

Output Directory and Files

The HTML report is named index.htm and created in the output directory batch_out in this example. All supporting files are located in the output directory.

Configuration File

[Common]
;DrvOptionsFilePath = ../
;UseFrames = 1
;
[Summary]
ReportType = DRCResultsSummary
GroupBy = Check
[histogram ]
ReportType = DRCHistogram
GroupBy = Check
Property = ERROR(%)
OrCheckFilter = rule_1.by_source.shorted
[colormap ]
ReportType = DRCColormap
GroupBy = Check
Property = ERROR(%)
OrCheckFilter = rule_1.by_source.shorted
[Results]
ReportType = DRCResults
GroupBy = [Summary]
SortBy = ERROR(%)
; include only three highest ERROR(%) values for each check in the table
SortOrder = Descending
RowCount = 3
