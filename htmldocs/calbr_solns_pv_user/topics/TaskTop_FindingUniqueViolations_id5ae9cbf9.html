Finding Unique Violations

You can use DFM Property and DFM Classify to output only unique violations from a set of DRC violations. The example constructs error marker polygons using an External operation. The rule check outputs only one error marker from a set of redundant violations. The redundant violations occur when more than one spacing violation occurs between the same two polygons. This is useful for reducing the number of DRC results.

This example is in the miscellaneous_checks/unique_violations example directory.

Procedure

  1. Derive spacing error markers between the polygons on METAL1.
    space_marker = EXT METAL1 <= 0.1 ABUT REGION
  2. Generate a unique property set on each METAL1 polygon.

    The properties proph and propv contain unique coordinate values for each polygon.

    corners = INT METAL1 < 0.05 ABUT REGION 
epairsh = DFM SPACE corners <= 0.05 BY INT HORIZONTAL 
epairsv = DFM SPACE corners <= 0.05 BY INT VERTICAL 

METAL1_props = DFM PROPERTY METAL1 epairsh epairsv OVERLAP ABUT ALSO 
               [proph = MIN(ECMIN(epairsh))] 
               [propv = MIN(ECMIN(epairsv))]
  3. Copy the property sets on each METAL1 polygon to the spacing error marker that touches it, assuming that each error marker only touches two METAL1 polygons.
    space_props = DFM PROPERTY space_marker METAL1_props 
                 OVERLAP ABUT ALSO MULTI 
                 [proph1 = PROPERTY(METAL1_props,proph,1)] 
                 [proph2 = PROPERTY(METAL1_props,proph,2)] 
                 [propv1 = PROPERTY(METAL1_props,propv,1)] 
                 [propv2 = PROPERTY(METAL1_props,propv,2)]
  4. Use DFM Classify to select one error marker per group of error markers with the same set of property values. 
    space_unique = DFM CLASSIFY space_props 
                PROPERTY "proph1" "proph2" "propv1" "propv2" 
                ABSOLUTE 0.001 UNIQUE KEEPPROPS

    UNIQUE KEEPPROPS copies properties to the output layer, which may be useful for debugging.

  5. Output the results with a DFM RDB operation and also write results to the standard DRC results database with a Copy operation. 
    unique_violations {
    COPY space_unique

// Set default options for DFM RDB operations 
   DFM DEFAULTS RDB CELL SPACE ALL CELLS CHECKNAME "%_l_"
   DFM RDB space_unique eqdrc.rdb 
   // for debug 
   DFM RDB METAL1_props eqdrc.rdb 
   DFM RDB space_props eqdrc.rdb  
}
