Derivation of Empty Layers

Empty layers can be derived in the rules. The following examples show the three possible derived layer types. 
// empty polygon layer
empty_shape = AREA(EXTENT) == 0

// empty edge layer
empty_edge = LENGTH(EXTENT) == 0

// empty error cluster layer
empty_error = EXT empty_shape < 1

These layer derivations can be used inside rule checks if desired; however, empty_error can only be passed to a layer operation that accepts error layers as input. An empty output layer is permitted in the DRC Results Database or in a DRC Check Map database.
