Electrical Rule Checks

Many Electrical Rule Check (ERC) problems can be solved using Calibre PERC. Detection of ERC problems includes Electrostatic Discharge (ESD) checks, Electrical Overstress (EOS) checks, multiple power domain crossing checks, voltage dependency checks, point-to-point current density and resistance checks, among other circuit reliability verification checks. It is advantageous to check for ERC issues in your source netlist to prevent problems in the layout and in the manufactured design. Most of these checks can also be performed on a layout netlist. Checks requiring parasitic resistance information can only be performed on the layout.

For complete information on the Calibre tool used for netlist-based circuit reliability checking, see the Calibre PERC User’s Manual.

For working code examples, see the shared/examples/calibre_perc directory in your Calibre software tree.

  • MOSFET Bulk Pin Connections
  • MOSFET Power Domain Connections
  • Back-to-Back Diode Ground Domain Separation
  • ESD Device Protection of I/O Pads
  • ESD Resistor Protection of Gates
  • Pin Voltage Checks
  • Current Density and Resistance Checks
