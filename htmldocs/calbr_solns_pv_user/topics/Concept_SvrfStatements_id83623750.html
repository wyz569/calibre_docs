SVRF Statements Sensitive to Rotation

Some SVRF statements are sensitive to rotation in general while others are sensitive to potential grid snapping induced by rotation.

There is also a special case when dealing with 1 dbu long edges. For example:

X = EXT Y < 0.1 ABUT == 90 REGION

When one of the edges of Y is >= 0.1um long and the other edge is 1 grid point long, the resulting region is preserved when the long edge is horizontal. When the short edge is horizontal, Calibre rounds away the error by increasing the precision. This prevents numerical precision limitations from causing erroneous output.
