Checking Equal Gate Pitch per Active Area

You can identify active area (AA) regions where gates are not equally spaced apart, have unequal widths, or both.

Figure 1. Gate Pitch

    Try It!

Calibre Advanced DRC (eqDRC) Tutorial and Example Kit

Go to this page on Support Center to download the complete eKit.

This example is in the grid_pitch_checks/equal_gate_pitch_per_aa example directory.

Procedure

  1. Derive gate spacing and width edge pairs:
    VARIABLE gate_spc 0.4   // max gate to gate spacing
VARIABLE gate_wid 0.2   // max gate width
spc = EXT gate < gate_spc OPPOSITE 
wid = INT gate < gate_wid OPPOSITE
  2. Determine active areas with unequal pitch. To do this, use a DFM Property operation with AA as the primary layer, and spc and wid as the secondary clustering layers:
    pitcherr_aa = DFM PROPERTY AA spc wid OVERLAP MULTI
                  [- = DRC_NE(MAX(EW(wid)), MIN(EW(wid))) 
                       + DRC_NE(MAX(EW(spc)), MIN(EW(spc))) ] != 0
                  [spc_diff = MAX(EW(spc)) - MIN(EW(spc))]
                  [wid_diff = MAX(EW(wid)) - MIN(EW(wid))]

    The rule check works as follows:

    • Use the MAX and MIN measurement functions to find the maximum and minimum width and spacing in the active area AA.

    • Use the DRC_NE comparison function to determine if the maximum and minimum width and spacing values are not equal. DRC_NE returns 1.0 if the arguments are not equal within a built-in tolerance, and returns 0 otherwise.

    • If either DRC_NE function returns 1.0, output the result, which is the AA layer polygon.

  3. (Optional) You can use a DFM Property Select Secondary operation to output the spc or wid error layer as a result rather than the AA layer polygon. 
    pitcherr_wid = DFM PROPERTY SELECT SECONDARY AA wid OVERLAP MULTI 
                   [ - = DRC_NE(MAX(EW(wid)), MIN(EW(wid))) ] > 0 
                   SELECT wid
                   [ wid = EW(wid)]
    
pitcherr_spc = DFM PROPERTY SELECT SECONDARY AA spc OVERLAP MULTI 
                   [ - = DRC_NE(MAX(EW(spc)), MIN(EW(spc))) ] > 0 
                   SELECT spc
                   [ spc = EW(spc)]
  4. Output the results using DFM RDB.
    // Set default options for DFM RDB operations
DFM DEFAULTS RDB CELL SPACE ALL CELLS CHECKNAME "%_l_" 
gate_pitch_active { 
   DFM RDB pitcherr_aa advDRC.rdb 
   DFM RDB pitcherr_wid advDRC.rdb 
   DFM RDB pitcherr_spc advDRC.rdb 
}
