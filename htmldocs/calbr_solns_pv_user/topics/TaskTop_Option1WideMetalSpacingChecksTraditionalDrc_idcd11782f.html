Writing Wide Metal Spacing Checks With Traditional DRC

Commonly, addressing wide metal spacing checks involves a series of single-dimensional design rules. A minimum space value is associated with each range of widths for a single band and both bands.

Conceptually, the spacing rules are described as follows:

Rule1: 0.09 minimum M1 Space if both bands >= 0.09 wide 
Rule2: 0.25 minimum M1 Space if one band   >= 0.36 wide 
Rule3: 0.30 minimum M1 Space if both bands >= 0.36 wide 
Rule4: 0.40 minimum M1 Space if one band   >= 1.44 wide 
Rule5: 0.50 minimum M1 Space if both bands >= 1.44 wide 
Rule6: 0.55 minimum M1 Space if one band   >= 4.50 wide 
Rule7: 0.60 minimum M1 Space if both bands >= 4.50 wide

Procedure

Write the wide-metal spacing rules to check a single band and both bands:
wide009 = Metal WITH WIDTH < 0.36 
wide036 = Metal WITH WIDTH >= 0.36 < 1.44 
all144 = wide009 OR wide036 
wide144 = Metal WITH WIDTH >= 1.44 < 4.50 
all450 = all144 OR wide144 
wide450 = Metal NOT WITH WIDTH < 4.50  
rule1_err { EXT wide009 < 0.09 OPPOSITE } 
rule2_err { EXT wide036 wide009 < 0.25 OPPOSITE } 
rule3_err { EXT wide036 < 0.30 OPPOSITE } 
rule4_err { EXT wide144 all144 < 0.40 OPPOSITE } 
rule5_err { EXT wide144 < 0.50 OPPOSITE } 
rule6_err { EXT wide450 all450 < 0.55 OPPOSITE } 
rule7_err { EXT wide450 < 0.60 OPPOSITE }

Results

The following analysis shows that the effectiveness of this approach can limit the erosion effects.

The following figure shows a case where both bands (or wires) are 1.44 units wide, and the separation distance is 0.49 units. This design fails rule 5.

Figure 1. Case 1

The following figure shows a case where the bands are 1.44 and 1.43 units wide, and the separation distance is 0.4 units. Rule 4 now applies instead of rule 5, and the design passes.

Figure 2. Case 2

Considering erosion susceptibility, the failing case shown in Figure 1 is actually better than the passing case shown in Figure 2. In the passing case, since the oxide area is smaller, more erosion will occur.

In Figure 3, one band has a width of 1.44 units, but the other band is only 0.09 units wide. Since one band is 1.44 units, the design fails rule 4 and the designer must increase the separation distance. Again, from a standpoint of susceptibility to erosion, Figure 3 is much better than Figure 2.
