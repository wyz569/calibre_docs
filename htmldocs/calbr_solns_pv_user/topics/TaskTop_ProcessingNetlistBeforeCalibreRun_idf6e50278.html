Processing a Netlist Before the Calibre Run

You can use Calibre Interactive to specify a trigger function that runs just before the Calibre run starts. This example defines a trigger function to process the source netlist. The Calibre Interactive internal trigger runs after source and layout export completes and before the Calibre nmLVS run starts.

Here are the desired actions for this Calibre nmLVS scenario:

  • Export the source and layout from the design tool.

  • Run a script to process the source netlist.

  • Run Calibre nmLVS.

  • Start Calibre RVE.

You can do this using Calibre Interactive with an internal trigger. The name of the source netlist is given to the trigger function as a trigger parameter. The following steps are needed to accomplish this:

  1. Create the shell script to process the source netlist.

  2. Specify the shell script on the Setup > Preferences > Triggers tab in Calibre Interactive.

  3. Set up the Calibre Interactive runset.

  4. Open your design and start the Calibre Interactive run.

The following procedure describes these steps in detail, including code examples.

Procedure

  1. Create the shell script for editing the netlist file. This example takes the netlist name as an argument and creates a copy. 
    #!/bin/sh

echo "executing internal trigger, copy source netlist"
# expects source netlist filename as argument
echo "source netlist is: $1"
cp $1 $1.copy

    Save the file and name it copy_netlist.sh.

  2. Open Calibre Interactive nmLVS.
  3. Do the following to specify the shell script copy_netlist.sh as a trigger function.
    1. Select Setup > Preferences, then choose the Triggers tab.
    2. In the Pre-Execution tab, click the Add Triggers button () to add a new trigger function.
    3. Enter the command and any arguments in the Trigger column. 
      copy_netlist.sh %s

      In this case the argument is the trigger parameter %s, which is replaced with the path to the source netlist at runtime. Available trigger parameters are listed on the Triggers tab.

    4. Click the checkbox in the Use column to enable the trigger.

  9. Start Calibre Interactive nmLVS and load the runset you saved in Step 7.
  10. Click Run LVS to start the run.

Results

The following actions occur in order when you click Run LVS:

  • The layout and source are exported from the design tool.

  • The shell script copy_netlist.sh executes.

  • The Calibre nmLVS run starts.

  • Calibre RVE opens.
