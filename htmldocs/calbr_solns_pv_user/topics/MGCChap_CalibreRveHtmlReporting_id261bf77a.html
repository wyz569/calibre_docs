Calibre RVE HTML Reporting

HTML reporting in Calibre RVE for DRC creates HTML pages corresponding to views in Calibre RVE. You can specify views of DRC result tables, result highlights, histograms, and colormaps. Controls are included for grouping, filtering, and sorting the results, and for customizing the report format.

DRC HTML reporting can simplify the communication and approval process within a design team and between the design team and external groups (such as the foundry) due to the following advantages:

  • HTML reports do not require manual processing once the configuration file is developed, so that the report creation process can be automated.

  • HTML reports are easy to share, and do not include access to any design data. Only the layout images you want to share are included.

  • HTML reports can be customized with a Cascading Style Sheet (CSS) or an HTML customization file, as described in “Output Format and Customization for DRC HTML Reports” in the Calibre RVE User’s Manual.

  • HTML reports can be automatically generated from Calibre Interactive nmDRC or Calibre RVE for DRC. When creating the report from Calibre RVE, the DRC HTML report is based on the view settings in your Calibre RVE session. See “Automatically Creating the DRC HTML Report” in the Calibre RVE User’s Manual.

For complete information on the configuration file format and command line for DRC HTML reporting, see “DRC HTML Reporting” in the Calibre RVE User’s Manual. The referenced section also includes an extended example and tips for troubleshooting.
