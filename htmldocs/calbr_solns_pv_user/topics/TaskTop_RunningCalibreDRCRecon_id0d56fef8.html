Running Calibre nmDRC Recon

Run Calibre nmDRC Reconnaissance mode to automatically execute a subset of rule checks in a rule file.

Prerequisites

  • A layout design.

  • A rule file, typically a foundry deck.

  • A Calibre RealTime Digital license and sufficient Calibre nmDRC/Calibre nmDRC-H license pairs for the run configuration.

Procedure

  1. (Optional.) For portions of the design you want to ignore, specify either of the following:
    LAYOUT WINDEL x1 y1 x2 y2
...
LAYOUT WINDEL CELL my_cell ORIGINAL
...
  2. (Optional.) Specify DRC Recon Add Check, DRC Recon Remove Check, DRC Recon Add Check By Layer, and DRC Recon Remove Check By Layer statements in the rule file to control the check set for the run.

    See “Selecting Checks in Calibre nmDRC Recon” for an example.

  3. To run Calibre nmDRC Recon mode, use the following command line (csh assumed):
    % calibre -drc -hier -turbo -recon rules >&! log

    Include the -hyper and -remote or -remotefile options as supported by your hardware. To run the set of checks that -recon does not run, specify “-recon inverse”.

    The -recon run options are also available from the Calibre Interactive nmDRC Inputs pane.

Results

The output files include the run transcript, a DRC Summary Report if specified, and a DRC Results Database. The database can be opened with Calibre RVE for cross-probing results into your layout editor.
