Finding Circular Polygons

You can use a DFM Circle Analyze or Inside Cell CIRCLE operation to find approximately circular shapes in a layout.

  • DFM Circle Analyze — For each polygon on the input layer, the operation finds the minimum enclosing circle C-out and the largest circle C-in that is contained within the polygon and concentric with C-out. The radii of these two circles are reported as DFM properties named OUTRADIUS and INRADIUS attached to each shape on the output layer.

    The OUTRADIUS and INRADIUS properties can be used in a DFM Property operation to select shapes that meet a desired criteria.

    DFM Circle Analyze merges the input layer before shapes are analyzed.

  • Inside Cell CIRCLE — The CIRCLE keyword causes the Inside Cell operation to return shapes that meet a built-in criteria. In particular, the angle between all polygon vertices must vary by less than 1%, and the distance between the center of the polygon extent and every vertex must vary by less than 1%. See the command definition for additional criteria.

    Inside Cell operates on unmerged original layer input, so it can output drawn circular shapes that overlap other shapes on the same layer. However, the output is merged, so overlapping shapes are merged on output.

The circle criteria for Inside Cell is built-in and cannot be changed, and is also very strict. When using DFM Circle Analyze, however, you can use the OUTRADIUS and INRADIUS properties to develop your own criteria for a circular shape.

    Try It!

Calibre Advanced DRC (eqDRC) Tutorial and Example Kit

Go to this page on Support Center to download the complete eKit.

This example is in the miscellaneous_checks/finding_circles example directory.

Procedure

  1. Use DFM Circle Analyze to attach the OUTRADIUS and INRADIUS properties to the output polygon layer. Use a DFM Property operation to select polygons with a ratio INRADIUS/OUTRADIUS > 0.9.
    circle_props = DFM CIRCLE ANALYZE MET1
circles_ratio_90 = DFM PROPERTY circle_props 
              [ - = PROPERTY(circle_props, INRADIUS)/
                    PROPERTY(circle_props, OUTRADIUS)] > 0.90
         // for 2020.3 and later, prevent selecting annular shapes 
              [ - = PROPERTY(circle_props, ANNULUS_INRADIUS)] == 0
  2. Use Inside Cell to select original layer polygons that meet the built-in CIRCLE criteria. 
    circles_inside_cell = INSIDE CELL MET1 "*" CIRCLE
  3. Output the results with a DFM RDB operation.
    circles {
    DFM RDB circles_ratio_90 circles.rdb CHECKNAME "%_l_" 
        CELL SPACE ALL CELLS
    DFM RDB circles_inside_cell circles.rdb CHECKNAME "%_l_" 
        CELL SPACE ALL CELLS
}
