Adding Constraints to Pattern Edges and the Output Marker

You can add constraints to a pattern to allow for variations in edge location. You can also constrain an output marker to be a fixed distance from a pattern edge that has constraints.

This procedure is similar to “Adding Layout Shapes Using Pattern Marker Output” and assumes you have the pattern library created for that task.



Pattern matching generates a polygon layer with the required layout corrections.

These videos demonstrate the steps in this procedure:

“Pattern Editing: Create Single Edge and Edge to Edge Constraint”

“Pattern Editing: Add Constraints to Markers”

Prerequisites

  • A pattern library with a pattern that matches the area of interest. Begin with the pattern created for “Adding Layout Shapes Using Pattern Marker Output”, which includes a custom marker.

  • A layout in GDS or OASIS format. The layout should be suitable for testing with your pattern library.

Procedure

  1. Open the pattern library in the Calibre Pattern Matching GUI:
    calibrepm lib.pmdb
  2. Edit the pattern to add single edge constraints to edges A and C.
    1. Select the pattern in the pattern list in the lower left of the GUI.
    2. In the toolbar above the pattern canvas, click the button to enter “Add Single Edge Constraint” mode.

      Click edge A (it will turn green), then move the cursor to the right edge of the constraint range and click, then to the left and click.

      Repeat for edge C.

    3. Click the Constraints tab at the bottom of the panel at the right of the GUI to view a listing of the two constraints just added.
  3. Add an edge to edge constraint between edges A and C, to indicate the constraint on the edge separation.
    1. Click the button to enter “Add Edge to Edge Constraint” mode.
    2. Click edge A.
    3. Click edge C.

      The Add E2E Constraint dialog box opens. Add a label if desired.

    4. Change the constraint values to read “Between 0.1 … Left” and “And 0.1 … Right,” and uncheck the “Inc.” (inclusive) box for both values. This specifies that the second edge (or its projection) can be up to 0.1 microns to the left or right of the first edge.


  4. Constrain the left edge of the custom marker to coincide with edge C:
    1. In the Constraints tab at the right of the GUI, click the icon to hide the C1, C2, and D1 constraints so they are not displayed. This makes it easier to see the pattern edges.
    2. Click the button to enter “Add Edge to Edge Constraint” mode.
    3. Click twice at the edge C location—once on the marker edge and once on the pattern edge.

      The Marker constraint M1 is entered in the Constraints tab, with parameters “Loc = e1”, indicating that the marker edge is coincident with pattern edge C.

  5. Constrain the right edge of the custom marker to be 0.1 microns to the right of edge A.
    1. Click the button to enter “Add Edge to Edge Constraint” mode.
    2. Click the right edge of the custom marker.
    3. Click edge A.

      The Marker constraint M2 is entered in the Constraints tab, with parameters “Loc = 0.1 + e1”, indicating that the marker edge must be the 0.1 microns to the right of pattern edge A.



  6. Click in the top toolbar to save the library.
  7. Use the GUI to run pattern matching with the library.
    1. Click the button in the top right of the GUI to start a pattern matching run with the open library.
    2. Select Layout File as the layout source and fill in the text field with the path to the layout.
      Tip

      You can also run on an open layout by selecting Active Layout as the layout source. See “Running Pattern Matching From the GUI” in the Calibre Pattern Matching User’s Manual.

    3. In the layers table, make sure the library layers correspond to the correct layer number in the layout. If not, click in the Layout Layers cell and select the correct layer number.

      Other options can remain at their default settings.

    4. Click Run.

    The transcript is displayed, along with a list of the files created by the run.

  8. Examine the results of the pattern matching run:
    1. Open the DRC results by clicking the folder icon in the DRC RDB row.


      The layout is opened in Calibre DESIGNrev and the DRC results database is opened in Calibre RVE.

    2. In Calibre RVE, select the pm_m_edit_out check and type “H” on the keyboard to highlight all results.

      The custom marker shapes are highlighted. Confirm that the shapes make the layout modifications that you want.

  9. (Optional) If you edited the library created with the procedure “Adding Layout Shapes Using Pattern Marker Output”, you can use the same updated rule file (edit_layout.svrf) to write the layers to an OASIS file.

    The pattern matching run in Step 7 generates a new compiled library named lib.svrf, which is used in the pattern matching rule file.

Results

The layout shown Figure 1 with the procedure “Capturing a Pattern From the Layout” has four locations that match the pattern. The two locations on the left match without constraints on the pattern edges. The two locations on the right match with constraints. The marker constraints ensure that one edge of the custom marker output is coincident with pattern edge C and the other edge is 0.1 microns from pattern edge A. As a result, the length of the custom marker output varies with the position of edges A and C.
