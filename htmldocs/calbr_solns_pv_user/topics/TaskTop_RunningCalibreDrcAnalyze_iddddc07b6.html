Running Calibre DRC Analyze

Run Calibre DRC Analyze to get error distribution results data for gross analysis of areas of a design that require attention.

Prerequisites

  • A layout design.

  • A rule file, typically a foundry deck.

  • A Calibre RealTime Digital license and sufficient Calibre DRC/Calibre DRC-H license pairs for the run configuration.

Procedure

  1. (Optional.) For portions of the design you might wish to ignore, specify either of the following:
    LAYOUT WINDEL x1 y1 x2 y2
...
LAYOUT WINDEL CELL my_cell ORIGINAL
...
  2. To run the complete set of rule checks, use the following command line (csh assumed):
    % calibre -drc -hier -turbo -analyze rules >&! log

    Include the -hyper and -remote or -remotefile options as supported by your hardware.

  3. (Optional.) To run a more fundamental set of checks, add the -recon option:
    % calibre -drc -hier -turbo -analyze -recon rules >&! log

Results

The output files include the run transcript, a DFM Summary Report called dfm.rep, and a DFM Database called analyze.dfmdb. The database can be opened with Calibre RVE for cross-probing results into your layout editor.

A DRC Summary Report is not produced; see the dfm.rep file instead. The DRC Results Database is empty. The analyze.dfmdb database contains all results.
