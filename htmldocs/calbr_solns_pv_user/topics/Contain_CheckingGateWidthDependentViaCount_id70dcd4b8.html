Checking Gate Width Dependent Via Count

You can identify via transitions that have an insufficient number of redundant vias to handle the current for the total gate width being driven by a particular metal line. You can do this by writing a rule check that verifies there is at least one via per unit gate width.
Figure 1. Gate Width Dependent Via Count

There are several different ways to approach the problem. Performance between these approaches can vary based on factors including design size and hierarchy scheme, reuse of derived layers and Connect statements elsewhere in the rule file, and whether incremental connectivity is enabled. You can test all three implementations and choose the one with the best performance for your design data.
