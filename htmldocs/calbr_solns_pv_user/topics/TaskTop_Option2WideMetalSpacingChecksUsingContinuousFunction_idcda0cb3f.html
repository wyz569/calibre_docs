Writing Wide Metal Spacing Checks Using a Continuous Function

You can write a wide metal spacing check by implementing a continuous function in terms of the width and space values.

To do this, find a curve of best fit to the discrete values defined by the traditional DRC rules, as shown in the following figure:

Figure 1. Traditional DRC Versus eqDRC

Note:

There are intervals that extend to the right of the stair step pattern for each minimum space value on the traditional DRC plot. This is caused by how the traditional DRC rules are defined. There is no upper constraint on width for one band or both bands, only that the widths must be greater than or equal to a minimum value. Because of this, for each space value, the total metal width can continue to increase without violating any of the traditional DRC rules.

An equation can be obtained from the red curve in terms of the two variables—the separation distance (space) and the total width of the bands (width1 plus width2). The equation can be put in the form of an error condition, where the check fails if the result is greater than 1:



The equation can be manipulated to find the change in spacing or total width needed to pass the check:

This example is in the device_routing_checks/wide_metal_spacing example directory.

Procedure

  1. Create a derived error layer where the edge clusters mark the separation distance between the bands:
    m_space = DFM SPACE Metal <= 0.60 BY EXT

    This layer is used as the primary input layer to the DFM Property operation in Step 3.

  2. Create a derived error layer where the edge clusters mark the width of each band: 
    m_width = DFM SPACE Metal <= 5.00 BY INT
  3. Create a derived error layer eqdrc_equation with three properties attached: error_value, increase_space_by, and reduce_sum_width_by:
    eqdrc_equation = DFM PROPERTY m_space m_width OVERLAP ABUT ALSO
                MULTI SPLIT ALL 
[error_value = 0.09/(EW(m_space)*LOG(2)) * log(EW(m_width)/0.09)] > 1 
[increase_space_by = (0.09/LOG(2))*(EW(m_width)/0.09)-EW(m_space)] 
[reduce_sum_width_by = EW(m_width)-(0.09*EXP((EW(m_space)*LOG(2))/0.09))]

    The eqdrc_equation layer is the subset of the primary input layer m_space for which the property constraints are met. The EW measurement function returns the separation distance of the input error layer. Because m_width is a secondary layer, the EW values are automatically summed over secondary geometries in the cluster, so that EW(m_width) returns the total width of the two metal bands that abut the m_space error cluster.

  4. Output the eqdrc_err layer to a DFM RDB file named eqdrc.rdb.
    eqDRC_rule_eq { 
    DFM RDB eqdrc_equation eqdrc.rdb CHECKNAME "%_l_" 
       ALL CELLS CELL SPACE 
       COMMENT "Wide metal spacing Rule: Use the property values to
       determine the error magnitude and adjustments needed to
       fix the violation" 
}

The following figure shows a result highlighted in Calibre DESIGNrev for a pair of failing metal bands.
