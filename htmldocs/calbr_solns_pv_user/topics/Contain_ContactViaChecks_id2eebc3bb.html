Name

Description

Counting Contacts in Source-Drain and Gate Regions

Checking Contact Balance

Checks that source-drain regions on both sides of a gate have an equal number of contacts, but allows a difference of one contact if both source-drain regions each have at least two contacts.

DFM Function, DFM Property, DFM RDB

Verifying Contact Run Length for Current Crowding

Quantifies current crowding through a gate based on the run length of contacts on each side of the gate and identifies gates that have the worst current crowding effects.

Coincident Edge, DFM Property, DFM RDB, External, Length

Verifying Contact Placement for Current Crowding

Checks for current crowding effects based on the number of contacts and their positions in the source and drain regions.
