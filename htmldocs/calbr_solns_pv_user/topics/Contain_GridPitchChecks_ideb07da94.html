This section includes grid and pitch checks.
Table 1. Grid and Pitch Check Solutions

Name

Description

SVRF Elements

Checking Polygon Centerline Alignment

Polygon centerlines must be aligned, where one polygon, such as a marker shape, overlaps another polygon from a different layer.

AND, NOT, Coincident Outside Edge, DFM Property, DFM DRB

Checking Equal Gate Pitch per Active Area

All gates in an active area region must have equal widths and must be equally spaced apart.

External, DFM Property, DFM DRB, Internal

Checking Gate Relative Pitch per Active Area

The centerline-to-centerline spacing of gates within the same active area must be spaced with an integer pitch value relative to each other.

AND, Variable, DFM Space, DFM Copy, DFM Property, DFM RDB

Checking Metal Routing Relative Pitch

The centerline-to-centerline spacing of routing wires must be spaced with an integer pitch value relative to each other.

Variable, DFM Space, DFM Copy, DFM Property, DFM RDB

Checking Contact Relative Pitch

The center points of contacts must be spaced with an integer pitch value relative to each other.

Variable, Extents, Grow, Offgrid, DFM RDB
