Reviewing DRC Analyze Results

Calibre RVE for DFM allows review of DRC Analyze results. The presentation is similar to DFM Report Card and Density RDB results. This topic shows some of the more commonly used features.

Prerequisites

  • A layout design and viewer.

  • An analyze.dfmdb results database.

  • A Calibre RVE license.

Procedure

  1. Open the design and Calibre RVE using your layout viewer.
  2. Open analyze.dfmdb in Calibre RVE using File > Open Database. Ensure the Database Type is DFM.

    (Optional.) This command line can be used to start the review session in Calibre DESIGNrev:

    calibredrv design.oas -rve -dfm analyze.dfmdb

    Calibre RVE opens with the Totals tab selected:



    The data is organized by rule check group in descending order of total result count.

  3. In the Totals tab, select View > Bar Chart.

    The data is presented in bar chart form:



  4. In the Totals tab, select View > Table to restore the tabular form.

    The table and bar chart data viewing modes are available in all tabs with rule check statistics.

  5. Select the Chip Summary tab.


    The data is organized by rule check name in descending order of result count. The rule check group names are also displayed.

  6. In the Chip Summary tab, choose a rule check name, right-click on it, and select Histogram Hierarchy Windows > Count.


    A panel opens with the rule check count data arranged by histogram bin:



    The bins comprise result count intervals of increasing values from left to right. Each bin corresponds to a set of data capture windows in the layout extent.

  7. In the histogram panel, right click in one of the bars and select Highlight This Bar.


    The corresponding data capture windows highlight with the same color in the layout:



  8. Repeat step 6 using the Colormap Hierarchy Windows > Count menu item instead.

    A panel opens with the rule check count data arranged by data capture window:



    The Scale represents results count per window in an increasing “temperature” color spectrum.

  9. In the colormap panel, right-click and select Colormap Windows.


    The data capture windows overlay the layout using the colormap pattern:



  10. Choose one of the data capture window boxes in the colormap panel. Right-click in that box and select Highlight Errors.


    The corresponding rule check errors highlight in the layout:



    Zoom in to the error region using your layout viewer control to see the details.

  11. In the Chip Summary tab, choose a rule check name, right-click on it, and select Browse Hierarchy Errors > Count.


    A rule_check_name.rdb tab opens:



    This is the usual ASCII RDB presentation for DRC results. You can use this to highlight individual results or groups of results in the layout.

  12. In Calibre RVE, select the Cell Summary tab.


    The data is arranged by cell name in decreasing order of results count. Cell area in um2 and instance count are given.
