Viewing and Editing Text Files in Calibre Interactive

The Calibre Interactive text editor lets you view and edit text files without leaving the application. Most of the files you use with Calibre Interactive are text files; these include control files, runsets, rule files, transcripts, and summary reports.

Procedure

  1. Choose File > View Text File.
  2. Enter the filename in the File Path field.
  3. Click OK to open the view window.

    By default, the text editor opens in view-only (read-only) mode.

  4. (Optional) Enable editing with one of the following:

    • Click the Edit button in the bottom of the text editor window. The Edit button is green when editing is enabled and red when editing is not enabled.

    • Choose Options > Edits Are, and enable Allowed.

  5. Use the Edit and Options menus for standard editing functions, to open a search dialog box, and to control the view.
  6. If you made changes, choose File > Save to save the file.
    Note:

    If you edit a rule file, you must save the file and reload it in order for the changes to take effect for the next Calibre run.

    Tip

    If you want the text editor to open with editing enabled by default, choose Setup > Preferences, click the Misc tab, and enable “Enable editing for text file viewer.”
