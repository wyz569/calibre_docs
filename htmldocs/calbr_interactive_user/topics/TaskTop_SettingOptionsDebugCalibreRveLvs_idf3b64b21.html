Setting Options for Debug with Calibre RVE for LVS

You can use Calibre Interactive nmLVS to set up the run so that you can view the results in Calibre RVE for LVS. Certain settings are needed to enable various debug features.

Prerequisites

  • A rule file for Calibre nmLVS.

  • Case sensitivity is handled correctly; see “Case Sensitivity in Calibre RVE” in the Calibre RVE User’s Manual for information.

Procedure

  1. Open Calibre Interactive nmLVS; see “Using Calibre Interactive to Perform LVS”.
  2. Specify the output database (SVDB) as follows:
    1. Click Outputs on the left panel.
    2. Enable “Create SVDB database.”
    3. Specify the “SVDB Directory” pathname in the text entry field.

    This adds the following statement to the Calibre Interactive control file:

    Mask SVDB Directory svdb QUERY

    where svdb is the path to your SVDB directory. The QUERY option is specified by default in Calibre Interactive. You can specify additional options on the Outputs pane, if desired.

  3. (Optional) Enable short isolation and short verification as follows:
    1. Choose Setup > LVS Options to open the LVS Options pane.
    2. Enable “Run Short Isolation.”
    3. Enable the following settings: “Output shorts by layer,” “Output shorts by cell,” “Isolate shorts in all cells,” and “Between all names.”

    See the section “Invoking Short Isolation” for further details.

  4. (Optional) Calibre RVE for LVS can display Design Fix Suggestions for certain types of common LVS problems. Do the following to enable Design Fix Suggestions in the results database:
    1. Choose Setup > LVS Options to open the LVS Options pane.
    2. Click the Report tab.
    3. For the LVS Report Options selection, enable “FX=Show detailed error analysis.”
    Tip

    When using Calibre Interactive, the default GUI setting of “None” for LVS Report Options overrides your rule file settings unless you click Load on the Rules pane to load the rule file settings.

    Tip

    If your SVDB (standard verification database) takes a long time to load into Calibre RVE, remove options A, AV, C, and CV from LVS Report Option if they are present. These options increase the size of the SVDB. Much of the information from these options is reported in the DETAILED INSTANCE CONNECTIONS section of the LVS Report. Also make sure that LVS Report Maximum ALL is not used.

    For a complete description of setting up a Calibre nmLVS run, see the Calibre Verification User’s Manual.
