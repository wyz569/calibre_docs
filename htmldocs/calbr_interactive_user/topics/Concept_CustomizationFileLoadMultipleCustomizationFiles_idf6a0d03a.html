Customization File to Load Multiple Customization Files

You can use the Tcl source command to create a customization file that loads multiple customization files; the following example shows such a file.

Example

# customization file: custom_load_all 
source custom_1_drc
source custom_2_drc
source custom_3_drc

The customization file custom_load_all is specified in the same manner as any other customization file. For example, to specify the customization file when you invoke Calibre Interactive from the command line, use the following command:

% calibre -gui -drc -custom custom_load_all

Each customization file specified in custom_load_all is loaded sequentially into Calibre Interactive.
