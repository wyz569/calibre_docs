Invoking Calibre Interactive From a Layout or Schematic Viewer

When you start Calibre Interactive from a layout or schematic viewer, communication is automatically set up between Calibre Interactive and the viewer. You can specify that Calibre Interactive use the open layout or schematic as input for the Calibre run.

Prerequisites

  • A supported viewer with the interface to Calibre installed. See “Integration to Design Tools” for a list of design tools with a Calibre interface, and “Interfacing Calibre Interactive to Layout and Schematic Viewers” for instructions on setting up the interface to Calibre.

  • Calibre Interactive is installed. See “Calibre Interactive System Requirements”.

Procedure

  1. Open your layout or schematic in a supported viewer.
  2. Select the desired Calibre Interactive application. The menu selection for Calibre Interactive depends on the design tool.

    Design Tool

    Menu Selection

    Calibre DESIGNrev family of layout viewers

    Choose Verification, then choose the run type as shown in Figure 1.

    Cadence Virtuoso and other Cadence tools

    Choose Calibre, then the run type

    Synopsys tools

    Choose Calibre, then the run type.

    Other tools

    Choose Calibre, then the run type.

Results

Figure 1 shows the Verification menu for Calibre DESIGNrev; the Calibre menu for other viewers will have similar entries and may have a submenu for the application.

When Calibre Interactive opens, certain fields are filled in based on the cell and view open in the design tool; see “Templates for File Naming” for more information.
