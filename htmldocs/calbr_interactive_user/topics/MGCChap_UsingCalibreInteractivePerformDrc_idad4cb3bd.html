Using Calibre Interactive to Perform DRC

The Calibre Interactive nmDRC Graphical User Interface (GUI) is an interface to Calibre nmDRC. The Calibre Interactive - nmDRC window allows you to specify command line options and select rule file statements that guide Calibre nmDRC.
