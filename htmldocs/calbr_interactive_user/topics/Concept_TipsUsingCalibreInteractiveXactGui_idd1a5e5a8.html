Tips for Using the Calibre Interactive xACT GUI

A search page, rule file loading, and other features make the Calibre Interactive xACT GUI easy to use.
Note:

If a page is not visible on the left panel of the GUI, choose Settings > Show Pages and select the desired page.

Table 1. Tips for the Calibre Interactive xACT GUI

To do this …

Do the following:

Search the GUI

Enter a search term in the entry field at the top right of the GUI. Use Ctrl-f or Ctrl-F to change focus to the search entry box.

See “Search Toolbar in Calibre Interactive xACT”.

View the control file

Click the Run xACT button while pressing the Ctrl key, or select File > View Control File.

The control file is opened in the Files page.

Load the rule file

Click the button on the Rules page.

Note: The rule file settings overwrite current GUI settings, including the path to the input design and the top cell name.

Specify additional rule files and rule file statements

Click Options on the left panel, enable “Include Rules Files,” and expand the control area to specify the additional rule files.

To specify additional rule file statements, enable “Include Rule Statements” on the Options page.

By default, additional rule files and statements are added before the main rule file. To change this, enable “Include additional rules files and statements after main rules file” on the Preferences page in the Rules area.

Specify additional layout or schematic input files

Click Database on the left panel and expand the Library area.

Start Calibre® RVE™ after the run

Click Run Control on the left panel, expand the “RVE Options” area, and enable “Show Results in RVE.”

Load or save a runset

File > Load Runset or File > Save Runset, or use Ctrl-r to save a runset.

By default, only non-default options are saved to the runset. To save all runset options, set the environment variable MGC_CALIBRE_SAVE_ALL_RUNSET_VALUES to 1 before starting the GUI.

Change the font

Settings > Change Font

Hide and show pages

Select Settings > Show Pages

Use keyboard shortcuts

You can use a hotkey to go to some pages. See “Calibre Interactive xACT GUI Reference”.
Note:

The runset option naming convention for some options changed in the 2018.4 release. As of 2018.4, certain options are prefixed with “cmn” instead of “pex”. Runsets saved with Calibre 2018.4 or later may report errors if they are loaded with an earlier version of Calibre.
