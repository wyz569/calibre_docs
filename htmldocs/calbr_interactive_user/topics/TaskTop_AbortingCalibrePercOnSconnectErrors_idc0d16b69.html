Aborting Calibre PERC on Sconnect Errors

If you use Sconnect statements in your rule file and encounter errors during your run, you can have Calibre PERC abort automatically on soft connection conflicts.

Corresponding rule file statement: LVS Abort On Softchk.

Procedure

  1. Click PERC Options to display the Options pane. If the PERC Options button is not available, choose Setup > PERC Options.
  2. Click the Supply tab.
  3. Enable “Abort PERC on Softchk errors.”
