CUSTOM::getCalibreVersionString

Command type: Customization command for Calibre Interactive.

Gets the current Calibre version string.

Usage

CUSTOM::getCalibreVersionString

Arguments

None.

Return Values

Returns a string with the Calibre version being run by the Calibre Interactive session. The string has the format v2012.4_0.28.

Examples

set cal_version [CUSTOM::getCalibreVersionString]

See the command CUSTOM::setOKCallbackError for a extended example that uses the command CUSTOM::getCalibreVersionString.
