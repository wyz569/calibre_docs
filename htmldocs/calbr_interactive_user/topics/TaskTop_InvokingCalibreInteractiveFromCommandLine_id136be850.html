Invoking Calibre Interactive from the Command Line

Command line invocation of Calibre Interactive can open the GUI or start a Calibre Interactive batch run.

Prerequisites

  • Calibre Interactive is installed. See “Calibre Interactive System Requirements”.

  • Required input files are available. See “Files and Directories Used By Calibre Interactive”.

Procedure

  1. (Optional) Navigate to the run directory on your Calibre host computer.
  2. Open Calibre Interactive with one of the following methods. Except for the first example, the examples are for Calibre Interactive nmDRC—substitute the appropriate command line argument of ‑lvs, ‑dfm, ‑perc, ‑pex, -xact, or ‑3dstack to open a different application.
    Table 1. Invoking Calibre Interactive from the Command Line

    If you want to …

    Do this:

    Display the Calibre Interactive palette.

    		 
    calibre -gui


    Choose the application you want to run.

    Display the Calibre Interactive nmDRC window.

    		 
    calibre -gui -drc

    Open the Calibre Interactive nmDRC window with a runset automatically loaded.

    		 
    calibre -gui -drc -runset my_runset_file

    Run Calibre Interactive nmDRC in batch mode.

    		 
    calibre -gui -drc -runset drc_runset -batch
