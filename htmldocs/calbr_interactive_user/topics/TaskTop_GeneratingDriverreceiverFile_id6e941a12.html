Generating a Driver/Receiver File

You can use Calibre PEX to create a driver/receiver file for use during parasitic extraction.

Procedure

  1. Click the PEX Options button on the left panel, or choose Setup > PEX Options if the button is not present.
  2. Click the Misc tab.
  3. Enable the “Generate driver/receiver file” checkbox and enter the name of the file and appropriate tags in the “Driver/Receiver File” and “Driver/Receiver Tags” fields.

    See PEX Generate Driver File Tag in the SVRF Manual for complete details.
