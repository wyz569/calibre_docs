Run Control, Remote Execution and Licensing Options

You can control how your Calibre PEX job is run by configuring options related to performance, remote and distributed computing, and license acquisition.

These topics are covered in detail in “Run Control Setup and Licensing”. Also see “Controlling PEX Step Execution”.

You can select the Calibre Cell/Block license for Calibre xRC and Calibre nmLVS from the Environment > Licensing sub-tab of the Run Control pane. The following restrictions apply when using the Calibre CB license:

  • The Calibre xRC CB license option is appropriate only for flat PEX runs on designs containing up to 100,000 transistors.

  • The Calibre xRC CB license can only be used for flat transistor or gate level extraction.

  • The Calibre Cell/Block license options for Calibre nmLVS and Calibre xRC should be selected if SOURCE names are requested in a PEX run of Calibre Interactive and you only have Calibre Cell/Block licenses.

  • The Calibre Cell/Block license is used for Calibre xACT 3D or xACT 3D/Hybrid runs. The option is ignored if it is selected.

The ‑flatten command line option is used when Calibre nmLVS is invoked during a Calibre Interactive PEX run in the following cases:

  • “Run hierarchical version of Calibre-LVS” is not enabled on the Performance tab of the Run Control pane.

  • “Use Calibre-CB license for LVS” is enabled on the Environment > Licensing tab of the Run Control pane.
