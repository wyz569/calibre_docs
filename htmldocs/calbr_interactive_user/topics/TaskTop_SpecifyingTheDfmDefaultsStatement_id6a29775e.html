Specifying the DFM Defaults Statement

You can specify the statement DFM Defaults RDB {GDS | OASIS} using Calibre Interactive nmDRC. The setting is on the DRC Options pane on the Database > DFM Defaults sub-tab. You can set default options for the RDB filename, format, cell name prefix and suffix, and the TOP keyword.

Prerequisites

  • Calibre Interactive nmDRC is open.

  • (Optional) Set a file-naming template for the DFM Defaults RDB filename. Choose Setup > Preference, Templates tab, Outputs tab, and set the “DFM Defaults File” template. See “Templates for File Naming”.

Procedure

  1. Click the DRC Options button on the left panel, or select Setup > DRC Options.
  2. Select the Database tab.
  3. Select the DFM Defaults sub-tab.
  4. Uncheck the box “Use settings from rules” in order to use the DFM Defaults RDB statement specified in the GUI.
  5. Set the options as desired.

    Option

    Description

    File

    The filename for the RDB. Required.

    Format

    The RDB format, which must be GDS or OASIS.

    Transform geometries to the top-level cell

    Adds the TOP keyword.

    Cell prefix

    Adds the specified string as a prefix to output cell names (the PREFIX keyword).

    Cell suffix

    Adds the specified string as a suffix to output cell names (the APPEND keyword).
