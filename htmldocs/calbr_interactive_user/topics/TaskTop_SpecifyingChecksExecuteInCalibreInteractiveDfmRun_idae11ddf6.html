Specifying the Checks to Execute in a Calibre Interactive DFM Run

The Check Selection Recipe specifies the checks that are executed in a Calibre Interactive DFM run. You can use a built-in check recipe or define your own check recipe.

See “Check Selection Recipe” for a full description of check recipes.

Prerequisites

  • The setting “Disable check selection” (Setup > Preferences, Disable tab) is not enabled.

Procedure

  1. Click Rules to display the Rules pane.
  2. Specify a rule file; see “Specifying a Rule File for a DFM Run” for additional details and a view of the Rules pane.
  3. Select a check recipe using the dropdown list for Check Selection Recipe.

    BUILT-IN RECIPES

    • Checks selected in the rules file — (Default) Run only checks selected in the rule file. This is the only built-in recipe that obeys DFM [Un]Select Check and DRC [Un]Select Check statements in the rule file.

      See the definition of DFM Select Check for the behavior of Calibre DFM if no DFM Select Check statements are specified in the rule file.

      If you are using Customization Files to select checks for a Calibre Interactive DFM run, you should always use “Checks selected in the rules file” to avoid conflicts between the check recipe and the Customization GUI.

    • All checks — Run all checks in the rule file, except those excluded by pre-processor directives. DFM [Un]Select Check and DRC[Un]Select Check statements are not considered.

    USER RECIPES

    • Select a previously defined custom check recipe.

      See “Basic Editing of Custom Check Selection Recipes” for information on creating a Check Selection Recipe.

Results

The checks specified in the check recipe are executed during the Calibre DFM run.
