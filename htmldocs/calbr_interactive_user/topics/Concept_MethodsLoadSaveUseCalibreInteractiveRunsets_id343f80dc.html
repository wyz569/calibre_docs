Methods to Load, Save, and Use Calibre Interactive Runsets

You can load a Calibre Interactive runset from the command line, from the GUI, or by defining an environment variable. Preference settings allow you to save the runset automatically.
Table 1. Loading, Saving, and Using Calibre Interactive Runsets

Action

Method and Description

Load a runset or multiple runsets from the GUI

File > Load Runset

To load multiple runsets, enter a space-separated list of runset names in the Runset File Path field.

Automatic runset loading, given in order of priority

(See “Loading Runsets on Invocation”)

  • Load a runset or multiple runsets from the command line

    (Priority 1, highest)

‑runset command line option:

   -runset my_runset

   -runset “runset1 runset2”

The ‑runset_options_display command line option can be used in batch operation.

  • Populate the dropdown list of runsets with an environment variable

    (Priority 2)

Define the environment variable MGC_CALIBRE_***_RUNSET_LIST to point to a file with the list of runsets. Replace *** with DRC, LVS, PERC, PEX, or DFM. Multiple files may be specified as a space-separated list.

  • Load a runset or multiple runsets with an environment variable

    (Priority 3)

Define the environment variable

MGC_CALIBRE_***_RUNSET_FILE with the runset(s); enclose multiple runsets in quotes. Replace *** with DRC, LVS, PERC, PEX, or DFM.

  • Load a runset or multiple runsets with a startup preference setting

    (Priority 4, lowest)

  1. Choose Setup > Preferences.

  2. Choose the Startup tab.

  3. Enable “Load runset files.” To load multiple runsets, click the down arrow and enter one runset per line.

Define a default runset

For DRC, name the default runset .cgidrcdb.def. Similar filenames are used for other applications.

Print runset options to the transcript in batch operation

Use the ‑runset_options_display command line option with the ‑batch switch. See the “calibre -gui” reference page for details.

Save a runset

File > Save Runset

Also see the option “Prompt to save changes before closing runset” on the Misc tab of the Setup Preferences dialog box.

Save the runset automatically

Enable one or both of these settings on the Misc tab of the Setup Preferences dialog box.

  • “Save runset each time Calibre is run”

  • “Save runset to current directory each time Calibre is run” (saves a copy of the runset to the directory from which Calibre Interactive was invoked)

Load the rule file when a runset is loaded

Enable “Automatically load rules file” on the Misc tab of the Setup Preferences dialog box.
