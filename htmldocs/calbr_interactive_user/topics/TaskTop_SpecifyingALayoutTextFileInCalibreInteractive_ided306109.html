Specifying a Layout Text File in Calibre Interactive

You can specify a Layout Text File in Calibre Interactive. The Layout Text File is used to specify free-standing cell-based text in the rule file.

Prerequisites

A text file of the correct syntax for the Layout Text File statement.

Procedure

  1. Click app Options on the left panel, where app is one of DRC, LVS, DFM, PEX, or PERC. If the app Options button is not present, choose Setup > app Options from the menu to display it.
  2. Click the Include tab.
  3. Enter the filename of a Layout Text File in the text entry field for “Include Layout Text File.”

    The Layout Text File statement is added to the Calibre Interactive control file. Layout Text statements in the Layout Text File are added to any Layout Text statements present in the rule file.
