Calibre Menu in Synopsys IC Compiler

The Calibre menu is available after Calibre interface is loaded. The menu has selections to start Calibre Interactive, Calibre RVE, and to selection setup options.

The Calibre menu in Synopsys IC Compiler has the selections shown in Figure 1. See “Loading the Calibre Interface to Synopsys IC Compiler” if you do not see the Calibre menu in the ICC GUI.

You can also add custom menu items to the Calibre integration menu; see “Custom Menu Items in Design Tools”.
