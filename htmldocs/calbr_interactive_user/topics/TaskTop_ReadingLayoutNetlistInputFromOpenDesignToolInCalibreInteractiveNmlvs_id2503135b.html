Reading Layout Netlist Input from an Open Design Tool in Calibre Interactive nmLVS

Layout netlist data for a Calibre Interactive nmLVS run is specified on the Inputs pane on the Layout tab. You can export the layout netlist from an open design tool. The layout netlist is used in Netlist vs Netlist runs.

Prerequisites

  • The Calibre nmLVS analysis type is “Netlist vs Netlist.” See “Specifying the Calibre nmLVS Run and Analysis Type”. The options available for layout data depend on the run and analysis type.

  • Calibre Interactive is connected to a supported schematic viewer; see “Communication with the Design Tool”.

  • The connected schematic viewer is set up to export the netlist in SPICE format.

Procedure

  1. Click the Inputs button on the left panel.
  2. Click the Layout tab.
  3. Enable “Export from schematic viewer” to specify that the layout netlist is exported from a connected design tool.

    The design tool must be set up to export the layout netlist in SPICE format.

    Batch runs: In addition, you must also disable the setting “Ignore layout and source export in batch mode” in the Setup Preferences dialog box on the Misc tab. Also see “Specifying Layout and Schematic Export for a Calibre Interactive Batch Run” for details on the viewer connection for a batch run.

  4. Specify the filename for the exported layout netlist in the “File” field.

    The exported netlist is written to the filename in the File field. When connected to a layout viewer, a system-generated SPICE netlist name appears in the “File” field by default. This name is taken from the “Extract to File” template setting (Choose Setup > Preferences, then click the Templates > Outputs sub-tab). However, you can choose a different filename.

  5. Verify the name of the top cell in the “Top Cell” field.

    When your netlist is exported from a connected schematic viewer, the top cell is determined from the current cell in the viewer.

  6. Possible next steps are:

    • “Source Netlist Input in Calibre Interactive nmLVS”
