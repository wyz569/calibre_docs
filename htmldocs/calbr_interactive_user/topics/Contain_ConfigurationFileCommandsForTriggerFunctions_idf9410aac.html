Configuration File Functions for Trigger Functions

Several functions are available for the Calibre Interactive xACT custom configuration file to specify pre- and post-execution trigger functions. The trigger functions can run before or after the batch Calibre run, and can run in the shell environment or the design tool environment.

See “Configuration File Format for xACT GUI Customization” for information on the file format and how to invoke Calibre Interactive xACT with a custom configuration file. The trigger functions added with the configuration file are displayed in the Triggers Pane.

Trigger Parameters

Certain filenames and other values specified in the GUI can be passed to the trigger function using trigger parameters.

add…Trigger

The functions to add trigger definitions specify the execution environment (shell or design tool) and timing for the trigger execution.
