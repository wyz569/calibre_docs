Reading Source Netlist Data from File in Calibre Interactive nmLVS

You can read source netlist data from file. A source netlist is required in Layout vs Netlist and Netlist vs Netlist runs in Calibre Interactive nmLVS.

Prerequisites

  • The Calibre nmLVS analysis type is “Layout vs Netlist” or “Netlist vs Netlist.” See “Specifying the Calibre nmLVS Run and Analysis Type”. The options available for layout data depend on the run and analysis type.

Procedure

  1. Click the Inputs button on the left panel,
  2. Click the Netlist tab.


  3. Uncheck “Export from schematic viewer” to specify that schematic netlist files provide the source schematic.
  4. Specify the format of the source netlist using the “Format” dropdown button list.

    • SPICE — The netlist is in SPICE format.

    • VERILOG — The netlist is in Verilog format. If you specify Verilog as the source format, Calibre Interactive calls a utility named v2lvs to translate the Verilog netlist to SPICE. Choose Setup > Verilog Translator to set up the translation options; see the section “Setting Up the Verilog Translator (v2lvs)”. The translated SPICE netlist is then used for LVS comparison; the SPICE netlist name is taken from the “Verilog Files” field.

    • MIXED — Select the “Mixed” option to use a mixed mode of both SPICE and Verilog files. Separate netlist entry fields are displayed for the SPICE and Verilog files. Verilog files are handled as noted for the VERILOG option.

  5. Specify the filename for the source netlist. To specify multiple files, click the down arrow and enter one netlist file per line. Alternatively, you can enter additional SPICE files in the LVS Options pane as described in a later step in this procedure.
  6. Specify the top cell name in the “Top Cell” field.
  7. (Optional) Specify additional SPICE netlist files. This may be necessary to provide INCLUDE files for the source netlist.
    1. Click the LVS Options button or choose Setup > LVS Options.
    2. Click the Database tab and the Library sub-tab.
    3. Use the buttons next to “Additional SPICE Files” to specify the required SPICE files.
      Note:

      Calibre Interactive verifies that the source netlist is complete before starting the run. You can turn off this check by disabling “Check source netlist is complete before running LVS” (Setup > Preferences, Misc tab). See “Miscellaneous Preferences” for information on the source netlist check.

Results

The appropriate statements are written to the Calibre Interactive control file at runtime.

If you specified multiple source netlist files, Calibre Interactive creates a file named _source.net_ with .INCLUDE statements for each netlist file. The file_source.net_ is used as the source netlist for the Source Path statement in the control file. Files entered in the Database > Library tab on the LVS Options pane are listed before files in the Netlist tab on the Inputs pane. If the source netlist format was MIXED, the SPICE netlist that was translated from the Verilog input is listed last.
