Specifying Calibre nmLVS Report Options

You can adjust the information included in the LVS Report file. The settings on the Report tab are all optional, and in most cases the default settings are adequate.

Procedure

  1. (Optional) To view the rule file settings for report options, click the Rules button in the left panel and click the Load button. This loads the rule file settings into the GUI.
  2. Click the LVS Options button on the left panel to display the Options pane. If the LVS Options button is not available, choose Setup > LVS Options.
  3. Click the Report tab.
  4. Select options for the LVS Report:

    For most Calibre nmLVS runs, you should not specify any of these options, unless you actually need the information they provide. The default Calibre nmLVS report is usually all that is needed for most LVS runs.

    1. Click the button next to “LVS Report Options.”
    2. Select the desired report options. The button label changes to show the selected options.

      You can also disable report options that are specified in the rule file.

  5. Set the maximum discrepancy count:

    Select the setting for “Max. discrepancies printed in report” as follows:

    • All — Include all discrepancies in the Calibre nmLVS report.

    • Count — Enter a value for the maximum discrepancy count. The default of 50 is adequate for most cases.

  6. Specify to report seed promotions:

    The Seed Promotions Report provides detailed information regarding the promotion of individual polygons to higher-level cells for the purpose of device recognition. This is an advanced option for Calibre nmLVS-H. It should be used for test purposes only.

    1. Enable “Create Seed Promotions Report.”
    2. Specify the maximum number of polygons to include in the report for each seed promotion.
