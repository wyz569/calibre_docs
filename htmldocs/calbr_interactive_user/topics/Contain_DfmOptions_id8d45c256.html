DFM Options

Calibre Interactive provides access to a wide range of Calibre DFM options. Some are available in the Setup menu, others are available through the DFM Options pane, which you access by clicking the DFM Options button on the left panel.

Some options are similar for all applications—see “Setting Database and General Options in Calibre Interactive”.

If the DFM Options button is not visible, choose the Setup > DFM Options menu item to add the DFM Options button to the left side of the Calibre Interactive.
