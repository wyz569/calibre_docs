Selecting ERC Checks

You use the Select Checks dialog box to specify which ERC checks to execute during your Calibre nmLVS run. You can also choose whether to abort an Calibre nmLVS run if an ERC check fails. Selections in the Select Checks dialog box are not saved to the runset and must be reset each time you open Calibre Interactive nmLVS.

You can also use a customization file to select ERC checks, as described in “Customization File to Select ERC Rule Checks”. If you are using a customization file, the customization settings initialize the settings in the Select Checks dialog box. Changes made in the Select Checks dialog box take precedence over the initial settings from the Customization GUI.

If you are running Calibre Interactive in batch mode, you must use a customization file to select the ERC checks or select the ERC checks in the rule file. The Select Checks dialog box is not available in batch mode.

Prerequisites

  • A Calibre nmDRC license. Refer to the Calibre Administrator's Guide for more details.

  • ERC checks are enabled, as described in “Running ERC Checks”.

Procedure

  1. Click LVS Options on the left panel to display the Options pane. If the LVS Options button is not available, choose Setup > LVS Options.
  2. Click the ERC tab on the LVS Options pane.
  3. Click the Select Checks button. The Select Checks dialog box is displayed.
    Figure 1. Select Checks Dialog Box

  4. For each of the individual rules or rule groups, click the rule or group name checkbox to cycle between the following choices:

    • Green Check Mark — Execute this check during the Calibre nmLVS run.

    • Orange Check Mark — Abort the Calibre nmLVS run if this check fails.

    • Red X — Do not execute this check during the Calibre nmLVS run.

  5. Choose View > Close to close the Select Checks dialog box.
