Providing Additional SPICE Netlist Files with the Source Netlist

You specify additional SPICE netlist files on the Database > Library sub-tab of the LVS Options pane. This may be needed to provide INCLUDE files for the main source netlist. This option is also available for Calibre Interactive PERC and Calibre Interactive PEX.

Tip

To specify both Verilog and SPICE files for the source netlist, select the MIXED format on the Netlist tab of the Inputs pane, as described in “Source Netlist Input in Calibre Interactive nmLVS”.

Prerequisites

  • Calibre Interactive nmLVS is open.

Procedure

  1. Click the LVS Options button or choose Setup > LVS Options.
  2. Click the Database tab and the Library sub-tab.
  3. Use the buttons next to “Additional SPICE Files” to specify the required SPICE files.

Results

The additional files are included before the netlist specified on the Inputs pane in the Netlist tab. Calibre Interactive creates a file named _source.net_ with .INCLUDE statements for each netlist file; _source.net_ is used as the source netlist for the Source Path statement in the control file.
