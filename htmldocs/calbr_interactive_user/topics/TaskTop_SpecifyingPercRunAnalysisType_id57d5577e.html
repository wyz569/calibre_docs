Specifying the PERC Run and Analysis Type

You can specify a hierarchical, flat, or Calibre PERC LDL run type, and choose to analyze the geometric layout, the layout netlist, or the source netlist.

Prerequisites

  • See these topics in the Calibre PERC User’s Manual for special requirements for the Calibre PERC-LDL applications:

    “Current Density Checks” (PERC LDL CD)

    “Point-to-Point Resistance Checks” (PERC LDL P2P)

    “High-Level Checks and Topological DRC” (PERC LDL DRC)

Procedure

  1. Click Inputs to display the Inputs pane.


  2. Choose the “Run” type as follows:

    • PERC (Hierarchical) — Run Calibre PERC hierarchically. This setting adds the ‑hier command line option.

    • PERC (Flat) — Run Calibre PERC flat.

    • PERC-LDL — Run the Calibre PERC logic driven layout (LDL) application. This setting uses the ‑ldl and ‑hier command line options.

  3. Choose your analysis type in the Analyze section. The available settings depend on the Run type, as follows:

    • “PERC (Hierarchical)” and “PERC (Flat)” runs:

      • Layout — Run Calibre PERC on a geometric layout database. Calibre PERC extracts a layout SPICE netlist and runs on that netlist.

        Layout analysis inserts the PERC Netlist LAYOUT command in the rule file and uses the ‑spice option on the command line.

      • Layout Netlist — Run Calibre PERC on an existing layout netlist. The netlist must be in SPICE format, and can be a netlist extracted during a previous Calibre PERC Layout analysis.

        Layout netlist analysis inserts the PERC Netlist LAYOUT command in the rule file; the -spice option on the command line is not used.

      • Source Netlist — Run Calibre PERC on a source netlist.

        Source netlist analysis inserts the PERC Netlist SOURCE command in the rule file; the -spice option on the command line is not used.

    • “PERC-LDL” runs:

      • Layout — Run Calibre PERC LDL on a geometric layout database.

      • Source-Based Flow — Run Calibre PERC LDL DRC on a source netlist. The rule file must include the “ldl::export_perc -source” command.
