Disable Preferences

You can disable several Calibre Interactive options and changes to certain settings on the Disable tab on the Setup Preferences dialog box (Setup > Preferences).
Table 1. Disable Preferences

Option

Remarks

Disable app Options

Disables editing of settings on the Options pane.

Disable waiver flow controls

(DRC/DFM/LVS only)

Disables editing of settings on the Waivers tab of the Inputs pane.

Disable check selection

(DRC/DFM only)

Disables changing the DRC or DFM Check Selection Recipe.

Disable check selection recipe editing

(DRC/DFM only)

Disables editing of custom check selection recipes.

Disable environment changes

Disables editing of environment variables defined in the rule file.

Disable customization changes

Disables editing of customization settings.

The OK button in the Customization Settings Dialog Box is made unavailable. The Customization GUI itself is still active but changes cannot be saved.

Disable device signature creation

(LVS only)

Disables the Create Device Signature Creation selection in the Setup menu.

Disable ERC changes

(LVS only)

Disables controls in the ERC tab of the LVS Options pane.

Disable transcript output

Disables all output to the transcript window except for reports of non-zero exit status.

You can choose Transcript > Echo to File to direct the transcript output to a file; however, you cannot use the other selections in the Transcript menu.

Note: Transcript output must be enabled when creating device signatures, as described in “Device Signature Creation with Calibre Interactive nmLVS”.

Disable Fast XOR max diff count control

(DRC only)

Disables the “Max Diff” control on the Inputs pane for FastXOR runs; this option is on by default, meaning the control is not displayed. Caution should be used when using this option. The “Max Diff” control adds the ‑maxdiff argument for DBdiff execution with FastXOR; see “FastXOR Database Comparison” in the Calibre Layout Comparison and Translation Guide for more information.

Disable Fast XOR options

Disables the Fast XOR Options tab on the Inputs pane for FastXOR runs. This disable preference is on by default, meaning the tab is not displayed. See “Fast XOR Options Tab”.

Disable Fast XOR generated rules file custom name

Disables the option to specify a custom name for the generated rule file in a FastXOR run. The “Rules File” option is on the Rule File Generation tab on the Inputs pane. This disable preference is on by default, meaning the option is not displayed. See “Rule File Generation Tab”.

Disable Fast XOR layer map and base layers combined usage

Enables you to simultaneously specify the Layer Map and Base Layer options on the Rule File Generation tab. By default, these options are mutually exclusive.

Release license after: ___ inactive hours.

Sets the time-out period for releasing the application’s license; the setting is saved in the runset. The minimum time-out period is five minutes.

If the time-out period is set with a higher precedence setting, that value is displayed and cannot be changed; for more information, refer to “License Timeout in Calibre Interactive”.
