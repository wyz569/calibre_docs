Setting Up and Starting a Calibre Interactive xACT Run

To set up the GUI for a run, click each of the page names on the left panel of the GUI and fill in the required fields.

Errors and warnings in the GUI settings are indicated as follows; hover over the icon to view a message about the problem.

A missing or incorrect setting. A red exclamation point ( ! ) is also displayed by the page name if any settings have an error.

A warning, such as for a conflicting setting. The run can proceed with warnings. The warning icon is not displayed next to the page name.

Prerequisites

  • The Calibre Interactive xACT GUI is open. See “Invoking the Calibre Interactive xACT GUI”.

Procedure

  1. (Optional) Select File > Load Runset to load a saved runset.

    To load multiple runsets, click the + button and select another file.

    Click a checkbox to select the runset or a group of runsets. Check multiple individual runsets to load them as a group. When multiple runsets are loaded, the runsets are loaded in order, with options loaded last having precedence if there is a conflict.



    Note:

    The runset option naming convention for some options changed in the 2018.4 release. As of 2018.4, certain options are prefixed with “cmn” instead of “pex”. Runsets saved with Calibre 2018.4 or later may report errors if they are loaded with an earlier version of Calibre.

  2. Click Rules on the left panel and specify the rule file and run directory.

    If desired, click the button to load the settings from the rule file into the GUI.

  3. Click Inputs on the left panel and specify the required input.

    (Optional) Check “Export from layout viewer” and/or “Export from source viewer” in the Layout Path and Source Path areas to export a design from a connected viewer—the active cell in the design tool is used as the layout or source input.

    Note that other settings in the GUI can affect the controls displayed on the Inputs page. For example, setting the PEX Netlist option “Use Names From” to “Source Names” on the Outputs page causes the Source Path control to be displayed on the Inputs page.

    Tip

    To specify additional layout or schematic input files, click Database on the left panel and expand the Library area.

    If source input is used and the format is VERILOG or MIXED, provide Verilog translation options on the V2LVS page.

  4. (Optional) If you are exporting a design from a connected design tool, click Run Control on the left panel and view the “Design Tool Settings.”

    If you invoked the GUI from a design tool, the design tool connection is made automatically. If you need to make a manual connection to a design tool, see “Connecting to a Design Tool in Calibre Interactive xACT”.

  5. Click each of the other page names on the left panel and set options as needed.
    Tip

    To quickly find a specific option, click Search in the left panel and enter a search string. Changes made in the Search page are reflected when you view the control in its original page.

  6. If any GUI pages have a by their name, provide the required input.
  7. (Optional) Choose File > Save Runset to save the GUI settings.
  8. Click the Run xACT button to start the run.

Results

The run transcript is displayed in the Transcript page, with any errors and warnings listed in the bottom portion of the page. Use the search toolbar in the upper right to search the Transcript page.
