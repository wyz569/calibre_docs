Configuration Commands to Execute Functions and Commands

The custom configuration file in Calibre Interactive has commands available that call other commands or functions. The commands specify a callback function and provide a way to execute a shell command.

Command

Description
connectSpecifies a function to execute on a signal from the Calibre Interactive xACT GUI.
launchCommandThe launchCommand function executes an external (shell) command.
launchCommandReturnStringThe launchCommandReturnString function executes an external (shell) command that returns a string.
