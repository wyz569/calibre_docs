Inputs for a Calibre Interactive nmDRC Run

Layout input for a Calibre Interactive nmDRC run can be exported from an open design tool or read from file. You can also specify multiple layout inputs.

Tip

You can set a preference for the setting of “Export from layout viewer” that depends on whether your Calibre Interactive session is connected to a layout viewer. By default, this preference is set so that “Export from layout viewer” is enabled if Calibre Interactive is started from a layout viewer, regardless of the saved runset value for the option. See “Export from Viewer Preferences”.
