Specifying Report Options for a Calibre Interactive PEX Run

You select report options on the Reports tab on the Outputs pane.

Prerequisites

  • Calibre Interactive PEX is open; see “Invoking Calibre Interactive Tools”.

Procedure

  1. Click Outputs on the left panel.
  2. Click the Reports tab and choose report options and specify filenames as appropriate.

    PEX Report — The results of your PEX run. The PEX Report is not generated if the extraction type is “No R/C.”

    If you chose SOURCEBASED or SCHEMATICONLY in the dropdown list of the Netlist tab, click the dropdown list on the right to use names from the source (SCHEMATIC) or the layout (LAYOUT) in the PEX Report.

    LVS Report — You must specify an LVS report name if you chose SCHEMATIC in the “Use Names From” dropdown list of the Netlist tab.

    Select from the following sub-tabs to generate the corresponding reports:

    Point to Point — See PEX Report Point2Point in the SVRF Manual for the meaning of the available options.

    The SOURCENAMES or LAYOUTNAMES keyword in the Report Point2Point statement is determined by the selection “Use Names From” on the Outputs pane, Netlist tab.

    Coupling Capacitance — See PEX Report Coupling Capacitance in the SVRF Manual for the meaning of the available options.

    Net Summary — See PEX Report Netsummary in the SVRF Manual for the meaning of the available options.

    Also see “Reports” in the Calibre xRC User’s Manual for information on the “Resistance Point-to-Point,” “Net-to-Net Coupling Capacitance,” and “Net Summary” reports.
