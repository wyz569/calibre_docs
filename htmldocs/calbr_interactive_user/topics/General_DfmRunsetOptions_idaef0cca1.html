dfmMGCCMPMetalLayerList Description

  • ltype — The type and description of each layer used (Metal 1, Metal 2, …)

  • lname — The name of each metal layer, which is saved in the dfmdb (M1, M2, …)

  • lnum — Defines metal layers with either the GDS layer numbers or predefined layer names

  • preetch — Pre-etch layers

  • deposel — Deposit selective layers

  • ebs — Dtch-back selective layers

  • ebl — Dtch-back layers

  • dmxnum — Obsolete. Dummy metal exclusion layer

  • vnum — Obsolete. Via layers

  • opc — Obsolete. opc fill shape layers

  • nonopc — Obsolete. nonopc fill shape layers

  • rthick — Model index

  • wins — Window size per layer (beta variable CMP_FAST_ENABLE)

  • arr — Array recognition parameter per layer (beta variable CMP_FAST_ENABLE)

  • lst — Layer stack topography

  • dish — Threshold of “Dishing” hotspot rule

  • erosion — Threshold of “Erosion” hotspot rule

  • dof — Threshold of “Depth of Focus” hotspot rule

  • antidish — Deprecated. TSMC specific parameter
