Reading Layout Input from an Open Design Tool in Calibre Interactive nmDRC

You can use the open cell in a your design tool as the layout input for a Calibre Interactive nmDRC run. Enable “Export from layout viewer” on the Inputs pane to use this mode.

Prerequisites

  • Calibre Interactive nmDRC is open and connected to your design tool. See the following topics for information:

    “Invoking Calibre Interactive From a Layout or Schematic Viewer”

    “Communication with the Design Tool”

  • Your design tool is properly set up to export the design to the layout format you select in Step 3.

Procedure

  1. Click Inputs on the left panel.
  2. Click the Layout tab.


  3. Use the “Format” dropdown button list and select the format of the exported layout data; GDSII, OASIS, and LEF⁄DEF are available. See “Export DEF Layout from Design Tool” for more information when using LEF⁄DEF input.

    For a full list of supported layout databases, refer to the “Layout Databases” section of the Calibre Verification User’s Manual. Also see “Third Party Database Options”.

  4. Enable “Export from layout viewer.”

    This specifies the layout viewer as the input source. The layout format must be GDSII, OASIS, or LEFDEF when “Export from layout viewer” is enabled.

    Batch runs: In addition, you must also disable the setting “Ignore layout and source export in batch mode.” Choose Setup > Preferences and the Misc tab to change the setting. Also see “Specifying Layout and Schematic Export for a Calibre Interactive Batch Run” for details on the layout viewer connection for a batch run.

  5. Verify the filename for the exported layout database in the “File” field.
    CAUTION:

    When “Export from layout viewer” is enabled, the file specified in the File or DEF File field is overwritten if it exists already. Do not enter a tapeout database in the File field, or use the exported layout named in the File field for tapeout or any other use.

    When exporting a layout database from a connected viewer, an intermediate file is created and saved to the file named in the File field. This filename is generated automatically based on the setting on the Templates > Inputs sub-tab of the Setup Preferences dialog box.

    If the filename is red, the file does not exist and Calibre Interactive creates the file. If the filename is green, the file exists and will be overwritten. Enter another filename if you want to keep the existing file.

  6. (Optional) If you have multiple layout inputs for your DRC run, choose Setup > DRC Options then the Database > Library tab to specify them. See “Specifying Multiple Layouts for a DRC Run”. The layout format must be GDSII or OASIS.
  7. Verify the top cell name; the top cell is determined from the current cell in the layout viewer when you export a layout database.
  8. (Optional) Enable the “Area” option to perform DRC checking on only a rectangular area of the layout. See “Performing Area DRC” for complete information.
  9. (Optional) You can apply waivers during the Calibre nmDRC run. Calibre Auto-Waivers controls are on the Inputs pane in the Waiver tab; see “Calibre Auto-Waivers in Calibre Interactive”.
  10. Possible next steps are the following:

    • “Specifying Multiple Layouts for a DRC Run”

    • “Specifying Outputs for a DRC Run”
