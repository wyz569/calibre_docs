Reading Layout Input from an Open Design Tool in Calibre Interactive DFM

You can use the open cell in a your design tool as the layout input for a Calibre Interactive DFM run. Enable “Export from layout viewer” on the Inputs pane to use this mode.

Prerequisites

  • Calibre Interactive nmDRC is open and connected to your design tool. See the following topics for information:

    “Invoking Calibre Interactive From a Layout or Schematic Viewer”

    “Communication with the Design Tool”

  • Your design tool is properly set up to export the design to the layout format you select in Step 3.

Procedure

  1. Click Inputs on the left panel of Calibre Interactive.
  2. Click the Layout tab.


  3. Use the “Format” dropdown button list and select the format of the exported layout data from GDSII, OASIS, or LEFDEF. For LEF⁄DEF format, see “Export DEF Layout from Design Tool” for more information.
  4. Enable “Export from layout viewer.”

    This specifies the layout viewer as the input source.

    Batch runs: In addition, you must also disable the setting “Ignore layout and source export in batch mode”; choose Setup > Preferences, then the Misc tab to find the setting. Also see “Specifying Layout and Schematic Export for a Calibre Interactive Batch Run” for details on the layout viewer connection for a batch run.

  5. Verify the filename for the extracted layout database in the “File” field. This file will be overwritten if it exists already.

    When exporting a layout database from a connected viewer, an intermediate file is extracted and saved to the file named in the “File” field. This filename is generated automatically based on the setting on the Templates > Inputs tab of Setup Preferences dialog box.

    If the filename is red, the file does not exist and Calibre Interactive creates the file. If the filename is green, the file exists and will be overwritten. Enter another filename if you want to keep the existing file.

    CAUTION:

    When “Export from layout viewer” is enabled, do not use the extracted layout named in the “File” field for tapeout or any other use.

  6. (Optional) If you have multiple layout inputs for your DFM run, choose Setup > DFM Options then the Database > Library tab to specify them. See “Specifying Multiple Layouts for a DFM Run”. The layout format must be GDSII or OASIS.
  7. Verify the top cell name. The top cell is determined from the current cell in the viewer.
  8. (Optional) Select the “Area” option to perform DFM checking on only an area of the layout. See “Performing Area DFM” for complete information.
  9. (Optional) If you are running Critical Feature Analysis (CFA) you can apply waivers during the run. Calibre Auto-Waivers controls are on the Inputs pane on the Waiver tab; see “Calibre Auto-Waivers in Calibre Interactive”.
