Setting Calibre PERC Options

You can specify options for your Calibre PERC run from the PERC Options panel. These settings generate SVRF specification statements in your control file. If the rule file you load has SVRF statements that correspond to these GUI options, the GUI is populated with the rule file settings. Whatever options you specify in the GUI override the rule file settings.

Some options are similar for all applications—see “Setting Database and General Options in Calibre Interactive”.

Choose Setup > PERC Options to display the PERC Options pane. This also adds the PERC Options button to the left panel.



The following options are described:

  • Setting Power Supply Options
  • Aborting Calibre PERC on Sconnect Errors
  • Specifying PERC Report Options and Result Count
  • Controlling Logic Gate Recognition in Calibre PERC
  • Setting Split Gate and Parallel MOS Reduction Options in Calibre PERC
  • Setting Filter Unused Device Options in Calibre PERC
  • Changing Trace Property Rules for the Calibre PERC Run
