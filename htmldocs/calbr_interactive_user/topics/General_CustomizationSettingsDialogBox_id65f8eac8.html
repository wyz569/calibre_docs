Customization Settings Dialog Box

To access: Customization button, or see “Methods to Open the Customization Settings Dialog Box” for methods to open the Customization Settings dialog box automatically or in batch mode.

The Customization Settings dialog box, or Customization GUI, contains the customization controls specified in the customization file. Several action buttons, such as Load and Reset, are also included.

Description

The customization file is read the first time Customization GUI is opened and not read again unless the customization file changes or a new customization file is loaded with the Load button. In particular, ‑initval settings in customization commands are only processed the first time the customization file is read.



Tip

You can disable changes to the customization settings with “Disable customization changes” on the Disable tab of the Setup Preference dialog box (Setup > Preferences).

Table 1 describes the function of the action buttons in the Customization Settings dialog box.

Buttons

Table 1. Action Buttons in Customization Settings Dialog Box

Action Button

Description

OK

Applies the current settings and closes the dialog box. The new settings are retained within the current Calibre Interactive session. The new settings are used in the next Calibre run unless overridden by later GUI actions.

If you make changes to the Customization GUI and then save a runset, the changes to the Customization GUI are saved to the runset as customization overrides. The override settings in the runset are applied the next time you load the runset and open the Customization GUI. See “Saving Customization Settings” for complete details.

Cancel

Cancel any changes to the customization dialog box and close the dialog box.

Reset

Reload the current customization file. Changed settings are discarded and customization override values in the runset are not applied. The dialog box remains open.

Load

Opens a dialog box to load a new customization file. Customization override values in the runset are not applied when the new customization file is processed.

Loading a new customization file causes the current customization settings to be ignored. The setting for “Customization File” on the Startup tab of the Setup Preferences dialog box is updated with the new customization file.

Exit

(only present for batch runs)

Closes the Customization Settings dialog box without running Calibre Interactive.
