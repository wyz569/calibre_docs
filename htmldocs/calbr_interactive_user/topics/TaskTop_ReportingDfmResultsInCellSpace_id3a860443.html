Reporting DFM Results in Cell Space

Calibre reports DFM results in top-cell coordinates by default. Choose the “Output cell errors in cell space” option to instruct Calibre to use local cell space coordinates when reporting errors. This option is only for DFM-H runs that output ASCII format results database.

Procedure

  1. Choose Setup > DFM Options to display the DFM Options pane.
  2. Click the ASCII Results tab.
  3. Check the “Output cell errors in cell space” option.
