Default Runsets in Calibre Interactive

You can use a default runset to specify custom default settings for Calibre Interactive. Default runsets are used only when no other runset is loaded.

To create a default runset, create a runset with the settings you want, save it to one of the following filenames depending on your application, and place the file in your home directory:

Table 1. Filenames for Default Runsets

Application

Filename

DRC

.cgidrcdb.def

DFM

.cgidfmdb.def

LVS

.cgilvsdb.def

PERC

.cgipercdb.def

PEX

.cgipexdb.def

The default runset is read in the following cases:

  • You invoke Calibre Interactive without loading a runset.

  • You choose File > New Runset during a Calibre Interactive session.

Calibre Interactive searches your home directory for a default runset file in the above two cases. If a default runset is found those settings are used instead of the Calibre Interactive built-in default values.

Note:

The default runset is not read when you load a runset. When you load a runset, the built-in Calibre Interactive defaults are applied first, then the settings in the runset being loaded are applied and overwrite the default settings.
