Using Calibre Interactive to Perform DFM

You can use Calibre Interactive to run Calibre DFM and Calibre® LFD™.

This chapter assumes you are familiar with the terms and concepts presented in the chapter “Running Calibre Interactive”. The objectives of this chapter are:

  • To guide you in setting up and performing a Calibre DFM or Calibre LFD run using Calibre Interactive.

  • To provide you with the knowledge you need to change existing settings so you can tailor each run to fit your current needs. When you are satisfied with the settings, you can save your settings as a new runset for use in your next session.
