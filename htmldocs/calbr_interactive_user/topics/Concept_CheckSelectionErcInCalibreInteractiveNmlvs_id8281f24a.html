Check Selection for ERC in Calibre Interactive nmLVS

The ERC rule checks that are executed during the Calibre run are selected with the Select Checks dialog box in Calibre Interactive nmLVS. You can also use a customization file to select checks.

Select Checks Dialog Box

Choose Setup > LVS Options, click the ERC tab, then click the Select Checks button to open the Select Checks dialog box. Selections made in the Select Checks dialog box are not saved to the runset. See these topics for details:

  • “Selecting ERC Checks”

  • “Running ERC Checks”

Customization File

You can use a customization file to select ERC checks for a Calibre Interactive nmLVS run. This method is useful because customization settings are applied in both interactive and batch runs of Calibre Interactive and can be saved to the runset. See these topics for more information on customization files:
