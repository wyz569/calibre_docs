Environment Variables for Calibre View

Several environment variables are available to control Calibre View generation.

The following table lists environment variables used in Calibre View generation. The Type column indicates whether the environment variable is available in the shell environment (env) or in SKILL scripts.

Also see “Global Variables for Calibre View Generation”.

Table 1. Calibre View Environment Variables

Variable

Type

Description

CALIBRE_HOME

env

Specifies the software tree invoked when Calibre Interactive or Calibre RVE is called from Pyxis (“Using Calibre Interactive with Pyxis”).

Also sets the location of the software tree for Calibre tools; see “Setting the CALIBRE_HOME Environment Variable” in the Calibre Administrator’s Guide.

CALIBRE_DISABLE_PPAR_WARNINGS

env

Set this environment variable to disable pPar evaluation warnings.

CALIBRE_FDIBA_ENABLE_NETEXPR_COPY

env

See “Handling of Nets With Inherited Connectivity”.

MGC_RVE_INIT_SOCKET_AT_STARTUP

env

Sets socket initiation preference for Cadence Virtuoso (“Table 2”) and Synopsys IC Compiler (“Setting Socket Connections with Synopsys IC Compiler and IC Compiler II”).

MGC_RVE_RELEASE_LICENSE_TIME

env

Sets the time in hours after which the Calibre RVE license is released if there is no activity (License Timeout in Calibre RVE” in the Calibre RVE User’s Manual).

If the environment variable MGC_CALGUI_RELEASE_LICENSE_TIME is not set, then MGC_RVE_RELEASE_LICENSE_TIME also sets the license timeout for Calibre Interactive. (“License Timeout in Calibre Interactive”).

MGCCALIBREMENUVIEWTYPE

SKILL

Specifies window type in which to load a Calibre menu (“Creating an Interface to Cadence Virtuoso”).

OA_HOME1

env

Sets OpenAccess library version to one other than the default. The OpenAccess default library that ships with Calibre is located at . ⁄shared⁄pkgs⁄icv_o in your Calibre software tree. (“Specifying Third Party Database Options in Calibre Interactive”)

Note: As of the 2013.3 release, the supported OpenAccess version in Calibre is oa22.43p006. Also see footnote.
