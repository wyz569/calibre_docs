Outputs Page in Calibre Interactive xACT

To access: Click Outputs in the left panel of the Calibre Interactive xACT GUI. (shortcut: Alt‑o)

The Outputs page controls the extraction mode and statements related to the extracted netlist and results database.



Objects

In most cases the options have an informative tooltip or correspond directly to a keyword in an SVRF statement. Refer to the SVRF Manual for keyword descriptions.

Option

Description

xACT Mode

Select an extraction mode of xACT, xACT 3D, or xACT 3D Select.

Extraction Type

Choose the parasitics to extract.

PEX Xact Corner

Specify process corners with the PEX XACT Corner statement.

Do not specify this option in the GUI if your rule file includes the PEX XACT Corner statement. This option cannot be specified with the “Corner” option in the GUI.

The extracted netlist path in the PEX Netlist statement is determined automatically by the GUI based on the corner information and cannot be edited.

If the netlist format is CALIBREVIEW, an extracted netlist and Calibre View is created for each corner.

Corners

Specify the process corners to write out. This option uses the ‑corner command-line option.

This option cannot be specified with the “PEX Xact Corner” option in the GUI.

PEX Netlist

Specify the options for the PEX Netlist statement. Use the GUI tooltips or refer to the SVRF Manual for the definition of the various options.

This area includes the option “View Netlist after xACT finishes.”

Mask SVDB

Specify options for the Mask SVDB Directory statement. Enable the checkbox to use this statement.

Tip: Enable “Show Results in RVE” on the Run Control page in the RVE Options area to start Calibre RVE automatically after the run.

Reports

Specify the LVS Report and PEX Report.

This area includes the options “View LVS Report after run finishes” and “View PEX Report after run finishes.” “PEX Report” must be enabled in order for “View PEX Report after run finishes” to be displayed.

PEX Report Point2Point

See PEX Report Point2Point in the SVRF Manual for the meaning of the available options.

Formatter

Specify options to configure the Calibre xACT formatter. The formatter step is not run separately; the following command line options are added to the -xact step.

  • Ground all coupling capacitors — Adds the -g option.

  • Display formatter warnings — Adds the -fmt_warnings option.

  • Display formatter information — Adds the -fmt_info option.
