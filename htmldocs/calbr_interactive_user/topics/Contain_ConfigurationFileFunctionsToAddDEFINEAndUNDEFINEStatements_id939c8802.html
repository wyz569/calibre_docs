Configuration File Functions to Add #DEFINE and #UNDEFINE Statements

Several functions are available for the Calibre Interactive xACT custom configuration file that add a #DEFINE and #UNDEFINE to the Calibre Interactive control file.

See “Configuration File Format for xACT GUI Customization” for information on the file format and how to invoke Calibre Interactive xACT with a custom configuration file.

Command

Description
addChoicesDefineAdds a control to specify a #DEFINE or #UNDEFINE statement with a value to the Calibre Interactive xACT GUI; the value is taken from an enumerated list. The statement is added to the control file.
addDefineAdds a control to specify a #DEFINE or #UNDEFINE statement with no value to the Calibre Interactive xACT GUI. The statement is added to the control file.
addInputDirTextDefineAdds a control to specify a #DEFINE or #UNDEFINE statement with a string value to the Calibre Interactive xACT GUI, where the string value is expected to correspond to a directory used for input. The statement is added to the control file. The control includes a text input field and a browser button for selecting the directory.
addInputFileTextDefineAdds a control to specify a #DEFINE or #UNDEFINE statement with a string value to the Calibre Interactive xACT GUI, where the string value is expected to correspond to an input file. The statement is added to the control file. The control includes a text entry field, a file browser button for selecting the file, and a view file button.
addIntegerDefineAdds a control to specify a #DEFINE or #UNDEFINE statement with an integer value to the Calibre Interactive xACT GUI. The statement is added to the control file.
addOutputDirTextDefineAdds a control to specify a #DEFINE or #UNDEFINE statement with a string value to the Calibre Interactive xACT GUI, where the string value is expected to correspond to a directory used for output. The statement is added to the control file. The control includes a text input field and a browser button for selecting the directory.
addOutputFileTextDefineAdds a control to specify a #DEFINE or #UNDEFINE statement with a string value to the Calibre Interactive xACT GUI, where the string value is expected to correspond to an output file. The statement is added to the control file. The control includes a text input field, a file browser button for selecting the file, and a view file button.
addRealDefineAdds a control to specify a #DEFINE or #UNDEFINE statement with a floating-point value to the Calibre Interactive xACT GUI. The statement is added to the control file.
addTextBoxDefineAdds a control to specify a #DEFINE or #UNDEFINE statement with a string value to the Calibre Interactive xACT GUI. The GUI control is a multi-line text box. The statement is added to the control file.
addTextDefineAdds a control to specify a #DEFINE or #UNDEFINE statement with a string value to the Calibre Interactive xACT GUI. The statement is added to the control file.
