Requirements for Calibre Interactive xACT

Several requirements must be met to use Calibre Interactive xACT.

  • Using Calibre on a platform running the AOI executable.

    The AOI executable supports platforms running Red Hat Enterprise Linux 6 (RHEL 6), RHEL 7, SLES 11 Service Pack 2, or SLES 11 Service Pack 3. See the Calibre Administrator’s Guide for more information.

  • The Calibre Interactive xACT GUI uses the XKEYBOARD extension. If you notice that keyboard input to the GUI fields is not correct, make sure your interface to the Linux machine running Calibre uses the XKEYBOARD protocol extension.
