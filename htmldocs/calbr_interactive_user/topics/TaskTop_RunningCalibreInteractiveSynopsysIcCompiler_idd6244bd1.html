Running Calibre Interactive with Synopsys IC Compiler

All Calibre Interactive applications are supported: Calibre nmDRC, Calibre nmLVS, DFM, Calibre PERC, and PEX.

You can use Calibre Interactive to easily change options for your Calibre run, since Calibre Interactive settings override settings in the rule file. You save all your Calibre Interactive settings in a runset for later use.

Prerequisites

  • “Loading the Calibre Interface to Synopsys IC Compiler” has been completed.

Procedure

  1. Launch Synopsys IC Compiler:
    icc_shell -gui

    You should see the Calibre banner displayed in the ICC command window; if not, see “Loading the Calibre Interface to Synopsys IC Compiler”.

  2. Open your library

    icc_shell> open_mw_lib my_milkyway_lib

    icc_shell> open_mw_cel top_cell

  3. Choose Calibre > Run nmDRC, or the desired Calibre Interactive application.
    Figure 1. Calibre Menu in Synopsys IC Compiler

    Calibre Interactive opens and displays the Inputs pane with the current cell selected.

    Note:

    Calibre Pattern Matching, Calibre DP, and dummy fill are run using the Run DRC selection. Calibre SmartFill is run using the Run DFM selection.

  4. (Optional) Load a runset with File > Load Runset; a runset includes all Calibre Interactive settings. See “About Calibre Interactive Runsets” for information.

    If your runset includes all the desired options, you may be ready to start a run; if so, skip to Step 13.

  5. Specify the input source as “Layout Export” and the desired format of the exported design.

    These steps instruct Synopsys IC Compiler to export the open design to Calibre. Synopsys IC Compiler controls the conversion to the output format.

    1. Set “Format” to GDSII, OASIS, or LEFDEF.

      See “Export DEF Layout from Design Tool” for additional details if exporting DEF format.

    2. Enable “Export from layout viewer.”
    3. Verify the filename for the exported layout database in the “File” field.

      An intermediate file is extracted and saved to the file named in the “File” field during layout export. This filename is generated automatically based on the setting on the Templates > Inputs sub-tab of the Setup Preferences dialog box (Setup > Preferences); see “Templates for File Naming”.



    4. Set streamout options in Synopsys IC Compiler. Consult the Synopsys IC Compiler documentation, or view Calibre > Layout Export and Calibre > Netlist Export in Synopsys IC Compiler for information.
  6. View the Rules and Outputs panes and fill in the required settings.
  7. (Optional) You can select the checks that are run in Calibre Interactive nmDRC, DFM, and Calibre nmLVS (for ERC checks). You can select checks by groups, by layer, and other criteria.

    See the following topics for more information:

    • “Specifying the Checks to Execute in a Calibre Interactive nmDRC Run”

    • “Specifying the Checks to Execute in a Calibre Interactive DFM Run”

    • “Selecting ERC Checks”

  8. (Optional) Calibre Interactive supports trigger functions which can execute at several points in your flow. Trigger functions are useful for customizing your flow and doing pre- and post-processing of design input and result output.

    Choose Setup > Preferences, then click the Triggers tab.

    See the following topics for information:

    • “Trigger Functions in Calibre Interactive” — Complete information.

    • “External Trigger Definitions in Synopsys IC Compiler” — External triggers, which execute before and after Calibre Interactive execution.

    • “Internal Triggers in Calibre Interactive” — Internal triggers, which execute before and after the Calibre run.

  9. Click Run Control on the left panel and review the settings. Performance on large designs can be improved by enabling the multithreaded (MT) and hyperscaling processing modes. See “Run Control Setup and Licensing” for more information.
  10. (Optional) You can run Calibre on a selected region of the design when using Calibre Interactive nmDRC and DFM.

    See “Performing Area DRC” for instructions.

  11. (Optional) View the control file for the Calibre Interactive run by clicking Ctrl‑Run DRC. The control file contains the settings that control the Calibre run; control file settings take precedence over rule file settings if there is a conflict. See “About Control Files” for more information.
  12. Choose File > Save Runset to save your Calibre Interactive settings; see “About Calibre Interactive Runsets” for information.
  13. Click Run DRC (or your selected application) on the left panel to start the Calibre run.

    Calibre Interactive switches the view to the Transcript pane, where you can view the progress of the Calibre run and any run-time errors and warnings. See “Transcript Pane in Calibre Interactive” for more information.

  14. If Calibre RVE did not open automatically, choose Calibre > Start RVE in Synopsys IC Compiler to view the Calibre results. See “Calibre RVE with Synopsys IC Compiler” in the Calibre RVE User’s Manual for more information.
