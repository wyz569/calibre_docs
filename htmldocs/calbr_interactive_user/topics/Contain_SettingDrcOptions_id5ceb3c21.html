DRC Options in Calibre Interactive

Calibre Interactive provides access to a wide range of Calibre nmDRC options. Some are available in the Setup menu, others from the Inputs and Outputs panes, and others are available through the DRC Options pane.

Some options are similar for all applications—see “Setting Database and General Options in Calibre Interactive”.

Do one of the following to access the DRC Options pane:

  • Choose Setup > DRC Options. This adds the DRC Options button to the left side of the Calibre Interactive nmDRC GUI and opens the tab.

  • Click the DRC Options button on the left panel if it is present.
