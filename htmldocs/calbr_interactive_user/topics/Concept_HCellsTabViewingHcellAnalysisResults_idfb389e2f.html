H-cells Tab for Viewing Hcell Analysis Results

Hcell analysis results are shown in four sub-tabs within the H-Cells tab of the Calibre Interactive nmLVS Inputs pane.

See the task “Setting up the Hcell Analysis Run” for a description of how to perform Hcell analysis.

When Hcell analysis is complete, the results are displayed as shown in Figure 1 in the following four sub-tabs:

  • H-Cells Evaluation — This tab displays the results of your Hcells evaluation. Use the Save H-cells button to save the selected hcell names to an hcell file. When asked whether to use this as an hcell file, selecting yes automatically enables the “Use H-Cells file” option.

  • Source Analysis — This tab displays information about the contribution of devices present given certain conditions and the resulting memory savings. Hover your mouse over the column heading for a description of the data provided.

  • Layout Analysis — This tab displays information similar to the Source Analysis tab, but for the layout netlist.

  • Hierarchy — This tab displays the hierarchical structure of both the layout and source.

Also see “Hcell and Hierarchical Analysis” in the Calibre Query Server Manual.
