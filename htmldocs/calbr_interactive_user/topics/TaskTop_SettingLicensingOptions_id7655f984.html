Setting Licensing Options

You can control licensing options using Calibre Interactive.

Procedure

  1. Select the Run Control pane.
  2. Click the Environment tab and then the Licensing sub-tab.
  3. Choose application-dependent license options:

    • Calibre nmDRC and Calibre nmLVS

      • Acquire CB license for RVE when using Calibre CB

        Select this option if you selected a run type of “Calibre-CB (flat)” on the Calibre Interactive nmDRC Inputs pane or “Calibre CB” on the Calibre Interactive nmLVS Inputs pane.

    • PEX

      Use these options if you are performing a flat extraction run. The Calibre CB license options for Calibre nmLVS and Calibre xRC should be selected if SOURCE names are requested in a PEX run of Calibre Interactive and you only have the Calibre Cell⁄Block (CB) license.

      • Use Calibre-xRC CB license — Can only be used for flat transistor or gate-level extraction, and on designs containing up to 100,000 transistors.

      • Use Calibre-CB license for LVS — The ‑flatten command line option is used in the Calibre nmLVS invocation.

      • Use Calibre-CB license for RVE

      Also see “Extracting a Block Using CB” in the Calibre xRC User’s Manual.

  4. If needed, set licensing options for remote hosts in the “Remote Calibre” section.

    You can set the “License File Name” and “License File Value.” See the section “Mentor Standard Licensing (MSL)” in the Calibre Administrator’s Guide for information on the license file. See “Configuring the Remote Environment” for other options related to setting up the remote environment.

  5. Enable or disable licensing queue behavior. If enabled, select one of the following options:

    • Disable Queuing — (‑nowait) The Calibre run is aborted if a license is not immediately available.

    • Wait for License — (‑wait <time>) The run is placed into a queue if a license is not immediately available.

      • Wait for ____ minutes — Select this option to specify a time parameter for the -wait argument in the Calibre command line. If a wait time is not specified, Calibre waits forever in the queue for a license.

    • Retry for License — (‑lmretry loop) Retry, or loop, until a license is available, subject to the following options:

      • Retry for: ____ minutes (MAXRETRY:<minutes>)

      • Wait for: ____ seconds between retries (INTERVAL:<seconds>)

      • Retry even when license errors occur (RETRY_ON_ERROR)

    When disabled, the Calibre command line does not include any licensing arguments.
