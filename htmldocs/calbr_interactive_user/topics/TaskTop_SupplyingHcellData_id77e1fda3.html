Supplying Hcell Data

Hcell files are used in hierarchical Calibre nmLVS to define the correspondence between layout and source cell names.

Prerequisites

  • Calibre Interactive nmLVS is open.

Video

If you don’t have an hcell list, you can use Calibre Interactive to create an initial hcell list, as demonstrated in the video “How To Create an Initial Hcell List with Calibre Interactive.”



Procedure

  1. Click the Inputs button on the left panel.
  2. Select “Hierarchical” for the “Run” type.
  3. Click the H-Cells tab.
  4. Select the desired options:

    • Match cells by name — Specifies automatic correspondence by name for cells in Calibre nmLVS-H comparison and should only be used if the layout cells actually contain the same devices as the source subcircuits of the same name.

      This option applies to LVS comparison (the ‑automatch command line option); this option does not apply to netlist extraction.

      If “Use H-Cells file” is also specified, automatched cells are added to the hcell list.

    • Use H-Cells file — Specifies to use an hcell file. The hcell file specifies cell correspondence for hierarchical LVS comparison. This option preserves hcells as subcircuits during both netlist extraction and LVS comparison.

    • Automatically generate an H-Cell list — Specifies to automatically generate an hcell list during circuit extraction using the Query Server Tcl shell. This option adds the ‑genhcells option to the command line. This option cannot be specified with “Match cells by name.”

      Check “Use default thresholds as hcell selection criteria” to use the default thresholds when selecting hcells. This option adds the ‑select keyword to the command line.

      Check “Use H-Cells Query Server Tcl script” to execute a user-defined Tcl script to generate the hcell list. If not specified, a default Tcl script is used.

      If “Use H-Cells file” is also specified, the hcell lists are concatenated.

      For more details, see the ‑genhcells option in the section “Calibre nmLVS and Calibre nmLVS-H Command Line” of the Calibre Verification User’s Manual.

    • Expand H-Cells to resolve false discrepancies — Specifies to examine LVS discrepancies and, in certain cases, to automatically perform additional comparison runs with fewer hcells. Enable the checkbox and select “Yes” to use this option. Uses the LVS EXPAND ON ERROR {YES | NO } statement.

    Note:

    The option “Match cells by number of placements” does not apply during LVS comparison or netlist extraction. It only applies during hcell analysis in Calibre Interactive, as described in “Performing Hcell Analysis in Calibre nmLVS”.
