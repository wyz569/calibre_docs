Reading Geometric Layout Input from File in Calibre Interactive PERC

Layout data for a Calibre Interactive PERC run is specified on the Inputs pane on the Layout tab. You can read the layout data from design files saved on disk. The layout netlist is extracted from the geometric layout.

Procedure

  1. Click Inputs to display the Inputs pane.
  2. Make sure that “Layout” is selected at the top of the Inputs pane in the Analyze section.
  3. Click the Layout tab.
  4. Uncheck “Export from layout viewer” to specify that the layout data is contained in the specified files.
  5. Select a layout format using the “Format” dropdown button list.

    For a full list of supported layout databases, refer to the “Layout Databases” section of the Calibre Verification User’s Manual. Also see “Third Party Database Options”.

  6. Specify the filename for the layout database in the “Layout File” field for GDSII and OASIS format. For other formats, see the following topics for additional information:

    • LEFDEF — see “Providing LEF/DEF Input”

    • OPENACCESS — see “Providing OpenAccess Input”

  7. Specify the top cell name in the “Top Cell” field.
  8. Enter the name of the extracted layout netlist in the “Layout Netlist” field. This option is for hierarchical runs only; in addition, it is not present for Calibre PERC-LDL runs.
  9. (Optional) If you have multiple layout inputs for your run, you can specify them using the Database > Library tab on the PERC Options pane.
