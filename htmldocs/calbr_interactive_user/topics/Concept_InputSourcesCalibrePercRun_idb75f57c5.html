Input Sources for a Calibre PERC Run

The origin of your input is different depending on whether you run Calibre Interactive connected to a layout or schematic viewer or not.

The following table shows these differences.

Table 1. Sources for Calibre PERC Input Data

Analysis Type

Viewer Connected

No Viewer Connected

Layout

One of the following:

a) The layout data comes from the currently-open layout viewer when the “Export from Layout Viewer” option is enabled.

b) The layout data comes from an existing set of layout files.

Layout data exists as one or more files used for input.

Layout Netlist

The netlist data comes from a file.

The netlist data comes from a file.

Source Netlist

One of the following:

a) The source netlist comes from the currently-open schematic viewer when the “Export from Schematic Viewer” option is enabled.

b) the source netlist comes from an existing set of files.

The source netlist comes from an existing set of files.
