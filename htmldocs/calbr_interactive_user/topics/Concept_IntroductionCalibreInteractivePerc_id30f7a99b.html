Introduction to Calibre Interactive PERC

The Calibre Interactive PERC Graphical User Interface (GUI) allows you to specify command line options and selected rule file statements that control a Calibre PERC run.

A common scenario for performing Calibre PERC from Calibre Interactive assumes that a runset file containing all the necessary settings already exists. Your time is spent running Calibre PERC and correcting errors in an interactive verification loop, as illustrated in Figure 1. In this scenario, you spend very little time setting up the Calibre run because you can load the necessary settings from a runset. You can set up Calibre Interactive to load the runset automatically on invocation as described in “Loading Runsets on Invocation”.
