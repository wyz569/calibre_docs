Performing Area DRC

You can run DRC checks on only a specified rectangular area of the layout. Area DRC checks the specified area plus a halo region around the area. You can specify the halo size and whether results in the halo region are filtered out or kept.

Prerequisites

  • “Calibre Interactive - nmDRC Requirements” are met

Note:

When performing Area DRC in the Cadence Virtuoso integration (version ICADVM18.1 or later), it is recommended that you open the design in Cadence Virtuoso Layout XL or Layout EXL instead of using the Layout Viewer application.

Procedure

  1. Click Inputs on the left panel and enter the layout database information as described in “Inputs for a Calibre Interactive nmDRC Run”.
  2. Enable the Area option.
  3. Select the area coordinates with one of the following methods:

    • Enter space-separated coordinates for the rectangular window you want to check. Coordinates are the lower-left and upper-right corners of the rectangle, in that order. You must use this method if not connected to a layout viewer.

    • If connected to a layout viewer, click the browse button (...) to select the area you want to check in the layout viewer. Click and drag out an area selection rectangle In the layout viewer. The coordinates of the area selection rectangle appear in the Area field of the Inputs pane.



      Tip

      When connected to a layout viewer, you can right-click the (...) button to highlight and zoom to the specified Area DRC region in the layout.

  4. (Optional) Set area DRC options, such as halo size and coordinate scaling, as follows:
    1. Click the DRC Options button on the left panel, or choose Setup > DRC Options if the left panel button is not present.
    2. Click the Area DRC tab, then select the desired options described in Table 1.
      Table 1. Area DRC Options

      Option

      Description

      Halo width for area DRC

      Specifies the size of the halo which expands the area DRC window.

      • Automatic — Width of halo is the greater of the width or height of the specified Area DRC region.

      • Size — Width of halo is an explicit user-specified width in user-units.

      Note: Depending on the geometry of your area selection, the automatic setting can result in a halo region that is much larger than needed, resulting in slow performance.

      Remove results from halo region after area-DRC

      (Only applies to the ASCII format results database.)

      Enabled — (default). As described.

      Disabled — Keep results in the halo region.

      Apply coordinate scaling

      Enabled — (default) Scale window coordinates by:

         (DRC Results Database
    Precision⁄Precision) *(Layout Magnify)

      Disabled — Do not apply scaling.
  5. Run DRC, as described in the section “Running Calibre Interactive nmDRC”.

Results

The following changes in output occur when Area DRC is enabled:

  • Results database — Results in the halo region are filtered out based on the setting “Remove results from halo region after area-DRC” on the Area DRC tab of the DRC Options pane. Result filtering only applies to ASCII format results databases.

  • Auxiliary databases — If an auxiliary results database is specified in the rule file, the results are filtered as specified for the main results database.

  • Transcript and summary report output — If the setting “Remove results from halo region after area-DRC” is enabled on the Area DRC tab of the DRC Options pane, then both the transcript and the DRC Summary Report report the number of results that were filtered out of the halo region.

    For example, the following line appears in both the transcript and the summary report:

    TOTAL DRC Results Generated:     422 (783), FILTERED 397, 
	FINAL Results Generated = 25
