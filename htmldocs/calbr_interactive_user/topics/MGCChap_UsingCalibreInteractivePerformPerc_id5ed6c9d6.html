Using Calibre Interactive to Perform PERC

The Calibre Interactive PERC GUI allows you to specify command line options and select rule file statements for a Calibre PERC run. You can save GUI settings to a Calibre Interactive runset and load them in future runs.

This chapter assumes you are familiar with the terms and concepts presented in the chapter “Running Calibre Interactive”.

You should also be familiar with the Calibre PERC User’s Manual.
