Calibre nmLVS Options in Calibre Interactive

You can specify options for your Calibre nmLVS run from the LVS Options pane. These settings generate SVRF specification statements in your control file. The options you specify in the GUI override the rule file settings.

Some options are similar for all applications—see “Setting Database and General Options in Calibre Interactive”.

If the LVS Options button is not visible, choose the Setup > LVS Options menu item to add the LVS Options button to the left side of the Calibre Interactive.
