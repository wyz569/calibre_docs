Specifying a Rule File for PEX Run

You need to specify a rule file for any PEX run. You can specify a rule file automatically with a runset or you can specify one manually using this procedure.

Prerequisites

  • Calibre Interactive PEX is open; see “Invoking Calibre Interactive Tools”.

Procedure

  1. Click Rules on the left panel.


  2. Specify the run directory name.
  3. Specify the rule filename. You can use the View button to view or edit the rule file.
  4. (Optional) Click Load to populate the GUI with the settings in the rule file.

    A rule file can also be specified in a runset. After loading that runset, the rule file is included in the control file but its settings are not be loaded into the GUI until you click Load.

    Tip

    The setting “Read Inputs/Outputs fields while loading rules file” (Setup > Preferences, Misc tab) controls whether or not rule file settings populate the GUI options on the Inputs and Outputs panes when Load is clicked. This setting is enabled by default. See Table 1 for more information.

    After you load a rule file, any information you specify in the GUI supersedes the information in your loaded rule file.

  5. (Optional) View the layer derivation tree at the bottom of the Rules pane; see “Layer Derivations Tree” for information.
