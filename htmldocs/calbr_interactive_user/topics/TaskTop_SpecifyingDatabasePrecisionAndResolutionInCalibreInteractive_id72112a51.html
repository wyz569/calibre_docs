Specifying Database Precision and Resolution in Calibre Interactive

You can specify the SVRF Precision and Resolution statements in the Calibre Interactive control file.

Procedure

  1. Click app Options on the left panel, where app is one of DRC, LVS, DFM, PEX, or PERC. If the button is not present, choose Setup > app Options from the menu to display it.
  2. Click the Database tab.
  3. Click the Precision tab.
  4. To set the precision, enable the “Specify precision” checkbox, and enter the desired precision in the text entry field. See Precision in the Standard Verification Rule Format (SVRF) Manual for information on setting this parameter.

    A compiler error results if the database precision does not match the rule file precision. To change this error to a warning and allow the DRC run to proceed, enable “Override layout value.”

  5. To set the resolution, select “Specify resolution” and enter the desired resolution in the text entry field. See Resolution in the Standard Verification Rule Format (SVRF) Manual for information on setting this parameter.
