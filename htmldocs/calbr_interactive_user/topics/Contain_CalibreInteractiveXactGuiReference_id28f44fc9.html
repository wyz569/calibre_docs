Calibre Interactive xACT GUI Reference

The settings for a Calibre xACT run are made in several pages selected from the left panel of the GUI.

In most cases the name of the GUI control corresponds to an SVRF command or command option. Tooltips provide additional information. Choose Settings > Show Pages to hide and show certain pages. Some pages have a shortcut, or hotkey, for going to the page; this is indicated in parentheses.

Table 1. Calibre Interactive xACT GUI Items

GUI Item

Description

Rules Page

(Alt+r / Alt+R)

Specify the rule file and the run directory on the Rules page.

Click the button to load rule file settings into the GUI. The rule file settings overwrite current GUI settings, including the path to the input design and the top cell name.

To specify additional rule files, click Options on the left panel, select “Include Rules Files,” and expand the control area to specify the additional rule files.

Inputs Page

(Alt+i / Alt+I)

Specify the input layout, schematic source (if used), and Xcell and Hcell files.

To specify additional layout or schematic input files, click Database on the left panel and expand the Library area.

Outputs Page

(Alt+o / Alt+O)

Specify parameters related to the output of the run, such as the extraction mode, process corner, PEX Netlist, Mask SVDB, and report options.

Options Page

(Alt+p / Alt+P)

Specify options related to extraction.

LVS Page

Specify LVS related options. You can specify the following SVRF statements: LVS Power Name, LVS Ground Name, LVS Recognize Gates, LVS Filter Unused Option, and LVS Report Option.

Inductance Page

Specify options related to inductance. This page is only displayed if inductance extraction is selected on the Outputs page or if “PEX Xact Control File” is checked on the Options page.

FieldSolver Page

Specify fieldsolver settings. You can specify the following SVRF statements: PEX Fieldsolver Mode … NETS, PEX Fieldsolver Mode … LAYERS, PEX Fieldsolver Boundary, and PEX Fieldsolver Cell_array.

This page is only displayed if the “xACT Mode” on the Outputs page is “xACT 3D” or “xACT 3DSelect.”

Database Page

Specify the SVRF statements Resolution and Precision, and additional layout and schematic input files.

OPENACCESS Page

Specify database conversion options for third party databases.

The page is only displayed if “Layout Format” on the Inputs page is OPENACCESS.

Also see “Providing OpenAccess Input with Calibre Interactive xACT”.

Environment Page

View and specify environment variables for the run.

V2LVS Page

Specify options for the v2lvs translator, which translates VERILOG format source input to SPICE format.

The V2LVS page is available when VERILOG or MIXED is selected as the “Source Format” on the Inputs page. This page is similar to the Setup Verilog Translator dialog box in Calibre Interactive nmLVS—see “Setting Up the Verilog Translator (v2lvs)”.

Run Control Page

(Alt+c / Alt+C)

Specify the run mode, processor information, options for invoking Calibre RVE, whether to run Calibre nmLVS, and settings for connecting to a design tool.

See “Connecting to a Design Tool in Calibre Interactive xACT”.

Triggers Page

Specify trigger functions that execute before and after the run.

Templates Page

Specify settings for file-naming templates.

Preferences Page

Specify settings for loading and saving the runset, saving the transcript, and loading a custom configuration file.

Search Toolbar

(Ctrl+f / Ctrl+F)

Search the GUI for a specified string.

Transcript Page

(Alt+t / Alt+T)

Display the transcript of the run. Errors and warnings are listed in the bottom portion of the page.

Right-click in the text display for a menu. Also see Transcript preferences on the Preferences Page.

Run xACT button

(Alt+d / Alt+D)

Starts the Calibre xACT run.

To view the Calibre Interactive control file, click the Run xACT button while pressing the Ctrl key.

Start RVE button

Open Calibre RVE and load the Mask SVDB database specified on the Outputs page.

Errors and warnings are indicated as follows:

A missing or incorrect setting. Hover over the icon for a message about the error. A red exclamation point ( ! ) is also displayed by the page name if any settings have an error.

A warning, such as for a conflicting setting. The run can proceed with warnings. Hover over the icon for a message about the warning. The warning icon is not displayed next to the page name.
