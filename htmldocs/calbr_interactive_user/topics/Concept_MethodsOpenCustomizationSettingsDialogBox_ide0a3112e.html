Methods to Open the Customization Settings Dialog Box

You can open the Customization Settings dialog box automatically when the runset is opened, manually with the Customization button, or during a Calibre Interactive batch run.

You can also load a different customization file with the Load button in the Customization GUI.

Requirements

  • A customization file is specified as described in “Methods to Specify a Customization File”.

  • Calibre Interactive is open.

Methods

Choose one of the following methods to open the Customizations Setting dialog box:

  • Automatically open the Customization GUI when the runset is loaded

  • Manually open the Customization GUI

  • Open the Customization GUI in Calibre Interactive batch mode

  • Specify and load a different customization file from the Customization GUI

Automatically open the Customization GUI when the runset is loaded

  1. Choose Setup > Preferences from the main menu, then click the Startup tab.

  2. Enable “Show customization dialog after opening runset.”

  3. Save the runset with File > Save Runset.

    The next time you load the runset, the Customization GUI is opened automatically.

Manually open the Customization GUI

  • Click the Customization button on the left panel of Calibre Interactive.

    If the Customization button is not present, make sure a customization file is specified.

Open the Customization GUI in Calibre Interactive batch mode

  • Specify the Customization GUI on the command line with the options “‑customgui filename ‑batch”. See “Customization GUI Operation in Batch Mode” for details.

Specify and load a different customization file from the Customization GUI

  1. Click the Customization button on the left panel of Calibre Interactive.

  2. Click the Load button at the bottom of the dialog box.

  3. Specify a new customization file.

  4. Click OK.

    The new customization file is loaded and the settings in the previously loaded Customization GUI are discarded. Customization override settings in the runset are not applied when the new customization file is opened using the Load button.

    The setting for “Customization File” on the Startup tab of the Setup Preferences dialog box is updated with the new customization file.
