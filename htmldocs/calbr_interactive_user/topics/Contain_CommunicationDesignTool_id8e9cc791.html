Communication with the Design Tool

When your Calibre Interactive session is connected to a supported layout or schematic viewer, you can use the open design as input for your Calibre run; this is referred to as layout and schematic export. Socket ports are used for communication between Calibre Interactive and the design tool.

Calibre Interactive can automatically set up communication with a design tool or you can manually make the connection.

Automatic Connection — If you invoke Calibre Interactive from a layout or schematic viewer, communication between the two applications is set up automatically. This invocation method is explained in “Invoking Calibre Interactive From a Layout or Schematic Viewer”.

Manual Connection — If you open a design tool after invoking Calibre Interactive, you can set up communication between the two tools through IPC sockets. See “Setting the Socket Port in Calibre Interactive”.
