Specifying Inputs for a Calibre PERC Run

The inputs you specify for a Calibre PERC run depend on the type of analysis you choose. The basic procedure for specifying Calibre PERC inputs is outlined here; see the individual sections for detailed instructions.

Procedure

  1. Select the type of Calibre PERC run, as described in “Specifying the PERC Run and Analysis Type”.
  2. Specify the inputs needed, depending on the analysis type:

    • “PERC (Hierarchical)” and “PERC (Flat)” runs:

      • Layout — Specifying Calibre PERC Inputs for Layout Analysis

      • Layout Netlist — Specifying Calibre PERC Inputs for Layout Netlist Analysis

      • Source Netlist — Specifying Calibre PERC Inputs for Source Netlist Analysis

    • “PERC-LDL” runs:

      • Layout — Provide layout information using instructions in “Specifying Calibre PERC Inputs for Layout Analysis” as a guideline.

      • Source-Based Flow — Provide layout and source information, using instructions in “Specifying Calibre PERC Inputs for Layout Analysis” and “Specifying Calibre PERC Inputs for Source Netlist Analysis” as a guideline.

    The source of the required input can be an open layout or schematic viewer, or a file, as explained in “Input Sources for a Calibre PERC Run”.

  3. Specify Hcells as described in “Supplying Hcell Data for a Calibre PERC Run”.
  4. If you are doing a layout analysis, specify other options as needed, as described in “Specifying Multiple Layouts for a Calibre PERC Run” and “Setting Database and General Options in Calibre Interactive”.
  5. (Optional) Specify a Layout Text file as described in “Specifying a Layout Text File in Calibre Interactive”.
