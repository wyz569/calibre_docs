Setup Preferences

You can control many behaviors of the Calibre Interactive GUI with setup preferences. Setup preferences are specified with the Setup Preferences dialog box, which is opened by selecting Setup > Preferences. Startup Preferences are saved automatically to special setup files, while all other preferences are saved to the runset.

See “Saving Setup Preferences” for information on saving preferences.

The Setup Preferences dialog box has the following tabs:

Table 1. Setup Preferences Dialog Box Tabs

Tab

Description

Startup

Defines startup behavior.

See “Startup Preferences”.

Templates

Defines file naming templates.

See “Templates for File Naming” and “Export from Viewer Preferences”.

Triggers

Specifies trigger scripts.

See “Trigger Functions in Calibre Interactive”.

Disable

Disables certain GUI options and sets license timeout behavior.

See “Disable Preferences”

Misc

Defines miscellaneous preferences.

See “Miscellaneous Preferences”

Info

Defines optional descriptive information about the runset.

See “Runset Type and Runset Info”.

CTO File

(applies to DRC only)

Defines the viewer layer map file to use when creating a CTO file.

See ““Creating a Check Text Override (CTO) File Using Calibre Interactive nmDRC”.

HTML Report

(applies to DRC only)

Specifies options to use when creating the DRC HTML Report. Creation of the DRC HTML Report is enabled with the “Create HTML Report” checkbox on the Outputs pane of Calibre Interactive nmDRC.

See ““Creating a DRC HTML Report from Calibre Interactive nmDRC”.
