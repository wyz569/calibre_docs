Setting Power Supply Options

You use the Supply tab to specify power and ground net names and to choose related options.

Procedure

  1. Click LVS Options to display the Options pane. If the LVS Options button is not available, choose Setup > LVS Options.
  2. Click the Supply tab.
  3. Enter the power and ground net names using one of these methods:

    • Enter a space-separated list of net names in the text field.

    • Click the Load from file button and specify a file. The file should be an ASCII text file with the net names specified as a space-separated list or given on separate lines.

      For example, loading the following file:

      A B C D
NET_IN1
NET_OUT2

      results in the list “A B C D NET_IN1 NET_OUT2” being entered in the text field.

  4. Enable “Abort LVS on power/ground net errors,” if desired. This option causes Calibre nmLVS to abort on encountering any power/ground net problems. An abort can be caused by any of the following reasons:

    • The power or ground names conflict with another name found on the net.

    • The same power or ground name was found on two disjoint nets.

    • An unattached power or ground name was found.

    • A stamp error occurred in a power or ground net.

      Tip

      You must specify power and ground net names if you invoke LVS logic gate recognition.
