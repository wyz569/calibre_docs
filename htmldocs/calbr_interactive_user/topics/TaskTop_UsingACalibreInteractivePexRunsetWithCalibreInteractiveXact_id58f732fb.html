Using a Calibre Interactive PEX Runset with Calibre Interactive xACT

Calibre Interactive xACT can read in runsets created for Calibre Interactive PEX. Supported runset options are automatically converted.

Prerequisites

  • A Calibre Interactive PEX runset

Procedure

  1. Invoke Calibre Interactive xACT and specify the PEX runset to load.

    You can specify the runset on the command line with “‑runset pex_runset” or load the runset from the GUI with File > Load Runset.

    If there were issues reading the PEX runset, the view automatically switches to the Transcript page, where warning, informational, and error messages are reported.

  2. If the Transcript page contains messages related to runset options, handle them as follows:

    Message

    Problem and Action

    WARNING … runset_option: Value of value is not supported …

    The option value is not supported. Supply a valid value in the PEX runset and reload the runset, or correct the problem in the GUI.

    WARNING … Conversion for runset_option not found -- (skipping)

    The PEX runset option does not have a conversion. The option may not be valid in the Calibre Interactive xACT GUI. Confirm the validity of the option and contact your Siemens representative if needed.

    WARNING … Keyword runset_option not recognized -- (skipping)

    The PEX runset option is not recognized. Confirm the validity of the option and contact your Siemens representative if needed.
    Note:

    A customization file used with Calibre Interactive PEX cannot be used in Calibre Interactive xACT.

  3. Look for icons in the left panel of the GUI, indicating an incorrect or missing setting.

    Click the page name with the error, locate the problem option, hover over the to view a message about the problem, and correct as needed.

  4. View each GUI page and search for the icon. The icon indicates a warning, such as an option setting that conflicts with another option. Hover over the icon to view a message about the problem and correct as needed. Warning icons are not displayed by the page name in the left panel of the GUI.
  5. Choose File > Save Runset As to save the runset in the new format. You are not allowed to overwrite the PEX runset.
