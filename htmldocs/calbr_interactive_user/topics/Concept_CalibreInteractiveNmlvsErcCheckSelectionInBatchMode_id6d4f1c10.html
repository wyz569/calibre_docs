Calibre Interactive nmLVS ERC Check Selection in Batch Mode

ERC checks in batch mode for Calibre Interactive nmLVS are selected using a customization file.

The settings in the Select Checks dialog box for ERC cannot be used in batch mode because Calibre Interactive runsets do not save the check selections in the Select Checks Dialog Box. You must select ERC checks in the rule file, or select them using a customization file.

A customization file can include a CUSTOM::CHECK command to select the rule checks that are run. The customization file is specified with the -customgui or ‑custom command line options in batch mode.

For example: 

calibre -gui -lvs -runset_options_display lvs_runset 
	-customgui lvs_custom_checks -batch > batch_gui.log

Where lvs_custom_checks is the name of the customization file; an example can be found at “Customization File to Select ERC Rule Checks”.
