Including/Excluding Nets

By default, Calibre extracts parasitics from all nets in your layout. You can include or exclude certain nets from extraction.

Procedure

  1. Click Outputs on the left panel.
  2. Click the Nets tab.
  3. Select the nets for parasitic extraction:

    • All Nets — Specify all nets for parasitic extraction.

    • Specified Nets — Select nets to include and exclude.

      You can specify top level and recursive nets to include or exclude from extraction.

      You can specify nets manually by listing them in the text field, or click the select button to select nets in a connected schematic viewer.

      • Manual selection — Specify source net names if you chose the SCHEMATIC option for naming nets in the extracted netlist; specify layout net names if you chose LAYOUT naming.

      • Select in schematic — The browse buttons are active only if SCHEMATIC is selected in the “Use Names From” dropdown list.

        Note: For Cadence Composer, you must press Ctrl-C on the keyboard to end the net selection process and enter the selected nets. (For Calibre 2012.3 and earlier releases, use the Esc key to end the net selection process.)

  4. If the extraction mode is “xACT 3D/xRC Hybrid,” then choose Setup > PEX Options and the xACT3D tab to specify the subset of nets extracted using the xACT 3D fieldsolver.

    Specify a setting for “Extract parasitics using xACT 3D for”:

    • All nets — xACT 3D extraction is done for all nets.

    • Specified nets — Specify nets to include or exclude from xACT 3D extraction. See the discussion with Step 3 for directions on selecting nets.

      For included nets, you can also specify the accuracy, as follows:

      200 — Less accurate, with faster runtime. This is the default.

      600 — More accurate, with slower runtime.

  5. (Optional) If you selected “Specified nets” and “Include” in Step 3, you can specify to keep coupling capacitance separate from intrinsic capacitance with the following steps:
    1. Click the PEX Options button or choose Setup > PEX Options.
    2. Click the Netlist tab and the Reduction and CC sub-tab.
    3. Enable “Keep coupling capacitors separate.”

    The ‑cselect command line option is used instead of ‑select if “Keep coupling capacitors separate” is enabled. This option may only be used with extraction types of C+CC, R+C+CC, and R+C+CC+L.

  6. SVRF commands: PEX Extract Include and PEX Extract Exclude.
    Tip

    You can include PEX Extract Include and PEX Extract Exclude statements in your rule file. When you click Load on the Rules pane, Calibre Interactive populates the Exclude and Include fields with the nets specified in the statements. Since GUI settings override the rule file settings, you must also select “Specified Nets” and the desired “Exclude” or “Include” options for the settings to take effect.

    Note: There is a limit of 16382 characters per line in the rule file for these SVRF statements. There is no character limit for entries in the corresponding GUI fields.
