Reading Layout Netlist Input from File in Calibre Interactive nmLVS

Layout netlist data for a Calibre Interactive nmLVS run is specified on the Inputs pane on the Layout tab. You can read the layout netlist from an existing file. The layout netlist is used in Netlist vs Netlist runs.

Prerequisites

  • The Calibre nmLVS analysis type is “Netlist vs Netlist.” See “Specifying the Calibre nmLVS Run and Analysis Type”. The options available for layout data depend on the run and analysis type.

  • A layout netlist in SPICE format. The layout netlist may be one extracted and saved from a previous Layout vs Netlist or Netlist Extraction run.

Procedure

  1. Click the Inputs button on the left panel.
  2. Click the Layout tab.
  3. Uncheck “Export from schematic viewer” to specify that the layout netlist is read from file.
  4. Specify the filename for the layout netlist in the “File” field.
  5. Specify the name of the top cell in the “Top Cell” field.
