Running Calibre Interactive PEX

Use the following procedure to start a parasitic extraction run from Calibre Interactive—PEX.

Prerequisites

  • Calibre Interactive PEX is open; see “Invoking Calibre Interactive Tools”.

Procedure

  1. Enter all necessary information in the Rules, Inputs, Outputs, and Run Control panes. Options may also be specified on the PEX Options pane, if desired. All left panel buttons are be green when the necessary information is entered.
  2. Choose Setup > PEX Options and select each of the option tabs to set additional options. All left panel buttons will be green when the necessary information is entered.
  3. If you want to run each step in the parasitic extraction process separately, see “Controlling PEX Step Execution” for instructions.
    Tip

    If you are using a TVF rule file, the TVF file is compiled and converted to SVRF format at each step. To save the converted rule file and re-use it for each execution step, enable “Save TVF Rules to SVRF before running Calibre” (Setup > Preferences, Misc tab).

  4. If desired, choose Setup > Preferences then the Triggers tab to specify triggers. See “Trigger Functions in Calibre Interactive”.
  5. Click Run PEX on the left panel of Calibre Interactive.

    If the run uses a source netlist, Calibre Interactive first verifies that the source netlist is complete; the source netlist check is case insensitive. If an internal pre-execution trigger is defined, this is done after the trigger executes. You can turn off the source netlist check by the choosing Setup > Preferences, clicking the Misc tab, and disabling the setting “Check source netlist is complete before running PEX.” See “Miscellaneous Preferences” for information on the source netlist check.

    When the run starts, the view switches to the Transcript pane, and any run output, including errors and warnings, is displayed there.

  6. View the transcript pane and examine the results of the run. See “Transcript Pane in Calibre Interactive”.
    Tip

    You can generate and view a control file without running Calibre by holding down the Ctrl key and clicking Run PEX. In addition, the calibre command line for the run is written to the transcript pane.

Results

If an error occurs in Calibre Interactive before the Calibre run is started, the error is displayed in a pop-up window or in the Calibre Quality Agent for Calibre Interactive.

When the Calibre run starts, the view switches to the Transcript pane, and any run output, including errors and warnings, is displayed here. See “Transcript Pane in Calibre Interactive”.

If “Start RVE after PEX” is checked, the results database is opened in Calibre RVE when the run is complete; see “Using Calibre RVE for PEX” in the Calibre RVE User’s Manual. In addition, the Calibre Interactive GUI may open a report file in a text viewer when the run completes; see “Viewing and Editing Text Files in Calibre Interactive”.
