Inputs for a Calibre Interactive DFM Run

Layout input for a Calibre Interactive DFM run can be exported from an open design tool or read from file. You can also specify multiple layout inputs.
Table 1. Sources for Layout Data in Calibre Interactive DFM

Open Cell in Design Tool

Files on Disk

Layout data comes from the active cell in the layout editor. Calibre Interactive prompts the editor to generate an intermediate layout file at run time.

 

Check “Export from layout viewer” on the Inputs pane.

Layout data exists as one or more layout files used for input. No viewer is connected to Calibre Interactive.

 

 

Uncheck “Export from layout viewer” on the Inputs pane.
Tip

You can set a preference for the setting of “Export from layout viewer” that depends on whether your Calibre Interactive session is connected to a layout viewer. By default, this preference is set so that “Export from layout viewer” is enabled if Calibre Interactive is started from a layout viewer, regardless of the saved runset value for the option. See “Export from Viewer Preferences”.
