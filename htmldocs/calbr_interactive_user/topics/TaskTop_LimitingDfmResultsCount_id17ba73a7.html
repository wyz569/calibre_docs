Limiting the DFM Results Count

You can control the number of results reported for each DFM rule check. This reporting option is only applicable to ASCII RDB output.

Procedure

  1. Click the DFM Options button on the left panel, or choose Setup > DFM Options if the button is not visible.
  2. Click the ASCII Results tab.
  3. Select the maximum result count in “Max errors generated per check”:

    • All — Report all results. Choose this option if you are using Calibre DFM to output a modified version of your layout as mask data.

    • Number — Enter the maximum allowed result count.
