Performing Area DFM

You can run Calibre DFM on only a specified rectangular area of the layout. Area DFM runs on the specified area plus a halo region around the area. You can specify the halo size and whether results in the halo region are filtered out or kept.

Procedure

  1. Click Inputs on the left panel and enter the layout database information as described in “Inputs for a Calibre Interactive DFM Run”.
  2. Enable the Area option.
  3. Select the area coordinates with one of the following methods:

    • Enter space-separated coordinates for the rectangular window you want to check. Coordinates are the lower-left and upper-right corners of the rectangle, in that order. You must use this method if not connected to a layout viewer.

    • If connected to a layout viewer, click the browse button (...) to select the area you want to check in the layout viewer. Click and drag out an area selection rectangle In the layout viewer. The coordinates of the area selection rectangle appear in the Area field of the Inputs pane.



      Tip

      When connected to a layout viewer, you can right-click the (...) button to highlight and zoom to the specified Area DFM region in the layout.

  4. (Optional) Set Area DFM options, such as halo size and coordinate scaling. Some options depend on the run mode. Options are not available for Critical Area Analysis (CAA) runs.
    1. Click the DFM Options button on the left panel, or choose Setup > DFM Options if the left panel button is not present.
    2. Click the Area DFM tab, then select the desired options described in the following table.
      Table 1. Area DFM Options

      Option

      Description

      Halo width for area DFM

      Specifies the size of the halo which expands the Area DFM window.

      • Automatic — Width of halo is the greater of the width or height of the specified Area DFM region.

      • Size — Width of halo is an explicit user-specified width in user-units.

      Note: Depending on the geometry of your area selection, the automatic setting can result in a halo region that is much larger than needed, resulting in slow performance.

      Remove results from halo region after area-DFM

      (Not available for CMP Analyzer runs. Only applies to the ASCII format results database.)

      Enabled — (default). As described.

      Disabled — Keep results in the halo region.

      Apply coordinate scaling

      Enabled — (default) Scale window coordinates by:

         (DFM Results Database
    Precision⁄Precision) *(Layout Magnify)

      Disabled — Do not apply scaling.

      Shrink factor

      (Only available for CMP Analyzer runs.)

      Apply a shrink factor to the layout. This is useful if you want to simulate the effect of an optical shrink factor on the layout.

      Adds Layout Magnify and DRC Magnify Results statements to the Calibre Interactive control file.

      Extent layer

      (Only available for CMP Analyzer runs.)

      Specifies an extent layer.

      The extent layer can be specified as a layer number or in layer_number.datatype format. If you specify only a layer number, then the datatype is assumed to be 0 (zero).
  5. If all other options are set, click the Run DFM button to start the run.

Results

The following changes in output occur when Area DFM is enabled:

  • Results database — Results in the halo region are filtered out based on the setting “Remove results from halo region after area-DFM”. Result filtering only applies to ASCII format results databases.

  • Auxiliary databases — If an auxiliary results database is specified in the rule file, the results are filtered as specified for the main results database.

  • Transcript and summary report output — If the setting “Remove results from halo region after area-DFM” is enabled, then both the transcript and the DFM Summary Report report the number of results that were filtered out of the halo region.
