Adding the PRIMETIME command to a PEX NETLIST Statement

You can add the PRIMETIME keyword to your Pex Netlist statement if you are using either the SPEF or DSPF netlist formats in RC mode. The optional keyword PRIMETIME creates a more compact netlist that the PrimeTime® static timing analysis tool can parse.

Procedure

  1. Click Outputs button on the left panel.
  2. Click the Netlist tab.
  3. Choose SPEF or DSPF for the netlist format in the “Format” dropdown button list.
  4. The PRIMETIME option is available at the bottom of the “Format” dropdown button list. Click the checkbox to enable it.

    Note: If the combination of netlist format and extraction type is not allowed the format indicator is printed in red.

  5. See PEX Netlist in the SVRF Manual for complete details.
