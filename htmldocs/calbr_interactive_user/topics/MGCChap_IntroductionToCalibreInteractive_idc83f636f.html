Introduction to Calibre Interactive

Calibre® Interactive™ provides a GUI interface to several Calibre tools. The interface enables you to customize rule file settings and run-time options. You can submit jobs on local, remote, or distributed computer resources.
Note:

As of the 2019.2 release, information on Calibre RVE is now in the Calibre RVE User’s Manual.
