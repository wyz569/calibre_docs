Preserving Cells During DRC

When performing hierarchical DRC, Calibre may expand certain cells for improved performance. The process of expanding a cell removes one level of hierarchy by promoting it into the parent cell. If a DRC error occurs in a cell that gets expanded, it is reported in the parent cell.

If you used Calibre RVE for DRC to waive certain errors from a previous Calibre nmDRC run, it is possible that an error in a waived cell will be reported in a parent cell if the cell with the waived result is expanded. For information on waiving results, see “Waiving, Marking, Sorting, and Grouping DRC Results in Calibre RVE” in the Calibre RVE User’s Manual.

If you used Calibre RVE to waive cell-resident errors from a prior run and you do not want the cells containing those errors to be expanded, you can specify that certain cells are preserved from expansion.

Prerequisites

  • You are running hierarchical Calibre nmDRC.

  • You know one of the following in order to specify which cells containing waived results should not be expanded during the Calibre nmDRC run:

    • Names of the cells that contain waived errors.

    • Name of the .waived file used by Calibre RVE for DRC. The file is automatically named drc_results_database.waived, where drc_results_database is the name of the DRC results database.

Procedure

  1. Click DRC Options on the left panel. If the DRC Options button is not present, choose Setup > DRC Options from the menu to display it.
  2. Click the Cells tab.
  3. If you wish to preserve all cells that contain waived results in the .waived file, enable “Preserve cells from RVE waiver files” and enter a filename in the associated text entry box.

    If you choose this option, the specified .waived file is automatically loaded into Calibre RVE for DRC if you start Calibre RVE from your Calibre Interactive session.

  4. Enable “Preserve Cells from list” to specify individual cells that are protected from expansion. You can enter cell names in these ways:

    • Enter a space-separated list of cells in the text entry box.

    • Read cell names from an existing waiver file — Click the Import from RVE Waiver File button and select a .waived file. All the cells that contain waived results are entered into the cell list. You can then delete or add cell names as needed.

      Note:

      Choosing to preserve cells from automatic expansion may increase run times in some cases.

    Corresponding rule file statements: Layout Cell List and Layout Preserve Cell List.
