Run Control Page in Calibre Interactive xACT

To access: Click Run Control in the left panel of the Calibre Interactive xACT GUI. (shortcut: Alt‑c)

The Run Control Page in Calibre Interactive xACT has settings for the host processor and run options, connection to a design tool, and Calibre RVE invocation.

Objects

Category

Description

Run Calibre On

Settings for the host processor.

  • Local Host — Calibre runs on the host currently running Calibre Interactive.

  • Remote Host — Calibre runs on the specified remote host.

  • Cluster Command — Calibre runs on a custom computer cluster. Enter the appropriate command in the Cluster Command field. You can use the replaceable parameters listed in the “Parameter Substitution Information” area.

    Synchronous checked — Both steps (LVS and xACT) are run. Calibre RVE and reports are displayed when the run is finished if appropriate options are set.

    Synchronous not checked — Only one step (LVS or xACT) is run. The run transcript, Calibre RVE, and reports are not displayed when the run finishes.

  • Platform LSF — Calibre runs on the IBM® Spectrum™ LSF® computing system.

Also see the Calibre Licensing and Remote Calibre Environment options when Remote Host or Cluster Command is selected.

Run Calibre

Settings for run mode (single-threaded, multithreaded, or Calibre MTflex), number of processors (‑turbo), and behavior if licenses are not available for all requested CPUs.

  • Single Thread — Single-threaded execution.

  • Multiple Threads — Multithreaded execution on a single host computer. Specify the number of CPUs to use in the entry field. (‑turbo num_processors)

  • All Threads — Multithreaded execution on a single host computer with all available processors. (‑turbo)

  • Distributed (MTflex) — Use your existing computer network as a distributed computing platform with Calibre® MTflex™.

    Specify the remote hosts file, which is used as the parameter for the ‑remotefile command line argument.

    If desired, specify Remote Data Server (RDS) and the number of servers. See “Remote Data Server” in the Calibre Administrator’s Guide.

RVE Options

  • Show Results in RVE — Automatically open the results database in Calibre RVE when the run finishes.

  • Start RVE after batch run — Automatically open Calibre RVE after a batch run finishes.

  • Run RVE on local host — When checked, Calibre RVE runs on the local host. When unchecked, Calibre RVE runs on the host specified for “Run Calibre On”.

  • Close RVE before running Calibre xACT — As described.

  • Close RVE when Calibre Interactive closes — As described.

Design Tool Settings

Settings for connecting to a layout or schematic design tool, including the socket number.

See “Connecting to a Design Tool in Calibre Interactive xACT”.

xACT Steps

Includes the following checkbox:

  • Run LVS — Run Calibre nmLVS if PEX Netlist specifies the use of source names. Also see “Check source netlist is complete before running xACT” on the Preferences page in the “Misc” area.

Calibre Licensing

The “Remote Calibre” section is available when “Run Calibre xACT on” is set to Remote Host, Cluster Command, or Platform LSF.

  • License File Name — Select LM_LICENSE_FILE or MGLS_LICENSE_FILE as the environment variable used for licensing.

  • Use — Specify whether to use the current license file value or to use the value specified in the License File Value field.

Calibre Interactive License Timeout

Setting the for license timeout.

The license timeout can also be set with the environment variable MGC_CALGUI_RELEASE_LICENSE_TIME; the default unit is hours. See “License Timeout in Calibre Interactive” for instructions on using other units.

Remote Calibre Environment

This section is available when “Run Calibre xACT on” is set to Remote Host or Cluster Command.

Specify the value of MGC_HOME, the remote shell, and the remote username.
