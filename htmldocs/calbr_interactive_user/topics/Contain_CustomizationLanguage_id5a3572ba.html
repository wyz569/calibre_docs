Customization Language

Customization files use Tcl/Tk, with API commands defined in the CUSTOM namespace. The CUSTOM commands control the elements in the customization dialog. The customization file is sourced by Calibre Interactive as a Tcl file; therefore all Tcl/Tk statements and constructs are available to the customization file developer.

Starting with Calibre version 2019.2, Calibre uses Tcl 8.6. See “Tcl Library in Calibre” in the Calibre Administrator’s Guide for other general information.

You can write a customization file even if you do not already know Tcl/Tk. This section covers some basic concepts in Tcl and provides a list of references.
