Source Netlist Input in Calibre Interactive nmLVS

Calibre nmLVS comparison runs require a source (or schematic) netlist. Source data can be exported from an connected schematic viewer or exist in database files. Netlist extraction runs do not use a source netlist.

Input Method

Comment

Source netlist input from a connected schematic viewer

See “Reading Source Netlist Data from an Open Schematic Viewer in Calibre Interactive nmLVS”.

The schematic viewer should be set up to export the netlist in SPICE or Verilog format.

Source netlist input from file

See “Reading Source Netlist Data from File in Calibre Interactive nmLVS”.

The netlist format can be SPICE, Verilog, or a combination of both types of files.
