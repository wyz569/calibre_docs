Reading Geometric Layout Input from an Open Design Tool in Calibre Interactive nmLVS

Layout data for a Calibre Interactive nmLVS run is specified on the Inputs pane on the Layout tab. You can export the layout from an open design tool. Geometric layout data is used in Layout vs Netlist and Netlist Extraction runs. The layout netlist is extracted from the geometric layout.

Prerequisites

  • The Calibre nmLVS analysis type is “Layout vs Netlist” or “Netlist Extraction.” See “Specifying the Calibre nmLVS Run and Analysis Type”. The options available for layout input depend on the run and analysis type.

Procedure

  1. Click the Inputs button on the left panel.
  2. Click the Layout tab.
  3. Enable “Export from layout viewer” to specify that the layout is exported from a connected design tool; see “Communication with the Design Tool” for more information.
  4. Select the exported layout format using the “Format” dropdown button list. The selections GDSII, OASIS, and LEFDEF are available if “Export from layout viewer” is selected. See “Providing LEF/DEF Input” for more information on LEF⁄DEF input.
  5. Specify the filename for the exported layout database in the “File” or “DEF File” field.

    If you are performing a Calibre Interactive batch run, see “Specifying Layout and Schematic Export for a Calibre Interactive Batch Run”.

  6. Verify the name of the top cell name in the “Top Cell” field. The top cell is determined from the current cell in the viewer.
  7. Enter the name for the extracted layout netlist in the “Layout Netlist” field. This option is for hierarchical runs only; Calibre nmLVS extracts the layout netlist from the geometric database and writes it to this filename.

    When connected to a layout viewer, a system-generated SPICE netlist name appears in the “Layout Netlist” field by default. This name is taken from the “Extract to File” template setting (Setup > Preferences, the Templates > Outputs sub-tab). However, you can choose a different filename.

  8. (Optional) If you have multiple layout inputs for your run, you can specify them using the Database > Library sub-tab on the LVS Options pane. See “Specifying Multiple Layouts for an Calibre nmLVS Run”.
  9. Possible next steps are:

    • “Specifying a Layout Text File in Calibre Interactive”

    • “Source Netlist Input in Calibre Interactive nmLVS”
