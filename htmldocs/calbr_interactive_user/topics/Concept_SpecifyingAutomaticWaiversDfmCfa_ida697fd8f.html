Specifying Automatic Waivers for DFM CFA

Calibre Auto-Waivers with Critical Feature Analysis allows you to waive certain results and run Calibre DFM with these waivers taken into account. The waiver_flow executable is used to generate waiver shapes, and can also be run from Calibre Interactive - DFM when you are performing Critical Feature Analysis (CFA).

The controls for Calibre Auto-Waivers are on the Inputs pane on the Waivers tab; see “Calibre Auto-Waivers in Calibre Interactive” for instructions on using this tab.
