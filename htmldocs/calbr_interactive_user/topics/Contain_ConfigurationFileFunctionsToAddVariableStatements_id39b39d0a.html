Configuration File Functions to Add Variable Statements

Several functions are available for the Calibre Interactive xACT custom configuration file that add a Variable statement to the Calibre Interactive control file.

See “Configuration File Format for xACT GUI Customization” for information on the file format and how to invoke Calibre Interactive xACT with a custom configuration file.

Command

Description
addChoicesVariableAdds a control to specify a Variable statement with a value to the Calibre Interactive xACT GUI; the value is taken from an enumerated list. The statement is added to the control file.
addListVariableAdds a control to specify a Variable statement to the Calibre Interactive xACT GUI, where the assigned value for the variable is a list. The statement is added to the control file.
addInputDirTextVariableAdds a control to specify a Variable statement with a string value to the Calibre Interactive xACT GUI, where the string value is expected to correspond to a directory used for input. The Variable statement is added to the control file. The control includes a text entry field and a browser button for selecting the directory.
addInputFileTextVariableAdds a control to specify a Variable statement with a string value to the Calibre Interactive xACT GUI, where the string value is expected to correspond to an input file. The Variable statement is added to the control file. The control includes a text entry field, a file browser button for selecting the file, and a view file button.
addIntegerVariableAdds a control to specify a Variable statement with an integer value to the Calibre Interactive xACT GUI. The statement is added to the control file.
addMultiChoicesVariableAdds a control to specify a Variable statement with a value to the Calibre Interactive xACT GUI. The value is taken from an enumerated list and multiple values can be selected. The statement is added to the control file.
addOutputDirTextVariableAdds a control to specify a Variable statement with a string value to the Calibre Interactive xACT GUI, where the string value is expected to correspond to a directory used for output. The Variable statement is added to the control file. The control includes a text entry field and a browser button for selecting the directory.
addOutputFileTextVariableAdds a control to specify a Variable statement with a string value to the Calibre Interactive xACT GUI, where the string value is expected to correspond to an output file. The Variable statement is added to the control file. The control includes a text entry field, a file browser button for selecting the file, and a view file button.
addRealVariableAdds a control to specify a Variable statement with a real value to the Calibre Interactive xACT GUI. The statement is added to the control file.
addTextBoxVariableAdds a control to specify a Variable statement with a string value to the Calibre Interactive xACT GUI. The GUI control is a multi-line text box. The statement is added to the control file.
addTextVariableAdds a control to specify a Variable statement with a string value to the Calibre Interactive xACT GUI. The statement is added to the control file.
