Reading Source Netlist Data from an Open Schematic Viewer in Calibre Interactive nmLVS

You can read source netlist data from a connected schematic viewer. A source netlist is required in Layout vs Netlist and Netlist vs Netlist runs in Calibre Interactive nmLVS.

Prerequisites

  • The Calibre nmLVS analysis type is “Layout vs Netlist” or “Netlist vs Netlist.” See “Specifying the Calibre nmLVS Run and Analysis Type”. The options available for layout data depend on the run and analysis type.

  • Calibre Interactive is connected to a supported schematic viewer; see “Communication with the Design Tool”.

  • The connected schematic viewer is set up to export the netlist in SPICE or Verilog format.

Procedure

  1. Click the Inputs button on the left panel,
  2. Click the Netlist tab.


  3. Enable “Export from schematic viewer” to specify that a connected schematic viewer supplies the schematic source netlist.

    Batch runs: In addition, you must also disable the setting “Ignore layout and source export in batch mode” in the Setup Preferences dialog box on the Misc tab. Also see “Specifying Layout and Schematic Export for a Calibre Interactive Batch Run” for details on the viewer connection for a batch run.

  4. Specify the format of the exported source netlist using the “Format” dropdown button list.

    • SPICE — The exported netlist is in SPICE format.

    • VERILOG — The exported netlist is in Verilog format. If you specify Verilog as the source format, Calibre Interactive calls a utility named v2lvs to translate the Verilog netlist to SPICE. Choose Setup > Verilog Translator to set up the translation options; see the section “Setting Up the Verilog Translator (v2lvs)”. The translated SPICE netlist is then used for LVS comparison; the SPICE netlist name is taken from the “Files” field.

    The “MIXED” option is not available when “Export from schematic viewer” is enabled.

  5. Specify the filename for the exported source netlist.

    The exported netlist is written to the filename in the “Files” field.

  6. Verify the name of the top cell in the “Top Cell” field. The top cell is determined from the cell currently open in the viewer when “Export from schematic viewer” is enabled.
  7. (Optional) Specify additional SPICE netlist files. This may be necessary to provide INCLUDE files for the source netlist.
    1. Click the LVS Options button or choose Setup > LVS Options.
    2. Click the Database tab and the Library sub-tab.
    3. Use the buttons next to “Additional SPICE Files” to specify the required SPICE files.

    The additional files are included before the netlist specified on the Inputs pane in the Netlist tab. Calibre Interactive creates a file named _source.net_ with .INCLUDE statements for each netlist file; _source.net_ is used as the source netlist for the Source Path statement in the control file.

    Note:

    Calibre Interactive verifies that the source netlist is complete before starting the run. You can turn off this check by disabling “Check source netlist is complete before running LVS” (Setup > Preferences, Misc tab). See “Miscellaneous Preferences” for information on the source netlist check.

Results

The appropriate statements are written to the Calibre Interactive control file at runtime.
