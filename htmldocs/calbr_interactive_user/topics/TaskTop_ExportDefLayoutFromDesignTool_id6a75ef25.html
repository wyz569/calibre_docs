Export DEF Layout from Design Tool

If your design tool is connected to Calibre Interactive and is capable of exporting the layout in DEF format, you can export the currently open design to Calibre.

Not available in Calibre Interactive PEX.

Procedure

  1. Click the Inputs button on the left panel of Calibre Interactive.
  2. Click the Layout tab.
  3. Select a Format of LEFDEF.
  4. Enable “Export from layout viewer.”
  5. Specify the DEF output file in the DEF Files field. The design streamed out by the design tool is written to this file; the file is overwritten if it already exists.
  6. Provide LEF Input as follows:

    • Tech File — The LEF technology file.

    • Files — One or more LEF files.

    • Directories — LEF directories.

    At least one of the Tech File, Files, and Directories entries must be filled in for LEF input. The Tech File entry is optional as long as the technology information is in one of the LEF file or directory entries. If the Tech File entry is not used, then the LEF technology file should be first in the list of LEF files.

  7. Specify the “Top Cell.”
  8. Set streamout options in your design tool.
  9. Set other database conversion options as described in “Third Party Database Options”.
    Tip

    You can define a file naming template for the DEF file. Choose Setup > Preferences, the Templates tab, the Database sub-tab, and specify a template in the “DEF File” field. See “Templates for File Naming” for more information.
