Transcript Pane in Calibre Interactive

You display the Transcript pane by clicking the Transcript button on the left panel of Calibre Interactive. The Transcript pane displays the Calibre transcript (also known as the log), which provides valuable information about your Calibre run.

You can turn off transcript output if desired; choose Setup > Preferences, click the Disable tab, and enable “Disable transcript output.”

If your Calibre run generates errors or warnings, Calibre Interactive displays each error and warning in tabs in the bottom portion of the Transcript pane. In Calibre Interactive xACT, when multiple configurations are loaded, messages from each configuration file are appended to the transcript window.

You can select any error or warning to scroll the top half of the Transcript pane to the relevant portion of the run transcript, as shown in Figure 1. If there are multiple errors or warnings listed in a tab, you can scroll through them with the up and down arrow keys on the keyboard. The menu selection Transcript > Save Errors and Warnings As saves only the errors and warnings to a file.

Figure 1. Transcript Pane in Calibre Interactive

The following table describes the contents of the Transcript menu in Calibre Interactive.

Table 1. Transcript Menu Contents in Calibre Interactive

Menu Selection

Description

Save As

Save the current transcript.

Echo to file

Write the transcript to the specified file as Calibre executes.

Save Errors and Warnings As

Save only the errors and warning from the transcript. The errors and warnings are grouped together with a count for each given at the top of each section.

Search

Search in the Transcript pane for a specific character string.
