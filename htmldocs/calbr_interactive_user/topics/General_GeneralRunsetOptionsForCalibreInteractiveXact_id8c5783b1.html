Calibre Interactive xACT-specific runset options are prefixed with pex and are described in “xACT Runset Options”. See “Calibre Interactive xACT Runset File Format” for important information.

This setting provides the value for the Resolution statement. It can be a single integer value, or a list of two integers to provide the step size in the x and y directions separately.

cmn.virtualConnect.netNames.value

The list of net names to connect by name for the Virtual Connect Name statement. The wildcard character “?” matches zero or more characters.

This option is used when cmn.virtualConnect.vconnect.value= “SPECIFIEDNAMES” (the “Virtual Connect Name” option).

cmn.virtualConnect.report.value

The setting for the “Virtual Connect Report” option, which controls warnings produced when a virtual connection is made.

  • NO — Do not report connections.

  • YES — Report connections.

  • YES UNSATISFIED — Only report virtual connections that are not physically connected higher in the hierarchy.

Specifies whether to limit the maximum number of virtual connect warnings issued when virtual connect reporting is enabled.

cmn.virtualConnect.vconnect.specified

Boolean controlling cmn.virtualConnect.vconnect.

This option controls the “Virtual Connect Name” checkbox, which determines whether to use the statement Virtual Connect Name.

  • false — Do not use. Disjoint nets with the same name are treated as separate nets.

  • true — Use. Disjoint nets are connected by name with the setting specified by cmn.virtualConnect.vconnect.value.

cmn.virtualConnect.vconnect.value

Specifies how to form virtual connections of named nets.

  • ALLNAMES — Virtually connect disjoint nets with identical names.

  • SPECIFIEDNAMES — Virtually connect disjoint nets that are in a specified list. The list is defined by cmn.virtualConnect.netNames.value.
