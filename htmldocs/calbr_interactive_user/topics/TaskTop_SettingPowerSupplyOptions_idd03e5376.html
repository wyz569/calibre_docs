Setting Power Supply Options

You use the Supply tab on the PERC Options pane to specify power and ground net names.

Corresponding rule file statements: LVS Power Name, LVS Ground Name

Procedure

  1. Click PERC Options to display the Options pane. If the PERC Options button is not available, choose Setup > PERC Options.
  2. Click the Supply tab.
  3. Enter the power and ground net names using one of these methods:

    • Enter a space-separated list of net names in the text field.

    • Select “Load from file” and specify a file. The file should be an ASCII text file with the net names specified as a space-separated list or given on separate lines.

      For example, loading the following file:

      A B C D
NET_IN1
NET_OUT2

      results in the list “A B C D NET_IN1 NET_OUT2” being entered in the text field.

      Tip

      The power and ground net names are used to populate the Calibre PERC net type keywords. See the Table PERC Net Type Keywords in the Calibre PERC User’s Manual.

      You must specify power and ground net names if you invoke LVS logic gate recognition.
