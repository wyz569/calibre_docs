Controlling Logic Gate Recognition in Calibre PERC

These options instruct Calibre to build the layout netlist using either transistors or logic gates.

Corresponding rule file statement: LVS Recognize Gates.

Procedure

  1. Click PERC Options to display the Options pane. If the PERC Options button is not available, choose Setup > PERC Options.
  2. Click the Gates tab on the PERC Options pane.
  3. Choose the desired gate recognition option. Simple gates use single-level logic (NAND, for example); non-simple gates use multi-level logic (AOI, for example).
  4. Decide whether to select the “Mix subtypes” option. If you do not select this option, transistors in series-parallel branches must all have the same subtype.
    Tip

    You must specify power and ground net names if you enable gate recognition. For more information on this topic, refer to “Setting Power Supply Options”.
