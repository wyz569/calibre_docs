Running Calibre Interactive nmDRC

Use the following procedure to start a Calibre nmDRC run from Calibre Interactive—nmDRC.

Prerequisites

  • A layout database.

  • A SVRF rule file.

Procedure

  1. Enter all necessary information in the Rules, Inputs, Outputs, and Run Control panes.
  2. Choose Setup > DRC Options and select each of the option tabs to set additional options. All left panel buttons will be green when the necessary information is entered.
  3. (Optional) Choose Setup > Preferences and click the Triggers tab to specify pre- and post-execution trigger functions. See “Trigger Functions in Calibre Interactive”.
  4. Click Run DRC on the left panel of Calibre Interactive.

    When the run starts, the view switches to the Transcript pane, and any run output, including errors and warnings, is displayed there.

  5. View the transcript pane and examine the results of the run. See “Transcript Pane in Calibre Interactive”.
    Tip

    You can generate and view a control file without running Calibre by holding down the Ctrl key and clicking Run DRC. In addition, the calibre command line for the run is written to the transcript pane.

Results

If an error occurs in Calibre Interactive before the Calibre run is started, the error is displayed in a pop-up window or in the Calibre Quality Agent for Calibre Interactive.

When the Calibre run starts, the view switches to the Transcript pane, and any run output, including errors and warnings, is displayed here. See “Transcript Pane in Calibre Interactive”.

If “Show results in RVE” is checked, the results database is opened in Calibre RVE when the run is complete; see “Using Calibre RVE for DRC” in the Calibre RVE User’s Manual. In addition, the Calibre Interactive GUI may open a report file in a text viewer when the run completes; see “Viewing and Editing Text Files in Calibre Interactive”.
