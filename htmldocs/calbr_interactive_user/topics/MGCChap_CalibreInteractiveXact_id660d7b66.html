Using Calibre Interactive xACT

You can use Calibre Interactive to run the Calibre® xACT™ direct netlisting flow. The GUI includes the following features:

  • A search option to easily locate controls

  • The ability to set environment variables within the GUI

  • GUI customization:

    • Design new panes, move the location of GUI controls, hide controls, and specify initial settings.

    • Attach a callback function to a signal from a GUI control. For example, a function to set netlist options in the GUI can be executed if the netlist format changes.

    • Specify #DEFINE, #UNDEFINE, and Variable statements in the Calibre Interactive control file.

  • Support for the direct netlisting flow in Calibre xACT and Calibre xACT 3D.

  • Integration with Calibre® DESIGNrev™, Cadence® Virtuoso®, Cadence® Encounter®, and Cadence® Innovus®.
