CUSTOM::SEPARATOR

Command type: Customization command for Calibre Interactive.

Inserts a horizontal separator line in the customization dialog box.

Usage

CUSTOM::SEPARATOR [-tool { DRC | LVS | PEX | DFM | PERC | tool_list }]
[ {{-master master_spec} [‑master_select {0 | 1}]} | ‑master_expr expr_spec ]

Arguments

-tool {DRC | LVS | PEX | DFM | PERC | tool_list }

If included, the statement applies only to the specified tool(s). To specify more than one tool, provide a Tcl list; for example: {DRC DFM}.

If ‑tool is not specified, the command applies to all tools.

{{-master master_spec} [‑master_select {0 | 1}]} | ‑master_expr expr_spec

An optional keyword and parameter set that specifies the current control as a primary control to another primary control in the customization file. In other words, this option specifies that the appearance of this GUI object in the customization window depends on the value of another control referenced by master_spec or expr_spec.

See “Primary-Dependent Commands” for a complete description of using the ‑master option.
