Runset Options for Template Parameters

Template parameters allow you to define certain Calibre Interactive GUI settings based on the layout or schematic database open in your design tool. These template parameters are saved in the runset.

The runset options that define template parameters are listed in Table 1. The default template, if there is one, is given in the first column. See the section “Templates for File Naming” for an explanation of file naming templates.

Template parameters that begin with the prefix “cmn” apply to all Calibre Interactive applications unless otherwise noted; templates parameters that apply to a specific application are prefixed accordingly.
