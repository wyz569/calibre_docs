Specifying Net Annotation Options When Reading Layout Databases

Calibre Interactive enables you to choose whether to read net names when loading LEF/DEF or OpenAccess layout databases. By default, net names are not annotated when reading these databases.

Procedure

  1. Click the app Options button on the left panel, where app is one of DRC, LVS, DFM, PERC, or PEX. If the button is not present, choose Setup > app Options from the menu to display it.
  2. Click the Database tab, and then click the OpenAccess/LEFDEF tab.
  3. Choose the Read Options category.
  4. In the area for “Read Net Names As”, select the method for adding annotations to the shapes that belong to a net:

    • Text — add annotations as text objects.

      All Cells — read net names from all cells.

      Top Cell — read net names from the top cell only.

    • Property — add annotations as properties using the specified property name.
