Setup Menu in Calibre Interactive

The contents of the Setup menu in Calibre Interactive depend on the application you are running.

The following table describes the contents of the Setup menu in Calibre Interactive.

Table 1. Setup Menu Contents in Calibre Interactive

Menu Selection

(application)

Description

Options

(all applications)

Displays the Options pane for the application and adds the Options button to the left panel of Calibre Interactive.

Set Environment

(all applications)

Opens the Setup Environment Variables dialog box, which allows you to view, edit, and define the environment variables for the run. See “Viewing and Editing Environment Variables”.

Verilog Translator

(nmLVS, PERC, PEX)

Opens the Setup Verilog Translator dialog box. See “Setting Up the Verilog Translator (v2lvs)”.

Delay Calculation

(PEX)

Opens the Setup Delay Calculation dialog box. See “Setting Delay Calculation” for more information.

Select Checks

(3DSTACK only)

Opens the Select Checks dialog box to select the checks that are executed. See “Selecting Specific Checks” in the Calibre 3DSTACK User’s Manual.

Layout Viewer

(all applications)

Opens the Setup Layout Viewer dialog box.

Communication with a layout viewer is automatically set up when you launch Calibre Interactive from the layout viewer and no additional setup is necessary in most cases. See “Communication with the Design Tool” and “Setting the Calibre Interactive Socket Port for a Layout Viewer”.

Schematic Viewer

(nmLVS, PERC, PEX)

Opens the Setup Schematic Viewer dialog box.

Communication with a schematic viewer is automatically set up when you launch Calibre Interactive from the schematic viewer and no additional setup is necessary in most cases. See “Communication with the Design Tool” and “Setting the Calibre Interactive Socket Port for a Schematic Viewer”.

Preferences

(all applications)

Opens the Setup Preferences dialog box. See “Setup Preferences”.

Show Tooltips

(all applications)

When enabled, causes Calibre Interactive to display pop-up tooltips when you hover the mouse over a button or field in the GUI.
