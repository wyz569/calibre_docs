Search Toolbar in Calibre Interactive xACT

To access: Enter a search term in the entry field in top right corner of the GUI. (shortcut: Ctrl-f)

The Search toolbar finds matches to a search string. The behavior depends on the GUI page that is active.



Description

Active Page

Behavior

Page other than Files or Transcript

Finds matches to a search string and places the GUI control in the Search page. Matches are also made to text in tooltips, values, and keywords.

You can make changes to the settings, if desired. Settings made in the Search page are reflected when you view the control in its original page.

If a match is found in a sub-option, the top-level option and all its sub-options are displayed.

The search examines all pages except Transcript and Files. Only visible pages and options are searched. For example, if the Inductance page is not visible, matches to “inductance” on the Inductance page are not found.

Transcript or Files

Searches in only the active page and highlights matches.
