Methods to Specify a Customization File

The customization file can be specified using the Calibre Interactive GUI, with an environment variable, or on the command line.

Requirements

  • A customization file. See “Customization Language” and “Customization Commands”.

Methods

Choose one of the following methods to specify a customization file:

  • Environment variable

  • In the Calibre Interactive GUI

  • Command line (interactive mode)

  • Command line (Calibre Interactive batch mode)

Environment variable

Set the MGC_CALIBRE_CUSTOMIZATION_FILE environment variable to the filename of the customization file.

In the Calibre Interactive GUI

  1. Choose Setup > Preferences to open the Setup Preferences dialog box.

  2. Click the Startup tab.

  3. Enable the “Customization File” checkbox and specify the filename of the customization file.

  4. Choose File > Save Runset to save the settings to the runset.

    If this runset is used to run Calibre Interactive in batch mode, the initial settings in the customization file are applied to the Calibre run; the Customization GUI does not open. If the runset contains customization override settings, those take precedence over the settings in the customization file.

Command line (interactive mode)

Specify a customization file from the command line with the -custom filename option. This option only works when you invoke a particular component of Calibre Interactive (for example, LVS), as shown in the following example:

calibre -gui -lvs -custom my_custom (correct)

	XX calibre -gui -custom my_custom (incorrect)

Command line (Calibre Interactive batch mode)

Specify a customization file in Calibre Interactive batch mode with either the ‑customgui or ‑custom command line options, depending on whether you want the customization dialog box to open or not. See “Customization GUI Operation in Batch Mode” for details.

Note:

The customization file specified on the command line with ‑custom or ‑customgui overrides a customization file set with the environment variable MGC_CALIBRE_CUSTOMIZATION_FILE.

Tip

You can also use the Load button in an open Customization Settings dialog box to specify a new customization file. See “Specify and load a different customization file from the Customization GUI”.

Results

When you invoke Calibre Interactive with a customization file, a Customization button appears in the left panel of Calibre Interactive. When you click the Customization button, the Customizations Settings dialog box is displayed with the custom GUI controls for that tool.

Tip

Choose Setup > Preferences, click the Startup tab, then enable “Show customization dialog after opening runset” to have the customization dialog box open automatically after loading a runset.

To disable changes to the customization settings, choose Setup > Preferences, the Disable tab, and enable “Disable customization changes.”
