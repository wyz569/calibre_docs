Running Only Checks Selected in the Rule File with a Custom Check Selection Recipe

You can subtract the check expression “Checks not selected in the rules file” from a custom check recipe in order to run only checks that are selected in the rule file. This step is necessary because most check expressions do not obey DRC [Un]Select Check and DFM [Un]Select Check statements in the rule file.

Prerequisites

  • Calibre Interactive nmDRC or Calibre Interactive DFM is running.

  • A rule file is loaded on the Rules pane.

Procedure

  1. Do one of the following to open a recipe for editing:

    • Create a custom check recipe. See “Creating a Custom Check Selection Recipe”.

    • Open an existing check recipe for editing:

      1. On the Rules pane, choose the recipe to edit in the “Check Selection Recipe” dropdown list.

      2. Click Edit.

    Leave the recipe open in the Check Selection Recipe editor dialog box.

  2. Select “Checks not selected in the rules file” in the Expression area.
  3. Enable “Subtraction.”
  4. Click the Apply expression button.
  5. View the “Recipe definition” area and confirm that “- Checks not selected in the rules file” is the last expression in the recipe definition.
