Reading Source Netlist Data from a Schematic Viewer in Calibre Interactive PERC

You can read source netlist data from a connected schematic viewer in Calibre Interactive PERC.

Corresponding rule file statements: Source Path, Source Primary, Source System

Procedure

  1. Click Inputs to display the Inputs pane.


  2. Make sure that “Source Netlist” is selected in the “Analyze” dropdown button list.
  3. Click the Netlist tab.
  4. Enable “Export from schematic viewer” to specify that a connected schematic viewer supplies the schematic source netlist.

    Batch runs: In addition, you must also disable the setting “Ignore layout and source export in batch mode” in the Setup Preferences dialog box on the Misc tab. Also see “Specifying Layout and Schematic Export for a Calibre Interactive Batch Run” for details on the viewer connection for a batch run.

  5. Specify the format of the exported source netlist using the “Format” dropdown button list.

    • SPICE — The exported netlist is in SPICE format.

    • VERILOG — The exported netlist is in Verilog format. If you specify Verilog as the source format, Calibre Interactive calls a utility named v2lvs to translate the Verilog netlist to SPICE. Choose Setup > Verilog Translator to set up the translation options; see the section “Setting Up the Verilog Translator (v2lvs)”. The translated SPICE netlist is then used for LVS comparison; the SPICE netlist name is taken from the “Files” field.

    The “MIXED” option is not available when “Export from schematic viewer” is enabled.

  6. Specify the filename for the exported source netlist in the “Spice Files” or “Verilog Files” field; the exported netlist is written to this file. The filename is generated automatically based on the “Source Path” template setting on the Templates > Inputs tab of the Setup Preferences dialog box.
  7. Verify the top cell name in the “Top Cell” field. The top cell is determined from the cell currently open in the viewer when “Export from schematic viewer” is enabled.
  8. (Optional) Specify additional SPICE netlist files. This may be necessary to provide INCLUDE files for the source netlist.
    1. Click the PERC Options button or choose Setup > PERC Options.
    2. Click the Database tab and the Library sub-tab.
    3. Use the buttons next to “Additional SPICE Files” to specify the required SPICE files.
      Note:

      Calibre Interactive verifies that the source netlist is complete before starting the run. You can turn off this check by disabling “Check source netlist is complete before running PERC” on the Misc tab of the Setup Preferences dialog box. See “Miscellaneous Preferences” for information on the source netlist check.

Results

The appropriate statements are written to the Calibre Interactive control file at runtime.

If you specified multiple source netlist files, Calibre Interactive creates a file named _source.net_ with .INCLUDE statements for each netlist file; _source.net_ is used as the source netlist for the Source Path statement in the control file. Files entered in the Database > Library tab on the PERC Options pane are listed before files in the Netlist tab on the Inputs pane.
