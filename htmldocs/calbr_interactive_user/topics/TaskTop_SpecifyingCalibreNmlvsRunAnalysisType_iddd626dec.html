Specifying the Calibre nmLVS Run and Analysis Type

You specify the type of Calibre nmLVS run on the Inputs pane.

Procedure

  1. Click the Inputs button on the left panel to display the Inputs pane.


  2. Choose the run type. This selects the type of run and the license to use.

    Run Type

    Description

    Hierarchical

    Run in hierarchical mode.

    Flat

    LVS comparison and circuit extraction are performed flat. The ‑flatten command line option is used.

    Recon

    Run Calibre nmLVS Reconnaissance Short Isolation. The -si command line option is used. Several options not used when running Calibre nmLVS Recon SI are hidden, and the “Step” option changes to “Mode”:

    • All Shorts — Perform both IO and Power Ground short isolation

    • Power Ground Shorts — Perform only Power Ground short isolation

    • IO Shorts — Perform only IO short isolation

    Calibre CB

    Use the Calibre CB (Cell/Block) flat license package. LVS comparison and circuit extraction are performed flat. The ‑cb and ‑flatten command line options are used.

    Note:

    See “Generating Waiver Shapes with Calibre Interactive” for instructions on using the “Autowaiver Creation” run type.

  3. Choose the analysis step from the “Step” dropdown list:.

    Analysis Step

    Description

    Layout vs Netlist

    Perform layout netlist extraction and LVS comparison in one step. You supply a layout database and the source netlist. Calibre extracts the layout netlist from the layout database. If the Run type is “Hierarchical”, the extracted layout netlist is saved.

    Netlist vs Netlist

    Perform LVS comparison in a separate step from layout netlist extraction. You supply a layout netlist file in SPICE format and a source netlist.

    Netlist Extraction

    Perform layout netlist extraction in a separate step from LVS comparison. You supply a layout database. The extracted layout netlist is saved.

  4. Supply other required data on the Inputs pane; see “Inputs for a Calibre Interactive nmLVS Run” for more information.
