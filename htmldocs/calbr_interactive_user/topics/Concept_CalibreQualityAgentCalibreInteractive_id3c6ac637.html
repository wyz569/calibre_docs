Calibre Quality Agent for Calibre Interactive

The Calibre Quality Agent dialog box is displayed when Calibre Interactive encounters an application error. You use the dialog box to send feedback about the error conditions to the Calibre team.

The Calibre Quality Agent dialog box is shown in the following figure.

Figure 1. Calibre Quality Agent Dialog Box for Calibre Interactive

Note:

No design data is sent with the Calibre Quality Agent emails.

The feedback from the Calibre Quality Agent email is very helpful to the Calibre team, especially when you include the steps leading up to the application error. The error and code traceback generated at the time of the error are included automatically, but are usually not enough information by themselves to correct an application error. A copy of the feedback email is automatically sent to the email address you provide.

Click the Details tab to view the automatically included error report and code traceback. Enable or disable “Include path to database” on the Details tab to choose whether the path to your runset (if any) is included.

The settings you choose for the Calibre Quality Agent dialog box remain in effect for the current Calibre Interactive session.
