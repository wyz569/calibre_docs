Getting Started with Customization Files

The customization file is specified in the Startup tab of the Setup Preferences dialog box.

Prerequisites

  • A customization file. See “Customization Language”, “Customization Commands”, and “Customization File Examples” for information.

Procedure

  1. In Calibre Interactive, choose Setup > Preferences to open the Setup Preferences dialog box.
    1. Click the Startup tab.
    2. Enable the “Customization File” checkbox and specify the filename of the customization file.
    3. (Optional) Enable “Show customization dialog after opening runset” to have the Customization GUI open automatically after a runset is loaded.
    4. Click OK to close the Setup Preferences dialog box.

    The Customization button is now displayed in the left panel of Calibre Interactive.

  2. Click the Customization button to open the Customization Settings dialog box.

    Make changes as desired in the Customization GUI.

  3. Click OK to apply the customization settings and close the Customization Settings dialog box.

    The customization settings are used in the next Calibre run unless overridden by later changes in the Calibre Interactive GUI.

  4. Choose File > Save Runset to save the runset.

    If you made changes to the Customization GUI, the changes to the Customization GUI are saved to the runset as customization overrides. The override settings in the runset are applied the next time you load the runset and open the Customization GUI.
