Inputs for a Calibre Interactive nmLVS Run

You specify input data and other input-related options in separate tabs on the Inputs pane.

The Inputs pane contains the following tabs:

Table 1. Inputs Pane in Calibre Interactive nmLVS

Tab

Description

Layout

Specify the layout input (geometric database or layout netlist) on the Layout tab; see “Layout Input in Calibre Interactive nmLVS”.

Netlist

Specify the source input (netlist) on the Netlist tab; see “Source Netlist Input in Calibre Interactive nmLVS”.

H-Cells

Specify hcell input and perform hcell analysis; see “Supplying Hcell Data” and “Performing Hcell Analysis in Calibre nmLVS”.

Signatures

Create device signatures; see “Device Signature Creation with Calibre Interactive nmLVS”.

Waivers

Specify waivers for ERC checks; see “Specifying Automatic Waivers for ERC”.
