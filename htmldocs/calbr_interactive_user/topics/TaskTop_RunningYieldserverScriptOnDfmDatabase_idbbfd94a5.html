Running a YieldServer Script on the DFM Database

You can run a YieldServer script on DFM database after it is created.

Prerequisites

  • The output results format is DFM Database on the Common tab of the Outputs pane.

Procedure

  1. Choose Setup > DFM Options to open the DFM Options pane.
  2. Click the DFMDB tab, as shown in Figure 1.
  3. Enable “Run YS Script.”
  4. Select one of the following:

    • Extract from Rule File — Run the YieldServer script in autostart mode. See DFM YS Autostart.

    • File Path — Provide the path to a YieldServer script.

      Enter the arguments for the YieldServer command in the “Arguments” field. Internal trigger parameters, such as %r (rule file) and %l (layout path), may be used for the arguments. The trigger parameters can be viewed by choosing Setup > Preferences, then the Triggers tab.
