Inputs for a PEX Run

The Inputs pane includes the tabs for each type of input. Use these tabs to specify the inputs needed for your PEX run.
Table 1. Inputs Pane for PEX

Tab

Purpose

Required

Layout

Specify the layout design database and format. See “Specifying the Layout Input in Calibre Interactive PEX” and “Specifying Multiple Layouts for a PEX Run”.

Yes

Netlist

Specify the schematic netlist. This is used when you want the extracted netlist to use source schematic names, or you want to control cell extraction for an ADMS extraction run. See “Specifying Schematic Netlist Input”.

No

H-Cells

Specify cell lists for performing non-flat extraction. See “Defining Hcell and Xcell Input (Gate-Level, Hierarchical Extraction and ADMS Only)”.

No

Blocks

Specify blocks for ADMS extraction. See “Manipulating Cell Extraction Options Using the Blocks Tab” and the Calibre xRC User’s Manual.

No

Probes

Specify probe points. See “Specifying Probe Points in Calibre Interactive PEX” and PEX Probe File in the SVRF Manual.

No
