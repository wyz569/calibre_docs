Split Gate and Parallel MOS Reduction Options in PEX

You can control how split gates are reduced using Calibre Interactive PEX.

Choose Setup > PEX Options, click the LVS Options tab, then select the options in the “Split Gate Reduction” area. These controls are the same as for Calibre Interactive nmLVS except for the tab label.
