Specifying a Check Selection Recipe

The Check Selection Recipe specifies the checks that are executed in a Calibre Interactive nmDRC or DFM run. You can use a built-in check recipe or define your check recipe.

Prerequisites

  • Calibre Interactive nmDRC or Calibre Interactive DFM is running.

  • “Disable check selection” is not enabled on the Disable tab of the Setup Preferences dialog box (Setup > Preferences).

Procedure

  1. Click the Rules button on the left panel to display the Rules pane.
  2. Specify a rule file.
  3. Choose a check recipe using the dropdown list for Check Selection Recipe, as shown in the following figure.
