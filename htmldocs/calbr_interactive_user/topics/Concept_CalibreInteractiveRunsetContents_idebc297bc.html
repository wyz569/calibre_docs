Calibre Interactive Runset Contents

A Calibre Interactive runset is a text file that contains variable names and values corresponding to settings in the Calibre Interactive GUI. A runset is used to restore saved settings to the GUI and to control Calibre Interactive batch runs.

A runset includes the following settings:

  • GUI settings.

  • Settings in the Setup Preferences dialog box (Setup > Preferences), except those on the Startup tab in the “Startup Runset” and “Calibre Interactive Window Position” areas.

Here is a brief example:

*drcRulesFile: rules
*drcRunDir: .
*drcLayoutSystem: GDSII
*drcResultsFile: drc.results
*drcResultsFormat: ASCII

The basic form of each line is the following:

*<variable>: <value>

Each variable corresponds to an item you can enter or change in the GUI. The name of the variable indicates which GUI (DRC, LVS, PEX, DFM, PERC) is affected by the value, and which item in the GUI is being set to that value. Runset variables preceded by “cmn” are common to all applications.

If an option is not listed in the runset, the Calibre Interactive built-in default setting is used.
