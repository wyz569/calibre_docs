Controlling Inductance Extraction

You can control inductance extraction using Calibre Interactive.

Procedure

  1. Click the PEX Options button on the left panel, or choose Setup > PEX Options if the button is not present.
  2. Click the Inductance tab.
  3. Specify the setting for “Extract Mode,” corresponding to the PEX Inductance Mode statement.

    LOOP — Loop inductance extraction mode. This is the default.

    PEEC — Partial RLK extraction mode, which is based on the Partial Element Equivalent Circuit method. If selected, the GUI automatically sets “Extract parasitics for” to “Specified Nets” on the Nets tab of the Outputs pane, and enables “Include” on the Top-Level Nets sub-tab.

    Clear the checkbox for “Extract Mode” to use the rule file setting.

  4. Set the following options, depending on whether you want to use the broadband model:

    To do this …

    Do the following …

    Generate a broadband netlist

    1. Enable the “Broadband” checkbox.

    2. Select one of the following options:

      • Operating Frequency — Specify the maximum operating frequency. This option uses the PEX Inductance Frequency statement with the MAXIMUM keyword.

      • Switch Time — Specify the switch time. This option uses the PEX Inductance Switch Time statement.

    Do not use the broadband model in netlist generation

    1. Disable the “Broadband” checkbox.

    2. Specify the operating frequency using the “Operating Frequency” option. This option uses the PEX Inductance Frequency statement.

  5. Use the “Net Filter” dropdown list to specify a value for the PEX Inductance Filter statement.

    If “File” is chosen from this list, two additional fields are enabled:

    Driver/Receiver File — Enter or browse for the file used by the PEX Inductance Driver File statement.

    Driver/Receiver Tags — Enter one of more optional tags; these are appended to the PEX Inductance Filter statement.

  6. If desired, enable the “Use Victim Path File” option and enter the path to the victim file. See the PEX Inductance Victim SVRF statement for information about the victim file format.
  7. Select the “Inductance Range” option to set the maximum distance used to determine inductance. “Cell Extent” is the default and results in the entire cell being used. You can also specify a maximum distance in microns. (SVRF statement: PEX Inductance Range)
  8. Use the dropdown list next to the “Nets to include in return path” checkbox to specify return path nets. Select either Ground, Power, Ground and Power, Average or None. Selecting Average uses the average results of ground only and power only. Selecting None writes the PEX Inductance Returnpath NONE statement to the control file. Ground is the default.

    Clear the “Nets to include in return path” checkbox to omit the PEX Inductance Returnpath statement from the control file.

    This option is not available if “Extract Mode PEEC” is selected.

  9. If desired, enable “Same net mutual inductance” to enable the extraction of mutual inductance between wire segments within the same net. The default is off. (SVRF statement: PEX Inductance Same Net Mutual)
  10. If needed, specify ground and power nets on the Misc tab of the PEX Options pane.
    1. Click the Misc tab on the PEX Options pane.
    2. In the fields for “Power nets” and “Ground nets,” select LAYOUT or SOURCE as the source for the net names, then enter one or more net names. Separate multiple net names with a space.
