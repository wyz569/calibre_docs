Specifying Parameters for Resistance Calculations

You can specify layer-specific parameters for resistance calculations in Calibre Interactive PEX.

This corresponds to the SVRF statement PEX Resistance Parameters.

Procedure

  1. (Optional) On the Rules pane, click Load, and the parameters from PEX Resistance Parameters statements in the rule file are loaded into the dropdown list of resistance parameters.
  2. Click the PEX Options button on the left panel, or choose Setup > PEX Options if the button is not present.
  3. Click the Misc tab.
  4. Enable the selection “Resistance parameters.”
  5. Enter the desired parameters for the PEX Resistance Parameters command in the text entry field. For example, to specify the TC1 and TC2 coefficients for the layer metal1, enter:
    metal1 TC1 0.003 TC2 0.012
  6. Click Add. This populates the dropdown list with the parameters you entered.
  7. (Optional) Add additional commands by repeating Steps 5 and 6.

    The text entry field lists the first parameter set from the dropdown list — simply overwrite the old text to enter a new parameter set. New parameter sets are added to the dropdown list if they are not duplicates of an existing entry, and previous entries are preserved.

  8. (Optional) To delete a command, select the command from the dropdown list and click Delete.

Results

If “Resistance parameters” is enabled, then the specified commands are used and PEX Resistance Parameters commands in the rule file are ignored.
