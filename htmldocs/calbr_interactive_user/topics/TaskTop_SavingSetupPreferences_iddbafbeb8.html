Saving Setup Preferences

Most Setup Preferences are saved to the Calibre Interactive runset, while Startup Preferences are automatically saved to special configuration files.

Procedure

  1. Choose Setup > Preferences and make sure all the preference settings are correct.
  2. Click OK to close the Setup Preferences dialog box.
  3. Choose File > Save Runset or File > Save Runset As to save a runset.
  4. Exit Calibre Interactive.

Results

All preferences on the Templates, Triggers, Disable, Misc, and Info tabs, plus the customization file settings on the Startup tab, are saved to the Calibre Interactive runset. See “Creating and Saving a Calibre Interactive Runset” for complete information.

Preferences on the Startup tab, except the customization file settings, are automatically saved to special setup files when you exit Calibre Interactive, as described in “Configuration Files for Startup Preferences”.
