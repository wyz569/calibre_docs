GUI Signals in Calibre Interactive xACT

You can connect functions to signals generated by the Calibre Interactive xACT GUI. Signals are generated when a property value changes and when a user-defined button is clicked.
Table 1. Signals from the Calibre Interactive GUI

GUI Signal

To connect a function …

Change in property value for a standard GUI control (those not added by the configuration file)

tool.option.propertyChanged.connect(cmd)

See connect.

Change in property value from an object added by the configuration file (for example, addDefine or addIntegerVariable)

ctrl_var.propertyChanged.connect(cmd)

See connect.

Button click from user-defined button

button_var.activated.connect(cmd)

See addAction.

New choice from a button with a dropdown list

button_var.valueChanged.connect(cmd)

See addChoicesAction.

 

Tip

You can block a run from executing by setting the property tool.blockRun to false. See “Tool Properties”.

You can attach a callback function to the Run button with the tool property tool.runCallback. See “Example: Attach Callback Function to Run Button With tool.runCallback”.
