CUSTOM::setOKCallbackError

Command type: Customization command for Calibre Interactive.

Specifies an error message that is displayed if the function specified by CUSTOM::setOKCallback returns -1.

Usage

CUSTOM::setOKCallbackError “msg_text”

Arguments

“msg_text”

The text of the error message, enclosed in quotes.

Description

If the function specified by CUSTOM::setOKCallback returns -1, a warning dialog box is displayed that includes the text of msg_text.

Examples

The following example shows a callback procedure that checks the Calibre version. If the version is too early, the callback procedure sets the error message with CUSTOM::setOKCallbackError and returns -1. 

# custom GUI controls ...

proc myOKCallback { } {

# check version and exit if not late enough
	set cal_version [CUSTOM::getCalibreVersionString]
	set version_year [string range $cal_version 1 4]
	 
	if {$version_year < 2013} {
		CUSTOM::setOKCallbackError "Calibre $cal_version is too early"
		return -1
	}

# rest of callback procedure ...
# ...
}

CUSTOM::setOKCallback myOKCallback
