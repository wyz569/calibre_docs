Reading Layout Input from File in Calibre Interactive DFM

Calibre Interactive can read layout input from database files. Uncheck “Export from layout viewer” on the Inputs pane to use this mode.

Procedure

  1. Click Inputs on the left panel of Calibre Interactive.
    Tip

    If you want to read information for the Inputs and Outputs panes from the rule file, choose Setup > Preferences, click the Misc tab, and enable the “Read Inputs/Outputs fields when loading rules file” option. (This option is enabled by default.) and rule file options that relate to the Inputs and Outputs panes are automatically applied when you click Load on the Rules pane.

  2. Click the Layout tab.


  3. Select a layout format using the Format dropdown list. For formats other than GDSII or OASIS, see the following topics for more information:

    • LEFDEF — see “Read LEF/DEF Input from Design Files”

    • OPENACCESS — see “Providing OpenAccess Input”

    For a full list of supported layout databases, refer to the “Layout Databases” section of the Calibre Verification User’s Manual. Also see “Third Party Database Options”.

  4. Uncheck “Export from layout viewer” to specify layout database files as the source of the layout. The checkbox “Export from layout viewer” is not displayed if the layout format does not allow layout export.
  5. Specify the filename for the layout database in the “File” field.
  6. (Optional) If you have multiple layout inputs for your DFM run, choose Setup > DFM Options then the Database > Library tab to specify them. See “Specifying Multiple Layouts for a DFM Run”. The layout format must be GDSII or OASIS.
  7. Specify the top cell name.
  8. (Optional) Select the “Area” option to perform DFM checking on only an area of the layout. See “Performing Area DFM” for complete information.
  9. (Optional) If you are running Critical Feature Analysis (CFA) you can apply waivers during the run. Calibre Auto-Waivers controls are on the Inputs pane on the Waiver tab; see “Calibre Auto-Waivers in Calibre Interactive”.
