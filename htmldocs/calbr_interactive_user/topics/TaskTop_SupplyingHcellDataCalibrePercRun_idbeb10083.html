Supplying Hcell Data for a Calibre PERC Run

Hcells are used in a Calibre PERC run to specify subcircuits in the input that are preserved.

See the description for the ‑automatch and -hcell command line options in the “calibre -perc” section of the Calibre PERC User’s Manual for more information. Unlike a Calibre nmLVS run, Calibre PERC does not compare layout and source cells, so the hcell data does not specify correspondence between layout and source cells.

Related rule file statements: Hcell, LVS Exclude Hcell.

Procedure

  1. Click Inputs on the left panel.
  2. Click the H-Cells tab.
  3. Specify the method for hcell processing:

    • Automatch — All subcircuits in the input netlist are preserved during the Calibre PERC analysis, except for subcircuits listed in an LVS Exclude Hcell statement.

      Automatch is recommended when analyzing a netlist. The Automatch option is not used by the circuit extraction phase.

      Automatch is only available for hierarchical runs.

    • Use H-Cells file — Specify the hcell filename. Subcircuits in the hcell list are preserved and written to the extracted netlist, and preserved during Calibre PERC analysis. Subcircuits listed in an LVS Exclude Hcell statement are excluded from the hcell list. Calibre PERC only reads the layout or source column of the hcell list, depending upon which type of analysis you specified.

      This option is recommended when analyzing a geometric layout and performing netlist extraction. This option is only available for hierarchical runs and flat geometric layout analysis.
