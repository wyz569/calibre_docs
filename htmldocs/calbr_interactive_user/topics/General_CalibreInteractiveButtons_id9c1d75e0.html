Calibre Interactive Buttons

The left side of the Calibre Interactive window contains buttons that either select the GUI pane that is displayed or initiate an action.

Table 1 describes the buttons on the left panel of Calibre Interactive. Some buttons are optional and must be enabled as described in the table.

Table 1. Buttons in Calibre Interactive

Button

Description

Rules

Displays the Rules pane for entering the rule file and run directory. You can also view the Layer Derivations Tree.

Inputs

Displays the Inputs pane for specifying input files and related options.

Outputs

Displays the Outputs pane for specifying the output database and related options.

Options

(Optional) Displays the Options pane for the application. Enable the Options button by choosing Setup > app Options, where app is one of DRC, LVS, DFM, PEX, or PERC.

Run Control

Displays the Run Control pane for specifying the host computer and other job control options. See “Run Control Setup and Licensing”.

Transcript

Displays the Transcript Pane in Calibre Interactive.

Customization

(Optional) Opens the Customization Settings Dialog Box, which is a user-coded GUI for specifying certain GUI controls and run parameters. Enable the Customization button by choosing Setup > Preferences and specifying a customization file on the Startup tab.

PEX steps

(Optional) These buttons allow you to individually select the Create PHDB, Create PDB, and Create Netlist stages of a PEX run. Enable the PEX steps buttons with “Show PEX steps in task button bar” on the Advanced tab of the Run Control pane.

Run

Starts the run.

Start RVE

Starts Calibre RVE and opens the results database specified on the Outputs pane.

Buttons for setup panes are green if all information is available and red if information is missing. Action buttons, such as Start RVE, are always black.
