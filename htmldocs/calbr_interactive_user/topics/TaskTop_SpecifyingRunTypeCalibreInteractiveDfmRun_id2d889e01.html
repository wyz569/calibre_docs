Specifying the Run Type for a Calibre Interactive DFM Run

The run mode in Calibre Interactive DFM determines whether you run Critical Area Analysis (CAA), Critical Feature Analysis (CFA), the CMP Analyzer, or Litho Friendly Design (LFD).

Procedure

  1. Click Inputs on the left panel of Calibre Interactive.
  2. Select a run mode using the Run dropdown list. The following table explains how Calibre is invoked for each combination of Run Mode and Results Format:
    Table 1. DFM Run Modes and Result Formats

     

    Run mode

    Command-line argument

    ASCII, GDSII, or OASIS Results Format

    DFM Database Results Format

    Critical Area Analysis (CAA)

    N/A

    -dfm

    Critical Feature Analysis (CFA)

    N/A

    -dfm

    CMP Analyzer

    N/A

    -dfm -cmp

    Litho Friendly Design (LFD)

    -lfd

    -dfm -lfd

Results

The following additional tabs appear, depending on the run mode:

  • CMP Tab — When you select the CMP Analyzer run mode, a CMP tab appears in the Inputs pane.

  • CAA Tab — When you select the Critical Area Analysis (CAA) run mode, a CAA tab appears in the Inputs pane.
