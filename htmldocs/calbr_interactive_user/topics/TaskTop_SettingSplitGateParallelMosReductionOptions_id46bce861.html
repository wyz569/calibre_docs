Setting Split Gate and Parallel MOS Reduction Options

These options control how split gates are reduced.

Procedure

  1. Click LVS Options on the left panel to display the LVS Options pane. If the LVS Options button is not available, choose Setup > LVS Options.
  2. Click the Gates tab on the LVS Options pane.
  3. Select GUI or rule file control of split gate reduction as follows:

    • “Use settings from rules” enabled — Use rule file setting and disable GUI control. No further GUI settings are needed.

    • “Use settings from rules” disabled — Use GUI settings, as described in the following steps.

  4. Enable or disable split gate reduction. Some options depend on the setting for “Reduce split gates.”

    • Enable “Reduce split gates” — This specifies “LVS Reduce Split Gates YES” to the Calibre Interactive control file. The following options are available:

      • Semi-split gates — Adds the SEMI ALSO keyword.

      • Only when within tolerance — Adds the WITHIN TOLERANCE keyword.

      • Mix types during reduction — Adds the MIX TYPES keyword.

      • Series-parallel split gate — Adds the SP ALSO keyword.

      • Only when the input order is same — Adds the SAME ORDER keyword.

      The controls for “Reduce parallel MOS transistors” and “Short equipotential nodes together” are not available when “Reduce split gates is enabled.”

    • Disabled “Reduce split gates” — This specifies “LVS Reduce Split Gates NO” to the Calibre Interactive control file. The following options are available:

      • Reduce parallel MOS transistors — As described; see LVS Reduce Parallel MOS.

        Note: This setting is on by default when “Reduce split gates” is enabled, since split gate reduction implies parallel MOS reduction.

      • Short equipotential nodes together — Short together the internal nets of basic split gate structures if the nodes are equipotential; see LVS Short Equivalent Nodes.
