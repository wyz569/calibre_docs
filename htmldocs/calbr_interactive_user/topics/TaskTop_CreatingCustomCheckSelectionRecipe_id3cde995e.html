Creating a Custom Check Selection Recipe

You can create a new custom check recipe or copy an existing recipe from the Check Selection Recipe Editor dialog box. Custom check recipes (also called user recipes) allow you to control which Calibre rule checks are executed during a Calibre nmDRC or Calibre DFM run. Check recipe definitions are saved in the Calibre Interactive runset, along with the currently active check recipe.

Prerequisites

  • Calibre Interactive nmDRC or Calibre Interactive DFM is running.

  • A rule file is available.

  • “Disable check selection” is not enabled on the Disable tab of the Setup Preferences dialog box (Setup > Preferences).

Procedure

  1. Click the Rules button on the left panel of Calibre Interactive to view the Rules pane.
  2. Specify a rule file. You cannot open the check recipe editor without specifying a rule file.
  3. Click the Edit button to the right of the Check Selection Recipe entry.

    A view of the recipe editor is shown in “Check Selection Recipe Editor Dialog Box”.

  4. Do one of the following:

    To do this …

    Take these steps …

    Create a new recipe

    1. Click New in the top button bar.

      A new recipe is initialized with the checks selected in the rule file.

    Copy an existing recipe

    1. Choose the recipe to copy in the Recipe dropdown list.

    2. Click Copy in the top button bar.
  5. Proceed to “Basic Editing of Custom Check Selection Recipes” or “Advanced Editing of Check Selection Recipes in Calibre Interactive”.
