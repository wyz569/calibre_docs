Saving Customization Settings

Settings in the Customization GUI are kept internally during the current Calibre Interactive session when you click OK in the Customization Settings dialog box. Customization settings are saved to the runset as customization overrides when you save a runset.

Prerequisites

  • A customization file is specified; see “Methods to Specify a Customization File”.

Procedure

  1. Click the Customization button on the left panel of Calibre Interactive to open the Customization Settings dialog box.
  2. Adjust settings as desired.
  3. Click OK to apply the settings and exit the dialog box.

    The new settings are used in the next Calibre run unless overridden by later GUI actions. The new settings are retained within the current Calibre Interactive session.

  4. Choose File > Save Runset to save a runset with the new customization settings.

    The new customization settings are saved as customization overrides to the runset option cmnCustomFileOverrideValues. The next time you load the runset, the customization override settings are used for the initial settings of the customization controls.
