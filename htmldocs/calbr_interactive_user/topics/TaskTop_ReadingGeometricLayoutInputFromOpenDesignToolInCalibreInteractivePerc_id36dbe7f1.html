Reading Geometric Layout Input from an Open Design Tool in Calibre Interactive PERC

Layout data for a Calibre Interactive PERC run is specified on the Inputs pane on the Layout tab. You can export the layout from an open design tool. The layout netlist is extracted from the geometric layout.

Procedure

  1. Click Inputs to display the Inputs pane.


  2. Make sure that “Layout” is selected at the top of the Inputs pane in the Analyze section.
  3. Click the Layout tab.
  4. Enable “Export from layout viewer” to specify that the layout is exported from a connected design tool; see “Communication with the Design Tool” for more information.
  5. Select the exported layout format using the “Format” dropdown button list. OASIS, GDSII, and LEFDEF formats are available if “Export from layout viewer” is selected.
  6. Specify the filename for the exported layout database in the “Layout File” or “DEF File” field. For LEFDEF format, also provide LEF input as described in “Providing LEF/DEF Input”.

    If you are performing a Calibre Interactive batch run, see “Specifying Layout and Schematic Export for a Calibre Interactive Batch Run”.

    CAUTION:

    When “Export from layout viewer” is enabled, the file specified in the “File” field is overwritten if it exists already. Do not enter a tapeout database in the “File” field, or use the extracted layout named in the “File” field for tapeout or any other use.

    When exporting a layout database from a connected viewer, an intermediate file is extracted and saved to the file named in the “Layout File” field. This filename is generated automatically based on the setting on the Templates > Inputs tab of Setup Preferences dialog box (Setup > Preferences).

    If the filename is red, the file does not exist and Calibre Interactive creates the file. If the filename is green, the file exists and will be overwritten. Enter a different filename if you want to keep the existing file.

  7. (Optional) If you have multiple layout inputs for your run, you can specify them on the Database > Library tab on the PERC Options pane. This is valid both for layout data from a file and exported from a layout viewer. See “Specifying Multiple Layouts for a Calibre PERC Run”.
  8. Verify the top cell name in the “Top Cell” field. The top cell is determined from the current cell in the viewer.
  9. Enter the name of the extracted layout netlist in the “Layout Netlist” field. This option is for hierarchical runs only; in addition, it is not present for Calibre PERC-LDL runs.

    When connected to a design tool, a system-generated SPICE netlist name appears in the “Layout Netlist” field by default. This name is taken from the “Extract to File” setting in the Setup Preferences dialog box on the Templates > Outputs tab. However, you can choose a different filename.
