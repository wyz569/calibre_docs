Specifying a Calibre PERC LDL Probe Path

You can specify the probe mapping file for a Calibre PERC LDL run using Calibre Interactive.

Related rule file statement: PERC LDL Probe Path

Prerequisites

  • The run type is “PERC-LDL” and the Analyze setting is “Layout” on the Inputs pane.

    The Probes tab on the Inputs pane is only visible when these settings are made.

Procedure

  1. Click Inputs on the left panel.
  2. Click the Probes tab.
  3. Check the “Use PERC LDL Probe File” checkbox and enter the path to the probe file.
