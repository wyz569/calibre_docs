CUSTOM::runsetVarExists

Command type: Customization command for Calibre Interactive.

Determines if the input parameter is a valid GUI control name.

Usage

CUSTOM::runsetVarExists name

Arguments

name

The name of the GUI control. The name is given by the corresponding runset option variable name. See “Option Descriptions” for a list of runset options.

Return Values

Returns 1 if name is a valid GUI control name, 0 otherwise.

Examples

This example determines if drcRulesFile is a valid GUI control name and prints a message based on the return value of CUSTOM::runsetVarExists. 

if {[CUSTOM::runsetVarExists drcRulesFile]} {
	puts "found runsetVar drcRulesFile"
} else {
	puts "no such runsetVar drcRulesFile"
}
