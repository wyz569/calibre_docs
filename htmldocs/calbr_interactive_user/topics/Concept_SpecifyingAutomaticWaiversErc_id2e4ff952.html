Specifying Automatic Waivers for ERC

You can specify the ERC rule checks to waive and the cells in which the waivers apply using the Calibre Auto-Waivers flow.

Calibre Auto-Waivers allows various classifications for the waivers that can be applied, such as how to handle multiple waivers that may intersect a result region and hierarchical interactions. See “Calibre Auto-Waivers” and “ERC Automatic Waivers” in the Calibre Auto-Waivers User’s and Reference Manual for complete information.

Calibre Auto-Waivers controls are on the Inputs pane in the Waiver tab; see “Calibre Auto-Waivers in Calibre Interactive”.

See “ERC Results Tab” in the Calibre RVE User’s Manual for information on waiving ERC results using Calibre RVE.
