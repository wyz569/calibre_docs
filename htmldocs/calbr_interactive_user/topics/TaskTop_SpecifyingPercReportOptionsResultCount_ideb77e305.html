Specifying PERC Report Options and Result Count

These settings control the verbosity of the Calibre PERC report and the maximum number of results reported.

Corresponding rule file statements: PERC Report Option and PERC Report Maximum.

Procedure

  1. Click PERC Options to display the Options pane. If the PERC Options button is not available, choose Setup > PERC Options.
  2. Click the Report tab.
  3. Enable the checkbox “PERC Report Options” to enable GUI control of the PERC Report Option settings. If this checkbox is not enabled, rule file settings are used and GUI settings are ignored.
  4. Select the desired settings for PERC Report Options:

    • Report:

      • All Net Types — Report all net types and type sets carried by a net. The default is to report only the types specified in a given rule check. (option ALL_NET_TYPE)

      • No Net Types — Do not report net types or type sets carried by nets. The default is to report net types and type sets. (option NO_NET_TYPE)

      Note: The GUI only allows one of “All Net Types” or “No Net Types” to be selected. If a rule file is loaded which specifies both options, this will be reflected in the GUI; “No Net Types” takes precedence.

    • Do not report passing cells — Report only failed cells. The default is to report both passed and failed cells. (option SKIP_PASSED_CELL)

    • Do not report waived results — As described. (option SKIP_WAIVED_RESULT)

    • Do not report pin information — As described. (option NO_DEVICE_PIN)

    • Do not report all path voltages — Only report the minimum and maximum path voltages on a net. Printing of symbolic voltages is not affected. (option MIN_MAX_VOLTAGE_ONLY)

    • Examine subtitles of all results — Report results according to the subtitle for the rulecheck. (option TRACK_RESULT_SUBTITLE)

  5. Select the maximum result count in “Max. discrepancies printed in report”:

    • All — Report all results.

    • Count — Enter the maximum allowed result count.
