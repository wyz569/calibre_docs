Setting Filter Unused Device Options

You can filter out specific unused devices in your layout and source using the Filter Unused Device Options table in the Gates tab of the LVS Options pane.

Procedure

  1. (Optional) To view the rule file settings for filter options, click the Rules button in the left panel, and click Load. This loads the rule file settings into the GUI.
  2. Click LVS Options on the left panel. If the LVS Options button is not available, choose Setup > LVS Options.
  3. Click the Gates tab on the LVS Options pane.
  4. Specify GUI or rule file control of the LVS Filter Unused Option [LAYOUT | SOURCE] statement as follows:

    • To use rule file settings — Disable the “Filter Unused Device Options” checkbox.

    • To enable GUI control — Enable the “Filter Unused Device Options” checkbox.

  5. If GUI control is enabled, do the following to specify options:
    1. Select the desired filter options, either individually or for multiple options as follows:

      • Individual selection — Specify device filtering using the checkboxes in the L (layout) or S (source) columns.

      • Multiple selections — Shift- and Ctrl-click in the filter description text to select multiple filter types, then right-click to display a filter options menu and make the desired selection.

        Tip

        Click to the right of the L or S in the column header to sort the options by selected state. This is useful to list all selected options at the top.

    2. (Optional) You can specify that an option only apply to specific device types and subtypes, rather than to all devices. Click in the Devices column and use the list box to select the desired device(subtype) combinations.

      Note: Some filter options are specific only to certain device types. In those cases, the device(subtype) specified in the Devices column must be consistent with the types implied by the LVS Filter Unused command option.

      Note:

      If the “Filter Unused Device Options” checkbox is enabled, but no options are selected in the table, then this unselects all LVS Filter Unused Option settings, including any that might be specified in the rule file.
