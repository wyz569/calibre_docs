Layout Input in Calibre Interactive nmLVS

Layout data for a Calibre Interactive nmLVS run is specified on the Inputs pane in the Layout tab. The input data and options you specify depend on the analysis type and whether you are reading layout data from file or exporting it from an open design tool. Depending on the format of the layout input, you can specify multiple layouts and other options.
