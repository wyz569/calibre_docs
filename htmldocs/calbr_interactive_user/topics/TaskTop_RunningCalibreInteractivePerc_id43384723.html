Running Calibre Interactive PERC

Use the following procedure to start a Calibre PERC run from Calibre Interactive PERC.

Procedure

  1. Enter all necessary information in the Rules, Inputs, and Outputs panes.
  2. Choose Setup > PERC Options and select each of the option tabs to set additional options. All left panel buttons will be green when the necessary information is entered.
  3. Specify options on the Run Control pane.

    In particular, you can specify “Number of hierarchical LVS licenses for PERC,” which uses Calibre nmLVS-H licenses for the circuit extraction portion of the run. This adds the ‑lvs_supplement command line option.

  4. If desired, choose Setup > Preferences, click the Triggers tab, and specify triggers. See “Trigger Functions in Calibre Interactive”.
  5. Click Run PERC on the left panel of Calibre Interactive.

    If the run uses a source netlist, Calibre Interactive first verifies that the source netlist is complete; the source netlist check is case insensitive. If an internal pre-execution trigger is defined, this is done after the trigger executes. You can turn off the source netlist check by choosing Setup > Preferences clicking the Misc tab, and disabling the setting “Check source netlist is complete before running PERC.” See “Miscellaneous Preferences” for information on the source netlist check.

    When the run starts, the view switches to the Transcript pane, and any run output, including errors and warnings, is displayed there.

  6. View the Transcript pane and examine the results of the run. See “Transcript Pane in Calibre Interactive”.
    Tip

    You can generate and view a control file without running Calibre by holding down the Ctrl key and clicking Run PERC. In addition, the calibre command line for the run is written to the Transcript pane.

Results

If an error occurs in Calibre Interactive before the Calibre run is started, the error is displayed in a pop-up window or in the Calibre Quality Agent for Calibre Interactive.

When the Calibre run starts, the view switches to the Transcript pane, and any run output, including errors and warnings, is displayed here. See “Transcript Pane in Calibre Interactive”.

If “Start RVE after PERC finishes” is checked, the results database is opened in Calibre RVE when the run is complete; see “Using Calibre RVE for PERC” in the Calibre RVE User’s Manual. In addition, the Calibre Interactive GUI may open a report file in a text viewer when the run completes; see “Viewing and Editing Text Files in Calibre Interactive”.
