Creating a Calibre Interactive Runset

You can create a Calibre runset file from the Calibre Interactive GUI.

Procedure

  1. (Optional) To save all runset options, define the environment variable MGC_CALIBRE_SAVE_ALL_RUNSET_VALUES in your shell environment.

    By default, only runset options that differ from the default setting are saved to the runset file.

  2. Start Calibre Interactive.
  3. Make settings for the run in the Calibre Interactive GUI.
  4. (Optional) Choose Setup > Preferences, then the Info tab.

    Provide the Runset Type and Runset Info information. If a runset has been tagged with this information, it is displayed in the Load Runset File dialog box.

  5. Choose File > Save Runset or File > Save Runset As to save the GUI settings to a runset.
