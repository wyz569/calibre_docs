Creating a CalibreView Setup File

The CalibreView setup file contains all the settings necessary to generate a CalibreView. You can use the file as input to the SKILL function mgc_rve_load_setup_file to create the CalibreView in batch mode, without opening the Calibre View Setup dialog box. The file can also be loaded into the Calibre View Setup dialog box in order to populate the dialog box with saved settings.

Note:

The file calibreview.setup is automatically created in the run directory after Calibre View generation. The file contains the Calibre View settings and is in the CalibreView Setup File Format. You can edit this file to create a setup file with different parameters.

You can also create a setup file using the Calibre View Setup dialog box.

Procedure

  1. Do one of the following, depending on whether you have generated a Calibre View:

    If …

    Do this …

    You have generated a Calibre View and want to use most of the same parameters

    Copy calibreview.setup to another filename, such as newcalibreview.setup.

    You have not generated a Calibre View and want a default setup file

    1. Execute the SKILL function:

      mgc_rve_default_setup_file

      This creates the file .calibreview_default_setup_file in the directory you started Cadence Virtuoso from.

    2. Copy .calibreview_default_setup_file to another filename, such as newcalibreview.setup.

    You want to enter parameters for the setup file from the Calibre View Setup dialog box

    1. From Cadence Virtuoso, select Calibre > Setup > Calibre View.

    2. Enter the filename for the setup file in the “CalibreView Setup File” field, or leave it blank to use the default filename of calibreview.setup.

    3. Enter parameters in the dialog box.

    4. Click the Save button at the bottom of the dialog box.

  2. Edit your CalibreView setup file (newcalibreview.setup for example) and provide required file names, library settings, and other parameters. See “CalibreView Setup File Format” for details.

    This step is not necessary if you created the setup file using the Calibre View Setup dialog box and entered all the desired parameters.

  3. Save and exit the file.
    Note:

    See “Creating a Calibre View in Batch Mode” for instructions on creating the CalibreView in batch mode using the CalibreView setup file.

    You can also load the file into the Calibre View Setup dialog box—browse to the file and click Load to populate the dialog box with the settings.
