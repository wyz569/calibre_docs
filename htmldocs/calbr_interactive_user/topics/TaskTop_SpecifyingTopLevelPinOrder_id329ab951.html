Specifying the Top Level Pin Order

You can specify the pin ordering for the netlist.

This functionality corresponds to the PEX Pin Order statement in the SVRF Manual.

Procedure

  1. Click the PEX Options button on the left panel, or choose Setup > PEX Options if the button is not present.
  2. Click the Misc tab.
  3. Enable “Create top level pin order.”
  4. Choose LAYOUT, SOURCE, or FILE from the dropdown list to specify the pin ordering. The default is LAYOUT.

    Enter the file name if you selected FILE.

  5. (Optional) If you specified SOURCE or FILE, you can enable “All Pins” to output all pins, including floating pins.
