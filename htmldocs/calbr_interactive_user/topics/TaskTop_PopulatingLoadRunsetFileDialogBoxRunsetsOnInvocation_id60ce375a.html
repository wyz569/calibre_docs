Populating the Load Runset File Dialog Box with Runsets on Invocation

Define the environment variable MGC_CALIBRE_DRC_RUNSET_LIST to point to a file with the list of runsets to display in the Load Runset File dialog box. Replace DRC in the environment variable name with LVS, PERC, PEX, or DFM for those applications. The dialog box is displayed with the specified list of runsets when you invoke Calibre Interactive. Multiple files may be specified as a space-separated list.

Note:

A runset specified on the command line takes precedence over a MGC_CALIBRE_***_RUNSET_LIST environment variable. See “Loading Runsets on Invocation” for the complete order of precedence for automatic runset loading.

Prerequisites

  • You have a set of Calibre Interactive runsets for your application.

Procedure

  1. Create a text file with the path to each runset on separate lines. You can use absolute or relative pathnames.
  2. Depending on your Calibre Interactive application, define one of the following environment variables with a text file containing the list of runsets:

    Application

    Environment Variable

    DRC

    MGC_CALIBRE_DRC_RUNSET_LIST

    LVS

    MGC_CALIBRE_LVS_RUNSET_LIST

    PERC

    MGC_CALIBRE_PERC_RUNSET_LIST

    PEX

    MGC_CALIBRE_PEX_RUNSET_LIST

    DFM

    MGC_CALIBRE_DFM_RUNSET_LIST

    If you have multiple files with runset lists, specify them as a space-separated list. The lists are combined when displayed in the Load Runset File dialog box.

  3. Start Calibre Interactive.

Results

Calibre Interactive displays the Load Runset File dialog box on invocation and displays the specified runsets in the “Recent Runsets” list.

Examples

Create the text file /user/calibre/my_runset_list with the following contents:

/user/runsets/my_runset1
/user/runsets/my_runset2
/user/runsets/my_runset3
/user/runsets/my_runset4

Define the environment variable as follows for Calibre Interactive nmDRC.

% setenv MGC_CALIBRE_DRC_RUNSET_LIST /user/calibre/my_runset_list

If you have two runset list files:

% setenv MGC_CALIBRE_DRC_RUNSET_LIST "runset_list1 runset_list2"
