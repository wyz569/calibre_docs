Specifying the Extraction Mode and Level for a Calibre Interactive PEX Run

The Outputs pane in Calibre Interactive PEX specifies the extraction mode and level.

Prerequisites

  • Calibre Interactive PEX is open; see “Invoking Calibre Interactive Tools”.

Procedure

  1. Click Outputs on the left panel.
  2. Select the Extraction Mode as follows:

    • xACT 3D — Use the xACT 3D fieldsolver. See the Calibre xACT User’s Manual.

    • xRC — Use model-based extraction. See the Calibre xRC User’s Manual.

    • xACT 3D/xRC Hybrid — Select “xACT 3D,” then enable the “Hybrid” checkbox. This runs hybrid xACT 3D/rule-based extraction (the ‑hybrid command line option). Choose Setup > PEX Options then the xACT 3D tab to select the nets for extraction.

    The following exceptions apply when “xACT 3D” or “xACT 3D/xRC Hybrid” is specified:

    • If “R” or “No R/C” is selected as the Extraction Type, the -3d command line option is not used and the “Accuracy” cannot be specified.

    • The Calibre-xRC CB license cannot be used; the -cb command line option is not used even if “Use Calibre-xRC CB license” is specified on the Run Control pane.

  3. If the extraction mode is “xACT 3D” or “xACT 3D/xRC Hybrid,” select the accuracy as follows:

    200 — Less accurate, with faster runtime. This is the default.

    600 — More accurate, with slower runtime.

    MEMS — Runs at the highest accuracy needed for MEMS designs.

  4. Choose the extraction level (Transistor Level, Gate Level, Hierarchical, In-Context or ADMS) using the first button in the Extraction Type section.

    See Run Control, Remote Execution and Licensing Options for instructions on specifying the Calibre-xrc CB and Calibre-CB licenses for your extraction run.

  5. Choose the type of parasitics to extract using the second button in the Extraction Type section. See Setting PEX Options for extracted netlist options.
  6. Choose the type of parasitic inductance to extract using the third button in the Extraction Type section. Inductance extraction is controlled using the Inductance page in the PEX Options pane. See Controlling Inductance Extraction for more information.

    To extract inductance, you must use Transistor Level or Gate Level as the extraction type. In addition, you must also extract resistance, or set No R/C and enable incremental extraction (“Generate PDB incrementally” on the Outputs > SVDB tab).

    Note:

    The second button in the Extraction Type section controls the information that is extracted into the parasitic database (PDB). To control the information from the PDB that is displayed in the netlist, choose Setup > PEX Options, then the Misc tab and select a setting for “Parasitics to output to RC netlist.” This setting allows you to set up your netlist to display a subset of what you have extracted to the PDB. For more information, refer to “Specifying Which Parasitics to Include in the Netlist”.
