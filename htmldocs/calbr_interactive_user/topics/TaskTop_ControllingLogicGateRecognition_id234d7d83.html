Controlling Logic Gate Recognition

These options control how Calibre performs logic gate recognition.

Procedure

  1. Click the LVS Options button on the left panel to display the Options pane. If the LVS Options button is not available, choose Setup > LVS Options.
  2. Click the Gates tab on the LVS Options pane.
  3. Choose the desired gate recognition option from the following selections:

    • Recognize all gates — Uses the ALL keyword in LVS Recognize Gates.

    • Recognize simple gates — Uses the SIMPLE keyword in LVS Recognize Gates.

    • Turn off — Uses the NONE keyword in LVS Recognize Gates.

  4. Decide whether to select the “Mix subtypes” option. If you do not select this option, transistors in series-parallel branches must all have the same subtype.
    Tip

    You must specify power and ground net names if you enable gate recognition. For more information on this topic, refer to “Setting Power Supply Options”.
