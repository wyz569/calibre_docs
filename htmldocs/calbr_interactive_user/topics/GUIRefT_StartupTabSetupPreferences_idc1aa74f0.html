Startup Tab for Setup Preferences

Access with: Setup > Preferences, Startup tab

The Startup tab in the Setup Preferences dialog box defines startup behavior for loading runsets, optional customization file settings, and the Calibre Interactive window position. For Calibre Interactive nmDRC and DFM, it also includes settings for imported check recipes files.

Description

The runset opening behavior specified on the Startup tab is overridden by command line invocation with a runset, or by setting either of the environment variables MGC_CALIBRE_***_RUNSET_LIST and MGC_CALIBRE_***_RUNSET_FILE. See the discussion with “Loading Runsets on Invocation”.

Except for the customization file and imported check recipe settings, the settings on the Startup tab are automatically saved to a setup file each time you exit Calibre Interactive; see Configuration Files for Startup Preferences for more information.

The customization file and imported check recipe settings are saved to the runset.

The customization file and check recipe settings are described in the following table; other settings are intuitive and not described.

Fields

Table 1. Customization and Check Recipe Settings in the Startup Preferences

Option

Description

Customization File

Enable the checkbox and specify a customization file to use the Customization GUI.

Show customization dialog after opening runset

 

(Default: true)

Automatically open the Customization Settings dialog box when a runset is loaded.

Warn if customization file has no DRC/DFM/LVS/PERC/PEX settings

 

(Default: false)

Display a warning message if the customization file does not have any settings that are valid for the current Calibre Interactive application.

Import Recipe Files

(DRC and DFM only)

Enable the checkbox and specify check recipe files to load on startup. You save a check recipe file with the Export button in the Check Selection Recipe editor dialog box. The listed check recipe files are imported at startup and listed in the USER RECIPES section of the check recipe list.
