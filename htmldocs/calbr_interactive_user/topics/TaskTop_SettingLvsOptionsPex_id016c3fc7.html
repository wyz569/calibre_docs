Setting LVS Options for PEX

You can specify LVS power and ground names and LVS report options on the LVS Options tab of the PEX Options pane.

Procedure

  1. Click the PEX Options button on the left panel, or choose Setup > PEX Options if the button is not present.
  2. Click the LVS Options tab.
  3. Enter the power and ground net names in the appropriate fields; use a space separated list to enter multiple nets.

    To enter power and ground net names using a file, see “Setting Power Supply Options” for the file format.

    Corresponding rule file statements: LVS Ground Name and LVS Power Name.

  4. Select the LVS Report Options using the dropdown list.

    For most runs, you should not specify any of these options, unless you actually need the information they provide. The default report options are usually all that is needed.

    1. Click the button to the right of “LVS Report Options”.
    2. Select the desired report options. The button label changes to show the selected options.

      You can also disable report options that are specified in the rule file.

    Corresponding rule file statement: LVS Report Option.

  5. Set other options as described in these sections:

    • “Logic Gate Recognition Options in PEX”

    • “Split Gate and Parallel MOS Reduction Options in PEX”

    • “Filter Unused Device Options in PEX”
