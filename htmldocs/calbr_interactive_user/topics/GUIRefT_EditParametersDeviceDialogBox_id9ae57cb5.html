Edit Parameters for Device Dialog Box

To access: Edit Device button on the Signatures tab on the Inputs pane of Calibre Interactive nmLVS,

Use the Edit Parameters dialog box to specify the layers, pin names, and options used for device signature creation in Calibre Interactive.

Description

You can specify and load a Layers File to pre-populate the dropdown lists and tables in the dialog box; see “Layers File for Device Signature Creation”. However, you can type in any entry field or table row to enter an arbitrary parameter, so loading a Layers File is not necessary.

Objects

Table 1. Device Parameter Fields for Create Device Signature

Device Parameter Field

Description

Device Name

The component name.

Seed Layer

The layer the device body is on; also known as the device layer.

Reference Layer

The reference layer defines the area around the seed shape used for deriving a signature.

Once a reference layer is used with a specific set of context layers, it may only be used with that set of context layers in the rule file.

Pin Name

Pin Layer

The pin name and layer pairs necessary for the Device statement.

Use the right-click menu to select or unselect all pins and to toggle the pin selection. You can also Shift-click in the header of the Select column to select/unselect all rows.

Swappable With

Enter the pin names that are interchangeable with this pin.

If this entry is left blank, the default pin swappability rule applies: pins on the same layer are interchangeable.

Context Layers

Shapes on the context layer that intersect the reference layer are considered in the signature.

You can specify multiple context layers. The order of the context layers matters, since the generated signature reflects the given order. Change the order of context layers by dragging the layer entry.

Use the right-click menu to select or unselect all layers and to toggle the layer selection. You can also Shift-click in the header of the Select column to select/unselect all rows.

You can use the layers file to map a list of context layers to a list name; see the *CONTEXT_LAYER entry in “Layers File for Device Signature Creation”.

Note: The seed layer and pin layers are not part of the signature calculation unless the layers are included in the context layer list.

Use only seed layers with attached text label

Text Label

Text Layer

When enabled, only device seed layers with the specified text label on the given text layer are used in the Device statement. The text label string is appended to the seed layer name and a new layer is created which is used in the Device statement. These statements are used:

<seed_layer>_<label> = <seed_layer> WITH TEXT
    "label" <text_layer>
DEV <device> <seed_layer>_<label> ...

Model Name

Use the specified model name (or component subtype).

Text Model Layer

Use the TEXT MODEL LAYER keyword and parameter,

Netlist Model

Use the NETLIST MODEL keyword and parameter

Netlist Element

Use the NETLIST ELEMENT keyword and parameter.
