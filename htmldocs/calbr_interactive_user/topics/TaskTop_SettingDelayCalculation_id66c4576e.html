Setting Delay Calculation

You can configure the delay calculation tool that runs on blocks marked for extraction.

Refer to Manipulating Cell Extraction Options Using the Blocks Tab for information on choosing blocks for extraction.

Procedure

  1. Click Setup > Delay Calculation.
  2. Choose one or more technology library files to be used by the delay calculator for obtaining timing information for primitive cells.
  3. Choose “Additional Verilog Files” in addition to those specified on the Inputs pane in the Netlist tab as necessary. These additional files are used to define the modules for delay calculation.
  4. Specify a file containing SDC (Synopsys Design Constraint) commands as necessary.
  5. Select either the TIME-IT or OTHER delay calculation tool. If OTHER is chosen, see Step 8.
  6. Choose either the SPEF or DSPF parasitic extraction file format. Before the delay calculator is run to create a Standard Delay Calculator file (SDF) for the module, the SDF extraction is first used by Calibre-xRC in the chosen format. SPEF is strongly suggested if Time-It is chosen as the delay calculation tool.
  7. Specify the MGC_HOME value for the Time-It delay calculation tool. This is the path for MGC_HOME or CALIBRE_HOME for your Calibre software tree.
  8. If OTHER is chosen as the delay calculator tool, the command line for that tool must be specified in the “Command to Invoke Delay Calculator.” The following parameters are recognized and substituted with the appropriate runtime values:

    Parameter

    Substitution

    %t

    Technology files

    %v

    Verilog files (Include both files you would specify in the Netlist pane and Additional Verilog Files field)

    %p

    Top module in Verilog

    %i

    Input parasitic file

    %f

    Parasitic file format

    %o

    Output SDF file

    %z

    SDC constraints file

    %m

    Cell name
  9. If the ADMS flow is enabled, the command specified in the “Command to Invoke Delay Calculator” field is run upon completion of the formatting step of Calibre-xRC. This command is used to create SDF files for blocks specified to be extracted in SDF format.
