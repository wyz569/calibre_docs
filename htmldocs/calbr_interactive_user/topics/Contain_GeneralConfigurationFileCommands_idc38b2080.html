General Configuration File Functions

The Calibre Interactive xACT configuration file has functions to add and remove pages, write to the transcript, add a separator line, and add text to the GUI. With the exception of the transcript function, these functions do not affect the Calibre run.

See “Configuration File Format for xACT GUI Customization” for information on the file format and how to invoke Calibre Interactive xACT with a custom configuration file.

Command

Description
addHeaderAdds a header label with a shaded background to the Calibre Interactive xACT GUI.
addLabelAdds a text label to the Calibre Interactive xACT GUI.
addMasterSpecifies that the visibility of a control depends on the state of another control (the primary control) defined in the Calibre Interactive xACT configuration file.
addMasterSelectSpecifies that the visibility of a control depends on the state of another control (the primary control) defined in the Calibre Interactive xACT configuration file. The dependent control is displayed if the primary control is not selected.
addMastersWithExprSpecifies that the visibility of a control depends on one or more other controls (the primary controls) that are defined in the Calibre Interactive xACT configuration file. The logic for displaying the dependent control is defined with an expression involving the primary controls.
addMastersWithFunctionSpecifies that the visibility of a control depends on one or more other controls (the primary controls) that are defined in the Calibre Interactive xACT configuration file. The logic for displaying the dependent control is determined by a function defined in the configuration file.
addPageAdds a custom page to the Calibre Interactive xACT GUI or changes the page order of an existing page.
addSeparatorAdds a separator line with an optional text label to the Calibre Interactive xACT GUI.
envVarsThe tool.envVars object includes function calls to read and set environment variables from the Calibre Interactive xACT custom configuration file.
FileIOThe FileIO component and associated functions provide the means to perform file I/O with the Calibre Interactive xACT custom configuration file.
getCalibreVersionReturns a string containing the Calibre version, for example v2017.3_0.74.
getProcessIDReturns the process ID for the Calibre Interactive GUI.
hasOwnPropertyReturns a Boolean value indicating whether the specified option name exists.
removePageRemoves a page from the Calibre Interactive xACT GUI. Pages with invalid objects are visible even when removed.
setItemOrderDefines the order of controls on a page in Calibre Interactive.
transcriptOutputs text to the transcript.
