Running ERC Checks

Calibre can execute Electrical Rule Check (ERC) statements during a Calibre nmLVS run.

Prerequisites

  • A Calibre nmDRC license in addition to the Calibre nmLVS license. Refer to the Calibre Administrator's Guide for more details.

Procedure

  1. Click LVS Options on the left panel to display the LVS Options pane. If the LVS Options button is not available, choose Setup > LVS Options.
  2. Click the ERC tab on the LVS Options pane.
  3. Enable the “Run ERC” option.


  4. Click the Select Checks button to select the ERC checks that are executed. Refer to the “Selecting ERC Checks” section for more information.
  5. Specify the ERC Results File.
  6. (Optional) Enable “Output ERC errors in cell space.” When enabled, this setting causes ERC results to be reported in cell space. This statement is necessary if you want to highlight ERC results in the cell they are reported in rather than the top cell, as described in the section “Viewing Results in Cell Space” in the Calibre RVE User’s Manual. This option is enabled by default. This option is required for Calibre Auto-Waivers.
  7. (Optional) Enable “Report results from pseudocells in” and select either “Pseudocells” or “Top cell.” See the tooltip for descriptions.
  8. Specify the ERC summary report filename.
