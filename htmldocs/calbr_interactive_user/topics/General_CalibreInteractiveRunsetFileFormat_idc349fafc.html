Calibre Interactive Runset File Format

Input for: Calibre Interactive

Created by: Calibre Interactive

Calibre Interactive runset files are built automatically using the Calibre Interactive GUI. Although not recommended, it is possible to manually edit the options in a runset file.

The majority of runset options can be set using the Calibre Interactive GUI. For example, the Calibre nmDRC runset option, drcLayoutSystem, can be set by choosing the Inputs pane and selecting an option from the “Format” dropdown list. In most cases finding the GUI element corresponding to each runset option is straightforward.

Format

The Calibre runset file is made up of runset options and their corresponding values. The syntax is as follows:

*OPTION: value

Parameters

See the section “Option Descriptions” for a listing of the OPTION and value pairs for each Calibre Interactive application.

Environment variables may be used for runset values that correspond to Calibre Interactive fields that accept a pathname, such as the rule file name or the run directory.
