Specifying Multiple Layouts for a DRC Run

If your primary cell has references to other additional database files you may need to load multiple layouts. You specify multiple layouts using the DRC Options pane in the Database > Library sub-tab.

Calibre Interactive checks the format of the additional database files and only includes files that match the format of the primary layout database.

Note:

The Library sub-tab is not available for FastXOR runs.

Prerequisites

  • Main layout database in GDSII or OASIS format.

  • Additional layout files in GDSII or OASIS format. Files compressed with gzip may be used. The additional layout files must be the same format as the main layout database.

Procedure

  1. Click Inputs, then click the Layout tab.
  2. Enter a layout file in the File field of the Layout tab.
  3. Select GDSII or OASIS for the Format, as appropriate.
  4. Choose Setup > DRC Options to open the DRC Options pane.
  5. Click the Database tab; then click the Library sub-tab.
  6. Use the buttons next to “Additional Layout Files” to add the required files.

    The additional files are added to the Layout Path statement. Only files that match the format of the primary database are added.

  7. If desired, enable the checkbox in the “Golden” column to add the GOLDEN keyword to the layout file specification in the Layout Path statement.

    The GOLDEN keyword is used when doing “golden” cell matching. See the Layout Path statement and the WITH MATCH GOLDEN keyword for the Extent Cell, Inside Cell, and Not Inside Cell statements for further information on the GOLDEN keyword.

  8. (Optional) In order to receive a warning if the additional layout files are not the same format as the main layout database, select Setup > Preferences, click the Misc tab, and enable “Warn when library layout file format differs from primary layout format.” The warning is displayed when you click the Run DRC button.

    Additional layout files that are not the same format as the primary database are ignored.
