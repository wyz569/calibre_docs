Calibre Interactive Overview

Calibre Interactive is a GUI interface to Calibre nmDRC, Calibre nmLVS, Calibre parasitic extraction tools, Calibre PERC, Calibre DFM, and Calibre 3DSTACK. You can easily adjust rule file settings and run-time options without editing the rule file, then submit the Calibre job on a local, remote, or distributed computer resource.

Calibre Interactive is integrated to most standard industry design tools, as discussed in the section “Integration to Design Tools”. When Calibre Interactive is connected to an integrated design tool you can make changes in the layout or schematic, then run Calibre on the modified version; the current layout or schematic is exported to Calibre prior to the start of the run.
