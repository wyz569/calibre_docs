Running Calibre Interactive nmLVS

Use the following procedure to start a Calibre nmLVS run from Calibre Interactive—nmLVS.

Prerequisites

  • All necessary input data and files.

Procedure

  1. Enter all necessary information in the Rules, Inputs, Outputs, and Run Control panes.
  2. Choose Setup > LVS Options and select each of the option tabs to set additional options. All left panel buttons will be green when the necessary information is entered.
  3. (Optional) To specify triggers, choose Setup > Preferences, then the Triggers tab. See “Trigger Functions in Calibre Interactive”.
  4. Click Run LVS on the left panel of Calibre Interactive.

    Calibre Interactive first verifies that the source netlist is complete; the source netlist check is case insensitive. If an internal pre-execution trigger is defined, the check for a complete source netlist is done after the trigger executes. You can turn off the source netlist check by disabling the setting “Check source netlist is complete before running LVS” (Setup > Preferences, Misc tab). See “Miscellaneous Preferences” for information on the source netlist check.

    When the run starts, the view switches to the Transcript pane, and any run output, including errors and warnings, is displayed there.

  5. View the transcript pane and examine the results of the run. See “Transcript Pane in Calibre Interactive”.
    Tip

    You can generate and view a control file without running Calibre by holding down the Ctrl key and clicking Run LVS. In addition, the calibre command line for the run is written to the transcript pane.

Results

If an error occurs in Calibre Interactive before the Calibre run is started, the error is displayed in a pop-up window or in the Calibre Quality Agent for Calibre Interactive.

When the Calibre run starts, the view switches to the Transcript pane, and any run output, including errors and warnings, is displayed here. See “Transcript Pane in Calibre Interactive”.

If “Start RVE after LVS finishes” is checked, the results database is opened in Calibre RVE when the run is complete; see “Using Calibre RVE for LVS” in the Calibre RVE User’s Manual. In addition, the Calibre Interactive GUI may open a report file in a text viewer when the run completes; see “Viewing and Editing Text Files in Calibre Interactive”.
