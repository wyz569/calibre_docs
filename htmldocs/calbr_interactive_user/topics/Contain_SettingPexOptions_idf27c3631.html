Setting PEX Options

You can customize the extracted parasitic netlist by selecting appropriate options in Calibre Interactive. The settings in Calibre Interactive override rule file settings and defaults for the parasitic extraction run. Most option settings are on the PEX Options pane, while some are on the Outputs pane.

Do one of the following to access the PEX Options pane:

  • Choose Setup > PEX Options. This action adds the PEX Options button to the left side of the Calibre Interactive PEX window and opens the tab.

  • Click the PEX Options button on the left panel if it is present.
