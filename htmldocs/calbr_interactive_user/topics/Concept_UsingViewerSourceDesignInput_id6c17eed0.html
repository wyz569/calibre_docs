Using a Viewer as the Source for Design Input

The “Export from layout” and “Export from schematic” settings on the Inputs pane of Calibre Interactive are used to specify the open layout or schematic in an connected viewer to be the input for your Calibre run. Layout and schematic export is possible for both interactive and batch runs. In batch mode, you must also disable the setting “Ignore layout and source export in batch mode” on the Misc tab in the Setup Preferences dialog box (Setup > Preferences) when doing layout or schematic export.

You can also define a preference to have the export setting depend on whether you started Calibre Interactive from a layout or schematic viewer. See “Export from Viewer Preferences”.

  • For communication with the design tool, see “Setting the Socket Port in Calibre Interactive”
