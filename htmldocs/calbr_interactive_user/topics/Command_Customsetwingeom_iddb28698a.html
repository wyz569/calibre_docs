CUSTOM::SETWINGEOM

Command type: Customization command for Calibre Interactive.

Sets the size and location of the customization window.

Usage

CUSTOM::SETWINGEOM widthxheight[±x_coord±y_coord]

Arguments

widthxheight

The parameters width and height are positive integers specifying the desired pixel dimensions of the window. The parameters width and height are separated by a lowercase ‘x’.

The geometry parameter has no spaces between any of the elements.

±x_coord±y_coord

The optional parameters x_coord and y_coord specify the desired location of the window on the screen, in pixels.

If x_coord is preceded by plus sign (+), it specifies the number of pixels between the left edge of the screen and the left edge of the window’s border; if preceded by a minus sign (-) then x_coord specifies the number of pixels between the right edge of the screen and the right edge of the window’s border.

If y_coord is preceded by plus sign (+) then it specifies the number of pixels between the top of the screen and the top of the window’s border; if y_coord is preceded by a minus sign (-) then it specifies the number of pixels between the bottom of the window’s border and the bottom of the screen.

Examples

The following example causes the window to display with its upper left corner at screen coordinate (10,20).

CUSTOM::SETWINGEOM 200x200+10+20
