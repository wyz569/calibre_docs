Basic Editing of Custom Check Selection Recipes

The basic editing controls allow you to construct check recipes based on a short list of common expressions, such as “Checks selected in the rule file” and “Checks with density”. You can also include and exclude individual checks and check groups.

Custom check recipes (also called user recipes) allow you to control which Calibre rule checks are executed during a Calibre nmDRC or Calibre DFM run. Check recipe definitions are saved in the Calibre Interactive runset, along with the currently active check recipe.

Prerequisites

  • Calibre Interactive nmDRC or Calibre Interactive DFM is running.

  • A rule file is loaded on the Rules pane.

  • “Disable check selection recipe editing” is not enabled on the Disable tab of the Setup Preferences dialog box (Setup > Preferences).

Procedure

  1. Click the Rules button on the left panel of Calibre Interactive to view the Rules pane.
  2. Specify a rule file. You cannot edit a check recipe without specifying a rule file.
  3. Choose the recipe you want to edit in the Check Selection Recipe dropdown list.
    Tip

    If you edit a built-in recipe, it is automatically saved as a user recipe with “(Modified)” appended to the recipe name. Only one modified version of each built-in recipe is saved at a time—new modifications overwrite any existing modified recipe.

  4. Click the Edit button at the far right of the Check Selection Recipe entry to open the Check Selection Recipe Editor dialog box.
  5. For basic recipe editing, make sure the dialog box appears as shown in the following figure. If it doesn’t, click the Advanced button to collapse the advanced controls.


  6. Use the Include and Exclude selections to select the types of checks in the check recipe.
  7. Enable and disable individual checks and check groups in the Recipe Checks area. The Groups tab is only displayed if check groups are defined with the Group statement.
  8. (Optional) Click Export to save the check recipe to a file; recipe files are given the .rcp file extension. The recipe file can be imported by other users with the Import button. Saved recipes can be imported by Calibre RealTime Custom and Calibre RealTime Digital.
  9. Click OK to save the recipe and exit the dialog box.
    Tip

    If you edited a built-in recipe, you can click the Rename button to give the modified recipe a new name.

  10. (Recommended) Choose File > Save Runset to save the recipe definition to a runset.
    Tip

    You must either save a runset or export the recipe to a file if you want the new check recipe to be available in later sessions of Calibre Interactive.
