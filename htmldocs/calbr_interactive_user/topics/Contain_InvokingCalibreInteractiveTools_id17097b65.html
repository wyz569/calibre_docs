Invoking Calibre Interactive Tools

You can invoke Calibre Interactive from a supported design tool or from the command line.

Invocation methods:

  • From a design tool with a Calibre interface.

    This method automatically sets up communication between the design tool and Calibre Interactive.

  • From a custom menu item in a supported design tool.

    See “Custom Menu Items in Design Tools”.

  • From the command line.

    When you invoke Calibre Interactive from the command line, you have the choice of running interactively (with the GUI) or in batch mode (without the GUI).
