Using the Other Cluster Option

The “Other Cluster” option provides a generic interface for configuring distributed computer solutions.

Prerequisites

  • For runs using remote hosts, configure your environment to allow rsh to execute your Calibre job remotely. This may include creating and editing a .rhosts file in your home directory.

Procedure

  1. Click the Run Control button on the left panel of Calibre Interactive.
  2. In the Performance tab, select “Other Cluster” from the “Run Calibre on” dropdown list.
  3. Enter the appropriate command for your cluster in the Queue Command text field.

    See “Replaceable Parameters for the Other Cluster Queue Command” for a list of the replaceable parameters that can be used in the queue command.

  4. (Optional) Set the “Synchronous” option as follows:

    Synchronous

    Behavior

    On

    Calibre Interactive waits for the Calibre run to complete. Calibre RVE and reports are displayed when the run is finished if appropriate options are set.

    Off

    Calibre Interactive executes the cluster command as a background process. The run transcript, Calibre RVE, and reports are not displayed when the run finishes.

  5. Choose Run Calibre options for your job. For details refer to Table 2 in the topic “Specifying the Host Processor and Run Options”.
  6. Click the Environment tab and configure the remote options appropriate for your run. For an example remote setup configuration, see Figure 1 in the topic “Configuring the Remote Environment”.
