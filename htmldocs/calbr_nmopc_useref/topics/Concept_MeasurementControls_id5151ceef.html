Measurement Controls

Create custom measurements for fragment length, distances, EPE, and CD using the Tcl commands.

The measurement controls are listed in Table 1.

Table 1. Calibre nmOPC Tcl-Based Measurement Commands Summary

Command

Description

MEASURE

Calculates distances from fragments on the first input layer to nearest fragments on the second input layer.

MEASURE_PER_FRAGMENT

Calculates distances from the specified fragment to the specified input layer.

MEASURE_PER_FRAGMENT_QUERY_COUNT

Returns the total number of retrievable edges.

MEASURE_PER_FRAGMENT_QUERY_FIRST

Initiates retrieval of edge data recorded by MEASURE_PER_FRAGMENT.

MEASURE_PER_FRAGMENT_QUERY_NEXT

Retrieves subsequent edge data recorded by MEASURE_PER_FRAGMENT (following MEASURE_PER_FRAGMENT_QUERY_FIRST).
