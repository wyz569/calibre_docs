Optional Setup File Operations

There are a number of additional optional operations that can be performed from the Calibre nmOPC setup file.

Table 1 provides a list of the supported optional setup file operations available and where they are described in detail.

Table 1. Optional Calibre nmOPC Setup File Operations

Operation

Notes

“Select Correction Method”

If no method is specified, Calibre nmOPC defaults to automatic fragmentation.

“Define Mask Rule Constraints”

A mask rule constraint is a type of spacing-based limitation where the rules are enforced on the photomask.

“Define Process Windows”

Process windows are the tolerable change in dose, focus, and mask sizing until the design fails to yield on silicon.

“Check Enclosure With Image Cost Functions”

Check for contact or via enclosure on the wafer with images.

Set maximum OPC iterations and maximum edge movement distance per iteration.

(See the max_iterations and max_iter_movement command descriptions in the “Calibre nmOPC Reference Syntax” chapter.)

Define how many iterations of OPC you want to perform and how far you want edges to be biased.

“Define No OPC Regions”

Establish regions where OPC will not be performed, assuming those areas are free of MRC violations.

“Smooth Biased Target Layers”

Correct small, irregular biasing on a target layer by smoothing.

“Adjust for Mask Corner Rounding”

Adjust mask polygons for corner rounding that occurs during lithographic exposure.

“Simulate Layer-Based Images Using Calibre OPCverify”

Use Calibre OPCverify commands to generate images based on layers created using setlayer.

“Create Custom Scripts for Calibre nmOPC” and the “Calibre nmOPC Custom Tcl Scripting Reference” chapter.

Use the Calibre nmOPC Tcl-based custom script to augment the existing nmOPC algorithms.
