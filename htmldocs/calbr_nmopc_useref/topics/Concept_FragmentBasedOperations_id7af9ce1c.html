Fragment-Based Operations

You can apply different operations to fragments, including measurements, retrieving EPE information, and setting displacements.

Table 1 lists the Calibre nmOPC Tcl-based fragmentation-based operations for performing operations on fragments during OPC.

Table 1. Fragment-Based Operations Commands Summary

Command

Description

auto_target_control

Automatic target control used to adjust a small four-point polygon for convergence

fragmaxedge

Applies the Calibre OPCpro minedgelength operation, but only to the fragments in the specified tag set.

fragtypetag

Creates a tag set based on a specified fragmentation operation type.

FRAGMENT_EPE_MEASURE_METHOD

Allows you to select between minimum, maximum, or average values of EPE for a tag set.

FRAGMENT_GET

Dumps the raw fragment information into a Tcl associative array named by the caller.

FRAGMENT_GET_DISPLACEMENT

Gets the fragment displacement for a single fragment or for an entire set of fragments (in array mode).

FRAGMENT_SET_DISPLACEMENT

Sets a “desired displacement” for one or more fragments.

FRAGMENT_ITERATOR first/next/release

Allows the caller to iterate over active fragments.

GET_FRAGMENT_EPE_INFO

Retrieves the EPE measurement method and value for specified fragment(s).

NEWTAG expression

Evaluates numerical properties about or attached to fragments.

NEWTAG property

Annotates a tag set with DFM properties.

PW_RETARGET

Applies PWOPC-based retargeting for OPC fragments in either a tag set or for all fragments.

SET_EPE

Sets EPEs on fragments, based on an input Tcl formula or expression.

SET_MEDIAN_EPE

Sets median EPE for a tag set of fragments.

TAG_FRAG_SRC

Gets the fragmentation rule that created the fragment.

TDBIAS

Sets the displacement (requires a call to FRAGMENT_MOVE unless the -force flag is used).
