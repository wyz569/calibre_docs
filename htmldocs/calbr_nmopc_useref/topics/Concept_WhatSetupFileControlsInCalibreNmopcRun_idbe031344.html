What the Setup File Controls in a Calibre nmOPC Run

In the Calibre nmOPC setup file, you control how simulations are performed, how corrections are applied, and any post processing or verification required.

You can design a setup file with minimal simulation controls because dense simulation samples data at evenly spaced grid points on a dense simulation grid. You do not need to specify the size of this grid because the optimal grid size, or Nyquist sampling distance (also referred to as “Nyquist value” in this document), is calculated based on the optical conditions.

Figure 1. Dense Grid Simulation

Correction controls can also be minimal because the tool fragments the polygon edges for you based on the NSD, calculates the edge placement error (EPE), and moves edges based on the EPEs.

Refer to “Variables, Keywords, and Commands Supported in the Setup File” to see what types of controls are available for the Calibre nmOPC setup file.
