Tcl Shell Hcell Analysis Commands

Hcell analysis commands provide information for constructing preliminary hcell correspondence lists.

Table 1 shows the summaries for each supported command. The hcells:: family of commands has equivalent Query Server commands from the non-Tcl interface. Whenever possible, the hcells:: family of commands should be used in place of the historical equivalents. A legend for error codes is presented under “Query Server Error and Failure Messages”.

Table 1. Hcell Analysis Commands

Commands

Description

hcells::add_matching_hcells

Adds cells identified with the hcells::automatch or hcells::placementmatch commands to the current hcell list.

hcells::automatch

Identifies layout and source cells that match by name.

hcells::clear_hcell

Removes a corresponding cell pair from the current hcell list.

hcells::clear_hcells

Removes all corresponding cell pairs from the current hcell list.

hcells::evaluate_current_hcells

Controls hcell effectiveness evaluation.

hcells::expand_unbalanced

Controls unbalanced hcell expansion.

hcells::hcell

Adds a single hcell pair to the current hcell list.

hcells::hcells

Creates a current hcell list from an hcell file.

hcells::placementmatch

Identifies corresponding layout and source cells that match by placement count.

hcells::print_hcells

Writes the current list of hcells.

hcells::print_hierarchy

Writes a hierarchy tree report of cell statistics.

hcells::read

Reads in the layout and source netlists, along with any Hcell statements that may be in the rule file.

hcells::select

Selects hcells in the current hcell list based upon effectiveness thresholds and generates a report of the current hcell statistics.

hcells::status

Displays the settings of the hcell evaluation system.
