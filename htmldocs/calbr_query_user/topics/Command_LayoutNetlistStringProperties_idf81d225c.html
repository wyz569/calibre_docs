LAYOUT NETLIST STRING PROPERTIES

CCI command. Corresponding Tcl shell command: dfm::set_layout_netlist_options ‑string_properties.

Controls the output of string properties. String properties produced with the device property computation language are output by the LAYOUT NETLIST WRITE command by default.

Usage

LAYOUT NETLIST STRING PROPERTIES {YES | NO}

Arguments

YES

Keyword that specifies string properties generated by a Device property computation program are output. This is the default.

NO

Keyword that specifies string properties are not output.
