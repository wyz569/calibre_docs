BROWSE DEVICELESS CELLS

Query Server command.

Controls how BROWSE commands report cells that do not contain any devices throughout their hierarchy (via cells, for example).

Usage

BROWSE DEVICELESS CELLS {NO | YES}

Arguments

NO

Keyword that specifies instances of cells without devices are not reported, but their contents are. This is the default.

YES

Keyword that specifies instances of cells without devices are reported, but their contents are not.

Acknowledgments

OK.

Examples
