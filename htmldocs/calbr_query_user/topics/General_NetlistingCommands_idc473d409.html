Netlisting Commands

Netlisting commands read, configure, or write SPICE netlists.

Commands in the dfm:: scope are documented in the Calibre YieldServer Reference Manual.

Table 1. Netlisting Commands

Command

Description

dfm::close_netlist

Closes a netlist handle generated by dfm::read_netlist.

dfm::list_layout_netlist_options

Lists options that are set using dfm::set_layout_netlist_options.

dfm::read_netlist

Reads a SPICE netlist specified by Layout Path or Source Path in the rule file and configures the netlist transformations. Used with dfm::write_spice_netlist.

dfm::set_layout_netlist_options

Configures the dfm::write_spice_netlist command for writing a layout netlist from the PHDB.

dfm::set_netlist_options

Configures the output of dfm::write_spice_netlist from a netlist read by dfm::read_netlist. Controls how comment-coded properties are written.

dfm::write_spice_netlist

Writes a SPICE netlist. If used without a dfm::read_netlist handle, writes a SPICE netlist similar to calibre ‑spice based upon the PHDB.

qs::write_layout_netlist_names

Writes a Layout Netlist Names correspondence file.

qs::write_separated_properties

Writes push-down separated properties to a file.
