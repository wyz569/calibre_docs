NETLIST HCELLS

Query Server command. Corresponding Tcl shell command: hcells::hcells.

Reads in an existing hcell list.

Usage

NETLIST HCELLS filename

Arguments

filename

A required pathname to a file containing an hcell list.

Description

Causes the filename to be read in and used as an hcell file (similar to the -hcell Calibre nmLVS command line option). Removes any existing hcells from the hcells list maintained by the system.

Acknowledgments
