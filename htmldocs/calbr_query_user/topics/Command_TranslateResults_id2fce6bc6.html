TRANSLATE RESULTS

Query Server setup command. Corresponding Tcl shell command: dfm::create_filter -translate.

Displaces output coordinates by specified values. The TRANSLATE RESULTS transformation is applied after any REFLECTX RESULTS, ROTATE RESULTS, or MAGNIFY RESULTS transformations (in that order), regardless of the order in which they are issued to the server.

Usage

TRANSLATE RESULTS {x_offset | y_offset}

Arguments

x_offset

A required integer in database units that is added to all x coordinates. The default is 0.

y_offset

A required integer in database units that is added to all y coordinates. The default is 0.
