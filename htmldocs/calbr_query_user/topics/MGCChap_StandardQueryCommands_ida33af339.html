Standard Query Commands

The standard mode Query Server responds to commands that query SVDB databases and the server environment. All commands return a brief acknowledgment upon execution. Some commands return a more verbose response. In either case, the reference pages discuss the messages that the server returns. For commands that have counterparts in the Query Server Tcl shell, the Tcl command equivalents are shown in the header of each reference page. The Tcl shell is the preferred interface.
