Administrative Commands

The administrative commands provide high-level control of the Query Server Tcl shell and session information.
Table 1. Administrative Commands

Command

Description

dfm::create_timer

Returns a timer object.

dfm::get_timer_data

Returns time and memory use information from a timer object.

dfm::reset_timer

Restarts a timer object.

qs::close_svdb

Closes the database without quitting Query Server.

qs::copy_svdb

Creates a copy of an SVDB database.

qs::exit

Exits the Query Server Tcl shell (same as qs::quit).

qs::help

Displays a table of all Query Server Tcl shell commands.

qs::open_svdb

Opens an SVDB database.

qs::password

Sets a password for the current SVDB.

qs::perf_stats

Starts the performance monitor.

qs::status

Provides the state of server settings.

qs::quit

Exits the Query Server Tcl shell.
