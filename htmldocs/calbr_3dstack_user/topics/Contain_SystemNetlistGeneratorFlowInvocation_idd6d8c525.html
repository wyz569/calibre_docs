System Netlist Generator Flow and Invocation

The System Netlist Generator allows you to easily create source netlists for complex stacked systems. It supports batch and interactive modes.

Calibre Interactive can invoke the System Netlist Generator to create a source netlist before a verification run. For more information on Calibre Interactive, see “Creating a Source Netlist”.
