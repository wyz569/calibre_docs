System and Miscellaneous Commands

System and miscellaneous commands specify the verification inputs, outputs, and run control options.
Table 1. System Control and Miscellaneous Commands

Name

Description
assembly_path_extensionSets the severity of circular GDS path extension errors.
export_connectivityExports the connectivity information into a netlist or annotated GDS file.
export_layout (Standard Syntax)Generates an annotated GDS of a specified portion of the assembly, including the layers of the specified placements.
export_templateExports a specified chip or placement from the stack into a GDS or OASIS file.
layer_props_fileSpecifies a layer properties file for Calibre DESIGNrev that sets custom layer visibility options.
reportGenerates a report file that contains information about the verification run. This is useful for debugging the design.
runRuns Calibre using the specified rule file and command-line options.
set_versionSets the syntax version of the Calibre 3DSTACK rule file.
source_filterApplies filtering options to the imported source netlist file.
source_netlistImports a source netlist file used for connectivity comparison. It is highly recommended that you use a source netlist during the verification of your assembly.
svrf_specsIncludes a file containing SVRF statements in the run.
warning_severitySpecifies the severity level for warning messages.
