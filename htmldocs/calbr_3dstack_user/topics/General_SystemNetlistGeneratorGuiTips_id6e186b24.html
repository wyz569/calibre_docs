System Netlist Generator GUI Tips

The graphical interface includes a number of features that allow you to quickly edit your design.

When you select a cell, a small rectangle appears in the cell’s bottom-right corner. Click this rectangle to select multiple cells as shown in the following figure.

Figure 1. Select Multiple Cells by Clicking and Dragging the Cell Corner

Edit properties by entering values for each field or by copying and pasting values from existing fields. Figure 2, Figure 3, and Figure 4 demonstrate how you can easily modify multiple cell properties at once.

Figure 2. Copy Existing Properties in the System Netlist Generator

Figure 3. Shift-select and Paste Into Multiple Cells in the System Netlist Generator

Figure 4. Updated Cells In the System Netlist Generator

Enable the Show generated netlist file in Calibre RVE SPICE File Viewer checkbox to view a graphical representation of the complete netlist.
