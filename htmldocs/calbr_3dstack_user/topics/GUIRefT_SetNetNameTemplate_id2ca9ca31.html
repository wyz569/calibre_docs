Set Net Name Template

Edit > Set Net Name Template

Controls how net names are generated when port connections are assigned.

Figure 1. Set Net Name Template

Objects

Table 1. Set Net Name Template Contents

Item

Description

Prefix

String that is prefixed to all net names defined in a sng::connect command that do not have an assigned net name.

Delimiters

Start end delimiters for the net. For example, { and }.

Save Net Name Template

Saves the net name template so that it is applied to port connections when they are created.
