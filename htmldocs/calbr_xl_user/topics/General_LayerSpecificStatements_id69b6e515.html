Layer-Specific Statements

This section describes the SVRF statements used to define your layout layer information.
Table 1. Layer-Specific SVRF Inductance Statements

Rule

Description

PEX Elayer

Specifies conductor layer equivalency and maps layer names in the layout to layer names you use (if different) when defining layer height information in the xCalibrate tool.

PEX Inductance Extract Layers

Declares the layers to process during inductance extraction.
