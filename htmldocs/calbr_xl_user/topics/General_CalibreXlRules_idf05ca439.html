Calibre xL Rules

Use Calibre xL inductance rules to control parasitic impedance extraction.

Following are the categories of SVRF rules for parasitic impedance extraction:

  • Layout database layer information rules — define the stacking height and thicknesses of conductor layers, some impedance information for each conductor layer, and equivalent layer information. See “Layer-Specific SVRF Statements”.

  • Parasitic impedance rules — define the frequency, range, return path, and other parameters. See “Calibre xL Statements”.
