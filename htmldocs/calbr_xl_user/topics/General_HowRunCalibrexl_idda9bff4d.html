How to Run Calibre xL

Calibre xL can be run from the command line or through a graphical user interface.

You can run the Calibre xL tool using either of following methods:

  • Creating a shell script and running it from the command line. See “Running Calibre xL From the Command Line”.

  • Interactively from the Calibre® Interactive™ graphical user interface. See “Running Calibre xL From Calibre Interactive ”.
