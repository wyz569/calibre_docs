Requirements for Generating a VT5 Resist Model

There are several prerequisites for generating a VT5 resist model.

Before you generate a resist model, you must have the following:

  • Test pattern — The design file from which you generated the mask for the test chip you used to gather empirical data. The pattern must use the following guidelines:

    • For calibration purposes, measurement sites need to be symmetric; in other words, the site marker needs to be placed such that it is either centered on a symmetrical edge (with loc=1) or in a space symmetrically equidistant from two parallel edges (with loc=0).

    • 2D patterns are supported, but must be symmetric in order for Calibre WORKbench to accurately measure the EPE, defined as (drawn CD - measured CD)/2.

    Note:

    If you do not use a test pattern generated by the Calibre WORKbench tool, you must use a test pattern that is a flattened layout file. Some modeling commands do not work on hierarchical layouts.

  • Sample spreadsheet — The sample spreadsheet that matches the test pattern design file. It must contain measurement data from the test chip.

  • Optical model — The optical model that simulates the optical system used to generate the test chip, previously generated in “Optical Model Creation”.

  • Resist model — The resist model that simulates the resist film behavior on the chip. A setup file must point to a valid resist model. If you do not have one, use the default resist model generated by Calibre WORKbench. While you can use this resist model as the starting point for designing a new model, you are not required to do so.

  • Setup file — The VT5 Center tool requires a valid setup file.

    Note:

    Opening a file for the RET Flow Tool also loads the optical and resist models referenced within that setup file.

    A default model is automatically copied into your home directory the first time you invoke the Calibre WORKbench application.The first time you generate a resist model, your setup file should point to the default resist model that is shipped with the software, ~/.calibrewb_workspace/models/defaultresist.mod
