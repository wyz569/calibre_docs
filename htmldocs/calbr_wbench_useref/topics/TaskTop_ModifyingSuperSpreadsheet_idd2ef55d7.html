Modifying the Super Spreadsheet

Use this technique to remove outliers present on one or more spreadsheets in order to help calibrate your process.

The SSS Process Window Analysis window uses the same Sample tab as VT5 Center. This means that you can use the interface to disable selected structures in any of the spreadsheets, and save the modified super spreadsheet for another VT5 Center run.

Prerequisites

  • A super spreadsheet file loaded in the SSS Process Window Analysis window

Procedure

  1. Click the Sample tab.
  2. Use the S tabs at the bottom to switch to individual sample spreadsheets (B in Figure 2).
  3. Edit the super spreadsheet as needed.
  4. When you have finished editing the super spreadsheet, click Export SSS (E in Figure 2) to save the modified file.
