Scripting Area Window

To access: From Calibre WORKbench, Litho > Calibre Pattern Generator (CPG). In the Calibre Pattern Generator, select the Scripting Area (SA) tool by clicking the Tool List icon on the top right and selecting SA Scripting Area in the menu.

The Scripting Area tool shows the Tcl script to edit and use for layout generation.

Figure 1. Scripting Area in Calibre Pattern Generator
