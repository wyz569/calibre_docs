Layout Generator Window

To access: From Calibre WORKbench, Litho > Calibre Pattern Generator (CPG). In the Calibre Pattern Generator, select the Layout Generator (LG) tool by clicking the Tool List icon on the top right and selecting LG Layout Generator in the menu.

The Layout Generator (LG) tool creates and edits the layout files used to generate test pattern layouts for RET modeling. The Layout Generator places the patterns, labels, and markers for test pattern design.

Figure 1. Layout Generator in Calibre Pattern Generator

Description

The Layout Generator creates and edits the layout files using pattern templates and libraries created by the Pattern Generator. The Layout Generator places the patterns, creates the labels for identification, and creates markers for identification and alignment.

Objects

Object

Description

Layout Configuration Display

Displays a list of layout configurations and their properties.

A selected layout configuration displays icons for Save Configuration, More Actions, and Close Configuration. More Actions items include Save As Library.

Add Configuration

Expands to New and Load.

New — Creates a new layout configuration.

Load — Loads an existing layout configuration file.

The Layout Viewer displays the layout configuration selected in the Layout Configuration Display.

Layout Viewer

Displays the layout configuration.


General Properties (gear) icon

Click to access Grid Properties, Layers Properties, DRC Properties, Sem Alignment Marker, Fine Alignment Marker, and Auto Focus Marker menu items.

Double-click to display a submenu with Label Properties, Gauges Properties, and Connectivity submenu items.


Grid Properties

Click to set the grid properties for pattern arrangement such as number of rows and columns, grid size and spacing, labels, and whether to allow clipping.


Layers Properties

Click to set layer properties such as pattern and layout label layers, drawing extents, and layer mapping.


DRC Properties

Click to specify an SVRF file for MRC configuration.


Sem Alignment Marker

Click to load a pattern library of SEM alignment markers.


Fine Alignment Marker

Click to load a pattern library of fine alignment markers.


Auto Focus Marker

Click to load a pattern library of autofocus markers.


Zoom All

Click to view entire layout configuration.


Clear Highlighting

Click to de-select highlighted items in the Pattern Viewer.


Verify MRC clean Layout

Click to start a DRC run using a user-specified Calibre nmDRC SVRF rule file.


Generate Test Layout

Click to save the layout configuration file and to create the layout and gauge files.

Pattern Library Table

Displays loaded pattern libraries and their templates.

  • To assign a pattern template to a grid, drag it from the table and drop it into a grid in the Pattern Viewer.

  • To add a pattern library, click “+” in the header.

  • To delete a pattern property, hover over the property name and click “X”.

  • To minimize or open the table, click the header.
