Using Calibre IPSE

The Calibre® Image Parameter Space Explorer (IPSE) tool and Calibre® Image Parameter Space Explorer Test pattern eXtender (IPSETX) tools are designed to display the image parameter space (IPS) coverage for a test pattern versus a layout design.

You can generate plots directly from a layout and compare it to log files from VT5 Center; you can also examine data generated by the samlinev2 Calibre tools batch command.

Once you have identified the places where your layout design is not covered by your test pattern results, you can use the Calibre Image Space Explorer Test pattern eXtender (IPSETX) tool to append the extra information to your design and spreadsheet.
