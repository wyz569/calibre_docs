set_out_gauge_file

Scope: Calibre Pattern Generator API Commands

Sets the name of the output gauge file.

Usage

$cpg set_out_gauge_file  "output_gauge_file"

Arguments

$cpg

A required argument specifying $cpg as a Tcl configuration object or variable.

output_gauge_file

A required argument specifying the name of the output gauge file. The filename must end with .gg and be enclosed in quotation marks.

Description

A required command specifying the name of the gauge file generated by the layout configuration file.

Examples

The following example saves the gauges to 3Bar.gg:

$cpg set_out_gauge_file "3Bar.gg"
