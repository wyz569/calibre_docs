configure_marker

Scope: Calibre Pattern Generator API Commands

Finds and configures a marker in a marker library.

Usage

$cpg configure_marker ‑name pattern_name [‑properties "{property_name property_value}..."]

Arguments

$cpg

A required argument specifying $cpg as a Tcl configuration object or variable.

‑name pattern_name

A required argument specifying the name of a pattern to use as a marker.

‑properties "{property_name property_value}..."

An optional argument specifying one or more pattern properties of a marker. The tool searches for a marker ID whose properties (such as CD) match the properties specified by this argument. The set of property name and value pairs must be enclosed with quotation marks. The braces in the syntax symbolize grouping and should not be included in the statement.

Description

An optional command that configures a marker and returns its ID. The marker ID can be used to add the marker as one of the following marker types:

  • SEM marker

  • fine alignment marker

  • autofocus marker

The identified marker is then used in an add_sem_marker, set_fine_alignment_marker, or add_auto_focus_marker statement.

Examples

This example searches a marker library that contains a pattern named pattern1 with properties named cd and cd2. When a pattern named pattern1 with cd equal to 100 and cd2 equal to 200 is identified, markerId is set to the ID of the matching pattern.

set markerId [$cpg configure_marker -name pattern1 -properties “cd 100 \
	cd2 200”]
