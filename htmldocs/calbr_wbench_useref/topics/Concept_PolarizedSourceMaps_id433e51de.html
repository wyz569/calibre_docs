Polarized Source Maps

For polarized sources, four graphs are displayed:

  • Upper left (Emag): Field amplitude |E|

  • Upper right (DOP): Degree of Polarization p

  • Lower left (atan Ex/Ey): Field vector angle, azimuthal tan (Ex/Ey)

  • Lower right (Ø (Ey) - Ø (Ex)): Phase shift between Ex and Ey

    Figure 1. Viewing Optical Kernels

For optical kernels, four graphs are displayed, representing the amplitude and phase components for both kernel and residue:

  • Amp (Kernel)

  • Phase (Kernel)

  • Amp (Residue)

  • Phase (Residue)

The kernels can also be displayed in Real and Imaginary format. The Residue graphs show differences in the kernel and reflected version of itself. The Residue X, Y, and XY radio buttons allow the user to select reflection through x axis, reflection through y axis, and reflection through both x and y axes for the residue.

The Range Min and Range Map controls select the display range scaling values on the side of the graphs. The Color Map dropdown sets the color scheme for the display.

One or more of these graphs can be selected for changes by clicking on the graph in the viewing area. A selected graph is highlighted in brighter green. Clicking a graph again unselects it. (Pale green graphs are unselected.)

Sectors View

If the optical model uses the Hybrid-Hopkins Abbe (HHA) method, it also contains source sectorization characteristics. You can visualize them using the File > Open Sector Map item and opening the Optical file associated with the optical model. A separate map for each polarization is displayed with the sector boundaries drawn. Symmetric sectors have the same colors across all polarization maps. Sector centers are marked with “X”s.

Tip

You can also calculate the sectorization information with the sectorinfo command, using the ‑mode sectorization parameter.
