Best Practices for Finding the VT5 Reference Threshold

Use these best practices specific to VT5 reference threshold calibration.

To verify that the chosen reference threshold is passing through the aerial image of all the structures:

  • Calibrate using all resist measured CD structures.

  • Check structures that have zero slope and no reference image. These can be caused by

    • A wrong reference threshold search range.

    • A test structure out of the process window range.

  • Enable force_epe_printing. This option optimizes the reference threshold so that all sample points resolve.

    • In VT5 Center, this control is a checkbox found on the VT5,3 tab.

    • In modelflow, this is the stage3a crosscheck argument.
