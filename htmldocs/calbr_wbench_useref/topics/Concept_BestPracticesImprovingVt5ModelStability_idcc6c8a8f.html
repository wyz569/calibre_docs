Best Practices for Improving VT5 Model Stability

Use these best practices to improve VT5 resist model stability.

  • Use an eigenvalue range between 0.01 and 0.001. Note that too small of a value causes overfitting; too large of a value results in underfitting.

    Eigenvalues are set using the Eigenvalue criterion field in VT5 Center, VT5, in tab.

  • Verify the model stability for each layer.

    A good stable model has density coefficients of +/- 0.15. You can see these values by examining the Threshold Coefficients graph in the Analyze Fitness dialog box.
