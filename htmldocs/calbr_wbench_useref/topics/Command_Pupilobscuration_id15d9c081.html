pupilObscuration

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Sets the transmission of the pupil to zero for points within a specified sigma of the pupil center.

Usage

pupilObscuration sigma

Arguments

sigma

A required argument specifying the outer sigma of zero transmission from the pupil center.

Description

This optional parameter sets the transmission of the pupil to zero for points within sigma of the pupil center. Valid ranges are 0 <= sigma <= 1.0. Used primarily with EUV lithography systems.

Note:

pupilObscuration 0 still blocks transmission at the (0, 0) point. To turn off this feature, remove the keyword from the optical file.

In optical files containing pupilObscuration, the image_normalization method must be either source_area or integ_source_area. This limitation ensures valid normalization for a dark field system (that is, when pupilObscuration sigma > source sigma) where the clearfield image intensity will be zero but an image can still be formed by light scattered by the mask.
