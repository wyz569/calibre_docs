pupilApodization

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Sets the pupil transmission model.

Usage

pupilApodization {edge_transmission et_val lossless_pt lp_val} | {file filename}

Arguments

edge_transmission et_val

A required argument set specifying the pupil transmission at the maximum NA, and can be between 0.5 and 1.0. This option also requires the lossless_pt argument to be specified.

Figure 1. Pupil Apodization Transmission Plot (edge_transmission)

lossless_pt lp_val

A required argument set specifying the radius in normal pupil coordinates (1.0 corresponds to the maximum NA) below which the pupil has no attenuation. It can be a value between 0.0 and 1.0.

Figure 2. Pupil Apodization Transmission Plot (lossless_pt)

Starting at coordinate lp_val, the pupil transmission decays quadratically from 1.0 to a value at the edge of the pupil given by et_val.

file filename

A required argument set specifying a file containing pupil apodization information. Its contents are interpolated to set the apodization value at each point in the pupil. If the filename includes a path, Calibre WORKbench tries to open the file from the following locations:

  • Using the relative path to the file, starting from the current working directory

  • Using the relative path to the file, starting from the Optical file location

  • As an absolute path

The text file must be in the following format:

  • A grid of space-separated numbers of N rows by N columns representing the pupil.

  • The first and last rows and columns of the grid must be tangent to the pupil boundary.

  • Points outside the pupil for which the apodization value is not known must be set to -1.

  • The known apodization values in the file can go beyond the pupil boundary. If a -1 value is near the boundary, it is not used for interpolation; the interpolation algorithm uses nearby points for valid values to use. If there are no valid points adjacent to a point in the pupil, it is considered an error.

Description

An optional pupil parameter specifying a model type for amplitude non-idealities in the pupil transmission that are independent of electric field polarization.
