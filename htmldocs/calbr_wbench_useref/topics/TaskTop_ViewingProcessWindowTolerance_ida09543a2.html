Viewing the Process Window Tolerance

Use the Analyze Process Window tool to view the common process window when process window data is available. This functionality is only available through CM1 Center.

Prerequisites

  • A super gauge data object

  • Calibre WORKbench invoked

Procedure

  1. Open CM1 Center (Litho > CM1 Center menu item).
  2. In CM1 Center, load the super gauge data file (Gauges > Open Super Gauge menu item).
  3. Select one or more gauges in the active gauge tab.
    Tip

    Click the Select button to bring up a dialog box that allows you to select gauges by structure name.

  4. Open the Analyze Process Window tool by selecting the Gauges > Process Window menu item.

Results

The selected gauges are shown in the Analyze Process Window.

The Common Process Window tab (Figure 1) shows the measured and simulated depth of focus, CD tolerance and dose latitude.

Figure 1. Analyze Process Window Dialog, Common Process Window Tab

The Plot View controls (A) can be used to find specific gauge plots in the window. They use the constructed notation shown in Table 1.

Table 1. Plot View Graph Abbreviations

Letter Code

Description

s or m

Simulated versus Measured

cpw

Common Process Window

g_#

Gauge number (from selected values)

lo or hi

Low and high values for the gauge (measured or simulated).

r or e

Rectangular or Ellipse

The Dose Latitude tab (Figure 2) shows the measured and simulated dose latitude compared versus the depth of focus.
