Comparing Bivariate Plots

The Bivariate Plots tab shows a number of standard image variables plotted against other image variables in the sample space.

The following variables can be plotted:

  • Densities

  • Imin

  • Imax

  • Slope

  • Sim(ulated) Threshold

  • Factor

    Figure 1. Bivariate Plot Tab

The points displayed in this tab rely on the items loaded in the File list at the bottom of the dialog box. In order to do a meaningful comparison, you must load at least one additional file from the following types:

  • the output of an earlier Calibre IPSE run

  • a sample data file created by a samlinev2 run

  • a modelflow (Version 1) log file (which can be created by the Calibre VT5 Center tool)

  • a modelflow_v2 log file (which can be created by the Calibre CM1 Center tool)

  • a Calibre nmModelflow Comprehensive Output Table (COT) CSV file generated with the following command (in Calibre nmModelflow):

    mdf gauges -export -view Cot -format csv filename

Use the File menu to load your data files:

  • Open IPS File — To open the output of the “Running a Simulation” task

  • Open Sam File — To open a sample data file

  • Open MfLog File — To open a modelflow_v1 or modelflow_v2 file

    Note:

    If the modelflow_v2 log file contains contour calibration information, the contour points also appear in the IPSE window.

    If the log file contains multiple calibration stages, you are prompted during the load action with a dialog box to select which stage to load the data from. Loaded files show the relevant calibration stage in the File list at the bottom of the window.

  • Open MDFv3 COT File — To open a Calibre nmModelflow file.

Each file is given its own list entry, color and symbol type, and is superimposed on the design in the Bivariate Plot tab.

Note:

An indication of good IPS coverage is when the points created by the image parameter space simulation follow the points from empirical data on a calibration test pattern. An indication of bad simulation result is when the points do not correspond well.
