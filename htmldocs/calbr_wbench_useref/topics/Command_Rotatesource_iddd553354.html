rotatesource

Optical Model Commands

Rotates a source map.

Usage

rotatesource -in input_file -out output_file -degrees {0 | 90 | -90}

Arguments

-in input_file

A required argument that specifies the path to the input source map file (.smf).

-out output_file

A required argument that specifies the path to the output source map file (.smf).

-degrees {0 | 90 | -90}

A required argument specifying the direction to rotate the output source map (90 for counterclockwise, -90 for clockwise, or 0 for no rotation).
