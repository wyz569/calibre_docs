Examining the CM1 Model for Quality

Follow this procedure when you have completed a fine calibration run to inspect the results in CM1 Center.

Prerequisites

  • You must have completed the procedure, “Performing Fine CM1 Calibration,” in order to have the fine calibration log used in this procedure.

Procedure

  1. In CM1 Center (Litho > CM1 Center), switch to the Optimize tab.
  2. Click the Import button in the lower right portion of the window. Open the fine calibration log.
  3. Select the new line appearing below the Import button. Your initial model evaluation may be performed by reviewing the CM1mean (mean CD error), CM1erms (CD error RMS), and CM1/AI diff (CM1 to Aerial Image difference) fields.
    Figure 1. Best Values and Evaluation Results

    Note:

    The following recommendation is based on limited internal experiments:

    If the CM1 model exhibits a CM1/AI difference >=15%, the model has a high probability of producing artificial extra printing features or false holes in the contour. Since a best practice is to explicitly set the CM1/AI maximum to 10%, this will usually not happen under normal circumstances.

  4. Click the Analyze button. The Result Record dialog box appears. On the left side is a results navigation tree with Gauges Fitness Data and Search Data trees that correspond to each of the stages run in the log.

    You can manipulate the graphs using the mouse, as shown in Table 1:

    Table 1. Navigating Analysis Graphs

    Control

    Action

    Right mouse button

    Hold, drag southeast and release to zoom in, hold, drag northwest, release to zoom out; click once to restore the default view size

    Left mouse button

    Select a point

    Ctrl‑left mouse button

    Toggle selection

    Shift‑left mouse button

    Add to selection

    Middle mouse button

    Pan

    Ctrl‑middle mouse button

    Show/hide graph legend

    Switching sections in the tree shows further analysis information about the CM1 model. Each of these display types are briefly described in “Analysis Graph Sections”.

    Tip

    Selecting two or more different log lines (click to select the first, then Ctrl-click to select the second) and clicking Compare brings up an enhanced Analysis graph that has the following additional features:

    • Per-structure type Show/Hide
    • A unique structure name displayed for structures where the two logs have matching structure lists
  5. Perform further optimizations as needed.
