Creating a Gauge File Using the Raw Data Filtering Tool

Use the Raw Data Filtering tool to create a gauge file from CD SEM files.

Prerequisites

  • Directory containing raw data (that is, CD SEM image and report files)

  • (Optional) Wafer map file.

  • (Optional) Design-based metrology (DBM) file or Condition (cond.txt) file.

  • (Optional) Layout file.

  • (Optional) Gauge layer number.

  • (Optional) Maximum CD in dbu.

Procedure

  1. In Calibre WORKbench, invoke the Calibre SEMSuite from the Litho > Calibre SEMSuite menu item.
  2. Select the Raw Data Filtering (RDF) tool by clicking the Tools icon at the top right and selecting Raw Data Filtering in the menu.


  3. Add an RDF session to the session list.
    1. Select Add Session and select the session type for the CD SEM machine used for data collection.
    2. In the Open Session dialog box, select the parent directory path for the CD SEM image files.
    3. (Optional) Specify input files specific to the CD SEM machine.
    4. Click OK to complete session setup.
  4. Classify images as accepted, rejected, or undecided.

    • Accepted images pass due to acceptable processing and metrology. Data from accepted images is included in target statistics and added to the output gauge file.

    • Rejected images fail due to either process (that is, lithography) or metrology issues. Data from rejected images is excluded from target statistics.

    • Undecided images may have questionable process or metrology quality. Data from undecided images is excluded from target statistics.

    Classify images individually or by target groups using the Quick Decisions tool:

    • To classify images individually, click a classification icon below the SEM image display. The tool advances to the next image in the Target Repeats Browser table. Classify all repeats in the target group.

    • To classify images for a target group using the Quick Decisions tool, click the Quick Decisions icon at the top of the Targets list.



      Select a threshold metric and specify its value. Select the classification decision and the target groups to apply the threshold to. Click Apply. The Target Repeats Browser table and plots update with the Quick Decisions results.

    Advance to the next target group by clicking the arrow on the right of the target statistics pane at the top. Alternatively, select a target group in the targets list on the left. Continue classifying images until all target groups are done.

  5. Specify dose and focus values to each die.
    1. Click the Wafer Map tab on the right to access the wafer map table.
    2. Add dose and focus values to the wafer map table to correspond to the wafer columns and rows (that is, Chip ID X and Chip ID Y). Click Apply to update the dose and focus values for the targets.
  6. To generate a gauge file, click the Generate Gauge File icon at the top of the Targets list.


    To use the RDF result for contour extraction, select Load generated file to CDF.

    The default output gauge file is gauges.gg. Click Generate.

Results

The output gauge file is saved using the specified filepath. The tool generates a folder with images classified as approved. Images from the same target group are saved in subfolders with the target group name.
