Dense Modeling Modes of Operation

The following sections describe the ways you can create CM1 and VEB models.

CM1 Modes of Operation

Use either of the following tools to create CM1 models:

  • CM1 Center is accessed inside Calibre WORKbench, using the Litho > CM1 Center menu item. The CM1 options are contained in a tab.

  • The modelflow_v2 batch command can be accessed as part of a Calibre batch command file or from the Calibre WORKbench Tcl command shell.

VEB Modeling Modes of Operation

You can create a VEB model in either the Calibre WORKbench CM1 Center GUI or using batch command mode (see the veb_create command).

Use either of the following tools to tune VEB models:

  • CM1 Center is accessed inside Calibre WORKbench, using the Litho > CM1 Center menu item. The VEB options are accessed using a tab.

  • The modelflow_v2 batch command can be accessed as part of a Calibre batch command file or from the Calibre WORKbench Tcl command shell.
