Calibrating for Variable Etch Bias Effects

Follow this procedure to further refine your CM1 model for variable etch bias (VEB) effects.

Prerequisites

  • The fine CM1 calibration log from completing the procedure, “Performing Fine CM1 Calibration”.

  • The corresponding gauge file created in “Creating a Gauge File for Variable Etch Bias (VEB)”.

Procedure

  1. In CM1 Center (Litho > CM1 Center), load the gauge file you created in “Creating a Gauge File for Variable Etch Bias (VEB).” (Gauges > Open menu item).
  2. Switch to the VEB tab (Figure 1). If you already have a VEB model, load it using VEB > Open. If you do not have a VEB model, you can create a default VEB model using VEB > Create Default or VEB > Model Form and adjust the values as needed.

    • To switch the default kernel type, type in the field containing gauss; accepted values are gauss or visible.

    • To select an internal visible kernel, change the Type field to “visible” and the looking field to “in”.

    • Add coefficient terms on the right side (BTERM) with the Add and Del buttons (A), adjusting the coefficient terms and expression fields as needed. The Expression field (B) is a text field that allows you to specify linear and non-linear terms (see “Using VEB Non-Linear Terms” for more information).

    • Additional VEB options can be specified in the VEB model text field (C).

      Figure 1. VEB Tab

  3. Switch to the Optimize tab. Click on the VEB tab at the bottom of the pane to raise it.
    Figure 2. CM1 Center, Optimize Tab, VEB Tab Mode

  4. Select the (O)ptimize run mode.
  5. Set the Fit field to either relative or absolute to select the mode.

    • relative means that the etch model parameters are determined by minimizing the difference between simulated and measured etch bias. The latter is determined as the difference between etch and resist CD: (CDetch – CDresist).

    • absolute means that the etch model is fit by minimizing the difference between simulated and measured etch CD data; in other words, the model is fit directly to CD data rather than to bias data.

  6. Modify the setup file to include the vetchbias keyword, setting it to the name of your VEB model.
  7. Click Run to search for VEB values that fit the given criteria.

    Similar to CM1 models, the lower and upper bounds found during the search are shown in the Optimize tab (for the Front, Full and Gradient search methods).

  8. To view the results of the optimization, select the new result log at the bottom of the window and click the Analyze button.

    • Expand the VEB tree and browse the sections (CD Fitting Section, CD Uniformity Section, and CD Linearity Section).

    • To view the search criteria as a bivariate plot, select the VEB Search tab at the top of the Analysis log.

  9. Click Accept: VEB when the run completes; you should also save the session file (File > Save Session) and the VEB model (VEB > Save menu item).
