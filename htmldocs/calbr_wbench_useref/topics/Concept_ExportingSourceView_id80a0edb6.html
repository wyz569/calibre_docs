Exporting the Source View

You can use the File > Export menu items to write the source file to the disk.

MathWorks MATLAB Format (Un-Polarized Source Only)

The Source View Tool lets you export the source data for single source optical models in MathWorks MATLAB format.

Once in MathWorks MATLAB format, you can view the source data as a contour plot or other format, as desired.

  1. Open the source map in Optics > Source View.

  2. Click File > Export to Matlab and save it in the MathWorks MATLAB file matlab.map.

  3. Switch to MathWorks MATLAB.

  4. Load the file. 

    s = load('matlab.map');

  5. Once the data is loaded into MathWorks MATLAB, you can use the standard MathWorks MATLAB commands. For example:

    • To see contours:

      contour(s);

    • To see a 3D surface:

      mesh(s);

    • To see the cutline through line x = 0:

      plot(s(:,50));

      where 50 is the index of the middle of the matrix.

Comma-Separated Value (CSV) Format

The Source View Tool also exports source data for optical models in a simple comma-separated value (CSV) format that can be opened in any spreadsheet program.

Each line of the CSV file contains three values, representing the X and Y coordinates and the source intensity value. Each value is immediately followed by a comma and no whitespace is allowed in the line.

The CSV file format contains all source points, including points with zero intensity, starting at (-limit,-limit) and ending at (+limit,+limit). The value of limit is typically 1.0. The X coordinate is the first varying coordinate. For example, after point (0.0, 0.0) is the point (0.01, 0.0).

  1. Open the source map in Optics > Source View.

  2. Click File > Export to CSV and save it to filename.csv.
