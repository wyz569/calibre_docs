Lithomodel Creation Wizard User’s Guide

The Lithomodel Creation Wizard is intended to ease the creation of litho models for Calibre tools that use them. Litho models are composed of existing optical, resist, and other related models that are in a central directory location.
Note:

Litho models can also be created using the Calibre nmModelflow wizard. See the Calibre nmModelflow User’s and Reference Manual for more information. The Calibre WORKbench wizard is intended for users who do not plan to use Calibre nmModelflow, and the wizards have some differences.
