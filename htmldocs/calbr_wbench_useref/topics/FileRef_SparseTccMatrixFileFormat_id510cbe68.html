Sparse TCC Matrix File Format

Optical Model File Formats

Input for: comparetccs and writetcctxt

Contains a description of a TCC matrix associated with an optical model.

Format

  • ASCII text file.

  • File extension .txt.

    [number of TCCs] 
i j m n real imag 
...

Parameters

number of TCCs

An optional line that contains the total number of TCC values that are given in the file (number of lines that follow).

i j m n real imag

Each subsequent line contains:

  • Four integer numbers that indicate the position in the matrix of the TCC value.

    • The first pair of indices (i,j) indicate the center of the pupil.

    • The second pair of indices (m,n) indicate the center of the conjugate pupil.

    • The i and m indices refer to shifts along the x-axis and can vary between ‑Nx*xord to +Nx*xord.

    • The j and n indices refer to shifts along the y-axis and can vary between ‑Ny*yord to +Ny*yord.

    By definition, x-ord = int(1/Lx * lambda/NA) and y-ord = int(1/Ly * lambda/NA), where Lx, Ly are the dimensions of the optical simulation window. The values for Nx and Ny depend on the source partial coherence. Note that the values of xord, yord, and the maximum coherence of the source along the x and y axes cannot be deduced from a sparse TCC file.

  • The fifth and the sixth values are the real and imaginary parts of the TCC value.

Examples

An example of a file that contains 63101 TCC elements. In this example, the four-dimensional TCC matrix has a total of 19^4 = 130321 elements. The first five lines are shown.
