sgd_get_prediction_error_stat

CM1 and VEB Commands

Reports the printing prediction error statistics for the input super gauge data object.

Usage

sgd_get_prediction_error_stat object [–vebfit [absolute]]

Arguments

object

A required argument that specifies the input super gauge data Tcl object.

-vebfit [absolute]

An optional keyword that tells the command to compute error statistics for etch data instead of for resist data. The optional flag “absolute” requests computation of the etch error stat for CDs.

Description

This command checks if the simulated CD meets the less than (LT:) or greater than (GT:) constraint for each gauge, and reports the printing prediction errors for the input super gauge data.

Examples

The following example code creates an object and then reports the resist and etch error statistics.

set obj [sgd_create –in sample.sdg]
set resist [sgd_get_prediction_error_stat $obj]
set etch [sgd_get_prediction_error_stat $obj –vebfit absolute]
