PSM Test Pattern Descriptions

PSM test patterns serve two purposes:

  • They provide information needed to create a phase-aware resist model.

  • They help you determine the capabilities of your mask-making and wafer printing processes with respect to phase shifting masks.

A phase-aware resist model is a resist model capable of simulating the affects of mask containing some binary regions and some PSM regions. You create this model using the same basic steps as you would use to create a standard resist model. You have the choice as to how you want to create your test pattern:

  • Create a single PSMext test pattern. This test pattern, and the spreadsheet associated with it, contain both the PSM and binary test structures.

  • Create two separate test patterns, one for PSM and one for the binary mask. When using this process you must do the following:

    1. Place both test patterns on the same test chip.

    2. After entering measurement data into the two spreadsheets, concatenate the two spreadsheets into a single file, which you use as input for VT5 Center.

As you take measurements from a PSM test pattern, enter them into the test pattern spreadsheet.

Isolated Line Width Row

Row 1

Spreadsheet file name:

psm_iso_width

The width of the phase regions varies. You can use this to determine the minimum necessary phase width for your process.

  • Col A:

    width = CD(w)-10

  • Col B: width = CD(w)

  • Col C:

    width = CD(w)+10

  • Col D:

    width = CD(w)+20

  • Col E-L: exponential increases up to 1200.

Isolated Line Height Row

Row 2

Spreadsheet file name:

psm_iso_height

The height of the phase regions is varied to help you determine minimum acceptable aspect ratio.

  • Col A: height = 4*CD(w)

  • Col B:

    height = 4CD(w)+40

  • Col C:

    height = 4CD(w)+80

  • Col D:

    height = 4CD(w)+120

  • Col E-L: exponential increases up to 4800.

Varied Gate Undersizing

Row 3

Spreadsheet file name:

psm_iso_sep

The separation between phase areas in these structures is progressively undersized. You use this structure to determine the behavior of the narrow channels and the proximity effects at the channel edges.

  • Col A: sep = CD(w)

  • Col B: sep = CD(w)-10

  • Col C: sep = 4CD(w)-20

  • Col D: sep = 4CD(w)-30

  • Col E-L: decreases to 20.

Row 4

Spreadsheet file name:

psm_iso_trans

The target layer width varies. You use this structure to examine the effect of varying gate-to-field width transition size.

  • Col A: target = CD(w)-10

  • Col B: target = CD(w)

  • Col C: target = CD(w)+10

  • Col D: target = CD(w)+20

  • Col E-L: increases up 2*CD(w).

Chromeless Dots, Lines and Polyps

These structures allow you to print pure phase edges. You can use these structures to investigate the ability to print phase edges of various designs. However, these structures should be disabled when building your model, as they do not have an associated target layer.

Varied Dense Line Pitch

Rows 9-16

Spreadsheet file name:

psm_pitch

The pitch varies.

Use these structures to:

  • Investigate iso-dense bias effects for PSM structures.

  • Determine the PSM pitch curve which is useful for performing OPC.

  • Determine the phase depth variation effects due to the etch effect called Aspect Ratio Dependent Etch (ARDE) or inverse ARDE.

Diagonal

These structures allow you to examine diagonal interactions between opposite phases. These structures should be disabled when building your model, as they do not have a single measurement point.
