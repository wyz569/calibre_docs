applyTCCsymmetry

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Applies symmetry to the TCC matrix.

Usage

applyTCCsymmetry { xy | xydiag | none }

Arguments

xy

An optional argument that averages the TCC values from positions (i,j,m,n), (-i,j,-m,n), (i,‑j,m,‑n) and (‑i,‑j,‑m,‑n), resulting in a model that exhibits identical symmetry with a model resulting from an x-dipole source or a y-dipole source.

xydiag

An optional argument that averages the TCC values from positions (i,j,m,n), (-i,j,-m,n), (i,‑j,m,‑n), (‑i,‑j,‑m,‑n), (j,i,n,m), (-j,i,-n,m), (j,-i,n,-m), and (-j,-i,-n,- m), resulting in a model that exhibits both X and Y reflection and four-fold rotation symmetry. Examples of sources with four-fold rotation symmetry are the following:

  • a standard source, which has a circular shape

  • an annular source

  • a QUASAR source

  • a QUAD source with 0 or 45 degree rotation

none

An optional argument that uses the original TCC matrix. This is the default behavior.

Description

An optional miscellaneous parameter that averages in all necessary respective TCC matrix combinations to enforce a specific model symmetry. This parameter affects the TCC matrix immediately after it is computed.

Note:

This parameter is incompatible with the hha argument. If hha is also specified, applyTCCsymmetry is ignored with a warning. The symmetry of the resulting optical model is dictated by the input optical parameters (such as the source map or Jones pupil file).

This option is a way to enforce a desired model symmetry in an optical model.

For example, if a measured source map is not symmetric because of measurement noise, the resulting optical model will also not be symmetric. There are two options to enforce model symmetry in this situation:

  1. Use the mergemap command to covert the measured sourcemap into a symmetric sourcemap.

  2. Use applyTCCsymmetry to enforce the desired symmetry on the output TCC matrix.

Both options will produce identical TCC matrices (to within numerical noise), and therefore also models with the specified symmetry.
