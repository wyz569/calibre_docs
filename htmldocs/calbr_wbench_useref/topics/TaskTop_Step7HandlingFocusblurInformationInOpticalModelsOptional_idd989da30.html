Step 7 - Handling Focusblur Information In Optical Models (Optional)

Focusblur modeling in Calibre WORKbench can be optionally modeled for use with CM1 models in the Optical Model Tool or by editing the focusblur keyword parameters in the optical model file. 

focusblur [lorentz gamma n] [gauss sigma] [rectangle width] 
[ssfile scanner_slit_file zrange low_val high_val]
[lsfile laser_spectrum-file lca lca_per_pm] 
[integorder num] [filterval fval]

Currently, the focusblur parameters can handle the following focus error types:

  • Focus errors because of chromatic errors of the laser source (finite source bandwidth)

    Uses the lorentz gamma n component. You can alternatively supply a laser spectrum measurement file (lsfile argument) that also contains the corresponding longitudinal chromatic aberration (lca) parameter in microns.

  • Focus errors because of mechanical vibrations of the wafer stage along the optical axis

    Uses the gauss sigma component.

  • Focus errors because of systematic (purposeful) vertical motion or tilt of the wafer stage during the scan.

    Uses the rectangle width and integorder num component. The user can also supply a file (ssfile argument) that contains measurements of the maximum source intensity within the slit and the corresponding range of focus in microns during the scan.

To add focusblur information to an optical model file, add the focusblur keyword as shown above.

To add focusblur information using the Optical Model tool, use the following procedure.

Prerequisites

  • Focusblur information from the scanner exposure tool

Procedure

  1. In the Optical Model tool, Kernels/Image tab, select the Focus Blur tab. It is in the Image panel on the right, about halfway down.
  2. Enter relevant values for the focusblur equation in the fields shown in Figure 1.
