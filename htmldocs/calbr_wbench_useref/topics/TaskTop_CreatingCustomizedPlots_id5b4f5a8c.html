Creating Customized Plots

In addition to the standard Bivariate and Checkerboard plots, you can create a custom plot by using the Customized Plots tab.

Procedure

  1. Click the Customized Plots tab.
  2. Select comparisons to be plotted by choosing an item from the Vs. dropdown lists.
    Tip

    Additional choices for density kernels are made available if your resist or VEB model contains them.

  3. Click Add.

Results

The selected plot pairing appears in the lower half of the window (4).

Figure 1. Customized Plots Tab

Tip

Similar to the Bivariate Plot tab, the Select Plots tab allows you to select one or more points and then click the Mark button. Points selected from the Customized Plots tab appear on layer 102 by default.
