cm1copy

CM1 and VEB Commands

Related Formats: CM1 Model File Format

Copies CM1 objects.

Usage

cm1copy cm1out cm1in

Arguments

cm1in

A required argument that specifies the input cm1 Tcl object.

cm1out

A required argument that specifies the output cm1 Tcl object.

Description

Use the cm1copy command in the Calibre WORKbench Tcl shell to copy one existing CM1 object (cm1in) to another (cm1out). The CM1 object initially represented by cm1out is deleted, and cm1out becomes a handle to a CM1 object identical to cm1in. Both cm1out and cm1in must be valid handles to CM1 objects, otherwise an error occurs and the command aborts.

Note:

You must pass cm1copy the actual objects (cm1obj1 and cm1obj2) rather than the Tcl variables $cm1obj1 and $cm1obj2; attempting to pass variables results in an error.

Examples

set cm1obj1 [cm1load cm1.mod]
set cm1obj2 [cm1compose –m 3]
# copy cm1obj2 to cm1obj1
cm1copy cm1obj1 cm1obj2
