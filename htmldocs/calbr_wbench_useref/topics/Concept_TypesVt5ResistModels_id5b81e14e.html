Types of VT5 Resist Models

The Calibre WORKbench application and the RET toolset support four types of sparse resist models:
Table 1. Types of Sparse Resist Models

Model Type

Description

Constant Threshold Resist Model (CTR)

Calculates the wafer print image based on the intensity of the aerial image only.

Variable Threshold Resist Model (VTR)

Calculates the wafer print image as a function of intensity and slope (the aerial image intensity plus how rapidly the intensity is changing).

Variable Threshold Resist Model Extended (VTR-E)

Calculates the wafer print image as a function of its aerial image intensity, slope, and aerial image factor.

VT5 Model (VT5)

Calculates the wafer print image as a function of four image parameters: Imax, Imin, slope, and factor, plus one or more convolution kernels.

For a description of the differences between VT5 model parameters and parameters for other types of models, refer to “Sample Data File”.
Tip

See Also CM1 / VEB Models — The VT5 models described in this chapter are based on simulation control sites and are used with Calibre OPCpro. The Siemens EDA CM1 and VEB models provide a dense simulation on a grid, but are used exclusively with Calibre nmOPC and Calibre OPCverify.

For more information on developing CM1 and VEB models, see “CM1 and VEB Model Development Tasks”.

For CTR, VTR, and VTR-E models, the model type defines which image parameters impact the model, the order of the model polynomials, and so on. The model types are selectable using buttons in the VT5 Center Resist tab.

For VT5 models, you design your own model, composed of one or two polynomials. You have control over the order, types of inputs, and use of convolutions.

If you use the CTR, VTR, and VTR-E model types, it is recommended that you emulate them with a VT5 model instead, because VT5 models calculate the aerial image parameters more precisely.
