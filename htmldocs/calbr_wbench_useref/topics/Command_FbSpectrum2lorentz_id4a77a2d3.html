fb_spectrum2lorentz

Optical Model Commands

Transforms the laser spectrum information in a file to its equivalent Lorentz parameters gamma (Γ) and n.

Usage

fb_spectrum2lorentz -lsfile laser_spectrum_file_name -lca lca_value [-lambda wavelength_in_nm] [-debug]

Arguments

-lsfile filename

A required argument specifying the input laser spectrum file.

-lca lca_value

A required argument specifying the longitudinal chromatic aberration (lca) parameter from the scanner, in um of defocus per pm of laser offset.

-lambda wavelength_in_um

An optional argument that sets the central wavelength in um. If this option is not specified, the command assigns the central wavelength based on the highest maximum intensity in the file. The wavelength numbers in the input file are not required to be symmetrical, but at least one wavelength number must have an intensity no larger than half of the maximum.

-debug

An optional argument that writes the result of the command to the console and the fitting details to the file fb_spectrum2lorentz.txt. The console results look similar to the following:

focusblur lorentz 0.150111 2.802075

The information written to the output file looks similar to the following:
