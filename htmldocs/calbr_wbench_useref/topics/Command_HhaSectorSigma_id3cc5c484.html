hha_sector sigma

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Define an annular geometrical HHA sector.

Usage

hha_sector sigma sigma [sigma_in si illumangle ia rotangle r]

Arguments

These arguments are similar to the source definition of an ASML_POLE. If any of the optional arguments are specified, then all of them must be specified.

sigma

A required argument specifying a circle of radius sigma in the pupil plane.

sigma_in si

An optional argument defining a sector bounded by an outer circle of radius sigma and an inner circle of radius si in the pupil plane.

illumangle ia

An optional argument defining a sector bounded by an outer circle of radius sigma and an inner circle of radius si in the pupil plane.

rotangle r

An optional argument defining a sector center about the angle r.

Description

An optional parameter set that is only used when “hha_sectorization manual” is also specified. It allows you to define an annular geometrical HHA sector.

  • Specifying only the value of sigma defines a circle of radius sigma in the pupil plane.

  • Adding the optional sigma_in, illumangle, and rotangle arguments defines a sector bounded in sigma space by an outer sigma and an inner si, with an illumination angle of ia centered about the angle r. These parameters are similar to the source definition of an ASML_POLE. If any of the optional parameters are specified, then all of them must be specified.

Multiple hha_sector statements are allowed.
