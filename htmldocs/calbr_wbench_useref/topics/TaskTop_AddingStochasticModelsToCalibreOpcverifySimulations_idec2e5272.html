Adding Stochastic Models to Calibre OPCverify Simulations

Stochastic modeling adds functionality to Calibre® OPCverify™

Prerequisites

  • The calibrated stochastic model file from “Simulating EUV Including Stochastic Models”

Procedure

  1. In a text editor, create a Calibre OPCverify command file with the following characteristics:

    • A standard SVRF header with a description of the input design file.

    LAYOUT PATH  "input.oas"
LAYOUT SYSTEM  OASIS
LAYOUT PRIMARY  "*"
LAYOUT INPUT EXCEPTION SEVERITY PRECISION_RULE_FILE 0
LAYOUT MAGNIFY AUTO
LAYOUT USE DATABASE PRECISION YES
DRC RESULTS DATABASE "output.oas" PSEUDO OASIS 
DRC MAXIMUM RESULTS ALL
LAYER m1 0

    • SVRF derived layer calls to Calibre OPCverify with a related mapping to output layers. 

    nom = LITHO EUV OPCVERIFY  m1 FILE setup_in   MAP nom
low = LITHO EUV OPCVERIFY  m1 FILE setup_in   MAP low
high = LITHO EUV OPCVERIFY  m1 FILE setup_in   MAP high
band = LITHO EUV OPCVERIFY  m1 FILE setup_in   MAP band
m1     { copy m1  }     DRC CHECK MAP m1        OASIS   1 0
nom    { copy nom }     DRC CHECK MAP nom         OASIS   2 0
low    { copy low }     DRC CHECK MAP low         OASIS   3 0
high   { copy high}     DRC CHECK MAP high        OASIS   4 0
band   { copy band}     DRC CHECK MAP band        OASIS   5 0

    • A Calibre OPCverify command script, enclosed by a LITHO FILE declaration, consisting of the following parts:

      • Commands to set system parameters:

        LITHO FILE setup_in [/*
modelpath .
layer m1
flare_longrange LITHOMODEL constant 0.02
processing_mode flat
tilemicrons 25
simulation_consistency 2
rasterizer_upsample_factor 2 1

      • Commands designating an image options block that defines the visible layers and litho model to use:

        image_options opt1 {
 layer m1 visible
 litho_model LITHOMODEL
}

      • Commands that create image contours using the stochastic option. These commands calculate the stochastic low and high bands, and then combine them as a process variation output with the pvband command:

        setlayer  nom     = image opt1
setlayer  low     = image opt1 stochastic low
setlayer  high    = image opt1 stochastic high
setlayer  band    = pvband low high
*/]
  2. Save and run the file:

    calibre -drc -hier -turbo -turbo_litho filename.svrf

Results

The output OASIS file contains the visual representation of the stochastic contour based on the stochastic compact model.
