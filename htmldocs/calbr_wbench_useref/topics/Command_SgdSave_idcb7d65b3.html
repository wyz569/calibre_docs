sgd_save

CM1 and VEB Commands

Related Formats: Super Gauge Data File Format

Saves the contents of the specified sgd Tcl object to the specified file.

Usage

sgd_save object –out filename

Arguments

object

A required argument that specifies the input sgd Tcl object.

–out filename

A required argument that specifies the output file name.

Examples

The following example code creates an super gauge data object (sgd_create), adds a gauge object to it (sgd_add) and saves the modified super gage data object to the file sample_new.sgd. 

set obj [sgd_create -in sample.sgd]
sgd_add $obj -gf “/tmp/sample.gg” -dose 1.04 -defocus 0.1 -weight 1.0
sgd_save $obj -out “/tmp/sample_new.sgd”
