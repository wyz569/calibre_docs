Return Values

After execution, the modelflow_v2 command returns to the Calibre WORKbench Tcl shell the final fitness statistics.

–ctrrms val –ctrmean val –resistrms val –resistmean val -resistdiff val –vebrms val ‑vebmean val

where:

  • -ctrrms val — CTR total RMS error in nm

  • -ctrmean val — CTR total mean error in nm

  • -resistrms val — resist total RMS error in nm

  • -resistmean val — resist total mean error in nm

  • -resistdiff val — percentage difference between resist and aerial image surfaces (named cm1diff in versions prior to 2008.4)

  • -vebrms val — etch total RMS error in nm

  • -vebmean val — etch total mean error in nm

If one of the optimization stages is not specified, the return values for the associated errors will be set to -1.

The best setup configuration (including the final optimized optical, resist, and etch models) will be retained after the modelflow_v2 optimization run completes. If stage1 is set and stage2 is off, the final CTR model is kept, but the nominal resist model will be deactivated.

If an error occurs, however, the nominal models will be kept.
