Sample Spreadsheet Files Overview

Test patterns have one or more spreadsheets associated with them to capture the data. The Siemens EDA test pattern generator facilities usually create a single spreadsheet to accompany a test pattern design file, which you use when entering the measurements taken on the chip post-process.

At a minimum, the modeling tools require one spreadsheet with completed data values to create a model from; these values are considered as a nominal centered dose and focus. When multiple spreadsheets are supplied for the same test pattern at different dose and defocus values, this “super spreadsheet file” is used by the modeling tools to create a process window model that can account for varying process conditions.

Gauge Files Overview

Gauge files are the CM1 modeling equivalent of sample spreadsheet files. They use gauge objects applied to test structures instead of the sample sites used in VT5 and earlier models. You can convert a sample spreadsheet file into a gauge file in the CM1 Center; for more information, see “Converting Sample Spreadsheets to Gauge Files”.
