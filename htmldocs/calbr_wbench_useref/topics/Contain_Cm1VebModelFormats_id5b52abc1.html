CM1 and VEB Model Formats

CM1 and VEB models use multiple structured formats to organize data.
Table 1. CM1 and VEB Model Formats

Format

Description
CM1 Model File FormatDefines a resist model file.
Custom Kernel File FormatDefines a custom kernel file.
List of ModelformsDescribes the set of model form templates used by modelflow_v2 and the CM1 model.
Model-Layer File FormatDefines a simpler setup file (containing just a model path and layer definition).
Mask Model FormatDefines a mask model.
Gauge Object ReferenceDescribes the gauge object, which is used by Calibre WORKbench to interact with CM1 and VEB objects.
Super Gauge Data File FormatDefines a super gauge file, which is a group of multiple gauge objects in a single file.
Variable Etch Bias (VEB) Model File FormatDefines a VEB model.
Group File FormatDefines an auxiliary input file that groups gauge objects together for more precise user control over gauge tolerances, shifts, skew factors, and weights.
