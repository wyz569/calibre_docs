stack_weight

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Sets weights of stack films to control the way the total pupil function of the optical system is calculated for topographical modeling.

Usage

stack_weight u
stack2_weight v

Arguments

u v

A required argument specifying weight values applied to the pupil function P of the corresponding film. The default value of the first film weight u is 1; the default weight value for additional films is 0.

Description

These optional parameters are used with the film2 (second stack) and film3 (third stack) parameters (see the film parameter description), and control the way the total pupil function of the optical system is calculated for topographical modeling.

  • If only film and film2 are specified, stack_weight sets the weight for the main stack, and the total pupil function P is the sum of the main stack (film, designated P1) and the second stack (film2, designated P2):

    P = uP1 + (1 - u)P2

    If stack_weight is not specified, u is assumed to be 1, which causes the second stack to have a weight of zero.

  • If film, film2, and film3 are specified, stack_weight and stack_weight2 should also be specified, and the total pupil function is the sum of all three stacks:

    P = uP1 + vP2 + (1 - u - v)P3

    If film3 is specified and stack2_weight is not specified, the weight of the second stack (v) is assumed to be zero.
