Generating the Super Spreadsheet Log

Before you can analyze the super spreadsheet information, you must generate a modelflow batch or VT5 Center log for it.

Prerequisites

  • You should be familiar with the use of VT5 Center, as described in “Model Optimization With the VT5 Center GUI”.

Procedure

  1. Run the test pattern through your lithography process at different dose and defocus conditions, and modify each instance of the sample spreadsheet with the measurement values from that dose and focus condition. Name the spreadsheets appropriately with the dose and focus values used to produce it to avoid confusion.
  2. Follow the instructions in “Creating a Super Spreadsheet” to create a super spreadsheet file.
  3. Open the VT5 Center tool (Litho > VT5 Center). In the Basic tab, type the log file name.
  4. Switch to the Sample tab. Open the super spreadsheet file by using the Sample > Open SSS menu item.
  5. Set up the other parameters in the VT5 Center tool as necessary.

    • Optics, in tab — You should use the optical model from the best dose and focus by default.

    • VT5, in tab — At least a nominal VT5 model should be used; however, an optimized VT5 model created using VT5 Center on the sample data from the best dose and focus optical model may be more effective.

  6. Run VT5 Center, which saves the result in the filename.log file and adds an entry to the bottom of the window.
  7. Proceed to the section “Using the SSS Process Window Analysis Tool”.
