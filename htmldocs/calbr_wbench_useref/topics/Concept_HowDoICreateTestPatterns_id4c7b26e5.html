Use the table to learn the flows and information for using test pattern tools in Calibre WORKbench.
Table 1. Calibre Test Pattern Generation at a Glance

Objective

Create simple test chip designs for use in model calibration. This is done with the Test Pattern tool, but can also be performed using a test pattern creation file, or by placing and exporting markers from the Calibre WORKbench GUI.

Test Pattern Inputs

Task Flow Summary

  • Pertinent information for design (Test Pattern tool)

or

  • Test Pattern cmd file (test pattern batch commands)

  1. In Calibre WORKbench, open the Test Pattern tool (V11/12).

  2. Fill out relevant data and generate the test pattern.

or

  1. Create a text file using the Test Pattern File Format syntax.

  2. Invoke Calibre WORKbench.

  3. Use the file as input to the tepCompile batch command.

Test Pattern Outputs

What to Look for in Results

  • Design file

  • Spreadsheet file

The empty column for Measurement must be filled in according to the results of your lithographic measurements.

Test Pattern Commands

Description

FEATURE Keyword

Defines a header section for a test pattern.

PATTERN Keyword

Defines a test pattern structure.

cdruler

Returns measurement data for each of the sample points in the specified spreadsheet file.

etchtp

Generates a version 10 etch test pattern and associated spreadsheet.

maitgen

Assembles a test pattern from multiple test patterns.

sss_add

Adds a spreadsheet object to a super spreadsheet object.

sss_cget

Retrieves information about a spreadsheet in a super spreadsheet object.

sss_config

Updates specifications for a spreadsheet inside a super spreadsheet object.

sss_create

Creates a super spreadsheet object.

sss_delete

Deletes a spreadsheet within a super spreadsheet object.

sss_save

Saves a super spreadsheet object.

tepCompile

Generates a custom test pattern based on specifications in a TPF file.

tepCreate

Generates a version 11/12 test pattern and associated spreadsheet.

tepCreate etch

Generates a version 11 etch test pattern and associated spreadsheet.

tepScal

Generates a SEM Calibration test pattern and associated spreadsheet.

tepVerification

Generates a verification test pattern and associated spreadsheet.

tpcreate

Generates a version 10 test pattern and associated spreadsheet.
