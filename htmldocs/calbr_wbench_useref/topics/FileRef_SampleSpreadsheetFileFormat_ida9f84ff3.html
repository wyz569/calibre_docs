Sample Spreadsheet File Format

CM1 and VEB Model Formats

The sample spreadsheet file contains tabular data for points in the associated test pattern. It is the primary input for sparse (VT5) modeling.

For dense modeling, the sample spreadsheet file format may be used as an alternate input source for CM1 and VEB models. Input sample spreadsheets are converted into gauge objects.

Contents

The sample spreadsheet file consists of tabular data, with each line in the file representing one row of data for a sampling point. Each row of spreadsheet data is separated from the next by a newline (return) character.

Comment lines are allowed, and must begin with the comment character ‘#’. Comment lines can be either unformatted text or a row of spreadsheet data that is currently disabled.

Each non-comment row has the following format:

name

row

col

X

Y

loc

target

other

meas

weight

Parameters

name

The name of the test structure (feature type). This can be any of the Siemens EDA structure types described in “Feature Types and Associated Subkeywords”. It can also be a user defined name for the test structure, which is an alphanumeric string that must contain at least one letter (but does not have to begin with a letter). The name may also contain underscores.

row

The row in which the test structure occurs in the test pattern (not the spreadsheet row). This is used by the VT5 Center Tool to plot data sequences.

col

The column in which the test structure occurs in the test pattern. This is used by the litho modeling Tool to plot data sequences within a given row.

X

The x-coordinate of the sample point, represented in database units.

Y

The y-coordinate of the sample point, represented in database units.

loc

The sampling location. If this is 0, it indicates the measurement is to the inside of the polygon edge nearest the (x,y) coordinate. If this is 1, it indicates the measurement is to the outside of the polygon edge nearest the (x,y) coordinate.

target

The drawn dimension on the mask, at wafer scale. Displayed in VT5 Center as the DrawnMask field.

other

A structure-specific value. For pitch lines, for example, this contains the space, so that (target + other) = pitch.

meas

The CD measurement for this structure.

weight

The relative weight of this structure for the purposes of model-fitting. Cannot be negative.
