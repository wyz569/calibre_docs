veb_setopt

CM1 and VEB Commands

Related Formats: Variable Etch Bias (VEB) Model File Format

Sets the global optParams etch model to the specified VEB model.

Usage

veb_setopt vebobj

Arguments

vebobj

A required argument that specifies the input VEB Tcl object.

Description

This command clones the input VEB object, sets the optParams model to this new (cloned) VEB model, and deletes the old optParams etch model if it was set. Cloning is done to prevent the new optParams etch model from being deallocated, which automatically happens in Tcl when out of scope where the input VEB object was specified.

Examples

#create veb object and writes it to string
set obj [veb_create –file “etch.mod”] 
veb_setopt $obj
