sgd_create_group_specs

CM1 and VEB Commands

Generates default group specs.

Usage

sgd_create_group_specs object [-gsfile file [-mode {w | a}]]

Arguments

object

A required argument specifying an input super gage data Tcl object.

-gsfile file

An optional keyword that specifies the input group definition file.

-mode {w | a}

An optional flag that specifies how the output file is opened. Accepted keywords are:

  • w – Create a new file or overwrite the existing one (Default).

  • a – Create a new file or open the existing one for appending at end-of-file.

Description

Generates default group specifications. This command extracts group names from the input super gauge data object, creates a group specification string with default values, and writes the string to the output file if -gsfile is specified. This command can be used only if all enabled gauges are split into groups, which requires that all values in the Group column cannot be ‘NA’ or ‘-1’.

Examples

set obj [sgd_create –in sample.sgd]
set group_spec [sgd_create_group_specs $obj -gsfile group.gsf]
