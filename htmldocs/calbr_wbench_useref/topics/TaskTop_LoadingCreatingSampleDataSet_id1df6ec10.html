Loading or Creating a Sample Data Set

This procedure describes the methods to supply one or more sample spreadsheet data sets to the VT5 Center tool, Sample tab. The modelflow batch command uses this data to fit the optical and resist models versus the simulated data during calibration runs.

Figure 1. VT5 Center GUI (Sample Tab)

Prerequisites

  • At least one sample data spreadsheet (.ss file) is required. It should contain the nominal dose and defocus conditions.

    Attempting to run VT5 Center without a sample data set present in this tab stops the run with an error.

    Tip

    Additional sample data spreadsheets at different dose and defocus conditions with corresponding measurements can be added in the Sample tab to form a super spreadsheet describing a process window.

  • The corresponding test pattern design file loaded in Calibre WORKbench

Procedure

  1. In VT5 Center, click the Sample tab.
  2. Use one of the following VT5 Center Sample menu items to add the first sample data spreadsheet file to the Sample tab.

    • Open SS File — Opens a previously saved sample data spreadsheet file.

    • Append New SS — Adds a new (blank) sample spreadsheet file.

      If you use this option, you must manually add the test structures present in the Calibre WORKbench layout file and the measurement data (see Table 1).

    • Open SSS — Opens a super spreadsheet file.

    • Extract SSS from Log — Uses a modelflow log file as input.

    If this is the only sample data you have available, skip to step 5.

  3. To include additional sample data spreadsheets, use one of the following Sample menu items:

    • Append SS File — Adds another saved sample data spreadsheet file in a new tab.

    • Append New SS — Adds a new (blank) sample spreadsheet to the tab list.

    Switch to the new tab and update its dose and defocus values. Add a weight value if needed to give certain sample spreadsheets different weight than others.

    Tip

    If you have more than 32 sample tabs, hold down and drag the middle mouse button, then move the mouse left or right to scroll the visible tab set.

    You can also choose the Sample > Edit FEM menu item to view and edit focus exposure model conditions for multiple spreadsheets. This raises a dialog box (Figure 2) with all current sample spreadsheets and their corresponding dose and defocus values. (Click the See button to raise a specific spreadsheet tab).

    Figure 2. Edit SSS FEM Dialog Box

  4. If you made changes or additions to the sample data set, save the modified sample set by using the Sample > Save as SS File or Sample > Save SSS As menu items to save the sample data spreadsheet or super spreadsheet file, respectively.
  5. Optionally, select the Sample > Clip Layout option to clip the layout based on the active sample structures that have measurements.
  6. Proceed to “Including Contour Calibration Information”.

Results

VT5 Center allows editing of the sample data on a per structure basis. Controls are shown in Table 1.

Table 1. Sample Data Spreadsheet Editing Controls in VT5 Center

Control

Description

Notes

View

Jumps the Calibre WORKbench viewer to the selected test structure.

You may need to zoom in on the structure in the Calibre WORKbench window to see the structure clearly.

Mark and Unmark

Adds or removes visible markers on selected test structures.

—

Select

Selects one or more structures based on specified criteria.

Use the arrow button to the right of the Select button to toggle Basic (select by structure name) or Advanced (multiple selection criteria; see “Advanced Sample Selector Dialog Box”) selection mode.

Enable

Re-enables disabled structures.

—

Disable

Disables marked structures.

Disabled structures are not used in calibration tests.

Add

Adds a new line to the currently active tab.

The structure definitions are set to default values; you must change them in order for the new structure line to be relevant.

Delete

Deletes selected structure(s).

The structures are deleted from the open copy of the sample spreadsheet; deletions are not saved unless the spreadsheet is saved.

Edit (dropdown)

Contains multiple editing controls.

—

  • Undo / Redo

Undoes or redoes the last action from the Edit menu or the Edit SSS FEM dialog box.

—

  • Set Weight

Sets the weight factor of marked lines.

Larger weight values indicate significance.

  • Set Value

Sets the value of highlighted fields to a value entered in a popup dialog box.

Most effective when a column is highlighted.

  • Copy DrawnMask to Measurement

Sets the Measurement field to be equal to the DrawnMask field, which represents the drawn target value.

—

  • Assign Simulation to Measurement

Sets the Measurement field to be equal to simulated results.

Using this option can cause overfitting.

  • Apply operations to all SS pages

Treats a super spreadsheet as if each corresponding cell were linked, such that an operation is carried across all spreadsheet tabs.

—
