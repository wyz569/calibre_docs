sgd_config

CM1 and VEB Commands

Related Formats: Gauge Object Reference, Super Gauge Data File Format

Sets parameters for a gauge object inside a super gauge data object.

Usage

sgd_config object gaugeSet_index [‑gf filename | ‑dose value | ‑defocus value | ‑weight value]
[‑type value] [‑plane value] [-opticaldir dir1][‑opticaldir2 dir2][‑opticaldir3 dir3]

Arguments

object

A required argument that specifies the input sgd object.

gaugeSet_index

A required argument that specifies an index in the gaugedata set within the sgd object. Accepted values for gaugeSet_index are the following:

  • integer — Resets the specified parameters of the gauge set determined by this index number, which must satisfy the inequality:

    0 < gaugeSet_index <= number_of_gaugefiles.

  • end — Resets the specified parameters of the last gaugedata set.

  • all — Resets the specified parameters in all gaugedata sets.

‑gf filename

An optional argument that specifies the input gaugedata file name. Specifying an empty string disables the field’s current setting.

‑dose value

An optional argument that specifies the input dose value as a dimensionless positive floating point number.

‑defocus value

An optional argument that specifies the input defocus value in microns (um).

‑weight value

An optional argument that specifies the input weight value as a dimensionless positive floating point number.

‑type value

An optional argument that specifies the gauge type. For version 1 supergauge objects, use CD. For version 2 supergauge objects, use either CD or SRAF.

‑plane value

An optional argument that specifies the metrology plane value in microns.

-opticaldir dir1

An optional argument that specifies the optical directory to read in optical information from. Specifying an empty string disables the field’s current setting.

-opticaldir2 dir2

An optional argument that specifies the input directory path for the second optical model to associate with the gauge data set. Specifying an empty string disables the field’s current setting.

-opticaldir3 dir3

An optional argument that specifies the input directory path for the third optical model to associate with the gauge data set. Specifying an empty string disables the field’s current setting.

Examples

The following example creates a super gauge object (using the sgd_create command), and then sets the dose value for gauge 1, the defocus value for the last gauge, and a weight of 1.0 for all gauges.

set obj [sgd_create -in “/tmp/sample.sgd”]
sgd_config $obj 1 -dose 1.04
sgd_config $obj end -defocus -0.14
sgd_config $obj all -weight 1.0
