Incorporating Calibre IPSE in Your Test Pattern Flow

Calibre IPSE is an image parameter space exploration tool that enhances test pattern creation by examining the coverage of a test pattern versus an actual design.

Detailed information on the use of the tool can be found in the section “Using Calibre IPSE”.

Note:

This best practice assumes you have already created an optical model (“Optical Model Creation”) and resist models (“VT5 Resist Model Creation”) and have them entered in the setup file. It does not work with Calibre ContourCal user flows.

The recommended flow with Calibre IPSE as a viewer tool is as follows:

  1. Select your verification (design) features and generate image parameter data with the samlinev2 command.

  2. Generate a calibration test pattern using a Siemens EDA version 12 test pattern (or your own calibration pattern using tepCompile and run samlinev2 on that test pattern).

  3. Compare the two patterns using Calibre IPSE to see how well the calibration pattern covers the verification data set.

  4. Modify your calibration pattern to address sites that are not well covered.
