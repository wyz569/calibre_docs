Optimizing the Optical Model

Follow this procedure to generate an optimized optical model. You perform these steps in CM1 Center Optics and Optimization tabs.

Prerequisites

  • The setup file from your original sample spreadsheet measurements or the model-layout file from “Creating a Model-Layer File”

  • The sample spreadsheet cleaned up in “Cleaning the Sample Spreadsheet and Gauge Object”

  • The corresponding layout file

Procedure

  1. Switch to the Optics tab.
  2. Select which optical model will be used as the nominal model using the radio buttons at the bottom of the tab (Figure 1).

    • This Page — Uses the model that you will load or define in this tab. If you select this option, continue to Step 3.

    • Gauge Linked — Uses the linked optical model from the Gauges tab. If you select this option, skip to Step 4.

    • Setup (or Model-Layer File) — Uses the optical model designated in the setup file specified in the Misc tab. If you select this option, skip to Step 4.

      Figure 1. CM1 Center, Optics Tab

  3. In the Optics menu, choose Open 1st Model. You will be prompted with a file browser dialog. Navigate to the optical directory (not the optical file) and click Open.

    The Optics tab fields on the left side of the tab are filled with the settings from the loaded optical file. Note that these settings correspond to the fields in the Calibre WORKbench Optical Model Tool.

    If you are modeling a double exposure system, choose Optics > Open 2nd Model and load the second optical model in the same way. The second optical model settings appear on the middle section of the tab.

    Note:

    See the section “Using Optical Models with Double Exposure Flows” for more instructions on how to set up double exposure systems.

    If you are modeling an implant exposure system, choose Optics > Open 3rd Model and load the third optical model. Contact your Siemens representative for more information on implant modeling.

  4. Switch to the Optimize tab (Figure 2 a), then select the Optics tab (b) in the middle of the page.
    Figure 2. CM1 Center, Optimize Optics Tab

  5. Select one or more parameters to tune for:

    • Click the checkbox in the center column (c) for the parameter. You should only choose to optimize values of interest that are also specified in the Optics tab; attempting to optimize for an empty field will result in an error.

    • Set start and end values. (d)

    • Set level values only for the full and gradient search methods. (d)

    • Optionally, set a Constraint by naming another value that will also be tuned for so that each of the linked tunable parameters is tested together.

  6. Switch to the Misc tab. Enter the either name of your setup file in the Setup field.
  7. Load the test pattern layout file into Calibre WORKbench.
  8. Using the mode selectors in the button bar, switch Optics to ‘O’ (Optimize), and CM1 and VEB to ‘X’ (off); this reduces the runtime by only working with the optical model.
  9. Click the Preview tab to check if there are any errors and inspect the modelflow_v2 command that will be run. Click the Run button to optimize the optical model.

    CM1 Center creates the new optical model in memory and a result log at the bottom of the window. It also returns the lower and upper bounds found during the search in the Optimize tab for the Front, Full and Gradient search methods.

  10. To view the results of the optimization, select the new result log at the bottom of the window and click the Analyze button.

    • Switch to the Optics Fitness tab and browse using the tree at the left side (CD Fitting Section, CD Uniformity Section, CD Linearity Section, and Contour Fitting Section).

    • To view the search criteria as a bivariate plot, select the Bivariates option from the Search Data tree at left side of the Analyze window.

      Figure 3. Optics Search Results

  11. To save the optimized optical model(s), use the Optics > Save 1st Model (and Save 2nd Model menu if you had a second optical model) option(s). If you attempt to enter an optical directory name that does not exist, CM1 Center creates the directory for you.
