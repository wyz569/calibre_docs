cm1load

CM1 and VEB Commands

Related Formats: CM1 Model File Format

Loads a CM1 object.

Usage

cm1load filename

Arguments

filename

A required argument that specifies the input CM1 file (required).

Description

Use the cm1load command in the Calibre WORKbench Tcl shell to create a CM1 object from a CM1 file. If the file does not exist, or is not a CM1 model file, then no object is created. Otherwise, the result of this command is a handle to the new CM1 object.

Examples

set cm1obj [cm1load cm1.mod]
