veb_getparam

CM1 and VEB Commands

Related Formats: Variable Etch Bias (VEB) Model File Format

Gets the value of the specified parameter from an input VEB object.

Usage

veb_getparam vebobj parameterName [index]

Arguments

vebobj

A required argument that specifies the input VEB Tcl object.

parameterName

A required argument that specifies a parameter name to be retrieved from the VEB object.

All model parameters are divided in two groups:

  • object parameters that belong to the VEB object as a whole.

  • term parameters are associated with a particular kernel in the VEB model expression.

If an object parameter is specified, then index must not be present. If a term parameter is specified, then a valid term index must also be specified. A valid index is an integer between 1 and (btermcount – 1) inclusive, where btermcount is the total number of linear coefficients (BTERMs) in the input VEB object.

All parameter names are case-sensitive. The following keywords return the associated values from the vebobj:

  • btermCount — Returns the number of terms in the VEB object, dimensionless.

  • BTERM0 — Returns the constant term (BTERM0) coefficient in nanometers.

Accepted term parameter names are the following:

  • u — Returns the kernel shift in nanometers.

  • es — Returns the kernel diffusion length in nanometers.

  • looking — Returns a string “in” or “out”, only for visible kernels

  • type — Returns the kernel type, either “gauss” or “visible”

  • BTERM — Returns the kernel term coefficient in nanometers.

index

An argument that specifies the input term index. It is required only if a term parameter is specified for parameterName.

Examples

This command loads the VEB object in the file veb.mod (veb_create command), then retrieves the parameters btermcount and the third es term.

set obj [veb_create –file “veb.mod”]

# get one of the object parameters, say, btermCount
set ncoeff [veb_getparam $obj btermCount]

# get one of the term parameters - the diffusion length 
# of the 3d term/kernel “es”
set s3 [veb_getparam $obj “es” 3]
