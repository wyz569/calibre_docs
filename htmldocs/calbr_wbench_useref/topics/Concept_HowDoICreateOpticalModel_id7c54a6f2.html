Optical Model Inputs

Task Flow Summary

  • Settings for use in the Optical Model tool

  • Test pattern file

  • Spreadsheet data

  1. Open the Optical Model Tool (Optics > Optical Model).

  2. Configure the Optical Model Tool with initial settings and generate a model.

  3. Open, then tune the optical model in VT5 Center (Litho > VT5 Center) or CM1 Center (Litho > CM1 Center).

  4. Optimize the final optical model.

Optical Model Outputs

What to Look for in Results

  • Optical model directory

Use VT5 Center or CM1 Center to view results:

  • CTRemean value: lower is better

  • CTRerms value: lower is better

Optical Model Commands

Description

fitmap

Calculates the composite source that best fits the specified source map.

kernsym

Reports on model kernel symmetry.

maskspectrum

Generates a mask spectrum file for the specified layout.

mergemap

Combines multiple source map files into a single averaged source map file.

opticsgen

Creates an optical model in batch mode.
