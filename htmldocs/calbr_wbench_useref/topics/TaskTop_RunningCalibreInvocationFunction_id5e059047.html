Running the Calibre Invocation Function

You can run the Calibre RET batch tools from the Calibre WORKbench application.

Prerequisites

  • You need to have the CALIBRE_HOME environment variable set to a valid MGC software tree

  • Have the following files:

    • A valid Calibre nmDRC SVRF rule file.

    • A Calibre WORKbench setup file.

Procedure

  1. In the Calibre WORKbench GUI, open the design file (File > Open Layout Files) to run the SVRF rule file on.
  2. In the RET Flow Tool (Litho > RET Flow Tool), load the setup file (File > Add Existing Setup).
  3. In the Calibre WORKbench GUI, select Options > Calibre Run. The Calibre Run Options window appears as shown in the figure.
    Figure 1. Calibre Run Options

    Click the Browse button next to the Rule File field, then browse and select an SVRF rule file.

  4. In the Calibre EXE field, type in the Calibre nmDRC invocation and any desired command-line switches in the Calibre Options field. (Note that the area of the layout you select is passed in as a flat design, so the output is also flat.)
