sgd_delete

CM1 and VEB Commands

Related Formats: Gauge Object Reference, Super Gauge Data File Format

Deletes a gaugedata set from a sgd Tcl object.

Usage

sgd_delete object index

Arguments

object

A required argument that specifies the input sgd Tcl object.

index

A required argument that specifies the index of the gauge set to be deleted. Accepted values for index are the following:

  • integer — Deletes the gaugedata set determined by this index number, which must satisfy the inequality:

    0 < index <= number_of_gaugefiles.

  • end — Deletes the last gaugedata set.

  • all — Deletes all gaugedata sets (and returns an empty sgd Tcl object).

Indexes for the remaining gaugedata sets are automatically reordered.

Examples

set obj [sgd_create -in “sample.sgd”]

# delete first and last sample sets
sgd_delete $obj 1
sgd_delete $obj end

# save the object to a file
sgd_save $obj -out “sample_new.sgd”
