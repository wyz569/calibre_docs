Test Pattern File Formats

Test pattern file format descriptions are described in the sections that follow.
Table 1. Test Pattern File Formats List

Format

Description

Sample Spreadsheet File Format

Used in the sample spreadsheet file. Contains coordinates and CD values for test structures.

Super Spreadsheet File

A collection of sample spreadsheet files. Contains the same information as a sample spreadsheet file arranged in an array.

Test Pattern File Format

Used to define a custom test pattern structure file.
