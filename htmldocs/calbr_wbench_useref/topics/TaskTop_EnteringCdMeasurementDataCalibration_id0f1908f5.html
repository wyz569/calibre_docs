Entering CD Measurement Data for Calibration

In this task, you manually enter CD measurement data for use in calibration. These specified “hotspots” will be used as additional alignment criteria.

Prerequisites

  • Accurate CD measurements from the verification test pattern

  • The corresponding layout

Procedure

  1. In the layout window, draw a selection rectangle around one of the “spots” for which CD is known.
  2. In the SEM tab of the SEMoverlay window, click the Get button in the first empty row of the Measurements Spots table.

    The SEMoverlay tool extracts the coordinates of the selection rectangle and enters them into the position column.



  3. Click on Show to open the Measurement SEM dialog box. The full title of this dialog box reflects the image and sample location under investigation.
  4. Click Browse to access the Load Image browser to find and open the SEM closeup image showing the exact measurement location.
  5. Verify, through visual inspection, that the measurement location matches the selection rectangle you drew in Step 1. If not, correct the problem:

    • If the measurement region is drawn incorrectly, redraw the selection box, then click the Get button for the row representing this measurement spot.

    • If you have opened the wrong closeup SEM, repeat Step 4 to open the correct SEM closeup.

    • If you are completely dissatisfied, click Clear to delete the selected row.

  6. In the Measurement SEM dialog box, click Find.


    The application compares the image topographies to locate the exact sampling location and mark it in the layout viewer.

  7. Adjust the image in the Measurement SEM dialog box as needed to read the CD value, then enter the CD value for that sampling spot in the SEM tab of the SEMoverlay window.
  8. Repeat for all remain CD sampling spots on the test structure.
