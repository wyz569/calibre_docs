Viewing Flare Map Information (Single Click Mode)

Use the Load Flare Map tool in Calibre WORKbench to view flare map information for EUV.

Prerequisites

  • Calibre WORKbench invoked

  • A design file containing EUV long range information, loaded into Calibre WORKbench

  • The corresponding flare map file

  • If you are viewing flare map intensity, the related litho model or global litho model

Procedure

  1. In Calibre WORKbench, select Litho > Load Flare Map.
  2. In the dialog box that appears, click Browse next to the PNG field to select the relevant .png file.


  3. Optionally, set the Flare Shift X and Y values if the flare map is offset from (0,0).

    If you are only testing the flare map percentage, skip to step 8.

  4. If you are viewing the flare intensity, select the Flare Intensity option and continue.
  5. Click the Browse button next to the Litho Model field and select a litho model or a global litho model.
  6. If you have loaded a global litho model, set the EUV X-Slit Center value to the center of the EUV field, even if it is 0.
  7. If the litho model includes black border models, set the EUV Field Center X and Y values to be the center of the EUV field.
  8. In the Calibre WORKbench main window, ensure that the tool is in Select mode, then draw a box over an area that corresponds to part of the flare map.
  9. In the Load Flare Map dialog box, click Flare.

Results

The Load Flare Map tool runs and shows the 2D flare map for the selected section of the design. If the resulting selection box is blank, try selecting a different section of the design. You may have selected an area that has no flare map coverage.



Note:

You can use Edit > Maps/Contours in the Calibre WORKbench main window to adjust the appearance of the flare map output.
