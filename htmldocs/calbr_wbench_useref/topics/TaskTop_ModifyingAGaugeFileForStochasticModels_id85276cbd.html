Modifying a Gauge File For Stochastic Models

Stochastic models require a modified version of a gauge file to input the stochastic high and low data for calibration.

Prerequisites

  • A standard gauge file from Calibre WORKbench or Calibre nmModelflow

  • Gauge measurements from a test chip

    This should be somewhere in the range of several hundred gauges measured from the test chip.

Procedure

  1. Open the gauge file in either a text editor or spreadsheet program.
  2. Add the following columns just to the left of the last column entry (the “Comment” column):

    LER1 — A required column listing the measurement value found at the topmost or leftmost edge of the gauge.

    LER2 — A required column listing the measurement value found at the bottommost or rightmost edge of the gauge.

    LWR — An optional column listing the measurement value for line width roughness (LWR). This column is only used for optimization of the LWR objective.

  3. Save the file.

Results

You can now use the gauge file as input to a stochastic model calibration command.
