Step 6 - Accounting for Mask Patterning and Resist Diffusion Effects (Optional)

Advanced users can customize the optical model to account for mask patterning, resist diffusion, or other factors that impact the aerial image.

There are two methods:

  • Specifying Kernel Diffusion (Pre-Convolution)

  • Specifying Image Diffusion (Post-Convolution)

Figure 1 represents these convolutions graphically.

Figure 1. Pre- and Post-Convolutions

When using the Optical Model Tool to generate an optical model, the pre- or post-convolutions are defined in the Kernels/Image tab.

The pre-convolution can be used when:

  • The properties of the mask writer that is used to produce the masks are known.

  • Information characterizing the rounding and other distortions introduced by the mask writer is known.

The pre-convolution should not be used when:

  • No information about the mask writer is available.

Usage Notes

  • Image diffusion features are useful for VT5 models in modelflow_v1 or VT5 Center tool because similar terms are not available in the VT5 resist model.

    Note:

    Image diffusion terms are available as part of the CM1 resist model for backwards compatibility, but are not recommended for inclusion with optical models used with modelflow_v2 or CM1 Center starting with the 2011.2 release.

  • In the VT5 Center tool, the double Gaussian image diffusions can be tuned during Stage 1 optimization using the following three keywords as a set:

    • imagediffusions1 beg end step — sigma for the first Gaussian

    • imagediffusions2 beg end step — sigma for the second Gaussian

    • imagediffusionw1 beg end step — weight for the first Gaussian. Note that the weight for the second Gaussian is calculated as 1- imagediffusionw1.

  • Special Considerations for Using Post-convolutions in 3D

    The standard post convolution using imagediffusion diffuses the image across the xy plane. Extending the post convolution with a Gaussian diffusion to include the z-axis lets you simulate diffusion within the resist film.

    In order to produce meaningful results when extending the convolution into the third (z) dimension, you should adhere to the following restrictions:

    • Define a Gaussian sigma (standard deviation) that is appropriate for all three dimensions.

    • Use only when additional diffusion is really needed. Data from the Cutline 2D tool can be used to determine this.

    • When using a vector model, the film stack should be fully characterized and the image to be diffused should be completely inside the film stack.
