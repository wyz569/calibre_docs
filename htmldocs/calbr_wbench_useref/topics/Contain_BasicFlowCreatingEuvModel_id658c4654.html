Basic Flow for Creating an EUV Model

Creating an EUV model is similar to creating a DUV model, but adds steps to the flow.


Table 1. EUV Calibration Flow

Step

Task

Described In

1

Create a flare model file.

  • “Creating a Flare Model File”

2

Calculate the flare map data.

  • “Generating a Flare Map With RET FLARE_CONVOLVE”

3

Create a nominal EUV optical model.

  • “Creating an EUV Optical Model File”

4

Create a DDM model to accompany the optical model.

  • “Adding HHA and DDM Information to the EUV Optical Model”

  • “Creating an EUV DDM Model Using the DDM MaskSim GUI”

5

Create a litho model to store the models you created.

  • “Creating an EUV-Enabled Litho Model”

6

Configure Calibre nmModelflow to calibrate the optical and resist models.

  • “Setting up an EUV Calibration Stage”

  • “Setting up EUV Calibration Jobs”

  • “Running EUV Calibration Jobs”

Opt

View design file information using the calibrated models.

  • “Viewing EUV Cutline Information”

  • “Viewing Flare Map Information (Single Click Mode)”

  • “Viewing Flare Map Information (RET Flow Tool)”

7

Use the calibrated model as input to Calibre® nmOPC™ and Calibre® OPCverify™.

  • See the Calibre nmOPC User’s and Reference Manual and the Calibre OPCverify User’s and Reference Manual

The primary difference between the DUV and EUV model flow are Steps 1 and 2; these additional steps create the flare model that accompanies the standard optical and resist models. You still create an optical model (Step 3) and DDM model (Step 4), except using EUV-appropriate settings, and you create the CM1 model using a standard resist modelform in Step 5 inside the Lithomodel/Setup/MDF Configuration Creation Wizard. Model calibration (Step 6) and use of the calibrated models in Calibre nmOPC and Calibre OPCverify (Step 7) require adding options that enable EUV models to be recognized by the Calibre tools.
