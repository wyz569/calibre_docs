cm1getparam

CM1 and VEB Commands

Related Formats: CM1 Model File Format

Retrieves the value of a specified parameter from a specified CM1 object.

Note:

The parameters retrieved may have been set by a previous cm1setparam command.

Usage

cm1getparam cm1obj parameterName [index]

Arguments

cm1obj

A required parameter that specifies the input CM1 Tcl object.

parameterName

A required parameter that specifies the input parameter name as either an object parameter or a term parameter. The command returns the value stored in that parameter name from the input CM1 Tcl object. Parameter names are case-insensitive.

The model parameter names are divided into two groups:

  • object parameters that belong to the CM1 object as a whole.

    Accepted object parameter names to retrieve are these:

  • anchor — Returns the index of the anchor term, dimensionless

  • coefmax — Returns the maximal allowed coefficient norm (not including the coefficient of the anchor term), dimensionless

  • consistency_setting — Returns the status flag of 1 (on) or 0 (off) for the consistency routines used in modelflow_v2 optimize and try regimes

  • const_term — Returns the constant term, dimensionless

  • maxorder — Returns the maximal allowed Gauss-Laguerre polynomial order

  • modeltype — Returns the model type, which is currently always “CM1”

  • resist_polarity — Returns the resist_polarity setting, either -1 (NTD) or 1 (PTD)

  • sidewall_bias_b — Returns the sidewall bias b value, dimensionless

  • sidewall_bias_c — Returns the sidewall bias c value, dimensionless

  • sidewall_bias_d — Returns the sidewall bias d value in nanometers (nm)

  • sidewall_bias_s — Returns the sidewall bias s value, dimensionless

  • sidewall_bias_t — Returns the sidewall bias t value, dimensionless

  • smooth_neutralization — Returns the neutralization type as a value (2 = spline smoothed neutralization, 1 = square root smoothed neutralization or 0 = off)

  • smooth_gradient — Returns the gradient smoothing type as a value (2 = square of the gradient, 1 = square root of the gradient, or 0 = off)

  • threshold — Returns the model threshold, dimensionless

  • version — Returns the version number, dimensionless

  • z_slope_a — Returns the z_slope a value, dimensionless

  • z_slope_b — Returns the z_slope b value, dimensionless

  • z_slope_c — Returns the z_slope c value, dimensionless

  • term parameters that belong to particular terms in the CM1 model expression.

    Accepted term parameter names to retrieve are these:

  • b or B — Returns the term neutralization constant, dimensionless

  • s or S — Returns the term diffusion length in nanometers (nm)

  • k or K — Returns the term derivative order, dimensionless

  • n or N — Returns the term power, dimensionless

  • p or P — Returns the term Gauss-Laguerre function order, dimensionless

  • curvature_d — For the contour curvature term, returns the diffusion length value in nanometers (nm)

  • curvature_s — For the contour curvature term, returns the smoothing length value in nanometers (nm)

  • curvature_t — For the curvature term, returns the threshold value, dimensionless

  • dissolution_span — For the dissolution term, returns the diffusion length value in nanometers (nm)

  • dissolution_threshold — For the dissolution term, returns the neutralization threshold value, dimensionless

  • electron_absorption_fov — For the electron absorption term, returns the diffusion length value in nanometers (nm)

  • electron_absorption_threshold — For the electron absorption term, returns the neutralization threshold value, dimensionless

  • horizontal_bias_a — Returns the horizontal bias a value, dimensionless

  • horizontal_bias_b — Returns the horizontal bias b value, dimensionless

  • horizontal_bias_c — Returns the horizontal bias c value, dimensionless

  • horizontal_bias_d — Returns the horizontal bias d value in nanometers (nm)

  • horizontal_bias_s — Returns the horizontal bias s value in nanometers (nm)

  • pattern_density_span — For the pattern density term, returns the diffusion length value in nanometers (nm)

  • pattern_density_threshold — For the pattern density term, returns the neutralization threshold value, dimensionless

  • plusminus — Returns the term neutralization type as a dimensionless value; two possible values are 1 for acid neutralization and –1 for base neutralization

  • mcoeff — Returns the term coefficient, dimensionless

  • scoeff — Returns the shrink term coefficient, dimensionless

index

An argument that specifies the input term index as a non-negative integer (required only if a term parameter is specified).

  • If an object parameter is specified, then index must not be specified.

  • If a term parameter is specified, then a valid index term must be specified. A valid index is an integer between 0 and (nterms - 1) inclusive, where nterms is the number of terms in the input CM1 object.

Examples

set cm1obj [cm1load cm1.mod]
# get one of the object parameters, say, index of the anchor term
set anch [cm1getparam cm1obj anchor]
# get one of the term parameters, say, diffusion length of the 4th term
set s4 [cm1getparam cm1obj S 4]
