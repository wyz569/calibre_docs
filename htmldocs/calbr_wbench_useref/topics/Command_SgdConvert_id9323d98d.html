sgd_convert

CM1 and VEB Commands

Related Formats: Gauge Object Reference

Converts a pre-2009.1 super gauge object to the current version.

Usage

sgd_convert object {2008 | 2009}

Arguments

object

A required argument specifying the input gauge data object.

{2008 | 2009}

A required argument specifying the format the input gauge object to be converted to:

  • 2008 — Converts gaugedata to the format accepted in 2008.4 and earlier releases.

  • 2009 — Converts gaugedata to the format accepted in 2009.1 and later releases.

Description

Converts a super gauge object between the pre-statistics version used in releases 2008.4 and earlier and the statistics-enhanced version used from 2009.1 to the present version. Specifically:

  • mode became stdev

  • count remained the same, but the functionality has changed

  • dist became 95%ci

This is a utility command mainly designed to resolve compatibility issues, in that the statistics-enhanced version of the gauge object is not backward-compatible and cannot be read with the sgd_create command of 2008.4 and earlier releases. The pre-2009.1 format is compatible with all releases.

Examples

# convert gaugedata to the format accepted in 2008.4 and earlier releases
sgd_convert aobj 2008
