configure_grid

Scope: Calibre Pattern Generator API Commands

Sets the array dimensions, cell size, and spacing for the layout.

Usage

$cpg configure_grid ‑rows num_rows ‑cols num_cols
‑grid_size gr_size ‑grid_space gr_space

Arguments

$cpg

A required argument specifying $cpg as a Tcl configuration object or variable.

‑rows num_rows ‑cols num_cols

A required argument set specifying the number of rows and columns in the layout file.

‑grid_size gr_size

A required argument specifying the cell size in user units.

‑grid_space gr_space

A required argument set specifying the distance between cells in user units.

Description

A required command set defining the number of rows, columns, and cell size and spacing for the layout configuration file.

Examples

The following example defines a 20-row, 20-column array of cells. The cell size is 2500 user units with 2600 user unit spacing between each cell:

$cpg configure_grid ‑rows 20 ‑cols 20 ‑grid_size 2500 ‑grid_space 2600
