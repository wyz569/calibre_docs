Creating a Pattern With the Calibre Pattern Generator

Use the Calibre Pattern Generator to create a test pattern to add to a layout.

Prerequisites

  • Knowledge of your test pattern parameters.

Video

The video series “Getting Started with the Calibre Pattern Generator” describes and demonstrates the main functionality and key workflow steps to create new patterns and layouts.

  • The “Overview” video describes the main functionality and key workflow steps to create new patterns and layouts.



  • The “Demonstration: Creating a New Pattern and Pattern Library” video demonstrates the creation of a new pattern and pattern library using the Pattern Creator.



Procedure

  1. In Calibre WORKbench, invoke the Calibre Pattern Generator from the Litho > Calibre Pattern Generator menu item.
  2. If needed, select the Pattern Creator (PC) tool by clicking the Tools icon at the top right and selecting Pattern Creator in the menu.


  3. Add a pattern to a pattern library using one of the following:

    • Load a pattern from a built-in library. For details, refer to Step 4.

    • Create a new pattern library and a new pattern. For details, refer to Step 5.

    • Capture a pattern from WORKbench. For details, refer to “Capturing a Pattern from Calibre WORKbench”.

  4. To add a pattern from a built-in pattern library:
    1. Select Add Library > Built in. Select one or more built-in libraries.


    2. Select and edit one or more patterns from the built-in libraries.

      Set the pattern property values to desired values and confirm gauge placement to collect the desired measurements.

  5. To create a new pattern library and new pattern:
    1. Select Add Library > New. Name the library and click OK.


    2. Add a new pattern by hovering over the library name and clicking +. Build the pattern by combining shapes and gauges.

      1. Add a shape or gauge by clicking on it in the Pattern/Gauge ribbon and dragging it to the approximate desired location for placement.

        • Shape options are Line, Lines List, and Lines Matrix.

        • The gauge configures horizontal and vertical gauges.

      2. (Optional) Use properties to specify dimensions and names for shapes and gauges.

        1. In the Pattern Properties table, create a new property by hovering over Pattern Properties and clicking +.

        2. Name the property and specify a value.

        3. Select the shape or gauge. To use math operations, type in the name of the property and any additional math operations used with the property. The math operations supported with properties are “+”, “-”, “*”, and “/”.

      3. (Optional) Add a gauge to the center of the pattern for a lines list or lines matrix by selecting the gauge and activating the Centred checkbox. The centered gauge moves to the pattern center.

      4. Specify the parameters of the shape or gauge by clicking on them and editing their values in Container Properties and Item Properties of the properties table.

        To view the current value of any property having variables or equations, hover the cursor over its entry in the properties table.

        The properties table supports using simple functions. The functions must be in lower-case with an expression in parentheses. The following functions are supported:

        • abs — The absolute value of the expression.

        • ceil — The smallest integer value that is greater than the expression.

        • cos — The cosine of the expression in degrees.

        • floor — The greatest integer value that is less than the expression.

        • pow2 — The square of the expression.

        • sin — The sine of the expression in degrees.

        • sqrt — The square root of the expression.

      5. Continue to add shapes and gauges until you complete the pattern.

  6. Save the pattern library by hovering over the library name and clicking the disk icon.
  7. If you are creating additional patterns for the pattern library, continue to add new patterns to the library.
  8. If you have completed adding patterns to the pattern library, continue to “Creating a Layout With the Calibre Pattern Generator” to create a layout with the pattern library.
