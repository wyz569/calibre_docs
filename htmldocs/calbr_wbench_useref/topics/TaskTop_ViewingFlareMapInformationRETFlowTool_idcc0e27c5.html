Viewing Flare Map Information (RET Flow Tool)

Use the RET Flow Tool in Calibre WORKbench to view flare map information for EUV.

Prerequisites

  • Calibre WORKbench invoked

  • A design file containing EUV long range information, loaded into Calibre WORKbench

  • The corresponding flare map file

  • The related litho model or global litho model

Procedure

  1. In Calibre WORKbench, select Litho > RET Flow Tool v2.0.
  2. In LM mode, load the litho model or global litho model file.
  3. Click the Flare button for at least one PWC if it is not already selected.
  4. In the Configure Flare section, change the Flare Long Range dropdown from “none” to “png” and set the relevant .png file.
  5. Optionally, set the Flare Shift X and Y values if the flare map is offset from (0,0).
  6. If you are using a global litho model, you must also set the EUV X-Slit Center value to the center of the EUV field, if it is something different than 0.
  7. If the EUV model includes black border models, you must set the EUV Field Center X and Y values to be the center of the EUV field.
  8. Add an input layer to the mask layer using the Layer Mapping list and set it to the visible layer for the flare map.
  9. In the Calibre WORKbench main window, ensure that the tool is in Select mode, then draw a box over an area that corresponds to part of the flare map.
  10. In the RET Flow Tool v2.0, click Map.

Results

The Map tool runs and shows the 2D flare map for the selected section of the design file. If the resulting selection box is blank, try the following:

  • Select a different section of the design. You may have selected an area that has no flare map coverage.



Note:

You can use Edit > Maps/Contours in the Calibre WORKbench main window to adjust the appearance of the flare map output.
