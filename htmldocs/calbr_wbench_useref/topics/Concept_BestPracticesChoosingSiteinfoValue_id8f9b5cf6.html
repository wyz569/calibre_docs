Best Practices for Choosing the siteinfo Value

When working with VT5 models, use these best practices for setting the most effective siteinfo value in your setup file.

Both VT5 Center and modelflow_v1 use a Calibre OPCpro setup file, which stores simulation information. In the setup file, the “siteinfo” keyword specifies how EPE test sites are configured. The siteinfo setting uses the syntax:

siteinfo [AERIAL | RESIST | SAMPLE] -num x -spacing y [-center z]

Sample Points (-num Argument)

The number of sample points (-num argument) you specify is multiplied by the -spacing setting to create the site. Find the best number of sample points based on your observation of the EPE spacing results.

  • Not using enough sample points causes the intensity curve to fail to reach Imax.
