gaugeResize

CM1 and VEB Commands

Related Formats: Gauge Object Reference

Resizes the length of the specified gauge or all gauges of the input gauge object.

Usage

gaugeResize object gauge_index {–extra integer | –scale value | -set value }

Arguments

object

A required argument specifying the input gauge data object.

gauge_index

A required argument specifying the index of the gauge to be modified. Accepted values for gauge_index are the following:

  • integer

    Resizes the gauge specified by this index number, which must be an integer between 0 and the size of the input gaugedata object-1.

  • end

    Resizes the last gauge of the input gaugedata object.

  • all

    Resizes all gauges of the input gaugedata object.

One of -extra, -scale, or -set is required:

–extra integer

Increases (integer > 0) or decreases (integer < 0) the gauge length by 2*integer in database units.

–scale value

Rescales the gauge length by the specified amount. the coordinates of the end gauge points are changed so that the new gauge length is equal to the old one multiplied by value.

-set value

Sets the length of the gauge to the specified value.

Examples

# resize all gauges by 200 dbu
gaugeResize aobj all –extra 200
# rescale gauge No 18 by a factor of 1.3
gaugeResize aobj 18 –scale 1.3
