Flare Intensity File Format

Input for: euv::fitflare

Describes the flare intensity used to calculate fractal kernel coefficients by a least squares regression.

Users that have detailed flare intensity map data can create a flare intensity file (.fif) with the experimental data in the file. This file can be turned into a flare model file with the euv::fitflare command.

Format

A flare intensity file must conform to the following restrictions:

  • Comments begin with a pound sign (#).

  • The file is a five-column space- or tab-separated table.

    x1 y1 w1 I1_meas I1_pred  
x2 y2 w2 I2_meas I2_pred
…
xm ym wm Im_meas Im_pred

Parameters

x

A required value specifying the x coordinate of the kernel, in microns.

y

A required value specifying the y coordinate of the kernel, in microns.

w

A required value specifying the fitting weight, which must be a non-negative number.

I_meas

A required value specifying the measured flare value at the x and y coordinates.

I_pred

A required value specifying the flare value that is predicted by a model as the output of the euv::fitflare command. The contents of this column are ignored as input. When creating this file for the first time, a value of 1 is acceptable.
