SEM Verification Overview

The Calibre SEMoverlay tool1simplifies model verification against SEM images. SEM verification provides improved model accuracy verifying model performance for 2D structures and for asymmetrical structures.

You perform the model verification against a SEM image with the following steps.

Table 1. SEM Image Verification Flow

Step

Task

1

Importing the SEM with Auto Alignment

2

Entering CD Measurement Data for Calibration

3

Extracting the SEM Contour

4

Comparing the Simulated Image to the Contour

SEM Verification Prerequisites

SEM verification requires the following inputs:

  • Verification Test Pattern

  • SEM Images of the Verification Test Pattern

  • CD Measurement Data

  • Calibre OPCpro Setup File

SEM Images of the Verification Test Pattern

You should take several SEM images for each of three to five key structures. These include:

  • A full structure image:

    • FOV (Field Of View): 3.5 um

    • Magnification: ~ 37.5k– 40k

    • Save in TIFF format



  • Three to four closeup images:

    • FOV (Field Of View): 1.75 um

    • Magnification: ~75k – 80k

    • Save in TIFF format

    Measurements can be space or width CD. The illustration shows three sampling spots.



    Each closeup should contain a marker box indicating the exact position of single measurement spot and text info containing the CD at that spot.

CD Measurement Data

You must know the CD at one spot on each closeup image. To make it easier to track this information, store these values as “text info” on the SEM itself.

Calibre OPCpro Setup File

You must have a Calibre OPCpro setup file referencing the models under investigation:

  • Optical Model

  • Resist Model

Good image quality is important for extracting a useful SEM contour. Bad images are those with poor contrast, focusing, or resolution.
