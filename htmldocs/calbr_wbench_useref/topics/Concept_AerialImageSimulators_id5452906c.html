Available Simulation Tools

The simulation tools available in Calibre LITHOview and Calibre WORKbench are in the RET Flow Tool.

Aerial Image Simulators

Aerial image simulators show the light intensity on one specific plane in the resist stack, measured at the specified defocus start value. They incorporate calculations for optically visible data layers (opc, visible, external, and correction) and only require an optical model to run. The tools highlighted in the button bar are aerial image simulation tools.
