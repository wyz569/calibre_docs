Generating a Test Pattern With the Calibre WORKbench GUI

Use the Calibre WORKbench Test Pattern tool to create a starting test pattern at your node level.

Prerequisites

  • Knowledge of your test pattern parameters

  • Review “How to Choose a Siemens EDA Test Pattern” to decide which test pattern you will need.

Procedure

  1. In Calibre WORKbench, invoke either the Test Pattern V11/12 or Test Pattern V10 generator from the Litho > Test Patterns menu item.
  2. Fill out the Generate Test Pattern V11/12 dialog box with the pertinent information and click OK or Generate. (Both buttons generate the test pattern, but OK closes the dialog box when the generator completes.)
    Figure 1. Test Pattern V11/V12 Generator

    Table 1. Contents of the Test Pattern V11/V12 Generator

    Control

    Comment

    Pattern Type

    Choose from: Standard, Contact, Verification, Etch, or PSM, depending on your test pattern type.

    Top Cell Name

    Used as the root of the cell tree.

    Database Unit, nm

    You will need to record these values for later use in the setup tool.

    Grid, nm

    CD width / CD space, nm

    The V11/V12 generator improves on the V10 generator in that it allows you to specify the width and space CDs separately.

    Version 11 or 12

    12 is the current default. Use a value of 12.2 for 45nm and smaller nodes.

    Output Files

    You can name the output test pattern and sample files individually. The V11/V12 generator can create:

    • GDS and OASIS test pattern files

    • Sample files in sample spreadsheet (SS, for modelflow_v1), gauge (GG, for modelflow_v2 and Calibre® nmModelflow™), and comma-separated value (CMi CSV) format.

    Markers and Border

    Selecting these options adds a border around the entire test pattern and SEM alignment markers in one of three types:

    • Odd/Even — 400nm x 400nm squares, with one square representing an even-numbered column and two squares representing an odd-numbered column. The test pattern contains one marker for each structure.

    • SEM Alignments — Twelve markers, one for every four test structures.

    • Cross — One marker for each structure.
    Figure 2. Test Pattern V10 Generator

    Table 2. Contents of the Test Pattern V10 Generator

    Control

    Comment

    GDS file

    You can only create GDS files with the V10 test pattern generator.

    Drawn Mask CD, nm

    Used for both width and space CD.

    Pattern type

    Sets the pattern type to one of Extended, PSM, PSMExt(ended), or Contact.

    Do not use Extended if you are working with a node smaller than 80nm, as the hammerhead patterns will be merged.

    Narrow

    Selecting this checkbox creates a narrower test pattern on the Y axis. For example, what would normally be a 130 nm pattern with the Narrow option checked creates a 100 nm wide pattern.

    Grid, nm

    You will need to record these values for later use in the setup tool.

    DB units per micron

    Sample Point at Center

    When selected, Calibre WORKbench adds a sample point at the center of each test structure.

Results

The new test pattern is written to the current working directory by default, and it is also automatically loaded into the Calibre WORKbench main window.
