How to Choose a Siemens EDA Test Pattern

Test patterns created by Calibre WORKbench are the fastest method to generate a test chip. Currently, versions V10, V11, and V12 of test patterns are available through the GUI interface.

Table 1 provides a summary of test pattern usage.
