calc_telecentricity

Optical Model Commands

Reports the telecentricity error of a source map.

Usage

calc_telecentricity ‑map sourcefile

Arguments

‑map sourcefile

A required argument set that specifies the path to the input source file.

Return Values

The command prints an ordered pair of telecentricity error values for the X and Y axes in units of milliSigma. A milliSigma represents one thousandth of a source sigma.

Source Telecentricity Error: X x_telecentricity Y y_telecentricity milliSigma

Description

This command reports the telecentricity error for a source. The telecentricity error is the average location of the source intensity and similar to the “center of mass”. A symmetric source has telecentricity error of 0 in the X and Y axes.
