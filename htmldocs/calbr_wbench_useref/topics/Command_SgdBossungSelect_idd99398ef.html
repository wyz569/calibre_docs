sgd_bossung_select

CM1 and VEB Commands

Related Formats: Gauge Object Reference, Super Gauge Data File Format

Identifies and selects focus-sensitive gauges in an input sgd object.

Usage

sgd_bossung_select object [-cdtype {resist_meas | resist_sim | etch_meas | etch_sim}]
[-out file] [-outenabled file] [-outcd file] [-maxrms val] [-cdswing val]

Arguments

object

A required argument that specifies the input sgd Tcl object. sgd_bossung_select works only if at least one Bossung replica group is present. This means that the input sgd object should include three or more process conditions with different defocus values.

Note:

Do not use height or SRAF gauge files as input. This command selects focus-sensitive CD gauges.

-cdtype

An optional argument that specifies the type of CD (gauge data column) to be used for parabolic approximation. Accepted values are:

  • resist_meas — Measured (empirical) resist CDs (default).

  • resist_sim — Simulated resist CDs.

  • etch_meas — Measured etch CDs.

  • etch_sim — Simulated etch CDs.

-out file

An optional argument specifying an output file containing only Bossung replica group gauges enabled. These gauges are marked; the comment column of each enabled gauge contains the Bossung group number the gauge belongs to.

-outenabled file

An optional argument specifying an output file that contains only enabled Bossung replica group gauges and non-empty gauge sets.

Tip

Use -outenabled instead of -out when the output file is intended as input for modelflow_v2, which does not accept empty gauge sets.

-outcd file

An optional argument specifying an output file for dumping Bossung replica CD data as functions of defocus.

-maxrms val

An optional argument specifying an upper limit in nanometers (nm) to be used for filtering out Bossung replica groups whose RMS error is greater than the specified value.

-cdswing val

An optional argument specifying a lower limit in nanometers (nm) to be used for filtering out Bossung replica groups whose CD swing is less than the specified value.

Description

Use the sgd_bossung_select command to identify focus-sensitive gauges in the input super gauge data (sgd) object. The command uses the following selection sequence:

  1. Finds all Bossung replica groups. A Bossung replica group is defined as a set of three or more enabled gauges that have all of the following:

    • Positive CDs in the specified gauge data column

    • Identical coordinates and doses

    • Different defocus values.

  2. For each group, it computes a parabolic approximation for CD as a function of defocus f using an RMS error fitting algorithm.

  3. Characterizes the quality of the parabolic approximation using the RMS error, the maximal absolute error and a CD swing value calculated using the maximal absolute defocus value in the input sgd object. The greater the CD swing the more sensitive this gauge to variations in defocus.

  4. Computes the mean focus center f center where the average is done over either all Bossung replica groups if no constraint is present, or only over those groups that satisfy the specified constraints on maximal RMS error or CD swing.

  5. Outputs simulation data for all groups or just for those that satisfy specified constraints to the screen and returned to Tcl in the output string.

This command ignores disabled gauges, as well as those that have nonpositive weight, simulated (if resist_sim or etch_sim is specified) or measured (if resist_meas or etch_meas is specified) CD values. The command does not support sgd objects with duplicated gauges, defined as gauges that have identical coordinates and belong to the same gauge set (process condition).

Because Bossung gauges are limited to CD-type data, height gauge files such as those used in 3D resist modeling and SRAF gauge files are not valid input.

Examples

# input gauges
set obj [sgd_create -in focus_calibration_allON.sgd]
set Bossung_data [sgd_bossung_select $obj -cdtype resist_meas 
   -out withFilter/bossung_replicas_out.sgd 
   -outcd withFilter/bossung_replicas_outcd.txt  
   -maxrms 7 -cdswing 10]
# save results
sgd_save $obj -out withFilter/focus_calibration_withFilter.sgd
set chan [open withFilter/Bossung_data.txt a]
puts $chan $Bossung_data
close $chan
