imageplanes

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Performs intensity interpolation for 3D resist models.

Usage

imageplanes nominal d0 [only | fixed | auto] {[num start step] | [ planes d1 d2 … dn ]}

Arguments

nominal d0

A required argument set specifying the nominal image plane d0 in um.

only | fixed | auto

An optional argument specifying the mode of the imageplanes command.

  • only

    Turns interpolation off.

  • fixed

    Uses the specified imageplane (from the num start step or planes d1 d2 … dn keywords).

  • auto

    Instructs Calibre WORKbench to internally select its image planes. This is the default.

num start step | planes d1 d2 … dn

An optional argument set specifying additional image planes that are stored with the optical model.

CAUTION:

These parameters are not the same as the ones in the description of the defocuslevels parameter.

Description

Note:

This Z-axis parameter is used specifically for 3D resist modeling, and replaces the defocuslevels parameter in the optical model.

An optional Z-axis parameter that performs intensity interpolation with special considerations for 3D resist models. This parameter is only valid for optical models with thin film stacks, and the specified image planes must be within the resist film.
