Generating a Source Map File Using fitmap

The fitmap command generates a parametrized source map by fitting source parameters to a known source map.

The main application of fitmap is to obtain a compact, parametrized representation of a measured source.

Prerequisites

  • A source map file in the Calibre format (see “Source Map File Format” for more information)

Procedure

  1. Copy the optical model file to the current directory you ran Calibre WORKbench from.
  2. Run fitmap from the Calibre WORKbench shell:
    fitmap -in filename -type {sourcetype param1 min max param2 min
max param3 min max} -profile erf

    The number of param min max arguments should match the composite source type. For example, if you have an annular source map named source_map_file.src, you would type the following: 

    fitmap -in source_map_file.src -type {annular sigma 0.0 1.0 sigma_in 0.0 1.0 dsigma 0.0 1.0} -profile erf
  3. Record the result string from the command.

    The result of the example command would be similar to: 

    -annular {weight 0.708 sigma 0.895 sigma_in 0.555} -fit 0.0370
  4. In Calibre WORKbench, open the Optical Model Tool (Optics > Optical Model). Follow the standard steps described in this chapter until you get to “Step 2 - Illumination Source Definitions”.
  5. Select the Composite Source type, then click Add to open the Add Source Element dialog box.
  6. Enter the values from step 3 of this procedure, discarding the -fit result.
  7. Continue optical model creation.
