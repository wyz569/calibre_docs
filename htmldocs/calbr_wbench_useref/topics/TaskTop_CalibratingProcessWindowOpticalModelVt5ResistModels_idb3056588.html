Calibrating a Process Window Optical Model for VT5 Resist Models

Advanced users might need to perform additional tasks when optimizing a super spreadsheet data set loaded in the Sample tab.

Procedure

  1. Build the optical model with as much information as possible (measured source, film stack, and Jones pupil information when available).
  2. Calibrate the beamfocus and def_start parameters using the information in the section “Calibration Best Practices for beamfocus and def_start Parameteters”.

    • You should use only a few focus-sensitive features over the focus exposure matrix (FEM) and calibrate for the nominal and process corner conditions.

    • Use a Front search algorithm with 128 iterations.

  3. Calibrate additional optical parameters (such as image_diffusion or apodization) using data at nominal conditions and the beamfocus and def_start values from step 2.
  4. For the VT5 model calibration step:

    • Use the optical model from step 3. Measurements at nominal focus conditions are sufficient to calibrate the VT5 model, but using the data from the nominal and corner conditions is recommended.

    • The variety and weights of test structures should represent the process of interest and the desired model accuracy by feature type.

  5. Verify the final model with the full process window data.
