The Flow for Using the Calibre IPSE Tool

The general usage flow for the Calibre IPSE tool is as follows:

  1. Calibrate optical and resist models from one or more test patterns.

  2. Use the Calibre IPSE tool to observe the image parameter space coverage of your calibration test pattern versus the sample data or Modelflow log data.

  3. Improve your image parameter space coverage in the following ways:

    • Add new features to the test pattern to better represent the coverage space, using the Calibre IPSETX tool

    • Incorporate Calibre® ContourCal tool data to capture unique feature types as contours

    • Use the process window data “as is” for the process
