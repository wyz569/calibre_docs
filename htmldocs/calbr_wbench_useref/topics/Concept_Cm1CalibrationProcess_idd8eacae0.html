CM1 Calibration Process

CM1 Calibration is a process that uses sample gauge files.

These are the files you created in the Pre-CM1 Calibration Process. You will be using the Calibre WORKbench CM1 Center tool to load and edit/modify the data.

Table 1. CM1 Calibration Process

Stage

Task

Description

1

Creating an Initial CM1 File

Create a ‘real’ CM1 file.

2

Performing Fine CM1 Calibration

Perform iterative calibration using more predictive elements.

3

Examining the CM1 Model for Quality

Check your calibrated model for accuracy.

Resuming Stopped Modeling Runs for Front Searches

Resume some runs that have been stopped.

Option

Exporting CM1 Center Settings as a modelflow_v2 Run Script

Export your settings as a modelflow_v2 run script file.
