Process Window Plot

To access: From Calibre WORKbench, Litho > PW Plot. In the Process Window dialog box, adjust settings and click Re-Plot.

The Process Window plot shows the common process window, best focus, and best dose across a set of gauges. It displays two types of plots in separate tabs - a Common Process Window (CPW) and exposure latitude (EL) versus depth of focus (DOF) plot. It provides controls to toggle individual gauges or adjust model settings to quickly see how they affect the CPW or EL vs DOF values.

Figure 1. Process Window Plot

Description

In the Process Window plot, you can change various settings under the Model Setting tab and click Re-Plot to see how the different polynomial types, shapes, and centering affect the common process window and the best dose and focus.

The Process Window tool also provides access to the Exposure Latitude vs DOF plot (EL vs DOF tab) and Bossung plots for individual gauges.

The area between the two sets of tabs is not editable, aside from the Show data points option and menu. Use the Model Setting tab to change EL% and DOF and the process window object settings. When you click Re-Plot, the Best Focus, Best Dose, and Bounding GIDs fields are re-calculated. The pushpin icons indicate which values are fixed (not varying) based on the model setting fields.

To see the points used in calculating the common process window, be sure Show data points is selected, then click a curve. By default, the points used show as solid markers. Use the dropdown list to additionally show points filtered out by the first pass (X markers) or the second pass (bowtie-shaped markers). To dismiss the points, click in the graph area or unselect Show data points. See “Usage Notes” for information on first and second pass filtering.

Model Setting tab

Settings affect the Common Process Window (cPW) calculations.

Model Form

The type of polynomial to use when modeling the common process window.

  • Poly13 — Default. Provides better fit for the DOF measurement.

  • Poly8 — A smaller polynomial provided for backwards compatibility. Poly8 is not recommended.

Type

The shape to fit inside the common area when measuring depth of focus (DOF). Select Elliptical (default) or Rectangular.

Target CD

Whether the DOF calculations should use the nominal print image CD or the drawn CD from the super gauge data (if available). Drawn is the default and recommended.

If you do not have a super gauge data file or if it does not include drawn CD measurements, the DOF calculation uses the nominal print image CD from simulation instead. Only set Nominal if you want to ignore the drawn CD.

Center

Forces the inscribed ellipse or rectangle in the common process window to be centered.

  • Dose — Default. This forces the dose variation to be symmetric around the dose value set in the Nominal field. It does not constrain the focus.

  • Focus — Similar to Dose, but forces the inscribed shape to be centered on the nominal focus.

  • None — Use for maximum common DOF. May be overly optimistic.

  • Both — Use to explore the common process window around nominal conditions.

CD Tolerance

The allowable deviation between measured and simulated CD on gauges. Gauges that cannot be resolved to within the tolerance are not included in the common process window plot. The default is 10%.

Dose Latitude

The extent of the common process window around the dose axis. The default is 5%.

CD Measurement

The type of sgd data used to plot the common process window.

  • Sim — Use simulated resist contours.

  • Meas — Use resist measurement CDs.

  • Sim Etch — Use simulated etch contours.

  • Meas Etch — Use etch measurement CDs.

  • CTR — Use CTR model.

RMSE Error Tol

Defines the threshold for outliers. The common process window is calculated with a two-step fitting process. Gauges whose CD at a given dose and focus vary more than the RMSE value are not included in the second part of the calculation. The default is 5.

Nominal

Specify a focus or dose to use in conjunction with Center. Dose is in percentage. (100% is 1.) Focus is in microns.

Changing the nominal focus and dose affects the inscribed shape.

Max DOF/Max Latitude

Specify a tolerance for depth of focus and exposure latitude. Both default to 0, allowing the full range of latitude and focus to be explored.

Use fixed DOF

When set, DOF is held steady at the specified value.

Weight & Filter tab

Settings used to remove outliers from the CD model.

Weight subtab

When enabled, a Gaussian weighting function is applied to the CD model data points centered on best dose and focus values by default.

  • Width Factor — Width of weighting function. Smaller values decrease the influence of data points that are furthest from the focus and dose centers. Default is 1.0.

  • Focus Center — Specify a focus center to use when applying weighting. The default behavior is to not center the focus.

  • Dose Center — Specify a dose center to use when applying weighting. The default behavior is to not center the dose.

Prefilter subtab

CD model data points are filtered before fitting the CD model polynomial. This is on by default.

  • Percent CD— Prefilter type that filters data points less than and greater than the specified percentage.

  • Min/Max CD— Prefilter type that filters data points less than and greater than the specified Min CD and Max CD values.

  • Min Dose and Max Dose— Filter data points below a minimum dose and above a maximum dose. Default is off.

  • Min Focus and Max Focus— Filter data points below a minimum focus and above a maximum focus. Default is off.

  • Prefilter data points— Specify data points to filter from CD model. Given in detail in “Remove data points from CD model at specific dose and focus conditions” of “Usage Notes” in “Table 1”.

Filter subtab

CD model data points are filtered after fitting the CD model polynomial. Default is enabled.

  • RMSE Multiplier — Filter data points from the CD model that are greater than a multiple of the RMSE. Default filter type is RMSE Multiplier with a value of 2.0.

  • Max Residual CD — Specify a residual CD value for removing data points from the CD model. Default is 1000.0 nm when on.

Coeffs tab

Displays coefficient values for the polynomial used for the CD model.

By default, the Process Window plot uses two-pass data filtering to improve model robustness. The first pass removes data based on a percentage of target CD. The second pass fits the 13-term or 8-term polynomial model and then removes points whose residuals exceed 2x RMSE or an absolute CD value. The model is then re-fit.

Of particular note, the window is truncated at the dose and focus limits and does not extrapolate past the available data.

The Model Stat columns in the Gauge Table tab include the following data:

  • Converged — Whether the gauge is included in the plot. Possible values are:

    • Converged — The model converged, and for some range of defocus the upper dose curve is greater than the lower dose curve.

    • Not Converged — Too few data points remain after filtering to fit a model. For a poly13 model, a curve requires 4 dose conditions, 5 focus conditions, and at least 14 clean points after filtering. For a poly8 model, a curve requires 3 dose conditions, 3 focus conditions, and at least 9 clean points after filtering.

    • Bad Fit — The fit is non-physical; for example, the slope of the dose is inconsistent.

    • No Window — There is no open window in the focus range.

    • Blank — Gauge is disabled.

  • Points — How many sampling points contributed to the curve.

  • Chi2 and Reduced Chi2 — Refer to Χ2. Can be used to identify a gauge with outlier data even when RMSE is small.

  • Rmse — Root mean square error. This value is higher for poor-quality data.

  • Stdev — Standard deviation.

  • R2 — Coefficient of determination.

  • Mean error and Mean abs error — Mean error and mean absolute error.
