Creating an EUV Optical Model File

An initial EUV optical model is most effectively created by using the Optical Model Tool in Calibre WORKbench.

The flare model you created in “Creating a Flare Model File” only models the flare effects. You also need to create an EUV optical model to simulate the EUV source.

Prerequisites

  • Calibre WORKbench GUI invoked

  • Information about your EUV illumination source

Procedure

  1. Open the Optical Model Tool (Optics > Optical Model).
  2. Create an optical model with the following characteristics:

    • Main tab

      • Model Name — Specify a name for the EUV optical model.

      • Wavelength — Specify a value of 13.5 (change from the default of 193 nm).

    • Kernels/Image tab

      • Optical Diameter — Specify a value greater than 4λ/NA. Siemens EDA recommends a value between 10 to 20 * (lambda/NA)/(1+sigma_max), which is between 0.21 and 0.43 for current EUV scanners.

      • Kernel Grid — Specify a value of approximately 0.003.

      • First image plane (also known as “defocus start”) — Specify 0.

    • Film Stack tab

      • Beam focus position — Specify 0.

      • Fill in the film information relevant to your film stack.

    • Pupil tab

      Choose the Coefficients Normalization for your Zernike normalization using the radio button. Peak specifies peak-to-valley normalization. RMS specifies the default RMS normalization.

    • Misc tab

      In the text field, add the pupilObscuration keyword with a sigma value in the center of the objective pupil corresponding to zero transmission. Valid ranges are 0 <= sigma <=1.0. However, note that a setting of 0 still blocks the center of the pupil.

  3. Click the Generate button.

Results

The Optical Model tool generates an optical directory that best represents the EUV illumination source.
