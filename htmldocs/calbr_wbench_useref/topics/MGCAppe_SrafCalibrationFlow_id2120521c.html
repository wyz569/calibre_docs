SRAF Calibration Flow

“SRAF printing” models can be produced using the Calibre WORKbench tool. SRAF printing models predict which SRAFs start printing during chip fabrication. These models are used during simulation to produce better OPC recipes.

You perform SRAF calibration using one of two methods:

  • Create a CTR model as a reference model and calibrate it in CM1 Center. This task is described in “Calibrating CTR Models With SRAF Gauges”.

    Figure 1. SRAF Calibration Phase 1: CTR Flow

  • Use Calibre nmModelflow to calibrate the model instead of CM1 Center. Using Calibre nmModelflow also allows you to calibrate Zplanes models.
