pupilPolarization

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Defines the polarized pupil transfer functions.

Usage

pupilPolarization { (none) | S | P | FILE path/filename.jpf | INLINE access_id }

Arguments

(none)

A required argument specifying no argument and causes pupilPolarization to be omitted.

If the J matrix is the unitary matrix for all points in the pupil, it is evident that there is no polarization change when going through the lens. This is the default setting.

S

A required argument specifying complete transmission of only the s-polarized plane waves.

If Jss = 1 and Jsp = Jps = Jpp = 0 for all points in the pupil, the pupil acts as an s-polarization filter, by allowing complete transmission of the s-polarized plane waves and eliminating p-polarized waves.

P

A required argument specifying complete transmission of only the p-polarized plane waves.

If Jpp = 1 and Jss = Jsp = Jps = 0 for all points in the pupil, the pupil acts as a p-polarization filter, by allowing complete transmission of the p-polarized plane waves and eliminating s-polarized waves.

FILE path/filename.jpf

A required argument specifying a Jones pupil file for modeling pupil polarization.

The general case of modeling pupil polarization requires a separate file that contains complete information about the Jones pupils. This functionality is similar to the source maps that model non-idealities in the illumination intensity. See the section “Jones Pupil Data File” for the specification of the Jones pupil file.

The file filename.jpf needs to exist in one of the following locations:

  • In the specified relative path to the file, starting from the current working directory

  • In the specified relative path to the file, starting from the Optical file location

  • As an absolute path

INLINE access_id

A required argument pointing to an inline pupilPolarization file defined with the pupilSampling keyword with the access_id specified in that inline file.

The Jones matrix J is given either in s-p or x-y coordinate systems as shown in the equations above. Each point in the pupil requires a set of eight numbers (the four elements of the J matrix are complex) to describe the rotation of the electric field polarization for the respective plane wave. (Recall that the coordinates of a point in the pupil correspond to direction cosines for the k-vector of the plane wave.)

In order to model the polarization transformations rigorously via the J matrix, eight separate pupil functions need to be provided for the optical simulation. These pupil functions are also known as “Jones pupils” or “Jones pupil data.”
