VT5 Resist Model Creation

A resist model is a mathematical function that uses the aerial image (the output of the optical model) as input to predict the threshold at which edges print.

Calibre WORKbench uses the VT5 Center tools to characterize the aerial image in terms of properties such as intensity, slope, and aerial image factor. It generates the resist model by optimizing the function’s polynomial coefficients to best fit the empirical measurements from a test chip.
