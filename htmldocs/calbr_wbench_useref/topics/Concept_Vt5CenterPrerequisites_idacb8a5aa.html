VT5 Center Prerequisites

VT5 Center requires certain inputs to perform the experiments. If any inputs are missing, it exits with an error.

You will need:

  • A test pattern GDS or OASIS file, either created by you or generated using the Calibre WORKbench Generate Test Pattern tool, described in “Test Pattern Creation”.

  • One or more sample test pattern spreadsheets based on that test pattern, which must have measurements filled in

  • An initial Calibre WORKbench setup (.in) file, containing company-specific information for your manufacturing process, as well as paths to:

    • A nominal (initial) optical or maskwriter model

    • A nominal (initial) resist model
