Comparing the Simulated Image to the Contour

In this task, you will run a simulated print image and compare it to the aligned SEM contour.

Prerequisites

  • Completed the task, “Extracting the SEM Contour”

Procedure

  1. Select the SEM outline for the first test structure you are using to evaluate your model.
  2. Specify which image to simulate using the Reference buttons in the Alignment portion of the SEM tab:

    • If your setup file references a VT5 model, select DPI.

    • If your setup file references any other type of resist model, select Aerial.

  3. Click Align.

    The SEMoverlay tool calculates a simulated contour and aligns the green SEM contour with it. Note that the tool does not display the simulated image. To view the simulation, you must select the test structure and click on the Contour button in the Calibre WORKbench toolbar.
