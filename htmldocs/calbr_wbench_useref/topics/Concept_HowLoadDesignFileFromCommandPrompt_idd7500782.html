How to Load a Design File From the Command Prompt

Load a design file using one of the following configurations: 
calibrewb filename

Use the -g argument to open the design at a specific cell in the design. 

calibrewb filename -l layerprop_file
calibrewb filename -dl default_layer_propfile

Use the -l (apply layer properties to only the specified file) and -dl (apply layer properties as defaults for any loaded file) arguments to customize the loaded design with the settings in the layer properties file.

If you specify the -l and -dl options in the same invocation with a layout, each option specified successively overrides like values of the previous option. The order of precedence is:

  1. The default file (the standard .calibrewb_workspace/layerprops as specified by -dl).

  2. Calibre detects the presence of a <layoutfile>.layerprops file.

  3. A file specified by -l.

The default and <layoutfile>.layerprops files are always searched for and used on all loads in the session; -l is only checked during invocation.

calibrewb -b bookmark

If you have defined Calibre WORKbench design file bookmarks in a previous session, you can also use the -b option to open the design file at the bookmark location.
