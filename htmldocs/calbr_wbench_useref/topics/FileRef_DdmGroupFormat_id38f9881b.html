DDM Group Format

3D Mask Modeling Reference

Input for: Model-Layer File Format

Collects multiple dense DDM models into a single group. This group can be calibrated inside modelflow_v2. Each group should include two or more DDM models, since the DDM group function extrapolates between the DDM models in a group.

A DDM group is contained inside a ddm_groupM statement in the model-layer file.

When a DDM group is loaded, the DDM model obtained by interpolating the group at its ddmgroupparam value is applied to each layer the group is lined to.

Format

ddm_groupM name { filename | ‘{’
	[ddmgroupparam value]
	paramtype slope 
	ddm ddmparval ddmfile
	...
	ddm ddmparval ddmfile
‘}’

Parameters

ddm_groupM name

A required parameter that defines a DDM group definition block. M can range from 0 to 4 inclusive, meaning that a maximum of five DDM groups are supported in a model-layer file.

filename

Either an external filename containing a DDM group definition or an inline definition in braces ({ }) is required to define a DDM group.

ddmgroupparam value

An optional parameter that specifies the nominal value for the DDM group parameter. value cannot be less than the minimum or maximum value of the individual DDM models in the group. By default, value is set to the center of the interval (minimum + maximum)/2.

paramtype slope

A required parameter that specifies the type of DDM group, which sets the interpolation routine to be used. Currently, only “slope” (using the stackslope parameter in the DDM file) is supported.

ddm ddmparval ddmfile

A required parameter that sets a parameter value and associated DDM library file to use when the slope is equal to that value. Each ddmparval value within the same group must be unique, but different DDM groups can have the same ddmparval value.

Examples

This code fragment from a model-layer declares three DDM groups.

  • Group 0 has two ddm libraries and an implied group parameter value of 87 (calculated as (85+89)/2).

  • Group 1 loads an ASCII file (ddmstack.txt) containing unknown DDM group information.

  • Group 2 explicitly sets the group parameter to 86, even though the three libraries inside would give a group parameter value of 84.5 (calculated as (80+89)/2).

    ddm_group0 DDM87 {
	paramtype slope
	ddm 85 ddm1.mod
	ddm 89 ddm2.mod
}
ddm_group1 ddm4 ddmstack.txt
ddm_group2 ddm7.mod {
	ddmgroupparam 86
	paramtype slope
	ddm 80 ddm0.mod
	ddm 84 ddm4.mod
	ddm 89 ddm9.mod
}
