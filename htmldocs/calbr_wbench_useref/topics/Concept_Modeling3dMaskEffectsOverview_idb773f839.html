Modeling 3D Mask Effects Overview

Calibre Tools implement rigorous mask simulation modeling using the Calibre MaskSim tool. Calibre MaskSim is an electromagnetic field (EMF) solver that uses a Finite Difference Time Domain (FDTD) method to compute simulated waves in a custom constructed mask.

The Calibre MaskSim tool is designed to address the need for accurate simulations for instances of thicker masks (such as in phase shift masks) and/or smaller wavelengths (where edge interactions can change the results) as shown in Figure 1.

Figure 1. Diffraction Depends on Illumination

Two components are needed to model 3D mask effects; an HHA-enabled optical model and a DDM model. Both of these models are only intended for use with dense modeling.
