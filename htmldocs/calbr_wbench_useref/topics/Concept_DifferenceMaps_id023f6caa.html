Difference Maps

The Difference utility (File > Open Maps Difference) allows you to see the differences between two source maps in order to assess the quality of a fitmap result versus an input illumination file. It can be used on two unpolarized source map files or two polarized source map files. If source map files with different pixel step sizes are compared, the source with the smaller pixel step size is interpolated to the larger pixel step size before determining the difference.
