pellicle

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Includes mask pellicle properties in the optical model.

Usage

pellicle thickness re_ref_index [ im_ref_index ]

Arguments

thickness

A required argument specifying the pellicle thickness in microns.

re_ref_index

A required argument specifying the real refractive index of the mask pellicle.

im_ref_index

An optional argument specifying the imaginary refractive index of the mask pellicle. The default value is 0.

Description

An optional pupil parameter that allows explicit modeling of a mask pellicle. The default behavior is no mask pellicle modeling.

A mask pellicle introduces apodization at the pupil that can be modeled analytically. Only simple pellicles that consist of a single film with uniform thickness and refractive index can be modeled.
