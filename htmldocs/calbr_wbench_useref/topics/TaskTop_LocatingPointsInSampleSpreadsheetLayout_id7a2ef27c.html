Locating Points in the Sample Spreadsheet and Layout

Drawing a banding box around site points in any graph in the Bivariate Plots tab selects the corresponding points as a group in all graphs. The grouped points are highlighted across each graph, which can show relationships based on the graph type.

To see the location of specific site points in the sample spreadsheet and layout viewer:

Procedure

  1. In the Bivariate Plots tab, optionally zoom by drawing a bounding box with the right mouse button (draw from northwest to southeast to zoom in; draw from southeast to northwest to zoom out).
  2. Left-click to select gauge points (shift-click to select multiple gauge points) in any Bivariate Plots tab. You can also draw a bounding box to select all points inside the box.
  3. Click the Mark button.

Results

Calibre WORKbench does the following:

  • Creates a layer for the marked points (100 by default) for the IPSE text files and adds them to the layout viewer. (You may have to zoom in to see the sites.)

  • Opens the Calibre WORKbench VT5 Center with the sample spreadsheet(s) loaded. Any selected log file points in the Bivariate Plots have their corresponding rows selected in the Sample tab.

    Note:

    Contour points cannot be marked and selected; only gauge points can be viewed in this manner.
