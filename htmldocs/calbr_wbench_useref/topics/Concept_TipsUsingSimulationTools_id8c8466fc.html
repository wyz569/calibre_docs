Tips For Using Simulation Tools

The Calibre LITHOview and Calibre WORKbench simulation tools have some common characteristics that apply to their use.

Guidelines for Simulation Frames

Tools that use simulation frames only calculate results inside of a selection frame (click the Select mode button, then left-click and hold, then drag to create a bounding box).

The maximum size of a simulation frame is 100x100 um. Attempting to simulate on an area larger than that displays an error dialog box.

Multiple simulation frames accumulate within the display. A frame can be selected for deletion (Delete key), or Raised and Lowered (right mouse button).

Guidelines for Generated Layers

The layers generated by the PRINTimage and ILT tools add a layer bump value to the OPC layer they operate on, writing the results on that layer number. You change the layer bump value in the Options pane. Access this dialog in the RET Flow Tool by clicking the Options button beneath the appropriate tool button.

Layers created by these tools are saved if you save the design file after running them.

Because the PRINTimage and ILT tools use a simulation frame, the maximum size of the information on the new layer is 100 x 100 um.
