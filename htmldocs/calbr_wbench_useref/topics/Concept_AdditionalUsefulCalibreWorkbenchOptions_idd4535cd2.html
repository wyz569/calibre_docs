Additional Useful Calibre WORKbench Options

Other options perform specific functions that affect the behavior of Calibre WORKbench on startup.
Table 1. Invocation Options

Option

Description

-shell

Specifies to use the shell interface instead of the GUI. To interact with the program you must type commands into the shell.

-a command

Runs a single batch command as if you had typed it from inside the shell.

-o overlayfile

Loads a previously-created overlay file (.ovly) during invocation. Refer to the Calibre MDPview User’s and Reference Manual for information on overlays.

-gui

Evaluates the script with the Tk environment and application GUI loaded and available for use by the script. Exits after processing.

‑HC

Enables High-Capacity (HC) mode for loading large layouts. Refer to the Calibre DESIGNrev Layout Viewer User’s Manual for information on High Capacity mode.

-threads number_of_threads

Specifies the number of threads to use in parallel macros. By default, the system determines and uses the maximum number of CPUs automatically.

-tb classic | origin | ignore | extent

Specifies how the extent of a text box affects the calculation of a cell bounding box. Used for cell library editing.

-help

-h

Prints command usage.

-version

Print the version number of the application.

-wait time

By default, a Calibre tool waits indefinitely for a license to become available. With the -wait option, a tool waits for any appropriate license for the number of minutes specified by the time argument; the tool then exits if no licenses become available before time expires.

Licenses can be acquired from this product or from any of the tool’s parent products. The order of precedence, from lowest to highest, is:

  1. Calibre® DESIGNrev™

  2. Calibre® MDPview™

  3. Calibre® LITHOview™

  4. Calibre® WORKbench™

License checking occurs in two stages. The first pass checks for a qualified license without a wait. If no license is acquired, a second pass is performed and a wait is done on the first existing qualified license.

For example, Calibre MDPview can use a Calibre MDPview, Calibre LITHOview, or Calibre WORKbench license, if one is available, but it cannot use a Calibre DESIGNrev license. Additionally, if you do not have a Calibre MDPview license, Calibre MDPview waits for a Calibre LITHOview or Calibre WORKbench license if one exists.

-nowait

Indicates to not wait if all licenses are unavailable. The layout viewer exits if it cannot get a license.
