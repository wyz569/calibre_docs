Modifying the Test Pattern With IPSETX

You can “repair” the holes (places where the test pattern does not cover the layout) in your test pattern using the Image Parameter Space Explorer Test pattern eXtend (also known as the “Testpattern X”) tool.

Procedure

  1. Once you have determined parts of your layout that do not have coverage in the test pattern, unselect the checkbox for the row containing the calibration data (the test pattern, usually). This will leave you with just the graph of the IPSE data from your design showing. For example, in Figure 1, line 1 is disabled.
  2. Select flyer points in any graph known to be lying outside the test pattern area by holding down the left mouse button and drawing a box around the points. You can also hold down the Shift key while clicking to highlight points in different areas of the design.
  3. When you have finished selecting flyer points, choose the Tools > TestpatternX menu item.
    Figure 1. Selecting Points for IPSETX

    The TestpatternX Settings dialog box appears (Figure 2):

    Figure 2. Testpattern X Settings Dialog Box

  4. Enter information on the output pattern, associated sample spreadsheet, and options for the markers.
    Table 1. Testpattern X Dialog Box Fields

    Field

    Description

    Suggested Value/Type

    Output Repair File

    Specifies the pathname to the file to be saved.

    Include a path if you do not want to write a file to the current directory.

    Output Format

    Specifies whether the output format is in GDS or OASIS.

    As appropriate

    Test Pattern Columns

    Specifies the number of columns across for test pattern clips.

    15 (default)

    Clips Radius (um)

    Specifies how much of a radius area around the measurement point to save to the clip.

    Varies

    Get Optical Diameter from Optical Model

    Selecting this check box uses the optical diameter from the optical model.

    On

    SEM Alignments

    Selecting this check box adds SEM alignment markers (Figure 3) to the output.

    As needed

    Output SS File

    Name the corresponding sample spreadsheet file.

    Include a path if you do not want to write a file to the current directory.

    Marker Half Width

    Sets the shape of the marker for each pattern using these fields, in microns.

    0.005

    Marker Half Length

    0.0025

    Figure 3. SEM Alignment Markers Added

  5. Click OK.
  6. The new test pattern can now be loaded and merged with the existing test pattern file to create the extended mask for testing purposes.
