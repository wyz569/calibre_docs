tepVerification

Test Pattern Commands

Generates a verification test pattern and associated spreadsheet file based on the specifications supplied in the test pattern file tpffilename. If no filename is specified, it generates a test pattern using the default parameter values, six structures in each column, and the width and space varying from 0.180 um up to 0.8 um.

Usage

tepVerification [tpffilename]

Description

The default parameter values are as follows:

  • topcellname: TOP

  • makers: none

  • border: 0

  • output files: pattern.gds + pattern.ss

  • dbgrid: 0.001um

  • designgrid: 0.001um

Arguments

tpffilename

The name of the test pattern file, specified as either an absolute path or relative to the directory from which you invoke the Calibre WORKbench application. For a complete description of the test pattern file format, refer to “Test Pattern File Format”.

Examples

Example 1: a verification test pattern using built-in default parameters.

% tepVerification

Example 2: a verification test pattern using parameters specified by TPF file mytpf.tpf.

% tepVerification mytpf.tpf
