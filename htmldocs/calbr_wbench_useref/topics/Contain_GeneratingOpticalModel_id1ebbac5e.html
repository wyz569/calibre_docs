Generating the Optical Model

Optical model generation is the result of selecting and configuring multiple options that define the characteristics of the optical model. Your choices must be appropriate in order to match the characteristics of your optical system (wavelength, NA, source, film stack, and so on).
Tip

The Optical File: When you generate the model, the Calibre WORKbench application creates an optical model directory, and populates it with the file named “Optical”, which stores the parameters, and a collection of .omf files containing the mathematical calculations (lookup tables).

  • For simple cases, the Optical file is the only file you need to recreate the entire model directory (see “Method 2 - Use the opticsgen Batch Command”). There are several exceptions to this when the Optical file references other files, such as measured source map file, Jones Pupil file, and scanner focus blur files.If you need to save file space, you can delete the .omf files.
  • If you need to share an optical model with another user, you can copy the Optical file and any files it references.
