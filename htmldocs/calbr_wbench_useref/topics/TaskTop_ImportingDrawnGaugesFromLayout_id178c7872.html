Importing Drawn Gauges From a Layout

An alternative option to creating gauge objects from sample file spreadsheets is to import them directly from a layout.

Prerequisites

  • The test pattern must have its gauges placed on a separate layer.

  • Gauges must be drawn paths (created externally or drawn manually using the Calibre WORKbench Path tool), markers exported by CM1 Center’s Export Marker functionality, or created using the $L create wire command, described in the Calibre DESIGNrev Reference Manual.

Procedure

  1. Invoke Calibre WORKbench, and open the test pattern with the Convert Paths to Polygons check box unselected.
  2. Invoke CM1 Center (Litho > CM1 Center menu item in Calibre WORKbench).
  3. In CM1 Center, select the Gauges > Import Markers menu item.
  4. In the dialog box that appears, fill in the text fields as follows:

    • Marker Layer — Specify the layer containing the drawn gauges.

    • Extra Search (%) — Set the percent of the gauge length to search for additional structures not crossed by the gauge. Structures found are used to fill in the Drawn and Other fields. The recommended value is 50%.

    • Force Horizontal/Vertical Gauges — Set this check box to force the imported gauges to be orthogonal (x1=x2 or y1=y2).

      This option is recommended, because angled gauges will have their coordinates recorded without measuring the drawn mask feature. The gauge will be imported in a disabled state.

    • Drawn layer — Specify the layer with the test structures.

  5. Click OK to import the markers as gauges.

Results

CM1 Center adds the new gauges to the current gauge tab. It names the structure “iso” if the gauge is over a line, and “inv iso” if the gauge is over a space.
