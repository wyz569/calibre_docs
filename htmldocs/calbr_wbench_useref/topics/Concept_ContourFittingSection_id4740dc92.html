Contour Fitting Section

The Contour Fitting section is only available when contour calibration was used as part of the model development process.

It displays the following graphs:

  • Contour difference values on a per-site basis (different blocks have a different color)

  • Contour differences on a per-block basis

  • Contour difference distribution by the amount of error

    Figure 1. Analysis Graph (Contour Fitting Section)

To to see the EPE locations for gauges on the Contour Diff tab, select one or more points (A) or one of the contour block bars (B) and click one of the following buttons:

  • Mark Layout highlights the points on the specified layer (layer 501 by default). You will see the error leader lines when you Mark and View the layout.

  • View Layout jumps to the position in the open layout. You must have the correct layout file open and be zoomed to a size that will show you meaningful information. Subsequent clicks of the View Layout button moves to the next selected point if you have selected a group of points.

    Tip

    Use the Contour Fitting tab graphs to select points with the highest EPE values and view them in the layout. Clicking View Layout will jump to the relevant part of the input contour.

  • Selecting a contour block, then clicking View Layout pans the window to the location of the contour block. To find the exact polygons that are used for the selected contour block, select one or more polygons on the clipping layer (cliplayer argument in the Contour tab), then select Object > Properties. If the selected polygon is part of the block, the Object Properties dialog box will show it in its Polygon tab.

    If the clip polygon has user properties defined and they include an attribute “CCAL_CLIP”, then it's value (for example, BLOCK_2 in Figure 2) will be used as the contour block's identifier during calibration, which will in turn show up in the log file and the analyzer.
