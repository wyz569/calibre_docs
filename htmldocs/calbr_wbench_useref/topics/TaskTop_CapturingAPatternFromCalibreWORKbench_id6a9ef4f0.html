Capturing a Pattern from Calibre WORKbench

Use the Calibre Pattern Generator to copy a pattern from Calibre WORKbench and save it to a pattern library.

Prerequisites

  • A layout file loaded into Calibre WORKbench.

Procedure

  1. Zoom into the feature of interest.
  2. Click Clip in the Layout Viewer Toolbar to activate interactive clip mode.
  3. Draw a bounding box around the features of interest.

    Using the left mouse button, draw a clip box around the feature to copy. For more information about clips, refer to “Creating and Working with User-Defined Clips” in the Calibre DESIGNrev Layout Viewer User’s Manual.



    When complete, the created clip displays a name that can be viewed in the Clips tab.

  4. Open the Calibre Pattern Generator from the Litho > Calibre Pattern Generator menu item.
  5. Select the Pattern Creator (PC) tool by clicking the Tools icon at the top right and selecting (PC) Pattern Creator in the menu.
  6. Select or create a pattern library to save the pattern to.
  7. Next to the library name, click the + icon to add a pattern.


  8. Add the clip to the pattern by clicking the Capture Pattern from CWB icon next to the library name, then clicking the Capture using layout clips icon.


    The clip appears in the Pattern Viewer. The Pattern Properties Table and Layer Properties Table populate information from the copied clip.

  9. As needed, adjust the values in the Pattern Properties and Layer Properties tables.
  10. Save the pattern library and close Calibre Pattern Generator.
