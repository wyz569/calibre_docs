SEMSuite User Guide

Manual extraction of useful data from CD SEM images can be a time-consuming process open to human error. The SEMSuite toolset provides the following tools to reduce the task time and human error involved with processing CD SEM images:

  • The Raw Data Filtering (RDF) tool processes CD SEM image and data files to produce a gauge file.

  • The Contour Data Flow (CDF) tool extracts contours from CD SEM images to produce a Sites CSV file.
