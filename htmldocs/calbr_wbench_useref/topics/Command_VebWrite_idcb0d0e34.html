veb_write

CM1 and VEB Commands

Related Formats: Variable Etch Bias (VEB) Model File Format

Outputs a VEB object to a string and optionally, to a file if one is specified.

Usage

veb_write vebin [filename]

Arguments

vebin

A required parameter that specifies the input VEB Tcl object.

filename

A required parameter that specifies the output file.

Examples

#write object to a string
set str [veb_write $vebobj]
puts stdout $str
#write object to a file
veb_write $vebobj “etch.mod”
