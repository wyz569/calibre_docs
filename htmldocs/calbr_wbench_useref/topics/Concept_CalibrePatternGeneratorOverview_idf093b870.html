Calibre Pattern Generator Overview

Test pattern generation is the first step in the litho modeling flow where a test layout is created with pre-determined geometries required by the RET modeling process. The Calibre Pattern Generator (CPG) is a flexible tool for creating basic and complex patterns for test layouts.

The Calibre Pattern Generator creates new patterns using several tools in the following workflow:



  1. The Pattern Creator (PC) tool creates patterns that are saved in a pattern library.

  2. The Layout Generator (LG) tool combines the patterns from a pattern library with labels and markers and saves it as a layout configuration file. The Layout Generator creates layout and gauge files from the layout configuration file.
