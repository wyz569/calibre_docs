Pre-loading Model Generation Data

To pre-load the model generation data, you load the optical model, spreadsheet data, and the setup file into Calibre WORKbench.

Prerequisites

  • Test pattern

  • Sample spreadsheet file with measurements filled in

  • Setup file

Procedure

  1. Invoke Calibre WORKbench and open the design file containing the test pattern (File > Open Layout).
  2. Invoke VT5 Center by selecting the Litho > VT5 Center menu option from the Calibre WORKbench main window.
  3. In VT5 Center, open the sample spreadsheet (by selecting Sample > Open SS File) and setup file (Basic tab, Setup field).

    Verify that the sample spreadsheet file is open in the Sample, in tab.

  4. In the setup file, make sure the siteinfo keyword is set as follows:

    • Type is set to SAMPLE. (Note that this is case-sensitive.)1

    • The -num parameter must be set such that it meets the resist criteria multiplier. This must always remain true even if you change the Sample Spacing setting (on the VT5 tab).

    • The -center parameter must be set to the middle of -num so half the points are on each side of the site.

  5. Click the Layer tab and verify the layer containing the test polygons is defined as the opc layer, and the correct GDS or OASIS layer number is shown.
