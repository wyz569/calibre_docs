Using VT5 Center as a Sample Spreadsheet Editor

The VT5 Center tool is a multi-tab, multifunction utility for creating models. The Sample, in tab is used to manage and modify sample spreadsheet file contents.

Use the editing capabilities in this tab in the following situations:

  • Entering data from the “Taking Measurements on a Test Pattern” procedure

  • Inspecting site locations

Prerequisites

  • A previously-generated sample spreadsheet file

  • The design file associated with the sample spreadsheet file

Procedure

  1. In Calibre WORKbench, invoke the Litho > VT5 menu item.
  2. Load a sample spreadsheet file (Sample > Open SS File).
  3. Modify the sample spreadsheet file with any of the following methods:
    Table 1. VT5 Center, Sample Tab Activities

    Action

    How To Do It

    Change a single value

    Click a field and type in it.

    Select multiple cells

    • Shift-click: Selects all cells between the current selection and the clicked cell.

    • Ctrl-click: Adds the clicked cell to the current selection.

    • Clicking a column header selects the whole column.

    Select multiple rows by type

    Click the Select button to bring up a dialog box that features all structure names.

    Add/remove a site marker

    Mark/Unmark buttons.

    Assign Multiple Values

    Select multiple cells, then:

    • Edit > Set Value to set a single value for all selected cells.

    or:

    • Edit > Copy DrawnMask to Measurement to copy the target (Drawn Mask) fields to the Measured field. Note that this will create a model that matches the data exactly.

      Use this option only when you are familiarizing yourself with VT5 Center.

    or:

    • Edit > Assign Simulation to Measurement to load the simulation values from a selected results log.

      Use this option to compare values from process window spreadsheet values.

    Show/Hide rows

    Click the + sign for a row (or multiple rows) and click the Enable/Disable buttons.
