Stochastic Model File Format

CM1 Model Supplement

A stochastic model file describes key characteristics of the EUV process to model uncertainty factors during simulations.

Format

version number
physical_dose dose_value
diffusion_length diff_value
[resist_efficiency resist_value]

Parameters

version number

An optional argument that sets the version number of the stochastic model. The default is 2. Version 2 supports stochastic failure rate calculations in Calibre nmModelflow and random contour generation in Calibre OPCverify.

physical_dose dose_value

A required argument that specifies the dose used on the scanner.

diffusion_length diff_value

A required parameter that specifies the lumped diffusion for the photonic process (acid and secondary electron effects).

resist_efficiency resist_value

An optional parameter that specifies the photon absorption efficiency in the resist. The default is 1.

Examples

physical_dose 40.0
diffusion_length 5.0
resist_efficiency 1.0
