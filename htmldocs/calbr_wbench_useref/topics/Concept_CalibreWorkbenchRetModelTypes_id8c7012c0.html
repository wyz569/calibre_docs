Calibre WORKbench RET Model Types

Calibre WORKbench creates multiple kinds of models.

Optical models that simulate the effects of:

  • Different kinds of source shapes

  • Film stack refraction effects

  • Immersion lithography

  • Jones pupils

  • Pellicle films

  • Focusblur process

  • EUV conditions

  • DDM models

  • Topography 3D models

Resist and etch models that simulate the pattern transfer from the resist to the underlying films on the wafer. Two model types are supported by Calibre WORKbench:

  • Sparse models, which use site-based sampling (CTR, VTR-E, and VT5).

    VT5 models include separate polynomials to model resist and etch effects on post-etch data.

  • Dense models, which use grid-based sampling and separate resist effects from etch effects (CM1/VEB). Dense simulation models can be used as follows:

    • Incorporating resist and etch effects, when the measurement data is obtained post-etch to create a CM1 model.

    • Incorporating resist effects only, when the measurement data is obtained post-develop to create a CM1 model.

    • Independently incorporating resist and etch by using two sets of measurements; use the post-develop data to create a CM1 model and use the post-etch data to create an independent Variable Etch Bias (VEB) model.

    • Direct VEB models incorporate exposure, resist and etch effects using measurement data obtained after etch, bypassing the CM1 and optical models.
