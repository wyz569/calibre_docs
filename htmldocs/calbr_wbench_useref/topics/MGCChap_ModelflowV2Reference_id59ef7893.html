modelflow_v2 Reference

The modelflow_v2 tool (accessed as a Calibre Tools batch command) optimizes optical, resist, and etch model parameters.

Unlike the original modelflow (version 1) tool, which involves site-based simulations and supports VT5 resist models, the modelflow_v2 tool is gauge-based and supports grid-based resist models of type cm1. It contains the following components:

  • multi-parameters optimizer

  • cutline- and grid-based aerial image simulators

  • grid-based resist simulator

  • edge-based etch simulator

  • polygon width and space measurer

This chapter specifically covers the automated model creation batch command modelflow_v2. For other commands related to CM1 and VEB models, see the “CM1 and VEB Model Reference” chapter.
