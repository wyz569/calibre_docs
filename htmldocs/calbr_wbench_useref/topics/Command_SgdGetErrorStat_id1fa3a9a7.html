sgd_get_error_stat

CM1 and VEB Commands

Computes error statistics for an input super gauge data object.

Usage

sgd_get_error_stat object [‑vebfit { absolute | relative }]
      [-gsfile file | -gsstring string]

Arguments

object

A required argument that specifies the input sgd Tcl object.

-vebfit

An optional keyword that tells the command to compute error statistics for etch data instead of resist (Default = compute error stat for resist). Accepted flags for this keyword are:

  • absolute – Computes etch error stat for CDs.

  • relative – Computes etch error stat for biases.

-gsfile file

An optional keyword that specifies an input group definition file (cannot be set if -gsstring is present).

-gsstring string

An optional keyword that specifies an input group definition string.

Description

Use the sgd_get_error_stat command to compute error statistics for the input sgd object. This command supports splitting gauges into groups. If group definitions are specified and all enabled gauges are split into groups (unique names in the column Group that are not “NA” or “‑1”), the command also reports group objectives as well as numbers of gauges that are within group tolerances.

Examples

set obj [sgd_create –in sample.sdg]
set resist [sgd_get_error_stat $obj -gsfile group.gsf]
set etch [sgd_get_error_stat $obj –vebfit absolute]
