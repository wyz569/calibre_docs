Best Practices for Simulation Tools

Best practices for simulation tools apply during aerial image simulation testing.

The following sections describe the requirements and best practices for performing simulations (the Map, PI, and Contour tools).

  • Requirements for Simulation Tools

  • Recommendations for Simulation Tools

Requirements for Simulation Tools

The items shown in the following table are required settings when using the simulation tools.

Table 1. Simulation Minimum Requirements

Requirement

Function

Notes

Setup file

Contains information used for simulation, including the location and names of optical and resist models.

Set by selecting Litho > RET Flow Tool v2.0, then opening the setup file or configuring a simulation session.

At least one optical file

Used to simulate the optical source.

Specified in the setup file.

One or more resist models

Used to simulate the reaction of the exposure film to the optical source.

Specified in the setup file.

A design file in the same scale

Used to run simulations on the design file to see how your design will likely look in silicon.

Open the file in Calibre WORKbench from the File menu.

Recommendations for Simulation Tools

  • In the RET Flow Tool v2.0, Layers field, make sure that the layers match the layers in the active design.

  • For the dprintiml command, hoodpix should be a power of 2 multiple of -simpixsz.
