Symmetrizing a Source Map Using the Source View Tool

When a source map is measured on an exposure scanner, it usually contains a small degree of asymmetry and some measurement noise. Since OPC models should generally have x-axis reflection, y-axis reflection, and often also 90-degree rotation symmetry, the measured source map has to be adjusted to achieve symmetric OPC.

Source map symmetrization can be performed using the Source View tool.

Prerequisites

  • A source map file in the Calibre format (see “Source Map File Format” for more information)

Procedure

  1. Open the Source View tool by selecting Optics > Source View.
  2. Load the source map by selecting File > Open Source Map.
  3. In Model Options, click Mirror X, Mirror Y, or Mirror Diagonal. Each of these buttons brings up a dialog box with the following options:

    • Take Maximum — Symmetrizes by copying the greater value of a pixel or its opposite to both locations.

    • Take Average — Symmetrizes by averaging the values of a pixel and its opposite in both locations. Combining an active pixel (100%) and an inactive pixel (0%) location results in a 50% value for both locations.

    • Copy — Copies half of the source map to the other half.

      • Copy down/Copy up — The Mirror X button copies across the X-axis.

      • Copy right/Copy left — The Mirror Y button copies across the Y-axis.

      • Copy to lower right side/Copy to upper left side — The Mirror Diagonal button copies diagonally across the XY-axis in the selected direction.

  4. Save the source map under a new name.
