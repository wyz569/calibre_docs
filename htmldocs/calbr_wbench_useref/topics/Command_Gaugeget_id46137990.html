gaugeGet

CM1 and VEB Commands

Related Formats: Gauge Object Reference

Gets the specified data associated with the gauge in a gauge object.

Usage

gaugeGet -g gauge_object {[-size] | [-enabledcount] | [-enabledweight] | [-row index]}
[-enabled] [-name] [-coordinates] [-location] [-drawn] [-other] [-cds]
[-cdse] [-precision] [-weight] [-measstat] [-comment]

Arguments

-g gauge_object

A required argument specifying a gauge data Tcl object. If no other arguments are specified, returns all gauge data records/lines.

Note:

The following options (-size, -enabledcount, -enabledweight, and -row) are mutually exclusive and cannot be specified together.

-size

An optional argument that returns the total number of gauges in the input gaugedata object.

-enabledcount

An optional argument that returns the number of enabled gauges in the gauge object.

-enabledweight

An optional argument that returns the sum of the weights of enabled gauges in the gauge object.

-row index

If no other row-related arguments (shown in the table) are set, this optional which returns the entire gauge data record at the specified index. Valid values for index are integer numbers between 0 and total number of gauges-1 in the input gaugedata object.

Description

The arguments for gaugeGet are independently specifiable. They return the named column value from a gauge object row. They are ignored if -size, -enabledcount, or -enabledweight are specified, but if a -row argument is supplied, information is returned only for the specified -row index. If a -row argument is not specified, the entire gauge object’s data is returned for any enabled rows of the specified column(s).

Table 1. gaugeGet Row Field Arguments

Argument

Column(s) Returned

-enabled

Enabled or disabled flag.

-name

Structure name.

-coordinates

X1 Y1 X2 Y2 coordinates.

-location

The type of measurement (0 for CD width, 1 for CD space).

-drawn

Drawn CD measurement value in nanometers (nm).

-other

Conditional field value based on the structure type.

-cds

Both the measured resist and simulated resist CD data.

-cdse

Both the measured etch and simulated etch CD data.

-precision

Precision specification column.

-weight

Weight specification column.

-measstat

All three measured statistics output columns (stdev, count and 95%ci).

-comment

Comment field.

Examples

gaugeGet -g gobj ; # get all gaugedata
gaugeGet -g gobj -size ; # get total number of gauges
gaugeGet -g gobj -enabled ; # get number of enabled gauges
gaugeGet -g gobj -cds ; # get resist CD data for all gauges
gaugeGet -g gobj -row 2 ; # get gaugedata for row 2 
gaugeGet -g gobj -row 127 -cds ; # get resist CD data for row 127
gaugeGet -g gobj -row 0 -weight -comment ; # get weight and comment for row 0 
gaugeGet -g gobj -row 0 -cdse ; # get etch data for row 0 
gaugeGet -g gobj -row 2 -enabled ; # get enabled flag for row 2
