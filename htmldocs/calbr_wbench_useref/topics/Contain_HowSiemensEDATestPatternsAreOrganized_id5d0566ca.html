How Siemens EDA Test Patterns Are Organized

Siemens EDA test patterns follow a specific organization and schema.

Figure 1 shows a V11 test pattern, which places information on different layers depending on how the information will be used.

Table 1. Siemens EDA Test Pattern Design Layers

Item

Layer (Color)

Description

Test structures

0 (green)

These are the structures to be measured by the SEM tool.

Annotations/comments

1 (blue)

These should not be measured. The top of a test pattern shows the type, CD, and version; other comments include row and column labels for visually locating a feature.

SEM calibration markers

2 (yellow)

These are only available in V11/V12 test patterns and require you to select a SEM alignment style.

Set this as visible in the Layers tab so that they will print on the mask.

SEM alignment marker

0 (green)

The SEM alignment marker will print on the test pattern chip and can be used to align your SEM tool.

Test structure information

3 (red)

These text labels are only available in V11/V12 test patterns and show information on the test structure they cover. You may need to zoom in on a structure and turn off layer 0 to read the information.
