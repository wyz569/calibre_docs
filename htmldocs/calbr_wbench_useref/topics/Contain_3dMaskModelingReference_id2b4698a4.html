3D Mask Modeling Reference

3D Mask Modeling uses a number of separate file reference formats to define its input and output libraries.
Table 1. 3D Mask Modeling Libraries

Format

Description

DDM Library Input File Format

Defines the mask characteristics of a 1D edge.

Domain Decomposition Method (DDM) Model Output File

Describes the DDM output mask. This file is generated by masksim.

Corner Model Output Format

Describes the supplemental corner signal libraries generated by masksim that accompany a DDM output mask.

Cross Talk Model Output Format

Describes supplemental cross talk libraries generated by masksim that accompany a DDM output mask.

DDM Group Format

Collects multiple dense DDM models into a single group.
