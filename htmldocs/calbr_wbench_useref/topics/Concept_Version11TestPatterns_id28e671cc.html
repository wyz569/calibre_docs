Version 11 Test Patterns

Test Pattern V11 lets you control both width and space CD.

It is characterized by increased annotation capabilities and variable sampling dimensions. The variable sampling dimensions are based on the CD. Sampling is every 20 microns when CD > 200, every 10 microns when 50 < CD < 200, and every 5 microns when CD < 50. This test pattern is used for general resist modeling and provides the user with an alternative to Test Pattern V10, which is still available.

A detailed explanation of the measurement data for each geometry is provided on the pages that follow.

iso

Row 1 — Isolated line basic geometries. Width varies:

  • Col 1: width = CD(w)-10

  • Col 2: width = CD(w)

  • Col 3: width = CD(w)+10

  • Col 4: width = CD(w)+20

  • Col 5-12: exponential increases up to 1.2 microns wide.

iso_pad

Row 2 — Padded iso lines width = CD(w), spacing varies:

  • Col 1: space = CD(s)-10

  • Col 2: space = CD(s)

  • Col 3: space = CD(s)+10

  • Col 4: space = CD(s)+20

  • Col 5-12: exponential increases up to 1.2 microns apart.

inv_iso

Row 3 — 5x5 micron patch with center line removed. Center varies:

  • Col 1: space = CD(s)

  • Col 2: space = CD(s)+10

  • Col 3: space = CD(s)+20

  • Col 4: space = CD(s)+30

  • Col 5-12: exponential increases up to 1.2 microns apart.

pitch 1

Row 4 — Seven dense lines with fixed width = CD(w)-10. Spaces vary:

  • Col 1: space = CD(s)

  • Col 2: space = CD(s)+10

  • Col 3: space = CD(s)+20

  • Col 4: space = CD(s)+30

  • Col 5-12: exponential increases up to 1.2 microns apart.

pitch 2

Row 6 — 12 of 24 structures containing seven dense lines with width = CD(w). In the first eight structures, spaces are incremented linearly, with 10 nm increments:

  • Col 1: space = CD(s)

  • Col 2: space = CD(s)+10

    :

  • Col 8: space = CD(s)+70

  • Col 9-12: exponentially increasing spaces, which continue in Row 7.

pitch 3

Row 7 — Continuation of the pitch structures in Row 6. Structures 7-12 in Row 6 and all structures in Row 7 have a fixed width = CD(w) and spaces are incremented exponentially from CD(s)+7 up to 1.2 microns.

pitch 4

Row 8 — Seven dense lines with fixed width = CD(w)+10. Space varies:

  • Col 1: space = CD(s)

  • Col 2: space = CD(s)+10

  • Col 3: space = CD(s)+20

  • Col 4: space = CD(s)+30

  • Col 5-12: exponential increases up to 1.2 microns apart.

pitch 5

Row 9 — Seven dense lines with fixed width = CD(w)+20. Spaces varies:

  • Col 1: space = CD(s)

  • Col 2: space = CD(s)+10

  • Col 3: space = CD(s)+20

  • Col 4: space = CD(s)+30

  • Col 5-12: exponential increases up to 1.2 microns apart.

inv_pitch

Row 11 — Inverse of six dense lines with fixed space = CD(w). Line width (also called inverse space) varies:

  • Col 1: inverse space = CD(s)

  • Col 2: inverse space = CD(s)+10

  • Col 3: inverse space = CD(s)+20

  • Col 4: inverse space = CD(s)+30

  • Col 5-12: exponential increases up to inverse space = 1.2 microns.

Note: the first eight structures are padded to ensure a minimum structure width.

Using the “Midheights” setting changes the height of the center lines.

line_end

Rows 12-14 — A matrix of two abutting lines with a gap.

  • Line width changes every six columns, ranging from CD(w) to 0.6 microns (four linear steps of 20 nanometers, two exponential steps.)

  • The gap changes: on each of the three columns ranging from CD(w)-20 to 0.6 microns (three linear steps of 20 nanometers, three exponential steps).

inverse_line_end

Row 16-18 — A matrix of 5x5 micron patches with two inverse lines.

  • Line space changes every six columns, ranging from CD(w) to 0.6 microns (four linear steps of 20 nanometers, two exponential steps).

  • The inverse gap changes: on each of the six columns ranging from CD(w)-20 to 0.6 microns (three linear steps of 20 nanometers, three exponential steps.)

contact

Row 19, Col 1-6 — Contacts (rectangular posts)

  • Col 1: width = CD(w)+60

  • Col 2: width = CD(w)+70

  • Col 3: width = CD(w)+80

  • Col 4-6: exponential increases up to 1.2 microns.

inverse_contact

Row 19, Col 7-12 — Inverse contacts (rectangular hole in 5x5 micron patch). The diameter of the hole varies:

  • Col 7: width = CD(s)+60

  • Col 8: width = CD(s)+70

  • Col 9: width = CD(s)+80

  • Col 10-12: exponential increases up to 1.2 microns.

dense_line_end

Row 21, Col 1-6 — 6 abutting lines with gaps. Line width is constant CD(w). Gap varies:

  • Col 1: gap = CD(s)

  • Col 2: gap = CD(s)+10

  • Col 3: gap = CD(s)+20

  • Col 4-6: exponential increase in gaps up to1.2 microns.

corners

Row 21, Col 7-12 — Two abutting corners with a gap. The gap varies:

  • Col 7: gap = CD(s)

  • Col 8: gap = CD(s)+10

  • Col 9: gap = CD(s)+20

  • Col 10-12: exponential increases up to 1.2 microns.
