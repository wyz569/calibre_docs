Environment Variables Featured in this Manual

The following environment variables are used by Calibre WORKbench.

Settable Environment Variables

Note that the environment variables in the table do not include beta key functionality. Contact your Siemens representative for relevant information.

Table 1. setenv Environment Variable List

Environment Variable

Description (value)

Notes

CALIBRE_HOME

Points to the location of the Calibre WORKbench main directory.

—

Optical model-related variables

PRINT_OPTICAL_MODEL_RUNTIME_INFO

Outputs optical model generation to the console window.

(1 = active, 0 = inactive)

“Optical Parameters File Format”.

WB_PRINT_MODEL

Writes the optical kernel to the disk.

(set to 1 to activate, 0 to deactivate (default))

“opticsgen”,

“vt5gen”.

SRC_MAP_STEP

Sets the grid size of an optical model output source map file.

(0.01 microns by default)

“Source Map File Format”.

LITHO_PROMOTION_TILING

Special settings used by Calibre IPSE.

“Requirements for Calibre IPSE/Calibre IPSETX”.

Resist model-related variables

CWB_MT_THREAD_COUNT

Limits the number of CPUs used during the Multi-Threaded Full and Multi-Threaded Newton search algorithms.

(set to a value between one and the maximum CPU limit or 32, whichever is less)

“modelflow (Version 1)”.
