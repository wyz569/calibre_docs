Table 1. Calibre Tools Simulation at a Glance

Objective

Given a pre-generated optical and resist model, use various simulation tools to predict how designs and changes to those designs appear after the wafer lithography process.

Related Documentation

  • “Simulations User Guide”

  • “Calibre WORKbench Simulation Reference”

Simulation Inputs

Task Flow Summary

  • Setup file

    • Optical model

    • Resist model

  • Design file

  1. Load the design (File > Open).

  2. Open or configure a new session file (Litho > RET Flow Tool v2.0).

  3. Select a simulation frame.

  4. Run simulation tools.

Simulation Outputs

What to Look for in Results

Updates layer(s) in viewer:

  • PI

  • Map

  • OPC (in the old RET Flow Tool)

  • PI shows projected aerial image contours.

  • Map and PI in dense mode show grid-based simulation data.

  • OPC creates a post-OPC design layer.

Statistics Graph:

  • Cutline

  • FInfo and EPE (in the old RET Flow Tool)

  • Data graphs show the intensity values.

  • EPE requires an OPCpro setup. For other session types, use FInfo.

Simulation Commands

Description

aerial

Simulates the aerial image intensity within an area defined by the specified bounding box.

cutline

Simulates the aerial image intensity along a cutline.

cutline2D

Simulates the cutlines for several z-planes within the resist to characterize standing waves in the resist.

dprintiml

Simulates a dense print image and returns one or more of the following: dense printimage, reference layer, threshold image.

iripple

Tests the interaction radius between features for use in evaluating ripple distance.

utp::table2sparse

Converts measurement data from a pitch array into the lookup_file format required for utp::verify.

utp::verify

Generates data comparing simulated CDs to predicted CDs.

vsbIndex

Sets the path and name of an index file for VSB11 chip processing.
