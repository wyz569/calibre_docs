Process Cache Actions

The process cache holds reusable information about the model calibration. You can manipulate the cache directly.
Table 1. Modelflow v2 Cache Actions

Action

Command

Clear the cache

modelflow_v2 -cache clear

Save the aerial image grids from the cache to the specified folder on disk

modelflow_v2 -cachesave dir

Load aerial image grids from the specified folder on disk into cache

modelflow_v2 -cacheload dir

Show the number of aerial image grids in the cache

modelflow_v2 -cachenumber
