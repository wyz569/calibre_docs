Pre-CM1 Calibration Process

Pre-CM1 calibration is a process for creating the initial data set and gauge files for CM1 model calibration.
Table 1. Pre-CM1 Calibration Process

Stage

Task

Description

1

Generating a Starting Optical Model

Generate an initial optical model.

2

Converting Sample Spreadsheets to Gauge Files

Create a gauge file from the sample spreadsheet.

Cleaning the Sample Spreadsheet and Gauge Object

Clean the sample spreadsheet data in CM1 Center and center the sample sites.

Creating a Clipped Layout File

Create a modified layout that only shows critical areas.

Importing Contour Calibration Information

Optionally import a previously-created CLI file and layout to CM1 Center.

Optional

Creating a Model-Layer File

If you do not have a setup file, create a model-layer file for ease of use.

3

Optimizing the Optical Model

Optimize the optical model in CM1 Center.

Using Optical Models for Process Window Optimization

Optionally, optimize multiple optical models using a process window analysis technique.

Using Optical Models with Double Exposure Flows

Optimize double exposure sources.

Resuming Stopped Modeling Runs for Front Searches

Resume some runs that have been stopped.

Optional

Exporting CM1 Center Settings as a modelflow_v2 Run Script

Export your settings as a modelflow_v2 run script file.

Importing Drawn Gauges From a Layout

Import gauges drawn directly in a layout.
