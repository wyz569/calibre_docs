magnification

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Sets the de-magnification factor (also known as the reduction factor) of the optical system.

Usage

magnification mag

Arguments

mag

A required argument specifying the de-magnification factor (also known as the reduction factor).

It is recommended that you set the proper value of de-magnification (typically 4.0) when using vector, thin-film image models. (Default is 1.0 previous to version 9, and 4.0 for version 9).

Description

An optional illumination system parameter specifying the de-magnification factor (also known as the reduction factor), which is the ratio of object (mask) to image size (wafer). It is defined as the ratio of image to object numerical aperture. This factor activates the so-called obliquity correction factor, which accounts for the different sizes of entrance and exit pupils and arises from energy conservation considerations.1

In the case of immersion lithography, the image NA can exceed 1. However, the object NA is always less than 1. In such cases, using the default value of de-magnification (1.0) will result in a user error.

  • Setting the reduction factor to larger than 1 is not recommended for scalar models.

  • Note that in the Calibre WORKbench Optical Model tool, this parameter is shown in the Main tab as Reduction Factor.
