Calibre Pattern Generator EUV Modeling Test Patterns

The built-in EUV modeling test patterns of the Calibre Pattern Generator are patterns and gauges suitable for calibrating models for EUV processes.

The Calibre Pattern Generator has built-in test patterns useful for the following phenomena that uniquely affect EUV patterning:

  • horizontal-vertical feature bias (HV bias)

  • aberration sensitivity

  • black border effects

  • flare sensitivity

2/3_Bar_H

Horizontal 3-bar pattern. Center bar pattern is 3x length of end bars.

Property Descriptions

Height — Applies 3 x Height vertically to middle bar and 1 x Height to top/bottom bars

S — Vertical space between middle bar and top/bottom bars

Width — Horizontal width of all bars

X — Center x-coordinate for pattern

Y — Center y-coordinate for pattern

Gauge Descriptions

Gauge_0 — Vertical height of bottom bar

Gauge_2 — Vertical height of top bar

Gauge_3 — Vertical gap distance between middle and bottom bars

Gauge_4 — Vertical gap distance between top and middle bars

Gauge_5 — Vertical height of middle bar
