Generating a Starting Optical Model

Follow this procedure to generate a starting optical model based on the known exposure conditions. You will also identify and remove gross outliers from the model.

Tip

If you already have an optical model that was created in Calibre WORKbench, you can skip this procedure.

Prerequisites

  • Knowledge of the correct optical parameters (such as NA and λ). For vector models, you will also need the film stack parameters (thickness, n, and k for each layer).

  • Calibre WORKbench GUI.You perform these steps in the Calibre WORKbench Optical Model tool. The Optical Model tool creates optical model data files based on your input that can be used with Calibre tools.

Procedure

  1. In the Optical Model Tool (Optics > Optical Model Tool menu item), enter the optical parameters to generate a nominal (non-optimized) optical model:

    • The model name, λ and NA values are entered in the Main tab.

    • The optical source shape is created in the Illumination tab.

    • n, k, thickness, and the film stack information are entered in the Film Stack tab.

    More detailed instructions on how to use the Optical Model Tool are covered in “Generating the Optical Model”.

  2. Click Generate to create the optical model.
