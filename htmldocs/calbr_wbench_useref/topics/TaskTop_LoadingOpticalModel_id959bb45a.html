Loading an Optical Model

This procedure describes the steps to set the initial optical model parameters. It is important to note the following:

  • The optical model displayed in this tab is used instead of the optical model defined in the setup file.

  • Accepting log results changes the optical model in the tab.

  • The optical model must be manually saved after running enough VT5 Center optimizations to find a satisfactory model.

  • Up to three optical models can be loaded; the additional optical models are used only as exposures for double exposure models or implant modeling and can be left blank in single exposure processes.

Prerequisites

  • Optical model directory, previously created using the steps in “Optical Model Creation”

Procedure

  1. In VT5 Center, click the Optics, in tab.
    Figure 1. VT5 Center GUI (Optics, in Tab)

  2. Choose Optics > Open 1st Model to load the primary optical model. This should be the nominal model at best dose and defocus.

    If you do not have the primary optical model, you can also click the Default button to create a standard optical model design, and alter it as needed.

  3. If you have a second optical model, choose Optics > Open 2nd Model to load it (or use the default).
  4. For implant modeling flows, choose Optics > Open 3d Model to load it (or use the default).
  5. Proceed to “Defining a VT5 Input Model”.
