Opening OASIS Files in Read-Only Mode

Calibre WORKbench can open an OASIS file in read-only mode. 
calibrewb -v filename.oas

Specifies an optional startup OASIS®1 layout file to be loaded as a view-only disk-resident file. If a disk-resident index file is not present, one is created. The -v option cannot be used with the -g option.

calibrewb -noedit

The -noedit option opens the GUI with all the commands that modify the layout disabled.
