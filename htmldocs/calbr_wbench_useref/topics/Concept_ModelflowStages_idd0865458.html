About modelflow Stages

modelflow performs three types of experiments, which you can run separately or in series. These experiments are referred to in modelflow as “stages”.

The stages are

  • Stage 1 — Optical model tuning

  • Stage 2 — Reference Threshold optimization

  • Stage 3 — VT5 Resist model optimization

modelflow reflects these stages in the Calibre WORKbench GUI as tabs, and as keyword stages in the modelflow batch command.

When you run modelflow, you decide which type or types of experiments to perform. However, some dependencies exist:

  • Tuning the optical model is dependent on having valid data.

  • Optimizing the reference threshold is dependent on having a good optical model plus valid data.

  • Optimizing the resist model is dependent on having the best reference threshold plus a good optical model plus valid data.

Run all of the stages at once for a comprehensive solution. Run each stage independently for quicker results.
