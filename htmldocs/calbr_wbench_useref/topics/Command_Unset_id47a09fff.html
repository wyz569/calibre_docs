unset

CM1 and VEB Commands

Related Formats: CM1 Model File Format, Gauge Object Reference, Super Gauge Data File Format

Deletes a CM1 object using its handle.

Usage

unset obj

Arguments

obj

A required argument specifying the handle of the CM1 object to delete.

Examples

Example 1 (CM1 Object)

#create cm1 object of modelform 11 and dump this template to a file
set cm1obj [cm1compose –m 11 –o “cm1.mod”]

# destroy the object
unset cm1obj

Example 2 (sgd Object)

set obj [sgd_create -in “sample.sgd”]

# delete first and last sample sets
sgd_delete $obj 1
sgd_delete $obj end

# save the object to a file
sgd_save $obj -out “sample_new.sgd”

# destroy the object
unset obj
