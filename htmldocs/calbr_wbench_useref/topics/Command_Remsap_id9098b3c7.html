remsap

Optical Model Commands

This command takes an optical model file containing a sourcemap and returns the sourcemap with the optical system pupil apodization removed.

This command is intended to avoid double counting of optical system apodization when the source map is measured at the exit pupil and the apodization is already accounted for in the Jones pupil file.

Note:

You can use the remove_source_apodization optical model parameter instead of remsap to perform the same functionality without having to run the command separately.

Tip

This command has a best practice associated with it. See “Use remsap to Adjust for Measured Source Map Distortion” for more information.

Usage

remsap {-opticalfile filename | -opticalstring text_string} -sourceout output_sourcemap
   [-pupilout output_pupil]

Arguments

-opticalfile filename | -opticalstring text_string

A required argument specifying the optical model as either a filename or a text string. The optical model must contain exactly one sourcemap.

-sourceout output_sourcemap

A required argument specifying the output sourcemap after the pupil apodization has been factored out. The output sourcemap is on the same grid as the input sourcemap.

-pupilout output_pupil

An optional argument specifying an additional output filename. If set, the computed apodization corresponding to the polarization properties of the input source is written out in a sourcemap file format to the specified file. The output grid contains the intensity value that is used for every point in the pupil when computing the output sourcemap from the input sourcemap.

For every point (f,g) in the pupil, the following expression is true:

Sout(f,g) = Sin(f,g) / Apod(f,g)

where:

  • Sin(f,g) — Is the intensity of the input sourcemap.

  • Sout(f,g) — Is the intensity of the output sourcemap.

  • Apod(f,g) — Is the local pupil transmission (computed as intensity, not as an electric field).

Note:

The output pupils Sout(f,g) and Apod(f,g) are computed with a pupil sampling distance that is equal to that of the input sourcemap.
