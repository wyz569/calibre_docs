cpgClipPattern

Scope: Calibre Pattern Generator Terminal Commands

Captures patterns from Calibre WORKbench.

Usage

cpgClipPattern ‑path clips_file ‑layout layout  [‑cell cell_name] ‑clips ‘{’‘{’x1 y1 x2 y2‘}’...‘}’ ‑inlayernumbers ‘{’layer...‘}’ ‑clipids ‘{’clipid1...‘}’ [‑inlayercolors ‘{’hex_colorcode...‘}’ ]

Arguments

‑path clips_file

A required argument specifying an .xml file to save the captured clips.

‑layout layout

A required argument specifying the layout to capture patterns from.

‑cell cell_name

An optional argument specifying the cell name to capture patterns from.

‑clips ‘{’ ‘{’x1 y1 x2 y2‘}’...‘}’

A required argument specifying sets of lower left and upper right coordinates of a clip to capture. Each set must be enclosed with braces ({}). At least one set of coordinates must be specified.

‑inlayernumbers ‘{’layer...‘}’

A required argument specifying the layer numbers to capture patterns from.

‑clipids ‘{’clipid1...‘}’

A required argument specifying the name of each captured clip. The list of clip names must be enclosed with braces ({}). The number of clip names must match the number of coordinate sets specified by ‑clips.

‑inlayercolors ‘{’hex_colorcode...‘}’

An optional argument specifying the color of the output layers in the Calibre Pattern Generator. The color value must be a hexadecimal number preceded by “#”. The number of colors must match the number of layers specified by ‑inlayernumbers. By default, color values are randomly generated.

Description

An optional command that captures patterns from Calibre WORKbench and saves them to an .xml file. Use the command in the Calibre WORKbench terminal window.

Examples

The following example captures two clips from the TOPCELL cell of layout0 and saves them to temp.xml:

cpgClipPattern ‑path temp.xml ‑layout layout0 ‑cell TOPCELL 
	‑clips {{‑649 2979 920 4272} {1686 2897 3550 4369}} 
	‑inlayernumbers {0 1 2} ‑clipids {Clip_1 Clip_2} 
	‑inlayercolors {#00FF00 #0000FF #FFFF00}

The example captures from layers 0, 1, and 2, and colors them green, blue, and yellow.
