Importing the SEM with Auto Alignment

In this task, you will import a SEM image and use the auto alignment functionality to match the image to a loaded verification test structure.

Prerequisites

  • Actual SEM images taken from the verification test chip available

  • The corresponding test pattern

Procedure

  1. Open the layout in Calibre WORKbench.
  2. Load the Calibre OPCpro setup file.
  3. Choose Litho > SEMoverlay. The Calibre SEMoverlay tool appears.
  4. In the layout window, select the test structure to use for verification by drawing a selection rectangle around the test structure.
  5. In the SEMoverlay window, click Import Image and load in the SEM image that corresponds to the selected test structure.
    Figure 1. Automatically Aligning a SEM

Results

The SEMoverlay tool aligns a SEM contour with a simulated aerial image contour that it generates.

  • It simulates the image contour using the optical model referenced in the setup file.

  • It creates a SEM contour at the default intensity level of 0.5.
