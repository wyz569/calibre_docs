sgd_cget

CM1 and VEB Commands

Related Formats: Gauge Object Reference, Super Gauge Data File Format

Gets parameter values from a specified sgd object or an entire gauge data object.

Usage

sgd_cget object [gaugeSet_index [parameter | ‑gaugeobject]]

Arguments

object

A required argument that specifies the input sgd Tcl object.

gaugeSet_index

An optional argument that specifies the index of the gaugeSet within the sgd object. Accepted values for gaugeSet_index are the following:

  • integer — Returns the values of the specified parameters or a Tcl object of the gauge set determined by this index number, which must satisfy the following inequality:

    0 < gaugeSet_index <= number_of_gaugefiles

  • end — Returns the specified parameters or Tcl object of the last gaugedata set.

  • all — Returns the values of the specified parameters for all gaugedata sets.

parameter

An optional argument specifying a particular parameter to value to return. Cannot be specified with ‑gaugeobject. Only one of the following parameter types may be specified per sgd_cget command:

  • ‑gf — An optional argument that fetches the name of the gaugedata file.

  • ‑dose — An optional argument that fetches the dose.

  • ‑defocus — An optional argument that fetches the defocus.

  • ‑type — An optional parameter that fetches the metrology type (CD or SRAF).

  • ‑plane — An optional parameter that fetches the metrology plane.

  • ‑weight — An optional argument that fetches the weight.

  • ‑opticaldir — An optional argument that returns the name of the first optical model directory.

  • ‑opticaldir2 — An optional argument that returns the name of the second optical model directory.

‑gaugeobject

An optional argument that fetches a Tcl object from the gaugedata set at the specified gaugeSet_index.

Note:

If none of the parameter switches (‑gf, ‑dose, ‑defocus, ‑weight, ‑gaugeobject, or ‑opticaldir) are specified, this command returns all parameter values (except gauge data objects) for the gaugedata set determined by the specified index. If gaugeSet_index is not specified, sgd_cget returns all parameter values (except gauge data objects) for all gaugedata sets entering the specified sgd object.

Examples

set obj [sgd_create -in sample.sgd]
# get the dose value associated with 2nd sample set
sgd_cget $obj 2 -dose
# get the values of all parameters associated with last sample set
sgd_cget $obj end 
# get defocus and dose values of all sample sets
sgd_cget $obj all -defocus -dose
# get the values of all parameters of all gaugedata set
sgd_cget $obj
# get 2nd sample set as Tcl object and dump this Tcl object to a file
set gd_obj [sgd_cget $obj 2 -gaugeobject]
#NOTE: do not use $gd_obj, you have pass to gaugeWrite the object itself
gaugeWrite -g gd_obj -o “gauges.gg”
