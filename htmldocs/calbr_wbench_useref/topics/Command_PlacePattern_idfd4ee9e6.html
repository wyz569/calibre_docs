place_pattern

Scope: Calibre Pattern Generator API Commands

Places a configured pattern in a specified row and column.

Usage

$cpg place_pattern ‑pattern_idx pattern_id ‑row row_num ‑col col_num

Arguments

$cpg

A required argument specifying $cpg as a Tcl configuration object or variable.

‑pattern_idx pattern_id

A required argument specifying the pattern template to place using a configured pattern id returned by configure_pattern.

‑row row_num ‑col col_num

A required argument specifying the rows and column to place the pattern template.

Description

A required command set that places a pattern template in a specified cell in the layout.

Examples

The following example incrementally places the patterns in pattern_list into cells by rows and columns: 

set p_count 0
	for {set i 0} {$i < $rows} {incr i} {
		for {set j 0} {$j < cols} {incr j} {
			if {llength $pattern_list > p_count} {
				$cpg place_pattern ‑pattern_idx [lindex $pattern_list \
				 $p_count] ‑row $i ‑col $j
			}
		incr p_count
		}
	}
