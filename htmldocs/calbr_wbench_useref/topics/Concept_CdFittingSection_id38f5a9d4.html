CD Fitting Section

This section is available for the Optics, CM1, and VEB Fitness trees.

Shows the accuracy of the data fitting results as a function of the gauges versus CD. You will need to scroll down to see additional results.

Figure 1. Analysis Graph (CD Fitting View)

The graphs available in the CD Fitting section (Figure 1) use the following codes:

  • Simulated CD (Gx_s)

  • Measured CD (Gx_m)

  • CD error (Gx_e) for each data point

Gauge files that contain a single structure type are displayed with one set of simulated, measured, and CD errors. Gauge files with multiple (differently-named) structures group their gauges by structure type.

The CD Fitting section contains the following graphs:

  • Simulated CD (CDs) versus Measured CD (CDm) plots

  • CD error: CDm - CDs plots

  • Diagonal plot of CDs versus CDm

  • Sorted CD error

  • Distribution of CD errors
