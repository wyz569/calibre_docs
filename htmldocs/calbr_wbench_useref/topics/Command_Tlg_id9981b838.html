tlg

Scope: Calibre Pattern Generator API Commands

Generates a layout from a layout configuration file.

Usage

tlg config_file

Arguments

config_file

A required argument specifying the path and name of the layout configuration file.

Description

A required command set that generates the test layout using a layout configuration file. The command is often the last statement in the script.

Examples

The following example generates a test pattern layout using the settings in the 3Bar_config.xml layout configuration file:

tlg 3Bar_config.xml
