Run Simulations

The primary usage mode of modelflow_v2 is to perform a simulation using an empirical data set. You will be providing as input gauge files and operating parameters to this command.

The modelflow_v2 command operating parameters are in groups.

  • modelflow_v2: Input Section Arguments

  • modelflow_v2: Output Arguments

  • Stage 1 Optical Optimization Modes and Parameters

    • modelflow_v2: Stage 1 Optimization Run Modes

    • modelflow_v2: Stage 1 Optimization Parameters

  • Stage 2 CM1 Resist Optimization Modes and Parameters

    • modelflow_v2: Stage 2 Resist Model Tuning Parameters

    • modelflow_v2: Stage 2 Optimization Parameters

  • Stage 3 Etch Optimization Modes and Parameters

    • modelflow_v2: Stage 3 Etch Model Tuning Options

    • modelflow_v2: Stage 3 Optimization Parameters

A list of Calibration Procedure Examples is provided after the option descriptions.
