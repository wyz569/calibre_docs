layout2gauge

CM1 and VEB Commands

Related Formats: Gauge Object Reference

Creates a gauge set from wires drawn on the layout.

Usage

layout2gauge -l wblayout -layer layer [-ssg]

Arguments

-l wblayout

A required argument that specifies an input layout name.

-layer layer

A required argument that specifies the layer to get the gauges from.

-ssg

An optional switch that instructs Calibre WORKbench to try to solve for subgauges. The command reads up to 2n+1 subgauges if this option is specified, and sets the gauge object’s count value to the number of gauges found. If -ssg is not specified, the command only reads the one central gauge and sets its count value to 0.

Description

Reads wires from the layout and transforms them to gauges. The wires should be drawn as paths on the layer specified in the -layer argument. See the section “Layout File as a Gauge Data Source (layout2gauge)” for more information.

Examples

set bobj [layout2gauge -l layout0 -layer 3]
