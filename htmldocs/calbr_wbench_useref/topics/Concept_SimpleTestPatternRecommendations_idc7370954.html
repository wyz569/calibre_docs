Simple Test Pattern Recommendations

Certain simple practices are meant for users who create test patterns with the Calibre WORKbench Test Pattern tool.

  • Use the latest test pattern version (12) created by the Test Pattern tool.

  • Make sure your node size matches the design size, including the grid size. Failure to do so will result in an error during simulation.
