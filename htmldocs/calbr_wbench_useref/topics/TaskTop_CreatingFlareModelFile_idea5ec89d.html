Creating a Flare Model File

A flare model file is required for creating the PNG flare map.

Prerequisites

  • A point spread function (PSF) file

  • Understand the Flare Model File Format and Flare Intensity File Format

Procedure

  1. Create a flare model file using one of the following techniques:

    • By creating a flare model file manually.

      Set arguments to use a computed fractal kernel to predict the flare model behavior by supplying values for the kf, ka, aspect, balance, s1, s2, rmin, rmax, and rsl arguments.

      You also need to enter the data from the PSF file between psf_start and psf_end headers, or use the “psfgridfile” option to point at a separate PSF file.

    • By creating a flare intensity file and using it as input to the euv::fitflare command.

      This option is useful for when detailed flare measurements are available.

  2. Save the file. You will use this file as the flare model input in the task “Generating a Flare Map With RET FLARE_CONVOLVE”.
