Creating a Gauge File for Variable Etch Bias (VEB)

Follow this procedure to prepare a gauge file with Variable Etch Bias (VEB) information.

Prerequisites

  • You must have variable etch bias information available for entry into a gauge file.

  • You must have a gauge file that you created in “Converting Sample Spreadsheets to Gauge Files.” A gauge file so created does not have measured VEB data, however; you will need to add this data either manually or by using a script.

Procedure

  1. Using your preferred text editor or by editing the gauge object in the Gauges tab, open the gauge file and add information for variable etch effects. You must edit the values in the MEtch (measured etch, column 13) column as appropriate. Both resist (Meas) and VEB (MEtch) columns must have positive values for VEB optimization, since VEB improves on the CM1 results.
  2. Save the modified gauge file under a different name.
