How to Run Calibre WORKbench on Remote Hosts

Run Calibre WORKbench on one or more remote hosts by adding Calibre® MTflex™ standard command line arguments to your invocation string.

For information on these arguments, see the Calibre Administrator’s Guide.

  • -mtflex — Starts a server process on a remote host, and connects the remote host to the primary host.

  • -remote — Specifies the names of remote hosts for Calibre MTflex.
