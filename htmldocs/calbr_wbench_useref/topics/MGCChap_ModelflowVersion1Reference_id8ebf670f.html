modelflow Version 1 Reference

The modelflow (version 1) tool uses site-based simulations to optimize both the supplied optical model and VT5 resist and etch model.

It contains the following features:

  • A multi-parameter optimizer

  • The ability to import pregenerated resist contours for calibration purposes

  • Support for focusblur calculations

  • Support for calculation of half-density kernels

  • The ability to model mask proximity

    Note:

    For CM1 models, use the modelflow_v2 tool instead. See “modelflow_v2 Reference” for more information.
