vertical_diffusion_shrink

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Specifies the shrinkage of NTD resists.

Usage

vertical_diffusion_shrink top amplitude length

Arguments

amplitude

An optional argument specifying the percentage of shrinkage in resist thickness.

length

An optional argument specifying the smoothness of the transition from no resist to resist.
