gaugeCreate

CM1 and VEB Commands

Related Formats: Gauge Object Reference

Creates an empty gauge object.

Note:

The object can be populated using the gaugeAdd command and edited using other gauge-related commands.

Usage

gaugeCreate

Arguments

None.

Return Values

The new gauge object is returned into the handle you supply for it.

Examples

set aobj [gaugeCreate]
