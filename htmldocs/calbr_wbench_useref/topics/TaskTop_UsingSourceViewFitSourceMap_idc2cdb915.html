Using Source View to Fit a Source Map

A source map can be created by combining several illumination primitives and applying an error function (ERF) to smooth the edges of the source if needed.

The source map is generated using the Source View tool, available through Optics > Source View. This tool displays a source map as either a map or a contour or both. The map can be annotated with grids, circles, and rays to measure the various properties of the illuminator. This function is used to manually estimate parameters of an existing source map.

Prerequisites

  • Illumination source map measured from a scanner or from an existing optical model

Procedure

  1. In Calibre WORKbench, open the Source View tool (Optics > Source View).
  2. To create an initial estimated source, select File > New Source Map. In Generate Options, choose one of the following options:

    • For quasar illumination, enter a non-zero value for the Illum Angle field. A value of zero results in an annular source.

    • For an ERF profile quasar illumination, enter a non-zero value in the Delta Illum Angle field.

  3. Click Generate to see the initial results of your estimates.
  4. To estimate the values for sigma, sigma_in, sigma_out, and any other circular parameters, enter the circle description in the Circles field of Map Options and click Apply to selected. The circle description format is as follows:

    radius [X Y] [thickness] [color]

    where:

    • X, Y are coordinates of the center, where the coordinates are normalized to 1, which corresponds to (2*pi)/(λ*NA).

    • X, Y, thickness, and color are optional.

    To draw multiple circles, enclose each circle definition in braces.

    Example: Using the line “0.1 {0.2 0.3 0.39 3 “red”}” will draw 2 circles:

    • The first circle is radius 0.1 and is centered at (0.0, 0.0).

    • The second circle is radius 0.2, is centered at (0.3, 0.39), has a line thickness of 3, and is colored red.

  5. To estimate the ERF change in sigma values, draw intensity contours:
    1. Enter the intensity (or a list of intensities as space separated floating point values) in the Contours field.
    2. Enter the color (or list of colors separated by spaces) in the Colors field
    3. Ensure the Contour check box is selected, and click Apply to selected.
  6. You can continue to adjust fields in Generate Options and Map Options as needed. When the source is regenerated using adjusted settings, any custom Map Options settings must be reentered. Select File > Save Source Map when you are satisfied with the results shown in the map display.
  7. To compare the actual approximation to the original source map:
    1. Obtain the measured source map or map from an existing model.
    2. Open the simulated source map in the Source View tool.

      When an optical model is generated through the Optical Model Tool, the pixel size for the model can be set in the Output Source Map Pixel field, located in the Misc tab of the Optical Model Tool. Creating an optical model results in the optical model itself plus a source map for the model, called illummap.smf.

    3. Compare the simulated source map for the new model to the original source map.
