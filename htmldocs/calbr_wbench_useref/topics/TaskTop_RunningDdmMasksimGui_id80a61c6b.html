Running the DDM MaskSim GUI

The DDM MaskSim GUI provides a quick template generator that uses values you specify to create the DDM library as output.

Prerequisites

  • Calibre WORKbench invoked

Procedure

  1. In Calibre WORKbench, select the Optics > DDM MaskSim menu item to open the DDM Edge Library Creation tool.
  2. Fill in the fields as described in Figure 1 and Table 1.
    Figure 1. DDM Edge Library Creation Tool, Main Tab

    Table 1. DDM Edge Library Creation Tool, Main Tab Notes

    Area

    Equivalent DDM Input Keyword

    Notes

    a

    gridsize

    Required argument. Fill in using appropriate values from your design specifications and mask stack film thickness.

    See the gridsize argument description in “DDM Library Input File Format”

    edgelength

    Optional argument. Default values are 8um for DUV and 2um for EUV.

    b

    illumination

    Required argument. Points to an optical file.

    chief_ray

    Optional for DUV models.

    Required for EUV optical models.

    c

    • bright_outputfile

    • dark_outputfile

    • compute_diagonals

    The checkboxes next to the Bright and Dark Output File fields must be selected to enable output. Files are written to the current working directory.

  3. Click the Mask tab. Select either a DUV or EUV mask with the Mask dropdown.

    Fill in required input using the guidelines in Table 2 and Table 3.

    Figure 2. DDM Edge Library Creation Tool, Mask Tab, DUV Mode

    Table 2. DDM Edge Library Creation Tool, Mask Tab, DUV Mode Notes

    Area

    Equivalent DDM Input Keyword

    Notes

    a

    • wavelength

    • reduction_factor

    Required input.

    b

    • substrate n (Refractive Index)

    • phasedepth

    • underetch

    • rounding

    • slope

    Only the Refractive Index field is required in this area.

    • rounding applies to a circle that inscribes the rounded corner (see Figure 1 in the section “DDM Library Input File Format”).

    c

    • film t (Thickness) n (Real Part) k (Imag Part)

    • stackslope

    • stackrounding

    Click Add Layer to create a new line, then click in the individual fields to edit them.

    • stackrounding applies to the corner where the film stack meets the air (see Figure 2 in the section “DDM Library Input File Format”).
    Figure 3. DDM Edge Library Creation Tool, Mask Tab, EUV Mode

    b

    • stackslope

    • stackrounding

    • outputplane

    Click Add Layer to create a new line, then click in the individual fields to edit them.

    c

    • capping_layer

    		  

    d

    e

  4. When you have completed configuring the fields, preview the appearance of the 3D mask by selecting Tools > View Mask.
    Figure 4. Mask View Window

  5. Optionally, select Options > More Options button. Inspect and set any additional options as needed in the dialog box that appears, then click OK. It controls the following simple DDM format keywords:

    • mincycle (Min Success Cycle)

    • maxcycle (Max Success Cycle)

    • domain {spatial | fourier} (Fourier is not currently supported.)

      Figure 5. More Options Dialog Box

  6. Click the Run button to generate the DDM library file(s).

Results

Depending on whether you specified a bright, dark, or both output file types, the new DDM library files are written to the disk. You can use these with other Calibre tools.
