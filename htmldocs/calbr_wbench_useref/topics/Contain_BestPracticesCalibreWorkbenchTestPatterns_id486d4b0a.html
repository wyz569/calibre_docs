Best Practices for Calibre WORKbench Test Patterns

Following test pattern best practices generally improves test pattern quality.

File formats used for test pattern creation are:

  • “Sample Spreadsheet File Format”.

  • “Super Spreadsheet File”.

  • “Test Pattern File Format”.

  • “Test Pattern Geometry Descriptions”.

Test Patterns come in two types: automatically generated using the Test Pattern tool, and user-created and configured in a test pattern definition file.

Table 1. Test Pattern Minimum Requirements

Requirement

Function

Notes

Sample file spreadsheet

Identifies the test structures in the test pattern file.

Spreadsheets start out with their Measurement columns empty and must be filled in from empirical data.

Contains test pattern shapes.

Test pattern shapes have their X and Y locations in the sample file spreadsheet.
