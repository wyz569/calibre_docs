add_sem_marker

Scope: Calibre Pattern Generator API Commands

Places a SEM marker in a layout configuration file.

Usage

$cpg add_sem_marker  ‑marker_idx marker_id

Arguments

$cpg

A required argument specifying $cpg as a Tcl configuration object or variable.

‑marker_idx marker_id

A required argument specifying the ID of the SEM marker to add.

Description

An optional command that places a SEM marker in a layout configuration file.

Examples

This example searches a marker library that contains a pattern named semmark1 with properties named cd and cd2. When a pattern named semmark1 with cd equal to 100 and cd2 equal to 200 is identified, markerId is set to the ID of the matching pattern. The SEM marker is then added to the layout configuration.

set markerId [$cpg configure_marker -name semmark1 \
	-properties “cd 100 cd2 200”]
$cpg add_sem_marker ‑marker_idx $markerId
