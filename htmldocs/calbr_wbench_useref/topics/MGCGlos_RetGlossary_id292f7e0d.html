RET Glossary

aberrations

Characterizations of any failure of rays from a given object to come to a single or common image point due to lens imperfections or lens design.

aerial image

The distribution of the intensity of projected incident light outside of the resist film.

apodization

Radial transmission loss of the projection optics, caused by imperfections in lens coating or other physical features.

assist feature

Non-printing features added to a mask to improve the printing of isolated features on the mask.

attenuated phase shifting mask

A phase shift mask where the non-clear areas of the mask are partially transmitting. Common uses for this mask type are contacts and isolated features.

background

The region of the design and mask that does not form a polygon.

background transmission

The normalized transmission value of the background, or field, part of the mask.

beamfocus

The image convergence of the projection lens in air, relative to the top of the resist.

bias

A change in the position of an edge of a polygon on a mask layer, resulting from moving the edge inward or outward as needed to improve printability. Biasing is a form Optical Process Correction.

binary mask

A conventional mask that has only two types of areas: completely clear and completely opaque.

clipping layer

Clipping layers are special layers containing shapes (usually rectangular) that enclose all or part of a single polygon structure to be given special treatment.

The Calibre ContourCal tool uses clipping layers in two ways:

  • As a context frame to enclose all of a single test pattern structure, placing a boundary on the simulation area (also referred to as a context clip).

  • As a frame for parts of a test pattern to be given a higher or lower weight setting.

Constant Threshold Resist model (CTR)

A resist model that defines the wafer print image as a specific contour of the aerial image. For example, a 0.3 constant threshold model predicts wafer features by assuming that the threshold for resist development is an aerial image with a minimum intensity of 0.3. The simulated printed image follows the contours of the 0.3 aerial image intensity.

contour block

Contour blocks are the method that Calibre ContourCal divides up the contours in analysis graphs. For each distinct clip polygon on a clipping layer, a single block is defined in the analysis graph containing all the points inside that clipping polygon.

control point

Locations on a control site where the RET tools calculate the image parameters.

control site

Locations on the opc layer where simulation data is evaluated for edge placement errors and image parameters.

Critical Dimension (CD)

The size of a feature to be printed that is important to dimensional control.

cutline

The light intensity calculation for a linear segment on the layout.

defocus

Is an offset from the best focus conditions in um. In simulations, it is added to the beamfocus value when beamfocus is present in the nominal model, or to the defocus start parameter of the optical model otherwise.

density

The area of the latent image of a structure, normalized by the total kernel area, weighted according to a kernel function.

Deep SubMicron (DSM)

Circuitry having trace widths less than 0.5 um.

Deep Ultraviolet (DUV)

Deep Ultraviolet lithography uses wavelengths of 248 and 193 nanometers.

Depth of Focus (DOF)

Range of focus error that a process can tolerate and still give good results.

diffraction

Modulation of light waves in response to an obstacle in their path.

dose

Is a dimensionless parameter specifying the dose ratio to the best dose. It is mapped during simulations to the setup file dose for the first mask (in other words, the setup file dose is multiplied by this dose).

edge

A line segment between two corners in a layout polygon.

Edge Placement Error (EPE)

The difference between the drawn edge of a polygon and actual printed edge.

EmpEPE

The edge placement error as measured on a test chip.

EmpTHRESH

The image intensity threshold at which printing actually occurred (measured).

factor

A Siemens EDA proprietary term referring to an image property measured at the ‘opc’ layer edge location. For CTR, VTR, and VTR-E models it is calculated at the control point closest to location of maximum slope. For VT5 models, it is the calculated at the reference threshold.

feature

Drawn or derived polygons in a design.

field mask

The second mask used in a phase shifted mask. It is primarily used for printing the field poly and preventing the second exposure of the gate poly.

fitness

A measure of how well a model matches the experimental data.

The fitness of a set of data is calculated by recording the total Edge Placement Error (EPE) between a set of sample data points and corresponding simulated data points. The object is to find a model (optical or resist model) that when applied to the sample data has an EPE value close to zero; this indicates a fairly accurate prediction model.

fragment

A line segment between two vertices.

fragmentation

The breaking up of a polygon edge into smaller fragments.

hammerhead

A shape resulting from the application of Optical Process Correction to line-ends. Hammerheads are characterized by rectangle shape applied over the line-end to prevent line-end pull-back.

horizontal calibration

Used in CM1 resist model calibration, horizontal calibration is a more robust approximation method to minimize EPE and CD differences than vertical calibration alone. It affects modelflow_v2 calibration by changing the approximation method used to find linear model coefficients so that it includes horizontal approximation at the measured printing positions along with the default vertical approximation.

implicit fragmentation

Fragmentation that is introduced at tile and hierarchy boundaries.

IMAX

The first local maximum of intensity along the specified region. For CTR, VTR, and VTR-E, this is along the site. For VT5, it is within a window with a total width of 2*searchRange (default searchRange = 0.5*lambda/NA).

IMIN

The first local minimum of intensity along the specified region. For CTR, VTR, and VTR-E, this is along the site. For VT5, it is within a window with a total width of 2*searchRange (default searchRange = 0.5*lambda/NA).

interfeature fragmentation

Fragmentation based on the proximity of nearby features. Interfeature fragmentation creates an initial vertex at the point on the edge that is closest to the feature, then adds pairs of vertices on either side of the initial vertex to create symmetrical fragments on both sides of that initial vertex.

intrafeature fragmentation

Fragmentation based on the internal proximity of vertices on the polygon being fragmented. Intrafeature fragmentation adds vertices at specified distances from the vertices on the polygon.

island-Layer fragmentation

Fragmentation based on the presence of other layers in the design. Island-layer fragmentation inserts vertices whether the edges on an opc layer or correction layer intersect edges or vertices on an island layer.

jog

A concave corner-edge-convex corner sequence in which the edge is shorter than a user-specified length and intended to be excluded from further fragmentation.

kernel

A filter or linear system that is a component of a model.

layer

A set of layout polygons having the same display or other properties.

linearity plot

A plot of achieved CD versus target CD.

loop

A closed chain of edge fragments that does not intersect another loop (including itself). Loops can be either positive or negative. Positive loops are just like regular polygons. Negative loops are holes. After the Calibre application performs fragmentation, the layout is a collection of loops, not a collection of polygons.

Manhattan geometries

Polygons that contain only 90 degree angles.

Mask Error Enhancement Factors (MEEFs)

A measure of the wafer image sensitivity to the mask feature width.

measurement site

A site in a test pattern where EPE is measured.

MEBES

An e-beam system layout data format, proprietary to ETEC.

model calibration

The practice of finding a set of physics-based equations that approximately predict the measured behavior of a test chip fabrication process. By using the equations you simulate the behavior of other layouts using elements of those structures on the test chip.

ModelEPE

The simulated edge placement error predicted by the process model. For VT5 models, this means using both the threshold and the bias polynomials.

Modelform

The term modelform refers to the basic form a model takes: the type of polynomial(s) (bias, threshold, or both), the polynomial order(s), the input parameters, and more.

ModelTHRESH

The simulated threshold (predicted by model).

Multi-Threading (MT)

Multiple instruction threads on a shared memory machine, all of which reside at the same virtual address space. Run times are faster on machines that contain multiple CPUs. CPU time is usually greater with multi-threading, but in real time, multi-threading is faster.

nominal model

A pre-existing model used as the starting point for VT5 model development. The nominal model supplies values for those parameters that are not optimized using Modelflow.

Numerical Aperture (NA)

A characteristic of a lens defined as the light gathering power of the objective.

Off Axis Illumination (OAI)

An illumination system characterized by a light source that is off-axis, or not directly perpendicular, relative to the mask plane. It is a means of overcoming restrictions on resolution of an optical system caused by capturing more first or higher order information into the entrance pupil.

optical diameter

The spatial diameter of influence of the optical system.

optical system

The focus behavior of the light source, lens, and stepper.

outlier

Data points that lie outside of a standard deviation bell curve.

P1 Unit

One P1 unit corresponds to 1 optical wavelength in photoresist. For example, a 248 nm laser illumination lambda and a 1.75 refraction coefficient results in a P1 unit that is equal to 141.7 nm.

partial coherence factor

A means of parameterizing partially coherent illumination. This parameter describes the fraction of the NA of the projection lens filled by the illuminating beam. A sigma greater than one is for practical purposes incoherent light. A usual range for projection lithography exposure tools is 0.6-0.7

Phase Shifted Mask (PSM)

A transmission mask coated with a transparent layer patterned so that the optical phases of apertures closest to each other are of opposite phase.

Photoactive Compound (PAC)

A component of resist that slows development. It is a parameter in rate of reaction calculations.

pitch

A characteristic of sets of parallel, equidistant polygons to be printed on a wafer, calculated as the polygon (line) width plus the space separating adjacent polygons (lines).

process model

A process model is a mathematical function that uses the aerial image (the output of the optical model) as input to predict the threshold at which edges will print.

Rayleigh Criteria

A criteria indicating that an image is nearly perfect. It can be summarized as follows: The optical path difference between the longest and the shortest paths leading to a selected focus does not exceed one quarter of a wavelength than an optical system.

reference threshold

The simulated threshold predicted by the nominal model. This term relates only to VT5 models, which use nominal models as the starting point for development of the actual VT5 model.

rendered source

A rendered source is an illumination source that has been rendered for a specific lithography tool and its illumination system configuration.

resist

A light-sensitive material that allows the transfer of a pattern onto a substrate.

resist profile

A description of the way resist responds to light. As photoresist is exposed, the absorption of light is not uniform.

reticle

A quartz mask with a layer of chrome or attenuated films forming the pattern (usually magnified 4x to 10x) of one device layer. This term is often used synonymously with the term “mask”.

ripple fragment

Symmetrical fragments created by adding vertices at either side of an initial vertex. The initial vertex can be created through interfeature, island-layer, or visible-layer fragmentation.

rule deck

A Standard Verification Rule Format (SVRF) rule file for performing Design Rule Checking (DRC) using Calibre nmDRC. The list of rules can be found in the Standard Verification Rule Format (SVRF) Manual.

sample spreadsheet

Sample spreadsheets refer to the data on measurement markers placed on a test pattern mask, where the actual locations of the markers are compared against the intended locations.

Scanning Electron Micrograph (SEM)

An image captured or viewed with a scanning electron microscope.

scattering bar

A bar shaped assist feature added to the mask to improve the printing of isolated features on the mask by normalizing the image contrast relative to dense features.

SEM and SEM contours

An abbreviation for Scanning Electron Microscope, the SEM tool is used in lithography to examine the results of a chip fabrication test from a mask. The edges of the design in the image created by a SEM tool can be described with a contour.

serif

Rectangles that cover and extend the corners of line ends to prevent line-end pullback.

sites

See control site.

slope

A measurement of the slope of the image intensity graph along a site. For CTR, VTR, and VTR-E models it is reported as the maximum slope of the intensity within site (or maximum slope of the log of the intensity when logslope option is turned on). For VT5 models, it is the slope at the reference threshold.

SRAF (Sub-resolution Assist Feature)

See assist feature.

stdout

A Linux® memory location. Writing a file to stdout means it is displayed in the shell window.

stepper

A photolithography tool used in the production of semiconductor devices. Its full name is step-and-repeat projection aligner. It works by transferring the image of a circuit or component from a master photomask image onto a small portion of the wafer surface.

Sum of Coherent Systems (SOCS)

An approximation to the exact Hopkins formulation of aerial image intensity. SOCS approximations are used by the Calibre RET Toolset in simulations.

SVRF

See rule deck.

tag

An association of a fragment or collection of fragments with a particular characteristic. Tags are created with the Calibre WORKbench newTag command.

target layer

The target layer defines the desired silicon image.

Tcl

A scripting language that can be used to write user-defined procedures and macros for use in Calibre WORKbench.

technology

The basic characteristics (size, materials, device type or types) of a specific semiconductor manufacturing method.

thin-film image

The distribution of the intensity of projected incident light inside the resist film.

threshold

The light intensity required to develop the resist material on a chip.

tiles

Areas that LITHO commands create when they process large areas of data. Implicit fragmentation can occur at the tile boundaries.

transmission value

The characteristic of a substance that determines how it affects the light passing through it.

trim mask

A special mask that typically covers the transition between phase regions and background to prevent this transition from printing on the wafer. So it “trims away” these unwanted features.

Variable Threshold Resist model (VTR)

The Variable Threshold Resist process model (VTR) calculates the wafer print image as a function of intensity and slope. That is, it predicts the wafer features by assuming that resist development is a function of the aerial image intensity plus how rapidly the intensity is changing.

weight

  • Weight settings can be specified for test pattern structures (VT5 Center) or gauges (CM1 Center); the higher the number, the greater the weight.

  • In the Calibre ContourCal tool, the weight setting is relative to the sample spreadsheet data; the formula for weight in this instance is:

    spreadsheet data weight = (1 - ContourCal data weight).
