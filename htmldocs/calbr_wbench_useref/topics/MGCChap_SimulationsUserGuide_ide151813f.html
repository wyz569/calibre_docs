Simulations User Guide

Calibre WORKbench is usable as an image simulator for designs. Several tools are available in the Calibre WORKbench GUI and through batch commands to calculate and display the effects of a process model set.

The RET Flow Tool is an interface that can be used to configure setup files used with Calibre OPCpro, Calibre WORKbench modeling, and Calibre LITHOview simulation. It also offers a method to run simulations that are not tied to a particular Calibre tool, which is the primary session type used by the Simulations User Guide.

Note:

All functionality for the Litho File Tool and associated utilities has migrated to the RET Flow Tool. As of the 2014.3 release, the RET Flow Tool appears in Calibre WORKbench by default. Refer to the Calibre WORKbench: RET Flow Tool User's Manual for complete information. As of the 2018.4 release, the Litho File Tool is unavailable.
