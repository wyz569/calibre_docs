Creating a Super Spreadsheet

You create a super spreadsheet from multiple spreadsheets (a maximum of 128 spreadsheets can be combined into a single super spreadsheet) in Calibre WORKbench. The super spreadsheet you create is used as input to the VT5 Center tool.

Prerequisites

  • At least one sample spreadsheet file (if you are using an empty spreadsheet file) or multiple spreadsheets saved with different names, corresponding to the dose and defocus values used for easy identification

  • The accompanying test pattern design file

Note:

Test pattern structures cannot begin with a number; Siemens EDA recommends you use names similar to the ones used by the Test Pattern Generator.

Test pattern structures must be centered, and the marker must be located on the edge of a polygon to be measured.

Procedure

  1. Open the VT5 Center (in the Calibre WORKbench main window, select Litho > VT5 Center) and click the Sample, in tab.
  2. Using the following Sample down menu items, create a super spreadsheet from one or more sample spreadsheets:

    • Open SS File — Opens an existing spreadsheet file. Any previously loaded tabs are lost.

    • Append SS File — Adds an existing spreadsheet file as a new tab.

    • Append New SS — Creates a blank spreadsheet tab.

    • Open SSS — Loads an existing super spreadsheet file. Any previously loaded tabs are lost.

    • Extract SSS from Log — Loads a super spreadsheet from a previously generated modelflow(v1) or VT5 Center log.

  3. Set the Dose and Defocus values for spreadsheet tabs as needed. You can also use the Sample > Edit FEM dialog box to set multiple dose, defocus, and weight values on a per tab basis.
