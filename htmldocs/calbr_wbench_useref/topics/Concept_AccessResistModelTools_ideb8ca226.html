Access for Resist Model Tools

The following table shows the tools used to create resist models and provides a description of where to find more information.
Table 1. Summary of Resist Model GUI Controls

Tool

GUI Location

Description

  • Information

RET Flow Tool

Litho > RET Flow Tool, use OPCpro session

Sets required values used by the resist model tools.

  • Calibre WORKbench: RET Flow Tool User’s Manual

VT5 Center

Litho > VT5 Center

Optimizes a resist model from a layout and sample spreadsheet (and supports multiple spreadsheets).

  • “Model Optimization With the VT5 Center GUI”

Process Window Analysis Tool

Litho > VT5 Center, load a log file, then click Plot

Uses multiple sample spreadsheets in a super spreadsheet to measure across a process window.

  • “Process Window Calibration Concepts”

ContourCal

Litho > VT5 Center, CLI tab

Imports SEM measured contours into the litho model tools. Optional tool.

  • “Importing Contour Calibration Files”

Image Parameter Space Explorer

Litho > IPS Explore

Highlights and creates test pattern addenda based on coverage. Optional tool.

  • “Using Calibre IPSE”
