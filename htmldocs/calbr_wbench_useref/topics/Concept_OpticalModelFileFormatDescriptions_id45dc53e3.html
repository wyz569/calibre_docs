Optical Model File Formats

Optical model file formats are used as inputs and outputs to optical model commands.
Table 1. Optical Model File Format Descriptions

Format

Description

Optical Parameters File Format

Comprises all supported keywords to configure an optical model.

Source Map File Format

Defines an optical source map.

Sparse TCC Matrix File Format

Defines a sparse TCC matrix text file.

Jones Pupil Data File

Defines a Jones pupil matrix.

DDM Library Input File Format

Defines a DDM configuration input file.

Domain Decomposition Method (DDM) Model Output File

Defines a DDM output library.

Cross Talk Model Output Format

Adds a cross talk library to a DDM model.

Stochastic Model File Format

Defines an EUV stochastic model file.
