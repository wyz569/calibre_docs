pupilSampling

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Sets the sampling distance in the pupil.

Usage

pupilSampling value

Arguments

value

A required argument setting the sampling distance in the pupil. The pupil is normalized to a unit circle, so a value of 0.02 implies a pupil grid of 100x100 points.

Description

An optional parameter setting the sampling distance in the pupil.

Note:

This optional miscellaneous parameter is used only if the engine is rapidTCC.

The program automatically chooses the exact pupil sampling value, which will generally be better (less) than the specified value. Setting this value is an accuracy versus speed trade-off. Generally, values greater than 0.02 should be avoided because of reduced calculation accuracy. Also, the time penalty for values smaller than 0.005 will be significant. Based on internal testing, the default of 0.01 value is a good compromise of speed and accuracy.
