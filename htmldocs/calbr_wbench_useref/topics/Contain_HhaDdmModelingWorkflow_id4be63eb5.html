HHA and DDM Modeling Workflow

The use of a domain decomposition method (DDM) library is optional for optical model calibration. DDM models are primarily used with HHA.

DDM models are created independently of the optical model using the following flow:

Figure 1. Complete DDM and HHA Flow

  1. Modify the optical model for HHA and DDM.

    The optical model requires some modification to enable the HHA and DDM libraries. See the section “Modifying an Optical Model File for DDM and HHA” for more information.

  2. Use one of the following methods to create a DDM output file:

    • Creating a DDM Library With masksim Batch

      Define a DDM library input file. Use the masksim batch command to create a Domain Decomposition Method (DDM) Model Output File.

    • Creating a DDM Library Using the MaskSim GUI

      Use the MaskSim GUI to create the DDM output file. This GUI is designed to accept user entered values or loaded files.

  3. Optimizing the HHA-Enhanced Optical Model

    Calibrate the modified optical model and DDM file together with modelflow_v2.

  4. An optical model that includes a DDM library requires additional handling depending on which Calibre tool the optical model is being used with. See the section “Using the HHA Library With Calibre OPCverify and Calibre nmOPC” for more information.
