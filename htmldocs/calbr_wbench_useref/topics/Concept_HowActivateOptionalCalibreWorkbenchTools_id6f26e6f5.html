How to Activate Optional Calibre WORKbench Tools

Some Calibre WORKbench tool functionality is only available when you provide a specific command-line option to Calibre WORKbench.

The following table shows the list of tools supported through command-line arguments.

Table 1. Calibre WORKbench Tool Arguments

Argument

Associated Tool

Functionality

-rve [-drc |-dfm | -pex] [rveFile] |

-rve [-lvs | -perc] [rveFile] [topcellname]

Calibre® RVE™

Results viewing tool to inspect RDB databases created during a Calibre run.

-mapi

Any Metrology API (MAPI)-based application

MAPI library-based Calibre WORKbench add-ons are all called with the -mapi option.

Calibre Metrology Interface

Used for Calibre MDPview.
