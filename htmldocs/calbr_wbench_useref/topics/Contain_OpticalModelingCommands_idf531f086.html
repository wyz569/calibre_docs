Optical Model Commands

Optical model file commands require specific inputs and parameters in order to produce optical model-related files.
Table 1. Optical Model Command Descriptions

Command

Description

activatelut

Creates the lookup tables for the specified optical model.

background

Defines the background transmission values of the mask for each exposure.

calc_telecentricity

Reports the telecentricity error of a source map.

clearfieldval

Reports the clearfield value for an optical model.

comparetccs

Compares two TCC matrices.

copyoptical

Copies an optical directory to a new location.

cwb_sample

Selects a set of representative gauges, sites, or clips from an input group.

fb_spectrum2lorentz

Converts laser spectrum file info into Lorentz parameters.

fitmap

Calculates the composite source that best fits the specified source map.

gauge_reduce

Reduces a set of input gauges to a representative subset.

interpolateapodization

Interpolates two apodization files based on their slit positions.

interpolateddmsectors

Creates a DDM custom library from a universal DDM input file.

kde

Computes coverage via Gaussian kernel density estimates of gauge parameters.

kernsym

Reports on model kernel symmetry.

layer

Defines layer characteristics.

masksim

Creates a DDM output file from a DDM input file or a generalized DDM visualization from a generalized input file.

maskspectrum

Generates a mask spectrum file for the specified layout.

mergemap

Combines multiple source map files into a single averaged source map file.

opticsgen

Creates an optical model.

optmatchweight

Modifies gauge weights.

remsap

Removes pupil apodization from a source map.

rotatesource

Rotates a source map.

sectorinfo

Analyzes an HHA file for illumination setting information.

socs2tcc

Converts a SOCS matrix to a TCC file.

socsregen

Regenerates an optical model with a different kernel amount.

writetcctxt

Writes sparse TCC matrix to a tcc.txt file.
