Contour Calibration Work Flow

Contour calibration is used during the resist model calibration process. The contour calibration file is an optional input to the modelflow/VT5 Center GUI (for VT5 resist models) and modelflow_v2/CM1 Center GUI (for CM1 resist models) tools.

The contour input must be given a weight value; it is used in addition to the required sample spreadsheet (VT5 model) and gauge object (CM1 model) data.

Table 1. Contour Calibration Task Flow

Stage

Description

1

Prepare the contour design files by adding contours to the design file with the drawn test structures and adding clip layers.

If you have contours in a GDS or OASIS file, use “Preparing Contour Design Files”.

If the contours are unclear or in an image file, use “Alternate Method to Create Contour Design Files”.

2

Prepare the file that provides the contour information to the resist model calibration tools as described in “Creating a Contour Layer Information File”.

Optional

Before using the contours to improve calibration, verify their quality. See “Contour Fitness Checks”.

3

Supply the contour information when calibrating models, as described in “Model Calibration With Contour Calibration Files”.

Optional

If you have a CM1 resist model and a VEB model, you can use the contours to improve VEB calibration as described in “Working With Input VEB Contours”.
