veb_duplicate

CM1 and VEB Commands

Related Formats: Variable Etch Bias (VEB) Model File Format

Duplicates an already existing VEB object represented by its handle, vebin.

Usage

veb_duplicate vebin

Arguments

vebin

A required argument that specifies a valid VEB Tcl object.

Return Values

This command returns a handle to a VEB object that is a duplicate of the input VEB object.

Examples

set obj [veb_create –file “veb.mod”]
set obj_duplicated [veb_duplicate $obj]
