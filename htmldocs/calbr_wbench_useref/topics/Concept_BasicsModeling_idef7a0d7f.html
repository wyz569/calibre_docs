Basics of Modeling

Each step in IC processing contributes to systematic distortion in the production of a circuit on silicon.

The most significant sources of distortion are:

  • mask writing

  • optical effects

  • resist development

  • etching

To correct for these distortions and improve the design-to-wafer fidelity, the Calibre WORKbench tool uses mathematical models that characterize the systematic component of these distortions through distinct physical processes. These models allow Calibre WORKbench to simulate how a mask will print on the wafer, then adjust the mask so that the resulting wafer image satisfies the design specifications.

About Modeling

Models of physical systems are categorized in terms of how well the system is understood.

  • An informed model is one for which the properties and interactions of all elements of the system are fully understood. With informed models, mathematical functions emulate the behavior of the system completely, and the simulation results have a high degree of accuracy.

  • A black box model is one for which the properties and interactions of the elements are either not understood, or are not quantifiable. With black box models, the mathematical functions are optimized to fit empirical data representing input and output. The drawback with black box modeling is that it only fits the empirical data used to create the model. If that data was complete enough to characterize the system, the simulation results will have a relatively high degree of accuracy, even for situations not within the scope of the empirical data. If the data was too limited, the simulations results will be unpredictable outside of the scope of the empirical data.

  • A white box model is one for which the properties and interactions of some elements of the system are understood or quantifiable, but others are not. Portions of the mathematical functions emulate the system while others are optimized to fit empirical data. White box models are considered a superset of black box modeling because some aspects still depend on empirical fitting to data.

Black Box Modeling in Lithography

For the purpose of lithographic modeling, it is often convenient to view the entire pattern transfer process as a single black box function that encompasses all process steps.

Figure 1. Black Box Model of Pattern Transfer

Modeling with the Calibre RET toolset separates the black box model of pattern transfer into two known subsystems: diffraction optics, and the resist development and etch process. Both of these subsystems are separate, measurable physical processes that occur in lithography.

Figure 2. The Black Box Showing the Two Subsystems

Using the Calibre WORKbench application, you create two models to predict distortion in your lithography process. These are:

  • Optical model — Predicts the intensity of light reaching the resist by modeling the optical system.

  • Process model — Predicts the behavior of the resist in response to the light that reaches it, and the subsequent development and etch process.

The optical model is a white box model. The process model is a black box model. Both models rely on empirical measurement data. Vector diffraction theory describes light diffraction from obstacles and mask structures; vector imaging theory describes image formation inside the thin film stack. Choosing the exact resist and defocus thickness settings is subject to 'tuning', based on empirical data.

Empirical Data for Black Box Modeling

Developing a black box or white box model requires that you begin with observable data. Practically speaking, observable data is anything that you can measure experimentally, including Critical Dimensions (CDs) such as mask CD, resist CD, etch CD, and SEM contours.

To obtain the data, you place a test pattern on a reticle, process the test pattern, then measure the resulting wafer CDs. To ensure that the test pattern data contains enough information to characterize the lithography process and its distortions, you must create specially designed test patterns containing a basic set of geometries that span multiple pitches, CDs, and configurations. The Calibre WORKbench application provides a test pattern generation tool that lets you tailor the test pattern to fit your process and design. The models you create with the Calibre WORKbench application use the distortions yielded by the basic set to predict other geometry distortions.

After you collect the empirical (measured) data from the test pattern you must enter it into the system-generated spreadsheet file, which you then use as input to the Calibre WORKbench modeling tools.
