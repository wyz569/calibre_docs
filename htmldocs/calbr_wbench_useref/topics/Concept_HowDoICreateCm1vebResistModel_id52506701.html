CM1/VEB Inputs

Task Flow

  • Optical model

  • Test pattern file

  • Gauge file (which can be created from spreadsheet data)

  1. Open the CM1 Center tool (Litho > CM1 Center).

  2. If necessary, convert spreadsheet files to gauge objects.

  3. Configure the tool with initial settings, including the optical model file.

  4. Run the tool to create a CM1 Center-compatible optical model file.

  5. Run the tool to create an initial resist model file.

  6. Use the analysis graphs to evaluate the model and rework parts of it that contain process variations.

  7. Tune the resist model until it is stable.

CM1 Model Commands

Description

modelflow_v2

Generate optical, CM1 resist, and VEB etch models.

clipLayout

Writes out the gauges within the specified clip to a layout file.

cm1compose

Creates a template CM1 object based on a modelform.

cm1copy

Copies a CM1 object to a specified object.

cm1duplicate

Duplicates a CM1 object.

cm1getparam

Retrieves a parameter value from a CM1 object.

cm1load

Loads a CM1 object from the disk.

cm1setparam

Sets a CM1 object parameter.

cm1stringload

Creates a CM1 object from a given string literal.

cm1write

Dumps the CM1 object to a string and/or file.

gauge2layout

Writes gauges to a layout.

gaugeGet

Gets data about a row in a gauge object.

gaugeMeasure

Measures gauges on a specified layer.

gaugeRead

Reads a gauge file into a layout.

gaugeResize

Resizes the length of a gauge.

gaugeSet

Sets data for a specified gauge or downsizes a gauge object.

gaugeWrite

Produces a gauge file from a gauge object.

gmWrite

Writes gauge measurements to a gauge file.

layout2gauge

Transforms wires to gauges.

sgd_add

Appends a gauge data object a super gauge object.

sgd_cget

Get parameter values for a gauge object inside a super gauge object.

sgd_config

Configures a data item inside a super gauge object.

sgd_create

Creates a super gauge object.

sgd_delete

Deletes a gauge object from inside a super gauge object.

sgd_from_sss

Converts a super spreadsheet file to a super gauge object.

sgd_opticsgen

Generates optical models from the super gauge object’s defocus values.

sgd_resize

Resizes gauge lengths inside a super gauge object.

sgd_save

Saves a super gauge object to a file.

ss2gauge

Produces a Tcl gauge object from a sample spreadsheet object.

unset

Destroys a gauge or super gauge object.

veb_copy

Copies a VEB object to another VEB object.

veb_create

Creates a VEB object.

veb_duplicate

Creates a duplicate of a VEB object.

veb_getparam

Retrieves a parameter from a VEB object.

veb_setopt

Sets the optical model to the specified VEB etch model.

veb_setparam

Sets a parameter inside a VEB object.

veb_write

Writes a VEB object to a string and the disk.
