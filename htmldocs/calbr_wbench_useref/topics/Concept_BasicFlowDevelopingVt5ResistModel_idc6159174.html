Tools for VT5 Model Development

Table 1 shows the information on the primary tool choices for VT5 model development.

Table 1. Differences Between VT5 Center, and modelflow v1

Tool

Purpose

Strengths

Drawbacks

VT5 Center

A GUI tool that calibrates by automatic search.

  • Can optimize multiple variables in the same run.

  • Can create and import multiple sample spreadsheets for process window optimization.

  • Can calibrate using contours.

Coarse control over model calibration.

modelflow (v1)1

Batch command version of the VT5 Center tool.

Allows for calibration and optimization without loading the GUI.

Lengthy configuration file if done by hand; you can also use the output of the VT5 Center Tool in batch mode.
