Dense Modeling Workflow

The following sections describe the modeling work flow for the CM1 and VEB models.

CM1 Modeling Work Flow

While the methodology for creating CM1 models is different from creating CTR, VTR, and VT5 models, the final result is a resist model that you use in the same way as any other resist model, except that CM1 is designed for use with Calibre nmOPC and Calibre OPCverify, and does not work with Calibre OPCpro.

VEB Modeling Work Flow

Create the VEB model after creating a CM1 model (see Figure 1). Both the CM1 Center tool and modelflow_v2 batch tool tune VEB to empirical data after CM1 optimizations are performed.

CM1 Modeling Versus VT5 Sparse Modeling

You can create a CM1 model in either the Calibre® WORKbench™ CM1 Center GUI or in batch command mode.

CM1 differs from the VT5 approach in that the VT5 methodology uses a variable threshold printing test at each layout location (as a function of image parameters, densities, and so on), and CM1 methodology fits to a single global threshold (as a function of neutralization and diffusion).
