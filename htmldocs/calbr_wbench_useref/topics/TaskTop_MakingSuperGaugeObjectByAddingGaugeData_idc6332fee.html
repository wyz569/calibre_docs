Making a Super Gauge Object by Adding Gauge Data

Multiple gauge files can optionally be concatenated into specialized dose- and focus-specific instances; these are known as “super gauge objects.”

Procedure

  1. In CM1 Center, selecting the Gauges > Append SS menu item allows you to add an additional sample spreadsheet file as a gauge object.

    or

    In CM1 Center, selecting the Gauges > Append menu item allows you to add a previously-saved gauge file as a gauge object.

    A new tab (Gn) appears at the bottom of the Gauges tab for the sample spreadsheet.

    Tip

    If you have more than 32 gauge tabs, click and hold the middle mouse button, then move the mouse left or right to scroll the visible tab set.

  2. Assign a defocus, dose, and weight value for the new gauge object (Figure 2). These values will be used to place the data in the gauge file in the proper location in a Bossung graph. Note that super gauge files can be used in CM1 Center even if they model only a single dose and defocus process condition.

    CM1 Center allows you to easily edit the dose, defocus, and weight values of the combined Focus Exposure Matrix (FEM) for the entire super gauge object in the Edit Super Gauge FEM dialog box (Figure 1). To use this feature, select the Gauges > FEM > Edit menu item.

    Figure 1. Edit Super Gauge FEM Dialog

    • Each row in the dialog box corresponds to one of the loaded gauge tabs. Editing the line changes the tab values when you click OK.

    • Selecting a row in the dialog box and clicking See switches the corresponding gauge tab to be the active tab.

  3. Optionally, add a precision tolerance in the Prec column (-1 is the default, which does not perform precision tolerance checks). The number of points that are within the precision value (| measured - simulated | <= precision) are output to the log file after an optimization run.
  4. Optionally, link the gauge file to pre-built optical models for that gauge file by using the Gauges > Link menu option, or pre-build and link models you select using the Gauges > Pre-build and link menu option.

    Linked optical directories associated with a gauge object can be used with the (T)ry and linked optical model modes (see “CM1 Center Optical Operating Modes”) during optical model tuning.

    Tip

    Up to three optical models can be linked to a gauge object; during calculations, the optical models are varied according to the dose and defocus values specified in step 2. Second optical models are used only to model double exposure systems.

    Note:

    If you are linking optical models, select the optical model directory, instead of the contents of the directory.
