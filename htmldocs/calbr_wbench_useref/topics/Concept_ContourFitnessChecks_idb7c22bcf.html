Contour Fitness Checks

It is a good idea to check the fitness of the imported contours when using them as a calibration aid. The following methods are available for viewing the fitness effects of a contour:

For VT5 modeling:

  • Use the ccalFitness command to quickly return a contour fitness EPE value. This command also applies additional contour layers to the open design file.

  • Search the VT5 Center log for the heading “BEGIN STAGE 1/2/3 CONTOUR DATA.” At this location, information about each contour block (see Table 1) is contained in the log, sorted by contour set.

  • Use the Analysis graphs: Calibre WORKbench adds special contour specific graphs to the Analysis tab, such as the Delta EPE (Equivalent) for All Contour Blocks graph in the Main tab. Each relevant graph shows information sorted by contour block. (See the section “VT5 Center Contour Calibration Support” for more information.)

For CM1 modeling:

  • Use the analysis graphs: Calibre WORKbench adds special contour-specific Contour Fitting sub tabs to the Optics Fitness, CM1 Fitness, and VEB Fitness tabs.

  • Run an optimization with a contour layer loaded and inspect the log results for the section “BEGIN POST STAGE 1/2/3 CONTOUR INFO.” At this location, information about each contour block (see Table 1) is contained in the log, sorted by contour set.

Table 1. Contour Information Log Output

Field

Description

rmsError

Calculated sum rmsError for the contour

meanError

Calculated mean error for the contour

sumWeight

Sum of the weight values applied for the contour

EMPX, EMPY

Empirical X and Y coordinates

MODX, MODY

Modeled X and Y coordinates

ERROR, CONTRAST, ILS_M, ILS_S, ILS_AVG

The amount of error (difference) between measured and simulated coordinates, for different metric types.

Tip

Sorting on these columns will find the largest EPE discrepancies.

WGHT

The weight value applied to this specific data point

IMAX, SLOPE, IMIN

The data values for the sample data point

The following fields only appear in VT5 modelflow logs:

ISLOPE, FACTOR

The VT5 data values for the sample data point

D1, D2

The Gaussian polynomial values for the sample data point

EMPTHRESH

Empirical threshold

MODTHRESH

Model threshold

EMPEPE

Empirical EPE

MODEPE

Model EPE

THMODEPE

Threshold model EPE
