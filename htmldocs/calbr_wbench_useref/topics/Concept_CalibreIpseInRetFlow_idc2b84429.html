Calibre IPSE in the RET Flow

The Calibre IPSE tool is used during the process model calibration phase.

You run the Calibre IPSE tool after:

  • Generating test pattern data

  • Generating process data for a design

Use missed coverage data found with the Calibre IPSE tool with the Calibre IPSETX tool in order to create a new test pattern with the additional data. You will then re-run your additional test pattern through your process, analyzing the new results in Calibre Modelflow.

Once you are satisfied with the output from VT5 Center, modelflow, CM1 Center, or modelflow_v2, you can use them to simulate further designs.
