Including Contour Calibration Information

This optional procedure describes how to use the VT5 Center CLI, in tab.

Use this tab only if you have contour data available. If you do not have this information, skip to “Loading an Optical Model”.

Figure 1. VT5 Center GUI (CLI Tab)

Prerequisites

  • While not a requirement, creating a CLI file before performing this task is recommended. Further information on contour calibration files can be found in this manual, in the appendix “Importing Contour Calibration Files”.

  • A design file with contour layers included.

  • Information on the contour layers in the design file, such as dose and defocus values, and layer numbers.

Procedure

  1. In VT5 Center, select the CLI, in tab.
  2. Fill in the template with one of the following methods:

    • Choose Contour > Open to load a pre-generated CLI file.

    • Choose Contour > Template to start from a template.

  3. Set the weight for the contour data in the upper right-hand corner of the tab.
  4. Proceed to “Loading an Optical Model”.
