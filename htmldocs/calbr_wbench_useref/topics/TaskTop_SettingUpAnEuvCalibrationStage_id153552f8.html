Setting up an EUV Calibration Stage

Stages represent simulation and calibration commands in a job. When you use the Flow Stage Wizard, you define simulation commands and set parameters and optimizer settings for an optimization run. Calibration jobs can be run independently, or can be linked with the results of another calibration job.

Prerequisites

  • Calibre WORKbench invoked, and the Calibre nmModelflow tool running

  • An active Lithomodel file in the database

Procedure

  1. Raise the Database Browser tab in the list of primary display tabs.
    Figure 1. Starting the Flow Stage Wizard

  2. Click Stage in the left column.
  3. Click the Add Calibration Stage button.


  4. In the Select Stage Type wizard, choose the type of stage to create, then click Next.
    Figure 2. Select Stage Type Dialog

    Table 1. Flow Stage Wizard Choices for EUV

    Type

    Description

    Calibrate

    • Mask

    • Optics

    • Zplanes

    • Resist

    • EUV Black Border

    • Etch

    Sets up a calibration run.

    • Choose Optics to calibrate the EUV optical model.

    • Choose Mask to calibrate the mask model (see Mask Model Format).

    • Choose Resist to calibrate the resist model.

    • Choose EUV Black Border to calibrate the black border model.

    • Zplanes, Topo, and CDSA are not used in EUV.

    Simulate

    Sets up a simulation run.

    Defocus Start Search + Center Focus on Through-Focus Data

    Adjusts both defocus_start and beamfocus as a single stage.

    Defocus Start Search + Center Focus on Nominal Data

    Similar to the Defocus Start Search + Center Focus on Through-Focus Data option, but for single focus conditions.

    Center Focus By Bossung Adjust on Through-Focus Data

    Adjusts the beamfocus by minimizing the mean focus shift value after running a Bossung filter on the initial data.

    Center Focus By Bossung Adjust on Nominal Data

    Similar to the Center Focus By Bossung Adjust option, but allows you to specify a desired focus center.

    Beamfocus/Defocus Start Full Search + Center Focus on Through Focus Data

    (Formerly MGC Best Practice for Beam Focus / Defocus Calibration)

    Sets up a beamfocus and defocus calibration run with preset commands, using best practice settings that are current for the release.

    These commands require focus-sensitive process window gauges. The stage uses a CTR model and applies Bossung adjustments after calibration.

    Build

    • Threshold Optimize

    • Build Resist

    • Build Etch

    • Stochastic Optimize

    • Build Topo

    Stochastic Optimize sets up a run where the included stochastic model is calibrated for use with the associated CM1 model.

    All other options are not used in EUV.

    Etch Bias Table Generation

    Not used in EUV.

    Center and Resize Gauges for Optimal Runtime

    Note:

    Using this option is recommended.

    When selected, this option adjusts gauge lengths to optimize runtime while maintaining accuracy.
  5. Fill out the wizard pages that appear relevant to the stage type you are creating. Each calibration stage type has a parameters setting page and a search optimization page. Depending on which stage you are creating, set the following best practice values:
    1. Run #1 (Beamfocus, Defocus Start) — Choose Optics calibration. Select the Exposure 1 checkbox for Beamfocus/Defocus Start.

      Parameter

      Recommended Value

      Notes

      BEAMFOCUS

      auto auto 15

      Requires change from default.

      DEF_START

      auto auto 15

      Requires change from default.
    2. Run #2 (Imagediffusion) — Choose Optics calibration, and also select the Exposure1 checkbox for Image Diffusion and the 1 Gaussian radio button.
      Table 3. Recommended Settings for EUV Optics Stage Wizard (Imagediffusion)

      Parameter

      Recommended Value

      Notes

      IMAGEDIFFUSION

      0 0.02 15

      Requires change from default.
    3. Run #3 (Mask Bias) — Choose Mask calibration, and set the Specify Number of Lithomodel Mask Layers dropdown to equal to the number of mask layers you have. The default setting of 2 gives you mask layers 0 and 1; however, if your mask bias is the same for all layers, set the field to 1.
      Table 4. Recommended Settings for EUV Mask Stage Wizard (Mask Bias)

      Parameter

      Recommended Value

      Notes

      MASK_MODEL_LAYERM_BIAS

      Use defaults

      Typically, all mask layers are calibrated as dependent on layer 0.

    4. Run #4 (Mask Cornerchop) — Choose Mask calibration, and set the Specify Number of Lithomodel Mask Layers dropdown to equal to the number of mask layers you have. The default setting of 2 gives you mask layers 0 and 1; however, if your mask bias is the same for all layers, set the field to 1.
      Table 5. Recommended Settings for EUV Mask Stage Wizard (Mask Cornerchop)

      Parameter

      Recommended Value

      Notes

      MASK_MODEL_LAYERM_CX

      0.002 0.01 10

      Requires change from default.

      Convex corners are calibrated dependent on layer 0.

      MASK_MODEL_LAYERM_CC

      0.002 0.01 10

      Requires change from default.

      Concave corners are calibrated dependent on layer 0.
    5. Run #5 (Resist) — Choose Resist calibration, and set the Select Resist Modelform dropdown to 21 if you have a full set of features available, or to 11 if you only have 1D features available.
      Table 6. Recommended Settings for EUV Resist Stage Wizard

      Parameter

      Recommended Value

      Notes

      S parameters

      Use defaults

      Earlier versions of the EUV model calibration flow recommended changing default values for S3 to S6. This is no longer required.

      Maxaidiff

      10 stable

      This is the default setting.

      Thresholdtolerance

      1.5

      This is the default setting.
    6. Optional Run (EUV Black Border) — Select EUV Black Border calibration. Note that setting the Symmetric Black Border checkbox changes the model parameters such that the left and right and top and bottom pairs are dependent on the left and top parameters, respectively.
      Note:

      If your litho model does not contain a black border model but you selected the EUV Black Border stage to calibrate, a generic black border model is automatically generated when this stage is run. However, it also requires you to issue the following command in the Command window:

      mdf setupfile create -euv_field_center <xcoord> <ycoord> 
-bbm_field_size <width_um> <height_um>
    7. Optional Run (Bossung Adjust) — Select the Center Focus By Bossung Adjust option if you intend to use a Bossung-adjusted gauge data file that contains focus-sensitive gauges. Set the parameters as shown in Table 7.

  6. The final screen of each Flow Stage Wizard displays the commands that Calibre nmModelflow runs when you click Finish.
  7. Repeat steps 3 through 6 until you have created the following stages:

    • Required — Optics x2, Mask x2, Resist

    • Optional — Black Border, Bossung Adjust

Results

Calibre nmModelflow adds an entry to the Stages database for each stage you create. These stages are used in the task, “Setting up EUV Calibration Jobs””.
