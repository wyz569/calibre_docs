setMeasurementStatistics

CM1 and VEB Commands

Related Formats: Gauge Object Reference

Computes confidence intervals from gauge object values.

Usage

setMeasurementStatistics gauge_object

Arguments

gauge_object

A required argument that specifies an existing gauge object to calculate the 95% confidence interval for.

Description

Computes the 95%ci (95 percent confidence interval) column in the gauge file, based on the gauge object values for stdev and count for enabled gauges. If an enabled gauge does not have both the StDev and Count values supplied, they are filled in with the mean value for stdev and count averaged over all available enabled gauges with supplied values.

This command returns an error if no stdev or count values are available for any enabled gauges.

Examples

#read gaugedata from file
set bobj [gaugeRead -f gauges.gg]
#set measurement statistics
setMeasurementStatistics bobj
