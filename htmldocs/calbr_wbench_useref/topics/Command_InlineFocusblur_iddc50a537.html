inline_focusblur

Parent topic: Optical Parameters File Format

Input for: opticsgen, modelflow (Version 1), modelflow_v2

Defines focus errors in the Optical file.

Usage

inline_focusblur inline_id ‘{’
   focusblur_block
‘}’

Arguments

inline_id

A required argument specifying an inline focusblur block by its identifier inline_id. The inline focusblur block identifier must be referenced by either an ssinline or lsinline argument in a focusblur command.

focusblur_block

An optional argument set specifying focus errors captured in either a scanner slit file or laser spectrum file. The block uses the same syntax as a file read from disk. See focusblur for descriptions of the file syntax. The block must be enclosed with braces.

Description

This optional parameter enables specifying focus errors in the Optical file instead of loading a separate file from disk. This parameter supports the content found in a scanner slit file or laser spectrum file.
