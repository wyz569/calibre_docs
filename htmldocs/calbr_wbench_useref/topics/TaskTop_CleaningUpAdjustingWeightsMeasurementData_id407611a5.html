Cleaning up and Adjusting Weights for Measurement Data

Cleaning up measurement data involves adjusting the sample spreadsheet so that invalid measurement points are not included in the calculations. Weighting important measurement points causes them to be valued more than others.

Prerequisites

Successful completion of the task “Pre-loading Model Generation Data”

Procedure

  1. In the Sample tab, disable rows with no measurement data, NA status, or bad data by selecting the row or rows, either manually or by using the Select dialog box as shown in Figure 1, and clicking Disable.
  2. Weight critical data points by selecting a row, clicking the Weight button, and adjusting the weight as needed.

    • Weight 1D semi-dense and isolated feature structure types at 60

    • Weight 1D dense feature types at 30 and 2D feature types at 1

      Note:

      For a full list of supported test pattern feature types, see the section “Feature Types and Associated Keywords” in “FEATURE Keyword”.

      Figure 1. Selecting and Weighting Data Points

  3. Save the adjusted sample spreadsheet file (Sample > Save As SS File). If you use a different filename, remember to load the updated sample file on later runs of VT5 Center.
  4. Proceed to create a resist model using VT5 Center. See the instructions in the section “Model Optimization With the VT5 Center GUI”.
