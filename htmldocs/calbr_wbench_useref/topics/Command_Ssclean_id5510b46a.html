ssclean

VT5 Resist Model Commands

This command goes through the spreadsheet file and either reports or fixes non-centered sample sites.

Usage

ssclean {fix | soft_fix | report} -ss input -f setup {-m gds | -l handle} [-o output]

Arguments

fix | soft_fix | report

Selects the command mode:

  • fix — Fixes off-edge sample points.

  • soft_fix — Fixes sample points, but does not center them at the middle of the fragment.

  • report — Does not fix sample points, but reports the bad data points in the output.

-ss input

The input spreadsheet to be fixed or reported about.

-f setup

The setup file.

-m gds | -l handle

GDS test pattern file or a layout object handle (cannot use both).

-o output

The output spreadsheet to create (only in fix mode).

Examples

An example of the results of a fix are shown in Figure 1. In this example there are two sites: the first is “fixed” by moving it to the center of the edge it projects onto. The second one cannot be fixed because it is unclear which edge the sample should be placed on. When the command encounters a sample sites that cannot be fixed, it reports this with an error message. Such unfixable sample sites should be examined in more detail by the user.

ssclean fix -ss sample.ss -m tp.gds -f setup.in
