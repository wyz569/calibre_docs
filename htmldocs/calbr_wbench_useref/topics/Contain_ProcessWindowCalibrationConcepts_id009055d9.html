Process Window Calibration Concepts

The following section explains some background behind process window calibration.

Single spreadsheet models assume a nominal centered dose and focus value for gathering pattern data. This assumption implies that the scanner used to expose the test pattern data remains consistent; however, many variables have an impact on the final printing image that cannot be accounted for in single nominal dose and focus values.

Process window calibration approximates this variance by using test pattern data at a range of dose and focus values; this is also known as a Focus Exposure Matrix (FEM). Creating a Bossung plot of the defocus values versus the resultant CD on a per-exposure-dose basis approximates a box within which all allowable CD values fall. This box is the process window, which describes the maximum dose and defocus variations that the process can accommodate.

You can use super spreadsheet files (multiple sample file spreadsheets at different dose and focus settings) to run model fitting over a process window in VT5 Center. Models created using calibration data from a variety of processing conditions lead to a greater predictive accuracy, as well as giving you the ability to view the potential issues from changes to the dose or defocus values. Fitness values for each process condition are determined and plotted, allowing you to visually see where they have process limitations.
