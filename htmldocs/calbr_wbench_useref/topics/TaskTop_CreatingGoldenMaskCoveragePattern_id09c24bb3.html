Creating a Golden Mask Coverage Pattern

Use Calibre IPSE to create a suggested sample spreadsheet from your golden mask file by following this task.

Prerequisites

  • A golden mask Modelflow (v1 or v2) log file

Procedure

  1. Load the golden mask Modelflow log into Calibre IPSE using the File > Open MfLog menu item.
  2. Select the Tools > Golden Mask > Suggest Model Coverage menu item. The following dialog box appears:
    Figure 1. IPSE Suggest Model Coverage Dialog

  3. Change the Golden Mask Id field to match the proper line in the files list.
  4. Similar to the task “Including Golden Mask Patterns in the Output”, use the Grid Size settings to limit the intervals of the grid selection criteria.
  5. Set an output name for the new sample spreadsheet file.
  6. Click OK.

Results

Calibre IPSE outputs a coverage pattern for the golden mask pattern, including a grid showing the granularity of the Grid Size settings for selections.
