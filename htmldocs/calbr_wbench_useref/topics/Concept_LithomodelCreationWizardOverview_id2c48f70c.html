Lithomodel Creation Wizard Overview

Litho models are a collection of models. You can specify one litho model instead of loading each type of models individually. Litho models are a key component of Calibre WORKbench. Calibre tools that use litho models can load all necessary models with a single command.

The Lithomodel Creation Wizard creates a litho model from optical, resist, and other process models you specify. Other ways of creating a litho model include specifying it as an output in the CM1 Center GUI or a modelflow_v2 script when creating new models, or using the Calibre WORKbench command “lithomodel create”.

Lithomodel Creation Wizard Workflow

The Lithomodel Creation Wizard creates litho model directories for the Calibre post-tapeout tools such as Calibre nmOPC, Calibre OPCverify, and Calibre nmSRAF. It is invoked from the Calibre WORKbench GUI using the Litho > Litho Model menu option. You use the Lithomodel Creation Wizard to navigate to models, and it creates the litho model directory and key file when you have finished entering relevant information. A key file contains pointers and links to directory contents.

The flow comprises three distinct sections:

  1. Set up the unique elements, such as the output directory and resist model. As part of this step, you set the number of exposures. For each exposure, Steps 2 and 3 are repeated.

  2. Choose optical models. An exposure can have multiple optical models, which can be pregenerated or generated by Calibre WORKbench.

  3. Define mask layers. An exposure can have multiple layers defined. The litho model mask layers will need to be listed in SVRF statements as input layers..

Lithomodel Creation Wizard Key Concepts

Certain concepts are important to using the Lithomodel Creation Wizard.

  • Lithomodel file — A file, which is named Lithomodel, located in theLithoModel directory. It contains information on all the models in that directory. The Calibre post-tapeout tools load this file and use it to locate other models in the directory.

  • Exposure — Up to 12 exposure conditions are supported.

    • In Lithomodel, an exposure is designated “mask n” where n is the exposure number, starting with 0.

    • Each exposure can have its own set of optical models and mask layers.

    • The same resist model is used for all exposures.

  • Nominal Optical Model — A pre-existing optical model generated at the nominal dose and defocus condition. A nominal optical model must be specified for each exposure.

  • Previously Computed Optical Model — After choosing “Add Defocus Model using existing model on disk”, a new exposure page refers to a choice of adding “previously computed optical models.” These optical models are used for process window functionality, and should be based off of the nominal optical model and varied by focus condition.

  • Focus Conditions To Generate Optical Models — After choosing “Add Defocus levels to Generate new Optical Models”, a new exposure page refers to a choice of adding “focus conditions to generate optical models”. This is an alternative choice to adding “previously computed optical models” and generates defocused optical models. This method requires a single nominal optical model and generates additional models based on defocus values you specify.

  • Mask Layer — Mask layers are one or more design layers associated with an exposure. These layers are the same input layers sent to Calibre OPCverify or Calibre nmOPC, and are associated using the image_options, denseopc_options, or pxopc_options block.

Litho Model Examples

The Lithomodel file is the central listing for any litho model.

Example 1

This example Lithomodel shows a litho model that has seven focus conditions ranging from ‑15nm to 15nm and one nominal condition, with two mask layers included that have cornerchop and bias specified.

version 1
resist resist.mod
mask 0 {
  optical Optical_nom
  optical optical_M1_F-15.opt focus -15nm
  optical optical_M1_F-10.opt focus -10nm
  optical optical_M1_F-5.opt focus -5nm
  optical optical_M1_F5.opt focus 5nm
  optical optical_M1_F10.opt focus 10nm
  optical optical_M1_F15.opt focus 15nm
  background 0.0225 -0.0097
  mask_layer 0 TRANS 0.95 0.00925 CX 0.0032 CC 0.0032 BIAS 0.0005 DDM ddm_M1_L0.ddm
  mask_layer 1 TRANS 0.95 0.00925 CX 0.0032 CC 0.0032 BIAS 0.00010 DDM ddm_M1_L1.ddm
}

Example 2

This example Lithomodel file features multiple mask layers, but only a single optical, resist, and VEB etch model.

version 1
resist resist.mod
etch veb1.mod
mask 0 {
  optical optical_test
  background  0.0225 -0.024
  mask_layer 0 TRANS 0.975 0.0212 CX 0.0075 CC 0.0075
  mask_layer 1 TRANS 0.975 0.0212 CX 0.0075 CC 0.0075
  mask_layer 2 TRANS 0.975 0.0212 CX 0.0075 CC 0.0075
  mask_layer 3 TRANS 0.975 0.0212 CX 0.0075 CC 0.0075
  mask_layer 4 TRANS 0.975 0.0212 CX 0.0075 CC 0.0075
}
