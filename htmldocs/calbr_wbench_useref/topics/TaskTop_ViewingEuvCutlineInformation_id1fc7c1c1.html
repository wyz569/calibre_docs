Viewing EUV Cutline Information

Use the RET Flow Tool in Calibre WORKbench to view cutline information for EUV.

Restrictions and Limitations

  • This task is intended for use with long range flare data only.

Prerequisites

  • Calibre WORKbench invoked

  • A design file containing EUV long range information, loaded into Calibre WORKbench

  • The corresponding flare map file

  • The related litho model

  • A Calibre OPCverify setup file that includes the flare_longrange model, similar to the following example code:

    modelpath ./models
layer M1
tilemicrons 40
optical_transform_size 768
imagegrid aerial 3 8
flare_longrange litho_md png flare_map.png
image_options MDF {
	layer M1 visible mask_layer 0
 	litho_model litho_md}
setlayer im = image MDF

Procedure

  1. In Calibre WORKbench, open the RET Flow Tool (Litho > RET Flow Tool).
  2. Create a new Calibre OPCverify session. Right click the session and select Open Setup, selecting the Calibre OPCverify setup file.
  3. Click Options underneath the ReCutline button and select the Flare option.

  4. In the Calibre WORKbench design window:
    1. Zoom in until the relevant test structures are visible.
    2. Click the Cutline button, then click the start and end points to draw a line across a test structure.

Results

The flare cutline appears in the Edit Cutline window.
