Exporting CM1 Center Settings as a modelflow_v2 Run Script

Exporting a run script can be used to run modelflow_v2 instead of reconfiguring CM1 Center every time. Use this procedure to create a run script.

Prerequisites

  • A completely configured CM1 Center that successfully generates at least an optical model log.

  • The test pattern design file used to create the gauge files.

Procedure

  1. Invoke Calibre WORKbench from the working directory containing your setup file and load the test pattern design file.
  2. Configure CM1 Center with known good working arguments or load a CM1 Center session file (.csf).
  3. Optionally, test the session by clicking the Run button.
  4. Save the session as a run file by selecting File > Export Run Script.
  5. Name the file prefix when prompted in the Export Run Script dialog box and click OK.

Results

A file named <prefix>_mfv2_lc.mfr will be added to the directory you invoked Calibre WORKbench from. The script can be run with the following command:

calibrewb -shell -s prefix_mvf2_lc.mfr
