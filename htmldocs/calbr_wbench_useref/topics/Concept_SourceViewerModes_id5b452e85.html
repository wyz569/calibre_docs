Source Viewer Modes

The Source Viewer (Optics > Source View) supports multiple types of optical model files:

  • Unpolarized source optical models, such as the ones generated using the tasks in this chapter.

  • Polarized sources (Figure 1).

  • Sector visualization.

  • Source file formats with .smf, .src, and .xml extensions.

Additional types of optical model file viewers are available in read-only mode:

  • Optical kernels (.bin) (Figure 1 in Polarized Source Maps), accessible using Optics > Kernel View

  • Jones pupil (.jp) files (Figure 1 in Jones Pupil View) accessible using Optics > Jones Pupil View

    Figure 1. Viewing a Polarized Source Map

For unpolarized sources, one graph is displayed, and the following options are available:

  • Source manipulations can be generated by choosing File > New Source Map and saved by choosing File > Save Source Map, which saves the source map in PROLITH™1-compatible format. This is useful for creating symmetrized source maps from measured data.

    Source maps can be exported in MATLAB™2, CSV, and Nikon formats.

    Tip

    The Generate controls appear in a flyout panel that appears on the right side of the Source View tool. This flyout only appears when you choose File > New Source Map.

  • Sources using the ASML ADEL file format with an .xml extension can be imported into the Source View tool and saved as a source map (.smf) file. Open the XML source file by selecting File > Open Source Map, change the File type to All Files (*), and open the XML file.

  • To change the output to have values for all points, select the Options > Save Zeros option. Choosing this option includes 0s in the .src file output. This is sometimes required to read in the file in the ASML rending software.

  • If a source contains pixel intensity variations, the Source View tool automatically calculates a Pupil Fill value and displays it at the bottom of the pupil image.

  • View controls are available at the bottom of the window as Map Options.

  • Asymmetric source maps can be rotated by clicking the Rotate button. A dialog box appears, giving you the option of rotating the source map counterclockwise (90) or clockwise (-90).

  • Some source maps can be viewed in log scale if the proper measurement data is found in the source file.

  • The Range Min and Range Max fields set the graph scaling for the selected plot(s).

    Note:

    Some changes to the viewing area only are applied when you click the Apply to selected button (A in Figure 2).
