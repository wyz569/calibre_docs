Bossung Graphs for Super Gauge Objects

If a super gauge data object is a multiple copy of the same data set, but at different process conditions (dose and defocus values vary), this will enable you to view the resulting process window for a chosen test pattern structure across dose and focus variations.

Tip

The Regexp field at the bottom of the window can be used to change the highlight color and symbol shape of any gauge line. Type the name of a gauge in the field (such as #4_M) and then select options to change. Your selections are immediately applied.

  • Choosing Gauges > Bossung Plot in CM1 Center shows the graph for one or more individually-selected test pattern structure lines. You can see which dose and defocus condition created a data point by selecting it and clicking See Gauge. The corresponding tab in CM1 Center will be raised with the gauge row highlighted.

    Note:

    Choosing the Next Field Options Append radio buttons allows you to accumulate Bossung plots.

    Figure 1. Bossung Plot

  • Choosing Gauges > Bossung Compare in CM1 Center allows you to compare Bossung curves for several multiply-selected test pattern structures in a single graph.

    Figure 2. Bossung Compare Plot
