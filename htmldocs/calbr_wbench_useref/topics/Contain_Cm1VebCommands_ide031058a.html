CM1 and VEB Commands

The following table shows the list of CM1 and VEB commands.
Note:

The sgd Tcl commands use handles to sgd Tcl objects rather than the objects themselves. These handles can be used as Tcl variables; they require a dollar sign ($) in front of them to be used. The only exception is the sgd_cget command. When used with the ‑gaugeobject option, the sgd_cget command returns a gauge data object which must be used (passed to other Tcl commands) without a $ sign.

Table 1. CM1 and VEB Command List

Command

Description
clipLayoutFlattens and clips the given layout.

cm1copyCopies CM1 objects.
cm1duplicateDuplicates CM1 objects.

cm1loadLoads a CM1 object.
cm1mflistLists the currently supported modelform formats for CM1 models.
cm1setparamSets a CM1 object parameter.
cm1stringloadCreates a CM1 object from a string literal.
cm1writeWrites a CM1 object to a string and, optionally, to a file.
createsrafgaugesGenerates a gauge or super gauge data file with SRAF image data from a SEM image.

gaugeAddAdds one or more gauges to the specified gauge object.
gaugeConvertConverts pre-2009.1 gauge objects to the current version.
gaugeCreateCreates an empty gauge object.

gaugeSetResizes or resets a gauge object.
gaugeWriteProduces a gauge file out of a gauge object. The output gauge file format has a syntax similar to that of a spreadsheet file. The output gauge file format is the following:flag structure_name row col x1 y1 x2 y2 loc drawn other meas simulated precision weight mode count dist [# comment]

sgd_addAppends a new gaugedata set to an existing sgd object.
sgd_bossung_selectIdentifies and selects focus-sensitive gauges in an input sgd object.
sgd_centerCenters gauges at the midpoints of associated drawn geometries.
sgd_cgetGets parameter values from a specified sgd object or an entire gauge data object.
sgd_cliplayoutOutputs areas of interest within clipping distance of gauges.
sgd_configSets parameters for a gauge object inside a super gauge data object.
sgd_convertConverts a pre-2009.1 super gauge object to the current version.
sgd_createCreates an empty Tcl object, intended for use as a super gauge data file.
sgd_create_group_specsGenerates default group specs.
sgd_deleteDeletes a gaugedata set from a sgd Tcl object.
sgd_from_sssConverts a super spreadsheet file to a Tcl super gauge object and returns a Tcl handle to this object.
sgd_get_error_statComputes error statistics for an input super gauge data object.
sgd_get_flareComputes flare values from a specified super gauge data object. Use the sgd_get_flare command to get flare values.
sgd_get_prediction_error_statReports the printing prediction error statistics for the input super gauge data object.
sgd_measureMeasures mask CDs on the specified layer and puts the measurements into the specified gauge record field or column.
sgd_mergeMerges two gauge or super gauge data objects or files.
sgd_opticsgenGenerates optical models for a super gauge data object.
sgd_rescaleRescales the specified super gauge object to a new size factor.
sgd_resizeResizes the length of a specified gauge or all gauges of the input super gauge data object.
sgd_saveSaves the contents of the specified sgd Tcl object to the specified file.
sgd_set_meas_statComputes the 95%ci (95 percent confidence interval) column based on the super gauge object values for stdev and count for enabled gauges. If an enabled gauge does not have both the stdev and count values supplied, they are filled in with the mean value for stdev and count averaged over all available enabled gauges with supplied values. This command returns an error if no stdev or count values are available for any enabled gauges.
sgd_set_rmssd_weightSets the weight of enabled gauges in a super gauge data object using its measurement statistics.
sgd_to_tableOutputs the supplied super gauge object to a table string, and if supplied, the specified file.

ss2gaugeCreates a gauge object from a sample data spreadsheet file.
unsetDeletes a CM1 object using its handle.
veb_composeCreates a VEB template object. The VEB object can be optionally output to a file, a Tcl variable for later use, or both.
veb_copyCopies an existing VEB object (vebin) to another (vebout). The target VEB object initially represented by vebout is deleted, and vebout becomes a handle to a VEB object identical to vebin. Both vebout and vebin must be valid handles to VEB objects, otherwise an error occurs, and nothing changes.
veb_createCreates a VEB object by loading it from a file or string.
veb_duplicateDuplicates an already existing VEB object represented by its handle, vebin.
veb_getparamGets the value of the specified parameter from an input VEB object.
veb_mflistGenerates the full list of currently implemented VEB model forms.
veb_setoptSets the global optParams etch model to the specified VEB model.
veb_setparamSets a VEB parameter for a loaded or created VEB object.
veb_writeOutputs a VEB object to a string and optionally, to a file if one is specified.
