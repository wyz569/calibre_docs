Calibre WORKbench Invocation

Calibre WORKbench is typically invoked from a Linux®1 command shell. Run Calibre WORKbench using command-line arguments, a run script containing commands, or from the Calibre WORKbench GUI for interactive simulation and model creation tasks.
Note:

Calibre WORKbench requires the CALIBRE_HOME environment variable to be set to point to the location of the Calibre tools installation directory, and $CALIBRE_HOME/bin should be in your environment path.
