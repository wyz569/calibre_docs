Files Used by Calibre WORKbench

Calibre WORKbench has certain standard file types and customary extensions.

The following file types are used by Calibre WORKbench:

Table 1. Files Used by Calibre WORKbench

File

Ext

Purpose

Notes

Optical

none

Contains a description of an optical model.

Can be loaded in the Optical Model Tool or run as an argument to the opticsgen command.

See “Optical Model Reference”.

Optical Kernel file

.bin

Contains a list of optical kernels.

See “opticsgen”.

Contour Layer Information file

.cli

Contains information about contour layer(s) in a loaded design.

See “Importing Contour Calibration Files”.

Design file

.gds

.oas

Contains the chip design. Loaded into the visual part of Calibre WORKbench.

Can be any supported file format.

Gauge file

.gg

Contains CM1 measurement data.

See “Gauge Object Reference”.

Setup file

.in

.setup

Configures the RET Flow Tool.

See “Creating a Setup File in RET Flow Tool”.

Chip Index file

.idx

Used for a VSB 11 chip index file.

See the command entry for “vsbIndex”.

Modelflow (v1 or v2), VT5 Center, or

CM1 Center Output file

.log

Contains reloadable results of a calibration run.

VT5 (VT5 Center) and CM1 (CM1 Center) log files are not compatible with each other.

Resist Model

.mod

Contains a description of a VT5 or CM1/VEB resist model.

See “VT5 Model File Format”.

See “CM1 and VEB Model Formats”.

Metrology Tool file

.mtf

Configures the Metrology Tool.

This command was deprecated in 2009.

See “maitgen”.

Modelflow Run file

.run

.mfr

Configures the VT5 Center Tool.

See “Inspecting Run Options and Running the Calibration”.

Super Gauge file

.sgd

Contains multiple gauge files at different process conditions.

See “Super Gauge Data File Format”.

Calibre WORKbench configuration file

.signaext

Contains environment settings.

Located in the $HOME/ directory.

Output Source Map file

.smf

Contains additional output from creating an optical model. Used by the Source View tool.

Named illummap.smf by default.

Sample Spreadsheet file

.ss

Contains sparse VT5 measurements.

Can also be used to generate gauge files.

“Sample Spreadsheet File Format”

Super Spreadsheet file

.sss

Contains measurements for multiple sample spreadsheets under different process conditions.

“Super Spreadsheet File”

Test Pattern Format file

.tpf

.TPF

Contains the definition for a custom test pattern.

“Test Pattern File Format”
