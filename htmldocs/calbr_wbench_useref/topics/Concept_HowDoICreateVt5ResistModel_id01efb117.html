Table 1. Calibre WORKbench VT5 Resist Models at a Glance

Objective

Using the optical model generated in the previous step, create a predictive model for the resist and etch development process. Usually this is a VT5 model, but CTR, VTR, and VTR-E models are also covered by this stage.

Process Model Inputs

Task Flow

Optical model

Test pattern file

Spreadsheet data

  1. Open the VT5 Center tool (Litho > VT5 Center).

  2. Configure the tool with initial settings, including the optical model file.

  3. Select a modelform and convolution kernel type.

  4. Run the tool to create an initial resist model file.

  5. Use the analysis graphs to evaluate the model and rework parts of it that contain process variations.

  6. Tune the resist model until it is stable.

Process Model Outputs

What to Look for in Results

VT5 resist model log file

Use VT5 Center to view results:

  • Look for outliers (data points far outside the EPE tolerance) and remove them.

  • Look for where the eigenvalue cross terms stop being significant.

Process Model Commands

Description

bestdose

Finds the optimized second exposure dose value.

cdsample

Simulates the sites specified in the samfile, then return resulting simulation data.

modelfitness

Runs a simulation using the models defined in the setup file to produce model fitness statistics.

modelflow (Version 1)

Creates an optimized VT5 model and an optimized optical model.

odanalyze

Analyzes errors introduced in the optical model from truncating the optical diameter.

openlib

Loads a GDS file, a setup file, or both into the Calibre WORKbench or Calibre LITHOview application.

resistgenv2

Creates a CTR, VTR, or VTR-E resist model.

samlinev2

Characterizes the aerial image in terms of Imax, slope, and other properties.

scalfit

Measures the fitness value of a SEM calibration contour.

scalcontour

Creates a SEM calibration contour.

setoptoptics

Resets the optical model file in use to be the specified file.

vt5gen

Creates a VT5 model.
