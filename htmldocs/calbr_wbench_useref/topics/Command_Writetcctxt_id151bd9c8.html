writetcctxt

Optical Model Commands

Loads the sparse TCC matrix inside an input optical directory and writes the sparse TCC matrix as a text file.

Usage

writetcctxt -in optical_model -out sparse_tcc_file [-noheader]

Arguments

-in optical_model

A required argument specifying the optical model directory.

-out sparse_tcc_file

A required argument specifying a sparse TCC filename.

-noheader

An optional argument that specifies that the input optical model is encrypted and has no header files.

Description

This command writes the file tcc.txt containing the TCC matrix in sparse format to a directory. In 2017.4, this command replaced functionality that was performed by comparetccs.
