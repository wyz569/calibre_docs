gaugeSet

CM1 and VEB Commands

Related Formats: Gauge Object Reference

Resizes or resets a gauge object.

Usage

Resize mode

gaugeSet –g gauge_object –size new_size

Reset mode using a string

gaugeSet –g gauge_object –row index –data string

Reset mode using individual fields

gaugeSet –g gauge_object –row index [-enabled {0|1}] [-cdmeas string ]
[-cdmeas_etch string ] [-weight value] [–comment string]}

Arguments

–g gauge_object

A required argument that specifies the input gaugedata object that is modified on output.

–size new_size

An argument that, when specified, instructs the command to resize the input gaugedata object to only have as many gauges as the specified size. The command deletes any gauges that have index numbers greater than or equal to new_size (which must be an integer, and cannot be greater than the size of the input gaugedata object).

–row index

An argument that, when specified, selects gauge editing mode. The gauge data at the specified index (row number in the gauge file) is modified, if it exists. It must be followed by at least one of the -data, -enabled, -cdmeas, -cdmeasnum, -cdmeas_etch, ‑cdmeasnum_etch, ‑weight, or -comment arguments.

-data string

An optional argument specifying input gauge string containing valid gaugedata format for an entire gauge row.

Note:

The following arguments set individual fields, which makes them incompatible with the -data argument.

-enabled {0|1}

An optional argument that sets the enable flag. Use a value of 0 to disable, or a value of 1 to enable.

-cdmeas string

An optional argument that sets the measured resist CD value.

-cdmeas_etch string

An optional argument that sets the measured etch CD value.

-weight value

An optional argument that sets the weight value of the row to the new value.

–comment string

An optional argument that sets an input string that is added to the specified gauge entry as a comment.

Description

Resizes the specified Setgauge object or resets object data associated with one of the gauges.

  • For resize operations, all gauges with indices greater than or equal to the specified size (integer number) are deleted.

  • For reset operations, specify the gauge to be changed by its index.

Examples

Resizes the gauge object to be no more than 274 gauges in size. Gauges numbered 275 and up are deleted.

# resize gauge object
gaugeSet aobj –size 274

Sets the data for gauge #46 to be the specified data string, which represents the entire gauge data object set.

# reset data for gauge No 46
gaugeSet aobj –row 46 –data “1 iso_cd 1 1 20750 10500 21250 10500 0 100 100 83.9 -1 -1 -1 -1 1 0 0 0” –comment “this gauge was modified”

Sets the data for the -cdmeas (resist measurement) and -cdmeas_etch (etch measurement) fields for gauge #32. No other fields are reset.

# reset the only the resist and etch measured cDs for row 32
gaugeSet aobj -row 32 -cdmeas GT:81.5 -cdmeas_etch 85.4
