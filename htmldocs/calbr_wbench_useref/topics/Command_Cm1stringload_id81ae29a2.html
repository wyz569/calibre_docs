cm1stringload

CM1 and VEB Commands

Related Formats: CM1 Model File Format

Creates a CM1 object from a string literal.

Usage

cm1stringload cm1string

Arguments

cm1string

Specifies the input string or Tcl variable representing a CM1 model (required).

Description

Use the cm1stringload command in the Calibre WORKbench Tcl shell to create a CM1 object from a string literal. The input parameter can also be a Tcl variable whose value is a string representing the CM1 model.

Examples

#create a string reperesenting CM1 model
set cm1string “version 9\nmodelType CM1\nDEFINE M0\nANCHOR 0\n”
append cm1string “COEFMAX 1.200000e-04\nTHRESHOLD 0.5\nMCOEF 0 1”
#load the string
set cm1obj [cm1stringload $cm1string]
