Requirements for Calibre IPSE/Calibre IPSETX

The Calibre IPSE tool and Calibre IPSETX tool require the following items to function:

  • A layout with an OPC layer loaded into Calibre WORKbench

  • An optical model

  • For simulations, a VT5 model

  • For simulations, a Calibre OPCpro setup file that contains the following information:

    • optical model location

    • VT5 model location

    • layout layer configuration

    • siteinfo setup file variable set to:

      RESIST -num 40 -spacing 0.02 -center 20

    • Environment variable settings of:

      sse LITHO_PROMOTION_TILING 1
      Tip

      Instructions for VT5 Center, samlinev2, and creating optical models, resist models, and setup files can be found elsewhere in this manual.
