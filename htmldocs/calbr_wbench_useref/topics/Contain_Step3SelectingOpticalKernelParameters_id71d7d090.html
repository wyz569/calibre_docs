Step 3 - Selecting Optical Kernel Parameters

Optical models in the Calibre WORKbench application use the Sum Of Coherent Systems (SOCS) model, which is an approximation of the full Hopkins simulation and provides a fast intensity calculation. This section describes the optical model components that define the SOCS kernel parameters and affect the accuracy of the approximation.
