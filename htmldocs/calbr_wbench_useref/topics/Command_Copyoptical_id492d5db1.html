copyoptical

Optical Model Commands

Copies an optical directory to a new location.

Usage

copyoptical -in inputmodel -out outputmodel

Arguments

-in inputmodel

A required argument specifying a previously created optical model directory.

-out outputmodel

A required argument specifying the new location and directory name for the copied optical model.

Description

The copyoptical command safely copies an optical model directory. Although optical model directories can be moved to any location, manually renaming them can corrupt the model. Using the copyoptical command preserves correct naming conventions.
