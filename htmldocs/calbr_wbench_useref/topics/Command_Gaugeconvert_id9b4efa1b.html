gaugeConvert

CM1 and VEB Commands

Related Formats: Gauge Object Reference

Converts pre-2009.1 gauge objects to the current version.

Usage

gaugeConvert object {2008 | 2009}

Arguments

object

A required argument specifying the input gauge data object.

{2008 | 2009}

A required argument specifying the format.

  • 2008 — converts object to the format accepted in 2008.4 and earlier releases.

  • 2009 — converts object to the format accepted in 2009.1 and later releases.

Description

Converts a gauge object between the pre-statistics version used in releases 2008.4 and earlier and the statistics-enhanced version used from 2009.1 to the present version. Specifically:

  • mode maps to stdev

  • the functionality of count changed

  • dist maps to 95%ci

This is a utility command mainly designed to resolve backward compatibility issues, in that the statistics-enhanced version of the gauge object cannot be read with the gaugeRead command of 2008.4 and earlier releases until converted. The pre-2009.1 format is compatible with all modelflow_v2 releases.

Examples

# convert gaugedata to the format accepted in 2008.4 and earlier releases
gaugeConvert aobj 2008
