Notes on Contour Weighting

The sum of the weight given to the contour data and the weight given to the sample/gauge data must be 1.0, and is calculated based on the weight value specified for the contour. For example, if the contour weight is specified as 0.33, then the sample/gauge data will be automatically given a weight of 0.67 by Calibre WORKbench.

This relative weighting is applied to all data from all contours and all sample/gauge data. So in the example above, if there are 700 data points in the sample/gauge data and 6000 data points from the contours, the contour data may still overwhelm the sample/gauge data by sample size, despite being nominally weighted only half as much. It is therefore important to note the relative sizes of the contour and sample/gauge data sets.

Selective clips that focus on line ends, corners, or other 2D features may help to reduce the total number of data points from contours.
