sgd_create

CM1 and VEB Commands

Related Formats: Super Gauge Data File Format

Creates an empty Tcl object, intended for use as a super gauge data file.

Usage

sgd_create [–in filename | -version {1 | 2}]

Arguments

–in filename

An optional argument that specifies an input gauge or super gauge data file.

-version number

An optional argument that sets the version parameter to the specified number when creating an empty super gauge data file.

  • 1 — (Default) Supports CD data only. This is the format used in versions prior to 2014.2.

  • 2 — Supports CD, SRAF, and height data. Use this when the file will be used for SRAF calibration.

If you do not specify ‑version, the object is created as version 1.

Return Values

This command returns a handle to an sgd object that can be used as a Tcl variable.

Examples

set obj [sgd_create -in “sample.sgd”]
