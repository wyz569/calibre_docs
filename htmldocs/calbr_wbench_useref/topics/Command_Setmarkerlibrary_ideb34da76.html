set_marker_library

Scope: Calibre Pattern Generator API Commands

Sets the marker library to read markers from.

Usage

$cpg set_marker_library {sem_marker | fine_marker | autofocus_marker } “marker_lib”

Arguments

$cpg

A required argument specifying $cpg as a Tcl configuration object or variable.

sem_marker | fine_marker | autofocus_marker

A required argument specifying the type of marker library to read from. The following marker library types are supported:

  • sem_marker — SEM marker.

  • fine_marker — Fine alignment marker.

  • autofocus_marker — Autofocus marker.

marker_lib

A required argument specifying the name of the marker library to read from. The filename must be enclosed in quotation marks.

Description

An optional command that specifies the marker library to read from for a specified type of marker library.

Examples

Example 1

The following example specifies alignment_markers_template.xml as a marker file to read SEM markers from:

$cpg set_marker_library sem_marker "alignment_markers_template.xml"
