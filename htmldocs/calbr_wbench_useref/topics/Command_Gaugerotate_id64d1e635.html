gaugeRotate

CM1 and VEB Commands

Related Formats: Gauge Object Reference

Converts the specified gauges of an input gauge object to be vertical or horizontal gauges. Gauge objects are rotated around their center point.

Usage

gaugeRotate -g gauge_object -l layout -layer layernum {-indexstring string | -indexfile file}
-force {vertical | horizontal} [-extra extra] [-etch] [-search distance] [-length factor]

Arguments

-g gauge_object

A required argument specifying the input gauge data object.

-l layout

A required argument specifying the corresponding input test pattern.

Note:

The input test pattern layout file must be flat. If it is hierarchical, it must be flattened using the layout copy (see the Calibre DESIGNrev Reference Manual) or clipLayout commands.

-layer layernum

A required argument specifying the layer in the layout to measure on.

-indexstring string | -indexfile file

One of either -indexstring or -indexfile is required. These two options are mutually exclusive:

  • -indexstring string — Specifies a set of gauges by index number to rotate.

  • -indexfile file — Specifies a file containing a set of index numbers for gauges to rotate.

The indices specified must be in the range from 1 to the number of gauges, or an error is returned.

-force {vertical | horizontal}

A required argument that specifies the direction to force the requested gauges into. Only the gauges requested have their orientation rotated.

-extra extra

An optional argument that lengthens rotated gauges by 2*extra nm. The default value is 200 nanometers (nm).

-etch

An optional argument that uses the etch measurement CDs for gauge measurements instead of the resist measured CDs, which is the default.

-search distance

An optional argument that sets the search distance in nm for edges to cross with the gauge The default value is 500 nanometers (nm).

-length factor

An optional argument that filters out gauges whose net length (before adding -extra) is factor less than the measured CD.

Examples

The following example code rotates the gauges listed in the index file “glist.txt” to be vertical, with a search distance of 800 nm. Gauges are lengthened by 2*100=200 nm.

gaugeRotate -g obj -l $l_obj -layer 0 -indexfile “/tmp/glist.txt” 
-force vertical -extra 100 -search 800
