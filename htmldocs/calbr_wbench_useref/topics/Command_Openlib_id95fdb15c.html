openlib

VT5 Resist Model Commands

Loads a GDS file, a setup file, or both into the Calibre WORKbench or Calibre LITHOview application.

Usage

openlib -m gds -f setup

Description

Once the setup file is loaded, it remains active until another setup file is loaded by openlib or any of the other commands that allow you to load a setup file, such as samlinev2.

Arguments

-m gds

A keyword and value pair specifying the path for the GDS file to load into the Calibre WORKbench application. You must specify either -m or -f or both.

-f setup

A keyword and value pair specifying the path for the setup file to load into the Calibre WORKbench application. You must specify either -m or -f or both.

Note:

To find out which setup file is currently loaded, issue the command gcsuf at the Tcl prompt.
