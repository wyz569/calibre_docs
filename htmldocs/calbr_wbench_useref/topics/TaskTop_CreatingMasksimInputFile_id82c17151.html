Creating a masksim Input File

The masksim input file contains instructions for Calibre WORKbench to create the DDM library.

Prerequisites

  • Calibre WORKbench available on your path.

  • Read the section “DDM Library Input File Format” to familiarize yourself with the input file parameters.

Procedure

  1. Create a masksim input file with your parameters of interest. The parameters are listed in the DDM Library Input File Format reference page.

    The parameters shown in Table 1 are required to be part of the text file:

    Table 1. Required DDM Input File Parameters

    Parameter

    Brief Description

    make_ddm_edge_library

    Signifies the file is a DDM edge library.

    wavelength

    Sets the simulation wavelength in microns.

    gridsize

    Sets the size of the simulation grid in microns.

    substrate

    Describes the substrate.

    illumination

    Sets the location of the optical input file you created in “Modifying an Optical Model File for DDM and HHA”.

    film

    Describes a film layer.

    bright_outputfile

    Sets output file name(s). At least one output file must be specified.

    dark_outputfile
  2. Invoke Calibre WORKbench using Calibre MTflex mode (as described in the Calibre Administrator’s Guide) to run masksim.
    calibrewb -remotefile remote_file -a masksim -i input_file \
-o sim_outfile
    Note:

    Starting with the 2014.1 release, using MTflex with at least 32 cores is now strongly recommended when the now-default cross talk libraries are generated in the DDM model by the masksim command. This is because cross talk library generation significantly increases masksim library generation time.

    If the number of cores utilized in the remote file is equal to or greater than the sum of the cross talk gaps and HHA sectors, the total run time should be less than the 13.4 release.

Results

Calibre WORKbench generates one or two output DDM files that can be used as input to other Calibre tools.
