Contour Calibration Overview

Contour calibration is a method for improving the accuracy of VT5 and CM1 resist model calibration by importing vendor-supplied SEM contours. It is primarily intended for use with 2D contours, and it supports image space parameter coverage data to improve model stability.

This appendix describes the steps to adapt a design file that has corresponding SEM contour data to use as a contour calibration input.

Note:

Information in this appendix represents a tool functionality collectively known as Calibre ContourCal.

Contour Calibration Functionality

Contour calibration can be used to improve the quality of models under the following situations:

  • When part of the chip design is not covered by the test pattern. You can discover this by using the Calibre Image Space Parameter Explorer (IPSE) tool.

  • When marginal structures (ones that resolve correctly only near the nominal dose and defocus values) are present in the design.

  • When errors occur in complex 2D patterns that are not detected by CD-based calibration. The use of contour information in modeling:

    • Improves pinching/bridging detection

    • Improves detection of misprints
