cm1setparam

CM1 and VEB Commands

Related Formats: CM1 Model File Format

Sets a CM1 object parameter.

Note:

These values can be retrieved using the cm1getparam command.

Usage

cm1setparam cm1obj parameterName [index] value

Arguments

cm1obj

A required argument that specifies the input CM1 Tcl object.

parameterName

A required argument that specifies the input parameter name as either an object parameter or a term parameter. Parameter names are case-insensitive.

The model parameter names are divided into two groups:

  • object parameters that belong to the CM1 object as a whole.

    Accepted object parameter names are these:

  • anchor — index of the anchor term, dimensionless

  • coefmax — maximal allowed coefficient norm (not including the coefficient of the anchor term), dimensionless

  • consistency_setting — flag to activate (1) or deactivate (0) special routines for improving optimization consistency for modelflow_v2 Optimize and Try regimes

  • const_term — constant term, dimensionless

  • curvature_d — Returns the contour curvature diffusion length term in nanometers (nm)

  • curvature_s — Returns the contour curvature smoothing length term in nanometers (nm)

  • curvature_t — Returns the contour curvature threshold term, dimensionless

  • dissolution_span — Returns the dissolution threshold diffusion length term in nanometers (nm)

  • dissolution_threshold — Returns the dissolution neutralization threshold term, dimensionless

  • maxorder — maximal allowed Gauss-Laguerre polynomial order (currently 7), dimensionless

  • modeltype — model type; can only be set to “CM1” for the current release

  • resist_polarity — resist polarity, -1 for NTD, 1 for PTD (default)

  • sidewall_bias_b — sidewall bias b value, dimensionless

  • sidewall_bias_c — sidewall bias c value, dimensionless

  • sidewall_bias_d — sidewall bias d value in nanometers (nm)

  • sidewall_bias_s — sidewall bias s value in nanometers (nm)

  • sidewall_bias_t — sidewall bias t value, dimensionless

  • smooth_gradient — flag as an integer value to turn gradient smoothing on (2 = square of the gradient, 1 = square root of the gradient, or 0 = off) (2, default).

  • smooth_neutralization — flag as an integer value to turn neutralization on (2 = spline smoothed neutralization, 1 = square root smoothed neutralization, or 0 = off) (2, default).

  • threshold — model threshold, dimensionless

  • version — version number

  • z_slope_a — z_slope a value, dimensionless

  • z_slope_b — z_slope b value, dimensionless

  • z_slope_c — z_slope c value, dimensionless

  • term parameters that belong to particular terms in the CM1 model expression.

    Accepted term parameter names are these:

  • b or B — Specifies the term neutralization constant, dimensionless

  • s or S — Specifies the term diffusion length in nm

  • k or K — Specifies the term derivative order, dimensionless

  • n or N — Specifies the term power, dimensionless

  • p or P — Specifies the term Gauss-Laguerre function order, dimensionless

  • curvature_d — For the contour curvature term, specifies the diffusion length in nanometers (nm)

  • curvature_s — For the contour curvature term, specifies the smoothing length in nanometers (nm)

  • curvature_t — For the contour curvature term, specifies the threshold, dimensionless

  • dissolution_span — For the dissolution term, specifies the diffusion length in nanometers (nm)

  • dissolution_threshold — For the dissolution term, specifies the neutralization threshold, dimensionless

  • electron_absorption_fov — For the electron absorption term, specifies the diffusion length value in nanometers (nm)

  • electron_absorption_threshold — For the electron absorption term, specifies the neutralization threshold value, dimensionless

  • horizontal_bias_a — Specifies the horizontal_bias a value, dimensionless

  • horizontal_bias_b — Specifies the horizontal_bias b value, dimensionless

  • horizontal_bias_c — Specifies the horizontal_bias c value, dimensionless

  • horizontal_bias_d — Specifies the horizontal_bias d value in nanometers (nm)

  • horizontal_bias_s — Specifies the horizontal_bias s value in nanometers (nm)

  • pattern_density_span — For the pattern density term, specifies the diffusion length value in nanometers (nm)

  • pattern_density_threshold — For the pattern density term, specifies the neutralization threshold value, dimensionless

  • mcoeff — term coefficient, dimensionless

  • plusminus — Specifies the term neutralization type, dimensionless, two possible values are 1 for acid neutralization and –1 for base neutralization

  • scoeff — Specifies the shrink term coefficient, dimensionless

index

An argument that specifies the input term index (required only if a term parameter is specified).

  • If an object parameter is specified, then index must not be specified.

  • If a term parameter is specified, then a valid index term must be specified. A valid index is an integer between 0 and (nterms - 1) inclusive, where nterms is the number of terms in the input CM1 object.

value

A required argument that specifies the input parameter value to set.

–rescale
Note:

The -rescale argument was obsoleted in the 2015.1 version.

An optional argument flag used to set the anchor coefficient to 1 and rescale the rest of the term coefficients to conform the norm limitation.

Description

Use the cm1setparam command in the Calibre WORKbench Tcl shell to set a specified parameter of a given CM1 object to a particular value. The list of accepted object and term parameters is the same as that for the cm1getparam command. The only exception is the object parameter maxorder, which is not available for cm1setparam.

The –rescale option has an effect only if the parameter name is anchor, coefmax, or mcoeff.

Depending on the parameter name, the input value must satisfy the following limitations:

  • 0 <= anchor < nterms–1, where nterms is the number of terms in the input CM1 object

  • b >= 0

  • s >= 0

  • coefmax >= 0

  • k = 0, 1, or 2

  • n = 1 or 2

  • 0 <= p <= maxorder, where maxorder is a hard coded model parameter (currently 7)

  • plusminus = 1 or –1

Examples

set cm1obj [cm1load cm1.mod]
# set one of the object parameters (threshold)
cm1setparam cm1obj threshold 0.3
# set one of the term parameters, mcoeff
cm1setparam cm1obj mcoeff 2 0.011
# set anchor
cm1setparam cm1obj anchor 3
