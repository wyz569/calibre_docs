How to Run a Batch Script

Run a Calibre WORKbench batch script (a text file that contains commands as if you had typed them in the Calibre WORKbench shell window) using the following command at the command prompt. 
calibrewb filename

Calibre WORKbench attempts to run the file as a Tcl script if it does not detect the file as a design file, bookmark file, or overlay file.

calibrewb filename -- arg1 arg2

Any strings that follow a pair of double-dashes (--) are passed as arguments to the script file, and are not interpreted as command-line arguments. 

calibrewb designfile -s scriptfile ...

When you have multiple arguments on the command line, use the -s option to designate a script file.
