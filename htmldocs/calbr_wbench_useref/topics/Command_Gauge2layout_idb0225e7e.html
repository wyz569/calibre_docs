gauge2layout

CM1 and VEB Commands

Related Formats: Gauge Object Reference

Writes a physical representation of gauges contained by a gauge data object into a layout object.

Tip

For more information on gauge objects and the relationship between the various gauge commands, see the section “Gauge Object Reference” in this chapter.

Usage

gauge2layout -g object [-l wblayout] -outlayer layer [-ssg] [-boxes size]

Arguments

-g object

A required argument specifying a gauge object (produced with ss2gauge or gaugeRead).

-l wblayout

An optional argument specifying a previously-loaded Calibre WORKbench layout object to read the measurements from. The default is the current layout.

-outlayer layer

A required argument specifying the layer number to receive the gauge output.

-ssg

An optional argument that causes gauge2layout to output subgauges as well as main gauges.

-boxes size

An optional argument that causes the gauges to be output as boxes instead of wires. The boxes defined using the bounding box of the wire and are sized up by the size value in database units (dbu).

Examples

gauge2layout -g aobj -l layout0 -outlayer 3 -ssg
