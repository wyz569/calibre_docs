Waiver Setup File

The waiver setup file controls waiver generation for the waiver_flow utility and specifies options in a verification run with calibre -waiver.

A single waiver setup file can be used for both operations, but it is highly recommended to separate the generation and verification waiver setup files. For information on waiver setup file statement usage for calibre -waiver and waiver_flow, see “Waiver Setup File Statement Usage”.

Note:

Calibre RVE is the recommended utility for generating waivers. Only use waiver_flow if batch mode is required and you are confident with its usage. For complete details on generating waivers from Calibre RVE, see “Waiver Shape Generation with Calibre RVE”. For more details on the waiver_flow tool, see “Waiver Shape Generation with the waiver_flow Tool”.
