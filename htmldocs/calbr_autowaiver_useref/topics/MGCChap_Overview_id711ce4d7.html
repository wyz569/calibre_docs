Calibre Auto-Waivers Overview

The Calibre® Auto-Waivers™ tool enables you to generate waivers for rules that are considered safe to ignore. You can then apply the waivers, which are geometrical shapes that resemble the original error result region, automatically in a verification run wherever an error result for the same waived rule appears.
