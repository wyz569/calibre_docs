Running Density Waiver Verification

You apply Density waivers in a physical verification run similar to DRC or ERC waivers. You will run density waiver verification whether you used Method 1 or 2 to generate your Density waivers.

Prerequisites

  • You have met the “Requirements for Running Calibre Auto‑Waivers”.

  • You have completed either “Method 1: Generating Density Waivers with DRC RVE” or “Method 2: Generating Density Waivers with waiver_flow”.

  • Density waivers were generated using the DRC Maximum Results ALL statement.

Procedure

Enter the following command:
calibre -drc -hier -waiver waiver_setup -turbo rules

Results

Waivers in the design are enforced automatically. The run also produces a calibre‑waiver.summary file that contains runtime data and messages and the following files viewable in DRC RVE:

File

Description

ASCII Results Database

DRC errors

density_waived.rdb

RDB file showing applied Density window waivers

used_waiver.rdb

RDB file containing the actual waiver shapes that were applied successfully to the DRC result

unused_waiver.rdb

RDB file showing potential Density window waivers that were not used. There are no properties available in this file.
