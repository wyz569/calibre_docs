Waiver Export of Error Edge Clusters by Calibre RVE for DRC

When Calibre RVE exports edge results it displays a popup message titled “Warning: Exporting Edges” because the resulting waiver shape may not match the result shapes generated by Calibre.

See “Waivers for Edge and Edge Cluster Outputs” for more information on this message and how Calibre Auto‑Waivers handles edge clusters.
