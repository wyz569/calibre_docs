ShowColumns

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: ByErrorResults, ByErrorResultsSummary

Optional key name that controls the display of report columns.

Syntax

ShowColumns = column_name {0 | 1}

Parameters

column_name

A required string that specifies the column. Multi-character wildcards (*) and single-character wildcards (?) are supported.

The default value is an empty list. If a column is not explicitly named in the list, a value is chosen automatically.

0

A required literal hides column_name.

1

A required literal shows column_name. This is the default.

Examples

This examples hides all of the Waived columns from the report:

ShowColumns = Waived* 0
