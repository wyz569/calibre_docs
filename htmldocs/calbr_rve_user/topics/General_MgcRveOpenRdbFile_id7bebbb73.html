mgc_rve_open_rdb_file

SKILL function that opens the specified ASCII format results database (RDB) in a running Calibre RVE session if one exists. If there is not a running Calibre RVE session, the RDB is opened in a new session.

The command opens the RDB file in an existing Calibre RVE session only if that session was started using Calibre > Start RVE from the Cadence Virtuoso menu or a previous mgc_rve_open_rdb_file or mgc_rve_open_rdb_file command. If the Calibre RVE session was started from Calibre Interactive, the mgc_rve_open_rdb_file command opens the RDB in a new Calibre RVE session.

Syntax

mgc_rve_open_rdb_file (“ rdb_file ”)

Arguments

  • rdb_file — A required argument giving the path to an ASCII format results database.

Examples

Enter the following in the CIW to open the RDB TOP.drc.results in an existing Calibre RVE session:

mgc_rve_open_rdb_file("TOP.drc.results")
