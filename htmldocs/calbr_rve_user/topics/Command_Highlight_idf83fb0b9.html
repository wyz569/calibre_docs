highlight

Tcl command to highlight a set of objects in the Calibre RVE SPICE file viewer.

Usage

$spice_api highlight {cell | instance | net | device} cell objs [colorIndex]

Arguments

$ spice_api

The API handle for the netlist. The API handle is returned by the command GetSpiceBrowserAPI.

{cell | instance | net | device}

Required argument specifying the type of object. Only one type may be specified.

If the type is cell, the cell name is used for both the cell and objs arguments.

cell

Required argument specifying the name of the cell in which to highlight.

objs

Required argument specifying a list of object names to highlight.

colorIndex

Optional argument giving an index for the highlight color.

Return Values

None.

Examples

Highlight the top cell. 

$api highlight cell top top

Highlight the net ibias in cell pllclk. 

$api highlight net pllclk {ibias}

Highlight instance XI19/XI20/Xi_0 in cell pllclk using color index 3. 

$api highlight inst pllclk {XI19/XI20/Xi_0} 3
