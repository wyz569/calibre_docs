Specifying the Layers Displayed When Highlighting in Cadence Virtuoso

You can also specify the layers that are shown when you highlight a result from a DRC check. Use the following rule check comment within a rule check in your rule file: 
@ RVE Show Layers: layer1 layer2 ... layerN

where the layern parameter can be specified in these ways for Cadence Virtuoso:

  • layer:purpose — You can use names or numbers for the layer and layer purpose. For example, MT1:drawing or 6:1.

  • layer_name — When you use only a layer name the purpose layer defaults to the drawing purpose layer.

The terminating delimiter for the layers list is a newline. All layers not specified in this statement are automatically hidden in the Cadence Virtuoso environment when any errors from the check are highlighted.

You must also enable the Calibre RVE option “Only show check-dependent layers while highlighting results (hides other layers)” on the Options tab in the DRC Browser category. This option is enabled by default.

If you use RVE Show Layers, you can use the Restore Default Layers button to toggle between displaying the default set of visible layers and the layers specified with RVE Show Layers. See the section “Restore Default Layers Toolbar”.

CAUTION:

RVE Show Layers temporarily changes the LSW layer visibility settings in Cadence Virtuoso. When Calibre RVE is invoked from Cadence Virtuoso, the LSW layer visibility settings are saved, and these settings are restored when Calibre RVE exits if you choose Calibre > Setup > RVE from Cadence Virtuoso and enable the “Restore layer visibility upon exit from RVE” setting. If “Restore layer visibility upon exit from RVE” is enabled, any changes you make within Cadence Virtuoso to the LSW layer visibility settings will be lost when Calibre RVE exits and restores the settings that were saved at startup.
