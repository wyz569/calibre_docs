Keyboard Shortcuts (Hotkeys) in Calibre RVE for DRC

Several keyboard shortcuts are defined for navigation and result selection in the tree and result views of Calibre RVE for DRC.
Table 1. Keyboard Shortcuts for Result Navigation in Calibre RVE for DRC

Keyboard key

Definition

Up arrow, down arrow

Move selection in Result View to next or previous result.

Ctrl-up arrow, Ctrl-down arrow

Move selection in Tree View to next or previous leaf. No selection is made in the Result view.

Left arrow, p, P

Move selection in Result View to previous result (up) and highlight.

Right arrow, n, N

Move selection in Result View to next result (down) and highlight.

Ctrl-left arrow,

Ctrl-P, Ctrl-p

Move selection in Tree View to previous leaf and highlight the first result in the cluster.

Ctrl-right arrow,

Ctrl-N, Ctrl-n

Move selection in Tree View to next leaf and highlight the first result in the cluster.

Page Up, Page Down

Scroll up or down a page in the display.

Ctrl+a

Select all.

Ctrl+b

Unselect all.

Ctrl+/

Select all in Tree View.

Ctrl+\

Unselect all in Tree View.

Home

Move view in Tree View to the left, if applicable.

Shift+Home

Move view to the top.

Ctrl+Home

Move view to the top and select the first item.

Ctrl+Shift+Home

Move view to the top and select all items from current selection to the top.

End

Move view in Tree View to the right, if applicable.

Shift+End

Move view to the bottom.

Ctrl+End

Move view to the bottom and select the last item

Ctrl+Shift+End

Move view to the bottom and select all items from current selection to the last.
