Creating and Editing Custom Results Filters in Calibre RVE for PERC

Custom result filters allow you to specify multiple conditions for a filter, such as check name, property value, waived state, and other conditions. Custom filters are saved and included in the filter list.

Prerequisites

  • A Calibre PERC or Calibre PERC LDL results database open in Calibre RVE for PERC.

Video

The video “How to Include and Exclude Rule Checks to Filter Results in Calibre RVE” demonstrates how to create a custom filter to include and exclude rule checks from the view.



Procedure

  1. Do one of the following to open the Filter Editor:

    • To create a new filter — Click the filter button and choose “Create Custom Filter.”

    • To edit an existing filter — Click the filter button , hover over the filter you want to edit, then click the icon, as shown in the following figure. If the filter is already selected, the icon appears without hovering.



  2. Select and define the desired filter expressions in the Filter Editor; you may select more than one expression. Standard glob-style pattern matching characters are allowed in the fields for the check and cell name.
    Table 1. Custom Filter Expressions in Calibre RVE for PERC

    Filter Expression

    Description

    Cell Name (OR)

    Filter on cell name. If multiple “Cell Name (OR)” expressions are selected they are combined with OR logic.

    Cell Name (AND)

    Filter on cell name. If multiple “Cell Name (AND)” expressions are selected they are combined with AND logic.

    Check Name (OR)

    Filter on check name. If multiple “Check Name (OR)” expressions are selected they are combined with OR logic.

    Check Name (AND)

    Filter on check name. If multiple “Check Name (AND)” expressions are selected they are combined with AND logic.

    State Changed By

    Filter on the username for the Inspect State.

    State Comment

    Filter on the Inspect State comment.

    State Date

    Filter on the Inspect State date

    State Value

    Filter on the value of Inspect State (Pending, Reviewed, or Waived).

    Property Exists

    Filter on whether a specified property name is attached to a result.

    Property Value (OR)

    Filter on a property value. If multiple “Property Value (OR)” expressions are selected they are combined with OR logic.

    Property Value (AND)

    Filter on a property value. If multiple “Property Value (AND)” expressions are selected they are combined with AND logic.
  3. Specify the filter name for a new filter (top of the Filter Editor form).
  4. Click Save to save the filter.
  5. Click Apply to apply the filter to the database. The Apply button also saves the filter.

    The Filter Editor remains open and active. However, it may be hidden behind the main Calibre RVE display. The main Calibre RVE display is not available while the Filter Editor is open.

  6. Click Close to close the Filter Editor. Changes to the filter are not saved unless either Apply or Save has been clicked.
    Tip

    You can delete a custom filter by hovering the mouse over the filter name, then clicking the red X. You cannot delete a custom filter that is currently applied to the database.
