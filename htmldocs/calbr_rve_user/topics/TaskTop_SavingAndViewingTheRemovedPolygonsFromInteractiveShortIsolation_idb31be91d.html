Saving and Viewing the Removed Polygons from Interactive Short Isolation

You can save the polygons that are designated for removal during Interactive Short Isolation. The polygons are saved in an ASCII results database (RDB) file that can be opened at a later time in Calibre RVE and shared with other team members. You can use the RDB to highlight the polygons in the layout and fix the short.

Prerequisites

  • You have used Interactive Short Isolation in Calibre RVE for LVS to mark short polygons with REMOVE and have run short verification. See “Verifying Short Repairs”. The Layout Shorts tab should look similar to the following image:



    If a REMOVE’d polygon is part of a via array, all vias in the array are listed and saved to the RDB.

Procedure

  1. Select File > Save Removed Polygons.

    The “Save Removed Polygons to Directory” dialog box opens.

  2. Enter a directory name and click Open.

    The directory is created if it does not exist.

Results

The polygons marked as REMOVE are saved as an ASCII DRC results database (RDB). The RDB file is saved in the specified directory with the following naming convention:

<short_id>_<start_net>_-_<end_net>_in_<cell>.rdb

For the short example shown in the preceding image, the RDB is named 1_gnd!_‑_vdd!_in_BUF4.rdb. There may be more than one RDB file in the directory, depending on your database and the number of fixed shorts present.

The RDB files can be shared with a team member working on the same database. To open the RDB in Calibre RVE, select File > Open Database and select a “Database Type” of DRC/ERC. The database is opened as a Layout Shorts tab. Double click on a polygon in the REMOVE list to highlight it.

Tip

If you use Calibre DESIGNrev as your layout viewer, you can also use it to save highlighted polygons, as an alternative to using Calibre RVE to save an RDB. First highlight the polygons in Calibre DESIGNrev, then use one of the following methods to save the polygons to a file for later use:

  • To save the highlighted polygons as an RDB, see the topic “Saving Highlighted Shapes as a Calibre nmDRC Results Database (RDB)” in the Calibre DESIGNrev Layout Viewer’s Manual. A video is included.
  • To save highlighted polygons as a GDS or OASIS layout, choose File > Export Layout in Calibre DESIGNrev, and choose GDS or OASIS as the format. See “Export Layout Dialog Box” for further details.
