Calibre RVE Highlight Options

Several options are available that allow you to control how Calibre RVE handles highlighting events.

Set Highlight Behavior

You can specify the highlight behavior, such as pan or zoom to highlights, using the Standard Toolbar or the Highlight menu.

Highlight menu options:

 

  • No view change

  • Pan to highlights

  • Zoom to Highlight

 

  • Clear Existing Highlights

  • Use Cell Map

  • Setup Cell Map
Toolbar highlight dropdown list:

The selections Use Cell Map and Setup Cell Map are only available from the Highlight menu; see “Map Calibre RVE Cells to Different Cadence Library for Highlighting”.

Set Highlight Options

To set highlight options, first open the Options tab with the Setup Options button () on the standard toolbar, then choose the Highlighting category. The Setup Highlighting Options pane has selections for magnifying and displacing coordinates and setting text and marker size. Highlighting options specific to DRC and DFM, and to LVS, PERC, and PEX are grouped together in a section of the Highlight Options pane. Click Apply to save changes made to the options panes.

See the following reference sections for information about the selections on the Options tab:

  • “Setup Highlighting Options Pane”

  • “DRC/DFM Highlighting Options”, Highlighting section

  • “LVS/PERC/PEX Highlighting Options”, Highlighting section

Set Highlight Layers and Colors

You can set additional application-specific highlighting options from the Highlight menu when a result tab is active; these application-specific menu selections are not available when the Options tab is active.

  • DRC — Highlight > Set Check Highlight Layers. Opens the Set Highlight Layers Dialog Box, which selects the layers used in the layout viewer for highlight operations. You can highlight each result to a new layer or highlight all results from a given rule check to a specified layer index or color.

    Also see “Special Highlighting Considerations for Layout Viewers” and “DRC Rule Check Comments for Calibre RVE” when using Calibre RVE for DRC.

  • LVS — Highlight > Setup Highlights. Assign layer colors and filters to use when highlighting; see “Highlights Tab” for complete information.

    Note:

    Cadence Virtuoso users must make an additional setting, as described in the section “Matching Calibre RVE Colors to Cadence Virtuoso Highlight Colors”.
