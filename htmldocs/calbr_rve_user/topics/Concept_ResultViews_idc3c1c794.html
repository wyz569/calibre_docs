Result Views

The Result View lists each of the results selected in the Tree View. Select the format of the Result View from the View > Result Options menu, You can choose List, Icon, or Details view, where the Details view presents the result in a table format.

The result view selections are described in Table 1. You can also use the dropdown list to select the result format. The icon displayed changes with the format selected.

Table 1. View > Result Options Menu Contents

Menu item

Definition

List (default)

Icons

Details

Select the display format for the Result Views.

List — Results are shown as their Result ID, with color indicating fixed or waived.

Icon — Results are shown as their Result ID, with an icon indicating fixed or waived.

Details — Individual results are shown in a table format.

Also see Display Conventions for DRC Results.

Result Data Pane

Show the Result Data Pane.

Result Tips

Show results information and waiver comments as tooltips for each result.

Figure 1. DRC Result View Formats

A description of the columns available in the Result View Details format is given in Table 2. Click the column selection icon to select columns to display; click and drag a column to change the column order.

Table 2. Column Headings for Result Views

Selection

Description

Result ID

A unique number for each result. Results retain their unique ID regardless of the grouping, sorting, or filtering selections.

Check

The check name.

Cell

The cell name.

Priority

The check priority. See “RVE Priority”.

Flat Count

The flat result count. For most cases the flat count is the same as the hierarchical count and by default is not displayed.

Type

The result type: E (edge) or P (polygon).

Fixed

The number of fixed results.

Waived

The number of waived results.

Waived Comment

The waiver comment. The column is not displayed automatically ( in column selection list); you must select the column with a check mark ().

Waived By

As described. The column is not displayed automatically ( in column selection list); you must select the column with a check mark ().

Waived Date

As described. The column is not displayed automatically ( in column selection list); you must select the column with a check mark ().

properties

The properties available with each check can be displayed.

Vertex Count

The number of vertices in the result shape.

Coordinates

The coordinates of the vertices in the result shape.
