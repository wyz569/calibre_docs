ColorSet

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: ByCellColormap, ByCellHistogram, ByErrorColormap, ByErrorHistogram, ByWindowColormap, ByWindowHistogram, ScatterPlot, Common section

Specifies a pre-defined color scheme for histogram and colormap reports or the line color for a ScatterPlot report.

Syntax

Usage with Histograms and Colormaps

ColorSet = {bluescale | custom | default | greyscale | heatmap1 | heatmap2 | redscale}

Usage with ScatterPlot

ColorSet = line_color

Parameters

bluescale

A required literal that specifies the blue scale pre-defined color scheme.

custom

A required literal that specifies the custom pre-defined color scheme. If specified, then the color scheme defined by the CustomColors key name is used.

default

A required literal that specifies the default pre-defined color scheme.

greyscale

A required literal that specifies the grey scale pre-defined color scheme.

heatmap1

A required literal that specifies the heatmap1 pre-defined color scheme.

heatmap2

A required literal that specifies the heatmap2 pre-defined color scheme.

redscale

A required literal that specifies the red scale pre-defined color scheme.

line_color

A Tk symbolic color name or an RGB color specified as a hexadecimal code (for example, #0000FF). This option is only used with the ScatterPlot report type.
