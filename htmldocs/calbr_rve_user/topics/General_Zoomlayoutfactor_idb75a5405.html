ZoomLayoutFactor

Key Value for DRC HTML Report Configuration File, highlighting category

Usage: Optional. Valid in DRCResults, DRCHighlight, and DRCColormap sections, and in Common section.

Inheritance: Standard Inheritance

Specifies a comma-separated list of zoom options to use when including layout images for DRCResults, DRCHighlight, and DRCColormap reports. One image is included for each specified option.

Allowed Values

zoom

A positive floating-point number representing a zoom factor.

Default: 0.7

By default, zoom specifies the ratio (result extent)⁄(zoom window). However, if ZoomLayoutFactorUU=1, then zoom is in user units and specifies the size of the halo region around the result.

fit

Specify fit to include a layout image zoomed out to show the complete cell.

coords

Specify coords to zoom to a layout window specified with absolute coordinates. The coordinates are specified by ZoomLayoutCoords.

Examples

ZoomLayoutFactor specifies two highlight images in this example, one at the default zoom factor of 0.7 and one with a complete view of the cell. 

[Highlight Report]
ReportType = DRCHighlight
GroupBy = Check
ZoomLayoutFactor = .7, fit
Tip

When using ZoomLayoutFactor=fit, you can use a Calibre DESIGNrev preferences file to specify a bounding box around the cell. Use DrvOptionsFilePath to specify the location of the preferences file and include the following line:

prefs_view_drawTopCellBbox 1
