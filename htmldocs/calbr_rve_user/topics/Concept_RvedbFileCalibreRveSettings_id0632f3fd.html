.rvedb File for Calibre RVE Settings

Calibre RVE options are saved to the .rvedb file located in your home directory by default. This file is accessed every time you invoke Calibre RVE. To return a setting to its default, you must specify the default explicitly. If you delete the .rvedb file, all settings return to their default values.

Default settings are not saved to the .rvedb file except in certain circumstances when using multi-level configuration management for Calibre RVE options.

Location of .rvedb File

The .rvedb file is located in your home directory by default.

You can specify the location of the .rvedb file to be other than your home directory by setting the environment variable MGC_RVEDB_DIR to a valid directory path. If MGC_RVEDB_DIR is set, the Calibre RVE tool looks in the specified directory for the .rvedb file on startup, and saves the .rvedb file to the specified directory when it exits. If the directory specified by MGC_RVEDB_DIR does not contain a .rvedb file, then default settings are used at startup. If the directory specified by MGC_RVEDB_DIR is not writable, then changes to Calibre RVE settings are not saved on exit.

You can specify multiple locations for the .rvedb configuration file; see “Multi-Level Configuration Management in Calibre RVE”.
