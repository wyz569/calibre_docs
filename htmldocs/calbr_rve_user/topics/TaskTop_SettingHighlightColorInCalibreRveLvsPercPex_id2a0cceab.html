Setting Highlight Color in Calibre RVE for LVS, PERC, and PEX

You can easily set the highlight color scheme from a dropdown list in the Calibre RVE toolbar. You can select a fixed highlight color, highlight each layer in a separate color, or cycle through a default or custom set of colors.

The highlighting color scheme applies to LVS, PERC, and PEX databases and to RDBs that are created as part of the results database, such as ERC, Pathchk, and PERC LDL databases.

Prerequisites

  • A results database from a Calibre nmLVS, Calibre PERC, or PEX run open in Calibre RVE.

Procedure

  1. Click the Highlighting color scheme icon to display the dropdown list. The button icon shows the current selection.


  2. Choose one of the options from the dropdown list. For each option except “Default colors” you can hover the mouse over the selection to display the  Edit icon.
    Table 1. Highlighting Color Scheme Options in Calibre RVE for LVS, PERC, and PEX

    Color Scheme

    Definition

    Default colors

    Cycles through the default set of highlight colors. A new color is used for each highlight action.

    Different colors are also used when multiple objects are highlighted at once if “Highlight objects in individual colors” is enabled (Options tab, Highlighting category, LVS/PERC/PEX Highlighting area) is enabled.

    Layer colors

    Highlights each design layer in the highlighted object with a different color. Click the Edit icon to open the Highlights Tab and select the layer colors.

    The following viewers are supported for color matching: Calibre DESIGNrev, Cadence Virtuoso, and Synopsys IC Compiler. With the Calibre DESIGNrev layout viewer, highlight layers are labeled with the layer name when this option is enabled. This selection is the same as the menu selection Highlight > Use Layer Colors.

    The “Layer colors” selection applies only to highlighting connectivity objects such as nets, devices, and ports in a connected layout viewer. The “Layer colors” selection is not available for source-based topological Calibre PERC runs.

    If “Layer colors” is selected as the highlight color scheme, schematic highlighting is done according to the most recent previously selected color scheme.

    Fixed color

    Use the selected color for each highlight action. Click the Edit button to select the color.

    Custom colors

    Cycles through a custom set of highlight colors. A new color is used for each highlight action. Click the Edit icon to open the “Choose highlight color” dialog box and select the set of colors.
