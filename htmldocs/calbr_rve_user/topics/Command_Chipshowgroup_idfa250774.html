ChipShowGroup

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: ChipSummary, TotalSummary, CellDrillDown, NetDrillDown, WindowDrillDown, Common section

Optional key name that controls display of the Group column.

Syntax

ChipShowGroup = {0 | 1}

Parameters

0

A required literal that hides the display of the Group column.

1

A required literal that displays the Group column. This is the default.
