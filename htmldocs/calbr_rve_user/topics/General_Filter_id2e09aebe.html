Filter

Key Value for DRC HTML Report Configuration File, filter category

Usage: Optional. Valid in all report types.

Inheritance: Filter Inheritance. If the GroupBy key references another section, then the Filter setting is ignored.

Specifies the name of a section with ReportType = DRCFilter. When this keyword is used, filter settings in the referenced section are used; filter settings in the current section are ignored.

Allowed Values

[filter_section]

The name of a section with ReportType = DRCFilter. The brackets are optional.

Examples

Filter = [MyFilterSection]
