Adding DRC RVE Rule Check Comments to the Rule File

You can add DRC RVE rule check comments to the rule check definition in your rule file.

Procedure

  1. Include the rule check comment in the rule check in your rule file. Rule check comments are preceded by the @ character. For example:
    min_ext_diff_poly {
	@ minimum enclosure of poly by diff
	@ RVE Highlight Color: blue
	enc diff poly < 1.25
}
  2. Make sure that rule check comments are output to the DRC results database. This is done by default unless you specify DRC Check Text NONE.
  3. Specify an ASCII results database in the rule file. For example, the following command specifies a database file named drc_database:
    DRC Results Database drc_database ASCII
  4. Run Calibre nmDRC.

Results

Calibre places the DRC RVE rule check comments in the DRC results database. Calibre RVE automatically processes them when it loads the results database.
