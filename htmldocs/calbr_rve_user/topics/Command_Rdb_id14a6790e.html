RDB

A DRC RVE rule check comment which creates a link to the specified side RDB in the check text pane of Calibre RVE for DRC. This rule check comment can only be placed within a rule check in the SVRF rule file.

Usage

          RDB rdb_file

Parameters

rdb_file

The filename of a side RDB (auxiliary database).

Description

The filename rdb_file is displayed as a link in the check text pane of Calibre RVE. Clicking the link opens the RDB in a new tab.

If a rule check creates more than one side RDB, multiple RDB rule check comments can be specified.

This rule check comment can only be placed within a rule check in the SVRF rule file; for instructions, see “Adding DRC RVE Rule Check Comments to the Rule File”. The RDB rule check comment cannot be specified in a DRC RVE Check Text Override File (CTO File) Format.

Examples

The following example shows the rule check comment placed in a Density rule check.

metal1_density { 
@ density metal1 greater than 50%, or 25%
@ RDB m1_density.rdb
DENSITY metal1 < 0.25 WINDOW 50 RDB m1_density.rdb
}

To see the link in Calibre RVE for DRC, do the following:

  1. Run Calibre as directed in “Adding DRC RVE Rule Check Comments to the Rule File”.

  2. Open the main results database in Calibre RVE.

  3. Enable View > Check Text Pane.

  4. In the tree view, select the check with the RDB rule check comment.

  5. Click the link in the check text pane to open the side RDB, as shown in Figure 1.
