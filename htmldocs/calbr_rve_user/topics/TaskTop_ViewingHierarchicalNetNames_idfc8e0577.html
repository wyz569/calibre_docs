Viewing Hierarchical Net Names

You can view the hierarchical names for a net using the Internal Schematic Viewer or the Calibre RVE SPICE File Viewer. Viewing hierarchical names is especially useful if a net crosses cell or instance boundaries and changes names.

Prerequisites

  • Open the netlist in the Internal Schematic Viewer or SPICE File Viewer:

    • Using the Internal Schematic Viewer in Calibre RVE for LVS, PERC, and PEX:

      In Calibre RVE, choose View > Schematics.

    • For viewing in the Calibre RVE SPICE File Viewer, enter the following at the command line:

      calibre -rve -spice netlist.sp

      where netlist.sp is a SPICE netlist.

Procedure

  1. Navigate to a view that contains the net of interest. If using Calibre RVE for LVS, PERC, or PEX, you can highlight the net or use the Finder or Info Pane search to find a net.
  2. Right-click on the net and select Hierarchical Names. The hierarchical names are displayed in descending order.
