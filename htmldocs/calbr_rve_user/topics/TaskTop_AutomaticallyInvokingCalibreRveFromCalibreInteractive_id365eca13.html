Automatically Invoking Calibre RVE from Calibre Interactive

You can instruct Calibre Interactive to start Calibre RVE automatically after the Calibre run finishes. Calibre RVE automatically opens the results database specified on the Outputs pane. If Calibre Interactive is connected to a supported design tool, Calibre RVE is automatically connected to the design tool.

Prerequisites

  • “Calibre RVE System Requirements”

Procedure

  1. Start Calibre Interactive and set up the options for your run.
  2. Enable the appropriate option to start Calibre RVE automatically. The location varies for each tool and there are restrictions on the database format:

    • DRC — Enable “Show results in RVE” on the Outputs pane.

      The Calibre nmDRC results database must be in ASCII format.

    • DFM — Enable “Show results in RVE” on the Outputs pane.

      The DFM output format must be DFM Database or ASCII.

    • LVS — Enable “Start RVE after LVS finishes” on the Report/SVDB tab on the Outputs pane.

      You must enable the Create “SVDB Database” option.

    • PEX — Enable “Start RVE after PEX” on the SVDB tab on the Outputs pane.

      You must enable the “Create SVDB Database” option.

    • PERC — Enable “Start RVE after PERC finishes” on the Outputs pane.

      You must enable the “Create SVDB Database” option.

  3. (Optional, for DRC and DFM) To start Calibre RVE as soon as results are available from the run, choose Setup > Preferences, click the Misc tab, and enable “Start RVE as soon as results are available.”

    For DRC and DFM runs you can instruct Calibre Interactive to start Calibre RVE as soon as the first check is completed, before the full run is finished. The progress and results from the run are displayed in real-time. Results are displayed up to the latest check that has completed. The option “Show results in RVE” on the Outputs pane must also be enabled.

  4. Click the Run button in Calibre Interactive to start the Calibre run.

Results

Calibre RVE automatically opens the results database specified on the Outputs pane after the Calibre run finishes. If Calibre Interactive is connected to a supported design tool, Calibre RVE is automatically connected to the design tool.

See Multithreaded Parallel Processing for Calibre RVE Operations (‑turbo Option) for how the ‑turbo option is set automatically in certain cases.

When Calibre RVE is invoked from Calibre Interactive, the settings for running on a remote host are taken from the Performance tab on the Run Control pane of Calibre Interactive.

If Calibre RVE for DRC is started automatically after a Calibre Interactive nmDRC run and the setting “Start RVE as soon as results are available” on the Misc tab of the Setup Preferences dialog box is enabled in Calibre Interactive, then the Group By setting in Calibre RVE must have a top-level grouping of “Check.” If the top-level grouping is not “Check”, the grouping is set to “Check/Cell.” Other grouping methods are not available until all results are loaded into Calibre RVE.
