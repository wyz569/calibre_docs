CheckHLLayer

Key Value for DRC HTML Report Configuration File, highlighting category

Usage: Optional. Valid in DRCResults and DRCHighlight report sections, and in the Common section.

Inheritance: Standard Inheritance

Specifies the highlight layer and color. The available options are to use the CTO file or check text, to use a different highlight layer for each rule check, or to use a single highlight color for all rule checks.

Allowed Values

RDB

The highlight color is determined by RVE Highlight Color and RVE Highlight Index statements in the CTO file or check text. If such statements are not found, a single highlight layer/color is used. (default)

Colors

A different highlight layer/color is used for each rule check and the highlight layer name is set to the check name. The CTO file or check text is not used.

None

A single highlight layer/color is used for all rule checks; the CTO file or check text is not used.
