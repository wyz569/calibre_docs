Waiving Topology Results in Calibre RVE for PERC

You can waive topology results using Calibre RVE for PERC and create a topology waiver description file based on the waived results. You can specify trigger functions that run before or after the waiver file is created.

See “Topology Waiver Application” in the Calibre PERC User’s Manual for a complete description of the Calibre PERC waiver flow.

Note:

The following procedure assumes one person is waiving results and creating the waiver description file. It is also possible for multiple users to waive results in the same Calibre PERC results database and create a merged waiver description file—see “Multi-User Waiver Flow” in the Calibre PERC User’s Manual.

Prerequisites

  • Calibre PERC results are open in Calibre RVE. See “Opening Calibre PERC Results in Calibre RVE”.

  • (Optional) To force the use of the waiver comment dialog box, set the environment variable MGC_CALIBRE_RVE_ENFORCE_WAIVER_COMMENT to 1 before starting Calibre RVE. See “Force Use of Waiver Comment Dialog Box”.

Procedure

  1. Choose Setup > Options to set options for waiving:
    1. Choose the Waivers category.
    2. Select the desired behavior for the following options:

      Option

      Description

      Show comment dialog while reviewing (PERC only) or waiving results

      Show a dialog box for entering waiver comments each time you waive a result.

      Specify initial comments

      (only displayed if “Show comment dialog …” is checked)

      Pre-populate the waiver comment dialog box with the text specified in the entry box. To specify multiple initial comments, select the dropdown menu on the right and click Add, then fill in the new text box. To delete an initial comment, select it in the dropdown menu, delete all of the text, and select a new comment in the menu.

      When you waive a result, the initial comment text is displayed in the Waived Result Comment dialog box. If multiple initial comments are defined, you can select an initial comment on the right hand dropdown menu, and the text currently open in the Options tab is the default option. You can edit the comment text when you waive a result, or right-click a result and choose Waive > Edit Waived Comment.

      Display waived results timestamps in GMT

      Enable the option to show timestamps in Greenwich Mean Time (GMT) rather than local time.

      Export PERC waivers with property criteria from database

      Add property criteria for the results to the waiver description file. For each property attached to a result, a ‑property or ‑propertyString parameter is added with the setting {propertyName == value}, where value is the corresponding property value in the results database.

      Use this option to make waiving a result conditional on the property criteria. See ‑property in “Topology Waiver Description File Format” in the Calibre PERC User’s Manual.

      When including property criteria, the Library and Delta export options are not available in the Export PERC Waiver Description File dialog box.

    3. Click Apply.
  2. (Optional) To specify a trigger function that runs before or after the waiver file is created, do the following:
    1. Choose the Waivers category on the Options tab.
    2. Enable “PERC Pre-Export Waivers” and/or “PERC Post-Export Waivers” and enter the trigger function call in the text field. The filename of the waiver file is appended to the list of parameters at during the run.
    3. Click Apply.
  3. Click the result tab for your database to make it active.
  4. Waive results with one of the following methods. You cannot waive results from checks that are specified SELECTTYPE UNWAIVABLE in the PERC Load statement.
    1. Right-click the result or group of results to waive.

      To waive all results in a cell, rule, or other grouping, right-click the group in the Tree View. To waive an individual result, right-click the result in the Result View. You can use Shift- and Ctrl-click to select multiple results.

    2. Choose one of the following to waive the result(s):

      The Waived External Net menu selection is only available for “Net” type results, and controls whether a result that is connected to a net external to the waiver_cell is waived.

      • Inspect State > Waived — Waive the result.

      • Inspect State > Waived External Net > Yes — Waive with “‑externalNet YES” in the waiver description file.

      • Inspect State > Waived External Net > No — Waive with “‑externalNet NO” in the waiver description file. This is the default and equivalent to Inspect State > Waived.

    CAUTION:

    If “Show only one placement for each hierarchical PERC result” was enabled when the database was loaded, then waiving the representative placement waives all placements for the result. See “Speeding Up Database Loading by Viewing One Placement of Results in Calibre PERC”.

    When waiving results, the setting of “Show only one placement for each hierarchical PERC result” should be the same between sessions to avoid confusion and undesired results.

    Waived results are indicated with a dim red X and a waiver description section is added at the end of the result text.

    Tip

    You can also waive a result or group of results with the W keyboard shortcut.

    Waived results are saved with the database.

  5. (Optional) If you want to export a subset of the waived results, select the results to export with one of the following methods:

    • Apply a result filter as described in “Result Filters in Calibre RVE for PERC”.

    • Select the results to waive using Shift- and Ctrl-click. You can select results in the tree and/or detailed view.

  6. Choose Tools > Export PERC Waiver Description File to create a waiver description file based on the waived results. Specify the following information in the dialog box:

    • Waiver Description File Name

      The pathname for the waiver description file. You can use environment variables in the pathname.

    • PERC Waiver Flow Type — Select the flow type from the following selections:

      • Full-chip — As indicated.

      • Library — Use this option if you are creating waivers for a small block or IP cell. This option adds the “‑parent cell” parameter to the waiver description, where cell is the top cell for the currently loaded results database.

    • Waiver Export Type

      • Full — Exports a waiver description file for use with the PERC Waiver Path statement.

      • Delta — Do not use with this procedure; it is used for the multi-user waiver flow.

    • Waivers to Export

      Use this option to export the subset of waived results selected in Step 5.

      • All — Export all waived results.

      • Displayed — Export only the waived results in the Calibre RVE display. If you applied a filter, only waived results that satisfy the filter are exported.

      • Selected — Export only the waived results selected with Shift- and Ctrl-click.

      The Displayed and Selected options are not available when the Delta export option is selected.

  7. Click OK to create the waiver description file.

Results

The output of the Export PERC Waiver Description File dialog box is a waiver description file; see “Topology Waiver Description File Format” in the Calibre PERC User’s Manual for complete details. Also see “Generating Topology Results Waivers” in the same manual for additional information in the “Results” section.

If you enabled “Export Waivers” in the Databases category on the Options tab, the specified trigger function is executed after the waiver file is created.

Waivers are applied automatically during a Calibre PERC run by using the PERC Waiver Path specification statement in the rule file with the name of the exported waiver description file as an argument.

Note:

If PERC Report Option SKIP_WAIVED_RESULT is specified in the rule file for a Calibre PERC run, then the database will not include results that are waived as part of the Topology Waiver Application.
