Using the Net Parasitics Table

The net parasitics table is displayed in the Parasitics tab of Calibre RVE for PEX. The net parasitics table gives a summary of the parasitic information for each net, and provides controls to display detailed information. The key actions described in this section are:

Prerequisites

  • Calibre RVE for PEX open to a SVDB that contains a net parasitics database. See “Starting Calibre RVE”.

  • Recommended: A layout viewer connected to the Calibre RVE session. See “Connection to a Design Tool” and “Interfacing With Layout and Schematic Viewers”.

  • Case is handled correctly in the rule file; see “Case Sensitivity in Calibre RVE”.

The opening view of Calibre RVE for PEX with a parasitics database should look similar to Figure 1. The Parasitics tab is opened automatically; the comparison and extraction results are opened only if they contain errors.

Procedure

  1. Verify that you have clean LVS extraction and comparison results.

    Clean LVS extraction and comparison results are indicated in the Navigator as shown in the following image.



    See “Examining Connectivity Extraction Results” and “Examining LVS Comparison Results” for more information.

  2. Set the behavior on highlight events using the Highlight menu or the toolbar list .
  3. Click the Parasitics tab to view the net parasitics table; the Parasitics tab is labeled with the database name and has the icon. The results of an extraction run are shown in a tabular view, as shown in Figure 1.

    The columns show the layout net name, source net name, resistor count, inductor count, total intrinsic capacitance, total coupled capacitance, and sum of intrinsic and coupled capacitance. Some parasitic databases include the coupling coefficient for mutual inductance.

    To sort the table by the values in a particular column, click in the column header.

    Figure 1. Net Parasitics Tab

    If the SVDB contains parasitic databases for more than one cell, you can open the net parasitics table for a different cell in a new tab by choosing View > Results > Parasitics and selecting the cell name.

  4. To find more information about a net, select a net in the parasitics table, and use the toolbar buttons or the right-click menu to highlight the net or display more information.
    Table 1. Parasitics Toolbar Buttons

    Icon

    Name

    Description

    Show Net Info

    Opens a tree view of the net in the Info Pane.

    Highlight Net

    Highlights the net in the attached layout and schematic viewers.

    Show Parasitics on Net

    Opens a detailed view of the parasitics on the net.
    Tip

    By default, an internal schematic viewer is opened on highlight events. To turn off automatic opening of the internal schematic viewer, choose Setup > Options, choose the Schematic Viewer category, and disable the selection “Show netlist schematics on highlighting.” For more information on navigating in the internal schematic viewer, see “Internal Schematic Viewer”.

  5. To view coupling capacitance to a particular net, rather than to all nets, enable the radio button next to the text entry field for “Coupling to” and enter the name of the net, as shown in Figure 2. You can enter multiple nets separated by spaces. Net names are case sensitive for finding coupling capacitance.

    To select the net in the layout rather than by name, click the layout icon: .

    The coupled capacitance to the chosen nets is displayed as shown in Figure 2, with the percentage of the total coupled capacitance given in parentheses.

    Figure 2. Coupling Capacitance to Selected Nets

    You can also specify coupling to selected nets by selecting multiple nets in the net table using Shift- and Ctrl-click. Then right-click a selected net to bring up a pop-up menu, and choose Show Coupling to Selected Nets.

  6. To find a net in the net table, enter the net in the “Find Nets” text field. Multiple nets may be entered separated by spaces. Net name are not case sensitive for the “Find Nets” function. The selected nets are moved to the top of the net table.

     

    Tip: You have a choice of using layout or source names to identify nets. Click the dropdown button to the right of the icon to switch between the two choices.

    Click the layout icon, , or the source icon, , to select the nets in an attached viewer. The nets are entered in the text field.

  7. Double-click a net in the net table to open the detailed parasitics view You can also select a net, then select Show Detailed Parasitics from the right-click pop-up menu, or use the toolbar icon . The view should look similar to that in Figure 3.
    Figure 3. Detailed Parasitics View

    Parasitic resistors (R), inductors (L), mutual inductance coupling coefficients (K), intrinsic capacitors (C), and coupled capacitors (CC) are shown in a tree view, each with their own icon. Details such as layers and properties for the selected item are shown in the area to the right.

    Click a parasitic icon to highlight the element in the connected viewers.

    The icon allows you to select an area in the connected layout viewer. The selected parasitic elements are highlighted only in the selected area.

  8. Click “Pt-to-Pt Res” or “Pt-to-Pt Ind” in the Type column to open the desired point to point calculator. See the section “Calculating Point-To-Point Resistance and Inductance” for details. Point-to-point inductance is only available if the parasitics database includes extracted inductance.
