Symbol Shape Elements

The symbol shape elements are parameters in the symbol shape definition in the Symbol Library file. The shape elements include arc, text, path, port, and others elements that are used to define a custom symbol shape. The Symbol Library file is used by the Internal Schematic Viewer and SPICE File Viewer in Calibre RVE to display custom symbol shapes.

See “Symbol Library File Format” for the definition of the Symbol Library file, including a complete example.

Calibre RVE supports the shape elements listed in Table 1.

Table 1. Symbol Shape Elements

Keyword

Shape element

arc

An arc or circle.

attrdsp

An instance attribute, used to display the cell or instance name at a specified location.

fpath

A closed polygon or a circle.

path

An open polygon path.

permute

Allowed pin-swap permutations.

pinattrdsp

An instance-pin attribute, used to display the pin name at a specified location.

place

A placement hint for latches and IO buffers.

port

A port. Port names must match the port names in the netlist for the custom symbol shape to be applied.

text

A text label.

Syntax conventions for the documentation are described in “Syntax Conventions”.
