Open ... DBs

Key Value for DRC HTML Report Configuration File, side RDB control category

Usage: Optional. Valid in Common section only.

Inheritance: Not applicable.

A group of key values that specifies whether to open and process any side (auxiliary) RDBs of the given type which are included in the main RDB. The default value varies by the RDB type.

Allowed Values

  • 0 — False; do not open or process side RDBs of this type.

  • 1 — True; open and process side RDBs of this type.

Key Names and Defaults

  • OpenCheckMapDBs

    Default: 0 (Do not open)

  • OpenDensityDBs

    Default: 0 (Do not open)

  • OpenDFMDBs

    Default: 0 (Do not open)

  • OpenLayoutInputExceptionDBs

    Default: 1 (Open)

  • OpenNetAreaRatioDBs

    Default: 0 (Do not open)

  • OpenOtherDBs

    Default: 0 (Do not open)

  • OpenPATHCHKDBs

    Default:1 (Open)

  • OpenWaiverDBs

    Default:1 (Open)

Examples

OpenWaiverDBs = 0
