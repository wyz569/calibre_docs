Calibre RDB Conversion: Working with Calibre Results in Place and Route Tools

You can convert results from an ASCII format Calibre results database (RDB) to error markers and route guides in your Place and Route tool. This allows you to use familiar tools when debugging errors and correcting the design. Calibre RDB Conversion is available for Synopsys IC Compiler and Cadence Encounter Digital Implementation (EDI).
Figure 1. Calibre RDB Conversion to Native Place and Route Formats

You can convert results shapes in the Calibre RDB to the following P&R objects:

  • DRC Error Markers — Allows you to use the internal Place and Route (P&R) error browser tool to view and debug errors.

  • Route Guides and Blockages — Route guides and blockages can aide in correcting certain design errors. For example, you can run Zroute in Synopsys IC Compiler or NanoRoute in Cadence Encounter to repair violations.
