Restore Default Layers Toolbar

Toggle the toolbar display with: View > Toolbars > Restore Default Layers

The Restore Default Layers button toggles the layer visibility in supported design tools.

The Restore Default Layers toolbar contains one button, as shown in the following figure:

Supported design tools: Calibre DESIGNrev, Cadence Virtuoso, and Synopsys IC Compiler

The effect of toggling the Restore Default Layers button is shown in Figure 1 with the Calibre DESIGNrev layout viewer. If a rule check or a DRC RVE Check Text Override File (CTO File) Format includes a RVE Show Layers check text comment, only the layers specified in the RVE Show Layers statement are displayed when you highlight a result from the rule check, as shown in the left side of Figure 1. Click the Restore Default Layers button to restore the default layer visibility, then click again to go back to the layer visibility defined by the most recent RVE Show Layers statement. This feature is only available with the Calibre DESIGNrev, Cadence Virtuoso, and Synopsys IC Compiler layout tools.
