Using the Voltage Trace GUI in Calibre RVE for PERC

The Voltage Trace GUI in Calibre RVE for PERC displays a schematic view of the voltage propagation for a net. The Voltage Trace GUI makes use of data from the perc::create_voltage_path command. The Voltage Trace GUI is available for both Calibre PERC and Calibre PERC LDL results.

The Voltage Trace GUI includes the following features:

  • Visual display of the voltage trace. Tooltips display the voltages on each net, pin, and device.

  • Multiple voltage mode that displays only elements that have a specified set of voltages.

  • Ability to highlight to connected design tools and the Internal Schematic Viewer from the Voltage Trace GUI.

  • Ability to select a path from the Voltage Trace GUI.

  • Keyboard shortcuts for zooming and panning. See “Navigating Tips for the Internal Schematic Viewer”.
