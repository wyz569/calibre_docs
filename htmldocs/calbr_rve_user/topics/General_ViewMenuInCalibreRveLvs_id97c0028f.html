View Menu in Calibre RVE for LVS

The View Menu includes some items that are specific to a Calibre RVE for LVS session. Some menu selections are only available when certain report tabs are selected, and this is noted in parentheses.
Table 1. View Menu Contents in Calibre RVE for LVS

Menu Item

(restriction)

Description

Results

Dropdown list of the available results databases. Select a database to open the report tab.

Reports

Dropdown list of the available text reports and files.

Rules

Opens the rule file used in the run.

Navigator

Opens the Navigator Pane, which provides easy navigation to the available report tabs, files, and session tools.

Info

Opens the Info Pane, which displays information on design elements in a tree view. Search capability for design elements in the layout is also included.

You can open the Info pane to a selected design element by right-clicking the design element and choosing Show … Info.

Finder

Opens the Finder Tab, which provides advanced design search and navigation for design elements in both layout and source.

Schematics

Opens the Internal Schematic Viewer. The schematic viewer includes search capability and netlist display. The schematic view is also opened when you click a blue or red design element in the detailed report view or highlight a selected item from the Finder Tab or Info Pane.

Tree Options

(Extraction, PERC, or Comparison Results tabs)

Display a cascading menu with options for sorting comparison and connectivity extraction results in the tree view. See the following tables:

  • Table 1 in the section “Extraction Results Tab”

  • Table 3 in the section “Comparison Results Tab”

  • Table 1 in the section “PERC Results Tab”
