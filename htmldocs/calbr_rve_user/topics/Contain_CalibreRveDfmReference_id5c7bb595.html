Calibre RVE for DFM Reference

The Calibre RVE for DFM interface appears when you invoke it from the command line, from Calibre Interactive, or from your layout viewer. You can view the results of your DFM run from this window. If you do not specify the DFM database on the command line, Calibre RVE prompts you for its pathname upon invocation.

The DFM Report Card is functional only if your database is one of the following:

  • A DFM database containing the required annotations, as described in the Calibre YieldServer Reference Manual.

  • An RDB with check names created using the naming conventions described in section “DFM Report Card” of the Calibre YieldAnalyzer and YieldEnhancer User’s and Reference Manual

Also, the ability to highlight data in your layout viewer from within the DFM Report Card is only available if your layout is loaded into the layout viewer and socket communication is set up between the layout viewer and Calibre RVE. You set up socket communication automatically when you launch Calibre RVE from your layout viewer.
