Calibre::import_route_blockages

Imports a file saved from a previous Calibre RDB conversion to Cadence Encounter route blockages. Used in the Cadence Encounter environment.

Usage

Calibre::import_route_blockages rdb_blk

Arguments

rdb_blk

A file saved from a previous Calibre RDB conversion to Cadence Encounter route blockages. The rdb_blk file is created with one of the following commands, where only the options specific to selecting route blockage conversion and file output are given:

Command

Options

Execution environment

Calibre::convert_to_route_blockages

‑output_file rdb_rg

Cadence Encounter

calibre -rve

‑convert_to edi_blk

‑output_file rdb_blk

Linux shell

Return Values

Returns 1 for success or 0 for failure.

Description

Imports a file saved from a previous Calibre RDB conversion to Cadence Encounter route blockages as routing blockages in the current design session. The design or cell containing the results must be the working design in Cadence Encounter or an error occurs.

The route blockages can be viewed in the Cadence Encounter session after the import. Turn on visibility of blockage type “Routing Blkg” in order to view the route blockages.
