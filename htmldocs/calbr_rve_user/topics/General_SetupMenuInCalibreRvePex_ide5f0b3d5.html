Setup Menu in Calibre RVE for PEX

This section describes the contents of the Setup menu that are specific to Calibre RVE for PEX.
Table 1. Setup Menu Contents in Calibre RVE for PEX

Menu Item

(restriction)

Description

Password

(Calibre RVE for LVS, PEX, and PERC)

Opens a dialog to set a password for the database.

 

Note: The password protects the PHDB and XDB, not the SVDB. By default, the PHDB and XDB are not loaded until they are needed1. Actions such a highlighting or using the Finder require the PHDB and XDB.
Parent Topic: Menu Bar in Calibre RVE for PEX
1 To change this behavior, choose Setup > Options, the Databases category, then expand the “Database Loading” section.
