Setup Menu in Calibre RVE for PERC

The Setup menu in Calibre RVE for PERC includes the Password selection.
Table 1. Setup Menu Contents in Calibre RVE for PERC

Menu Item

(restriction)

Description

Password

(Calibre RVE for LVS, PEX, and PERC only)

 

Not available when viewing Calibre PERC LDL results.

Opens a dialog box to set a password for the database.

 

Note: The password protects the PHDB and XDB, not the SVDB. Actions such a highlighting or using the Finder require the PHDB and XDB. By default, the PHDB and XDB are not loaded until they are needed. To change the default behavior, open the Options tab, choose the Databases category, and expand the “Database Loading” section.
