RotateXLabels

Key Value for DRC HTML Report Configuration File, histogram and colormap category

Usage: Optional. Valid in DRCHistogram report type, and in Common section.

Inheritance: Standard Inheritance

Specifies the orientation for X-axis labels of a histogram.

Allowed Values

0

Print X-axis labels horizontally. (default)

1

Print X-axis labels vertically.
