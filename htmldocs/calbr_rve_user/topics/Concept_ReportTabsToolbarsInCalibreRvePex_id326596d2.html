Report Tabs and Toolbars in Calibre RVE for PEX

The report tabs and toolbars in Calibre RVE for PEX provide the tools you need to analyze your parasitic extraction results.

Extraction Results Tab — This tab shows the results from the netlist extraction component of the Calibre nmLVS run, if present. Extraction results should be clean for a PEX run, and by default the tab is not opened if the results are clean. If you have extraction warnings or errors, see “Examining Connectivity Extraction Results” in the Calibre RVE for LVS section.

Comparison Results Tab — This tab show your LVS comparison results. Comparison results should be clean for a PEX run, and by default the tab is not opened if the results are clean. See “Comparison Results Tab” and “Examining LVS Comparison Results” in the Calibre RVE for LVS section.

Parasitics Tab — The Parasitics tab displays the results of a parasitic extraction run in a tabular view, as shown in “Session Window for Calibre RVE for PEX”. Details on using the features of the Parasitics tab are given in the section “Using the Net Parasitics Table” and in the reference section “Parasitics Tab”.

Standard RVE Toolbar —The Standard Toolbar in Calibre RVE is used control the behavior on highlight events, open a new database, and open the Options Tab in Calibre RVE.

Parasitics Toolbar — The parasitics toolbar is available when the Parasitics tab is active, as seen in the following figure. The selections in the parasitics toolbar are also available when you right-click a net in the parasitics table.
