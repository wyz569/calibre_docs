Hiding Unconnected Pins In the Internal Schematic Viewer

You can hide unconnected pins in cells and instances with more pins than a specified threshold value. The cell outline is shown as a dashed line if pins are hidden.

Prerequisites

  • A Calibre results database is open in Calibre RVE for LVS, PERC, or PEX.

Procedure

  1. Click the Options pane button (), choose the Schematic Viewer category, and enable “Hide unconnected pins on cells/instances with more than ___ pins.” This option is on by default.
  2. Open the source or layout netlist in the Internal Schematic Viewer.
  3. Select a cell or instance in the hierarchy tree to view. If the cell has more than the specified number of pins, the cell outline is shown as a dotted line.
  4. In the hierarchy tree, click a device within the cell to add it to the view. The device pins are automatically shown, but all other pins remain hidden. An example is shown in the following figure, where the device MP0 in cell pll is shown.


  5. Right-click on the cell outline for the following menu options:

    • Show pins — Shows all pins. Only displayed when pins are hidden.

    • Hide pins — Hides unconnected pins, unless they are selected to show in the Show/Hide Pins dialog box. Only displayed if all pins are shown.

    • Show/Hide Pins — Opens the Show/Hide Pins dialog box. Use the dialog box to select individual pins to show or hide.

    The above menu selections are only shown if “Hide unconnected pins on cells/instances with more than ___ pins” is enabled.
