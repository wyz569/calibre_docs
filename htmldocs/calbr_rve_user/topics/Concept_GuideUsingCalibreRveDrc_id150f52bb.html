Guide to Using Calibre RVE for DRC

Some common tasks in Calibre RVE for DRC are highlighting, grouping, filtering, and waiving results.

The following table lists some common DRC results viewing tasks and gives the location of the control. Also see “Keyboard Shortcuts (Hotkeys) in Calibre RVE for DRC”.
