Using Calibre RVE for LVS

This chapter describes how to use Calibre RVE (Results Viewing Environment) for LVS. Instructions for the tasks of isolating shorts, examining Calibre nmLVS comparison results, and using Design Fix Suggestions are included. Detailed information for the GUI elements is given in a reference section.

This chapter assumes that you have an SVDB from a Calibre nmLVS run open in Calibre RVE, and are familiar with the general operation of Calibre RVE. See these sections for general information on Calibre RVE:
