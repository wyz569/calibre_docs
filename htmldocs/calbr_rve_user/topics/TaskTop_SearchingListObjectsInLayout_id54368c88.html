Searching for a List of Objects in the Layout

You can use the Info Pane to search for a list of objects in the layout.

Tip

The Finder Tab > Find Particular Objects can also search for a list of items; the Finder tab search allows for search in the source netlist and application of filters.

Prerequisites

  • Calibre RVE for LVS, PERC, PEX or 3DSTACK is open to a valid results database.

Procedure

  1. Choose View > Info to open the Info Pane.
  2. Enter the search list using the following format:
    object1 ... objectn [in cell]

    The optional “in cell” qualifier specifies the cell to search in, and must appear at the end of the list. The search is in the top cell by default. Example list: 

    ibias iref in clock_cell
    Figure 1. Search for List of Objects in Info Pane

  3. Click the Info button () or press Enter from the Info entry field to start the search.

    The results are returned in the results area, as shown in Figure 1. All object types for a particular design element are returned; for example, an object named ibias may be listed as both a net and a port.

  4. Highlight or view information about the objects with one of the following methods:

    • Click the Show Highlight Menu button () to view highlighting and display options. You can choose to highlight all objects of a particular type, such as all nets or all instances.

    • Click the colored icon next to an object to highlight it.

    • Expand the entry in the result listing to view details.

    Note: you can click the Show Highlight Menu button () prior to populating the Info pane result list.
