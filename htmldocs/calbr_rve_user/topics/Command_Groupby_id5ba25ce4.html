GroupBy

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: ByErrorColormap, ByErrorHighlight, ByErrorResults, ByErrorResultsSummary, CellDrillDown, CellSummary, ChipSummary, Common section, NetDrillDown, NetSummary, TotalSummary, WindowDrillDown, WindowSummary

Optional key name that specifies the order in which to group columns in the text report, or bars in the bar chart.

Syntax

GroupBy = {Bins | Metrics} | hierarchy_specification

Parameters

Bins

A required literal that specifies the grouping by bins for rule-based reports. If an invalid value is specified, no reports are generated for the section. Can indirectly reference other sections.

Metrics

A required literal that specifies the grouping by metrics for rule-based reports. This is the default. If an invalid value is specified, no reports are generated for the section. Can indirectly reference other sections.

hierarchy_specification

A required key value that specifies the grouping for error-based reports. See the GroupBy key for DRC HTML reporting for a discussion of hierarchy specification.
