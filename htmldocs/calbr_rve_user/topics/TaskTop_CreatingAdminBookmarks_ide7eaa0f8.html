Creating Admin Bookmarks

You choose Export Bookmarks in the Bookmarks menu to create an admin bookmark file. If you already have an admin bookmark file, you can use this procedure to add new admin bookmarks to the admin bookmark file. Admin bookmarks are permanent bookmarks associated with the results database.

Procedure

  1. Set up each view you want to define as an admin bookmarks and save it as a user bookmark, as described “Creating and Managing User Bookmarks in Calibre RVE for DRC”.
  2. Delete any user bookmarks that you do not want to save as admin bookmarks. See Figure 1 for instructions on deleting a bookmark.
  3. Click the Bookmarks icon, then select Export Bookmarks. This opens the “Export bookmarks” dialog box, as shown in the following figure.


    Enter the base filename for the admin bookmark file with one of the following methods:

    • Bookmark filename is same as results database name — In this case, the admin bookmarks are read automatically when the database is opened. The admin bookmark file must be located in the same directory as the results database.

      Example: drc.results.xrvedb if the results database is named drc.results.

    • User-provided bookmark filename — In this case, you must specify the location of the admin bookmark file with the MGC_RVE_DB_OPTIONS_FILE environment variable.

      Example:

      setenv MGC_RVE_DB_OPTIONS_FILE /admin/perc_voltage_bookmarks.xrvedb

      If MGC_RVE_DB_OPTIONS_FILE is used to specify an admin bookmark file, any results_db.xrvedb file is ignored, where results_db is the name of the results database.

Results

Each user bookmark in the session is saved to the filename specified in the Export Bookmarks dialog box.

If your Calibre RVE session already has some admin bookmarks defined, the user bookmarks defined for the current session are added to the existing admin bookmarks.
