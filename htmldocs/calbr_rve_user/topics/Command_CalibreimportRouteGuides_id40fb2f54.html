Calibre::import_route_guides

Imports a file saved from a previous Calibre RDB conversion to Synopsys IC Compiler route guides. The cell containing the results you want to import must be the working design in your Synopsys IC Compiler session. The route guides can be viewed in the Synopsys IC Compiler session after the import. Used in the Synopsys IC Compiler environment.

Usage

Calibre::import_route_guides  rdb_rg

Arguments

rdb_rg

A file saved from a previous Calibre RDB conversion to Synopsys IC Compiler route guides. The rdb_rg file is created with one of the following commands, where only the options specific to selecting route guide conversion and file output are given:

Command

Options

Execution environment

Calibre::convert_to_route_guides

‑output_file rdb_rg

Synopsys IC Compiler

calibre -rve

‑convert_to icc_rgf

‑output_file rdb_rg

Linux shell

Return Values

Returns 1 for success or 0 for failure.
