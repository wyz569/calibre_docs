Starting Calibre RVE

You can start Calibre RVE from a supported design tool, from a Calibre Interactive session, or from the command line. Starting Calibre RVE from a viewer or a Calibre Interactive session connected to a viewer automatically connects Calibre RVE to the viewer.

For information on the Calibre RVE command line, see “Calibre RVE Command Line Reference”.

Prerequisites

  • Calibre RVE System Requirements are satisfied. This includes platform support, licensing, and connections to supported layout and schematic editors.

Procedure

  1. Start Calibre RVE with one of these methods:

    To open Calibre RVE …

    Do the following …

    From a design tool

    Choose one of the following:

    • Verification > Start RVE

    • Calibre > Start RVE

    depending on your design tool.

    From Calibre Interactive

    Click Start RVE on the left panel of Calibre Interactive, as shown in Figure 1
  2. If you opened Calibre RVE from a design tool, enter the results database name, the database type, and other options as needed. See Figure 1 for a view of the opening dialog box.

    When you open Calibre RVE from Calibre Interactive, the database specified on the Outputs pane is opened automatically.

    Figure 1. Starting Calibre RVE

Results

The results are shown in the Calibre RVE session window. Calibre RVE is automatically connected to an open design tool for highlighting in the following cases:

  • You opened Calibre RVE from the design tool.

  • You opened Calibre RVE from a Calibre Interactive session that is connected to the design tool.
