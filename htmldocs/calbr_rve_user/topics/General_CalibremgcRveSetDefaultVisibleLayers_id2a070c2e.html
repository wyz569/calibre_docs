Calibre::mgc_rve_set_default_visible_layers

Command in Synopsys IC Compiler and IC Compiler II that saves the current set of visible layers as the default set of visible layers.

Syntax

Calibre::mgc_rve_set_default_visible_layers

Description

This function saves the current set of visible layers as the default set of visible layers. It is used internally by Calibre RVE as part of the Restore Default Layers feature (see “Restore Default Layers Toolbar”.)
