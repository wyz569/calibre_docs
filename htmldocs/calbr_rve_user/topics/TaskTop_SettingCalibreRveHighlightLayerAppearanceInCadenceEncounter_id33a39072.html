Setting Calibre RVE Highlight Layer Appearance in Cadence Encounter

You can control the appearance of the layers used for Calibre RVE highlights.

Procedure

  1. Choose Calibre > Setup > RVE from the Cadence Encounter menu; this opens the Calibre Setup RVE Highlight Layers dialog box.
  2. Make your selections for layer appearance.
  3. Select Save to write the Tcl commands for setting the highlight layer appearance to a file. In later Encounter sessions, you can set the layer appearance by loading this file after the Calibre—Encounter integration code is loaded.
