The Calibre Menu in Cadence Virtuoso

The Calibre menu in Cadence Virtuoso layout and schematic windows (windows of type maskLayout or schematic by default) has the selections to start Calibre Interactive, start Calibre RVE, set up options, and map schematic nets.

The following tables describe the menu contents:

  • Calibre Menu (Table 1) — The menu items available from a layout or schematic window.

  • Calibre Setup Menu (Table 2) — The items in the Calibre > Setup submenu in a layout or schematic window.

Table 1. Calibre Menu in Cadence Virtuoso

Menu Item

Description

Run nmDRC

Starts Calibre Interactive DRC. (not available in schematic view)

Run DFM

Starts Calibre Interactive DFM. (not available in schematic view)

Run nmLVS

Starts Calibre Interactive LVS.

Run PERC

Starts Calibre Interactive PERC.

Run PEX

Starts Calibre Interactive PEX. (not available in schematic view)

Start RVE

Starts Calibre RVE.

Clear Highlights

Clears Calibre RVE highlights in the current cell.

Setup

See Table 2.

OPC Workbench

Contains a submenu for using Calibre WORKbench, if installed.

RealTime DRC

See the Calibre RealTime Custom User’s Manual.

Map Schematic Net

Only available in the schematic view. Opens the dialog box “Map Schematic Nets to CalibreView Nets.”

See the sections “Mapping Schematic Nets to Calibre View Nets for Simulation and Plotting” and “Backannotation of Parasitic Values to the Schematic in Cadence Virtuoso” in the Calibre Interactive User’s Manual.

 

The Calibre > Setup menu has the following items:

Table 2. Calibre Setup Menu in Cadence Virtuoso

Menu Item

Description

Layout Export

Displays a dialog that controls options used while exporting layout databases to Calibre. See “Layout Export in Cadence Virtuoso” in the Calibre Interactive User’s Manual.

Netlist Export

Controls options used while exporting schematics to Calibre. See “Netlist Export in Cadence Composer” in the Calibre Interactive User’s Manual.

Calibre View

Allows you to set up Calibre View options for Calibre parasitic extraction.

Environment variables and SKILL variables can be used as part of a filename. See “Calibre View Setup Dialog Box” in the Calibre Interactive User’s Manual.

RVE

Allows you to set up your Cadence Library for Calibre RVE and your Highlight Layers. See “Setup RVE Dialog Box in Cadence Virtuoso”.

Socket

Specifies the socket that Calibre Interactive and Calibre RVE use to communicate with Cadence Virtuoso. The socket is not initialized until needed.

If you want to initialize the socket during the loading of calibre.skl, set the environment variable MGC_RVE_INIT_SOCKET_AT_STARTUP to any value before loading calibre.skl. The socket number is displayed in the CIW window.

The socket port number may be specified by the MGC_CALIBRE_LAYOUT_SERVER environment variable, as described in “Setting the Calibre RVE Socket Port for a Layout Viewer”.
