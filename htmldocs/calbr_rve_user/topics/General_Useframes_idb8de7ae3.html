UseFrames

Key Value for DRC HTML Report Configuration File, customization category

Usage: Optional. Valid in Common section only.

Inheritance: Not applicable.

Specifies whether to use HTML frames to display the report.

When frames are used, the left frame displays the report index and DRCResultsSummary reports. The right frame displays the report header, footer, and all other reports. The values Report, GroupBy, Check, and Cell create separate HTML pages for reports other than the DRCResultsSummary; these pages are accessible from the report index with links. These settings are useful with large results databases, as they break the report into smaller files that load faster.

When printing the report, UseFrames = 0 (the default) is most convenient.

Allowed Values

0

Do not use HTML frames. (default)

1

Use HTML frames.

Report

Use HTML frames and create separate HTML files for each report that is not a DRCResultsSummary report. The index in the left frame includes a link to each report.

GroupBy

Create the DRC HTML report as with the Report option. In addition, separate each DRCResults report into separate HTML files corresponding to the first level of the GroupBy setting for the report.

Check

Create the DRC HTML report as with the Report option. In addition, if a DRCResults report is grouped by Check as the first level, create separate HTML files corresponding to each check.

Cell

Create the DRC HTML report as with the Report option. In addition, if a DRCResults report is grouped by Cell as the first level, create separate HTML files corresponding to each cell.

Examples

The following Common section specifies UseFrames = GroupBy. The initial view of the DRC HTML report is shown in the figure following the example. 

[Common]
LayoutWidth = 200
LayoutHeight = 200
WorldView = 1
UseFrames = GroupBy
ShowCheckText = 0
