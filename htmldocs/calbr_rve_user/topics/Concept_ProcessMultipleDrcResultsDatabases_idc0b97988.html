Process Multiple DRC Results Databases

You can process multiple DRC results databases (RDBs) with the same DRC HTML report configuration file.

Use one of the following methods:

  • Automatically for side RDBs included in the main results database; see “Automatically Process Side (Auxiliary) RDBs”.

  • On the command line:

    calibre -rve -drc rdb_1 rdb_2 -report html_rpt.ini -outputdir html_out

    where rdb_1 and rdb_2 are the results databases to process. In this case the results databases are assigned the default RDB type of “General”. Highlighting or creating colormaps is not possible with this method, since a DRC HTML Report Database Map File is not specified.

  • With a DRC HTML Report Database Map File:

    You can use the ‑input db_to_layout_map_file command line option to specify a file with multiple databases to process. The DRC HTML Report Database Map File is also used to specify a layout file for highlights and colormaps, and allows you to assign an rdbType to a results database. All these features are used in the following example for PERC LDL, which produces ASCII type DRC results databases.

    • Example DRC HTML Report Database Map File (db_map.txt):

      ; RDB(required), RDBType, Layout, Layer Properties, layout type
$CD_DIR/perc.rep.cd.rdb,perc_cd,$CD_DIR/simple.hp3.mult.port.gds,,
$P2P_DIR/perc.rep.p2p.rdb,perc_p2p,$P2P_DIR/simple.hp3.diode.gds,,

      Two results databases are processed ($CD_DIR⁄perc.rep.cd.rdb and $P2P_DIR⁄perc.rep.p2p.rdb); the databases are assigned a user-specified rdbType of perc_cd and perc_p2p, respectively. Different layout files are specified for each database to use for creating the layout images. Each results database is processed in the order specified and an index is generated for each results database.

      Note: To create an HTML report for PERC LDL results, you must process the RDBs created by the Calibre PERC LDL run.

    • Example configuration file (rpt_perc_ldl.ini):

      [Common]
UseFrames = 1
LayoutWidth = 200
LayoutHeight = 250
WorldView = 0
[PERC LDL CD]
ReportType = DRCColormap
GroupBy = Cell
ApplyTo = perc_cd
Property = ERROR(%)
[PERC LDL P2P]
ReportType = DRCColormap
GroupBy = Cell
ApplyTo = perc_p2p
Property = R(ohms)

      This example creates a colormap for each results database. The ApplyTo keyword with the user-specified rdbType is used so that the correct property is colormapped for each database.

    • Example command line (broken into two lines):

      calibre -rve -drc -input db_map.txt -report rpt_perc_ldl.ini \
	‑outputdir html_rpt_perc_ldl
