TrueColormap

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: ByCellColormap, ByWindowColormap, and Common section

An optional key name that specifies whether to use truecolor in colormaps. This key is enabled by default and causes the bin distribution to be determined automatically for colormaps.

Syntax

TrueColormap = {0 | 1}

Parameters

0

A required literal that disables truecolor.

1

A required literal that enables truecolor. This is the default setting.

The bin distribution for colormaps is determined automatically. If PropRangeDistributionType=linear, PropLinearDivisionCount is ignored for colormaps. If PropRangeDistributionType=custom, only the range endpoints specified by PropCustomRangeDistribution are used for colormaps.
