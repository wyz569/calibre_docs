Calibre RVE System Requirements

Certain requirements for the computing platform, licensing, and environment variables must be met in order to run Calibre RVE.

The following list gives the requirements for Calibre RVE:

  • Platform support — Calibre RVE is available on all supported platforms found in the Calibre Administrator’s Guide.

  • Licensing — Calibre RVE requires a Calibre RVE/Query Server product license. This can be a standalone license or part of the Calibre Cell/Block (CB) product package.

    The following Calibre RVE applications require additional licenses:

    • Export Waivers in Calibre RVE for DRC — requires a Calibre Auto-Waivers license.

    • Interactive Short Isolation in Calibre RVE for LVS — requires a Calibre nmLVS license.

    • Calibre RVE for DFM, and any applications that open a DFM Database — requires a Calibre YieldServer license.

    See the Calibre Administrator’s Guide for complete licensing information.

    See “License Timeout in Calibre RVE” for information on setting the license timeout period.

  • Environment variable for software tree — Before invoking Calibre RVE, you must set an environment variable to the location of your Calibre software tree. See “Setting the CALIBRE_HOME Environment Variable” in the Calibre Administrator’s Guide for details.

  • Layout Design Tool (Recommended) — Access to a layout editor or viewer with a Calibre integration is recommended. When using Calibre RVE, Calibre results can be highlighted in the design for easier debugging.

    Integrated layout viewers include Calibre DESIGNrev, Siemens Sierra Olympus/Pinnacle, Siemens L-Edit, Cadence Virtuoso, and Synopsys IC Compiler. See “Integration to Design Tools” for a complete list of integrated viewers.

  • Schematic Design Tool (Optional) — Access to a schematic viewer with a Calibre integration is optional.

    Calibre RVE for LVS/PERC/PEX includes an internal schematic viewer that displays rendered schematics from the netlist, so an external schematic viewer is not required.

    Integrated schematic viewers include Siemens IC Station/Pyxis Schematic, Siemens Tanner S-Edit, and Cadence Composer. See “Integration to Design Tools” for a complete list of integrated viewers.
