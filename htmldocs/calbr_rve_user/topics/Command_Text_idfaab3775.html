text

Shape element for Symbol Library file

The text element defines a text label for the symbol.

Usage

text string justify[.orient] x y size[i]

Parameters

Parameter

Description

text

A required keyword

string

The text string. Use “\n” in the string for a newline. For example: “test\nobject”

justify[.orient]

The justification and optional orientation, where the justification specifies the origin of the attribute and the orientation specifies horizontal (default) or vertical text.

  • justify — One of -ll, -lc, -lr, -cl, -cc, -cr, -ul, -uc, or -ur (lower-left, lower-center, lower-right, …, upper-right, respectively)

  • orient — Append .v to justify to specify vertical text. For example: -ll.v

x y

The x and y coordinates for the attribute.

size[i]

The font size and optional “i” character to indicate an invisible attribute. For example: 12 or 10i
Tip

Invisible attributes are shown by default. You can change this with the setting “Show ‘invisible’ properties specified in symbol library files” located on the Options tab in the Schematic Viewer category.

Examples

text "test" -cl 5 -32 10 
text "test\nobject" -cl 5 32 10
text "vertical" -cr.v 10 32 10
