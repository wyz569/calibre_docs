Viewing Properties in Calibre RVE for DRC

DRC ASCII results databases can contain properties attached to geometric results. Such properties are typically created by Density and DFM operations. The database properties are shown by default in the Details format and Results Data views of the report tab. You can also instruct Calibre RVE to display database properties when highlighting results and create a histogram or colormap.

Calibre RVE can also calculate several properties associated with a DRC result. The calculated properties are defined in Table 1. Choose Setup > Options, the DRC Browser category, and enable “Show calculated properties” and “Classify result shapes” (for #SC) to turn on the display of calculated properties. See Table 2 for other controls related to property display.

Table 1. Calculated Properties in Calibre RVE for DRC

Property Name

Property calculated for:

Definition

#EC

Edge pair

Common edge length, or common edge projection. Same as the EC() measurement function used in DFM Property.

#EW

Edge pair

Edge width, or minimum separation distance between the edge pair. Same as the EW() measurement function used in DFM Property.

#EL

Edge pair

The average edge length.

#PA

Polygon

Polygon area.

#PP

Polygon

Polygon perimeter.

#SC

Any result

Shape Class. Calibre RVE uses an internal algorithm to classify the shape for each result, and the #SC value is an index into the set of different shapes found in the results database.

The #SC value differs for result shapes that vary by rotation or mirroring but are otherwise identical; the #SC value does not vary by displacement. The #SC value may vary from run to run, but is constant across Calibre RVE sessions for the same results database.

#Flipped

Results from cloned cells

See “Results From Cloned Cells”. #Flipped and #Rotated are displayed if present; it is not necessary to enable “Show calculated properties.”

#Rotated

Several controls affect the display and highlighting of properties attached to a DRC results database, as given in Table 2.

Table 2. Controls for Property Display in Calibre RVE for DRC

Task

Control

Display properties in tables.

  • Database properties — Shown by default if present in database.

  • Calculated properties — Enable “Show calculated properties” and “Classify result shapes” (for #SC) on the Options tab in the DRC Browser category.

To display properties in the Result View, select either View > Results Options > Details or View > Results Options > Result Data Pane.

Choose the properties displayed in the tables.

Click the column filter icon and select the columns to display. See “Selecting Columns to Display”.

Histogram a property.

Select View > Histogram and select the property to histogram or right-click in the tree view. You must have properties present in the database, or have “Show calculated properties” enabled. See “Histogram Window”.

Histograms may only be created for numeric properties.

Colormap a property.

Select View > Colormap and select the property to colormap, or right-click in the tree view. The color scheme and data distribution defined on the Setup > Options > Histograms pane are used to create the colormap.

Colormaps may be created for properties with numeric, string, or mixed numeric and string values. See “Creating Histograms and Colormaps”.

Change display precision.

Adjust the value in the “Apply precision to display of reals” field on the Options tab in the Session category.

This setting affects the display of numbers in tables and the display of property values in highlights.

Group results by property.

Select View > Tree Options > Group By > Statistics …. This groups the tree by property name.

Group results by property value.

Select View > Tree Options > Group By > Custom > property1, and continue selecting the group hierarchy. The tree view is grouped first by the value for property1, then according to remaining group order. See Figure 1.

Display properties in the layout viewer when highlighting.

Enable “Show Properties” on the Options tab in the Highlighting category, “DRC/DFM Highlighting” section.

Note: To display calculated properties when highlighting you must also enable “Show calculated properties” on the Options tab in the DRC Browser category.

Also see the RVE Show Properties rule check comment.

Tip

Set the precision for the display of property values with the “Apply precision to display of reals” setting in the Session category on the Options tab.

Change text size for property display in the layout viewer.

Adjust the value in the “Highlight Text Size” field on the Options tab in the Highlighting category, “Layout Highlighting” section.

If you are using the Calibre DESIGNrev layout viewer, see “Setting Text Size for Highlighting with Calibre DESIGNrev” for additional settings.

Also see the RVE Highlight Text Size rule check comment.
