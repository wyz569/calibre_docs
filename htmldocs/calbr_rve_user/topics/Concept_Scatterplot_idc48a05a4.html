ScatterPlot

This is an example of the ScatterPlot report.
Figure 1. ScatterPlot Report

Required keys: XProperty, YProperty

Optional keys: XLayer, YLayer, XAxis, YAxis, PlotTitle, LineName, Sorting, ColorSet

Example

; Create Scatter plot
[Scatter Plot]
; required
ReportType = ScatterPlot
XProperty = Thickness_Z1
YProperty = Density

; optional
; XLayer = YLayer if neither or only one specified, 
; all layers with data are used if neither XLayer nor YLayer is specified
XLayer = M2_data_detail
; YLayer
PlotTitle = "Density v Z1"
XAxis = Z1
YAxis = Density
; LineName = M2
; Sorting = X
; ColorSet = orange
