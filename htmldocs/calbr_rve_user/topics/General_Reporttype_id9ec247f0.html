ReportType

Key Value for DRC HTML Report Configuration File, report specification category

Usage: Required in all sections except the Common section. Not valid in the Common section.

Inheritance: Not allowed.

Specifies the type of report to generate, except for ReportType of DRCFilter. Filter settings apply to all report types.

Allowed Values

DRCResultsSummary

Corresponds to the tree (hierarchy) view in Calibre RVE for DRC. The key GroupBy determines the tree view.

If another report section, such as DRCHighlight, references the DRCResultsSummary section with the GroupBy key, links to the report section are automatically placed in the DRCResultsSummary report.

If a report section does not reference a DRCResultsSummary with the GroupBy key, you can place links to the report in the DRCResultsSummary with the LinkFrom key.

DRCResults

Corresponds to the result view in “Details” format in Calibre RVE for DRC.

DRCHighlight

Corresponds to a “Highlight all” action performed in the tree view in Calibre RVE for DRC. “Highlight all” is performed on each leaf in the GroupBy hierarchy.

DRCHistogram

Corresponds to a Histogram action in the tree view in Calibre RVE for DRC. A histogram is created for each leaf in the GroupBy hierarchy. You can specify the property to histogram with the Property key; otherwise a histogram is created for each property in the database.

DRCColormap

Corresponds to a Colormap action in the tree view in Calibre RVE for DRC. A colormap is created for each leaf in the GroupBy hierarchy.You can specify the property to colormap with the Property key; otherwise a colormap is created for each property in the database.

DRCFilter

Indicates the section contains filter specifications that can be referenced by other sections. Use the Filter keyword to reference a DRCFilter section. See Example —Named Filters in “Applying Filters to DRC HTML Report Sections”.
