Setting the Calibre RVE Socket Port for a Schematic Viewer

You can set the socket port manually through the Calibre RVE interface or by using an environment variable. By default, Calibre RVE uses socket port 9199 to communicate with the schematic viewer. The tool automatically searches for an available socket if it cannot obtain the default socket. It initializes the first available socket between the numbers 5000 and 9999 and reports the socket number through a dialog box.

Full instructions are given in the section “Manual Connection to a Viewer”.

Prerequisites

  • Calibre RVE must be open if you are setting the socket number through the GUI.

Procedure

Set the schematic viewer socket port with one of the following methods:

To set the schematic socket port with …

Do this …

Calibre RVE

  1. Choose Setup > Design Tools in Calibre RVE.

  2. Enable “Schematic Design Tool.”

  3. Do one if the following, depending on connection type:

    Same Connection: If your layout and schematic viewers are running as the same application, do the following:

    1. Enable “Use the same connection as the layout tool” in the Schematic Design Tool section.

      Calibre RVE sends schematic and layout highlight information over the same socket connection.

    2. In the Layout Design Tool section, enter the Host Name and Socket Number. The default host name is “localhost”.

    Different Connection: If your schematic viewer is a separate application from the layout view, do the following:

    1. Disable “Use the same connection as the layout tool.”

      Calibre RVE sends layout and schematic highlight information over separate socket connections.

    2. Enter the port number in the “Socket Number” field in the section below the “Use the same connection as the layout tool” setting.

  4. Click the connect button () to start the connection. The “Tool Information” field is filled in when the connection is made. The status is reported in the Connection Icon in Calibre RVE.

  5. Click Apply (in the left panel of the Options tab).

Environment variable

Set the MGC_CALIBRE_SCHEMATIC_SERVER environment variable as shown here for the C-Shell:

setenv MGC_CALIBRE_SCHEMATIC_SERVER [<host>:]<port>

The <host> parameter is set to localhost by default, but can be set to a different host. The <port> parameter must be a numerical value.

To disable socket communication to the schematic viewer, set MGC_CALIBRE_SCHEMATIC_SERVER to “ ” (blank). The schematic server then does not attempt to open the communication socket.
