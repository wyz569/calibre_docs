Window Menu in Calibre RVE

The Window menu has several selections that are common to all Calibre RVE applications.
Table 1. Window Menu Contents in Calibre RVE

Menu Item

Description

Tab Control

  • New tab group

  • Move to next tab group

  • Move to previous tab group

  • Close

These selections control the position of report tabs. The selections are only available when a report tab is selected.

Choose “New tab group” to create a new pane within the session window, and move the report tab to the new pane. The other selections work as indicated.

Window Control

  • Restore

  • Minimize

  • Maximize

  • Undock

  • Close

  • Close All

  • Dock All

These selections control the position of windows such as the Internal Schematic Viewer, histogram window, and Navigator pane. With the exception of Dock All, the selections are only available if such a window is docked within the Calibre RVE display and selected.

Window Arrangement

  • Cascade

  • Tile Horizontal

  • Tile Vertical

  • None

These selections control the arrangement of multiple windows such as the Internal Schematic Viewer, histogram window, and Navigator pane. The windows must be docked within the Calibre RVE display. The selections are not available unless such a window is selected.
