Viewing Separate Placements of a Result in the PERC Results Tab

A rule in Calibre PERC may produce more than one placement of a result, and each instance or placement of a result can be displayed in the Result View. The option “Show only one placement for each hierarchical PERC result” on the Options tab in the LVS/PERC/PEX category must be disabled when you load the database in order to view separate placements of a result.

The hyperlinks and text (except coordinates) for each result are specific to that placement. The display in the Result Data Pane does not adjust the text of the coordinate display, but instead adjusts the coordinates as needed when a hyperlink is clicked.

Prerequisites

  • Calibre PERC results are open in Calibre RVE. See “Opening Calibre PERC Results in Calibre RVE”.

Procedure

  1. Click the Options button () to open the Options pane.
  2. Choose the category LVS/PERC/PEX from the left column if the Setup LVS⁄PERC⁄PEX Browser Options pane is not displayed.
  3. Disable “Show only one placement for each hierarchical PERC result”.
  4. Click Apply.
  5. Click the PERC Results tab to make the results tab active.
  6. Choose File > Reload Database if you changed the setting of “Show only one placement for each hierarchical PERC result” in Step 3.

    The option setting will be in effect the next time you open a Calibre PERC database.

    Note:

    You can waive individual placements with “Show only one placement for each hierarchical PERC result” disabled. If selected placements for a result have been waived with “Show only one placement for each hierarchical PERC result” disabled, the waived status for the first placement is displayed when you view results with “Show only one placement for each hierarchical PERC result” enabled.

    When waiving results, the setting of “Show only one placement for each hierarchical PERC result” should be the same between sessions to avoid confusion and undesired results.
