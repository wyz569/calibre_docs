Viewing Drawn Layer/Purpose Pairs in the Calibre RVE for LVS Display

You can specify a layer map file in Calibre RVE for LVS, PERC, and PEX in order to have the layer name/purpose pair displayed in Calibre RVE. This makes it easier to correlate the Calibre layers names to the layer names used in your layout viewer. You specify the layer map file on the Options tab, LVS/PERC/PEX category with the “Use Viewer Layer Map file” option.

This option affects the display of original layer names in Info results, the layer filter table on the Highlights tab (Highlight > Setup Highlights), the Filter Layers table on the Finder tab, and the layout net right-click Highlight > Highlight on Layer dialog box. If already open, these tabs must be closed and re-opened in order to see the effect of the option. The option does not affect highlighting.

Prerequisites

  • A layer map file with the following format: 

    LayerName  LayerPurpose  GDS_number  GDS_Datatype

    where LayerName and LayerPurpose are the layer and purpose names used in the layout viewer. (This is the same layermap file format used for Cadence Virtuoso streamout.)

  • A Calibre nmLVS, Calibre PERC, or parasitic extraction results database is open in Calibre RVE.

  • The Calibre rule file is available from the results database. Calibre RVE uses the Layer and Layer Map statements in the rule file when “Use Viewer Layer Map file” is enabled.

  • The persistent hierarchical database (PHDB) is part of the results database. See the PHDB keyword for the Mask SVDB Directory statement.

Procedure

  1. Click the () Options button to open the Options tab.
  2. Select the LVS/PERC/PEX category.
  3. Enable “Use Viewer Layer Map file” and specify the layer map file.
  4. Click Apply.

Results

When you view Info results or view the layout net right-click Highlight > Highlight on Layer dialog, the drawn layer/purpose pair is shown for original layers and the Calibre layer name is shown for derived layers. When you view the Filter Layers table on the Finder tab or the layer filter table on the Highlights tab, a Drawn Layer column is displayed in addition to the Layer column.

If multiple drawn layers are mapped to the same Calibre layer, Calibre RVE displays the “drawing” layer if one exists; if multiple drawing layers are mapped, Calibre RVE arbitrarily selects one. If there is no drawing layer, Calibre RVE arbitrarily selects a layer name for display. Similarly, if one GDS layer is mapped to multiple Calibre layers, Calibre arbitrarily selects a layer name for display.

The option does not affect the display of derived layers and does not affect highlighting.
