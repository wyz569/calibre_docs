Displaying Result Data as Properties in Cadence Virtuoso

Calibre RVE for DRC can send result data as properties when highlighting a result in Cadence Virtuoso.

Procedure

  1. Open your layout in Cadence Virtuoso.
  2. Set the highlight layer type in Cadence Virtuoso as follows:
    1. Choose Calibre > Setup > RVE from the Cadence Virtuoso menu bar to open the Setup RVE dialog box.
    2. Select “Warning Markers” or “Error Markers” for the setting “RVE Highlight layers.”
    3. Click OK to close the dialog.
  3. Choose Calibre > Start RVE to start Calibre RVE and open your DRC results database.
  4. In Calibre RVE, choose Setup > Options, the Highlighting category, expand the “DRC/DFM Highlighting” area, and enable the setting “Annotate result data to highlights as properties,” then click Apply.
  5. Highlight a result by double-clicking it in the Calibre RVE display.
  6. View the properties as follows:
    1. In Cadence Virtuoso, choose Verify > Markers > Explain to open the marker text display box.
    2. Click in the result marker. The marker text display box shows the result properties; you may need to expand the display box and scroll up to view all of the properties.
