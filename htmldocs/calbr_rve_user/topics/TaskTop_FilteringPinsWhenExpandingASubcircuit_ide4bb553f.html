Filtering Pins When Expanding an Instance

You can select the pins that are displayed when you expand an instance. This is useful for instances with many pins so that you can focus on the pins of interest. When expanding an instance within the sub-hierarchy of a given instance, the filter dialog includes the name of the external net that is connected to the pin—this makes it easy to select pins to show based on the connected net.

Prerequisites

  • A SPICE netlist is open in the Internal Schematic Viewer or the Calibre RVE SPICE File Viewer.

  • The option “Hide unconnected pins on cells/instances with more than ___ pins” is enabled. This option is on by default. The option is in the Schematic Viewer category of the Options pane.

Video



Procedure

  1. Click an instance to display it.

    If the number of pins exceeds the pin threshold the instance is shown with a dotted outline and no pins are displayed.

  2. Perform the following steps to selectively expand the instance.

    Step

    Explanation or Screenshot

    If needed, right-click on the instance outline and select Show/Hide Pins to open the dialog box. Select the pins to display.

    Double-click a pin to view the nets and instances connected to the pin.

    Right-click on the instance in the sub-hierarchy and select Show/Hide Pins.

    Select the pins/nets to show. The currently displayed pins are selected for you. You can enter a space separated list of filter terms in the filter field.

    The “Net” column enables you to select pins based on the external net the pin is connected to.

    Result: Only the selected pins are shown.
