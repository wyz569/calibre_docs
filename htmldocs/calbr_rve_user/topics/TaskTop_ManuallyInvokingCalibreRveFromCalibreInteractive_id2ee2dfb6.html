Manually Invoking Calibre RVE from Calibre Interactive

You can open Calibre RVE manually from Calibre Interactive with the Start RVE button. Calibre RVE attempts to open the results database specified on the Outputs pane. If Calibre Interactive is connected to a supported design tool, Calibre RVE is automatically connected to the design tool.

Prerequisites

  • “Calibre RVE System Requirements”

Procedure

  1. Start Calibre Interactive and set up the options for your run.
  2. Make sure the option to invoke Calibre RVE automatically is not enabled. These options are explained in “Automatically Invoking Calibre RVE from Calibre Interactive”.
  3. Click the Run button in Calibre Interactive to start the Calibre run.
  4. Click the Start RVE button on the left panel of Calibre Interactive after the run is finished.

Results

Calibre RVE attempts to open the database defined in the Outputs pane. If Calibre Interactive is connected to a supported design tool, Calibre RVE is automatically connected to the design tool.

See Multithreaded Parallel Processing for Calibre RVE Operations (‑turbo Option) for how the ‑turbo option is set automatically in certain cases.

When Calibre RVE is invoked from Calibre Interactive, the settings for running on a remote host are taken from the Performance tab on the Run Control pane of Calibre Interactive.
