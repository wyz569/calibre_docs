View Menu in Calibre RVE for PERC

The View menu in Calibre RVE for PERC includes selections for viewing reports, setting view options, and creating histograms and colormaps.

Some menu selections are only available when viewing results from a Calibre PERC LDL run, and this is noted in parentheses.

Table 1. View Menu Contents in Calibre RVE for PERC

Menu Item

(restriction)

Description

Results

Lists the available results databases. Choose an item to open it in a new result tab.

Reports

Lists the available text reports and files. Choose an item to open it in a new report tab.

Rules

Lists the rule file(s). Choose an item to open it in a new report tab.

Navigator

Displays the Navigator Pane.

Info

Displays the Info Pane.

Finder

Opens the Finder Tab.

Schematics

Opens the Internal Schematic Viewer.

Tree Options

Displays a menu of display and grouping options for the Tree View. The selections depends on whether you are viewing Calibre PERC or Calibre PERC LDL results.

See Table 2 in this topic.

Result Options

Displays a menu of display options for the results view in the results tab. The selections depends on whether you are viewing Calibre PERC or Calibre PERC LDL results.

See Table 3 in this topic.

Check Text Pane

Toggles display of the Check Text pane, which displays the comment text for the check selected in the Tree View.

Histogram

Creates a histogram of the property you select. The histogram displays results for the cells or checks selected in the Tree View.

See “Creating a Histogram of Property Values”.

Colormap

(PERC LDL only)

Creates a colormap of the property you select. The colormap displays results for the cells or checks selected in the Tree View.

See “Histograms and Colormaps” and “Creating Histograms and Colormaps”.

 

The contents of the View > Tree Options menu are described in the next table. The contents vary depending on whether you are viewing Calibre PERC results or Calibre PERC LDL results. Some menu selections are only available when certain report tabs are selected, and this is noted in parentheses.

Table 2. View > Tree Options Menu in Calibre RVE for PERC

Menu Item

Description

Group By

Specify the grouping hierarchy for the tree.

  • Check / Cell

  • Cell / Check

  • Check

  • Cell

Group results with the specified hierarchy.

  • Statistics / Check / Cell

  • Statistics / Cell / Check

    (Calibre PERC LDL only)

Group first according to property name, then as indicated. Statistics, such as min, max, and mean, are reported for each property.

Results which include a given property are shown under the property name; a result which includes multiple properties is shown under each property.

  • Custom

Specify a custom grouping hierarchy. Use this selection to group by a property value.

Choose the desired hierarchy using the menu selections; click the top selection of a cascading menu to end the grouping hierarchy. See “Grouping Results in Calibre RVE for PERC”.

Show Empty Checks

Show empty checks in the Tree View.

This setting does not affect the display of checks that were aborted or not executed due to a runtime error; such checks are shown with the icon in the tree view.

Show Clean Cells

Show clean cells in the Tree View. Clean cells are only displayed if the top level grouping is by cell. Also see the setting “Show clean cells on startup” in the LVS⁄PERC⁄PEX category of the Options tab.

Show Branch Prefix

Show the label prefix for each branch in the tree view. For example, “Check rule_1” is shown in the tree view if this option is enabled, while “rule_1” is shown if the option is disabled.

 

The contents of the View > Result Options menu are described in the following table. The contents vary depending on whether you are viewing Calibre PERC results or Calibre PERC LDL results. Some menu selections are only available when certain report tabs are selected, and this is noted in parentheses.

Table 3. View > Result Options Menu in Calibre RVE for PERC

Menu Item

Description

List

Icons

Details

Choose the display format for the Result View:

List — Results are shown as their Result ID, with color indicating fixed or waived.

Icon — Results are shown as their Result ID, with an icon indicating fixed or waived.

Details — Individual results are shown in a table format.

Result Data Pane

Show the Result Data Pane.

Show Top Level Net Names

(topological results only)

In the Result Data Pane, also show the top-level net name for each placement-specific net. The highest logically equivalent net name is displayed after the net and path types.

Note: If the netlist is not already loaded, Calibre RVE will load the netlist when a result is first viewed with this option enabled. This may take some time for large netlists.

Result Tips

(PERC LDL only)

Show results information and waiver comments as tooltips for each result.

Result IDs

(PERC LDL only)

Display the unique result ID in the first line in the Result Data Pane.
