Setup RVE Dialog Box in Cadence Virtuoso

Open with: Calibre > Setup > RVE from Cadence Virtuoso

The Setup RVE dialog box controls highlighting behavior for the Cadence Virtuoso integration, such as custom colors, layer visibility, libraries, and highlight layers.

The Setup RVE dialog box has the entries described in the following table.

Table 1. Setup RVE Dialog Box in Cadence Virtuoso

Selection

Description

Edit-in-place while highlighting

  • Enabled — Calibre RVE highlights to the open window in Cadence Virtuoso. Calibre RVE automatically ascends or descends the hierarchy if necessary in order to highlight in the correct cell.

  • Disabled — Calibre RVE opens a new window for the highlight action if the cell and view for the highlight is not open as the top cell.

Allow custom colors

Enable this setting to match Calibre RVE colors to Cadence Virtuoso highlight colors. See “Matching Calibre RVE Colors to Cadence Virtuoso Highlight Colors”.

Restore layer visibility upon exit from RVE

As described; see “Restoring Layer Visibility Settings for Cadence Virtuoso” for complete details.

Libraries RVE should look in for:

Layout cells: ______

Schematic cells: ____

As described. Calibre RVE searches libraries in the order specified. Newly opened libraries are placed at the beginning of the list.

To have newly opened libraries placed at the end of this list, set the environment variable MGC_CALIBRE_SEARCH_LIB_ORDER to APPEND. The default for this environment variable is PREPEND.

Also see these SKILL functions to set the library list:

mgc_rve_set_layout_libpath

mgc_rve_set_schematic_libpath

mgc_rve_set_current_library

You can also define a map between the cells listed in the Calibre results database and a specific library/cell/view combination in the Cadence design tool. See “Map Calibre RVE Cells to Different Cadence Library for Highlighting”. The cell map table is automatically populated if you open Calibre RVE from Calibre Interactive, and the cell map table has higher precedence than the library list specified in the Setup RVE dialog box.

Also see “Set Cadence Library to Highlight In for Calibre RVE”.

RVE Highlight Layers

Set the layers used for highlighting; see “Specifying the Highlight Layers in Cadence Virtuoso”.
