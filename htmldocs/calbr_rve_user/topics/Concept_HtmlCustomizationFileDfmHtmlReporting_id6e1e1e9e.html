HTML Customization File for DFM HTML Reporting

An HTML customization file allows you to insert custom code in the header and footer of the summary report, and the header of each section.

Specify the path to this file with the CustomizationFilePath key name.

The following table contains the IDs that you can assign to div tags in the file.

Table 1. Customization File div Tags

div ID

Description

Report.CustomReportHeader

Everything contained in this division is included in the summary report between the Calibre header that appears at the top and the main report index table.

Report.CustomReportFooter

Everything contained in this division is included in the summary report as a footer after the final section report.

SectionName.CustomSectionHeader

Everything contained in this division is included in the summary report before the report index table for the section specified with SectionName.

The following code shows how to implement these tags in the customization file:

<div id="Report.CustomReportHeader">
	<!-- custom header code goes here -->
</div>

<div id="Report.CustomReportFooter">
	<!-- custom footer code goes here -->
</div>

<div id="WindowSummary.CustomSectionHeader">
	<!-- custom header code for the WindowSummary section goes here -->
</div>
