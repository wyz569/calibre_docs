Finder Tab

Access with: View > Finder or from the Navigator in Calibre RVE for LVS, PERC, and PEX.

Requires: Rule file. All variables referenced in the rule file must resolve. Some search types require the PHDB or XDB; see Mask SVDB Directory in the SVRF Manual.

The controls in the Finder tab allow you to search for design elements in the source or layout side of the design. You can specify options for narrowing the search to specific design elements, layers, or device types.

Item

Comment

Search in Layout or Source?

As described.

What type of search?

Select from:

  • Particular Objects

  • Objects at Location

  • Instances of Cell

  • Unconnected Net Instances

  • Find Path

  • Isolate Path on Layout Net

If Source is selected for “Search in Layout or Source?”, then only the Particular Objects and Find Path searches are available.

Search in which cell?

Choose a cell name from the dropdown list. This field is used for all types of search.

Finder Filters

You can apply filters to the layers, devices, and area (window) that are searched. Check the items you want included in the search. The type of filter available depends on the search type. Some filters apply to portions of the search results.

See “Search and Selection Filters in the Finder Tab”.

Search Results area

Each search result is listed in a tree view that can be expanded.

Right-click a result for a context-sensitive highlighting menu.

To highlight an object, right-click it and select the Highlight command. You can also press H to highlight the currently selected object, N to select and highlight the next object, and P to select and highlight the previous object.

Some search and selection filters apply when expanding certain nodes in a search result; see “Search and Selection Filters in the Finder Tab”.
