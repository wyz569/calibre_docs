Setting the Calibre RVE Socket Port for a Layout Viewer

You can set the socket port manually through the Calibre RVE interface or by using an environment variable. By default, Calibre RVE uses socket port 9189 to communicate with the layout viewer. The tool automatically searches for an available socket if it cannot obtain the default socket. It initializes the first available socket between the numbers 5000 and 9999 and reports the socket number through a dialog box.

Some layout viewers, such as Calibre DESIGNrev, also have a GUI interface to set the socket port — see the instructions for the specific viewer. Full instructions for Calibre RVE are also given in the section “Manual Connection to a Viewer”.

Prerequisites

  • Calibre RVE must be open if you are setting the socket number using the GUI.

Procedure

Set the layout socket port with one of the following methods:

To set the layout socket port with …

Do this …

Calibre RVE

  1. Choose Setup > Design Tools in Calibre RVE.

  2. Enter the “Host Name” information in the Layout Design Tool section; “localhost” is the default.

  3. Enter the port number in the “Socket Number” field.

  4. Click the connect button () to start the connection. The “Tool Information” field is filled in when the connection is made. The status is reported in the Connection Icon in Calibre RVE.

  5. Click Apply (in the left panel of the Options tab).

Environment variable

Set the MGC_CALIBRE_LAYOUT_SERVER environment variable as shown here for the C-Shell:

setenv MGC_CALIBRE_LAYOUT_SERVER [<host>:]<port>

The <host> parameter is set to localhost by default, but can be set to a different host. The <port> parameter must be a numerical value.

To disable socket communication to the layout viewer, set MGC_CALIBRE_LAYOUT_SERVER to “ ” (blank). The layout server then does not attempt to open the communication socket.
