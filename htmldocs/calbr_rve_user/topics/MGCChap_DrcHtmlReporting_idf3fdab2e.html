DRC HTML Reporting

This chapter describes how to create HTML formatted reports of Calibre DRC results. You can create HTML pages corresponding to views in Calibre RVE for DRC for any ASCII format DRC results database (RDB). You can specify views of result tables, tree views, result highlights, histograms, and colormaps. Controls are included for grouping, filtering, and sorting the results, and for customizing the report format.

More examples — See the chapter “Calibre RVE HTML Reporting” in the Calibre Solutions for Physical Verification manual for examples using Calibre Auto-Waivers, Calibre PERC LDL CD, and Calibre® LFD™ results.

DFM RVE batch reporting — See “DFM HTML Reporting”. DFM RVE batch reporting uses a similar format, but is very specific to the YieldAnalyzer report types and capabilities.
