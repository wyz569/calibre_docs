Saving DRC Results Data to CSV Format from Calibre RVE for DRC

Calibre RVE for DRC can save results data to a CSV (comma-separated value) formatted file. The CSV file can then be imported into most spreadsheet tools for later analysis.

You can filter the saved data by applying a DRC Result Filter; only data that meets the applied filter is saved. In addition, you have the option of saving displayed results, selected results, or property statistics. You can also specify to save only the data columns that are displayed. These options are discussed in the procedure below, which uses a Density RDB as an example.

Prerequisites

  • A DRC results database is open in Calibre RVE for DRC.

Procedure

  1. (Optional) If desired, apply a DRC Result Filter, as described in the section “Filtering DRC Results”.

    You can use this step to filter the results so that only the subset you want to analyze is saved. Only results that satisfy the applied DRC Result Filter are saved.

  2. (Optional) If you want to save only the displayed data columns, choose View > Result Options > Details, then click the button to select columns to show and hide.

    This is recommended for large results databases to reduce the size of the CSV file. It can also make it easier to analyze the CSV file.

  3. (Optional) If you want to manually select specific results to save, use Shift- and Ctrl‑click to select the results to save.
  4. Select File > Save Results As.

     

    Enter the filename for the saved data, including a file extension, if desired. Select a File Type of CSV.


  5. Select the results you want to save in the “Save” dropdown list:

    • Displayed Results — Save all results shown in the Calibre RVE display.



      If empty checks are shown in the tree view, they are saved with a zero in the first column of the CSV file.

    • Selected Results — Save only results selected with Shift- and Ctrl-click (Step 3).



      Empty checks are saved if no results are selected in the result view and at least one empty check is selected in the tree view; empty checks are saved with a zero in the first column of the CSV file.

    • Property Statistics — Save the data displayed when View > Tree Options > Group By > Statistics is selected, as shown below. This option is only available when properties are present in the results database.



  6. (Optional) Check “Visible Columns Only” to save only the data columns that are displayed.
  7. (Optional) Uncheck “Open after Save” if you do not want to view the CSV file within Calibre RVE.
  8. Click Save.
