Symbol Library File Reference

Symbol Library (*.sym) files define custom shapes for symbols displayed by the Internal Schematic Viewer or SPICE File Viewer.

See “Symbol Library Files for Custom Symbol Shapes” for directions on how to load the Symbol Library files and how they are used by Calibre RVE.

An example file, netlist, Calibre RVE configuration file (.rvedb), and shell script are located at $CALIBRE_HOME/shared/examples/rve/symlib in the software tree.
