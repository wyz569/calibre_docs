Troubleshooting Calibre RVE Highlighting to Cadence Virtuoso

Some common problems when highlighting to Cadence Virtuoso are not finding the correct library and cell and opening or not opening a new window.

The following problems are discussed:

  • Problem: Calibre RVE does not highlight to the correct library and cell

  • Problem: A new window is opened when highlighting to a new cell

Problem: Calibre RVE does not highlight to the correct library and cell

Cause: This problem can happen if a cell exists in multiple libraries or you have multiple libraries open in Cadence Virtuoso.

Solution: See the following topics:

“Set Cadence Library to Highlight In for Calibre RVE”

“Map Calibre RVE Cells to Different Cadence Library for Highlighting”

Problem: A new window is opened when highlighting to a new cell

Cause: “Edit-in-place while highlighting” is disabled in the Setup RVE dialog box. When “Edit-in-place while highlighting” is disabled a new window is opened if the cell and view for the highlight is not already open as the top cell. The Cadence Virtuoso setting for edit-in-place or descend edit has no effect on the highlighting behavior in Calibre RVE.

Solution: In Cadence Virtuoso, choose Calibre > Setup > RVE and enable “Edit-in-place while highlighting.”

Also see the section “Highlighting with Edit-In-Place and Descend Edit in Cadence Virtuoso”.
