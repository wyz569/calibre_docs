Find Objects at Location

Access with: View > Finder, then choose “Objects at Location” for “What type of search?”

Searches in: Layout

Requires: PHDB. See Mask SVDB Directory in the SVRF Manual.

Use “Objects at Location” search on the Finder tab to find objects at a specific location in the layout, using optional search filters.



Objects

Table 1. Find Objects at Location Search Criteria

Item

Description

Find Objects of Type

Select the design elements to search for.

Objects at Location

Enter a space-separated coordinate set, use the dropdown list of previous search items, or click the layout icon () to select a location in a connected layout viewer.

Find Instances at

Select “Top level of hierarchy” (default) or “All level of hierarchy.”

Layout Icon          

Select location in layout viewer.

Show Highlight Menu                     

Click to display a menu with options for highlighting and displaying information about the object.

Search Filters

You can apply filters to the layers and devices that are searched. Check the items you want included in the search. Some filters apply when expanding nodes in the search results. See “Search and Selection Filters in the Finder Tab”.

Usage Notes

Tip

You can also find objects at a location in the layout using the Info Pane; click the layout button next to the search field to select a location in the layout.
