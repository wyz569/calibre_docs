DRC RVE Check Text Override File (CTO File)

The DRC RVE check text override file (CTO file) allows you to specify DRC RVE rule check comments for a DRC results database and load the settings into a Calibre RVE for DRC session. DRC RVE rule check comments allow you to specify highlight colors, display layers, document links, and other settings related to highlighting results. The CTO file also defines categories for filtering by category. When you use a CTO file you can use DRC rule check comments without editing the rule file or running Calibre again.

You can create the CTO file using any text editor. Additionally, you can use Calibre Interactive or Calibre RVE to automatically create a CTO file with RVE Show Layers statements to control layer visibility when highlighting results. You can also specify to generate Groups and Layers categories when creating a CTO file.
