Extraction Results Tab

The Extraction Results tab displays the results of netlist extraction. It provides a tree view and a detailed view of the results listed in the Extraction Report.
Figure 1. Extraction Results Tab

The tree view shows the extraction warnings in a tree format that can be expanded or collapsed. Extraction results can be sorted by cell or warning type using the menu selection View > Tree Options, as given in Table 1.

Table 1. View > Tree Options Menu for Extraction Results Tab

Menu Item

Description

View By Layout Cell

Sort results first by cell name.

View by Warning Type

Sort results first by warning type.

Usage Notes

A left-click in a blue text element in the detailed view highlights the design area in the layout viewer and opens the schematic view.

A right-click in a blue element displays a context sensitive menu with options for highlighting and displaying more information.

The Extraction Report created by Calibre uses net ID numbers rather than net names. Calibre RVE converts the net ID to a net name for highlighting purposes. In some cases the net name in the geometric layout may be different than the net name in the extracted layout netlist. This may be apparent when you view the right-click menu for a net in the Extraction Results tab. For example, a net named X0/1 may be listed as X0/n1 in the right-click menu, where X0/1 is the net name in the layout and X0/n1 is the net name in the extracted layout netlist.

The text extraction report can also be viewed with the View > Reports > Extraction Report menu selection.
