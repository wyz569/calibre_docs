Getting Started with Calibre RVE

The Calibre Results Viewing Environment (RVE) is a graphical user interface for viewing and analyzing Calibre results databases. The basic invocation methods and some configuration options and setup procedures are common to all Calibre RVE applications.

For a general introduction to Calibre RVE, see “Calibre RVE Overview”.
