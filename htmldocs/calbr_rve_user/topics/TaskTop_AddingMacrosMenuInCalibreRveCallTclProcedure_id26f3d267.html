Adding the Macros Menu in Calibre RVE to Call a Tcl Procedure

The Macros menu is created when a Calibre RVE macro configuration file is present. The Macros menu is a special menu that allows you to run Tcl/Tk procedures directly from Calibre RVE; in Calibre RVE for DFM you can also run YieldServer commands.

Procedure

  1. Create the macro configuration file and save it to an accepted filename and location, as described in “Calibre RVE Macro Configuration File”.

    The file should contain Tcl procedure definitions and at least one rve::addmenu command. See the “Examples” section.

  2. Start Calibre RVE.

Examples

The following macro configuration file creates a menu item named Add Annotation under the Macros menu, as shown in the “Results” section. The oncmdclick procedure is called when Macros > Add Annotation is selected.

rve::addmenu command -label "Add Annotation" -command oncmdclick

proc AddAnnotation {} {
	dfm::create_rev    
	set layerName M1_data_detail
	dfm::add_annotation -layer $layerName -annotation "NEW_LAYER_TITLE"
	-value $layerName]\n"	dfm::save_rev
}
proc oncmdclick {} {
    AddAnnotation
}
