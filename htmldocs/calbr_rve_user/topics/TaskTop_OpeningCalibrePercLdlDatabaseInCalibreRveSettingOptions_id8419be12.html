Opening the Calibre PERC LDL Database in Calibre RVE and Setting Options

You can open the DFM database created by a Calibre PERC LDL run in Calibre RVE for PERC. The P2P or CD results databases (RDBs) are displayed in a DRC-format report tab. The following procedure describes how to open the DFM database and set options for highlighting results and nets.

Prerequisites

  • A successful Calibre PERC LDL run. See “Current Density Checks” and “Point-to-Point Resistance Checks” in the Calibre PERC User’s Manual.

    The PINLOC keyword must be included in the DFM Database statement for pin highlighting to work.

  • A Calibre RVE license.

  • A layout viewer with an interface to Calibre RVE. See “Integration to Design Tools”.

Procedure

  1. Open the layout in a supported layout viewer
  2. Depending on your layout viewer, choose Calibre > Start RVE or Verification > Start RVE.
  3. Select “PERC” for the “Database Type” and open the DFM database from the Calibre PERC CD or P2P run.


    The P2P or CD results database is opened and displayed using a DRC Results Tab automatically unless “Automatically open PERC P2P/CD/LDL database on startup” is disabled (Options tab, Databases category, “Database Loading” section). If the P2P or CD tab does not open, choose View > Results to open the results tab.

    Although Calibre RVE for DRC is used to display the results, many of the features of Calibre RVE for PERC, such as net highlighting, connectivity information, and right-click menus, are also available, as shown in Figure 1.

    The results for a Calibre PERC LDL P2P run are shown in the following figure; the view for Calibre PERC LDL CD results is slightly different. The Navigator pane is not shown in the figure. If standard Calibre PERC results are included in the DFM database, they are opened automatically.

    Figure 1. Calibre PERC LDL Results in Calibre RVE

  4. (Optional) Click the Group By button () to select the grouping for the Tree View.
     

    Use the Custom grouping option to group by properties. When specifying a Custom grouping, click the top entry of a menu selection to end the grouping hierarchy; see “Grouping Results in Calibre RVE for PERC” for complete instructions.

     

  5. (Optional) Disable View > Tree Options > Show Empty Checks if you only want to view checks that produced results.
  6. To set highlight colors, see “Setting Highlight Color in Calibre RVE for LVS, PERC, and PEX”.
  7. Click the Options button () to open the Options pane and set the following options:

    • To add property data to the result highlights:

      1. Choose the Highlighting category.

      2. Expand the “DRC/DFM Highlighting” area.

      3. Enable “Show Properties.”

    • To automatically show connectivity objects in the Internal Schematic Viewer:

      1. Choose the Schematic Viewer category.

      2. Enable “Show netlist schematics when highlighting connectivity objects,” “Show Schematic Pane,” “Show Hierarchy Pane,” and “Show Text Pane.”

  8. Proceed to “Examining Calibre PERC LDL Results in Calibre RVE”.
