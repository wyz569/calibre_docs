Linking to Custom Reports with Calibre RVE

You can specify for Calibre RVE to display a link to a custom report. If your Calibre run creates a custom text or HTML-formatted report, you cause Calibre RVE to display a link to the report, allowing you to easily open the report from the Calibre RVE environment. This feature is available in Calibre RVE for DRC, LVS, PERC, and PEX. The implementation depends on which application you are running.
