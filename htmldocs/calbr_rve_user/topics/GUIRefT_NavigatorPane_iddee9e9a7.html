Navigator Pane

Access with: View > Navigator in Calibre RVE for LVS, PERC, and PEX.

The Navigator Pane provides easy access to the information and controls in your Calibre RVE for LVS, PERC, or PEX session.

Description

The Navigator is shown in “Session Window in Calibre RVE for LVS” and its contents are described in the following table. If desired, you can dismiss the Navigator pane to provide more space in the Calibre RVE display and use the controls in the View menu instead.

Objects

Table 1. Navigator Pane Contents

Element

Description

Results

This section lists the results available from your SVDB. A Calibre nmLVS run can contain the following results:

  • Extraction Results

  • Layout Shorts

  • Comparison Results

  • PERC Results (PERC Run only)

ERC

Only available if ERC results are present.

  • ERC Results — Opens ERC Results in a new window, using the DRC Results tab.

  • ERC Summary — Opens the text ERC Summary report in a new tab.

Reports

Text view of reports and files in the SVDB. A typical LVS run will include:

  • Rules File

  • Extraction Report

  • LVS Report

  • LVS Summary Report

Also see “Searching for Text Strings”.

Rules

Opens the rules file used for the run.

View

Info

Opens the Info Pane with a tree view of the selected design element. The Info Pane provides a quick search of design elements.

Finder

Opens or switches to the Finder Tab, which provides advanced design search and navigation for design elements in both layout and source.

Schematics

Opens the Internal Schematic Viewer in a new window. The schematic view is also opened when you click a blue design element in the detailed report view or highlight an item.

Tip: To turn off automatic opening of the Internal Schematic Viewer, click the Setup Options button () to open the Options tab, choose the Schematic Viewer category, and uncheck the selection “Show netlist schematics when highlighting connectivity objects.”

Setup —    Options

Open the Options tab. See “Options Tab in Calibre RVE” and “Setup LVS/PERC/PEX Browser Options Pane”.
