Key Name Reference

This section provides of summary of key names.

Table 1 contains a summary of the key names you can use in the configuration file—see “Configuration File in DFM HTML Reporting” for additional information.

Note:

It is not necessary to enclose a value in quotes unless it contains a space.

Table 1. DFM HTML Reporting Key Name Summary

Key Name

Description

AnalyzeCellUsageFile

Specifies the path to the cell usage file.

AnalyzeDDKFile

Specifies the path to the DDK file.

AnalyzeMemoryConfigurationFile

Specifies the path to the memory configuration file.

AnalyzeMemoryIntermediateResultsByCell

Enables intermediate results calculation.

AnalyzeNormalizationArea

Specifies the normalization area in um^2.

AnalyzeShrinkFactor

Specifies the shrink factor.

AnalyzeUmcPfxFile

Specifies the path to the UMC PFX file.

AnalyzeUmcPfxFilePassword

Specifies the UMC PFX file password.

AnalyzeWhatIfConfigurationFile

Specifies a path to a file for a what-if analysis.

AnalyzeYieldModels

Specifies one or more yield models.

BinsToInclude

Specifies a list of bin names to include in the report.

CellsToInclude

Specifies a list of cell names to include in the report.

ChipHighlightWorstBins

Controls highlighting of the worst bins.

ChipShowGroup

Controls display of the Group column.

ChipShowPriority

Controls display of the Priority column.

ChipShowType

Controls display of the Type column.

ColorSet

Specifies a pre-defined color scheme.

CssFilePath

Specifies the path to a Cascading Style Sheet (CSS) file used to format the summary report (index.htm).

CustomColors

Specifies a list of colors for use with a custom color scheme.

CustomizationFilePath

Specifies the path to a valid HTML file that contains div tags with predefined IDs.

DefaultColors

Specifies a list of colors to be displayed by default.

DrvOptionsFilePath

Optional key name that specifies the directory for a Calibre DESIGNrev preferences file.

FilterBin

Specifies a bin name on which to apply the filter.

FilterByFlatMetrics

Controls sorting by flat values.

FilterMetric

Specifies the name of a metric on which to apply the filter.

FilterOperation

Specifies the filter operator.

FilterValue

Specifies the filter threshold value.

GroupBy

Specifies the order in which to group columns in the text report, or bars in the bar chart.

GroupsToInclude

Specifies all groups and results in one table per group.

Highlight

Specifies whether the ByErrorResults report includes an image of the result highlighted in the corresponding layout file.

IncludeFlatMetrics

Specifies if flat metrics are calculated and included in the report.

IncludeInSummaryReport

Controls whether or not the section appears in the summary report.

IncludeWindowCoordinates

Specifies if window coordinates are included in the report.

InvertColors

Reverses the order of the displayed color scheme.

LinkReports

Controls display of section reports in the summary report.

LogarithmicYAxis

Controls display of the Y axis as either a linear or logarithmic scale.

MetricsToInclude

Specifies a list of metric names to include in the report.

MetricsToShareRange

Specifies a list of metrics that share a custom range for histograms and colormaps.

NetsToInclude

Specifies a list of net names to include in the report.

NumericFormat

Defines the format for floating-point values. Applies to all plots.

NumericPrecision

Defines the precision of floating-point values. Applies to all plots.

OutputFormat

Specifies a list of formats to which the output is written. Not all formats are valid for all report types.

PartitionOutputBy

Specifies creating separate reports for each rule that has a violation.

PrioritiesToInclude

Specifies a list of priorities to include in the report.

Property

Specifies the property name, which should be used to assign specific color to each error shape in generated layout color map.

PropCustomRangeDistribution

Specifies endpoints of histogram distribution ranges when PropRangeDistributionType is set to “custom”

PropLinearDivisionCount

Specifies the number of histogram range divisions when PropRangeDistributionType is set to “linear”.

PropLinearRange

Specifies the endpoints of histogram distribution ranges when PropRangeDistributionType is set to “linear”.

PropRangeDistributionType

Defines how distribution ranges are specified when generating histograms. Choose either custom or linear.

ReportType

Specifies the type of report the section describes.

RotateXLabels

Controls the vertical orientation of the histogram distribution values in the report.

RowCount

Specifies the number of rows to include for each leaf in the GroupBy hierarchy.

Rows

Specifies the number of rows to return for the section.

RulesToInclude

Specifies a list of rules to include in the report.

SetColormapMinSize

Sets the maximum pixel dimension for colormaps.

SetColormapMaxSize

Sets the minimum pixel dimension for colormaps.

SetHistogramMinSize

Sets the maximum pixel dimension for histograms.

SetHistogramMaxSize

Sets the minimum pixel dimension for histograms.

ShowColormapBorders

Controls the display of borders for colormaps.

ShowColumns

Controls the display of report columns.

Sort

Specifies the order in which to sort the results for the section.

SortBy

Specifies the sort order in which to sort the results.

UseAdjustedRange

Controls the display of ranges of different layer properties for creation of histograms and colormaps.

WaivedByFilter

Filters waived results by a specified user.

WaivedCommentFilter

Filters waived results by comment.

WaivedDateFilter

Filters results that match a specified date constraint.

WaivedStateFilter

Filters waived results by state.

WaiveStatesToInclude

Specifies which errors to include in the report.

WindowsToInclude

Specifies a list of windows to include in the report.

WorldView

Specifies whether to include an image of the Calibre DESIGNrev World View along with the layout image.

WorstBinsQuarter

Specifies the quarter for which the worst bins are highlighted.

XProfileInclude

Specifies the exact Y positions of the X direction profile to include in report.

YProfileInclude

Specifies the exact X positions of the Y direction profile to include in report.

YSScript

Controls post-processing execution by Calibre YieldServer.

ZoomLayoutFactor

Specifies zoom factors for included layout images.
