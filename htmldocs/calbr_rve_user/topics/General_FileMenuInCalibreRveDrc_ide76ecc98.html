File Menu in Calibre RVE for DRC

The File menu in Calibre RVE for DRC has selections for saving the database or a subset of it and for viewing side or auxiliary RDBs.

The contents of the File menu in Calibre RVE for DRC are shown in Table 1. Menu selections common to all Calibre RVE applications are described in the section “File Menu in Calibre RVE”.

Note:

In Calibre RVE for DRC, the open database selections and the corresponding toolbar button are disabled while Calibre RVE is processing results from a Calibre Interactive run with “Start RVE as soon as results are available” enabled on the Setup > Preferences tab in Calibre Interactive.

Table 1. File Menu Contents in Calibre RVE for DRC

Menu Item

Description

Save Results As

Opens a Save Results dialog box for saving the results database.

File Name

The saved filename. Add a file extension if desired; a file extension is not added automatically.

File Type

  • RDB — Save in ASCII DRC results format.

  • CSV — Save in comma-separated value format. All fields, including empty columns, are exported in CSV format.

Save

  • Displayed Results — Save all results shown in the Calibre RVE display.

  • Selected Results — Save only results selected using Shift- and Ctrl‑click.

  • Property Statistics — Save the data shown when View > Tree Options > Group By is set to Statistics; this saves the min, max, range, mean, and standard deviation for each property. Only available when File Type is CSV.

See “Saving DRC Results Data to CSV Format from Calibre RVE for DRC” and “Saving a Subset of DRC Results as an RDB” for an example.

Only results that obey the applied DRC Result Filter are saved.

Empty checks are saved in CSV format with a zero in the first column. They are saved in either of the following cases:

  • “Displayed Results” is selected in the Save dialog and empty checks are shown in the tree view.

  • “Selected Results” is selected in the Save dialog, no results are selected in the result view, and at least one empty check is selected in the tree view.

Side RDBs

Opens a cascading menu with the available side (auxiliary) RDBs. See “Viewing Auxiliary Databases” for more information.
