RotateHighlightsInAllCells

Key Value for DRC HTML Report Configuration File, highlighting category

Usage: Optional. Valid in Common section only.

Inheritance: Not applicable.

Specifies whether shapes are rotated in the top cell only (default) or all cells when RotateHighlightsBy is specified.

Allowed Values

0

Rotate shapes only in top cell. (default)

1

Rotate shapes in all cells.
