ApplyTo

Key Value for DRC HTML Report Configuration File, report specification category

Usage: Optional. Valid in all report types, and in Common section.

Inheritance: Section Reference allowed: Indirectly for Derived Reports or Common

Specifies a list of RDB types that the report section applies to.

If the ApplyTo key is not specified, then an RDB type of “General” is used. If ApplyTo is specified in a report section, the section applies only to the listed RDB types; “General” must be listed explicitly for the section to apply to the “General” RDB type.

Allowed values

General

(default) The database type of “General” is used if ApplyTo is not specified in the report section. Note: The “General” database type is assigned when a database is specified on the command line.

All

Apply the report section to all RDB types, including “General.”

rdbtype

One or more RDB types from the following list: Density, General, NetAreaRatio, DFM, Waiver, PATHCHK, LayoutInputException, CheckMap, Other.

user_type

One or more user-defined RDB types from the DRC HTML Report Database Map File.

Multiple rdbtype and user_type values may be specified in a comma-separated list. See the rdbType parameter in “DRC HTML Report Database Map File” for a complete discussion of database types in the context of DRC HTML reporting.

Examples

ApplyTo = General, Density
ApplyTo = NetAreaRatio
ApplyTo = All
