RVE Highlight Text Size

A DRC RVE rule check comment which specifies the text size for text objects added during highlight operations.

Usage

          RVE Highlight Text Size: size

Parameters

size

A required argument which specifies the text size for text objects; size is a positive real number. A space is required between the colon (:) and size.

Description

This rule check comment specifies the text size for text objects added during highlight operations. Calibre RVE for DRC adds text objects to highlight shapes when property data is included in the highlight operation; see RVE Show Properties.

See “Methods to Add DRC RVE Rule Check Comments” for instructions on specifying the rule check comment. You can add the rule check comment to the rule check in the rule file or specify it in a Check Text Override (CTO file.

Examples

The following rule check shows an example using RVE Highlight Text Size.

// rule file...
density_m1_check{
	@ RVE Show Layers: metal1 
	@ RVE Show Properties: DV "DA metal1"
	@ RVE Highlight Text Size: 1.5
	DENSITY metal1 < 0.25 WINDOW 50 RDB lab2_m1_density.rdb
}

The following example shows a DRC RVE Check Text Override File (CTO File) Format with the RVE Highlight Text Size rule check comment:

# DRC RVE check text override file for XYZ
#
density_m1_check 
RVE Show Layers: metal1 
RVE Show Properties: DV "DA metal1"
RVE Highlight Text Size: 1.5
# ...
