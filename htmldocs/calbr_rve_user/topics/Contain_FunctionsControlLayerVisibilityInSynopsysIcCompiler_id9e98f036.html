Functions to Control Layer Visibility in Synopsys IC Compiler and IC Compiler II

You can control layer visibility with the Restore Default Layers Toolbar or use functions available in Synopsys IC Compiler and IC Compiler II.

If you use the RVE Show Layers check text comment to control layer visibility during highlighting for DRC results, you can use the Restore Default Layers Toolbar toggle between the default set of visible layers and the layers specified by RVE Show Layers.

The functions that enable this feature are available in the Synopsys IC Compiler and IC Compiler II command interface.

Command

Description
Calibre::mgc_rve_set_default_visible_layersCommand in Synopsys IC Compiler and IC Compiler II that saves the current set of visible layers as the default set of visible layers.
Calibre::mgc_rve_toggle_visible_layer_setCommand in Synopsys IC Compiler and IC Compiler II that toggles the design layer visibility.
