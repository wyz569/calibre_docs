DFM HTML Reporting

The Calibre DFM HTML report tool is a batch interface to the DFM database version of the DFM Report Card.

DFM HTML reporting allows you to generate tabular score data in both HTML and CSV format, GIF images of bar charts, and color maps and histograms through a highly-customizable configuration file.

Use DFM HTML reporting to do the following:

  • Save sets of data from the DFM Report Card.

  • Use other applications or scripts to further analyze data from the DFM Report Card.
