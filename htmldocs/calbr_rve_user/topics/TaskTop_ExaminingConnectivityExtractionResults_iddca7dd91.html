Examining Connectivity Extraction Results

The results from the connectivity extraction component of the Calibre nmLVS run are shown in the Extraction Results tab. Extraction errors should be corrected first when debugging LVS errors, since they can lead to other errors.

Prerequisites

  • An SVDB from a Calibre nmLVS run which included netlist extraction.

  • The layout database for your LVS run.

  • A layout viewer with an interface to Calibre RVE. See “Connection to a Design Tool”.

Procedure

  1. Open your layout in Calibre DESIGNrev or in another integrated layout viewer.
  2. Start Calibre RVE from your layout viewer, as described in the procedure “Invoking Calibre RVE From a Layout or Schematic Viewer” and open your SVDB.
  3. Click the Extraction Results tab and view the results. A typical view is shown in Figure 1.
    Figure 1. Extraction Results Tab

    The tree view shows the extraction warnings in a hierarchical view that can be expanded or collapsed.

  4. (Optional) Choose View > Tree Options to sort the results by cell or warning type.
  5. Select a result in the tree view and view the details in the bottom pane of Calibre RVE.

    The detailed view (bottom pane) displays only the first 10 results by default. If there are more than 10 results, a message is displayed with directions to view the next group of results. You can change the number of results displayed with the setting “Show Detailed Results” (Options tab, LVS/PERC/PEX category)

    A left-click in a blue text element in the detailed view highlights the design area in the layout viewer and opens the schematic view. A right-click in a blue element displays a context sensitive menu with options for highlighting and displaying more information. The text extraction report can also be viewed with the View > Reports > Extraction Report menu selection.
