Tools Menu in Calibre RVE for DRC

The Tools menu in Calibre RVE for DRC has selections related to waivers and Check Text Override (CTO) files.
Table 1. Tools Menu Contents in Calibre RVE for DRC

Menu Item

Description

Import Waivers

Imports a Calibre RVE waived results file and applies the waivers to the currently selected results tab. See “Waived Results in Calibre RVE for DRC”.

Export Waivers

Opens the Export Waived Results dialog box, which is used to export waiver shapes for use with Calibre Auto-Waivers. See “Export Waived Results Dialog Box” for details.

Create Waiver Report

Saves a text report of the results that are waived using Calibre RVE. The report is opened in a new tab.

Create CTO File

Opens the Create CTO File dialog box. The generated CTO file includes RVE Show Layer statements based on the rule checks in the rule file. See “Creating a Check Text Override (CTO) File Using Calibre RVE”

Import CTO File

Loads a DRC RVE Check Text Override File (CTO File) Format. The CTO file specifies rule check comments which provide additional control over the Calibre RVE highlight actions; see “DRC Rule Check Comments for Calibre RVE”.

If a CTO file is already loaded, all CTO file data from the current Calibre RVE session is removed, except for checks that were locked with RVE Waive.

Reset CTO File

Removes all CTO file data from the current Calibre RVE session, except for checks that were locked with RVE Waive. A new CTO file can then be loaded without merging with a previous CTO file.

Export to Layout

Displays the Export to Layout Control Bar, which allows you to export results on a cell-by-cell basis to the layout editor.

Create HTML Report

Creates a DRC HTML report from the active results tab. See “Creating a DRC HTML Report from Calibre RVE for DRC”.
