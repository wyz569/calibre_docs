Session Window in Calibre RVE for LVS

The session window is arranged to provide convenient viewing of the Calibre nmLVS results and reports, highlighting of results in a connected layout viewer, and schematic browsing for comparison between layout and schematic.

Figure 1 shows a typical view on opening an LVS session in Calibre RVE. The session opens with tabs for each result database generated by your Calibre nmLVS run; however, databases with clean results are not opened automatically. The View menu and the tabs are used to customize the display.

Figure 1. Opening Session Window in Calibre RVE for LVS

Here are the major features of Calibre RVE for LVS:

Toolbars —The following toolbars are available to speed up your LVS debugging session:

  • Standard Toolbar in Calibre RVE — Available for all views.

  • Short Isolation Toolbar — Available when the Layout Shorts Tab is active.

  • LVS Toolbar — Sets view options; the contents of the toolbar depend on which report tab is selected.

  • Search Control — Finds text in the Extraction, Comparison, and PERC Results tabs and text display tabs. For additional information about text display tabs, see “Viewing Text Files in Calibre RVE”.

Navigator — Provides easy navigation to the reports, files, and tools available in the session. However, you can close the Navigator and use the selections in the View menu if you want more room in the session window.

Report Tabs — The different results in your SVDB are arranged in windows called report tabs. The arrangement of the report tabs can be customized for your work flow. The most common report tabs are:

  • Extraction Results — Displays results from the netlist extraction phase of the Calibre nmLVS run.

    See “Examining Connectivity Extraction Results” and “Extraction Results Tab”.

  • Layout Shorts — Displays the results of short isolation, if you have enabled it in your rule file. The tab has several features to simplify viewing a short and pinpointing the short location. You can view a summary of all shorts and a detailed listing of each polygon in the short; filter short polygons by cell, layer, and net; highlight shorts by cell, layer, and net; and assign (within RVE) a net label to a polygon.

    See “Interactive Short Isolation” and “Layout Shorts Tab”.

  • Comparison Results — Displays and sorts the LVS comparison errors in your run. You can cross-probe results in your layout editor and a schematic viewer. Schematic viewing can be done in the Internal Schematic Viewer or in a connected external schematic editor. Fix Suggestions (if enabled) provide easy to understand suggestions for fixing certain types of errors.

    See “Examining LVS Comparison Results”, “Using Design Fix Suggestions for LVS Comparison Results”, and “Comparison Results Tab”.

You may see other reports, such as ERC results, depending on your Calibre nmLVS run. See “Using Calibre RVE for PEX” for examining parasitic results and “Using Calibre RVE for PERC” for examining PERC results.

LVS sessions use the icon to designate the top cell of the design.

Schematic Browsing — Schematic viewing of both source and layout is integrated into Calibre RVE using schematic rendering. The Internal Schematic Viewer includes search capability and netlist display.

You can use a connected external schematic editor rather than the internal schematic viewer.

Info Pane — The Info Pane displays detailed information about design elements. Right-click a design element in any detailed view and choose Show … Info to display hierarchical information about the element in the Info Pane. The Info Pane also includes search capability for layout elements.

Finder Tab — The Finder Tab provides advanced search capability in both the layout and source. Searches can be restricted by object type. You can also search for devices in a given layout path, and for devices with unconnected layout nets.
