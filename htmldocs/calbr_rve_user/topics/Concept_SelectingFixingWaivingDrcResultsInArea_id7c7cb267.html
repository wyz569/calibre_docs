Selecting, Fixing, and Waiving DRC Results in an Area

If you are working in a particular region of the layout, you can fix, waive, and select results in that region.

Select in area — You can select results in an area using Ctrl-S, or right-click a check or cell name and choose Select in Area. The results for the selected check or cell that are within the area you specify in your viewer are selected (highlighted in blue in the Results View). See Figure 1 for a view of a result selected in a Results View; result number one is selected in the figure.

Fix and Waive in area — You can use Ctrl-F and Ctrl-W to fix and waive results in an area. You are prompted to select an area in your layout viewer. Alternatively, you can choose Select in Area from the right-click menu to select the results, then right-click the cluster and choose Fix or Waive from the menu.
