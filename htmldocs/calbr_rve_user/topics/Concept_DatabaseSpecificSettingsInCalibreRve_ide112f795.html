Database-Specific Settings in Calibre RVE

In Calibre RVE for DRC and Calibre RVE for PERC, some options, such as the “Group By” hierarchy and result filters, are associated with a specific results database and restored when you open that database again.

Database-specific options are saved to the .rve directory located in your home directory. If you delete the .rve directory, all database-specific settings return to their default value for all databases. If the .rve directory does not have write permission the database-specific options are not saved.

You can disable the saving of database-specific options by setting the environment variable MGC_RVE_USE_USER_DB_OPTIONS to 0. For example:

setenv MGC_RVE_USE_USER_DB_OPTIONS 0

Setting MGC_RVEDB_DIR to define the location of the .rvedb configuration file does not affect the location of the .rve directory for database-specific options.
