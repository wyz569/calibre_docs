Report Tabs in Calibre RVE

Reports and results are organized in report tabs that can be arranged to fit your work flow. In many cases results are presented in a tree view that can be expanded or collapsed. Depending on the application, you can customize the view in the report tabs using the View menu, settings on the Options tab, and right-click menus.

Tab Management — You can close a tab by clicking the x at the right edge of the active tab label. You can change the order of the tabs in a tab group by dragging the tab to the desired location within the tab group.

You can create a new tab group and control the report tabs displayed in each tab group by right-clicking in a tab label for a menu of options. The right-click menu is shown in Figure 1.

Figure 1. Customizing Tab Placement

Window Management — Most windows can be resized by dragging the borders. Some, like the Internal Schematic Viewer window in Calibre RVE for LVS, can also be moved. The arrangement of these windows is controlled by the selections in the Window menu: Cascade, Tile Horizontal, and Tile Vertical.

Some windows, such as the Internal Schematic Viewer and the Info Pane in LVS sessions, and the Histogram window in DRC and DFM sessions, can also be undocked from the main session window using the undock icon shown in Figure 2. Return an undocked window to the session window with the dock icon, or use Window > Dock All.
