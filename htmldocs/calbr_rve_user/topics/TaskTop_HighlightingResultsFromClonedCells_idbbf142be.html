Highlighting Results From Cloned Cells

You can highlight results from cloned cells in the context of the original cell or in the top cell. You can also highlight to a layout generated by Calibre with cloned cells.

Prerequisites

  • A Calibre nmDRC results database produced from a run with the following statements in the rule file:

    • The specification statement DRC Cell Name Yes XFORM.

    • One of the Layout Clone … specification statements.

  • The Calibre nmDRC results database is open in Calibre RVE for DRC and Calibre RVE is connected to a supported layout viewer; see “Starting Calibre RVE”.

Procedure

  1. Select whether to view the results at the top level or in the context of the original cell:

    The “Highlight DRC results in context setting” is on the DRC Toolbar and in the Options tab in the Highlighting category, “DRC/DFM Highlighting” area.

    • Highlight DRC results in context ON — Results are highlighted in the context of the original cell. All results in cloned cells are transformed to the original cell.

      It is possible for different cloned cells to have different results when the result depends on the orientation or placement of the cell. For example, this can happen if a layer derivation or rule check includes an “==” constraint in the Angle operation, making the result orientation-specific. Such results are transformed to the original cell, with a possible change in orientation.

      When viewed at the top level, all results in the original cell are highlighted in all placements of the cell, regardless of which cloned cell the result originated from.

    • Highlight DRC results in context OFF — Results are highlighted in the top cell with the correct orientation.

    If your results are orientation-specific and you want to view them in the top cell, it is best to set “Highlight DRC results in context” to OFF.

  2. (Optional) If you are highlighting to a generated layout that contains cloned cells, rather than the original layout, set the option instructing Calibre RVE to use cloned cell names and highlighting information:

    Click the Options button (), choose the DRC Browser category, and check the “Preserve cloned cells for output layout” option.

  3. Locate a cell and check combination with the #Flipped and #Rotated properties shown in the Results View—this indicates the result is from a cloned cell.
  4. Double-click the result to highlight it. An example is shown in Figure 1, depending on the setting of “Highlight DRC results in context.”
