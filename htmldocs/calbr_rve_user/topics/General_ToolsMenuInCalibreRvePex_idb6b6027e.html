Tools Menu in Calibre RVE for PEX

This section describes the contents of the Tools menu that are specific to Calibre RVE for PEX.
Table 1. Tools Menu Contents in Calibre RVE for PEX

Menu Item

Description

Export CalibreView

Opens a dialog for creating the CalibreView. Select from:

  • Create new view with devices and nets

  • Annotate existing view with net geometry

See “Creating a Calibre View” in the Calibre Interactive User’s Manual for more information.

Mutual Inductance

Opens the tab for creating the victim file needed for Calibre xL inductance extraction. See “Selecting Nets for Mutual Inductance”.
