Map Calibre RVE Cells to Different Cadence Library for Highlighting

You can define a map between the cells listed in the Calibre results database and a specific library/cell/view combination in the Cadence design tool. This feature is useful if cells have been copied to a different library or if the cell exists in multiple libraries. The cell map table enables Calibre RVE to highlight to the correct cell in the Cadence design tool. Choose Highlight > Setup Cell Map to open the Map RVE Cells dialog box.

Note:

If you open Calibre RVE from Calibre Interactive, the Library Name, View Name, and Top Cell values are passed to Calibre RVE and entered into the cell map table. These values must be non-empty and the top cell must not be a wildcard (*). This enables Calibre RVE to automatically highlight to the correct library, cell, and view.

The cell map table has higher precedence than the Setup RVE dialog box opened from Cadence Virtuoso with Calibre > Setup > RVE; see “Setup RVE Dialog Box in Cadence Virtuoso”.

Prerequisites

  • Calibre RVE is connected to Cadence Virtuoso with an OpenAccess database or Cadence Composer. See “Invoking Calibre RVE From a Layout or Schematic Viewer”.

Procedure

  1. Check Highlight > Use Cell Map.

    If Use Cell Map is not checked, highlights are made to the currently open library/cell/view and the entries in the cell mapping table are ignored.

  2. In Calibre RVE, choose Highlight > Setup Cell Map to open the Map RVE Cells dialog box.

    The table includes a row for each cell found in the Calibre results database.

  3. Click in the columns for Library, Design Cell, and View to change the value. Use the Add, Remove, and Unmap buttons as needed.

    Only rows that are added by the user can be removed.

  4. Click OK when finished.

Results

Highlights to a given cell in the Calibre RVE database are made to the library/cell/view specified in the Map RVE Cells dialog box.

The contents of the Map RVE Cells dialog box are saved when you exit Calibre RVE.
