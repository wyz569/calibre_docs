Using Calibre RVE with Synopsys IC Compiler

The following procedure starts Calibre RVE and makes a socket connection to Synopsys IC Compiler.

Prerequisites

  • “Loading the Calibre Interface to Synopsys IC Compiler”

Procedure

  1. Start Calibre RVE as follows from Synopsys IC Compiler or Calibre Interactive:

    • Start Calibre RVE from Synopsys IC Compiler:

      1. Choose Calibre > Start RVE from the ICC menu bar.

      2. Enter your database name, the database type, and other options as needed in the Calibre RVE dialog box (Figure 1). Communication between Calibre RVE and your viewer is established automatically.

        For complete details on the available settings, see “Invoking Calibre RVE From a Layout or Schematic Viewer”.

    • Start Calibre RVE from Calibre Interactive:

      Click Start RVE on the left panel of Calibre Interactive (Figure 1). This opens the database specified on the Outputs pane. If your Calibre Interactive session is connected to a viewer, a connection is made between the viewer and Calibre RVE.

      Also see “Automatically Invoking Calibre RVE from Calibre Interactive”.

      Figure 1. Starting Calibre RVE from Synopsys IC Compiler

  2. (Optional) In Synopsys IC Compiler, choose Calibre > Highlight Colors and set the fill pattern and line width for highlights, if desired.


    You can set the highlight color and other highlighting options using Calibre RVE. For example, you can set the highlight color for a rule check in Calibre RVE for DRC and set the highlight color for each design layer in Calibre RVE for LVS. See “Calibre RVE Highlight Options”.

  3. Analyze and debug your results. See “Using Calibre RVE” for an introduction to Calibre RVE, and the application specific chapters in this manual for detailed information.
