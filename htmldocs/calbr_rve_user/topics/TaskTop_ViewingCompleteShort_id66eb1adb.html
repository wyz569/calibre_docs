Viewing the Complete Short

The first step in interactive short isolation is to highlight the complete short in the layout viewer using a different highlight color for each layer.

This method applies to shorts on texted nets; for untexted nets, see “Isolating Shorts Between Untexted Nets”.

Prerequisites

  • Requirements described in “Interactive Short Isolation”

Procedure

  1. Open your layout in Calibre DESIGNrev or another supported layout viewer.
  2. Start Calibre RVE from your layout viewer, as described in the procedure “Invoking Calibre RVE From a Layout or Schematic Viewer”, and open the SVDB.
  3. Click the Layout Shorts report tab.


    The Short Isolation Toolbar is available when the Layout Shorts tab is selected. It allows you to quickly highlight the polygons that make up the path of the short and toggle the highlight in short cell context option. You can also use different keyboard strokes to highlight polygons and navigate the selection.



    Keyboard Key

    Action

    h, H

    Highlight current polygon

    Left arrow, p, P

    Highlight previous polygon

    Right arrow, n, N

    Highlight next polygon

    Up arrow

    Move to previous selection

    Down arrow

    Move to next selection
  4. Depending on your preference, use the Short Isolation Toolbar button to choose between highlighting in the top cell or in the context of the short cell:

     Highlight in top cell

     Highlight in short cell

    If the short crosses cell boundaries, the short is automatically highlighted in the top cell.

  5. Set highlighting options. Enable Highlight > Zoom to Highlights and Highlight > Use Layer Colors.

    Each layer of the short is highlighted in a different color when Use Layer Colors is selected.

  6. In the short summary view, click the short you want to analyze. The selected short appears with a blue background in the summary view and the polygons making up the short are shown in the detailed view.
  7. Highlight the short in the layout. You can use one of these methods:

    • Click the Highlight Short icon in the short isolation toolbar.

    • Right-click the short in the summary view, then choose Highlight by Layer > All Polygons in the pop-up menu.

      Note: When highlighting using the right-click menu, the menu choices Highlight Short and Highlight by Layer are independent of the menu setting Highlight > Use Layer Colors.

    The short is highlighted in the layout viewer. If it is not, you may need to set up the connection between Calibre RVE and the layout viewer. See the section “Manual Connection to a Viewer”.

    When the short is highlighted, your view should look similar to that shown in Figure 1.

    Figure 1. View of Interactive Short Isolation

  8. Proceed to “Viewing Details of Short Polygons”.
