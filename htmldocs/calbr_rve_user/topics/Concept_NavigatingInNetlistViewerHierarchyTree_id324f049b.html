Navigating in the Netlist Viewer and Hierarchy Tree

Selecting an element in other tabs or windows of your LVS session automatically highlights the element in the netlist viewer and hierarchy tree.

To navigate directly in the netlist viewer and hierarchy tree, use these methods:

  • Hierarchy tree — Expand and collapse the tree view as needed. Clicking an element shifts the netlist and schematic view to that element. The corresponding line in the netlist view is indicated with a >>. Also see Table 1 for right-click menu options.

  • Netlist view — Click an element to highlight all occurrences of that element in the netlist and highlight it in the layout and schematic viewers. To extend the highlight selection to an equivalent net in a referenced subcircuit, use Add to Select Env. Also see Table 2 for right-click menu options.

Note:

For information on the Select Environment, see “Select Environment in the Internal Schematic Viewer” Figure 1 in the topic “Selecting and Highlighting Equivalent Nets Using the Select Environment”.
