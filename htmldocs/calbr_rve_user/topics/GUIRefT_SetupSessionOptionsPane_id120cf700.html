Setup Session Options Pane

Access with: Setup > Options, then choose the Session category

Use this pane to make session settings common to all Calibre RVE applications.

Fields

Table 1. Setup Session Options Pane Contents

Setting

Description

Main window size and position

Select window size and position when Calibre RVE is opened. Select from:

  • Default — Use defaults

  • Last — Use settings in effect when Calibre RVE was last closed.

  • Custom — Use settings in effect when the Apply button is clicked.

Position of tabs (for tabbed views)

Select the location of the report tab labels.

Auto-arrange graphical windows

Select method for arranging multiple windows (such as multiple windows of the internal schematic viewer).

Show Terminal Pane

Displays a terminal window with a command prompt.

Show memory usage

Display memory usage in the status bar.

Font size

Select the font size for all Calibre RVE windows.

Use bold font

Use bold font for display.

Apply precision to display of reals

Enter the number of significant digits to display in tables and when displaying property values in highlights.

Default: 6

Use scientific notation

Use scientific notation in table and charts.

Release license after inactivity of

Enable this option to set a license timeout.

If the time-out period is set with a higher precedence setting, that value is displayed and cannot be changed; for more information, refer to “License Timeout in Calibre RVE”.

Enable Quality Agent

Display the Calibre Quality Agent for Calibre RVE dialog box when an application error occurs.

Exit confirmation

Show exit confirmation dialog box.
