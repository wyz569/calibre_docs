Setup Histogram and Colormap Options Pane

Access with: Setup > Options, then choose the Histograms category

Use this pane to make settings for histograms and colormaps in Calibre RVE for DRC and DFM.

Fields

Table 1. Setup Histogram and Colormap Options Pane Contents

Section

Options and Description

Data Distribution

Choose a linear or custom bin distribution for histograms. If “Enable truecolor” is checked, the range endpoints are used for colormaps but the number of bin divisions is determined automatically.

Linear — Use a linear bin distribution. You can control the number of divisions and the range as follows:

  • Data Min:Max — Use the minimum and maximum values in the dataset for the range.

  • Custom Min:Max — Specify the minimum and maximum values for the range.

  • Use Adjusted Range — Use the custom range specified with the DFM_LAYER_PROPERTY_RANGE annotation. See “Layer Annotations” in the Calibre YieldAnalyzer and YieldEnhancer User’s and Reference Manual.

  • Normalize (CAA only) — Use the original metric's range for the memory metric colormap rather than its (the memory metric's) range. Used for the CAA memory flow.

Custom — Enter a list of values specifying the bins.

Color Scheme

Select the color scheme from the button dropdown list. For the Custom color scheme, you can choose colors from a color selection dialog box.

The number of divisions in the color scheme preview is controlled by the number of divisions selected in the Data Distribution control.

Invert color scheme — Reverse the color order on display.

Histograms

  • Rotate X-Labels — as described.

  • Logarithmic Y Axis — as described.

  • Show Range Controls — display the range controls in the histogram window.

  • Y Axis Maximum Result Count — as described.

These controls are also available by right-clicking in the histogram window.

Colormaps

(DFM Browser only)

Show borders — Show borders for colormaps displayed within the DFM Browser.

Enable Zoom Synchronization — Synchronize the zoom level for colormaps displayed within the DFM Browser.

Enable Truecolor — Use truecolor. The colormap bin distribution is determined automatically when this option is enabled.

Profile Plots

(CMP only)

Plot Settings — Select the symbol used for profile plots.

Grid Settings — Select the line type used for the grid in profile plots.

Show exact position input — Allow exact position input for profile plots displayed in the DFM Browser. See the explanation for the figure “Profile Plot” in the Calibre CMPAnalyzer User’s Manual.
