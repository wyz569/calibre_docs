Waived Results in Calibre RVE for DRC

There are times when you want to mark a particular result as waived. A waived result is a DRC violation that is considered acceptable for the design and process. Waiver information is typically available from the foundry. Waiving results allows you to acknowledge the existence of acceptable errors in your design, therefore making your DRC debugging sessions more efficient.

Calibre provides the followings two methods for managing waivers:

  • Calibre Auto-Waivers

    Calibre Auto-Waivers manages waivers by generating waiver shapes that are merged with your design so that waivers can be applied automatically during a verification run. These waivers take into account hierarchical polygon interactions, as well as interactions of multiple waivers that may apply to a single result.

    See “Calibre Auto-Waivers Overview” in the Calibre Auto-Waivers User’s and Reference Manual and “Calibre Auto-Waivers in Calibre Interactive” in the Calibre Interactive User’s Manual for information on this method of managing waivers.

  • Calibre RVE Waived Results

    Results can be waived using the Calibre RVE interface; results can be waived individually, by check, or by cell. Waived results are stored in a drc_results_database.waived file.

    This flow is simple and easy to use, but hierarchy and cell name changes are difficult to handle. See the rest of this section for more information on using Calibre RVE waived results.

See “Calibre Auto-Waivers Versus the .waived File Method” in the Calibre Auto-Waivers User’s and Reference Manual for a more detailed comparison of these two methods.

Marking results as waived in Calibre RVE for DRC does not change your DRC results database. Instead, information about waived results is saved in a text file named drc_results_database.waived in the directory of the DRC results database. If this waiver file exists, it is automatically read by Calibre RVE before the DRC database is loaded and the waived errors are marked as waived in the display. Errors remain marked as waived even when the DRC database is regenerated by Calibre nmDRC.

After you have built a comprehensive waiver file, it can be shared and imported into Calibre RVE for DRC by other designers on your team.

Note:

The drc_results_database.waived file is automatically saved each time a change is made to the waived results in your database. If you want to save a particular version of the .waived file, you should copy it to another filename.

By using these features you can efficiently track and communicate the state of your DRC cycle with your team. The following tab lists common tasks when working with waivers in Calibre RVE.

Table 1. Waiver Tasks in Calibre RVE for DRC

Task

Comment

Save a Calibre RVE .waived file

The file drc_results_database.waived is automatically saved each time you make a change to the waived results in your database.

Mark a result as waived

Right-click a result and choose Waive or press W on the keyboard.

Include waiver comments

Enable the option “Show comment dialog while reviewing (PERC only) or waiving results” in the Waivers category on the Options tab. Waiver comments are displayed as tooltips in Calibre RVE.

To force the use of the waiver comment dialog box, set the environment variable MGC_CALIBRE_RVE_ENFORCE_WAIVER_COMMENT to 1 before starting Calibre RVE. See “Force Use of Waiver Comment Dialog Box”.

Pre-populate the waiver comment dialog box

Enable the option “Specify initial comment” in the Waivers category on the Options tab and fill in the text box that is displayed.

Edit a waiver comment

Right-click a result and choose Waive > Edit Waived Comment.

Import a Calibre RVE .waived file

Tools > Import Waivers

This is useful if the Calibre RVE .waived file was created by another user and is located in a different directory or has a different filename than drc_results_database.waived. The waived results are applied to the currently selected results tab.

Generate a waived report file

Choose Tools > Create Waiver Report in Calibre RVE, or use the waiver report generation option in the Calibre RVE batch command. See the Waiver Report Generation section in “calibre -rve” for more information.

This creates a text report of the results that are waived using Calibre RVE and opens the report in a new tab.

Generate a waiver geometry file for Calibre Auto-Waivers

Tools > Export Waivers

If your rule file includes either of the specification statements Layout Clone Transformed Placements and Layout Clone Rotated Placements, see “Waived Results from Cloned Cells” for important information on how waiving results from cloned cells is handled.
