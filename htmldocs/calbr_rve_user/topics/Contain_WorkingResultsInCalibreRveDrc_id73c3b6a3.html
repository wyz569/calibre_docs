Working with Results in Calibre RVE for DRC

This section summarizes the controls for highlighting, marking, sorting, grouping, and filtering results.

Also see the section “DRC Rule Check Comments for Calibre RVE”. You can do the following with DRC rule check comments:

  • Specify the highlight layer index number or color.

  • Specify a hyperlink to a document. The hyperlink is displayed in the check text pane.

  • Specify the display priority for a rule check, enabling you to sort the result tree by priority.

  • Specify the layers that are shown in a connected layout tool when highlighting a result.

    Note:

    Calibre PERC LDL runs create DRC results databases in ASCII format. See the section “Viewing Calibre PERC LDL Results in Calibre RVE” for instructions on viewing these results.
