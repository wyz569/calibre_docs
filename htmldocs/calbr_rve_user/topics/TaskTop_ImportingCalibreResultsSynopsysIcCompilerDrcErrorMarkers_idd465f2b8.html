Importing Calibre Results as Synopsys IC Compiler DRC Error Markers

You can convert results from an ASCII format Calibre results database (RDB) to Synopsys IC Compiler DRC error markers. The conversion process automatically imports the error markers to the Synopsys IC Compiler session, where you can view them in the Error Browser.

Prerequisites

  • A Calibre RVE license.

  • The Calibre interface is loaded; see “Loading the Calibre Interface to Synopsys IC Compiler”.

  • A Calibre results database in a supported format; see “Requirements for Converting Calibre RDBs to Native Place and Route Format”.

Procedure

  1. Make the cell containing the results you want to convert the working design in Synopsys IC Compiler.
  2. Issue the Calibre::convert_to_drc_errors command at the icc_shell prompt.

    For example, to convert all DRC results for the results database drc.results:

    icc_shell> Calibre::convert_to_drc_errors drc.results

    To convert results for rule check M2.1 in cell block1:

    icc_shell> Calibre::convert_to_drc_errors drc.results -cell block1 -checks M2.1

    Refer to the Calibre::convert_to_drc_errors reference page and “Configuring Calibre RDB Conversion to Native Place and Route Format” for details. You can specify options for side RDBs, x- and y-offset, and other parameters.

    Tip

    Some physical verification flows apply an offset to the design before verification. Use the -offset option to correct for the offset so that the results are displayed correctly in the original design.

    The Synopsys ERR view is opened and closed during the conversion. At completion the transcript window displays an information message with “Saved design named <cell>.err;<version>.”

  3. View the imported Calibre results in Synopsys Error Browser as follows:
    1. Choose Verification > Error Browser in Synopsys IC Compiler.
    2. Select “Thirdparty DRC” in the Open Error View dialog box and click the Browse button (…) to select the file.
    3. Select the appropriate cell name in the dialog box and confirm that the time stamp and version of the file is correct.
  4. Use the Error Browser to view Calibre DRC errors in your layout.
