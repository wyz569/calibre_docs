AnalyzeYieldModels

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: Common section only

Optional key name used for reanalysis. Specifies one or more yield models.

Syntax

AnalyzeYieldModels = yield_model_list

Parameters

yield_model_list

A required literal or comma-separated list of literals that specify the yield model. Choose from the following:

  • Yield_Poisson

  • Yield_Exponential

  • Yield_Seeds

  • Yield_Murphy

  • Yield_NegativeBinomial
