Highlighting Using Text Selection in Calibre RVE for LVS and Calibre RVE for PERC

If your Calibre flow produces a custom text report, you can open the report in Calibre RVE, then highlight design elements and view object information from the report. Selection-based highlighting is available from any text file open in a text tab.

For example, you can highlight nets and pins listed in the LVS Report or PERC Report using selection-based highlighting.

Prerequisites

  • Using Calibre RVE for LVS or Calibre RVE for PERC.

  • The results database must include a PHDB (persistent hierarchical database) or a netlist.

Procedure

  1. Choose File > Open Text File to open your custom text report.
  2. Select the text for the design element you want to highlight.
    Tip

    See “Creating Links to Custom Reports in Calibre RVE for LVS”, and use the lvs_ExtOutFile statement to create a link which opens the report within Calibre RVE for LVS.

  3. Right-click the design element and do the following:
    1. If the right-click menu includes the Actions selection, select it, then select Load in the Load Database Question pop-up dialog box. The Actions menu selection is not shown if the PHDB or netlist is already loaded.
    2. If needed, select the correct cell and choose Source or Layout in the Set Context dialog box (top right corner). You can also select a cell name, right-click, and select Set Context to set the cell context.
    3. Choose the desired action from the right-click menu. If the right-click menu does not include highlight selections, make sure the cell context is correct in the Set Context dialog box.
