Results Right-Click Menu

You can get a context sensitive menu by right-clicking a check name, result cluster, or individual result in the result view.

The menu selections are described in the following table.

Table 1. Results Right-click Menu Contents

Menu Item

Keyboard shortcut

Description

Highlight

H

Highlight the selected results.

Fix > Fix

Fix > Undo Fix

F

Mark the selected results as indicated

Waive > Waive

Waive > Undo Waive

W

Mark the selected results as indicated.

Waive > Edit Waived Comment

E

Edit a waiver comment.

Histogram

(right-click in tree view only)

none

Select the property you want to create a histogram for. See “Histograms and Colormaps” and “Creating Histograms and Colormaps”.

Colormap

(right-click in tree view only)

none

Select the property you want to create a colormap for. See “Histograms and Colormaps” and “Creating Histograms and Colormaps”.

Highlight All

(right-click in tree view only)

none

Highlight all results in the database.

Select All

(right-click in tree view only)

Ctrl-A

Select all results in the database.

Highlight in Area

Ctrl-H

Prompts you to select an area in the connected layout viewer. The selected results that are within the layout area are highlighted.

Fix in Area

Ctrl-F

Prompts you to select an area in the connected layout viewer. The selected results that are within the layout area are marked as fixed.

Waive in Area

Ctrl-W

Prompts you to select an area in the connected layout viewer. The selected results that are within the layout area are marked as waived.

Select in Area

Ctrl-S

Prompts you to select an area in the connected layout viewer. In the Results View, the results that are within the layout area are selected (given a blue background).

Note: You can use Ctrl-F mark results as fixed in an area, and Ctrl‑W to mark results as waived in an area.
