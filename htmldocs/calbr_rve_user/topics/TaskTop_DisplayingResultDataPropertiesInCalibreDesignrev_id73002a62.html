Displaying Result Data as Properties in Calibre DESIGNrev

Calibre RVE for DRC can send result data as properties when highlighting a result in Calibre DESIGNrev.

Procedure

  1. Open your layout in Calibre DESIGNrev.
  2. (Optional) To enable viewing result data in a pop-up window, choose Options > Object in Calibre DESIGNrev, and enable “Show object info in popup window.”
  3. Open your DRC results database in Calibre RVE for DRC.
  4. Choose Setup > Options to open the Options tab in Calibre RVE.
  5. Expand the “DRC/DFM Highlighting” area, and enable “Annotate result data to highlight shapes as properties.”
  6. Click Apply.
  7. Highlight a result in the layout.
  8. In Calibre DESIGNrev, select the highlighted result; if you cannot select it, make sure that both Path and Polygon are enabled for the selection mode.

    If “Show object info in popup window” is enabled, a pop-up window with the result data appears in the upper left.

    If “Show object info in popup window” is not enabled, choose Object > Properties to open the Object Properties dialog box; the result properties are displayed in the “Properties” area.
