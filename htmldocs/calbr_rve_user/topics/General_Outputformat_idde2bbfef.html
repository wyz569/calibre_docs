OutputFormat

Key Value for DRC HTML Report Configuration File, report specification category

Usage: Optional. Valid in Common section only.

Inheritance: Not applicable.

Specifies the format of the report. The default output format is HTML.

See “Output Format for DRC HTML Reports” for a list of browsers that support MHTML format reports.

Allowed Values

html

Output in HTML format. (default)

mht

Output a single file namedreport.mht in MHTML format.

The key UseFrames must be 0 (the default) if OutputFormat is mht.
