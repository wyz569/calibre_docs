Navigating Tips for the Internal Schematic Viewer

There are various options for panning, zooming, highlighting, and expanding elements in the schematic view.

You can navigate in the schematic view using any of the following methods:

  • Navigation icons in the top left corner of the schematic view.

    The Mute Colors () button causes all schematic elements except the highlighted element to be displayed in gray. This makes it easy to find a highlighted element within a dense display.

  • Keyboard keys. First click in the schematic view, then use the keyboard keys listed in the following table. These shortcuts also apply to the SPICE File Viewer, the schematic view for Find Path results, and the Voltage Trace GUI in Calibre RVE for PERC.
