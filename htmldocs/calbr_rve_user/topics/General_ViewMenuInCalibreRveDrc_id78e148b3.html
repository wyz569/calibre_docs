View Menu in Calibre RVE for DRC

The View menu in Calibre RVE for DRC has selections for viewing reports and setting options for the tree and result views.

The contents of the View menu in Calibre RVE for DRC are shown in Table 1. Menu selections common to all Calibre RVE applications are described in the section “Highlight Menu in Calibre RVE”.

Table 1. View Menu Contents in Calibre RVE for DRC

Menu Item

Description

New Browser

Open a new tab for the selected results database.

This can be used to set up side-by-side views of the same results database.

Rules File

Summary File

Open the rules file or summary file in a text window.

The Search Toolbar in Calibre RVE for DRC is active for the text window.

Tree Options

Select display and grouping options for the tree view. See Table 1 in the section “Tree Views”.

Result Options

Select display options for the results view. See Table 1 in the section “Result Views”.

Check Text Pane

Display the Check Text area. Check text for the check selected in the tree view is displayed.

Also see “Check Text” in the Calibre Verification User’s Manual.

Histogram

Colormap

Create a histogram or colormap of the property you select. The histogram or colormap displays results for the cells or checks selected in the tree view.

See “Histograms and Colormaps” and “Creating Histograms and Colormaps”
