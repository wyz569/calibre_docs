mgc_rve_open_db

SKILL function that opens the specified results database in a running Calibre RVE session if one exists. The results database may be a file or directory. If there is not a running Calibre RVE session, the database is opened in a new session.

The command opens the database in an existing Calibre RVE session only if that session was started using Calibre > Start RVE from the Cadence Virtuoso menu or a previous mgc_rve_open_db or mgc_rve_open_rdb_file command. If the Calibre RVE session was started from Calibre Interactive, the mgc_rve_open_db command opens the RDB in a new Calibre RVE session.

Only one Calibre nmLVS, PERC, PEX, or DFM database (svdb or dfmdb) may be open at a time in Calibre RVE. If one of these database types is open in Calibre RVE, it is closed when a new database of one of these types is opened.

Syntax

mgc_rve_open_db(“db” @optional “top_cell”)

Arguments

  • “db” — A required parameter specifying the path to the results database. The database may be a file or directory. The supported database types are DRC(ASCII format), LVS, PERC, PEX, and SPICE. SPICE files are opened in the Calibre RVE SPICE File Viewer.

  • “top_cell” — An optional argument specifying the name of the top cell; only applies to LVS, PERC, and PEX databases.

Examples

Enter the following two commands in the CIW to open the svdb results database and the RDB nar.rdb in the same Calibre RVE session:

mgc_rve_open_db("svdb")
mgc_rve_open_db("nar.rdb")
