Highlight Layout Net/Device/Pin Dialog Box

Access with: Highlight > Highlight Layout Net/Device/Pin

The Highlight Layout Net/Device/Pin dialog box highlights a net, device, or pin in the layout.

Figure 1. Highlight Net/Device/Pin Dialog Box

Description

This menu selection is available in Calibre RVE for DFM, or in Calibre RVE for DRC if an associated DFM Database is also open. PERC LDL CD and PERC LDL P2P applications produce DRC Results databases in ASCII format and an associated DFM Database.

Objects

Table 1. Highlight Net/Device/Pin Dialog Box Contents

Menu Item

Description

Layout Cell

The layout cell to highlight in.

  • Net

  • Device

  • Pin

Select the type of element to highlight in the dropdown list, then enter the name in the text box. Backslash is used for the hierarchy separator.

Pins are entered with the following format:

device:pin

Examples:

(net) INA

(device) M0

(device) X14/M1

(pin) D1:pos

Highlight by layer

Highlight polygons on different layers in different colors.

Highlight

Create the highlights.

Clear Highlights

As described.
