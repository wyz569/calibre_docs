Using Calibre RVE for PERC

The Calibre Results Viewing Environment (RVE) for PERC provides a graphical interface for viewing and analyzing the results from Calibre PERC runs. You can highlighting results, sort and filter results, and view a rendered schematic.

This chapter assumes that you have a Calibre PERC or Calibre PERC LDL database open in Calibre RVE, and are familiar with the general operation of Calibre RVE.
