Calibre RVE Overview

Calibre RVE (Results Viewing Environment) is a graphical user interface that allows you to view and debug output from Calibre results databases. Calibre RVE is integrated to most standard industry design tools. When Calibre RVE is connected to a supported design tool, you can highlight Calibre results in the layout or schematic; viewing the result in the layout or schematic makes it easier to correct the design error.

Calibre RVE has multiple interfaces. These are:

  • Calibre RVE for DRC — This interface allows you to browse DRC, ERC, and 3DSTACK results databases. Objects in the database are the results of design rule checks in your rule file. A result can be an error, or it can be geometry created in a rule check. Calibre RVE allows you to highlight results in an attached layout viewer. You can also mark results as fixed or waived, and sort results by check, cell and fixed/waived state.

    If you are using Calibre Auto-Waivers, Calibre RVE for DRC can export the waiver shapes to a database, or merge the waiver shapes into your layout database.

    For detailed information on Calibre RVE for DRC, see “Using Calibre RVE for DRC”.

  • Calibre RVE for DFM — This interface allows you to browse the contents of a DFM database. The DFM report card is integrated with Calibre RVE for DFM and allows you to browse errors and generate color maps and histograms of the errors.

    For detailed information on Calibre RVE for DFM, see “Using Calibre RVE for DFM”.

    You can also create HTML reports based on the DFM results; see “DFM HTML Reporting”.

  • Calibre RVE for LVS — This interface allows you to investigate and debug your Calibre nmLVS/Calibre nmLVS-H discrepancies and short isolation results by using the SVDB directory. Calibre RVE for LVS includes the ability to cross-probe between layout views, schematic views, and SPICE netlist browsers (layout and source).

    Calibre RVE for LVS includes an internal schematic viewer that displays rendered schematics from the netlist. This is useful for cross-probing between layout and source. You can also use an external schematic editor to view schematics.

    For detailed information on Calibre RVE for LVS, see “Using Calibre RVE for LVS”.

  • Calibre RVE for PERC — This interface allows you to investigate and debug Calibre PERC results, including Calibre PERC LDL results. Calibre RVE for PERC includes all the features of the LVS application; in addition, many of the features of Calibre RVE for DRC are available when viewing Calibre PERC LDL results.

    For detailed information on Calibre RVE for PERC, see “Using Calibre RVE for PERC”.

  • Calibre RVE for PEX — This interface allows you to investigate and debug Calibre parasitic extraction results by using the SVDB directory. Parasitic element are displayed in a table view. You can calculate point-to-point resistance and coupling capacitance to selected nets. Calibre RVE for PEX includes all the features of the LVS application.

    For detailed information on Calibre RVE for PEX, see “Using Calibre RVE for PEX”.

  • DRC and DFM HTML Reporting — DRC and DFM HTML reporting allow you to create HTML pages based on views in Calibre RVE for DRC or Calibre RVE for DFM. The HTML reports are easy to share.

    For detailed information, see “DRC HTML Reporting” and “DFM HTML Reporting”.

  • RDB Conversion — You can convert results from an ASCII format Calibre results database (RDB) to error markers and route guides in your Place and Route tool. This allows you to use familiar tools when debugging errors and correcting the design.

    See “Calibre RDB Conversion: Working with Calibre Results in Place and Route Tools”.

  • Calibre RVE SPICE File Viewer — The SPICE file viewer displays the netlist hierarchy, a rendered schematic view, and the netlist.

    The SPICE file viewer includes a Tcl scripting interface to specify a schematic view and save the view to a postscript or Scalable Vector Graphics file.

    See “Calibre RVE SPICE File Viewer” and “Scripting Interface to the Calibre RVE SPICE File Viewer”.

If you intend to use Calibre in an interactive editing environment, you probably want to invoke Calibre RVE from your layout editor or from Calibre Interactive, as explained in “Starting Calibre RVE”. When Calibre Interactive opens Calibre RVE, it establishes communication between Calibre RVE and the layout or schematic viewer from which you invoked Calibre Interactive, allowing you to cross-probe the results. With cross-probing, you select a result in Calibre RVE and view the result in the layout or schematic viewer.
