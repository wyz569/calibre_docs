Highlights Tab

Access with: Highlight > Setup Highlights or the Setup Highlight Filters icon () in the toolbar in Calibre RVE for LVS, PERC, and PEX. The Highlights Tab is not available with the ERC Results tab.

The Highlights tab in Calibre RVE specifies which layers and devices are highlighted in your layout viewer, assigns custom highlighting colors to layers, and specifies highlighting in only certain regions of the design. By default, all layers and devices are highlighted in the complete design. The Highlights tab is available in Calibre RVE for LVS, PERC, and PEX.

Note:

The Layout Shorts tab does not use the Highlights tab settings for layer filters.

Description

Tip

You can apply the search and selection filters from the Finder Tab to highlighting by enabling Apply Finder Filters while highlighting in the filter area of the Finder Tab. Access to the Highlights tab is disabled if “Apply Finder Filters while highlighting” is enabled on the Finder tab.

Figure 1. Highlights Tab

Highlight filters are saved when you exit Calibre RVE for LVS, and restored the next time you open Calibre RVE. The filter icon is blue when a filter is applied, and a pop-up notice in the lower right corner informs you that a filter is applied the first time you highlight with a filter applied.

The table in the next section lists the available filter controls, with links to the detailed reference section.

Objects

Table 1. Filters on the Highlights Tab

Filter

Description

Layer Filters and Colors on the Highlights Tab

Selects colors for highlight layers and selects the layers that are displayed on highlight events.

Device Filters on the Highlights Tab

Selects the devices that are highlighted. The device filter is applied when highlighting devices on nets.

Window Filters on the Highlights Tab

Limits highlighting to specified rectangular regions.
