File Menu in Calibre RVE

The File menu has some selections that are common to all Calibre RVE applications.
Table 1. File Menu Contents in Calibre RVE

Menu Item

(restriction)

Description

Open Database

Displays the Open Calibre Database dialog box where you specify the pathname to a results database.

Directory and File Browsing - Click the browse button (…) to open a file browser. The file browser is opened in the current working directory by default. To open the file browser in different directory, enter the directory path in the text entry field, then click the browse button while pressing the Ctrl key on the keyboard. The file browser will open in the specified directory, or the parent directory if a file pathname is specified.

Also see “Specifying a Calibre RVE Working Directory”.

The Open Database icon is located on the far left of the standard toolbar.

Close Database

Close the database.

Reload Database

Reloads the current database. It does not clear discrepancies or results marked as fixed.

You can also reload the current database by clicking the Open Database button at the same time you press the Ctrl key on the keyboard.

Open Text File

Opens a file browser, then opens the selected text file in a new tab. When a text file is open, additional menu selections are available in the File and View menus, as described in “Viewing Text Files in Calibre RVE”.

Recent Databases

A cascading menu lists the most recently opened databases.

Recent Text Files

A cascading menu lists the most recently opened text files.

Exit

Exit Calibre RVE.
Note:

In Calibre RVE for DRC, the open database selections and toolbar button are disabled while processing results from a Calibre Interactive run with the “Start RVE as soon as results are available” enabled on the Misc tab in the Setup Preferences dialog box.
