Setting the Socket Port for Calibre RVE

Calibre RVE uses certain default port numbers for socket communication with a connected layout or schematic viewer. The port numbers can also be set manually, as described here. Individual layout and schematic viewers may have additional methods for setting the port numbers; see the section for the specific viewer.
