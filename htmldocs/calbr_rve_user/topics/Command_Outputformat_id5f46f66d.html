OutputFormat

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: ByCellColormap, ByErrorColormap, ByErrorHighlight, ByErrorResults, ByErrorResultsSummary, ByWindowColormap, CAPlot, CellDrillDown, CellSummary, ChipSummary, Common section, MemoryPlot, NetDrillDown, NetSummary, TotalSummary, WindowDrillDown, WindowMemoryPlot, WindowSummary

Required key name that specifies a list of formats to which the output is written. Not all formats are valid for all report types.

Syntax

OutputFormat = list_of_formats

Parameters

list_of_formats

A required comma-separated list of one or more literals that specify the reporting format—refer to Table 1. Choose from any of the following:

  • csv

  • chart

  • html
