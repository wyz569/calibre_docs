Cadence QuickView

Calibre RVE is integrated to the Cadence QuickView Signoff Data Analysis Environment, allowing highlighting from Calibre RVE. There are some version compatibility restrictions.

The version requirements are given in the following table. The “Highlight in Area” action, in which an area is selected in the viewer, is only available in certain versions.
