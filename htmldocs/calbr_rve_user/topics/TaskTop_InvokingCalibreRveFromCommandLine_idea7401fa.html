Invoking Calibre RVE from the Command Line

You can invoke Calibre RVE from the command line.

Prerequisites

  • “Calibre RVE System Requirements”

  • An existing Calibre results database or SPICE file.

Procedure

  1. (Optional) Navigate to the directory containing the Calibre results database or SPICE file.
  2. Use one of the following commands to invoke Calibre RVE from the command line:

    Calibre RVE for …

    Command line call

    DRC

    % calibre -rve -drc drc_database1

    where drc_database is an ASCII DRC results database. You may specify more than one database to open.

    DFM

    % calibre -rve -dfm dfmdb

    where dfmdb is a DFM database or a DFM database directory.

    LVS

    % calibre -rve -lvs svdb TOPCELL

    where svdb is an SVDB directory and TOPCELL is the top cell.

    PERC

    % calibre -rve -perc svdb TOPCELL

    where svdb is an SVDB directory and TOPCELL is the top cell. The SVDB directory should have PERC results.

    PEX

    % calibre -rve -pex svdb TOPCELL

    where svdb is an SVDB directory and TOPCELL is the top cell. The SVDB directory should have PEX results

    SPICE file

    % calibre -rve -spice spice_file

    where spice_file is a SPICE netlist. You can use the option “-top cell” to specify the top cell.

    If a database or directory path is not given in the command line, the application prompts you for the path.

    Tip

    If you later open a layout viewer, see “Manual Connection to a Viewer” to establish a connection between Calibre RVE and the layout viewer.
