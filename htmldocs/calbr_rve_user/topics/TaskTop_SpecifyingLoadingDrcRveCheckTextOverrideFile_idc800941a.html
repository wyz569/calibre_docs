Specifying and Loading the DRC RVE Check Text Override File

The DRC RVE Check Text Override File (CTO File) is used to configure a Calibre RVE for DRC session with DRC RVE rule check comments. DRC RVE rule check comments allow you to specify highlight colors, display layers, document links, and other settings. You can load the CTO file using an environment variable, automatically based on file naming, or with the Tools > Import CTO File menu selection.

For support of the CTO file outside of Calibre RVE for DRC, see these topics:

  • “Specifying Highlight Colors in DRC HTML Reporting”

  • DRC Rule Check Comments for Calibre RealTime Custom in the Calibre RealTime Custom User’s Manual

Procedure

  1. Create or obtain a CTO file. See these topics for information:

    “DRC RVE Check Text Override File (CTO File) Format”

    “Creating a Check Text Override (CTO) File Using Calibre RVE”

    “Creating a Check Text Override (CTO) File Using Calibre Interactive nmDRC”

    Creating the CTO file using Calibre RVE or Calibre Interactive results in a CTO file with RVE Show Layers statements to control layer visibility while highlighting.

  2. Specify the CTO file with one of the following methods, given in order of precedence:

    • MGC_RVE_CTO_FILE environment variable (highest precedence)

      Set the environment variable MGC_RVE_CTO_FILE to the pathname to the DRC RVE check text override file. Calibre RVE for DRC applies this file to all DRC databases opened during the Calibre RVE session, including auxiliary databases.

      Also see the section “Loading a CTO File When Starting Calibre RVE from Calibre Interactive”.

    • drc_database.cto file for automatic loading

      Name the CTO file drc_database.cto and place it in the same directory as drc_database, where drc_database is the Calibre nmDRC results database. Calibre RVE for DRC reads the file drc_database.cto when it opens drc_database and sets MGC_RVE_CTO_FILE to this filename. This method is used if the environment variable MGC_RVE_CTO_FILE is not defined in the shell environment.

      Calibre RVE for DRC applies the CTO file to all DRC databases opened during the Calibre RVE session, including auxiliary databases.

    • Tools > Import CTO File menu selection (lowest precedence)

      Choose Tools > Import CTO File in Calibre RVE and enter the name of a check text override file. The file is applied to the active results tab.
