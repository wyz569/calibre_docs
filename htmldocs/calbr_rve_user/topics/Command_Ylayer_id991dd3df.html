YLayer

Key value for Configuration File in DFM HTML Reporting

Supported ReportTypes/Sections: ScatterPlot

Optional key name for the ScatterPlot report that specifies layer names for the Y axis property.

Usage

YLayer = layer_names

Arguments

layer_names

A comma-separated list of layer names.

Description

YLayer specifies the layers used for the YProperty when creating the scatter plot.

If only one of XLayer and YLayer is specified, XLayer = YLayer is assumed. If neither XLayer or YLayer is specified, scatter plots are generated for all layers that have simulation results.
