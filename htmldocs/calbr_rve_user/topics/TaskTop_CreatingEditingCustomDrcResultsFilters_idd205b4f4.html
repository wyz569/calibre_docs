Creating and Editing Custom DRC Results Filters

Custom DRC result filters allow you to specify multiple conditions for a filter, such as check name, property value, waived state, and other conditions. Custom filters are saved and included in the dropdown list of filters.

Video

The video “How to Include and Exclude Rule Checks to Filter Results in Calibre RVE” demonstrates how to do the following:

  • Create a custom filter to include and exclude rule checks

  • Edit and delete a custom filter



Procedure

  1. Do one of the following to open the Filter Editor:

    • To create a new filter — Click the filter button and select “Create Custom Filter” (see Figure 1 in the topic “Applying DRC Results Filters”).

    • To edit an existing filter — Click the filter button , hover over the filter you want to edit, then click the icon, as shown in the following figure. If the filter is already selected, the icon appears without hovering.



  2. Select and define the desired filter conditions in the Filter Editor. Unix glob-style pattern matching characters are allowed in the fields for the check and cell name.
    Table 1. Custom Filter Expressions in Calibre RVE for DRC

    Filter Expression

    Description

    Cell Name (OR)

    Filter on cell name. If multiple “Cell Name (OR)” expressions are selected they are combined with OR logic.

    Cell Name (AND)

    Filter on cell name. If multiple “Cell Name (AND)” expressions are selected they are combined with AND logic.

    Check Name (OR)

    Filter on check name. If multiple “Check Name (OR)” expressions are selected they are combined with OR logic.

    Check Name (AND)

    Filter on check name. If multiple “Check Name (AND)” expressions are selected they are combined with AND logic.

    Fixed State

    Filter on the Fixed State.

    Waived State

    Filter on the value of Waived State.

    Waived By

    Filter on the username for the Waived State.

    Waived Comment

    Filter on the Waived comment.

    Waived Date

    Filter on the Waived date.

    Property Exists (OR)

    Filter on whether a specified property name is attached to a result. If multiple of these expressions are selected, they are combined with OR logic.

    Property Exists (AND)

    Filter on whether a specified property name is attached to a result. If multiple of these expressions are selected, they are combined with AND logic.

    Property Value (OR)

    Filter on a property value. If multiple of these expressions are selected, they are combined with OR logic.

    Property Value (AND)

    Filter on a property value. If multiple of these expressions are selected, they are combined with AND logic.

    Area

    Filter on a specified region in the layout. Specify the cell to filter in, then select whether to filter within or not in a rectangle or a custom polygon.

    Click the button to draw the region:

    • Rectangle — With a layer selected in the layout viewer, click and drag your cursor to draw a rectangle. The resulting coordinates of the lower left and upper right vertices appear in the Coords field at the bottom of the Filter Editor dialog box.

    • Polygon — With a layer selected in the layout viewer, click to place the vertices of your desired region. The resulting coordinates of the polygon’s vertices appear in Coords field at the bottom of the Filter Editor dialog box.

      In Calibre DESIGNrev, double-click when placing the final vertex to finish drawing your polygon.

    Alternatively, enter the coordinates directly in the Coords region without drawing in the layout viewer:

    • Rectangle — Enter the lower left and upper right vertices.

    • Polygon — Enter pairs of x, y coordinates.

    Press the button to highlight the defined region in the layout view.
  3. Specify the filter name for a new filter (top of Filter Editor form).
  4. Click Save to save the filter.
  5. Click Apply to apply the filter to the database. The Apply button also saves the filter.

    The Filter Editor remains open and active. However, it may be hidden behind the main Calibre RVE display. The main Calibre RVE display is not available while the Filter Editor is open.

  6. Click Close to close the Filter Editor. Changes to the filter are not saved unless either Apply or Save has been clicked.
    Tip

    You can delete a custom filter by hovering the mouse over the filter name, then clicking the red X. You cannot delete a custom filter that is currently applied to the database.
