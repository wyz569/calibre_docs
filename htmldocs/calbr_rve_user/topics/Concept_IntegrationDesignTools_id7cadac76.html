Integration to Design Tools

Calibre RVE can establish connections to most standard industry design tools. This section describes where to find setup and operation information.

Table 1 lists the design tools that support a Calibre integration. The Setup and Operation Information column indicates whether the integration is automatic or requires configuration.

The Calibre Interfaces web page has additional information about the integration to different design tools:

www.mentor.com/products/ic_nanometer_design/calibre-integration/
