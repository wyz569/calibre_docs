mgc_rve_toggle_visible_layer_set

Toggles the design layer visibility.

Syntax

mgc_rve_toggle_visible_layer_set

Description

This function toggles the design layer visibility between the default set and the layers specified by the most recently processed RVE Show Layers check text comment. The command is used internally by Calibre RVE as part of the Restore Default Layers feature.
