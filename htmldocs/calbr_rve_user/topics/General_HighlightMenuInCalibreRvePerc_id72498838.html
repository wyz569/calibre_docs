Highlight Menu in Calibre RVE for PERC

The Highlight menu in Calibre RVE for PERC includes items setting highlight options. Some menu selections are only available when viewing results from a Calibre PERC LDL run.
Table 1. Highlight Menu Contents in Calibre RVE for PERC

Menu Item

(restriction)

Description

Setup Highlights

Opens the Highlights Tab, from which you can specify layer, device, and window filters, and specify highlighting by layer color.

 

This selection is only available for layout-based Calibre PERC and Calibre PERC LDL results.

Use Layer Colors

Toggles the “Use layer colors while highlighting” option on the Highlights Tab. This selection is only available for layout-based Calibre PERC and Calibre PERC LDL results.

 

When enabled, design layers in the highlighted object are colored with the colors specified on the Highlights tab. Supported for the Calibre DESIGNrev, Cadence Virtuoso, and Synopsys IC Compiler layout viewers.

 

In the Calibre DESIGNrev layout viewer, highlight layers are labeled with the layer name when this option is enabled.

Highlight all

(PERC LDL only)

Highlight all results.

Set Check Highlight Layers

(PERC LDL only)

Opens the Set Highlight Layers Dialog Box, where you can assign the layers used for highlighting according to the rule check.
