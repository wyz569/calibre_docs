Examining Calibre PERC LDL Results in Calibre RVE

You can highlight a result in an attached layout viewer, and view connectivity objects (nets, instances, devices, and pins) in the Internal Schematic Viewer.

You may also want to see these sections in the Calibre PERC User’s Manual:

  • “Debugging Current Density Errors Using Calibre RVE”

  • “Highlighting LDL CD Simulation Results”

  • “Debugging P2P Resistance Errors Using Calibre RVE”

  • “Highlighting LDL P2P Simulation Results”

The following procedure gives general instructions for viewing Calibre PERC LDL Results in Calibre RVE.

Prerequisites

  • “Opening the Calibre PERC LDL Database in Calibre RVE and Setting Options”

Procedure

  1. Click the P2P or CD results tab in Calibre RVE. If the tab is not present, choose View > Results.
  2. (Optional) Click the Setup Highlight color scheme button () to set the highlight color; see “Setting Highlight Color in Calibre RVE for LVS, PERC, and PEX”.
  3. To highlight a result, select it in the Tree View or Result View, then press H on the keyboard. You can also double-click a result in the Result View to highlight it, or right-click for more options. A view of a Calibre PERC LDL CD result highlighted in Calibre DESIGNrev is shown in the following figure.


  4. To create a histogram of a result property, select the result set of interest in the Tree View, then right-click in the property column header in the results table and choose Histogram prop, where prop is the property to histogram. The Histogram selection is not available for all columns in the results table.

    A histogram of the property values appears, as shown below for the R(ohms) property in a Calibre PERC LDL P2P run.



    To modify the appearance of the histogram, right-click in the histogram window and choose Show Range Controls. A control panel appears with various options for changing the appearance of the histogram; see “Creating Histograms and Colormaps”.

  5. To highlight a net, select the desired result, then click the net icon (, for CD results) or the net text (for P2P results) in the Result Data Pane. Figure 1 shows a view of a net highlighted by layer in Calibre DESIGNrev and highlighted in the Internal Schematic Viewer.

    If the Internal Schematic Viewer does not open automatically, make sure options are set as described in Step 7 of “Opening the Calibre PERC LDL Database in Calibre RVE and Setting Options”.

    Figure 1. Calibre PERC LDL Net Highlighting

    For CD results, you can also highlight a net by clicking the net icon () in the Tree View when it is grouped by the net property, or by clicking the net icon in the column header of the Result View in details format. P2P results do not have the net icon.

    Click the right mouse button on an item for additional options, as shown in the next figure:



  6. To get more information about nets, devices, instances, and ports, use the Finder Tab (View > Finder) or the Info Pane (View > Info from the main menu or Show Info from a right-click menu).
