Parasitics Tab

Use this report tab to the view the net parasitics table and access detailed information about selected nets. If parasitic extraction with multiple corners was performed, there is a Parasitics tab for each process corner.
Figure 1. Parasitics Tab View

Click the column header to sort the values. For the Layout Net and Source Net columns, you can use a hierarchical or alphanumeric sort. The default is hierarchical; right click to change the sort type.

The Parasitics tab includes the following information and controls:

Table 1. Net Parasitics Table

Item

Description

R Count

Number of parasitic resistors on the net.

P2P R Count

Number of point-to-point resistance calculations for the net. The number is the total of results listed in the point-to-point report, if it exists, and results calculated in the Point-to-point Resistance Calculator. See “Calculating Point-To-Point Resistance and Inductance”.

L Count

Number of parasitic inductors on the net.

P2P L Count

(not shown)

Number of point-to-point inductance calculations for the net. The number is the total of results listed in the point-to-point report, if it exists, and results calculated in the Point-to-point Inductance Calculator. See “Calculating Point-To-Point Resistance and Inductance”.

K Count

Number of coupling coefficients for parasitic mutual inductance on the net. Not included in all databases.

C Total

Total intrinsic capacitance.

CC Total

Coupled capacitance.

If the number is also reported as a percentage, the value is the coupled capacitance to the nets listed in the coupling capacitance control area. To be displayed, the percentage must be above the threshold of 1.0e-20.

C+CC Total

Sum of intrinsic and coupled capacitance, where the coupled capacitance selection is indicated in the coupling capacitance control area.

Find Nets Control

Search for a specific net or nets in the parasitics table.

Coupling Capacitance Control

Calculate coupled capacitance to selected nets.

Right-click menu

Use the right-click menu with one or more selected nets. Select multiple nets using Shift‑ and Ctrl‑click.
