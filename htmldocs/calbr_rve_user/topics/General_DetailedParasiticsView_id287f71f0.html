Detailed Parasitics View

The detailed parasitics view shows information on the parasitic elements for a selected net.

Display the detailed parasitic view by doing one of the following:

  • Double-click a net in the net parasitics table.

  • Select a net in the net parasitics table, right-click the selected net, and choose Show Detailed Parasitics.

  • Select a net in the net parasitics table and click the icon in the parasitics toolbar.

You should see view similar to that shown in the following figure:

Figure 1. Detailed Parasitics View

By default the detailed view tab is named with the layout net name. If source names are available and you want to use them, choose Setup > Options, choose the LVS/PERC/PEX category, and uncheck “Use the layout net name in PEX detailed parasitics tab.”

The parasitic elements are shown in a tree view that can be expanded and collapsed.

The properties displayed for parasitic resistors are given in following table. The properties reported for a particular parasitic resistor depend on the rule file settings. The units are determined by the parasitic extraction run. The default unit for length is microns; this can be changed with the RSCALE keyword to PEX Netlist.
