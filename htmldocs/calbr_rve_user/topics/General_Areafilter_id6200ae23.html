AreaFilter

Key Value for DRC HTML Report Configuration File, filter category

Usage: Optional. Valid in all report types, and in Common section.

Inheritance: Filter Inheritance. If the GroupBy key references another section, then the key and value are read from that section and any locally specified value is ignored.

Include only results that are within (or not in) the specified region.

Allowed Values

[filterType] cellName lowerLeftX lowerLeftY upperRightX upperRightY

Gives the filter type, cell name, and the vertices of a rectangular region.

A filter type of IN_RECT represents within the rectangular region, and a filter type of OUT_RECT represents not in the region. If this argument is not provided, the default is IN_RECT.

[filterType] cellName x1 y1 x2 y2 x3 y3 ...

Gives the filter type, cell name, and the vertices of a polygon region.

A filter type of IN_POLY represents within the polygon region, and a filter type of OUT_POLY represents not in the region. If this argument is not provided, the default is IN_POLY.

Examples

CellFilter = IN_RECT XO_abc 10 10 110 110
CellFilter = OUT_POLY XO_abc 10 10 30 40 60 70 100 40
Note:

The ! character can also be used as a substitute for OUT_RECT.

CellFilter = ! XO_abc 10 10 110 110
