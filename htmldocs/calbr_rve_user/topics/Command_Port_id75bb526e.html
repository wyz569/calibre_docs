port

Shape element for Symbol Library file

The port element defines a port name and location in the symbol shape. Port names in the symbol definition must match the port names in the netlist for the custom symbol shape to be applied.

Usage

port port_name direction -loc x_port y_port x_stub y_stub

Parameters

Parameter

Description

port

A required keyword.

port_name

The name of the port.

direction

direction is specified as io, dir, or io.dir

io — The electrical direction, which can be one of the following:

  • input | in — Input port.

  • output | out — Output port.

  • inout | ino — Inout port.

dir — The direction to connect from, as follows:

  • left —Connect from the left side.

  • right —Connect from the right side.

  • top —Connect from the top.

  • bottom | bot — Connect from the bottom.

  • unknown — The direction is unknown and the tool determines the direction to connect from.

-loc

A required keyword.

x_port y_port

The x and y coordinates of the port connection.

x_stub y_stub

The x and y coordinates of the port stub end.

Examples

The ending non-numerical entry is not shown in the examples. 

port A in -loc -20 -10 0 -10 
port Y out -loc 40 -10 28 -10
port CK in.bot -loc -20 -30 0 -30
