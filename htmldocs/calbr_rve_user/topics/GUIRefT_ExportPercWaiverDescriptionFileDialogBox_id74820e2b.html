Export PERC Waiver Description File Dialog Box

Access with: Tools > Export PERC Waiver Description File

Creates a Calibre PERC waiver description file based on the waivers in the database. This option only applies to topology waivers.

Objects

Table 1. Export PERC Waiver Description File Dialog Box Contents

Item

Description

Waiver Description File Name

The pathname for the waiver description file that is created.

The %T pattern is replaced with the PERC topcell, and environment variables are replaced with their values. For example, entering %T_$USER.waived creates a file with the name TOPCELL_jdoe.waived.

PERC Waiver Flow Type

  • Full-chip — As indicated.

  • Library — Use this option if you are creating waivers for a small block or IP cell. This option adds the “‑parent cell” parameter to the waiver description, where cell is the top cell of the loaded results database.

Waiver Export Type

  • Full — Exports a waiver description file for use with the PERC Waiver Path statement.

  • Delta —Creates a delta waivers file. This file is used in the Calibre PERC Multi-User Waiver Flow.

Waivers to Export

  • All — Export all waived results.

  • Displayed — Export only the waived results in the Calibre RVE display. If you applied a filter, only waived results that satisfy the filter are exported.

  • Selected — Export only the waived results selected with Shift- and Ctrl-click.

The Displayed and Selected options are not available when the Delta export option is selected.
