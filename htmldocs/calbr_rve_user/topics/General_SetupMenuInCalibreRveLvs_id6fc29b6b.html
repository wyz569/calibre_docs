Setup Menu in Calibre RVE for LVS

The Setup menu has some selections that are specific to a Calibre RVE for LVS session.
Table 1. Setup Menu Contents in Calibre RVE for LVS

Menu Item

Description

Password

Opens a dialog to set a password for the database.

Note: The password protects the PHDB and XDB, not the SVDB. Actions such a highlighting or using the Finder require the PHDB and XDB. By default, the PHDB and XDB are not loaded until they are needed. To change the default behavior, open the Options tab, choose the Databases category, and expand the “Database Loading” section.
