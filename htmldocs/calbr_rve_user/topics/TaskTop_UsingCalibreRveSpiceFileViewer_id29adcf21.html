Using the Calibre RVE SPICE File Viewer

Calibre RVE includes a SPICE file viewer which displays the netlist hierarchy, a rendered schematic view, and the netlist. The SPICE file may be a compressed file ending in the .gz or .Z file extensions.

The Calibre RVE SPICE File Viewer is very similar to the Internal Schematic Viewer. Most menu selections and toolbar buttons for the SPICE File Viewer are easy to understand. Tooltips are available for the toolbar buttons.

While the Internal Schematic Viewer reads power and ground names from the rule file and uses them in the rendered schematic, the Calibre RVE SPICE File Viewer does not do this because it does not have access to a rule file. Likewise, the Internal Schematic Viewer determines schematic symbols from the device recognition statements in the rule file, but the Calibre RVE SPICE File Viewer does not. The SPICE File Viewer includes options to set case sensitivity, device names, and power and ground nets.

Prerequisites

  • A SPICE file.

    The following compressed file formats are supported:

    • Files ending in the .gz file extension and compressed in the gzip format.

    • Files with the .Z file extension. Such files are opened with the gunzip utility.

Procedure

  1. Open the SPICE File Viewer with one of these methods:

    • Invoke Calibre RVE from a supported design tool, enter the SPICE netlist name in the database field, and select “SPICE File” as the database type.

      See “Invoking Calibre RVE From a Layout or Schematic Viewer”.

    • Choose File > Open Database from an existing Calibre RVE session, enter the SPICE netlist name in the database field, and select “SPICE File” as the database type.

    • From the command line as follows, where spice_file is a SPICE netlist:

      calibre -rve -spice spice_file

      You can also specify the top cell:

      calibre -rve -spice spice_file ‑top topcell
  2. If a schematic view is not shown when you start the Calibre RVE SPICE File Viewer, do the following:
    1. Make sure that View > Show Schematics Pane is enabled.
    2. Click a subcircuit in the hierarchy tree (left panel).
  3. Click the Options button () to open the Options tab.
  4. Click the Schematic Viewer category.
    1. Set options in the top area. These options are common to the Internal Schematic Viewer and the SPICE File Viewer.
    2. Expand the area “Standalone SPICE Netlist only” to set options specific to the SPICE File Viewer, such as case sensitivity, NMOS and PMOS device type names, and power and ground nets.

      Unlike with the Internal Schematic Viewer, these options must be set explicitly. Do not clear the entries for the power and ground nets.

    See “Setup Schematic/Netlist Viewer Options Pane”.

  5. Use the Select menu to set the “select environment.” See “Select Environment in the Internal Schematic Viewer” for information.
