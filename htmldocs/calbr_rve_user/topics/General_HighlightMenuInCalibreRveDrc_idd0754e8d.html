Highlight Menu in Calibre RVE for DRC

The Highlight menu in Calibre RVE for DRC has selections for highlighting in the top cell or the context of the result and for assigning the layers used for highlighting.
Table 1. Highlight Menu Contents in Calibre RVE for DRC

Menu Item

Description

See Table 1 for the Highlight menu selections that are common to all Calibre RVE applications.

Highlight in Context

Specifies to highlight results in the coordinate space of the cell rather than top-level coordinate space. This checkbox is disabled if you did not specify the DRC Cell Name YES XFORM specification statement in the rule file. For details refer to the section “Viewing Results in Cell Space”.

Ctrl-T toggles the setting of this option.

Highlight All

As described.

Set Check Highlight Layers

Opens the Set Highlight Layers Dialog Box, where you can assign the layers used for highlighting.

Also see “Special Highlighting Considerations for Layout Viewers”.
