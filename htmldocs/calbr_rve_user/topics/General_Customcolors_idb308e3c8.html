CustomColors

Key Value for DRC HTML Report Configuration File, histogram and colormap category

Usage: Optional. Valid in DRCHistogram and DRCColormap report types, and in Common section.

Inheritance: Standard Inheritance

Specifies a custom color palette to use for histograms and colormaps when ColorSet=custom.

Allowed Values

color1, color2, …

Comma-separated list of colors. Colors may be specified by name or RGB value.

Examples

ColorSet = custom
CustomColors = #99004D, #990099, #4D0099, #004D99, #000099
