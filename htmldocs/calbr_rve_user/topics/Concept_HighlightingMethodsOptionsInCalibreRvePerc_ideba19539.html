Highlighting Methods and Options in Calibre RVE for PERC

You can use several methods to highlight a result in Calibre RVE for PERC.

  • Select a tree node or result in the Tree View, then right-click and choose Highlight.

    If multiple results are selected in the Tree View, than all results and their associated objects are highlighted. For example, if you select a rule which has multiple results in the Tree View, then right-click and choose Highlight, all results from the selected rule are highlighted.

  • Select a result in the Results View, then right-click and choose Highlight.

    You can highlight multiple results by selecting multiple results with Shift- and Ctrl‑click.

  • Click a design element displayed in blue text in the Result Data Pane.

  • Right-click a design element displayed in blue text, then choose a highlight action from the pop-up menu.

Note:

If “Show only one placement for each hierarchical PERC result” is enabled on the LVS/PERC/PEX category pane of the Options tab, only one placement is highlighted by default when highlighting. You can highlight all placements by right-clicking a result and choosing Highlight All Placements; however, a maximum of 5000 placements is highlighted in order to provide reasonable performance.

See “Speeding Up Database Loading by Viewing One Placement of Results in Calibre PERC” for information on “Show only one placement for each hierarchical PERC result.”

Highlights are displayed in the Internal Schematic Viewer and any external design tools that are connected. If the Internal Schematic Viewer does not open when you highlight a result, choose View > Schematics, then repeat the highlight action.

After using PERC LOAD XFORM to generate transformed schematic data, both the original and the transformed representations of the schematic are viewable.

You can set highlight colors and filters as follows:

  • The Highlights Tab () and the Highlight > Use Layer Colors option are available if you are viewing a results database from a layout-based Calibre PERC run. Filters can be set on the Highlights tab.

  • The highlight color scheme can be set with a button on the toolbar; see “Setting Highlight Color in Calibre RVE for LVS, PERC, and PEX”. The setting “Layer colors” is only available for layout-based Calibre PERC runs.
