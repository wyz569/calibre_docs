PropLinearRange

Key Value for DRC HTML Report Configuration File, histogram and colormap category

Usage: Optional. Valid in DRCHistogram and DRCColormap report types, and in Common section.

Inheritance: Standard Inheritance

Specifies the distribution range for a histogram or colormap when PropRangeDistributionType=Linear.

Allowed Values

x_min, x_max

A pair of floating-point values specifying the range. The strings min and max may also be used to specify the minimum and maximum values in the data set.

auto

(default) Specifies to use the minimum and maximum values in the data set as the distribution range.

Examples

PropLinearRange = auto
PropLinearRange = 0.2,12.5
PropLinearRange = 1.0,max
