Reference for Calibre RDB Conversion to Native Place and Route Tool Formats

Several commands are available for converting results from an ASCII format Calibre results database (RDB) to native Place and Route tool formats. A check map file is required when converting to route guides or route blockages.
Table 1. File and Command Summary for Calibre RDB Conversion

Command or File

Summary

Calibre::convert_to_drc_errors

Converts results in a Calibre RDB to DRC error marker format and imports the converted results to the current session. The command is executed from the design tool command prompt.

Calibre::convert_to_route_guides

(Synopsys IC Compiler)

Converts results in a Calibre RDB to Synopsys IC Compiler route guides and imports the converted results to the current session. The command is executed from the icc_shell prompt.

Calibre::convert_to_route_blockages

(Cadence Encounter)

Converts results in a Calibre RDB to Cadence Encounter route blockages and imports the converted results to the current session. The command is executed from the encounter command prompt.

Check Map File Format for Calibre RDB Conversion

Input file that associates one or more layers to each rule check. The check map file is required when converting to route guides or route blockages; the file is optional when converting to DRC error markers.

calibre -rve Command Line for Calibre RDB Conversion

Converts results in a Calibre RDB to native Place and Route tool formats and saves the results to a file. The command is executed from the Linux shell. Use one of the import commands to import the saved file to the design tool session.

Calibre::import_drc_errors

Imports a file saved from a previous Calibre RDB conversion to DRC error markers. The command is executed from the design tool command prompt. The errors can be viewed in the design tool after the import.

Calibre::import_route_guides

(Synopsys IC Compiler)

Imports a file saved from a previous Calibre RDB conversion to Synopsys IC Compiler route guides. The command is executed from the icc_shell prompt. The route guides can be viewed in the Synopsys IC Compiler session after the import.

Calibre::import_route_blockages

(Cadence Encounter)

Imports a file saved from a previous Calibre RDB conversion to Cadence Encounter route blockages. The command is executed from the encounter command prompt. The route blockages can be viewed in the Cadence Encounter session after the import.
