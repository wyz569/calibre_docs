Opening Calibre PERC Results in Calibre RVE

You can start Calibre RVE from the command line or a design tool and view the results from a Calibre PERC run.

Prerequisites

  • A Mask SVDB Directory containing Calibre PERC results.

Procedure

  1. Start Calibre RVE with one of these methods:

    If you want to …

    Do the following:

    Start Calibre RVE from the command line

    calibre -rve -perc svdb

    Start Calibre RVE from your layout or schematic viewer

    1. Depending on your design tool, choose one of the following:

      • Verification > Start RVE

      • Calibre > Start RVE

    2. For Database, specify the database and the top cell.

    3. For Database Type, select PERC.

    If there were runtime errors, the PERC Runtime Errors and Warnings Dialog Box is displayed if “Show LVS/PERC runtime errors and warnings on startup” is enabled in the LVS/PERC/PEX category of the Options tab.

  2. After Calibre RVE opens, click the Options button () to open the Options pane and set session and highlighting options.

Results
