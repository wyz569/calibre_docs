Results From Cloned Cells

Results from cloned cells are displayed in Calibre RVE with the #Flipped and #Rotated properties in the Details format of the Result View.

The SVRF statements Layout Clone Transformed Placements and Layout Clone Rotated Placements instruct Calibre nmDRC to use cell cloning while constructing the hierarchical database (HDB). The CL property in the results database indicates the transformation of the cloned cell. Calibre RVE derives the #Flipped and #Rotated values from the CL property.

You can highlight results from cloned cells in the context of the original cell or in the top cell. Results from cloned cells are waived in the original cell.
