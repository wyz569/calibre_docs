Isolating Shorts Between Untexted Nets

There may be times when you need to find a short between untexted layout nets. These types of shorts can be difficult to locate and the Isolate Path on Layout Net search on the Finder tab can assist with this task. Shorts between untexted nets can be reported by the LVS comparison stage, often just as a discrepancy stating that a layout net corresponds to two source nets. It is very helpful to include LVS Report Option FX in the Calibre run; this option reports shorts between untexted nets and provides a fix suggestion.

Note: The short isolation features on the Layout Shorts Tab in Calibre RVE for LVS depend on the LVS Isolate Shorts statement, which performs connectivity extraction and finds shorts between texted nets. Therefore, the Layout Shorts tab is not helpful in finding shorts between untexted nets.

Prerequisites

  • An SVDB from a Calibre nmLVS run which includes LVS comparison. See Rule File Requirements for Examining Results with Calibre RVE for LVS.

  • LVS Report Option FX specified in the rule file. It can also be specified in Calibre Interactive—nmLVS; see “Specifying Calibre nmLVS Report Options” in the Calibre Interactive User’s Manual for more information.

  • The layout database for your LVS run.

  • The source netlist for your LVS run.

  • A layout viewer with an interface to Calibre RVE. See “Connection to a Design Tool”.

Procedure

  1. Open your layout in Calibre DESIGNrev or in another supported layout viewer.
  2. Start Calibre RVE from your layout viewer, as described in the procedure “Invoking Calibre RVE From a Layout or Schematic Viewer” and open your SVDB.
  3. Disable Highlight > Clear Existing Highlights.
  4. Click the Setup Highlight color scheme icon and select “ Default colors” from the dropdown list.
  5. View the short circuit error in the Comparison Results tab. An example is shown below with View > Tree Options > View by Fix Type selected:


  6. In the detailed view, click the shorted layout net to highlight it.
  7. For each of the source nets, right-click and choose Highlight Devices on Net. The devices on each source net are highlighted in different colors.
  8. Choose View > Finder to open the Finder Tab.
  9. Select “Layout” for “Search in Layout or Source?”.
  10. Select Isolate Path on Layout Net for “What type of search?”.
  11. Select the cell to search in.
  12. Select “All Paths Per Net In Layout.” A dialog box opens for selecting the path endpoints in the layout.


  13. Click in the layout on starting and ending points on the shorted net that you want to check. The highlighted devices for each source net can guide you in selecting points on the layout net. If the endpoints are on the same physical net, the path is highlighted in the layout viewer; otherwise “No common net” is displayed in the dialog box.

    Continue clicking in the layout to find paths between locations on layout nets. The dialog remains open, and each path is added to the Finder search result list.
