Specifying the Highlight Color and Layer Visibility When Highlighting DRC Results to Synopsys IC Compiler

You can set the highlight color and layer visibility when using Calibre RVE with Synopsys IC Compiler. This can aid in debugging DRC results.

In the IC Compiler II toolbar, select Calibre > Highlight Colors to open the Set Highlight Colors dialog box. You can change the highlight color, the fill pattern, and the line width of the border, as well as toggle the option to cycle colors while highlighting.

You can also use DRC Rule Check Comments for Calibre RVE to control highlight color and the design layers that are shown when a result is highlighted. See the following rule check comments:

  • RVE Highlight Color — Specifies the highlight color used when a result from the rule check is highlighted.

  • RVE Show Layers — Specifies the design layers that are shown when a result from the rule check is highlighted.

Procedure

See “Methods to Add DRC RVE Rule Check Comments” for instructions.
