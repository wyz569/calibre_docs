Device Filters on the Highlights Tab

Access with: Highlight > Setup Highlights, then enable “Use Device Filter while highlighting.” See Figure 1.

When “Use Device Filter while highlighting” is enabled on the Highlights tab, only device types that are selected with a check mark in the device filter are highlighted. If no device type is selected in the filter or the setting is disabled, all device types are highlighted. The device filter is applied when highlighting devices on nets.

Description

The filter selections only apply when highlighting multiple elements. If a single element, such as a specific device, is highlighted, it is always be highlighted. Device filters override layer filters if both are applied.

Select default devices — Right-click in the device filter table and choose Select Common Devices. This selects the MP, MN, MD, ME, L, R, C, LDD, LDDD, LDDM, and LDDP device types.

Right-click in the device filter for a pop-up menu to toggle, select, and clear the device types in the filter.
