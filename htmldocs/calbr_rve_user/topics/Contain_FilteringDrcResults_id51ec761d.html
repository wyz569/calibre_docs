Filtering DRC Results

DRC results filters allow you to select a subset of your DRC results to display in Calibre RVE. System-defined filters are available, or you can create and save a custom filter. You can also use the command line to filter results and save them in new results databases. See the following sections:

  • Creating and Editing Custom DRC Results Filters
  • Filtering DRC Results by Category
