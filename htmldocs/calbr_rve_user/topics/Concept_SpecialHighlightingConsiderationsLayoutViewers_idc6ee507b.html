Special Highlighting Considerations for Layout Viewers

Depending on your layout viewer, there may be some special highlighting methods or considerations to take into account.

Details about the Calibre RVE interface to integrated layout viewers are given in “Interfacing With Layout and Schematic Viewers”. The following topics may be of interest:

Specifying a highlight layer

  • Specifying a highlight layer or highlight color — You can set highlight layers and colors for individual rule checks from Calibre RVE or in the rule file:

    • Highlight > Set Check Highlight Layers menu item — Opens the Set Highlight Layers Dialog Box.

    • Rule check comments in your rule file — Set either the highlight layer color or the highlight layer index using a rule check comment.

      Highlight Layer Color — You can set the highlight layer color in a rule check comment; this method applies when using the Calibre DESIGNrev/WORKbench, Synopsys IC Compiler, Cadence Virtuoso, and Synopsys Laker layout editors.

      See the command reference section “RVE Highlight Color”.

      Highlight Layer Index — You can set the highlight layer index in a rule check comment; this method works for any supported layout editor.

      See the command reference section “RVE Highlight Index”.

  • Highlighting with user-defined keyboard shortcuts — see “Highlighting in Cadence Virtuoso With User-Defined Bindkeys (Keyboard Shortcuts)” and “Using Calibre RVE with Pyxis”.

  • Specify layers shown when highlighting — In Calibre DESIGNrev, Synopsys IC Compiler, and Cadence Virtuoso, you can specify the layers that are shown when you highlight a result from a DRC check. See the command reference section “RVE Show Layers” for details.

    If you use RVE Show Layers, you can use the Restore Default Layers button to toggle between displaying the default set of visible layers and the layers specified with the most recent RVE Show Layers statement. See the section “Restore Default Layers Toolbar”.

  • Calibre DESIGNrev considerations — See “Calibre RVE with Calibre DESIGNrev and Calibre WORKbench” for information on setting text size and other highlight options.
