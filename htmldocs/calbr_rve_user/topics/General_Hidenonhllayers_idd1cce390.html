HideNonHLLayers

Key Value for DRC HTML Report Configuration File, highlighting category

Usage: Optional. Valid in DRCResults, DRCHighlight, and DRCColormap report sections, and in Common section.

Inheritance: Standard Inheritance

Specifies whether to hide all layers other than highlight layers. This option is useful in top-level overview snapshots, where hiding the design layers provides a better view of the highlights.

Allowed Values

0

Do not hide non-highlight layers. (default)

1

Hide non-highlight layers.

Examples

[Common]
LayoutWidth = 200
LayoutHeight = 200
HighlightInContext = 1
ShowHighlightsInTopCell = 1

[Overview Snapshot]
ReportType = DRCHighlight
GroupBy = Overview
ZoomLayoutFactor = fit
WorldView = 0
HideNonHLLayers = 1
CheckHLLayer = Colors
