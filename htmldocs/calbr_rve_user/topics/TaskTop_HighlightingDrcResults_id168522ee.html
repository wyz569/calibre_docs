Highlighting DRC Results

You can set various options that control how results are highlighted and highlight the result using several methods. You can highlight clusters (groups of results) from the tree view.

Prerequisites

  • A Calibre nmDRC results database is open in Calibre RVE for DRC.

Procedure

  1. Set highlight options from the Highlight menu or a dropdown list from the standard toolbar, as shown here:


  2. If desired, click the Highlight in Cell Context button () or select Highlight > Highlight in Context to highlight results in the cell they are reported in rather than the top cell, as described in the section “Viewing Results in Cell Space”. Ctrl-T toggles the setting for this option.
  3. If desired, select Highlight > Set Check Highlight Layers to set the highlight color. This opens the Set Highlight Layers Dialog Box, where you can then select a highlight color for each rule check.
  4. Highlight a result with one of the following methods:

    • Use the buttons in the highlight toolbar (). Use Ctrl-click with the toolbar buttons to highlight all results in the cluster.

    • Double-click a result.

    • Right-click a result or cluster for a highlight pop-up menu. See “Results Right-Click Menu” for a description of the menu selections.

    • Use the H keyboard key. See “Keyboard Shortcuts (Hotkeys) in Calibre RVE for DRC” for other keyboard shortcuts.
