Highlight in Lower Cell Dialog Box

To access: Highlight in Cell action for a net

The Highlight in Lower Cell dialog box is displayed for certain LVS discrepancy types when the net has no shapes in the selected layout cell.

Note:

For example, the Highlight in Lower Cell dialog box can occur when highlighting the nets that make up a “Possible hierarchical short chains to non-floating extra pins” discrepancy. This occurs if Calibre nmLVS added ports to the extracted netlist in order to include the possible hierarchical short chain. Such added ports have no corresponding geometry in the layout; however, Calibre can determine where such a port connects in a lower level cell.

See “Tracing Possible Hierarchical Short Chains to Non-Floating Extra Pins”.

Description

Highlight in Lower Cell

Continue with the highlight action in the current cell context. This action results in the message that the layout cell has no shapes on the net.
