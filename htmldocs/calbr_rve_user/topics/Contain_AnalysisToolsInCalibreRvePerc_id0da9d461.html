Analysis Tools in Calibre RVE for PERC

Calibre RVE has several features that allow you to sort, group, and filter results according to database properties. These features allow you to focus on a particular set of results as you debug a design. For example, you might attach a property named “Severity” to results, and use that property to group the tree hierarchy or filter the results in Calibre RVE. You can also create a histogram of a database property.
