Bookmarks for Result Tab Views in Calibre RVE for PERC

Bookmarks save the view options for the result tab so that you can easily return to a previous view for a debug session. A bookmark saves information such as the applied result filter, the Group By setting, the column selections, and more.

There are two types of bookmarks:

  • User Bookmarks — User bookmarks are created by the user. They are automatically saved in the $HOME/.rve/dbOptions directory when you exit Calibre RVE. You can delete a user bookmark or change its name.

  • Admin Bookmarks — Admin bookmarks are permanent bookmarks associated with the results database. You create admin bookmarks by exporting the user bookmarks in a session to a special file with the .xrvedb file extension. See “Creating and Managing Admin Bookmarks” for information.

You use the Bookmarks menu to save a new bookmark and to open a bookmarked view.
