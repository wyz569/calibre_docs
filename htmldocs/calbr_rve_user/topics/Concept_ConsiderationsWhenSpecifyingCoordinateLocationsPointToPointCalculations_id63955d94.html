Considerations When Specifying Coordinate Locations for Point-to-point Calculations

The parasitic extraction tool takes some extra steps when using a coordinate as one of the end points for calculating point-to-point resistance or inductance. If the specified coordinate does not coincide with the end point of an extracted resistor or inductor, the end point of the closest extracted element is used in the calculation.

For details, see the command PEX Report Point2Point in the SVRF Manual.

If coordinate substitution occurs because the specified coordinate does not coincide with the end point of an extracted element, the point-to-point calculator in Calibre RVE enters the new coordinates into the coordinate entry field, and uses them for the calculation.

If you want to be warned in advance of coordinate substitution, choose Setup > Options, select the LVS/PERC/PEX category, and enable the setting “Ask to choose closest coordinate in PEX P2P”. Calibre RVE displays a pop-up warning when this option is enabled if the specified coordinate does not coincide with the end point of an extracted element. The warning reports the specified coordinate and the closest end point and asks if the user wants to continue or cancel the operation. Calibre RVE also labels the two points in the layout viewer, draws a line segment between them, and zooms into the location.
