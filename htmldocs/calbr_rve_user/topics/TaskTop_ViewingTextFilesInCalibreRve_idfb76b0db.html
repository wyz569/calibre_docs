Viewing Text Files in Calibre RVE

Text files are opened in a new tab. You can save text files using the File menu, and control the display using the View menu. Some Calibre RVE applications have additional methods for opening specific text files.

Procedure

  1. Choose File > Open Text File.

    Some specific files, such as the rules file and text report files may be available in the View menu or the Navigator Pane of Calibre RVE LVS, PERC, and PEX.

  2. Once a text file is open in a tab, some additional selections are available in the View and File menus, as listed in Table 1.
    Table 1. Menu Selections for Text Files

    File Menu

    View Menu

    Save File

    Allow Edits

    Save File As

    Note:

    Note on large files — By default, files larger than 50 Mbytes are opened in view-only mode and the option View > Allow Edits is not available.
