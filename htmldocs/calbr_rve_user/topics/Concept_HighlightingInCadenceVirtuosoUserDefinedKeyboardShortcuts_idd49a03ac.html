Highlighting in Cadence Virtuoso With User-Defined Bindkeys (Keyboard Shortcuts)

Several highlighting functions are available in the Calibre interface to Cadence Virtuoso. You can map these functions to keyboard shortcuts (or hotkeys).

The following functions are available in the Cadence Virtuoso layout editor:

mgc_rve_hl_next_error
mgc_rve_hl_prev_error
mgc_rve_hl_curr_error

Following the directions for your layout editor, map keyboard shortcuts to these functions to highlight the next, previous, and current errors, respectively.

Prior to using these commands, you must have started Calibre RVE for DRC and established a socket connection between the layout editor and Calibre RVE. These shortcuts allow you to highlight results without leaving the editing environment. If multiple Calibre RVE windows are open, these commands communicate with the last window that had input focus.
