Export Waived Results Dialog Box

Open with: Tools > Export Waivers in Calibre RVE for DRC

This dialog box is used to export waiver shapes for use with Calibre Auto-Waivers.

Description

The following sections in the Calibre Auto-Waivers User’s and Reference Manual describe how to use Calibre RVE to export waivers:

  • “Creating Embedded Waiver Shapes Using Calibre RVE”

  • “Creating Non-Embedded Waiver Shapes Using Calibre RVE”

  • “Exporting Waiver Shapes to a Cadence Virtuoso Database Using Calibre RVE”

Figure 1. Export Waived Results Dialog Box

See the tables in the next section for information on each tab in the dialog box:

Objects

Table 1. Design Tab Contents for Export Waived Results Dialog Box

Setting

Description

Export format

Specifies the format for the exported waiver shapes:

  • GDSII

  • OASIS

  • Virtuoso — Waiver cells are exported to a Cadence Virtuoso library and placed in the design. See “Exporting Waiver Shapes to a Cadence Virtuoso Database Using Calibre RVE”in the Calibre Auto-Waivers User’s and Reference Manual for more details.

Output file

(GDSII and OASIS export)

The filename of the exported waiver database when “Export format” is GDSII or OASIS.

The default filename is topcell.waived.gds ortopcell.waived.oas. If the waiver name includes %I (such as %I.waived.gds), it will be replaced with the top cell name of the RDB file.

Cell library

(Virtuoso export)

The Cadence Virtuoso library containing the top cell of the open results database.

When Calibre RVE places the waiver cells in the design, it traverses the cell hierarchy in the open design, starting from the top cell location indicated by “Cell library,” and places the waiver cells at the correct locations.

View

(Virtuoso export)

The name of the Cadence Virtuoso cell view for the top cell of the results database. Exported waiver cells are placed in this cell view.

Waiver library

(Virtuoso export)

The name of the Cadence Virtuoso library to contain the waiver cells. Exported waiver cells are first saved to this library, then placed in the design.

Use streamin template

(Virtuoso export)

Specifies the streamin process for writing waiver cells to the “Waiver library,” as follows:

  • Enabled — Calibre RVE uses the specified streamin template when writing the waiver cells to the waiver library. All parameters in the streamin template are used except for the input file and library name.

  • Disabled — Calibre RVE uses a system-generated streamin template file when writing the waiver cells to the waiver library.

See “Waiver Export to a Cadence Virtuoso Database for Calibre Auto-Waivers” for additional details.
Table 2. Options Tab Contents for Export Waived Results Dialog Box

Setting

Description

Waiver setup file

A waiver setup file.

Import button

Loads the specified waiver setup file. The layer and datatype settings in the waiver setup file are used to control the waiver export process; all other setting in the waiver setup file are ignored.

See “Importing Layer and Datatype Settings from a Waiver Setup File in Calibre RVE” in the Calibre Auto-Waivers User’s and Reference Manual for more information.

Merge from file

(Only available for GDSII and OASIS export format)

  • Disabled — Waivers shapes are saved to the “Output file” specified on the Design tab.

  • Enabled — Waiver shapes are merged with the layout database specified by “Output file” or “Other file.” If the precision of the results database differs from the precision of the layout database, a prompt is displayed with option to resolve the problem.

Output file

New waiver shapes are merged with the database specified for “Output file” on the Design tab.

Other file

New waiver shapes are merged with the specified database. The merged database is saved to the new file specified by “Output file” on the Design tab. “Output file” is overwritten if it exists.

Place waiver cell in intermediate cell

Place waiver cells in an intermediate cell instead of the parent cell. See the section “Intermediate Container Cells” and ADD_WAIVER_HIERARCHY in the section “Waiver Setup File Format for Waiver Generation” in the Calibre Auto-Waivers User’s and Reference Manual.

This option is automatically enabled if “Annotate waiver cells with IP_MATCH checksum text” is selected.

Generate waivers in cloned cells

Enables waiver generation for cloned cells. Cloned cells can be generated by statements in your rule file, such as Layout Clone Rotated Placements. If this option is not enabled, waivers are not generated for results in cloned cells.

Note:

This option cannot be used with IP_MATCH validation. IP_MATCH checksums are not compatible with waivers in cloned cells.

Output precision

Specify the precision (output precision) for the output waiver geometry database. The output waiver shapes are scaled by (output precision / results database precision).

“Output precision” may not be specified with “Magnify coordinates by.”

Magnify coordinates by

Scale the waiver shape coordinates by the specified value. The precision of the output waiver geometry database is the same as the DRC results database precision.

To correct for a precision difference between the results database and the layout database, the magnification factor should be (results database precision / layout database precision). For example, if the layout database precision is 1000 and the precision of the results database is 10,000, you should specify 0.1 for “Magnify coordinates by.”

“Magnify coordinates by” may not be specified with “Output precision.”

Text Magnification

Specify a magnification level for texts written during waiver creation. Text magnification only applies when the output format is GDS. The default is 0.005. See TEXT_MAG parameter for the waiver setup file.

Magnify Text Spacing

Magnify the spacing between text written during waiver creation; this only applies when the output format is GDS. The magnification level specified for the “Text Magnification” option is used; the default of 0.005 is used if “Text Magnification” is not specified.

See the MAGNIFY_TEXT_SPACING parameter for the waiver setup file.

Annotate waiver cells with cell checksum text

Write a geometry checksum to new waiver cells. The checksum is in hexadecimal format.

Annotate waiver cells with Calibre version text

Annotate waiver cells with a text object indicating the Calibre version.

Annotate waiver cells with rule checksum text

Annotate waiver cells with a checksum of the rule that produced the result. You must also specify a rule file.

See “Rule Checksum Annotations” in the Calibre Auto-Waivers User’s and Reference Manual.
Table 3. Criteria Tab Contents for Export Waived Results Dialog Box

Setting

Description

Annotate waiver cells with waiver criteria text

If enabled, waiver description information is added as text objects to the waiver cell.

When waiver criteria is included in the waiver cells, you can use the data to create the waiver description file for the calibre ‑waiver run. See the EXTRACT keyword for the WAIVER_CRITERIA parameter in the section “Waiver Setup File Format for Waiver Verification” in Calibre Auto-Waivers User’s and Reference Manual.

Input waiver criteria file(s)

Specifies one or more waiver criteria files.

By default, Calibre RVE annotates waiver cells with a global criteria of SINGLE 100 100. If non-default criteria values are desired, specify the path to a waiver criteria file containing different criteria settings.

See the section “Waiver Criteria File Format” in the Calibre Auto-Waivers User’s and Reference Manual for definitions of the waiver criteria.
Table 4. Layers Tab Contents for Export Waived Results Dialog Box

Setting

Description

Layer and Data Type

Layer number and datatype settings for the exported waiver shapes. Enter the desired values in the table.

See “Waiver Setup File Format for Waiver Generation” in the Calibre Auto-Waivers User’s and Reference Manual for a description of the options and their defaults; also see “Text Object Annotations” in the same manual.

The comment, user name, and date entries are used for the Calibre RVE waiver comment that is entered when “Show comment dialog while reviewing (PERC only) or waiving results” is enabled on the Options tab in the Waivers category.
Table 5. IP_MATCH Tab Contents for Export Waived Results Dialog Box

Setting

Description

Annotate waiver cells with IP_MATCH checksum text

Specifies to generate an IP_MATCH signature for each cell in the “IP_MATCH input file” that has a waived result in the results database.

The option “Place waiver cells in intermediate cell” is automatically enabled if “Annotate waiver cells with IP_MATCH checksum text” is enabled.

See “Creating IP_MATCH Signature Checksums Using Calibre RVE” in the Calibre Auto-Waivers User’s and Reference Manual for more information.

IP_MATCH input file

A GDSII or OASIS format layout database that contains the cells you want to use for IP_MATCH.
