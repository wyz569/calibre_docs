DRC Results Tab

The results from a Calibre nmDRC run are displayed in a report tab labeled by the database name. The tab has a Tree View and a Result View that lists individual results. Each view has display options that are discussed in their respective reference sections.

See “Session Window in Calibre RVE for DRC” for a view of the session window in Calibre RVE for DRC.

You can save a particular configuration of view settings using bookmarks, as described in “Bookmarks for Result Tab Views in Calibre RVE for DRC”.
