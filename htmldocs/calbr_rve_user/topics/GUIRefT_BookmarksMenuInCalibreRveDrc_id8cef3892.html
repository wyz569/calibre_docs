Bookmarks Menu in Calibre RVE for DRC

Access with: Click the Bookmarks icon () in the summary area of the results tab, at the right, above the Result View area

The Bookmarks menu has selections to save a new bookmark and open a bookmarked view in Calibre RVE for DRC. Bookmarks save the view options for the result tab so that you can easily return to a previous view for a debug session. A bookmark saves information such as the applied result filter, the Group By setting, the column selections, and more.

Note:

The Bookmarks menu is shown in the following figure. If there are no bookmarks defined for your result database, Add Bookmark is the only menu selection displayed.

Enable the checkbox to open the bookmarked view in a new results tab when you select a bookmark. The name of the new tab has “(n)” appended to the database name, where n is an integer. You can right-click in the tab and select Rename to change the name of the results tab.
