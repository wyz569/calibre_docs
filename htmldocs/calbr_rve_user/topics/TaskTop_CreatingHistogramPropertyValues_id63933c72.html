Creating a Histogram of Property Values

A histogram is a useful way to view the distribution of property values for a result. Histograms can only be created for numeric properties.

Prerequisites

  • You have one of the following database types is open in Calibre RVE for PERC:

    • A Calibre PERC results database with properties attached to the results. See “Example to Generate Calibre PERC Results with Database Properties”

    • A Calibre PERC LDL results database. See “Opening the Calibre PERC LDL Database in Calibre RVE and Setting Options”.

Procedure

  1. If desired, choose Setup > Options then the Histograms & Colormaps category to make selections for the bin distribution and display properties of the histogram. These settings are used when you create the histogram in Step 3.
  2. In the Tree View, select the cell or check name. Use Shift- and Ctrl- click to select multiple cells or rule checks.

    Only the results that belong to the selected cells or checks are shown in the histogram or colormap.

  3. Create the histogram or colormap with one of the following methods:

    • View > Histogram, then select the property to histogram.

    • Right-click in the Tree View and choose Histogram, then select the property.

    Histograms and colormaps are not available for LDL disjoint path (NaN) results.

  4. (Optional) You can right-click in the histogram window for a menu with additional options. “Show Range Controls” displays controls for the bin distribution and color scheme. Click Update to refresh the histogram view after changing the display options.
