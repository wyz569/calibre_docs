Sorting and Grouping DRC Results

You can group results in the tree view by cell name, check name, property name, or property value. Use the View > Tree Options > Group By menu selection or right-click in the first column header to select the grouping method.


See Table 1 for a description of the View menu, and the reference section “DRC Results Tab” for more information on grouping and sorting results.

Click in a column header to sort the table or tree by that value. The default action is to sort results by the check order in the database.

Note:

If Calibre RVE DRC is started automatically after a Calibre Interactive run and the setting “Start RVE as soon as results are available” on the Misc tab of the Setup Preferences dialog box is enabled in Calibre Interactive, then the Group By setting in Calibre RVE must have a top-level grouping of “Check.” If the top-level grouping is not “Check”, the grouping is set to “Check/Cell.” Other grouping methods are not available until all results are loaded into Calibre RVE.
