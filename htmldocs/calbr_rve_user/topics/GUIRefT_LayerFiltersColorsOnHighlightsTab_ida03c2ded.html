Layer Filters and Colors on the Highlights Tab

Access with: Highlight > Setup Highlights. See Figure 1.

The layer filter and color controls in the Highlights tab select colors for highlight layers (for supported viewers only) and select the layers that are displayed on highlight events.

Objects

Table 1. Highlights Tab - Layer Filter and Colors Controls

Item

Description

Skip contact/via layers while highlighting

 

(Enabled by default)

When selected, contact layers appearing in a Connect … BY statement are not included in highlighting actions. The layer filter is automatically enabled and contact layers are un-selected when this option is enabled.

Contact layers are given by the layerC parameter in the following Connect statement:

Connect layer1 layer2 BY layerC

If a layer appears as both a contact and non-contact layer, then the layer is not filtered out.

 

This option is also located on the Options tab in the Highlighting category in the “Layout Highlighting” expandable section.

Note: Since “Skip contact/via layers while highlighting” requires use of the layer filter, “Use Layer Filter while highlighting” is automatically enabled and cannot be disabled if “Skip contact/via layers while highlighting” is selected.

Use Layer Filter while highlighting

When selected, highlights appear only on the selected layers. The layer filter is applied when highlighting nets, device-pins on nets, and ports on nets. Device filters override layer filters if both are applied.

Use layer colors while highlighting

 

(checkbox is located below the filter table)

When selected, design layers in the highlighted object are colored with the specified colors. Supported for Calibre DESIGNrev, Cadence Virtuoso, and Synopsys IC Compiler.

In Calibre DESIGNrev, highlight layers are labeled with the layer name when this setting is on.

This setting is the same as Highlight > Use Layer Colors. Also see “Setting Highlight Color in Calibre RVE for LVS, PERC, and PEX”.

More highlighting options

Click to go to the “LVS/PERC/PEX Highlighting” area on the Options tab in the Highlighting category; see “LVS/PERC/PEX Highlighting Options” for details.

Column

Click in a column header to sort the table by that column.

Filter

A check ( ) selects the layer for display during highlighting in the layout. If no layers are selected or the setting is disabled, all layers are highlighted.

Right-click to open a pop-up menu that includes Toggle Layer Filter, Select All Layers, and Clear Layer Filter.

This column is present if “Skip contact/via layers while highlighting” or “Use Layer Filter while highlighting” is enabled.

Color

Highlight color for that layer.

Click in a color box to bring up a dialog to choose a new color, or use the right-click menu.

Changes to the default color selections are saved to the .rvedb File for Calibre RVE Settings. Calibre RVE loads highlight colors from this file on subsequent invocations.

This column is present if “Use layer colors while highlighting” is selected.

Layer Number

The internal layer number used by Calibre RVE.
