HistogramOutputFormat

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: ByCellHistogram, ByErrorHistogram, ByWindowHistogram, Common section

Optional key name that specifies a list of formats to which the output is written for histograms. This option is ignored for other report types.

Syntax

HistogramOutputFormat = list_of_formats

Parameters

list_of_formats

A required comma-separated list of one or more literals that specify the reporting format—refer to Table 2. Choose from any of the following:

  • csv

  • chart

  • txt

Defaults to chart if this key value is not specified.
