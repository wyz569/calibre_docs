Calibre RVE with Synopsys IC Compiler

Synopsys IC Compiler is fully supported for all Calibre RVE applications: Calibre RVE for DRC, LVS, PEX, PERC, and DFM. Calibre RVE provides an easy to use GUI for analyzing Calibre results and allows you to highlight the results in an attached Synopsys IC Compiler design window.

All Calibre RVE features are supported in the integration to Synopsys IC Compiler. See “Getting Started with Calibre RVE” for general instructions on using Calibre RVE, and application-specific chapters in the manual for detailed information.
