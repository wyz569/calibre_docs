Sections

The Common section (specified as [Common]) is an optional section used to specify default values for key names not specified in other sections.

For more information on key names, refer to “Key Name Reference.”

All other sections are used to generate reports that are written to the output directory. The following rules apply to section names in the configuration file:

  • Section names must be unique. Order of appearance for sections is irrelevant.

  • Key names must be unique within a section. Order of appearance for key names is irrelevant.

  • If a key name is not specified in an optional section, and it exists in the Common section, its value from the Common section is used for the optional section.

  • Each optional section must contain a value for the ReportType key. If a ReportType key is not specified directly, it must be specified in the Common section.
