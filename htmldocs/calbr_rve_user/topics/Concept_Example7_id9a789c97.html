Example 7: Ranges for Histograms and Colormaps

This example configuration file sets up custom ranges for histograms and colormaps. 
[Common]
BinsToInclude = *
MetricsToInclude = *
RulesToInclude = *
IncludeFlatMetrics = 1
MetricsToShareRange = M1.*:lambda*, M2X.*:lambda*
[ErrorHistogram]
ReportType = ByErrorHistogram
RulesToInclude = M1.OPEN, M1.SHORT, M2X.OPEN, M2X.SHORT
MetricsToInclude = ca
BinsToInclude = *
MetricsToShareRange = M1.OPEN:ca M1.SHORT:ca, M2X.OPEN:ca M2X.SHORT:ca
[CellColormap]
ReportType = ByCellColormap
RulesToInclude = M1.*, M2X.*
MetricsToInclude = *
BinsToInclude = *
MetricsToShareRange = M1.OPEN:area M1.SHORT:area, M2X.OPEN:area M2X.SHORT:area
