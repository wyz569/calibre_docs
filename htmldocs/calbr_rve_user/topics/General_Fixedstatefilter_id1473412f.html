FixedStateFilter

Key Value for DRC HTML Report Configuration File, filter category

Usage: Optional. Valid in all report types, and in Common section.

Inheritance: Filter Inheritance. If the GroupBy key references another section, then the key and value are read from that section and any locally specified value is ignored.

Include only results with the specified fixed state.

Allowed Values

All

Include all results. (default)

Fixed

Include only fixed results.

NotFixed

Include only not fixed results.
