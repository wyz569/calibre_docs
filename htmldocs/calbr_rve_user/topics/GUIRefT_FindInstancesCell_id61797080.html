Find Instances of Cell

Access with: View > Finder, then choose “Instances of Cell” for “What type of search?”

Searches in: Layout

Requires: PHDB. See Mask SVDB Directory in the SVRF Manual.

The search type “Instances of Cell” on the Finder tab finds instances of a given layout cell. The search is made at the top level by default, but can be expanded to all levels.
