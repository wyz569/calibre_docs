Session Window in Calibre RVE for LVS

The main session window in Calibre RVE for LVS includes a tree (hierarchical) and detailed view of results, toolbars for common actions and settings, and an optional Navigator pane.

The main session window is shown in Figure 1. A brief description of the window areas is given in Table 1.

Figure 1. LVS Session Window in Calibre RVE

Table 1. LVS Session Areas in Calibre RVE

Session Area

Description

Menu Bar

Provides standard menu items. See Menu Bar in Calibre RVE for LVS.

Toolbar

Provides shortcuts to commonly used commands. Additional toolbars are available when specific report tabs are selected. See Toolbars in Calibre RVE for LVS.

Navigator Pane

Provides easy navigation to the available report tabs, files and tools. Duplicates the View menu.

Terminal Window

Provides a Tcl shell (not shown in Figure 1.) To open, choose View > Terminal. The Terminal is disabled when a SPICE file is the only file open.

Report Tabs

Provide a tabbed views of results. See LVS Report Tabs and Detailed Views.

Tree View

Provides a hierarchical view of results.

Detailed View

Provides a detailed view of a selected result. See LVS Report Tabs and Detailed Views.
