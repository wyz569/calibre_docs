View Menu in Calibre RVE for PEX

This section describes the contents of the View menu that are specific to Calibre RVE for PEX.
Table 1. View Menu Contents in Calibre RVE for PEX

Menu Item

Description

Results

Dropdown list of the available results databases. Select a database to open the report tab.

Reports

Dropdown list of the available text reports and files.

Navigator

Opens the Navigator Pane.

Info

Opens the Info Pane.

Finder

Opens the Finder Tab.

Schematics

Opens the Internal Schematic Viewer.

Net Info

Opens the Info Pane to the selected net.

Parasitics on Net

Opens the detailed parasitics view to the selected net.

Only Show Parasitics Nets

A toggle selection; if disabled all nets are displayed in the parasitics table. Default: enabled.
