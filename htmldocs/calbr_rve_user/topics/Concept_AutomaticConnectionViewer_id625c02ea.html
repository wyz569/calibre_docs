Automatic Connection to a Viewer

Automatic connection between Calibre RVE and a viewer is made when you start Calibre RVE from the viewer.

See “Invoking Calibre RVE From a Layout or Schematic Viewer”.

The Connection Icon () in the lower right corner of Calibre RVE displays the connection status and includes menu items to connect, disconnect, and open the Options tab.

If the automatic connection fails, see “Manual Connection to a Viewer”.

An automatic connection to a viewer is attempted if you start Calibre RVE without connection to a viewer (standalone mode) then later attempt to highlight a result. However, this automatic connection attempt is not made if you explicitly disconnect from a viewer using Disconnect in the Connection Icon dropdown menu.
