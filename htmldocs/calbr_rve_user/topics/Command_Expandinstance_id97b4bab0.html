expandInstance

Tcl command to expand an instance in the Calibre RVE SPICE file viewer.

Usage

$spice_api expandInstance inst

Arguments

$ spice_api

The API handle for the netlist. The API handle is returned by the command GetSpiceBrowserAPI.

inst

Required argument specifying the instance name to expand. The cell name is used for a top-level cell.

Examples

$api expandInstance pllclk
$api expandInstance pllclk/XI19
