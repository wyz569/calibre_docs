View Menu in Calibre RVE for DFM

The menu items available from the View menu change based on which tab you currently have selected.

The following table describes each of the commands available from the View menu. For instructions on using each of the options available from each tab, see “Calibre RVE for DFM Report Card” in the Calibre YieldAnalyzer and YieldEnhancer User’s and Reference Manual.

Table 1. View Menu Commands in Calibre RVE for DFM

Command

Description

Browse

Calibre RVE opens the standard window, displaying the errors or capture windows from the selected window.

 

Hierarchically — Displays hierarchical results for the cells on the selected layer.

 

Flat — Displays all results after they are promoted to the top level.

Browse Errors

Generates a separate Calibre RVE window that allows you to browse errors for the selected rule.

Cell Summary

Highlights the Cell Summary tab and displays scores for the selected rule by cell.

Copy to Create

Copies SVRF code directly from the selected layer and pastes it into the SVRF Layer Operation field.

Create Layer

If you interactively specify SVRF code that contains rule checks or operations that create new layers, these layers are added directly to the DFM database. Note that any layers you create using the Create Layer button exist only in memory and are lost when you exit Calibre RVE.

Delete Layer

Deletes a layer from the database. This change is persistent on open revisions only. Deleting a layer on a frozen revision requires that the change be saved to a new revision. For more information on revisions, see “DFM Database Revisions” in the Calibre YieldServer Reference Manual.

Drill Down

Dynamically generated tab that lets you see how different rules affect a particular cell or window.

Refresh

Redraws the DFM Report Card.

View Layer

As described.

Window Summary

Displays the impact by window for all rules.
