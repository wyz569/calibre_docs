Cascading Style Sheet (CSS) File for DFM HTML Reporting

You can specify a CSS file to customize your DFM HTML report. The DFM HTML reporting tool generates a template CSS file if you do not specify a CSS file.

You specify this path using the CssFilePath key name. The template file, default.css, appears in the output directory and contains basic parameters that control the background color and table style in the summary report.

Note:

If you edit the default.css file, make sure that you specify the path to it using the CssFilePath key name. Otherwise, the tool overwrites this file the next time you run the DFM HTML reporting tool.
