PERC Info Results Tab

The PERC Info Results tab displays Calibre PERC informational rule checks. The tab is present if the SELECTTYPE INFO keyword set is included in the PERC Load statement.

Information in the PERC Info Results tab is presented in the same manner as for the PERC Results Tab.
