Setting View Options in Calibre RVE

The Options tab in Calibre RVE has option categories for the general session view, highlighting, and application-specific views.

Procedure

  1. Do one of the following to open the Options tab:

    • Click the Setup Options button () on the standard toolbar

    • Choose Setup > Options from the main menu.

  2. Choose the Session category for viewing options common to all Calibre RVE applications.

    The Setup Session Options pane has selections for setting window and tab positions, for display settings such as number precision, and for license timeout.

  3. Select other option categories as needed.
  4. Click Apply to save changes made to the options panes.
