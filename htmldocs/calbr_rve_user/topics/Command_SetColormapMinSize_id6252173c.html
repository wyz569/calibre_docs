SetColormapMinSize

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: ByCellColormap, ByWindowColormap

Optional key name that specifies the minimum pixel dimension for colormaps.

Syntax

SetColormapMinSize = integer

Parameters

integer

A required integer that specifies the minimum pixel dimension for colormaps.

Description

Can not be used in conjunction with SetColormapMaxSize.
