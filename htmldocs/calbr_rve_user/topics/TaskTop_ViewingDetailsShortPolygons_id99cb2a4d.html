Viewing Details of Short Polygons

You can use Calibre RVE for LVS to view the individual polygons that make up a short.

The highlight polygon icons in the toolbar, , allow you to highlight the previous polygon, the current polygon, or the next polygon, respectively. The following steps zoom into each polygon in the short in sequence.

Note:

In the shorts database created by Calibre nmLVS, 2×2 dbu squares are added to the polygonal paths between shorted text objects. The text object base points intersect these squares. This is to ensure text objects in the shorts database are not isolated from short paths. The squares are on the layers that the text objects are attached to, and the squares abut or overlap other shapes in a short path.

These 2×2 dbu squares are seen as the first and last polygons in the short path displayed by Calibre RVE, but do not correspond to actual layout geometries.

Prerequisites

  • Requirements described in “Interactive Short Isolation”.

  • “Viewing the Complete Short”

Procedure

  1. Clear existing highlights with Highlight > Clear Highlights or the toolbar icon.
  2. Enable Highlight > Zoom to Highlights and Highlights > Clear Existing Highlights.
  3. Select the second polygon in the short detailed view and highlight it using either the highlight icon in the short highlight toolbar or the right-click menu.

    If desired, click the column selection icon in the header of the first column of the detailed view for optional column selections.

  4. Highlight the next polygon using the Highlight Next Polygon icon .

    Note: If the short crosses cell boundaries, the detailed view will include a Cell column. If this is the case, you may want to switch between highlighting in cell context and at the top level. (See Step 4 in the section “Viewing the Complete Short”.)

  5. Continue clicking the

    icon to view each polygon in the short. This may help you determine where the short is.

    You can also use various keyboard strokes to perform these highlighting actions.

    Keyboard Key

    Action

    h, H

    Highlight current polygon

    Left arrow, p, P

    Highlight previous polygon

    Right arrow, n, N

    Highlight next polygon

    Up arrow

    Move to previous selection

    Down arrow

    Move to next selection
