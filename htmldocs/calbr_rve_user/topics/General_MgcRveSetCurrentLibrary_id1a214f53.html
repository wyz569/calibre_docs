mgc_rve_set_current_library

Specifies the first library that Calibre RVE searches in for cells to highlight to. When called without any arguments the current cell and view is used.

Syntax

mgc_rve_set_current_library(@optional (cv nil))

Arguments

  • cv — An optional argument giving a cellview.

Description

Calibre RVE uses a list of libraries to determine what libraries it looks in for the cellview to highlight results to; libraries are searched in the order specified. The function mgc_rve_set_current_library places a library at the top of existing library search list and behaves as follows depending on the arguments:

  • nil or no argument — Place the current library at the front of the library search list.

  • cv — Places the specified cellview at the front of the library search list.

The updated library search list is returned and reflected in the Setup RVE dialog box (Calibre > Setup > RVE).

You can set the full library search list with mgc_rve_set_layout_libpath and mgc_rve_set_schematic_libpath.
