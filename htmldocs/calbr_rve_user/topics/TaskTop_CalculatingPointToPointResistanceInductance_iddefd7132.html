Calculating Point-To-Point Resistance and Inductance

Calibre RVE for PEX can calculate point-to-point (P2P) resistance and inductance between selected points on a net. You can specify pins, ports, and coordinates as the end points.

Prerequisites

  • Calibre RVE for PEX is open with a SVDB that contains a net parasitics database. See “Starting Calibre RVE”.

    For point-to-point inductance calculations, the parasitics database should include extracted inductance.

  • Make the desired setting for “Use extracted PDB for calculating PEX P2P” on the Options tab, LVS/PERC/PEX category.

    “Use extracted PDB for calculating PEX P2P” is enabled by default. Unselect this setting if you want to use the reduced PDB to calculate P2P values.

  • Recommended: A layout viewer connected to the Calibre RVE session. See “Connection to a Design Tool” and the Appendix “Interfacing With Layout and Schematic Viewers”.

Procedure

  1. Double-click the net you are interested in to open the net in the detailed view.
  2. Click “Pt-to-Pt Res” in the Type column to open the Point-to-point Resistance Calculator, or “Pt-to-Pt Ind” to open the Point-to-point Inductance calculator. The Point-to-point Resistance Calculator is shown in the following figure; the interface for Point-to-point Inductance is similar.
    Figure 1. Point-to-point Resistance Calculator

    If your SVDB includes a point-to-point report (generated with the PEX Report Point2Point SVRF statement) with results for the net, those results are listed in the table.

  3. Choose “Layout” or “Source” as the source for net names in the list box.
  4. Select the end points for the calculation.

    • Pin andPort — Use the dropdown list to select an end point for the resistance or inductance calculation. Select All to calculate between all device/pin pairs on the net, or all ports.

    • Coordinate — Specify a coordinate for the point-to-point calculation. This option is only available if net names are from the layout.

      1. Select the layer using the dropdown list.

      2. Enter a space or comma-separated coordinate pair in the entry field, or click the layout icon to choose coordinates in your layout viewer. Coordinates are given in user units. The net is automatically highlighted in the layout viewer if “Highlight parasitics after calculation” is enabled in the Select P2P Locations dialog box. You may want to disable this option if highlighting takes a long time.

      Also see “Considerations When Specifying Coordinate Locations for Point-to-point Calculations” for important information.

  5. Click the Calculate button. The calculated value is reported in the table area below the controls.
  6. Right-click an entry in the table and choose the desired highlighting method from the following selections:

    Selection

    Description

    Highlight Selected

    Same as Highlight > Same Port Open

    Highlight > Same Port Open

    Ports with the same net name are not shorted.

    Highlight > Same Port Short

    Ports with the same net name are shorted.

    Highlight > Shortest Path

    (P2P Resistance only)

    Highlight only the shortest path.

    This method is useful when the net is a power or ground net, as it prevents highlighting of paths through all connected devices.

    Same Port Open and Same Port Short correspond to keywords in PEX Report Point2Point; there is no corresponding keyword for Shortest Path.

  7. Choose File > Export Parasitics > Pt-to-Pt Resistance or Pt-to-Pt Inductance to save the results. If there is an existing point-to-point report, the new calculations are added to the report.
