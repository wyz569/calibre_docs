CssFilePath

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: Common section only

Optional key name that specifies the path to a Cascading Style Sheet (CSS) file used to format the summary report (index.htm).

Syntax

CssFilePath = pathname

Parameters

pathname

A required string that specifies the path to a valid CSS file with read access.
