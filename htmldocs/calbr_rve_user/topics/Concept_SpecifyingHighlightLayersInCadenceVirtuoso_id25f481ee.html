Specifying the Highlight Layers in Cadence Virtuoso

The Setup RVE dialog box (Calibre > Setup > RVE) allows you to choose which layers Calibre RVE uses to do highlighting. The default is layers y0 to y9. You can also choose the nine hilite layers, the warning marker layer, or the error marker layer. The MGC_RVE_HIGHLIGHT_LAYERS variable (either specified as an environment or SKILL variable), can also be used to choose from among any of these four choices of highlight layers. The valid values for MGC_RVE_HIGHLIGHT_LAYERS are y0, hilite, warning, and error.
