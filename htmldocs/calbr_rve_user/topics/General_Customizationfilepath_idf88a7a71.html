CustomizationFilePath

Key Value for DRC HTML Report Configuration File, customization category

Usage: Optional. Valid in Common section only.

Inheritance: Not applicable.

Full path to an HTML customization file. This file contains HTML div tags with predefined id’s.

Allowed Values

path

Full path to valid HTML file with read access.
