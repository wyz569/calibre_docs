Saving DRC Results Data to CSV Format from the Command Line

You can save a Calibre nmDRC results database in CSV (comma-separated value) format from the command line, without displaying the Calibre RVE for DRC interface.

Prerequisites

  • A Calibre nmDRC results database in ASCII format.

Procedure

  1. Use the following command at the shell:

    %calibre -rve -drc drc_db_file -output_csv csv_out.csv

    (Optional) Include one or both of the following switches to expand the CSV output:

    • -write_check_text — Outputs check text data

    • -write_top_coords — Outputs cell transformation details and the transformed results expressed in top level coordinates.

    Including both switches results in the following command at the shell:

    %calibre -rve -drc drc_db_file -output_csv csv_out.csv -write_check_text
       -write_top_coords

  2. Open csv_out.csv in a standard spreadsheet program that can read CSV files.

Results

All results are written to the CSV file, with coordinates written in cell space. The Calibre RVE calculated properties #SC, #EL, #PP, and so on, are not included. The Calibre RVE GUI is not displayed.

Only check text within the DRC results database is processed—any Check Text Override (CTO) file associated with the database is not processed.
