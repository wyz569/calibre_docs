LVS Runtime Errors and Warnings Dialog Box

To access: Click the LVS Runtime Errors button in the top area of the result tab.

The LVS Runtime Errors and Warnings dialog box displays errors and warnings from the Calibre nmLVS run. The dialog box is displayed automatically at startup if “Show LVS/PERC runtime errors and warnings on startup” is enabled in the LVS/PERC/PEX category of the Options tab.



Objects

Object (type)

Description

Show Warnings (checkbox)

  • Enabled (default) — Display both errors and warnings.

  • Disabled — Display only errors.

The “Show Warnings” checkbox is unavailable if there are only warnings or only errors from the run.

Headline (display region)

Displays the errors and warnings (if enabled). Click an item to view message details in the lower display area.

Proceed (button)

Click to dismiss the dialog box and view results in Calibre RVE.

Usage Notes

The dialog box displays warnings if only warnings are present, regardless of the setting for the “Show Warnings” checkbox.

The setting for the “Show Warnings” checkbox is saved to the .rvedb file.

You can specify a custom message that is added to the end of the warning or error message with the option “Specify custom message to be appended at the end of LVS/PERC runtime errors” on the LVS/PERC/PEX category of the Options tab.
