ColorSet

Key Value for DRC HTML Report Configuration File, histograms and colormap category

Usage: Optional. Valid in DRCHistogram and DRCColormap report types, and in Common section.

Inheritance: Standard Inheritance

Specifies a pre-defined color palette to use for histograms and colormaps.

Allowed Values

default

Use the default color palette. (default)

colorset

Use a pre-defined color palette, where colorset is one of the following:

  • heatmap 1

  • heatmap 2

  • greyscale

  • redscale

  • bluescale

custom

Use a custom color set specified by CustomColors.
