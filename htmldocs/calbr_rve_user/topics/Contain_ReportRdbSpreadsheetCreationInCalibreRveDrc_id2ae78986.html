Report, RDB, and Spreadsheet Creation in Calibre RVE for DRC

You can create an HTML formatted report of the results that are displayed in Calibre RVE for DRC or save the results data to a CSV (comma-separated value) formatted file. HTML reports make it easy to share results between groups. A CSV formatted file can be imported into most spreadsheet tools for later analysis.
