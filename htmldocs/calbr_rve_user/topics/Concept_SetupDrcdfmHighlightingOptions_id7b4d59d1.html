Setup DRC/DFM Highlighting Options

You can set options for showing layers and properties on the DRC/DFM Highlighting Option pane.

Refer to “DRC/DFM Highlighting Options” for a description of the options.

Also see the option “Skip contact/via layers while highlighting” located on the Options tab, Highlighting category, in the “Layout Highlighting” area. This option applies when highlighting connectivity objects (nets, devices, instances, and pins). When selected, contact layers appearing in a Connect … BY statement are not included in highlighting actions.
