Use Rule Check Name for Highlight Layer with Calibre DESIGNrev

Calibre DESIGNrev labels the highlight layer with the rule check name if you selected “Colors” in the Set Highlight Layers Dialog Box of Calibre RVE for DRC. This selection assigns a highlight layer color to each rule check and all errors from a given rule check are highlighted to the specified color.

See “Set Highlight Layers Dialog Box” for complete information.
