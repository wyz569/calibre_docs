GroupBy

Key Value for DRC HTML Report Configuration File, report specification category

Usage: Required in all sections unless LinkFrom is used. Valid in Common section.

Inheritance: Allowed in report sections other than DRCResultsSummary

In the DRCResultsSummary report, GroupBy specifies the grouping hierarchy for the report. In all other reports, one report is output for each GroupBy criteria.

If LinkFrom is used in a section, the GroupBy key is not required; see LinkFrom for information.

Allowed Values

In all sections:

h_level[/h_level]…

Hierarchy specification consisting of any combination of Check, Cell, and property names separated by the ‘/’ character. For example: GroupBy = Check/Cell

In sections other than DRCResultsSummary:

[section_name]

Indicates key value inheritance from section_name, where section_name specifies a section with a DRCResultsSummary report.

[section_name] index

Where index is one-based index to the GroupBy hierarchy specified in section_name. This specification functions as with [section_name], except that the GroupBy hierarchy is extended only to the index level.

[section_name]/hier_extension

Where hier_extension specifies a valid extension to the GroupBy hierarchy specified in section_name.

In a DRCHighlight section only:

Overview

Specifies that highlights from all cells are grouped together in a single snapshot; the HighlightInContext and ShowHighlightsInTopCell parameters must also be set to 1. See the Examples section.

Examples

Example 1

Group first by the hierarchy specified in the “summary” section, then group by the area property. This example is not valid in a DRCResultsSummary section.

GroupBy = [summary]/area

Group first by Check, then by Cell. This example is valid in any section. 

GroupBy = Check/Cell

Example 2

This example creates an overview snapshot. 

[Common]
LayoutWidth = 200
LayoutHeight = 200

[Overview Snapshot]
ReportType = DRCHighlight
GroupBy = Overview
HighlightInContext = 1
ShowHighlightsInTopCell = 1
ZoomLayoutFactor = fit
WorldView = 0
HideNonHLLayers = 1
CheckHLLayer = Colors
