Calibre RVE with Calibre DESIGNrev and Calibre WORKbench

Choose Verification > Start RVE to start Calibre RVE from Calibre DESIGNrev or Calibre WORKbench.
Note:

If a highlight, colormap, or export to layout operation takes longer than one second, Calibre RVE displays a progress dialog with a Cancel button. Since Calibre RVE communicates with Calibre DESIGNrev directly across a socket, some shapes may still appear in Calibre DESIGNrev from a canceled operation. These shapes must be removed manually.
