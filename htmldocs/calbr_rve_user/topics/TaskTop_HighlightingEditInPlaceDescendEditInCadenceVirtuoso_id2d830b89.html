Highlighting with Edit-In-Place and Descend Edit in Cadence Virtuoso

The Calibre RVE setting “Edit-in-place while highlighting” in the Setup RVE dialog box controls whether Calibre RVE highlights in the open window or opens a new window for each highlight. The Cadence Virtuoso setting for edit-in-place or descend edit has no effect on the highlighting behavior in Calibre RVE.

Procedure

  1. In Cadence Virtuoso, choose Calibre > Setup > RVE to open the Setup RVE dialog box.
  2. Select the desired behavior for “Edit-in-place while highlighting”:

    • Enabled — Calibre RVE highlights to the open window in Cadence Virtuoso. Calibre RVE automatically ascends or descends the hierarchy if necessary in order to highlight in the correct cell.

    • Disabled — Calibre RVE opens a new window for the highlight action if the cell and view for the highlight is not open as the top cell.

  3. Set other options as desired; see “Setup RVE Dialog Box in Cadence Virtuoso”.
  4. Click OK to close the dialog box.
