mgc_enc::openRDBFileInRVE

Opens an ASCII format results database (RDB) in a running Calibre RVE session if one exists. If there is not a running Calibre RVE session, the RDB is opened in a new session. The command is entered at the Cadence Encounter command prompt.

The command opens the RDB file in an existing Calibre RVE session only if that session was started using Calibre > Start RVE from the Cadence Encounter menu or a previous mgc_enc::openRDBFileInRVE or mgc_enc::openDBInRVE command. If the Calibre RVE session was started from Calibre Interactive, the mgc_enc::openRDBFileInRVE command opens the RDB in a new Calibre RVE session.

Syntax

mgc_enc::openRDBFileInRVE rdbFileName

Arguments

  • rdbFileName — A required argument specifying the path to an ASCII format results database file.

Examples

For example, enter the following in the Cadence Encounter command prompt to open the RDB M1.5_result.rdb in an existing Calibre RVE session:

encounter 2> mgc_enc::openRDBFileInRVE M1.5_result.rdb
