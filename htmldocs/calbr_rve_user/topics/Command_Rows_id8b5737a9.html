Rows

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: CellDrillDown, CellSummary, ChipSummary, NetDrillDown, NetSummary, TotalSummary, WindowDrillDown, WindowSummary

Optional key name that specifies the number of rows to return for the section.

Syntax

Rows = integer

Parameters

integer

A required integer greater than 0 (zero) that specifies the number of rows. The default value is an asterisk (*), which includes all rows.

If an invalid value is specified, no reports are generated for the section. Can indirectly reference other sections.
