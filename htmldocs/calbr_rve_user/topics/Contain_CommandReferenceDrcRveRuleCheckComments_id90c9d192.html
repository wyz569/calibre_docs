Command Reference for DRC RVE Rule Check Comments

DRC RVE rule check comments provide additional control over the Calibre RVE display and highlighting actions. The rule check comments are placed in your DRC Results database or a check text override (CTO) file and interpreted by Calibre RVE for DRC.

See “DRC Rule Check Comments for Calibre RVE” for general information on using DRC RVE rule check comments.

CTO files can be generated automatically; see “Creating a Check Text Override (CTO) File from the Rule File”.

The syntax conventions used for the rule check comment descriptions are listed in “Syntax Conventions”.

  • RVE Highlight Index
  • RVE Highlight Color
  • RVE Highlight Text Size
  • RVE Link
  • RVE Priority
  • RVE Show Layers
  • RVE Show Properties
  • RVE Waive
  • RDB
