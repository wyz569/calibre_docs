Calibre::openRDBFileInRVE

Opens the specified ASCII results database (RDB) in a running Calibre RVE session, if one exists. If there is not a running Calibre RVE session, the RDB is opened in a new session. The command is entered in the Synopsys IC Compiler or IC Compiler II command prompt.

The command opens the RDB file in an existing Calibre RVE session only if that session was started using Calibre > Start RVE from the Synopsys IC Compiler or IC Compiler II menu or a previous Calibre::openRDBFileInRVE command. If the Calibre RVE session was started from Calibre Interactive, the Calibre::openRDBFileInRVE command opens the RDB in a new Calibre RVE session.

Syntax

Calibre::openRDBFileInRVE rdbFile

Arguments

  • rdbFile — A required argument giving the path to an ASCII format results database.

Examples

Enter the following at the icc_shell or icc2_shell command prompt to open the RDB drc.results in a existing Calibre RVE session:

icc_shell> Calibre::openRDBFileInRVE drc.results
