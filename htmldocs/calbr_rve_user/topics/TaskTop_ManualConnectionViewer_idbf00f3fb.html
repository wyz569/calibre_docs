Manual Connection to a Viewer

Although automatic connection between Calibre RVE and a viewer is usually more convenient, you can also manually set the socket number and start the socket connection. When using Calibre RVE for DRC, you can connect to two layout design tools.

Manual connection to a viewer is needed in these cases:

  • You open the viewer after starting the Calibre RVE session.

  • Calibre RVE is unable to use the default socket number for communicating with the viewer. The default socket number is 9189 for layout viewers and 9199 for schematic viewers.

  • You want to highlight to two layout design tools with Calibre RVE for DRC.

Tip

The Connection Icon () in the lower right corner of Calibre RVE displays the connection status and includes menu items to connect, disconnect, and open the Options tab.

Procedure

  1. Open your layout or schematic viewer.
  2. Activate the socket in the viewer(s).

    • Calibre DESIGNrev — Choose Verification > RVE/CI Setup from the main menu. In the Server section of the dialog box, enter a socket number and click Start.

    • Cadence Virtuoso — Choose Calibre > Setup > Socket from the Cadence Virtuoso main menu. Enter the socket number and click OK.

    • Other viewers — Activate a socket following the instructions in the user’s manual for the viewer.

  3. In Calibre RVE, choose Setup > Design Tools.
  4. For a layout design tool, enable “Layout Design Tool.”
    1. Enter the host name and socket number for the viewer.
    2. Click the connect button () to start the connection.

    If the connection is successful, the design tool information is displayed in the “Tool Information” field. However, Synopsys Laker, unlike other layout tools, does not report the viewer name to back to Calibre RVE, so the most recently connected viewer or the default viewer is displayed as the design tool.

  5. (Optional, for DRC only) Check “Secondary Layout Design Tool (DRC Only)” to connect to a second layout viewer. This is useful for highlighting to a viewer such as Calibre DESIGNrev for debugging and to a different layout design tool for actual editing.
    1. Activate a socket in the second layout design tool, as described in Step 2.
    2. Enter the host name and socket number for the secondary layout viewer.
    3. Click the connect button () to start the connection.

    Standard highlight actions are made to both layout tools; however the highlight, waive, fix, and select in area actions only apply to the first connected layout tool. Also see “Debugging DRC Results Using Two Layout Design Tools”.

  6. For a schematic design tool, enable “Schematic Design Tool.”
    1. Select the company and design tool using the radio buttons and dropdown menu.
    2. Do one of the following, depending on your schematic design tool:

      If …

      Do this …

      Schematic and layout tool share a socket

      Check “Use the same connection as the layout tool.”

      Schematic and layout tool are independent and do not a share socket

      Uncheck “Use the same connection as the layout tool” and enter the hostname and socket number for the schematic tool.

      Tip: Cadence Virtuoso and Cadence Composer are examples of layout and schematic tools that share a socket connection.

    3. Click the connect button () to start the connection.
  7. Click Apply in the left panel to save changes to the socket or host information.
