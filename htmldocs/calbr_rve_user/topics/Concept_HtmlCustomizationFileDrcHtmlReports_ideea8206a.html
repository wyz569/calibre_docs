HTML Customization File for DRC HTML Reports

The HTML customization file allows you to add custom HTML code, such as text, images, or links, to the DRC HTML report. The customization file uses HTML div tags to add a header and footer to the DRC HTML report.

The following table contains the ID’s that you can assign to div tags in the file.

Table 1. Customization File div Tags

div ID

Description

Report.CustomReportHeader

Everything contained in this division is included in the DRC HTML report between the Calibre header the main report index table.

Report.CustomReportFooter

Everything contained in this division is included in the DRC HTML report as a footer after the final section report.

SectionName.CustomSectionHeader

Everything contained in this division is included before the title of the section specified with SectionName.

The CustomizationFilePath key name specifies the path to the customization file; the key is placed in the Common section of the configuration file. You can also specify the customization file when creating a HTML report using Calibre RVE; see “Creating a DRC HTML Report from Calibre RVE for DRC”.

Example

The following customization file example places text in the header and footer of the DRC HTML report. It also places a CustomSectionHeader before a section named “histogram”. 

<div id="Report.CustomReportHeader">
<h2>DRC batch report for design XYZ</h2>
</div>

<div id="Report.CustomReportFooter">
<h1>Company Confidential</h1>
</div>

<div id="histogram.CustomSectionHeader">
<h2>Custom Header for histogram section</h2>
</div>

A custom Cascading Style Sheet (CSS) can be used to define formatting for the custom header and footer. The following code can be added to the default CSS file; see “Using a Custom Cascading Style Sheet (CSS) in DRC HTML Reporting”. 

 #Report\.CustomReportHeader
 {
  color: blue;
 }
 #Report\.CustomReportFooter
 {
  text-align: center;
 }
 #histogram\.CustomSectionHeader
 {
  color: blue;
 }

See “Example for DRC HTML Reporting” for a view of a DRC HTML report created with this customization file.
