Example 9: Chip Summary Report

This configuration file example creates a ChipSummary batch report.

The reports is filtered for rows as follows:

Group = DIFF and Priority = P1 and Lambda_ANF >= 0.01 and WCA > 130

[ChipReports]
ReportType = ChipSummary
GroupsToInclude = DIFF
PrioritiesToInclude = P1
FilterMetric = Lambda_ANF, WCA 
FilterOperation = >= , >
FilterValue = 0.01, 130
