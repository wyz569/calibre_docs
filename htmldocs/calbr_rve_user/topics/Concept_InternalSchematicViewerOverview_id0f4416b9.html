Internal Schematic Viewer Overview

Calibre RVE for LVS uses schematic rendering technology to provide schematic viewing for both source and layout. Schematic comparison allows you to quickly see the cause of a LVS comparison error. The schematic window includes search capability and netlist display. The Internal Schematic Viewer obeys the case-sensitivity specified by the Layout Case and Source Case statements in the rule file.

Highlighting an element in the main session window automatically causes the element to be highlighted in all schematic views: the rendered schematic, the netlist display, and the hierarchy tree. Highlighting in the internal schematic viewer obeys the highlight settings zoom, pan, and keep view; however, if pan or zoom is set, the highlighted element is centered in the schematic.

Tip

To easily find a highlighted element within a dense display, click the Mute Colors () button in the top left corner of the schematic display. All schematic elements except the highlighted element are displayed in gray.

Figure 1. Internal Schematic Viewer

You enable and disable the Internal Schematic Viewer with View > Schematics and the “Show netlist schematics on highlighting” and “Show Schematic Pane” settings on the Options tab in the Schematic Viewer category.

The Internal Schematic Viewer performs logic gate recognition by default, and displays schematic instances as the equivalent logical gate. To disable this feature, unselect “Automatically recognize gates in sub-circuit with fewer than __ nets” on the Options tab in the Schematic Viewer category.

Tip

If the schematic view appears unresponsive, check for a pop-up message hidden behind the session window and close it.

You can also open a SPICE file for viewing independent of Calibre RVE for LVS⁄PERC⁄PEX; see “Calibre RVE SPICE File Viewer”.
