Internal Schematic Viewer Toolbar

The Internal Schematic Viewer toolbar has buttons for common options and actions.


Table 1. Internal Schematic Viewer Toolbar Contents

Icon

Description

Set schematic view options:

  • Schematics — Display the rendered schematic.

  • Device Properties — Display device properties in the schematic view.

  • Draw Power/Ground — As described.

  • Bulk Nets — Display bulk nets in the schematic viewer.

  • Print Schematic — Opens the Print Schematic dialog box, from which you can print the schematic view. Enable “Print to File” to save to file instead of printing. After you click the Print button when printing to file, you have the option to select a file type of post script (*.ps) or scalable vector graphics (*.svg).

Also see option descriptions in “Setup Schematic/Netlist Viewer Options Pane”.

Set display and sort options for the hierarchy tree.

Note: if the hierarchy tree view is turned off, then the different schematic views are arranged in a tabbed format.

Set netlist view options. Select from:

  • Netlist Text — turns the netlist display on and off.

  • Line Numbers

  • Wrap Lines

  • Status Bar

Erase highlights.

Dropdown list:

  • Clear Highlights — clear all highlights.

  • Clear Selections — clear local highlights only (not in attached viewers).

Set the select environment options. Select from:

  • Go to Select Environment

  • Clear Select Environment

  • Add Search Results to Select Environment

 

The select environment applies to the instances in the netlist, and allows you to extend net selection (highlighting) to equivalent nets in a referenced subcircuit. See Select Environment in the Internal Schematic Viewer and Figure 1 in “Selecting and Highlighting Equivalent Nets Using the Select Environment”.

Open or close the Internal Schematic Viewer Find Bar.

Schematic view history. The arrows scroll through the schematic views that have been displayed.
