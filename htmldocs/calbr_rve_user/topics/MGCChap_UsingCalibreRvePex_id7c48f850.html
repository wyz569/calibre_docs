Using Calibre RVE for PEX

This chapter describes how to use the Calibre Results Viewing Environment (RVE) for PEX. It covers the key features of the tool and a run-through of common tasks using the net parasitics table. A reference section describes each of the GUI elements.

This chapter assumes that you have a PEX database open in Calibre RVE, and are familiar with the general operation of Calibre RVE. See these sections for general information on Calibre RVE:
