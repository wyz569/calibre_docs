Saving a Subset of DRC Results as an RDB

You can save selected results from a Calibre nmDRC results database as a new RDB. You can use a filter to select the results that are saved or use Ctrl- and Shift-click to select the results. The new RDB can be used to share results or as input to DRC HTML reporting.

Prerequisites

  • A DRC results database open in Calibre RVE for DRC.

Video



Tip: Alternatively, you can use Calibre DESIGNrev to save the shapes on a layer as an RDB. See the topic “Saving Highlighted Shapes as a Calibre nmDRC Results Database (RDB)” in the Calibre DESIGNrev Layout Viewer’s Manual, which includes a video.

Procedure

  1. Do one of the following to select the results you want to save:

    • Click the filter button to apply a DRC Result Filter. You can also click the Bookmarks icon () and select a bookmarked view that includes a DRC Result Filter.

    • Manually select specific results to save using Shift- and Ctrl‑click.

  2. Select File > Save Results As to open the Save Results dialog box.
  3. Enter a filename for the new RDB.
  4. Select a file type of “RDB.”
  5. Select the results you want to save in the “Save” dropdown list:

    • Displayed Results — Save all results in the Calibre RVE display. If you applied a filter, only results that satisfy the filter are saved.

    • Selected Results — Save only the results selected with Shift- and Ctrl-click.

  6. (Optional) Enable “Open after Save” to open the new RDB after it is created.
  7. Click Save.

Results

An ASCII formatted results database (RDB) is created. You can use the RDB in any application that accepts Calibre ASCII formatted results databases. For example, you can share the file with a team member so they can view the results in Calibre RVE or you can create a DRC HTML report from the RDB.
