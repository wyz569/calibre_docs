GetSpiceBrowserAPI

Tcl command that returns the API object for a netlist that is loaded in the Calibre RVE SPICE file viewer.

Usage

::rve::SpiceBrowserAPI::GetSpiceBrowserAPI netlist

Arguments

netlist

The handle for a loaded netlist.

Return Values

The API object for the netlist.

Examples

set api [::rve::SpiceBrowserAPI::GetSpiceBrowserAPI [lindex $netlists 0]]
