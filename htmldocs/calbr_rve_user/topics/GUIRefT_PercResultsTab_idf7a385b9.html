PERC Results Tab

The PERC Results tab is available if your Calibre PERC run produced a Mask SVDB Directory.

Usage Notes

If the results include locations, then the PHDB (persistent hierarchical database) is loaded and the locations appear as clickable links in the Result Data Pane, with the locations translated to be relative to the top cell. However, if a result has multiple placements, the locations reported in the display are not correct; see “Viewing Separate Placements of a Result in the PERC Results Tab” for more information. If there is no PHDB, locations do not appear as clickable links in the Result Data Pane. See the Mask SVDB Directory statement for more information on the PHDB.

Some Calibre PERC analysis flows generate multiple netlists with flattened hierarchy and the analysis is done on the flattened netlist. Calibre RVE for PERC recognizes this, and uses the hierarchical extracted layout netlist in the internal schematic viewer for display purposes.

Note:

It is recommended that the version of Calibre used to create the results database match the version of Calibre RVE used to view the results. Some information may not be present if you use the current version of Calibre RVE to view an older results database.
