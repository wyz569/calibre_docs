printSchematicToFile

Tcl command to output the schematic view to a file.

Usage

$spice_api printSchematicToFile output_file
[-colormode {bg_white | bg_black | mono}]
[-orientation {auto | landscape | portrait}]
[-aspect {full | view}]
[-filetype {.ps | .svg}]

Arguments

$ spice_api

The API handle for the netlist. The API handle is returned by the command GetSpiceBrowserAPI.

output_file

Required argument that specifies the output file name. Filenames that end in .ps are saved in postscript format; filenames that end in .svg are saved in Scalable Vector Graphics format.

-colormode {bg_white | bg_black | mono}

Optional switch and argument specifying the color scheme.

  • bg_white — (default) Use a white background. This option inverts the colors used in the interactive version of the Calibre RVE SPICE file viewer.

  • bg_black — Use a white background. This option uses the same colors as the interactive version of the Calibre RVE SPICE file viewer.

  • mono — Use black and white only, with a white background.

-orientation {auto | landscape | portrait}

Optional switch and argument specifying the orientation. The auto setting chooses the best fit orientation.

-aspect {full | view}

Optional switch and argument specifying the view.

  • full — print the full design

  • view — print what's currently visible

-filetype {.ps | .svg}

Optional switch and argument to force the format of the output file.

  • .ps — postscript format

  • .svg — Scalable Vector Graphics format

Examples

Print to a postscript file named capture.ps. Default options are used for color and orientation. 

$api printSchematicToFile capture.ps

Print to a SVG file with a black background. 

$api printSchematicToFile capt_black_bg.svg -colormode bg_black
