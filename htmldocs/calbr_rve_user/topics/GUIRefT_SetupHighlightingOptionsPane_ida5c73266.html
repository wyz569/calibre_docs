Setup Highlighting Options Pane

Access with: Setup > Options, then choose the Highlighting category

Use this pane to select highlighting options that are common to all Calibre RVE applications.

Fields

Table 1. Setup Highlighting Options Pane Contents

Item

Comment

On New Highlighting

No view change

Keep the same view when highlighting.

Pan view to highlights

Shift the view when highlighting, but keep the same zoom factor.

Zoom to highlights by ___ (select Ratio or User Units in dropdown list.)

Enter a positive number for the zoom control.

Ratio — The size of zoom window is the ratio of the highlighted area to the zoom control. A number less than one will result in a zoom window larger than the highlighted area. Default is 0.7.

User Units — The size of the zoom window is the highlighted area plus a halo given by the zoom control in user units.

Clear existing highlights before showing new highlights

Automatically clear existing highlights before showing results of next highlight operation.

Clear highlights in design tools when RVE exits

As described.

Layout Highlighting Options

Split polygons with more than _________ vertices

This is useful for layout viewers that have a maximum number of vertices for polygons that are displayed. Default is 4000.

Magnify highlight coordinates by _________

As described.

Displace highlight coordinates in

  • Top Cell

  • All Cells

Select this option to add a displacement of (DeltaX, DeltaY) in user units to coordinates from Calibre RVE while highlighting in the layout viewer.

Rotate highlight shapes in

  • Top Cell

  • All Cells

(Only supported in Calibre RVE for DRC)

Select this option to rotate a shape counterclockwise by the specified angle. You can enter a floating-point number, or use the scroll arrows to increment the value by 90 degrees.

Select “Reflect on X axis” to flip the shape about the x axis.

Highlight Marker Size: _____ user-units

Sets the marker size. The default is 4.

Highlight Text Size:

_____ user-units

Sets the text size. Default is 0.1.

If you are using the Calibre DESIGNrev layout viewer, see “Setting Text Size for Highlighting with Calibre DESIGNrev” for additional settings.

Skip contact/via layers while highlighting

 

(Enabled by default. Valid for all Calibre RVE applications except DRC.)

When selected, contact layers appearing in a Connect … BY statement are not included in highlighting actions. See Layer Filters and Colors on the Highlights Tab and the Highlights Tab for a complete description.

 

This option is on by default.

Display connectivity object name in highlight layer labels

For Calibre DESIGNrev/WORKbench layout viewers only. The highlight layer label is of the form object_name:layer_name; otherwise only layer_name is used.

Usage Notes

When using the Internal Schematic Viewer, if “Pan view to highlights” or “Zoom to highlights” is selected, this corresponds to centering the element in the Internal Schematic Viewer on a highlight event. No View Change is obeyed.
