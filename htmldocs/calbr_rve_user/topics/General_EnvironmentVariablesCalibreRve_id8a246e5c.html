Environment Variables for Calibre RVE

Several environment variables are available to control the behavior of Calibre RVE.

The following table lists environment variables used in Calibre RVE. The Type column indicates whether the environment variable is available in the shell environment (env) or in SKILL scripts.

Table 1. Calibre RVE Environment Variables

Variable

Type

Description

CALIBRE_HOME

env

Specifies the software tree invoked when Calibre Interactive or Calibre RVE is called from Pyxis.

Also sets the location of the software tree for Calibre tools; see “Setting the CALIBRE_HOME Environment Variable” in the Calibre Administrator’s Guide.

MGC_CALIBRE_DB_DIR

env

Sets the Calibre RVE working directory (“Specifying a Calibre RVE Working Directory”).

MGC_CALIBRE_DISABLE_GUI_TERMINAL

env

Disables the Terminal pane, preventing access in the Calibre RVE window. The controls View > Terminal and Setup > Options > Session > Show Terminal Pane are disabled, and the cmnShowCommandPrompt runset option is not honored.

MGC_CALIBRE_DISABLE_STRIP_REPLICATED_DEVICE_SUFFIX

env

Calibre RVE removes the suffix string “==N” when highlighting replicated devices to an external schematic tool. Devices are replicated if LVS Spice Replicate DevicesYES is in the rule file.

MGC_CALIBRE_ECHO_VIEWER_MENU_CMDS

env

When set, specifies to echo custom menu commands to the layout viewer’s transcript; see “Creating Custom Menus in Design Tools” in the Calibre Interactive User’s Manual.

MGC_CALIBRE_ENABLE_ONM

env

See “Map Cadence Netlist Names to Schematic Names for Highlighting from Calibre RVE”.

MGC_CALIBRE_LAYOUT_SERVER

env

Specifies a hostname and communications socket port numbers for Calibre RVE and Calibre Interactive to use for communication with a connected layout viewer (“Setting the Calibre RVE Socket Port for a Layout Viewer”).

MGC_CALIBRE_LAYOUT_SERVER_NAME

env

This environment variable is set automatically by the tool to the type of layout viewer. The allowed values are the same as those for the cmnLayoutServerType runset variable.

MGC_CALIBRE_LAYOUT_TMP_FILE

env

Sets the name of the intermediate file used in layout editor communication. See “Highlight data file” in “Setup Design Tools and Connection to RVE Pane”.

MGC_CALIBRE_MRU_SIZE

env

Sets the list length of the “most recently used” lists in Calibre Interactive, Calibre Interactive xACT, and Calibre RVE. The environment variable also controls the list length for the Recent Runsets and Recent Text Files menu selections.

The default list length is 20. The maximum length is 50.

MGC_CALIBRE_NO_PIPO_WARNING_CHECK

env or SKILL

When enabled, the check for duplicate cell names during layout export from Cadence Virtuoso is disabled; see “Layout Export in Cadence Virtuoso” in the Calibre Interactive User’s Manual.

MGC_CALIBRE_PRESERVE_MENU_TRIGGER

env or SKILL

Set this variable to preserve other menu triggers registered before Calibre. Used when setting up the Calibre menu for Cadence Virtuoso. (“Creating an Interface to Cadence Virtuoso”).

MGC_CALIBRE_RVE_NAME_DELIMITER

env

Sets the cell name delimiter for Calibre RVE.

If the variable is set to one character, that character is used to enclose names. If the variable is set to two characters, the first character delimits the start of the name, and the second character delimits the end of the name (See “Defining Name Delimiters in Calibre RVE” for the default settings).

MGC_CALIBRE_SCHEMATIC_SERVER

env

Specifies a hostname and communications socket port numbers for Calibre RVE and Calibre Interactive to use for communication with a connected schematic viewer (“Setting the Calibre RVE Socket Port for a Schematic Viewer”).

MGC_CALIBRE_SCHEMATIC_SERVER_NAME

env

This environment variable is set automatically by the tool to the type of schematic viewer. The allowed values are the same as those for the cmnSchematicServerType runset variable.

MGC_CALIBRE_SEARCH_LIB_ORDER

env

Specifies whether to place newly opened libraries at the beginning or end of the library search list for Cadence Virtuoso. This affects the fields “Libraries RVE should look in for: Layout Cells” and “Libraries RVE should look in for: Schematic Cells” in the Setup RVE Dialog Box in Cadence Virtuoso. To prepend newly opened libraries to the list, set the environment variable to PREPEND; this is the default. To append, set the environment variable to APPEND.

MGC_CALIBRE_VIEWER_MENU_CMDS

env

Specifies the path to a menu customization file; see “Creating Custom Menus in Design Tools” in the Calibre Interactive User’s Manual.

MGC_RVE_CTO_FILE

env

Specifies the path to a DRC RVE Check Text Override File (CTO File) Format.

MGC_RVE_DB_CLOSE_TRIGGER

env

Sets a trigger to run in Calibre RVE for DRC after a database has been closed.

MGC_RVE_DB_OPEN_TRIGGER

env

Sets a trigger to run in Calibre RVE for DRC after a database has been loaded.

MGC_RVE_DISABLE_CLEAR_HILIGHT

env

By default, this environment variable is not set. In this case, Calibre records all highlighted objects, instead of clearing all highlighted objects. This recording operation enables Calibre RVE to selectively clear highlights for fixed or waived results.

When set, no recording of highlighted objects occurs; in this case, Calibre RVE is unable to selectively remove fixed or waived results highlights. Be aware that the recording operation slows down the performance of highlighting.

MGC_RVE_DISABLE_TURBO

env

Disable the automatic inclusion of the ‑turbo option for Calibre RVE when invoked from Calibre Interactive or a supported layout editor (“Multithreaded Parallel Processing for Calibre RVE Operations (‑turbo Option)”).

MGC_RVE_HIGHLIGHT_LAYERS

env or SKILL

Sets the use of warning or error layers by Calibre RVE in Cadence Virtuoso (“Specifying the Highlight Layers in Cadence Virtuoso”).

MGC_RVE_HTML_BROWSER

env

Specifies the HTML browser executable to use when viewing links created with the RVE Link rule check comment or when opening the DRC HTML Report (see “Creating a DRC HTML Report from Calibre RVE for DRC”).

Also see MGC_RVE_PDF_VIEWER.

MGC_RVE_INIT_SOCKET_AT_STARTUP

env

Sets socket initiation preference for Cadence Virtuoso (“Table 2”) and for Synopsys IC Compiler and IC Compiler II (“Setting Socket Connections with Synopsys IC Compiler and IC Compiler II”).

MGC_RVE_LAYOUT_CASE_FOLD

env or SKILL

Sets the case sensitivity behavior of highlight cells by Calibre RVE in Cadence Virtuoso (“Case Sensitivity in Calibre RVE with Cadence Virtuoso”).

MGC_RVE_NO_DUAL_DB

env

When defined, Calibre RVE does not strip trailing “$D1$” and “$D2$” characters from cell names when sending highlight information to the layout editor. Dual database runs in Calibre nmDRC add “$D1$” and “$D2$” to the end of cell names, and by default Calibre RVE strips these trailing four characters from cell names. This behavior can also be set with the “Strip dual database cell name suffixes while highlighting” option in Calibre RVE. (“Handling Cell Names with $D1$ and $D2$ from DRC Dual Database Runs”)

MGC_RVE_PDF_VIEWER

env

Specifies the path to the PDF viewer executable to use when opening a PDF document, where the link to the PDF document was created with the RVE Link rule check comment.

MGC_RVE_PRE_OPEN_TRIGGER

env

Sets a trigger to run in Calibre RVE for DRC immediately before loading a database.

MGC_RVE_RAISE_WINDOW

env or SKILL

If set to any value, the window the highlight is made in is raised to the front if it is not already at the front. In SKILL, the value must not be nil. (“Overview of Highlighting in Cadence Virtuoso with Calibre RVE”).

MGC_RVE_RELEASE_LICENSE_TIME

env

Sets the time after which the Calibre RVE license is released if there is no activity (“License Timeout in Calibre RVE”). The default time unit is hours.

MGC_RVE_SHOW_TRUE_RES_LOCS

env

By default, Calibre RVE for PEX highlights of the actual node locations of resistors.

For 2012.1 and earlier releases, highlighting used resistor node locations after removal of shorts and reduction. To revert to the old behavior, set this environment variable to 0.

MGC_RVE_SOURCE_CASE_FOLD

env or SKILL

Sets the case sensitivity behavior of highlight cells by Calibre RVE in Cadence Virtuoso (“Case Sensitivity in Calibre RVE with Cadence Virtuoso”).

MGC_RVE_USE_MARKERS (deprecated)

env or SKILL

Sets the use of warning or error layers by Calibre RVE in Cadence Virtuoso.

MGC_RVE_USE_USER_DB_OPTIONS

env

You can disable the saving of database-specific options by setting this environment variable to 0. See “Database-Specific Settings in Calibre RVE”.

MGC_RVEDB_DIR

env

Sets the directory path for the .rvedb File for Calibre RVE Settings. You can define multiple directories separated by a colon; See “Multi-Level Configuration Management in Calibre RVE”.

MGCCALIBREMENUVIEWTYPE

SKILL

Specifies window type in which to load a Calibre menu (“Creating an Interface to Cadence Virtuoso”).

OA_HOME1

env

Sets OpenAccess library version to one other than the default. The OpenAccess default library that ships with Calibre is located at . ⁄shared⁄pkgs⁄icv_o in your Calibre software tree. See “Specifying Third Party Database Options in Calibre Interactive” in the Calibre Interactive User’s Manual for more information.

Note: As of the 2013.3 release, the supported OpenAccess version in Calibre is oa22.43p006. Also see footnote.
