Select and Highlight Objects in Design Tools Using Finder

You can use the Particular Objects search in the Finder tab to select and highlight objects in an attached design tool. You can select multiple objects.

The process is illustrated in the following figure.

Figure 1. Select and Highlight Objects in Design Tools Using Finder

If you are using Cadence tools, note the following usage details:

  • When selecting objects in Cadence Virtuoso, you must press Ctrl-C to end the selection process.

  • When highlighting to Cadence Virtuoso, if “Highlight found objects” is enabled and the layout window is closed, it is possible for the system to hang. This is due to an internal Cadence Virtuoso issue and the workaround is to press Ctrl-C twice.
