Setup Schematic/Netlist Viewer Options Pane

Access with: Setup > Options, then choose the Schematic Viewer category

This option pane controls settings for the Internal Schematic Viewer in Calibre RVE for LVS⁄PERC⁄PEX.

Note:

For option panes common to all Calibre RVE applications, see “Options Tab and Common Calibre RVE Option Panes”.

Fields

Table 1. Setup Schematic/Netlist View Options Pane Contents

Setting

Description

Show netlist schematics when highlighting connectivity objects (Nets / Devices / Instances / Pins)

When enabled, the layout and source netlists are opened in the Internal Schematic Viewer when highlighting connectivity objects (nets, devices, instances, and pins).

The setting of this option is checked the first time a highlight is made for a database. If a schematic view is closed after the first highlight event, that view will not automatically open on later highlight events. You can open the schematic view again by choosing View > Schematics.

Show Schematic Pane

Show the rendered schematic in the Internal Schematic Viewer.

Show Hierarchy Pane

Show the hierarchy (tree) view in the Internal Schematic Viewer.

Show Text Pane

Show the netlist browser in the Internal Schematic Viewer.

Show transistor bulk pins

Enable to show bulk nets in the schematic view.

Draw power/ground nets connected to instance pins

Disabled — (default) Power/ground nets are not drawn when connecting to instance pins. The instance pins are drawn normally; when you click a pin the appropriate power/ground net is drawn.

Enabled — Include power/ground nets which are solely connected to instance pins in the drawn schematic.

Show device properties

Enable to show device properties in the schematic view using one of the following display conventions:

  • As in netlist

  • Use SPICE notation

  • Use Engineering notation

Hide unconnected pins on cells/instances with more than ___ pins

Enable to hide unconnected pins in cells and instances with more pins than the specified threshold value. The cell outline is shown as a dashed line if pins are hidden. Additional menu options are available when you right-click on the cell outline if this option is enabled.

All pins for the cell or instance are shown if this option is disabled. This option is on by default.

See “Hiding Unconnected Pins In the Internal Schematic Viewer” for information on using this option.

Detect pin directions in sub-circuit with fewer than _____ pins

Pin direction is detected in schematic instances with fewer than the specified number of pins. Existing schematics must be reopened for this setting to take effect. Schematic display times for large circuits can increase significantly if this value is set too high.

Automatically expand sub-circuit with fewer than _____ nets

Schematic instances containing fewer than the specified number of nets are automatically expanded when any of the contents are shown.

Automatically recognize gates in sub-circuit with fewer than ____ nets

Schematic instances containing fewer than the specified number of nets are displayed as the equivalent logical gate. Gate patterns for INV, NAND, NOR, AOI, and OAI are recognized. Only the strict patterns for AOI and OAI shown in “Regular CMOS Gates” are recognized. Gates ending with an inversion (NAND, NOR, AOI, OAI) followed by an inverter are reduced to AND, OR, AO, or OA. In addition, chains of inverters are reduced to the equivalent inverter or buffer and parallel devices are reduced to their equivalent device. Parallel capacitors, resistors, inductors, and diodes are reduced to their equivalent device. Reduced devices are displayed with a new level of hierarchy which contains the non-reduced logic. The value specified for this option must be greater than or equal to 5 for the reduction to take place.

Note:

Logic reduction is done based on a dictionary sort of the devices by name. Only devices that are adjacent to each other in dictionary sort order are considered for reduction.

Symbol Library files

The Symbol Library files supply user-defined symbol shapes. Enter or browse to the Symbol Library files (*.sym). Enter one file per line or use the Add button in the file browser to select multiple files.

See “Symbol Library Files for Custom Symbol Shapes”.

Show ‘invisible’ properties specified in symbol library files.

As described.

Spice File Viewer only

These options only apply to the Spice File Viewer; the corresponding behavior in the Internal Schematic Viewer is determined by the rule file associated with the results database. See “Using the Calibre RVE SPICE File Viewer”.

Select objects in case-sensitive manner

When enabled, only objects that match in a case-sensitive manner are selected when you select an object in the netlist view of the Spice File Viewer. This option is off by default.

NMOS device type names

Enter a space separated list of device names to be displayed as nmos in addition to the default of n?, where the ? wildcard matches any number of characters.

PMOS device type names

Enter a space separated list of device names to be displayed as pmos in addition to the default of p?, where the ? wildcard matches any number of characters.

Power nets

Enter a space separated list of net names for power; the default is vdd? vcc?, where the ? wildcard matches any number of characters.

Note:

Do not clear the power or ground nets selections—schematic display and automatic gate recognition depend on correct entries for power and ground.

Ground nets

Enter a space separated list of net names for ground; the default is gnd? ground? vss?, where the question mark (?) is a wildcard character that matches any number of characters.
