Calibre::import_drc_errors

Imports a file saved from a previous Calibre RDB conversion to DRC error markers. Used in the Synopsys IC Compiler or Cadence Encounter environment.

Usage

Calibre::import_drc_errors  rdb_drcerr

Arguments

rdb_drcerr

A file saved from a previous Calibre RDB conversion to Synopsys IC Compiler or Cadence Encounter DRC error markers. The rdb_drcerr file is created with one of the following commands, where only the options specific to selecting DRC error marker conversion and file output are given:

Command

Options

Execution environment

Calibre::convert_to_drc_errors

‑output_file rdb_drcerr

Synopsys IC Compiler or Cadence Encounter

calibre -rve

‑convert_to {icc_drc | edi_drc}

‑output_file rdb_drcerr

Linux shell

Return Values

Returns 1 for success or 0 for failure.

Description

Imports a file saved from a previous Calibre RDB conversion to DRC error markers as error markers in the current design session.

Importing Errors to Cadence Encounter

In Cadence Encounter the errors are viewed in the Violation Browser. Close the Violation Browser using the Close button before issuing the Calibre::import_drc_errors command. For some Cadence Encounter versions the conversion process can hang if the Violation Browser is open during the process or if you close the Violation Browser with the X icon in the top right corner of the window.

An error occurs if you attempt to import results from a cell that is not the working design in the Cadence Encounter session.

Importing Errors to Synopsys IC Compiler

In Synopsys IC Compiler, the errors are viewed in the Error Browser after the import as a “Thirdparty DRC” error view.

If you import results from a cell that is not currently open, the cell is automatically opened.
