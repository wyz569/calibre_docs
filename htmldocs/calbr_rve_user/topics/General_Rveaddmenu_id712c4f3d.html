rve::addmenu

The rve::addmenu command adds items to the Macros menu in Calibre RVE.

Syntax

rve::addmenu command -label menu_name -command tcl_proc

Parameters

  • -label menu_name — Required switch and parameter, where menu_name specifies the label for the menu item.

  • -command tcl_proc — Required switch and parameter, where tcl_proc specifies the Tcl procedure to run when the menu item is selected.
