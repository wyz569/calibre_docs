Display Conventions for DRC Results

Calibre RVE for DRC uses certain conventions to mark results as clean, fixed, or waived.

Result status — Right-click a result or cluster for a pop-up menu with options for marking a result as fixed or waived. Result status is indicated with the following colors and icons:

Clean     

Fixed     

Waived     

Error     

Result ID — A unique number for each result. Results retain their unique ID regardless of the grouping, sorting, or filtering selections.

Column selection — The columns in a table view can be selected by clicking in the column selection icon in the first column to get a dropdown list, as shown in Figure 1. Some columns in the Result View are hidden by default if they have the same data value across the cluster being displayed.

Tip

You can change the order of the columns in the tree view by dragging the column header.

Sorting — Sort a table by a column value by clicking in the header of the column.
