Show ... DBsAsChecks

Key Value for DRC HTML Report Configuration File, side RDB control category

Usage: Optional. Valid in Common section only.

Inheritance: Not applicable.

A group of key values that specifies whether to show side (auxiliary) RDBs of the given type as checks in the tree view. The default setting depends on the database type.

Allowed Values

  • 0 — False; do not show as a check in the tree view.

  • 1 — True; show as a check in the tree view.

Key Names and Defaults

  • ShowCheckMapDBsAsChecks

    Default: 1 (Show as check)

  • ShowDensityDBsAsChecks

    Default: 0 (Do not show as check)

  • ShowDFMDBsAsChecks

    Default: 0 (Do not show as check)

  • ShowLayoutInputExceptionDBsAsChecks

    Default: 1 (Show as check)

  • ShowNetAreaRatioDBsAsChecks

    Default: 0 (Do not show as check)

  • ShowOtherDBsAsChecks

    Default: 0 (Do not show as check)

  • ShowPATHCHKDBsAsChecks

    Default: 1 (Show as check)

  • ShowWaiverDBsAsChecks

    Default: 1 (Show as check)

Examples

ShowPATHCHKDBsAsChecks = 0
