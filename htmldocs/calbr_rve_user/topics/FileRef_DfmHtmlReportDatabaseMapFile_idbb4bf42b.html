DFM HTML Report Database Map File

Input for: DFM HTML Reporting

The database map file (db_to_layout_map_file) is an input to DFM HTML reporting and specifies the DFM database directory as input for DFM HTML reporting. If result highlight images are included in the DFM HTML report, the layout database file and an optional Calibre DESIGNrev layer properties file are also specified in the file.

Format

  • ASCII text file.

  • Comma-separated parameters.

  • A single listing of a DFM database.

  • No support for comment text.

Syntax

dfmdb_path , layout_file , layer_properties_file

All commas are required.

Parameters

dfmdb_path

A required parameter specifying the DFM results database.

layout_file

A required parameter specifying the path to and name of the of the layout.

layer_properties_file

A parameter specifying the path to and name of the layer properties file. Refer to “layerprops File Format” in the Calibre DESIGNrev Layout Viewers User’s Manual for complete information.
