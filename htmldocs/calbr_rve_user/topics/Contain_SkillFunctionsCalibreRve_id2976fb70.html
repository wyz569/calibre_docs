SKILL Functions for Calibre RVE

Several SKILL functions are available for Calibre RVE.

SKILL Function

Description
mgc_calibre_start_rveStarts Calibre RVE from the SKILL command interpreter or a SKILL script.
mgc_rve_open_dbSKILL function that opens the specified results database in a running Calibre RVE session if one exists. The results database may be a file or directory. If there is not a running Calibre RVE session, the database is opened in a new session.
mgc_rve_open_rdb_fileSKILL function that opens the specified ASCII format results database (RDB) in a running Calibre RVE session if one exists. If there is not a running Calibre RVE session, the RDB is opened in a new session.
mgc_rve_set_current_librarySpecifies the first library that Calibre RVE searches in for cells to highlight to. When called without any arguments the current cell and view is used.
mgc_rve_set_default_visible_layersSaves the current set of visible layers as the default set of visible layers.
mgc_rve_set_layout_libpathYou can specify the libraries Calibre RVE looks in for layout cells using a SKILL function. This function corresponds the “Libraries RVE should look in for” setting in the dialog box opened with Calibre > Setup > RVE in Cadence Virtuoso.
mgc_rve_set_schematic_libpathYou can specify the libraries Calibre RVE looks in for schematic cells using a SKILL function. This functions corresponds the “Libraries RVE should look in for” setting in the dialog box opened with Calibre > Setup > RVE in Cadence Virtuoso.
mgc_rve_toggle_visible_layer_setToggles the design layer visibility.
