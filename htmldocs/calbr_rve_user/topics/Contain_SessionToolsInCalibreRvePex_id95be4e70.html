Session Tools in Calibre RVE for PEX

Calibre RVE for PEX provides a variety of tools for making the task of investigating net parasitics easier.

Table 1 lists each tool and how to access it. With the exception of the point-to-point calculations and the Mutual Inductance tab, the tools are common to all Calibre RVE for LVS, PERC, and PEX applications.

Table 1. Calibre RVE for PEX Session Tools

Tool

Access with

Use for

Navigator Pane

View > Navigator

Easy navigation to reports, files, and tools in Calibre RVE.

Info Pane

View > Info

View detailed information about design elements in a hierarchical display.

Finder Tab

View > Finder

Search for design elements. Searches can be restricted by object type, net, and layer.

Internal Schematic Viewer

View > Schematic

View rendered schematics for layout and source.

Highlights Tab

Highlight > Setup Highlights

Select layers and layer colors for highlighting. Also specify device and area filters.
