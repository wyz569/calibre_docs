Calibre DESIGNrev Commands for Calibre RVE

There are several batch commands in Calibre DESIGNrev that are relevant when using Calibre RVE. There are commands to open an ASCII format results database (RDB), convert polygons to an RDB file, and move RDB error markers to a different level in the layout.
Table 1. Calibre DESIGNrev Commands for Calibre RVE

Command

Description

cwbLoadRdbFileInRve

Opens the specified RDB file in a running Calibre RVE session, if one exists. If there is not a running Calibre RVE session, one is started.

cwbOpenDBInRve

Opens the specified results database in a running Calibre RVE session, if one exists. The results database may be a file or directory; the supported database types are DRC (ASCII format), LVS, PERC, PEX, and SPICE. If there is not a running Calibre RVE session, one is started.

$L rdbout

Convert all polygons on a layer into RDB output.

$cwb runRveOnRveOutput

Starts Calibre RVE and opens an RDB file if one is specified. If a Calibre RVE session is already open, that session is closed.

$cwb runMultiRveOnRveOutput

Starts Calibre RVE and opens an RDB file if one is specified, without closing existing sessions.

$L pushmarkers

Moves Calibre RVE ASCII database markers to levels relevant to marked polygons.

You can also use the File > Export Layout selection in Calibre DESIGNrev to save polygons to an RDB file. See “Saving Highlighted Shapes as a Calibre nmDRC Results Database (RDB)” in the Calibre DESIGNrev Layout Viewer User’s Manual.
