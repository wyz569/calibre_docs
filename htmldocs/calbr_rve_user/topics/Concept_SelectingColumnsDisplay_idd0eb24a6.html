Selecting Columns to Display

You can select the columns that are displayed in the tree or details view in Calibre RVE for DRC.

The columns displayed in the tree or table view can be selected by clicking the column selection icon to get a dropdown list or by right-clicking in the column header for a context sensitive menu. Both methods are shown in Figure 1.

Figure 1. Column Selection Methods

Some columns are hidden by default if there is no relevant data in the column. You can select the display parameter for each column by clicking through the following possible choices:

Always display the column.

Do not display the column.

Use automatic selection for column display. The column is displayed only if there is non-zero data in the field. The Waived By and Waived Date columns are exceptions; you must enable these columns with a check mark to display them.
Tip

You can change the order of the columns in the tree view by dragging the column header.
