Tools Menu in Calibre RVE for LVS

The Tools menu has some selections that are specific to a Calibre RVE for LVS session.
Table 1. Tools Menu Contents in Calibre RVE for LVS

Menu Item

Description

YieldAssist Report

Opens a file browser for the YieldAssist Report. The file opens in a report browser, which also displays TestKompress and FastScan reports.

Export CalibreView

Opens a dialog for creating the CalibreView, which is used in the Cadence® Analog Design Environment. Select from:

  • Create new view with devices and nets

  • Annotate existing view with net geometry

The extracted connectivity is exported as a Calibre View; see “Creating a Calibre View” and “Creating a Post-LVS Calibre View” in the Calibre Interactive User’s Manual for more information.

The Export Calibre View dialog also allows you to export all net polygons to the Calibre View of a cell. All instances in the cell are flattened before polygons are exported. This feature requires the Calibre Connectivity Interface (CI) license, and also requires that the CCI keyword is specified in the Mask SVDB Directory SVRF statement (which is equivalent to the Generate Calibre Connectivity Interface data setting in Calibre Interactive LVS and PEX).

To make sure the Calibre RVE cells are mapped to the correct Cadence library, choose Highlight > Setup Cell Map before creating the Calibre View. This is not necessary if you started Calibre RVE from Calibre Interactive. See “Map Calibre RVE Cells to Different Cadence Library for Highlighting”.
