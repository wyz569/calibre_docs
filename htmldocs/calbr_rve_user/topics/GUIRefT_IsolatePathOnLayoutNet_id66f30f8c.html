Isolate Path on Layout Net

Access with: View > Finder, then choose “Isolate Path on Layout Net” for “What type of search?”

Searches in: Layout

Requires: PHDB. See Mask SVDB Directory in the SVRF Manual.

The “Isolate Path on Layout Net” search on the Finder tab locates a path between two endpoints on a layout net. The endpoints must be on the same physical net. The returned path is not necessarily the shortest path.

Description

There are three modes of operation, “Single Path Per Net,” “All Paths Per Net,”and “All Paths Per Net in Layout,” as shown in Figure 1 and described in a table later in this topic.

The results are entered in the Finder search result list with the IP () icon; you can click the icon to highlight the path in the layout viewer. A layer filter may be specified; see the “Search Filters” entry in the table of options later in this topic.

Description

How many to isolate?

  • Single Path Per Net — Finds one path between the path endpoints in the From Location and To Location fields. The first path found is returned.

  • All Paths Per Net — Finds all paths between the path endpoints in the From Location and To Location fields.

  • All Paths Per Net in Layout — Opens a dialog box to select path endpoints in the layout viewer; the dialog box remains open until dismissed. The first path found is automatically highlighted. All results are displayed in the dialog box and entered in the Finder search results list.

From Location

To Location

The endpoints for the Isolate Path on Layout Net search. The endpoints must be on the same physical net in the layout. In “Single Path” and “All Paths Per Net” mode the endpoints may be entered as a space-separated coordinate pair or selected in the layout viewer. In “All Paths Per Net in Layout” mode, a dialog box opens to select the endpoints in the layout viewer.

Layout Icon  

Select a location in an attached layout viewer in “Single Path Per Net” and “All Paths Per Net” modes.

Show Highlight Menu  

Click to display a menu with options for highlighting and displaying information about the location.

Search Filters

You can specify a layer filter which is used when selecting points in the layout viewer for the “From Location” and “To Location” fields. The layer filter is not used for the path search.

See Search and Selection Filters in the Finder Tab.
