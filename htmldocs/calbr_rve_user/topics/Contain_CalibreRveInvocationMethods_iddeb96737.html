Calibre RVE Invocation Methods

Calibre RVE can be started in several ways, depending on your usage.

The following list summarizes the most common ways to start Calibre RVE. Make sure that you have fulfilled all the requirements for Calibre RVE as given in the section “Calibre RVE System Requirements”.

“Invoking Calibre RVE From a Layout or Schematic Viewer”

Communication between Calibre RVE and the viewer is established automatically. Use this method if you already have results from a Calibre run.

“Automatically Invoking Calibre RVE from Calibre Interactive”

Calibre RVE is started after the Calibre run finishes and opens the database listed on the Outputs pane of Calibre Interactive. If your Calibre Interactive run is connected to a layout or schematic viewer, communication between Calibre RVE and the viewer is established automatically.

“Manually Invoking Calibre RVE from Calibre Interactive”

This method is similar to automatically starting Calibre RVE from Calibre Interactive, except that you start Calibre RVE manually.

“Invoking Calibre RVE from the Command Line”

Use this method if you only want to view results in the Calibre RVE interface, without a connection to a layout or schematic viewer.
