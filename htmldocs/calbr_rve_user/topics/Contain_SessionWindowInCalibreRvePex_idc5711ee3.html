Session Window in Calibre RVE for PEX

The session window in Calibre RVE for PEX opens with report tabs for each report generated from your PEX run. Depending on your SVDB, you may see different reports. If parasitic extraction with multiple corners was performed, there is a Parasitics tab for each process corner. The View menu items and the report tabs are used to customize the display.

Table 1. Calibre RVE for PEX Session Areas

Area

Description

Toolbar

The Standard Toolbar in Calibre RVE is available in all views. The Parasitics Toolbar is available when the Parasitics tab is active.

Navigator Pane

Provides easy navigation to the available report tabs, files and tools. Selections are duplicated in the View menu.

Report Tabs

Tabbed views of results. See PEX Report Tabs and Detailed Views in the reference section.

Net Parasitics Table

A table view of net parasitics. See “Using the Net Parasitics Table” and “Parasitics Tab”.

 

Double-click a net or use the Show Parasitics on Net icon to open the Detailed Parasitics View.
