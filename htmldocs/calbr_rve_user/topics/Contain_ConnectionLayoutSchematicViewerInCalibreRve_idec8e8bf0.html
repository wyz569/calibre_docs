Connection to a Design Tool

It is very useful to have your Calibre RVE session connected to a layout viewer so that you can highlight results, and cross-probe results between the geometric layout and the source.

All Siemens layout viewers are supported and have a Calibre or Verification menu for accessing Calibre RVE. “Interfacing With Layout and Schematic Viewers” discusses how to set up the Calibre interface in other design tools.

You can also connect to a schematic viewer; however, because Calibre RVE for LVS, PERC, and PEX have an internal schematic viewer, this is optional. See “Internal Schematic Viewer” for a instructions on using the internal schematic viewer.

The connection between Calibre RVE and your viewer can be made automatically or manually. You can highlight to two layout design tools when using Calibre RVE for DRC. The Connection Icon () in the lower right corner of Calibre RVE indicates the connection status and includes menu selections to connect, disconnect, and open the Options tab.
