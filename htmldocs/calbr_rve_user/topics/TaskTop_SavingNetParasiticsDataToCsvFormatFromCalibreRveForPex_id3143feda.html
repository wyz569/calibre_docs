Saving Net Parasitics Data to CSV Format from Calibre RVE for PEX

Calibre RVE for PEX can save net parasitics data to a CSV (comma-separated value) formatted file; the file can then be imported into most spreadsheet tools for later analysis.

Prerequisites

  • A net parasitics tab is open in Calibre RVE for PEX.

Procedure

  1. (Optional) If desired, open the detailed parasitics view for a net in order to export the detailed information for that net.
  2. Select File > Export Parasitics > Export To CSV.
  3. In the Export Parasitics dialog box, specify the name of the CSV file.
  4. Select the information to export:

    • Summary Table — Save the whole net parasitics table as it currently appears in the GUI.

    • Net Detailed Parasitics — Save information on the parasitic elements for a specified net whose detailed parasitics view is open. This option is only available if a detailed parasitics view is open.

      After selecting this option, specify the net to export and the type of parasitic elements to export. The nets currently open in the detailed parasitics view are listed, and the types listed match the available types for the currently selected net.

  5. (Optional) Specify whether to open the CSV file in Calibre RVE after exporting.
  6. Select Export.

Results

Results are saved to the CSV file according to the selection in Step 3.
