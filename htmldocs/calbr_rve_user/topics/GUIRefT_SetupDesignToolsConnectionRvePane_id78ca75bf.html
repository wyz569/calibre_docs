Setup Design Tools and Connection to RVE Pane

Access with: Setup > Design Tools

Use this pane to control options for connecting to a layout or schematic viewer.

Tip

The connection status is reported in the Connection Icon in Calibre RVE, located in the lower right corner. The Connection icon also includes a dropdown menu to connect, disconnect, and open the Options tab to this pane.

Layout Design Tool

     Disconnect button

Tool Information

Displays the design tool connection. When connected, the name of the design tool is displayed.

Synopsys Laker, unlike other layout tools, does not report the viewer name to back to Calibre RVE, so the most recently connected viewer or the default viewer is displayed. As a workaround, you can set the environment variable MGC_CALIBRE_LAYOUT_SERVER_NAME to “Synopsys:Laker”.

Highlight data file

Default: query_results

The name of the intermediate file for query results. Make sure the filename is not the same as any other file in use, or the existing file will be overwritten.

The environment variable MGC_CALIBRE_LAYOUT_TMP_FILE also defines the path to the highlight data file.

Secondary Layout Design Tool (DRC Only)

Select the checkbox to enable highlighting to a second layout tool. Used with Calibre RVE for DRC only. See “Debugging DRC Results Using Two Layout Design Tools”.

Connection Options

The host, socket, and other connection options have the same meaning as described with “Layout Design Tool”
