LVS/PERC/PEX Highlighting Options

Access with: Setup > Options, choose the Highlighting category, then expand the “LVS/PERC/PEX Highlighting” section.

These options controls highlighting actions for the results browsers in LVS, PERC, and PEX sessions.

Fields

Table 1. LVS/PERC/PEX Highlighting Option Pane Contents

Item

Description

Highlight Groups of Reduced Devices

If enabled, highlight all devices in a group of reduced devices if a device is part of the group.

Use layer colors while highlighting

Highlights each design layer in the highlighted object with a different color. See Layer Filters and Colors on the Highlights Tab and the Highlights Tab for a complete description. See “Setting Highlight Color in Calibre RVE for LVS, PERC, and PEX” for additional options.

Highlight objects in individual colors

Highlights each connectivity object in a different color when multiple objects are highlighted at once; this occurs when “Highlight all …” is selected or “Highlight found objects” is enabled in the dialog to select objects from a viewer. Only applies if the highlighting color scheme is “Default colors.”

Highlight PEX P2P parasitics after selecting coordinates

When selecting P2P coordinates in the layout viewer, this option specifies whether to automatically highlight P2P parasitics after the calculation is complete.

Display connectivity object name in highlight polygons

Adds the name of the net, device, instance or port as text to each highlighted polygon. See “Displaying Net and Object Names with Highlight”.

Highlight connectivity objects in Schematic Design Tool

When enabled, connectivity objects (nets, devices, instances, and pins) are highlighted in the connected schematic viewer.

Highlight objects also by corresponding layout names.

When enabled, connectivity objects (nets, devices, instances, and pins) are highlighted in the connected schematic viewer using corresponding layout names. Only available when “Highlight connectivity objects in Schematic Design Tool” is enabled. Automatically enabled by Calibre Interactive-nmLVS in Netlist vs Netlist mode.

Highlight layout nets/devices/ports in:

Context cell — Highlight objects in the cell they are in. Default.

Top cell — Highlight objects in the top cell.

Highlight polygons in layout shorts in:

Short cell — Highlight short polygons in the layout cell the short is found in.

Top cell — Highlight short polygons in the top cell.

Layout location query filter distance:_____ user-units

The query filter distance must be a non-negative number in user units. A larger number causes a larger geometric area to be searched.

This value is used in layout location queries for ports only. Without a filter, only ports that overlap the location are returned. Specifying a filter causes Calibre RVE to return ports within the filter distance of the search location.

Marker Scale for P2P/CD/LDL results

Sets the scale of the marker size for highlighting Calibre PERC LDL P2P and CD simulation results. The default size is 0.01.

Enter a floating point number, where a number less than one reduces the size of the marker. The setting affects the markers used to highlight sources, sinks, and connections.

Strip SPICE instance prefix of: _______

Select this setting to remove the specified SPICE instance prefix before highlighting hierarchical instances. This option only applies during highlighting to a connected external schematic design tool. Default: On, prefix X

This setting controls the mapping from source netlist instance names to schematic instance names. If the netlister appends SPICE prefixes to every instance in the schematic when it creates the source netlist, this setting should be selected. Enter the prefix that the netlister adds to each instance in the schematic.

Strip SPICE device prefix of length: ____ characters

Select this setting to remove SPICE device prefixes of the indicated length before highlighting the device in the schematic. This option only applies during highlighting to a connected external schematic design tool. Default: On, 1 character.

This setting controls the mapping from source netlist device names to schematic device names. If the netlister appends SPICE prefixes to every device in the schematic when it creates the source netlist, this setting should be selected.

Usage Notes

Also see the following options located in the Highlighting category in the “Layout Highlighting” area:

  • Skip contact/via layers while highlighting — When selected, contact layers appearing in a Connect … BY statement are not included in highlighting actions. This control is also location on the Highlights tab (Highlight > Setup Highlights); see Layer Filters and Colors on the Highlights Tab for a full description.

  • Display connectivity object name in highlight layer labels — For Calibre DESIGNrev/WORKbench layout viewers only. The highlight layer label is of the form object_name:layer_name; otherwise only layer_name is used.
