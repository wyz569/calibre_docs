Results Tree - Group by Check and Cell

You can group results in the tree view by check and cell name using the View > Tree Options > Group By selection or the Group By button.

Grouping by Cell/Check is shown in Figure 1; select View > Tree Options > Group By > Cell / Check to get this view. Other grouping options are Check/Cell, Check, and Cell. You can also click the Group By button to select the grouping method.

Figure 1. Tree View Grouped by Cell / Check

Result Sorting — Click in a column header to sort results by that value.

Result selection — Select a single result or cluster by clicking it. Use Shift- and Ctrl-click to select multiple results. All selected results are displayed in the Result View. See DRC Result Filter to filter results.

Column Headings — The values given in Table 1 are available for display in the results tree when grouped by check or cell name. To select which columns are displayed, click the column selection icon in the first column to get a dropdown list, or right-click in the column header for a context sensitive menu.

Table 1. Column Headings for Tree View

Selection

Description

Priority

The check priority. See “RVE Priority”.

Result Count

The hierarchical result count.

Flat Count

The flat result count. For most cases the flat count is the same as the hierarchical count and by default is not displayed.

Fixed Count

Number of fixed results.

Waived Count

Number of waived results.

Check Text

The check text.

Check text can also be viewed in the Check Text Pane by enabling View > Check Text Pane. Also see “Check Text” in the Calibre Verification User’s Manual.
