Examining Not Compared, Incorrect, and Empty LVS Comparison Results

In some cases the Comparison Results tab in Calibre RVE for LVS displays only the top cell with a status of “Incorrect” or “Not compared” and the detailed view contains only text information with no links for highlighting and probing design elements. This can happen if Calibre encounters an error and halts processing before starting the LVS comparison phase. If processing stops without the Calibre nmLVS tool providing any comparison status or data, Calibre RVE reports “Comparison Not Performed” as the status.

A few of the common reasons for the Calibre nmLVS tool to halt processing before starting the LVS comparison phase are the following:

  • An input error, such as a missing file or a syntax error.

  • LVS Report Option includes an option that specifies to abort processing on certain errors.

  • Cell or net name mismatch.

In most cases the LVS Report contains useful information. In some cases the LVS Report is not produced and you need to examine the run transcript for information.

The following steps can help to determine the problem:

  • Search in the Comparison Results tab for “Error”. If no text is displayed in the detailed view, open the LVS Report or the run transcript and search for “Error”. Many errors are explained in the section “Secondary Messages” in the Calibre Verification User’s Manual. The search should be case-insensitive.

  • Open the rule file and determine what options are specified for the LVS Report Option statement. The options LPE, N, NCA, NE, SPE, and W cause processing to halt if certain errors are encountered.

  • In the LVS Report, review the sections “Initial Numbers of Objects” and “Numbers of Objects After Transformation”. A mismatch in the number of objects between layout and source can indicate a problem in either the layout or the source netlist.

  • Review the section “LVS Report” in the Calibre Verification User’s Manual. The following sections in particular may be useful:

    • “Analysis of the LVS Report”

    • “Errors and Warnings”

    • “Overall Comparison Results”

    • “Input Errors”
