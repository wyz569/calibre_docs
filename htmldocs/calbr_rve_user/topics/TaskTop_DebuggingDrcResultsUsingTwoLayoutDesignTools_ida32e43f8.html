Debugging DRC Results Using Two Layout Design Tools

Calibre RVE for DRC can highlight to two layout design tools, enabling you to debug in one design tool and edit the layout in a different tool. For example, you can debug the DRC results in Calibre DESIGNrev and make the required fix in your place and route tool.

Highlights are made to both layout design tools. However, actions that take place in a selected layout area, such as highlight, waive, fix, and select in area, only take place in the first (primary) connected layout tool.

Prerequisites

  • A Calibre DRC results database.

Procedure

  1. Open your design in the two layout tools you want to highlight to.
  2. Invoke Calibre RVE from the primary layout tool. Actions such as highlight and select in area only take place in the primary layout viewer.

    • Calibre DESIGNrev and other Calibre viewers — Choose Verification > Start RVE.

    • Most other design tools — Choose Calibre > Start RVE.

  3. Specify the database path and a “Database Type” of DRC/ERC in the Calibre RVE dialog box and click Open.

    The results database opens in Calibre RVE and the connection to the design tool is made automatically, as indicated by the Connection Icon () in the lower right of Calibre RVE. If the connection is not successful, check the viewer socket, as described in “Manual Connection to a Viewer”.

    Tip

    The socket number for a connection is reported from the connection icon.

  4. Setup the socket in the secondary layout viewer. Make sure the socket number is not the same as that used by the primary layout viewer connection.

    • Calibre DESIGNrev — Choose Verification > RVE/CI Setup from the main menu. In the Server section of the dialog box, enter a socket number and click Start.

    • Cadence and Synopsys tools — Choose Calibre > Setup > Socket from the main menu of the design tool. Enter a socket number and click OK.

    • Other viewers — Activate a socket by following the instructions in the user’s manual for the viewer.

  5. In Calibre RVE, click the connection icon and select Options, or choose Setup > Design Tools from the main menu.
  6. Check “Secondary Layout Design Tool (DRC Only)” in the Setup Design Tools and Connect to RVE pane.
  7. Enter the connection information:
    1. Enter the host name and socket number (from Step 4) for the secondary layout viewer.
    2. Click the connect button () to start the connection.

    If the connection is successful, the design tool information is displayed in the “Tool Information” field.



  8. Highlight results as usual from Calibre RVE.

    The highlights are made in both layout viewers. Debug and edit as needed.
