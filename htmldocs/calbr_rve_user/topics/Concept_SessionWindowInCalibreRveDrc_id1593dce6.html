Session Window in Calibre RVE for DRC

The session window is designed to help you analyze the results from a Calibre nmDRC run. It allows you to view, sort, analyze, and track DRC results throughout your verification process.

Figure 1 shows the view you see on opening a DRC session in Calibre RVE. The session opens with a side by side tree view and detailed result view.

Figure 1. Opening Session Window in Calibre RVE for DRC

These are the session areas and their functions:

Toolbars — See “Toolbars and Control Bars in Calibre RVE for DRC”.

Report Tabs — Results from each database are presented in separate tabs. You can arrange tabs for side-by-side comparison, as explained in “Report Tabs in Calibre RVE”. See “DRC Results Tab” for details about the report tab in Calibre RVE for DRC.

Note: The opening view does not have tabs; separate tabs appear when a text file, the Options tab, or a second database is opened.

Summary Area — The summary area includes the following:

  • Result Filter — Apply or create a result filter to select a subset of results for display. See “Filtering DRC Results” for complete details.

  • Top cell

  • Result summary - Reports the number of results and the total fixed and waived.

Bookmark () — Choose or save a bookmark for the view configuration. See “Figure 1”.

Tree View — A hierarchical display of the results is shown in the tree view. From the tree view you can do the following:

  • Sort and group — Sort and group results according to check name, cell name, or property using the selections in View > Tree Options > Group By.

  • Analyze — Click a check name to select it; the results from that check are displayed in the Result View.

  • Filter — Click the filter icon to select a subset of results for display, as described in “Filtering DRC Results”.

    See the sections “Viewing Properties in Calibre RVE for DRC” and “Histogram Window” for details on viewing properties.

Result View — The Result View lists all the results for the checks selected in the Tree View. You can choose the format of the Result View using the menu selection View > Result Options, or with the icon at the right edge of the summary area at the top of the report tab, as shown at right.

 


You can do the following from the result view:

  • Highlight results in a layout viewer — Double-click a result, or right-click a result for a menu of highlight options (see Figure 1).

  • Mark results — Right-click a result and choose Fix or Waive.

  • Analyze results — View details about each result in the Result Data Pane, or choose the Details format to view results in a table format.

Result Data Pane — Displays details about each result. The Result Data Pane is enabled with View > Result Options > Result Data Pane.

Check Text Pane — Displays the check text for the rule check selected in the Tree View. Turn the display of this area on and off with View > Check Text Pane.

Message Area — Displays information about the selected result or cluster.

Right-click menus — Right-click a result or result cluster for a pop-up menu to highlight the result or mark it as fixed or waived. Right-click in the table header for sorting and grouping selections.
