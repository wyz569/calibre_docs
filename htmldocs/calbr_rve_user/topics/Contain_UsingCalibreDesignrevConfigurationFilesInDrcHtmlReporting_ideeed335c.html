Using Calibre DESIGNrev Configuration Files in DRC HTML Reporting

You can use the Calibre DESIGNrev layerprops and preferences files to control the layout display in result highlight images created by DRC HTML reporting. There are several methods for using these files.

File

Methods for Use

layerprops

In order of precedence:

  1. Specify a layerprops file in the database map file. See “DRC HTML Report Database Map File”.

  2. Use the layerprops file in a Calibre DESIGNrev configuration directory specified with the environment variable MGC_CWB_CONFIG_DIRS. See “Multilevel Configuration” in the Calibre DESIGNrev Layout Viewers User’s Manual.

preferences

See “Calibre DESIGNrev Display Preferences File for DRC HTML Reporting”.

Note:

The preferences and layerprops files in $HOME/.calibrewb_workspace are not used unless MGC_CWB_CONFIG_DIRS specifies that directory.
