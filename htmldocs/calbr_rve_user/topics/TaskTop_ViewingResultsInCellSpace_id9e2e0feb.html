Viewing Results in Cell Space

The Calibre RVE Highlight > Highlight in Context menu item specifies that results are highlighted in the cell in which they are reported, rather than in the top cell. This feature depends on results being reported by cell in your DRC results database, which is enabled with the DRC Cell Name specification statement. This option is only available for hierarchical Calibre nmDRC.

You can also view ERC results from a Calibre nmLVS run in cell space; in this case use the ERC Cell Name specification statement to report results by cell. This option is only available for hierarchical Calibre nmLVS.

Prerequisites

  • A DRC or ERC results database with results reported by cell; this can be done with one of the following methods:

    • Specify in the rule file with one of the following statements:

      • Output results in cell coordinate space and provide the transformation data necessary to convert those coordinates to top-level space:

        DRC Cell Name YES CELL SPACE XFORM

      • Output results in top-level coordinate space and provide the data necessary to convert those coordinates to cell space:

        DRC Cell Name YES XFORM

      Use the ERC Cell Name specification statement to report results by cell when running ERC checks in Calibre nmLVS.

    • Specify in Calibre Interactive. See “Reporting DRC Results in Local Cell Space or Top Cell Space” or “Running ERC Checks” in the Calibre Interactive User’s Manual for more information.

Procedure

  1. Open the results database in Calibre RVE for DRC.
  2. Enable highlight in context using one of these methods:

    • Select Highlight > Highlight in Context.

    • Select Setup > Options to open the Options tab, then select the Highlighting category, expand the “DRC/DFM Highlighting” area, and enable “Highlight DRC results in context.”

    • Press Ctrl-T to toggle the setting for the option.

    • Use the Highlight in Context toolbar button (On: ; Off:).

      Tip

      If the “Highlight in Context” setting is not available in Calibre RVE, confirm that the DRC results database was created as described in the “Prerequisites.”
