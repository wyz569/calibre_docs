CalculateProperties

Key Value for DRC HTML Report Configuration File, report specification category

Usage: Optional. Valid in Common section only.

Inheritance: Not applicable

Specifies whether to calculate and show the calculated properties #PP, #PA, #EL, #EC, and #EW.

See “Viewing Properties in Calibre RVE for DRC” for a definition of these properties. See ClassifyShapes for specifying the #SC (shape class) property.

Allowed Values

0

Do not include calculated properties. (default)

1

Include calculated properties
