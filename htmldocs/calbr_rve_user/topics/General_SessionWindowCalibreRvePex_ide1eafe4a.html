Session Window for Calibre RVE for PEX

The session window in Calibre RVE for PEX includes a net parasitics table and a Find Nets search toolbar.

The main session window is shown in the following figure. If parasitic extraction with multiple corners was performed, there is a Parasitics tab for each process corner. A brief description of the window areas is given in Table 1.

Figure 1. Calibre RVE for PEX Session Window

Table 1. Calibre RVE for PEX Session Areas

Session Area

Description

Menu Bar

Standard menu items. See Menu Bar in Calibre RVE for PEX.

Toolbar

Shortcuts to commonly used commands. Additional toolbars are available when specific Report Tabs are selected. See Toolbars in Calibre RVE for PEX.

Navigator Pane

Provides easy navigation to the available report tabs, files and tools. Duplicates the View menu.

Terminal Window

Provides a Tcl shell (not shown in Figure 1.) To open, choose View > Terminal. The Terminal is disabled when a SPICE file is the only file open.

Report Tabs

Tabbed views of results. See PEX Report Tabs and Detailed Views.

Net Parasitics Table

A detailed view of a selected result. See PEX Report Tabs and Detailed Views. Double-click a net to open the Detailed Parasitics View.
