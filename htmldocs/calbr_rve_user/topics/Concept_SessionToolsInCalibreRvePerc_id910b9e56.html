Session Tools in Calibre RVE for PERC

Calibre RVE for PERC provides a variety of tools for making the task of investigating Calibre PERC results easier.

Table 1 lists each tool and how to access it. The tools in Table 1 are common to all Calibre RVE for LVS, PERC, and PEX applications.

Table 1. Session Tools in Calibre RVE for PERC

Tool

Access with

Use for

Navigator Pane

View > Navigator

Easy navigation to reports, files, and tools in Calibre RVE.

Info Pane

View > Info

View detailed information about design elements in a hierarchical display.

Finder Tab

View > Finder

Search for design elements. Searches can be restricted by object type, net, and layer.

Internal Schematic Viewer

View > Schematics

View rendered schematics for layout and source.

Highlights Tab

Highlight > Setup Highlights

Select layers and layer colors for highlighting and apply filters for highlight actions
