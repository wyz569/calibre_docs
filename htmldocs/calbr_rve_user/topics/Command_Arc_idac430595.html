arc

Shape element for a Symbol Library file

The arc element defines a circle or an arc. Make the start and end points equal to define a circle. Note: fpath can also be used to define a circle.

Usage

arc x_start y_start x1 y1 [x2 y2] … x_end y_end

Parameters

Parameter

Description

arc

A required keyword

x_start y_start

The x and y coordinates of the starting point.

x1 y1 [x2 y2] …

The x and y coordinates of one or more intermediate points.

x_end y_end

The x and y coordinates of the end point.

Examples

arc 40 0 30 -10 15 -15
