Calibre::openDBInRVE

Opens the specified results database in a running Calibre RVE session, if one exists. The database may be a file or directory. If there is not a running Calibre RVE session, the results database is opened in a new session. The command is entered at the Synopsys IC Compiler or IC Compiler II command prompt.

The command opens the results database in an existing Calibre RVE session only if that session was started using Calibre > Start RVE from the Synopsys IC Compiler or IC Compiler II menu or a previous Calibre::openDBInRVE or Calibre::openRDBFileInRVE command. If the Calibre RVE session was started from Calibre Interactive, the command opens the results database in a new Calibre RVE session

Syntax

Calibre::openDBInRVE db [top_cell]

Arguments

  • db — A required parameter specifying the path to the results database. The database may be a file or directory. The supported database types are DRC (ASCII format), LVS, PERC, PEX, and SPICE. SPICE files are opened in the Calibre RVE SPICE File Viewer.

  • top_cell — An optional argument specifying the name of the top cell; only valid with LVS, PERC, and PEX databases.

Examples

Enter the following at the icc_shell or icc2_shell command prompt to open the svdb database in a existing Calibre RVE session:

icc_shell> Calibre::openDBInRVE svdb
