Methods to Add DRC RVE Rule Check Comments

There are two ways to add DRC RVE rule check comments: in the rule file or in a Check Text Override File (CTO File).

  • In the rule file — Place DRC RVE rule check comments in the rule check with a preceding ‘@’, just as you would other rule comments. You must run Calibre to have the DRC RVE rule check comment included in the results database.

  • In a DRC RVE Check Text Override File — Place DRC RVE rule check comments in a DRC RVE Check Text Override File (CTO File), which is read by Calibre RVE. This method is useful if you want to add DRC RVE rule check comments to an existing DRC results database without editing the rule file and running Calibre nmDRC again.
