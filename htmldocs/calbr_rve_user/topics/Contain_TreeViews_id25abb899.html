Tree Views

The tree view in Calibre RVE for DRC displays results grouped by check name, cell name, property name, or property value, depending on the selection from the View > Tree Options > Group By menu. Custom groupings can be defined. The View > Tree Options menu also controls the display of empty checks.

The selections in the View > Tree Options menu are shown in Table 1. The table has links to sections with more detail on each grouping method.
