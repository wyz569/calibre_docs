Calibre RVE for DFM Report Card

This section describes the Calibre RVE for DFM report card.

Figure 1 shows the DFM report card that appears when you invoke Calibre RVE and load a DFM database. The DFM report card displays the yield information concerned with the designs you are analyzing based on the annotations that you define in your rule file.

All actions related to investigating your DFM results are initiated from here. Calibre RVE automatically reads the DFM report card option file to configure the report card, if one exists, or uses defaults if one does not exist.

Figure 1. Calibre RVE for DFM Report Card

For information and procedures on using the DFM report card to analyze your design, see:

  • “Calibre RVE for DFM Report Card” in the Calibre YieldAnalyzer and YieldEnhancer User’s and Reference Manual.

  • CFA Tutorial with accompanying design data on Support Center.
