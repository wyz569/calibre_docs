mgc_calibre_start_rve

Starts Calibre RVE from the SKILL command interpreter or a SKILL script.

Syntax

mgc_calibre_start_rve(@optional (t_dir nil) (t_topCell nil))

Arguments

  • t_dir — An optional parameter giving the results database file or directory.

  • t_topCell — An optional parameter giving the name of the top cell. The t_dir argument must be present if t_topCell is used.

Description

The function mgc_calibre_start_rve() behaves as follows, depending on which arguments are specified:

  • No arguments — Calibre RVE opens to the startup dialog box, where you can enter the results database and optional topcell name. See “Invoking Calibre RVE From a Layout or Schematic Viewer”.

  • t_dir only — If only the t_dir argument for the results database is given, the behavior depends on the type of results database as follows:

    • database file — If t_dir is a database file, such as an ASCII DRC results database, Calibre RVE is opened to that database.

    • database directory — If t_dir is a database directory, such as an SVDB, then Calibre RVE is opened to that database directory. Calibre RVE searches for a top cell in the database.

  • t_dir and t_topCell — If both the t_dir and the t_topCell arguments are given, the behavior depends on the type of results database as follows:

    • database file — If t_dir is a database file, such as an ASCII DRC results database, Calibre RVE is opened to that database. The parameter t_topCell is ignored.

    • database directory — If t_dir is a database directory, such as an SVDB, then Calibre RVE is opened to that database directory and to the specified top cell.

Example

The following command given in the Cadence command interpreter window opens Calibre RVE for LVS to the database “svdb” at the top cell named “TOP”.

mgc_calibre_start_rve("svdb" "TOP")
