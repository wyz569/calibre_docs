mgc_enc::openDBInRVE

Opens a results database in a running Calibre RVE session if one exists. The results database may be a file or directory. If there is not a running Calibre RVE session, the RDB is opened in a new session. The command is entered at the Cadence Encounter command prompt.

The command opens the database in an existing Calibre RVE session only if that session was started using Calibre > Start RVE from the Cadence Encounter menu or a previous mgc_enc::openDBInRVE or mgc_enc::openRDBFileInRVE command. If the Calibre RVE session was started from Calibre Interactive, the mgc_enc::openDBInRVE command opens the database in a new Calibre RVE session.

Syntax

mgc_enc::openDBInRVE db [top_cell]

Arguments

  • db — A required parameter specifying the path to the results database. The database may be a file or directory. The supported database types are DRC(ASCII format), LVS, PERC, PEX, and SPICE. SPICE files are opened in the Calibre RVE SPICE File Viewer.

  • top_cell — An optional argument specifying the name of the top cell; only applies to LVS, PERC, and PEX databases.

Examples

Enter the following in the Cadence Encounter command prompt to open the svdb results database in an existing Calibre RVE session:

encounter 2> mgc_enc::openDBInRVE svdb
