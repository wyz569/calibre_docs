Viewing Voltage Trace Data for a Single Voltage

Calibre RVE for PERC can generate a schematic view of the voltage propagation for a net. In single voltage mode, the GUI displays the trace for a selected voltage.

Prerequisites

  • A Calibre PERC license. See “Calibre PERC” in the Calibre Administrator’s Guide for more information.

  • A Mask SVDB Directory containing Calibre PERC results or a DFM Database containing Calibre PERC LDL results.

    The database must be from a hierarchical run (the ‑hier command line option) for the Voltage Trace GUI to be available.

  • The command perc::create_voltage_path must be used in the initialization procedure for the rule check.

  • The Calibre PERC database is open in Calibre RVE for PERC.

  • (Optional) Choose the desired setting for “Show PERC voltage paths in a new Voltage Trace window” on the Options tab in the LVS/PERC/PEX category. If this setting is disabled, the current window is closed before opening a new window.

Procedure

  1. Enable View > Result Options > Details to display result details.
  2. Enable View > Result Options > Result Data Pane.
  3. Select a result in the tree view that is from a check that includes the perc::create_voltage_path command.
  4. Right-click a result and choose Trace > Load Trace Data if the menu option is present. This step is only needed once during the Calibre RVE session.
  5. Right-click a result and choose Trace > Voltage voltage (for a net result) or Trace > Pin pin-net > Voltage voltage (for a device result), where voltage is the voltage you want to trace. You can also right-click in the Result Data pane for the Trace menu.


    Tip

    To display separate voltage traces for several single voltages, right-click and choose Trace > Multiple Voltages , then select the desired voltages and enable “Trace each voltage individually.”

    A schematic view of the voltage trace is displayed, as shown in Figure 1 in the “Results” section. The trace includes only nets and pins that have the selected voltage; however terminating pins without the selected voltage are shown if they are connected to a device that does have pins with the selected voltage.

    Figure 1 shows a voltage trace with more than one path, while Figure 2 shows a trace with a single path. A tree view of the voltage path is created automatically for the single path case.

  6. Hover over a net or device to view the voltages that are present. A voltage that is applied directly to a net due to a command in the rule file has the “ud” (user-defined) attribute.
  7. (Optional) Click any schematic element to highlight it in a connected design tool and the Internal Schematic Viewer. If you previously closed the Internal Schematic Viewer, choose View > Schematics to open it again.
  8. (Optional) If the tree view of the path was not created automatically, click the Select Path button and select elements to form a path from start to end. Click Done to end the path selection. The path is saved for the duration of the Calibre RVE session. Selecting a path in this manner can be a useful way to save elements for later analysis.
  9. (Optional) You can right-click any net and choose Trace Voltage on Net <net> to trace the voltage on that net in a new Voltage Trace window.

    This can be useful in these circumstances:

    • The hover text for the net does not include voltage information. For example, a net that is not directly a part of the trace, such as the gate of a device, does not include voltage information. You can right-click such a net and display a new voltage trace.

    • The complete voltage trace is not displayed due to the number of elements. You can right-click a net and continue the voltage trace.
