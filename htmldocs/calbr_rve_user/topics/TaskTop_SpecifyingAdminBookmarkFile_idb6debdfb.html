Specifying the Admin Bookmark File

You can specify the admin bookmark file either automatically by following a naming convention or with an environment variable. Admin bookmarks are permanent bookmarks associated with the results database.

Procedure

  1. Create the Admin bookmark as described in “Creating Admin Bookmarks” or obtain an Admin bookmark file from another team member.
  2. Specify the Admin bookmark with one of the following methods:

    • Automatic — Name the admin bookmark file results_db.xrvedb, where results_db is the name of the results database, and place it in the same directory as the results database. This file is automatically read when you open the results database.

    • Environment variable — Specify the location of the admin bookmark file with the MGC_RVE_DB_OPTIONS_FILE environment variable, as in this example: 

      setenv MGC_RVE_DB_OPTIONS_FILE /admin/perc_voltage_bookmarks.xrvedb

      If MGC_RVE_DB_OPTIONS_FILE is used to specify an admin bookmark file, any results_db.xrvedb file is ignored, where results_db is the name of the results database.

Results

The Admin bookmarks are listed in the “Admin Bookmarks” section of the Bookmarks menu.
