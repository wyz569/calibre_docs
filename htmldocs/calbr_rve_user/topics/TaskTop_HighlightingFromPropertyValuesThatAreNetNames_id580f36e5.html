Highlighting from Property Values that are Net Names

You can easily highlight from a property value that resolves to a net name. Calibre RVE attempts to match property values to net names in the layout. If a match is found, Calibre RVE adds appropriate right-click menu selections in the detailed view for the result.

For example, a result might make use of property annotations placed on derived layer objects by the DFM Property operation. The NETID, GLOBALNETID, and PSPROPERTY functions are useful in handling net IDs.

Prerequisites

  • A Calibre nmLVS, Calibre DFM, or Calibre PERC database is open in Calibre RVE.

Procedure

  1. Choosing View > Result Options and check the “Result Data Pane” check box to display the Results Data pane.
  2. Select a result.
  3. Right-click the net value for a result property and choose Layout Cell cell, then the desired action in the sub-menu.

    Note:

    Some databases automatically include certain properties, such as DN, Sink, or Source, that contain net numbers and net names. The right-click menu for such properties may include entries for both the explicit property name and value and the corresponding net name, resulting in duplication. For the preceding example, the right-click menu for the DN property results in menu selections “DN Net X0/X2/10” and “Net ibias”, which correspond to the same net.
