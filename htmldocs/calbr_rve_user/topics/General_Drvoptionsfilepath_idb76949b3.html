DrvOptionsFilePath

Key Value for DRC HTML Report Configuration File, highlighting category

Usage: Optional. Valid in Common section only.

Inheritance: Not applicable.

Specifies the directory for a Calibre DESIGNrev preferences file.

If this key is specified, the options in the specified preferences file are combined with the options in other preferences files used by the tool. See “Calibre DESIGNrev Display Preferences File for DRC HTML Reporting” for information on file locations and precedence.

Allowed Values

path

The full path to a directory with a valid Calibre DESIGNrev preferences file with read permission.

Default: $CALIBRE_HOME/shared/pkgs/icv.$(VCO)/examples/rve/templates/batchreporting/

where $VCO is returned by the utility $CALIBRE_HOME/bin/calibre_vco and indicates the vendor, CPU, and operating system.

Examples

DrvOptionsFilePath = $HOME/calibre_HTML

Use the preferences file in $HOME/calibre_HTML.
