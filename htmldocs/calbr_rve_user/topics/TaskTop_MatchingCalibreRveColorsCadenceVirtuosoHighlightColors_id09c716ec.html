Matching Calibre RVE Colors to Cadence Virtuoso Highlight Colors

You can use the same colors for highlighting in Cadence Virtuoso that are used for Calibre RVE histograms and colormaps.

Procedure

  1. In Cadence Virtuoso, choose Calibre > Setup > RVE to open the Setup RVE dialog box.
  2. Enable the setting “Allow custom colors.” This setting does not apply to marker colors.
    Note:

    When the Cadence Virtuoso session is closed, a dialog prompts you to save the display resource file. Click Cancel to discard the changes.
