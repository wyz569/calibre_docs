Waiving Results Using Calibre RVE for DRC

You can mark a result as waived using Calibre RVE for DRC. You can also enable options to include a waiver comment with the result and to execute trigger functions before and after the waiver export.

If your rule file includes either of the specification statements Layout Clone Transformed Placements and Layout Clone Rotated Placements, see “Waived Results from Cloned Cells” for important information on how waiving results from cloned cells is handled.

Prerequisites

  • A Calibre nmDRC results database is open in Calibre RVE for DRC.

  • (Optional) To force the use of the waiver comment dialog box, set the environment variable MGC_CALIBRE_RVE_ENFORCE_WAIVER_COMMENT to 1 before starting Calibre RVE. See “Force Use of Waiver Comment Dialog Box”.

Procedure

  1. (Optional) Enable the option to add a waiver comment as follows:
    1. Choose Setup > Options to open the Options tab.
    2. Choose the Waivers category.
    3. Enable “Show comment dialog while reviewing (PERC only) or waiving results.”
    4. If you want to pre-populate the waiver comment dialog box, enable “Specify initial comments” and fill in the text box that is displayed. To specify multiple initial comments, select the dropdown menu on the right and click Add, then fill in the new text box. To delete an initial comment, select it in the dropdown menu, delete all of the text, and select a new comment.

      When you waive a result, the initial comment text is displayed in the Waived Result Comment dialog box. If multiple initial comments are defined, you can select an initial comment on the right hand dropdown menu, and the text currently open in the Options tab is the default option. You can edit the comment text when you waive a result, or right-click a result and choose Waive > Edit Waived Comment.

    5. Click Apply.
  2. (Optional) To specify a trigger function that runs before or after the waiver file is created, do the following:
    1. Choose the Waivers category on the Options tab.
    2. Enable “DRC Pre-Export Waivers” and/or “DRC Post-Export Waivers” and enter the trigger function call in the text field. The file name of the waiver file is appended to the list of parameters at execution time.
    3. Click Apply.
  3. (Optional) By default, the W keyboard key is used to waive and unwaive results. To prevent the W keyboard key from unwaiving results, do the following:
    1. Choose the Waivers category on the Options tab.
    2. Enable “Restrict the ‘W’ keyboard shortcut to Waive only (DRC only)”. When this option is enabled, the W keyboard does not perform an operation if any selected results are already waived. If all selected results are unwaived when pressing W, all selected results are waived.
    3. Click Apply.
  4. Click the result tab for your database to make it active.
  5. Select the result or result cluster you want to waive.

    You can select a result in the Result View or select a tree node in the Tree View. You can select multiple items with Shift- and Ctrl‑click.

  6. Use the W keyboard key to waive the result(s) or right-click the selection and choose Waive > Waive.

Results

Waived results are indicated with a dim red X ().

The waived results are saved in the file drc_results_database.waived in the directory of the DRC results database, where drc_results_database is the name of the results database. See the discussion “Waived Results in Calibre RVE for DRC” for a discussion of the Calibre RVE .waived file and how to use it. To import a Calibre RVE .waived file, choose Tools > Import Waivers.

Note:

In Calibre Interactive nmDRC, if you choose the option “Preserve cells from RVE waiver files” on the Cells tab of the DRC Options pane, the specified Calibre RVE .waived file is automatically loaded when you start Calibre RVE from your Calibre Interactive session.

You can click the filter icon () and select a filter to display only waived or unwaived results.
