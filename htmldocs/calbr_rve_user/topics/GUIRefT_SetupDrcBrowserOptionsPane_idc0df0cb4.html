Setup DRC Browser Options Pane

Access with: Setup > Options, then choose the DRC Browser category.

This option pane controls settings for the DRC Browser in Calibre RVE.

Fields

Table 1. Setup DRC Browser Options Pane Contents

Setting

Description

Properties

Show calculated properties

Show calculated properties for polygons (area, perimeter) and edges (length, width). See “Viewing Properties in Calibre RVE for DRC”.

Classify result shapes

Show the shape class (#SC) property in the detailed view.

See #SC in “Viewing Properties in Calibre RVE for DRC”for a definition.

Cell Names

Preserve cloned cells for output layout

  • Off — Default. Use original cell name and highlighting information for cloned cells. This setting should be used when highlighting to the original layout.

  • On — Use cloned cell name and highlighting information for cloned cells. This setting should be used when highlighting to a generated layout that contains cloned cells. Cloned cells have a suffix of $Cnum$.

This option affects handling of cloned cells produced by Calibre when Layout Clone Rotated Placements or Layout Clone Transformed Placements is included in the rule file.

The results database is reloaded when this setting is changed.

Check Text Override Files

Append RVE LINK statements to existing check text

If the DRC Results database includes RVE Link rule check comments, merge RVE LINK rule comments found in the database with those found in the DRC RVE Check Text Override File (CTO File) Format. This setting is on by default.

Side RDBs

RDB Type table

Select the desired behavior for each RDB type:

  • Show as Check — Show the RDB type as a check in the tree view. Additional information about the side RDB is given in the check text pane (enabled with View > Check Text Pane)

  • Auto Open — Automatically open all side RDBs of this type when the main results database is opened.

Also see “Viewing Auxiliary Databases”.
