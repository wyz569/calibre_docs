Setting Case-Sensitivity, Highlight, and View Options for the Scripting Interface to the SPICE File Viewer

Prior to printing schematic views with the Tcl scripting interface to the Calibre RVE SPICE File Viewer, you must set the viewer options. The viewer options are set within Calibre RVE and saved to the .rvedb file. The settings in the .rvedb file are used when the Tcl script is executed.

You have the option of saving the .rvedb file and using the MGC_RVEDB_DIR environment variable to point to the directory containing the .rvedb file; this allows you to specify and save settings specifically for use with the Scripting Interface to the Calibre RVE SPICE File Viewer.

Note:

The settings for case sensitivity, NMOS and PMOS device type names, and power and ground nets must be set explicitly for the SPICE File Viewer, unlike the case when viewing a results database in Calibre RVE. When viewing a results database, these options are determined from the rule file associated with the results database; however, a rule file is not available when viewing a SPICE netlist with the SPICE File Viewer.

Procedure

  1. Open Calibre RVE. It is not necessary to load a results database.
  2. Click the button to open the Options tab.
  3. Choose the Highlighting category.
  4. Select the setting for “Clear existing highlights before showing new highlights.”

    The setting for “Clear existing highlights” controls whether multiple highlights or only a single highlight are visible in the schematic image. For example, if “Clear existing highlights” is enabled, only one highlight is included in the output file.

  5. Choose the Schematic Viewer category.
  6. Set options in the top portion of the tab. See “Setup Schematic/Netlist Viewer Options Pane”.
  7. Expand the area “Standalone SPICE Netlist only.”
    1. Select the desired option for “Select objects in case-sensitive manner” as follows:

      • Disabled — (default) Selection in the netlist is case-insensitive.

      • Enabled — Selection in the netlist is case-sensitive.

      For example, suppose you have a subcircuit definition with net A and port a. If “Select objects in case-sensitive manner” is enabled, selecting net A only selects and highlights all other A nets. If “Select objects in case-sensitive manner” is disabled, then port a is also selected and highlighted when net A is selected.

    2. (Optional) Enter NMOS and PMOS device type names. The default is n? and p?, respectively, where the ? matches any number of characters.
    3. (Optional) Edit the default power and ground net names, if needed. Do not clear the entries.
  8. Click Apply.
  9. Close Calibre RVE.

    The .rvedb file is saved to your home directory by default, or to the location specified by the MGC_RVEDB_DIR environment variable.

  10. (Optional) If desired, copy the .rvedb file to a location for use when printing schematics with the scripting interface. This allows you to always use the desired settings, regardless of what settings are in use for your Calibre RVE session.

    For example:

    cp ~/.rvedb ~/rvedb_dir_for_scripting
    setenv MGC_RVEDB_DIR ~/rvedb_dir_for_scripting

Results

You are ready to print schematics, as described in “Using a Tcl Script to Print a Schematic View to File with Calibre RVE”.
