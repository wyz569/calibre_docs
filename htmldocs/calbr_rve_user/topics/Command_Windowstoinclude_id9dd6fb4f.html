WindowsToInclude

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: WindowDrillDown, WindowMemoryPlot

Required key name that specifies a list of windows to include in the report.

Syntax

WindowsToInclude = windowID

Parameters

windowID

A required comma-separated list of window IDs or a reference to a section that outputs window IDs. Can directly reference other sections. Multi-character wildcards (*) are supported.
