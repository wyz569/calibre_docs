Setup DFM Browser Options Pane

Access with: Setup > Options, then choose the DFM Browser category.

The Setup DFM Browser option pane controls settings for the DFM Browser in Calibre RVE.

Objects

Table 1. Setup DFM Browser Options Pane Contents

Setting

Description

Show Layer Browser

Enable to show the Layer Browser tab.

Layers

Specifies the layers to display. Select from:

  • All

  • DFM ANALYZE layers ONLY

  • DFM ANALYZE detail layers ONLY

Columns

Specifies the columns to display for layer information. Select from:

  • Number — The Calibre layer number.

  • Comment — The check text for the layer.

  • Annotations — The current layer annotations.

  • Operation —The operation used to create the layer.

On double-click, browse layer:

Specifies how to browse the layer when double-clicked:

  • Hierarchically — As described.

  • Flat — As described.

Chip Summary

Specifies the columns to display for the chip summary. Select from:

  • Type

  • Group

  • Priority

  • Description

Highlight the Worst Bins

Select to highlight the worst bins on the Chip Summary Table and the CAA Plot.

Disable Waiving of Flat Errors

When enabled, the right-click Waive menu selection is removed from the results tab opened with a right-click Browse Flat Errors in the Chip Summary tab.

Keep sorting

Select to save the column sorting settings when you exit Calibre RVE. The settings are restored when you next open Calibre RVE.
