Property

Key Value for DRC HTML Report Configuration File, histogram and colormap category

Usage: Optional. Valid in DRCHistogram and DRCColormap report sections, and in Common section.

Inheritance: Standard Inheritance

Comma-separated list of property names to use for DRCHistogram and DRCColormap reports. All properties are included by default (Property=*).

Allowed Values

property

The value property is a valid database property name. Wildcards (*) may be used. The default value is *—include all properties.

 

The following calculated properties may be used: #SC, #EC, #EW, #EL, #PA, #PP

 

The Shape Class property (#SC) property is included in DRC HTML reports by default; other calculated properties can be included by setting CalculateProperties=1.
