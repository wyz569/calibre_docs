Tools Menu in Calibre RVE for PERC

The Tools menu in Calibre RVE for PERC includes items related to managing waivers and for loading a Check Text Override (CTO) file. Some menu selections are only available when viewing results from a Calibre PERC LDL run.
Table 1. Tools Menu Contents in Calibre RVE for PERC

Menu Item

(restriction)

Description

Export PERC Waiver Description File

(Topological PERC only)

Opens the Export PERC Waiver Description File Dialog Box, which contains settings for creating a waiver description file based on the waived results in your database.

Import PERC Delta Waiver Files

(Topological PERC only)

Imports a delta waivers file. See “Multi-User Waiver Flow” in the Calibre PERC User’s Manual.

Reset PERC Waivers

(Topological PERC only)

Resets the results view to state (pending, reviewed, or waived) from the most recent Calibre PERC run.

Import Waivers

(PERC LDL only)

Imports a Calibre RVE waived results file and applies the waivers to the currently selected Calibre PERC LDL results tab. See “Waived Results in Calibre RVE for DRC”.

Export Waivers

(PERC LDL only)

Opens the Export Waived Results dialog box, which is used to export waiver shapes for use with Calibre Auto-Waivers. See “Exporting Waiver Geometry Using Calibre RVE” for details. Although a waiver geometry file can be created, there is no method to apply these waivers to a Calibre PERC LDL run.

Create Waiver Report

(PERC LDL only)

Saves a text report of the results that are waived using Calibre RVE. The report is opened in a new tab.

Create CTO File

Opens the Create CTO File dialog box. The generated CTO file includes RVE Show Layer statements based on the rule checks in the rule file. You need both a Calibre PERC LDL CD database and a Calibre PERC rule file. See “Creating a Check Text Override (CTO) File Using Calibre RVE”.

Import CTO File

(PERC LDL only)

Loads a DRC RVE Check Text Override File (CTO File) Format. The CTO file specifies rule check comments which provide additional control over the Calibre RVE highlight actions.

Note:

The RVE Highlight Color check text comment has no effect when highlighting from a Calibre PERC LDL database—the highlight color is determined by the “Highlight color scheme” setting; see “Setting Highlight Color in Calibre RVE for LVS, PERC, and PEX”. The RVE Show Layers check text comment is supported.

Export to Layout

(PERC LDL only)

Displays the Export to Layout Control Bar in Calibre RVE for DRC, which allows you to export result shapes on a cell-by-cell basis to the layout editor.
