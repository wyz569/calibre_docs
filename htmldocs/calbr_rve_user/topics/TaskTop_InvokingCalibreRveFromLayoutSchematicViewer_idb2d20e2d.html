Invoking Calibre RVE From a Layout or Schematic Viewer

Invoking Calibre RVE from a layout or schematic viewer automatically creates a connection between Calibre RVE and the viewer.

Prerequisites

  • “Calibre RVE System Requirements”

Procedure

  1. Depending on your viewer, do one of the following:

    • Calibre DESIGNrev and other Calibre viewers — Choose Verification > Start RVE.

    • Other design tools — Choose Calibre > Start RVE.

    You should see the Calibre RVE startup dialog box as shown in Figure 1.

  2. Enter the path for your database and the (optional) top cell name in the text entry field.

    Example for an LVS database:

    svdb top_cell

    Example to open multiple ASCII DRC results databases:

    drc_results.db density_1.rdb density_2.rdb
    Tip

    You can open multiple DRC/ERC results databases in the file browser by using Shift- and Ctrl-click to select filenames. The file type must be DRC/ERC files.

  3. Select the correct database type from DRC/ERC, DFM, LVS, PERC, PEX, or SPICE File.
  4. Select desired options.

    • Use Calibre-CB license — Select this if you are using the Calibre Cell/Block license package. See “Calibre CB” in the Calibre Verification User’s Manual. This option may not be specified with “Run multi-threaded RVE.”

    • Run multi-threaded RVE — (‑turbo option) Select this to allow multithreaded processing for operations that support this option, such as Interactive Short Isolation in Calibre RVE for LVS and DRC operations executed from a YieldServer script or the Layer Browser tab in Calibre RVE for DFM. This option is not available if the database type is DRC/ERC or SPICE, and may not be specified with “Use Calibre-CB license.”

      See the section “Multithreaded Parallel Processing for Calibre RVE Operations (‑turbo Option)” for how the ‑turbo option is set automatically in certain cases.

    • Run RVE on — Select the computer that Calibre RVE runs on from the following selections:

      • Local Host — The default.

      • Remote Host — Enter the remote host name in the text field to run Calibre RVE on a remote host.

      • Cluster — Enter the queue command in the text field to run Calibre RVE on a computer cluster.

      For complete details on specifying a remote host or cluster, see “Running Calibre RVE on a Remote Host”.

  5. Click Open.
