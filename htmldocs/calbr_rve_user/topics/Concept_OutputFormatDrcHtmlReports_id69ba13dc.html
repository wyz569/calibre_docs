Output Format for DRC HTML Reports

You can specify the format of the report as standard HTML (the default) or single file MHTML with the OutputFormat key.

HTML output is the default; it is not necessary to specify an HTML format report. To specify a single file MHTML format report, include “OutputFormat = mht” in the Common section of the configuration file. Do not include the quotes.

Supported Browsers for MHTML Formatted Reports

The MHTML formatted report has been tested with the following browsers:

  • Firefox version 10.0.12 with UnMHT Extension Version 5.7.7 on Linux®1

  • Firefox version 24.0 with UnMHT Extension Version 6.3.3 on Microsoft Windows

  • Internet Explorer version 8.0.7601.17514 on Microsoft Windows (no extension required)

  • Google Chrome version 30.0.1599.69m on Microsoft Windows (no extension required)

    In MHTML format when viewing the report using the Google Chrome browser, the link to the rule file does not resolve.

    Tip

    If you are creating the DRC HTML report from Calibre RVE or Calibre Interactive, you can set the environment variable MGC_RVE_HTML_BROWSER to the path for a browser executable to specify the HTML browser that is opened when viewing the DRC HTML Report. A recent browser version is recommended, as unexpected results may occur in older browsers. Your default browser is used if this environment variable is not set.
