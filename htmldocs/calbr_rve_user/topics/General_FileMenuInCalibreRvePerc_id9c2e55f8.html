File Menu in Calibre RVE for PERC

The File menu in Calibre RVE for PERC includes the Save Results As selection.

See Table 1 for menu items common to all applications.

Table 1. File Menu Contents in Calibre RVE for PERC

Menu Item

Description

Save Results As

  • Calibre PERC LDL results — “Saving DRC Results Data to CSV Format from Calibre RVE for DRC”
