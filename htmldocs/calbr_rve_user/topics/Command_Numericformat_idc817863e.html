NumericFormat

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: Common section only

Optional key name that defines the format for floating-point values.

Syntax

NumericFormat = {auto | float | sci}

Parameters

auto

A required literal that reports numerical values as either floating-point numbers or in scientific notation.

float

A required literal that reports numerical values as floating-point numbers.

sci

A required literal that reports numerical values in scientific notation.

Description

Applies to CAPlot, CellDrillDown, CellSummary, ChipSummary, MemoryPlot, NetDrillDown, NetSummary, TotalSummary, WindowDrillDown, WindowMemoryPlot, WindowSummary.
