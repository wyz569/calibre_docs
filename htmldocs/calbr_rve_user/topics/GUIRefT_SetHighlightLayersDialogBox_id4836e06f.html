Set Highlight Layers Dialog Box

Access with: Highlight > Set Check Highlight Layers

Assigns the layers or layer colors that are used in the layout viewer for highlight operations.

Figure 1. Set Highlight Layers Dialog Box

Objects

Table 1. Set Highlight Layers Dialog Box Contents

Menu Item

Description

Override Check Defaults

Enabled — Select Auto, Colors, or Layer Indexes for the highlight layer behavior. The selections in the dialog box override settings from RVE Highlight Color and RVE Highlight Index rule check comments if they are present in the DRC results database.

Disabled — (default) The highlight layer behavior is controlled by RVE Highlight Color and RVE Highlight Index rule check comments in the DRC results database; the selections in the dialog box are not active.

When “Override Check Defaults” is disabled, the dialog box displays what is found in the results database:

  • No rule check comment — “Auto” indicated.

  • RVE Highlight Color — “Colors” indicated and settings displayed.

  • RVE Highlight Index — “Layer Indexes” indicated and settings displayed.

If both RVE Highlight Index and RVE Highlight Color comments are found, the display indicates “Colors”.

When “Override Check Defaults” is disabled, highlight color selections saved to the .rvedb configuration file are not used.

Auto

Use a new layer for each highlight operation.

Colors

Assign a layer color for each rule check. All errors from a given rule check are highlighted to the specified color.

Color assignment only works with Calibre DESIGNrev/WORKbench, Cadence Virtuoso, Synopsys IC Compiler, and Synopsys Laker layout editors. Color selections that alter the default selections are saved to the .rvedb File for Calibre RVE Settings.

If Reset Color or Reset All Colors is chosen in the right-click menu, then colors are reset to the default color or the color specified in the results database with a RVE Highlight Color rule check comment, if present.

When Colors is selected with the Calibre DESIGNrev or WORKbench layout viewer, the highlight layer is labeled with the check name.

Layer Indexes

Assign a layer index for each rule check. All errors from a given rule check are highlighted to the specified layer index.

Indexes are not assigned automatically; you must assign an index to each layer. To assign an index, do one of the following:

  • Click in the layer column, enter a value, then press Enter on the keyboard. Do this for each rule check.

  • Right-click and choose Linearize to automatically assign all layer indexes starting from one.

 

Right-click menu:

  • ++ — Increase index by 1

  • -- — Decrease index by 1

  • Set — Open dialog box to set the layer index. You can select multiple rule checks, then open the dialog box to assign the same layer index to all selected rule checks.

  • Linearize — Automatically assign all layer indexes starting from one.

 

Most layout viewers limit the number of highlight layers that are used. Make sure the range of layer indexes entered in the Highlight Layers dialog box does not exceed the number of available highlight layers in the layout viewer. If this happens, the highlight layers used wrap, possibly causing unexpected results.

Layer index assignment works with all layout editors. Layer index selections are not saved when you exit Calibre RVE.

OK

Accept selections.

Color selections that alter the default selections are saved to the .rvedb file. Layer index selections are not saved when you exit Calibre RVE.

Usage Notes

You can also select the layer assignment method from the Options tab; see the setting “Override check defaults to set the highlight layer” in the Highlighting category, “DRC/DFM Highlighting” section. Default colors are assigned automatically when you select Colors. Layer indexes are not assigned automatically if you select Layer Indexes on the Options tab; you must assign layer indexes using the Set Highlight Layers dialog box for the setting to take effect.
