Requirements for DRC HTML Reporting

DRC HTML Reporting requires a Calibre nmDRC results database and a configuration file. Other optional input files are required depending on report options and customization.

The following are required:

  • A Calibre RVE license.

  • A DRC results database. Results database files compressed with gzip may be used.

  • A DRC HTML Report Configuration File Format.

  • If layout images are included in the report, the following are needed:

    • A layout database in GDS, OASIS, or one of the formats supported by Calibre MDPview.

    • A Calibre DESIGNrev license (for layout databases in GDS or OASIS format) or a Calibre MDPview license.

    • A DRC HTML Report Database Map File.

  • File permissions for /tmp/.X11-unix and /tmp/.X11-pipe must be set to 1777.

    DRC HTML reporting uses a virtual frame buffer (XVfb) to create the report images without opening a display and these files are used by the virtual frame buffer. Your system may already have these permissions set correctly; if not, the tool reports an error.
