Property Statistics Tree - Group by Property Name

The property tree displays the results grouped by property and gives statistics on each property. Select View > Tree Options > Group By > Statistics … for this view.

You can also click the Group By button to select the grouping method.

Figure 1. Property Statistics Tree

You can also right-click in a column header for a context sensitive menu of view options. You can select the columns that are displayed by clicking in the column selection icon to get a dropdown list.

Property Right-click Menu — Right-click a property or result, then select Histogram or Colormap to display a histogram or colormap of the selected property. See “Creating Histograms and Colormaps” and “Histograms and Colormaps” for more information. The complete right-click menu is given in Table 1.

Tip

You may need to resize the window sections when a histogram is first displayed.
