Using Calibre RVE for DRC

Calibre RVE for DRC displays results in a tree view and includes options for sorting and filtering the results. You can highlight results in a connected layout viewer and mark results as fixed or waived.

This chapter assumes you have a DRC Results Database or RDB open for viewing and are familiar with the general operation of Calibre RVE. See “Getting Started with Calibre RVE” for general information on Calibre RVE. Also see Calibre RVE for DRC Quick Reference for a quick reference card with the main features of Calibre RVE for DRC.

You can create HTML reports based on views in Calibre RVE for DRC; see “Creating a DRC HTML Report from Calibre RVE for DRC”.

You can convert results from an ASCII format Calibre results database (RDB) to error markers and route guides in your Place and Route tool. This allows you to use familiar tools when debugging errors and correcting the design. See “Calibre RDB Conversion: Working with Calibre Results in Place and Route Tools”.

Video: The video “Getting Started With Calibre RVE for DRC” demonstrates setting options, highlighting and waiving results, viewing properties, creating a histogram, and setting the highlight color.
