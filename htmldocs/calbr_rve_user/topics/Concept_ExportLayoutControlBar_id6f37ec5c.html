Export to Layout Control Bar

The Tools > Export to Layout menu selection opens a toolbar that allows you to export results on a cell-by-cell basis to the layout editor.
Figure 1. Export to Layout Control Bar

Table 1. Export to Layout Control Bar Contents

Control

Description

Export Displayed Results or Selected Results

Select what results are exported in the dropdown list:

  • Displayed Results — Export the results shown in the Calibre RVE display. You can use the DRC Result Filter to filter results.

  • Selected Results — Export the results that are selected in the Calibre RVE display. You can use Shift- and Ctrl-click to select multiple results.

to Layer

Specify the name or number of the layer to export results to.

 

Cadence Virtuoso users can specify a layer name or number and an optional layer purpose separated by “ ” (space). If the layer purpose is not specified, it defaults to drawing.

in Cell

Select “Top Cell” or “Selected Cell” to export the results to that cell.

Export

Click the Export button to export the results to the layout.

Usage Notes

  • Calibre RVE does not provide for deletion of such exported results. The added layers can not be removed with the Undo command in the layout editor. However, they can be selected and deleted within the layout editor.

  • All the settings that affect highlighting also affect the export to layout operation, such as the Highlight > Highlight in Context setting. The settings “Show Check Names”, “Show Result Numbers”, and “Show Properties” (on the Options tab in the Highlighting category, “DRC/DFM Highlighting” area) allow for the export of check names, result numbers, and properties along with the results.

    CAUTION:

    Follow your company’s tape-out methodologies when generating databases for delivery to foundries for manufacture. The Tools > Export to layout operation in Calibre RVE for DRC is not intended for this purpose.
