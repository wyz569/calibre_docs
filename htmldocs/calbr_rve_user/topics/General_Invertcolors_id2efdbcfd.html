InvertColors

Key Value for DRC HTML Report Configuration File, histogram and colormap category

Usage: Optional. Valid in DRCHistogram and DRCColormap sections, and in Common section.

Inheritance: Standard Inheritance

Specifies the order in which colors are used for histograms and colormaps.

Allowed Values

0

Use colors first to last from the list of colors. (default)

1

Use colors in reverse order (last to first).
