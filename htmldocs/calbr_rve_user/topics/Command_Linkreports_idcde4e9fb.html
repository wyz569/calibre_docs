LinkReports

Key value for Configuration File in DFM HTML Reporting

Supported Report Types/Sections: ByCellColormap, ByCellHistogram, ByErrorColormap, ByErrorHighlight, ByErrorHistogram, ByErrorResults, ByErrorResultsSummary, ByWindowColormap, ByWindowHistogram, CAPlot, CellDrillDown, CellSummary, ChipSummary, Common section, MemoryPlot, NetDrillDown, NetSummary, TotalSummary, WindowDrillDown, WindowMemoryPlot, WindowSummary

Optional key name that controls display of section reports in the summary report.

Syntax

LinkReports = {0 | 1}

Parameters

0

A required literal that excludes hyperlinks from the summary report.

1

A required literal that includes hyperlinks in the summary report. This is the default.
