Viewing perc::trace Results in Calibre RVE

You can use Calibre RVE for PERC to highlight the nets and pins listed in the output of a perc::trace command.

The perc::trace command is used in voltage propagation rule checks. Given an initial input net the command traces backward through the netlist reporting the net's driven pins and their voltage characteristics along the way. The perc::trace command produces text output that can be viewed in the Result Data Pane of Calibre RVE.

You can use the Voltage Trace GUI or selection-based highlighting to view results.

Prerequisites

  • A Calibre PERC results database that includes results from a perc::trace command.

  • The Calibre PERC database is open in Calibre RVE for PERC. See “Examining the Calibre PERC SVDB”.

Procedure

  1. Enable View > Results Options > Result Data Pane.
  2. Select a result in the result view. The output of the perc::trace command is displayed in the Result Data pane.
  3. To open the Voltage Trace GUI, right-click a result in the result view and choose Trace.

    See “Using the Voltage Trace GUI in Calibre RVE for PERC” for details.

  4. To highlight a specific net or pin listed in the perc::trace output, do the following:
    1. Select the net or pin name in the Result Data pane.
    2. Right-click and choose Highlight Pin or Highlight Net.

    See “Highlighting Using Text Selection in Calibre RVE for LVS and Calibre RVE for PERC” for a more detailed description.
