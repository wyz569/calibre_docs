Controlling What Level of Detail You See

In some cases the window size needed to view all of your data can be very large in relation to the size of the geometries. When this occurs, your geometries may not display properly. To ensure that you see the geometries correctly you can change the size of the smallest object that is displayed.

Procedure

  1. The first method to adjust the level of details displayed is as follows:
    1. Choose Edit > Preferences.
    2. Select View tab.
    3. Change Minimum Object size to draw text entry box to 0.
    4. Apply changes.

      To improve drawing performance, Calibre MDPview applies a resolution process known as “culling.” With this process, if something in the view is smaller than x (specified by Minimum Object size to draw), then a box of fixed size x by x is drawn in its place. For example, if you specify that 10 pixels is the smallest size limit, anything with a side less than 10 pixels is drawn as a 10x10 box. This can lead to some drawings becoming coarser, but culling greatly improves performance.

  2. Another method that prevents the dropping of shapes (at a cost of increased draw time) is as follows:
    1. Select the View menu. The Choose Viewing Detail dialog appears.
    2. In the Chose Viewing Detail dialog box, select Custom, then set the minimum object pixel size and detailed reference pixel size to 1.
    1. Click Apply.

Results

Calibre MDPview updates its display to the detail level set.
