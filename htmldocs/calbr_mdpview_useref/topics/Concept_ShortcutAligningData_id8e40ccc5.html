A Shortcut for Aligning Data

If the two layouts area magnified, mirrored, and rotated properly, but still not aligned right, you can use the Move function to reposition the data. The challenge here is that to select a “cell” in a merged layout1, you must have only the cell outline visible, but to align it properly, you must see the contents. You can get around this by changing the view depth after putting the cursor in Move mode and selecting the cell.
