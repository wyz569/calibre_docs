mpcCompile

Test Layout Generation Commands

Generates a proximity, uniformity, and mask model test layout (combines the execution of ptpCompile, uniCompile, and mtpCompile).

Usage


          mpcCompile
          mtpfilename
          
          ptpfilename
          
          unifilename
        

Arguments

mtpfilename

Specifies the input file. For information on the input file. Refer to “mtpCompile Input File Format” for further information.

ptpfilename

Specifies the proximity test pattern specification file. Refer to “ptpCompile Input File Format” for further information.

unifilename

Specifies the uniformity test pattern specification file and corresponding gauge file. The file format is the same as the ptpCompile input file. Refer to “ptpCompile Input File Format” for further information.

Description

This command combines the ptpCompile, mtpCompile, and uniCompile commands into a single execution. The inputs are the same for each of the separate commands.
