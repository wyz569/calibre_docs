Additional Modeling Operations in MPC Center

MPC Center also contains two additional tabs, Contour and Remote.

  • The Contour tab in MPC Center is identical to the Contour tab (see Figure 1) described for the CM1 Center as documented in the Calibre WORKbench User’s and Reference Manual. This tab is used to read a Contour Layer Information (CLI) file, which is supplies information about an additional contour calibration layout file.

    Figure 1. Contour Tab

  • If you have a distributed hardware system, the Remote tab (see Figure 2) is used to create a remote host file and implement distributed processing. See the Calibre WORKbench User’s and Reference Manual for complete details.

    Figure 2. Remote Tab
