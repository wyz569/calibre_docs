uniCompile

Test Layout Generation Commands

Generates a uniformity test mask layout and corresponding spreadsheet.

Usage

uniCompile unifilename

Arguments

unifilename

Specifies the uniformity test pattern specification file and corresponding gauge file. The file format is the same as the ptpCompile input file (see “ptpCompile Input File Format”).

Description

Generates a uniformity test mask layout and corresponding spreadsheet.

Figure 1. Example uniCompile Output Pattern

Examples

Create a uniformity test pattern:

% uniCompile uniformity.in
