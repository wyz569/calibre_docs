pxSMO Job:cDOF Pane

To access: In the Session Browser, activate a session and click Job:cDOF in the Setup Items tree.

The pxSMO Job:cDOF pane enables setup of the SMO run to maximize common depth of focus before execution, or viewing of settings when the run is complete. If EL/ILS optimization is selected, the labels change to Job:EL and the SMO run will maximize exposure latitude.

Description

The pxSMO Job:cDOF pane manages the settings that maximize common depth of focus for a specified exposure latitude. If EL/ILS optimization is selected, then the labels change to Job:EL and job maximizes exposure latitude for a specified depth of focus. Most settings are used by both job types.

Description

Extra PW conditions for extra print suppression table

Turns on SRAF print suppression for specified process window conditions. Nominal condition does not appear in the table.

Optimize

Specifies the process window value to optimize.

Selecting Depth of Focus freezes the exposure latitude value during optimization.

Selecting EL/ILS freezes the depth of focus value during optimization.

PW Type

Specifies the shape to inscribe in the process window for calculating depth of focus. Default is Ellipse.

Exposure Latitude

Specifies the exposure latitude percentage when common depth of focus is optimized. Default is 6 percent.

DOF Starting Value

Specifies the initial depth of focus in nanometers when common depth of focus is optimized. Default is 100 nm.

EL Starting Value

Specifies the initial exposure latitude when EL/ILS is optimized. Default is 6 percent.

Depth of Focus

Specifies the depth of focus in nanometers when exposure latitude is optimized. Default is 100 nm.

Improved Nominal Convergence

Improves EPE for the nominal process window condition. Default is on.

Weight — Specify higher values to improve EPE. Default is 10.

Focus Maximization Factor

Controls the relative emphasis given to common depth of focus during process window optimization. Used only with CDOF jobs.

Focus Base Points

Specifies the range of defocus conditions for through-focus aerial image calculation. Used only with CDOF jobs.

Points

Specifies the number of focus base points to use for through-focus aerial image calculation. Used only with CDOF jobs.

Extra Print Suppression Factor

Controls the relative emphasis given to SRAF print suppression. Default is 10.

Dose Maximization Factor

Controls the relative emphasis given to exposure latitude during process window optimization. Used only with EL jobs. Default is 100.

Center Dose Optimization

Center Focus Optimization

Enables optimization of the dose center and focus center during process window optimization. Default is optimization disabled.

Job:EL Options

File Save

Saves files for debugging. File names can be specified.

  • Source — Saves source map file.

  • Mask — Saves OASIS file of all clips.

  • Image — Saves contours of all clips.

  • Images (cdof) — Saves input layers and process window contours.
