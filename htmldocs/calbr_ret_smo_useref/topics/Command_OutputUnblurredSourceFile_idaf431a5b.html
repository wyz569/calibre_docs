output_unblurred_source_file

PxSMO Setup File Commands

Outputs an illuminator-level source map file at the end of a session run.

Usage

output_unblurred_source_file filename

Arguments

filename

A required argument that specifies the name of the file for the source map at the end of the session run. The convention is to use a .src suffix, but this is not applied automatically.

The source map is in text format. Each line consists of x and y coordinates and the filtered and smoothed intensity of a pixel.

Description

An optional command that specifies the name of an illuminator-level source map file saved at the end of a session run. The illuminator-level source differs from a reticle-level or wafer-level source by removing blurring caused by the optical system. The command is added to the general section of a setup file to produce a source suitable for the design of a diffractive optical element (DOE).

The file is saved in the session directory.
