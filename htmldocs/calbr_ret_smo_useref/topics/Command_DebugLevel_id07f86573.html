debug_level

PxSMO Setup File Commands

Writes optimization rates to stdout.

Usage

debug_level {0 | 1}

Arguments

0

A required argument indicating no extra information should be printed. This is the default.

1

A required argument that causes additional information to be printed to standard output (stdout). The additional information includes the following:

  • Optimization rates and clip weights.

  • For CDOF optimization, post-optimization values for focus_range, focus_center, and dose_center variables.

Description

The debug_level command controls whether optimization values are output. This can be useful for tuning recipes.

Note:

Do not use this command in a production setup file. It makes the log file very large and slows down the run.

Examples

debug_level 0
