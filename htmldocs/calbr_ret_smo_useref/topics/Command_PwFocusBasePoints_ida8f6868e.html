pw_focus_base_points

PxSMO Setup File Commands

Specifies defocus conditions for through-focus aerial image calculation. For job types other than CDOF, this command is required when focus_range or focus_center is specified by the pw_variables command.

Usage

pw_focus_base_points focus_list

Arguments

focus_list

A required argument specifying a set of defocus conditions in nanometers. At least two values must be specified. This command is applied per job. For a CDOF job, the default set is a range from -100 nm to +100 nm with 20 nm steps.

Description

A required command when focus_range or focus_center is specified by the pw_variables command in a job that is not job type CDOF. A CDOF job applies a default set of defocus conditions from -100 nm to +100 nm with 20 nm steps. Custom defocus conditions for a CDOF job can be specified using pw_focus_base_points.

Examples

pw_focus_base_points -100 -80 -60 -40 -20 0 20 40 60 80 100
