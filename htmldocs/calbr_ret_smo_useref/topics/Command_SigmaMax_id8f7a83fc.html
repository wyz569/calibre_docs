sigma_max

PxSMO Setup File Commands

Controls the source manufacturability constraint on maximum allowable sigma for the optimized source.

Usage

sigma_max sigma

Arguments

sigma

A required positive number less than or equal to 1. A source pixel with its center outside sigma_max is also included in final source. The default value is 0.98.
