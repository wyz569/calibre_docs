set_on_target

PxSMO Setup File Commands

Improves EPE for the nominal process window condition.

Usage

set_on_target cond_name [cond_name...] [weight weight]

Arguments

cond_name

A required argument that specifies the names of process window conditions. At least one process window condition name must be specified and additional names are optional. The name must also be used as the cond_name in pw_condition and pw_select statements.

weight weight

An optional argument that specifies the weight to give to the EPE defined for this process window condition. The default is 1.0.

Description

An optional command that improves the EPE for specified process window conditions. The name of the process window condition must be specified as the cond_name. For a CDOF job, specifying the nominal condition is recommended.

Examples

set_on_target nominal weight 5.0
