suppress_extra_printing

PxSMO Setup File Commands

Specifies SRAF printing suppression for process window conditions or process window condition groups.

Usage

suppress_extra_printing {cond_name | pw_group_name} [cond_name...] [pwgroup_name...] [weight weight]

Arguments

cond_name | pw_group_name

A required argument specifying the name of at least one process window condition or process window condition group. A process window condition name must also be used as the cond_name in a pw_condition statement. A process window condition group name must also be used as the pw_group_name in a pw_condition_group statement. Each name must also be specified in a pw_select statement. The default action is SRAF printing suppression applied to all conditions and groups.

weight

An optional argument specifying a weight given to SRAF printing suppression. The default is 1.0.

Description

An optional command that minimizes SRAF printing at specified process window conditions or process window condition groups. In a CDOF job with no conditions or groups specified, the weight value is applied to all groups selected for the job.

Examples

suppress_extra_printing pw_outer1 PWG1 5.0
