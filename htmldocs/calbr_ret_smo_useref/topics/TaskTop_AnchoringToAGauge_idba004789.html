Anchoring to a Gauge

You can anchor the litho model threshold to a gauge that is critical to the design.

Anchoring adjusts the threshold used by the litho model to make a specified anchor gauge print on target for a specified bias. The bias is a uniform sizing applied to all target edges of the clip to produce a mask layer for simulation. Anchoring is available in both the Free-Form pxSMO and Parametric Explorer session modes.

Anchoring can be activated before, during, and after the SMO step. The Pre-SMO step performs anchoring and sets the threshold used during SMO. The Post-SMO anchoring step performs an anchoring refinement after the SMO step. Anchoring during the SMO step uses a dynamic anchoring process. During each pxSMO iteration, anchoring checks the print quality of the anchor gauge and adjusts the litho model threshold. This improves the robustness and consistency of the optimization.

Prerequisites

  • A layout/gauge pair and session.

Procedure

  1. In the Setup Items tree, select pxSMO. In the pxSMO Setup pane, click the Set button next to Anchor/Thres/Dose.
  2. In the Anchor/Threshold/Dose Settings Dialog Box, select the stage to apply anchoring and an anchoring method.
  3. If anchoring on a selected gauge, select an anchor gauge from the pulldown menu.
  4. To apply CD bias to the anchor feature, activate Bias and specify either an absolute or relative mask feature size.
    Note:

    The mask feature sizing is applied to the feature; half of the relative mask feature sizing is applied to each edge.

    To specify the type and size range of features to apply CD bias, fill out the Bias Config table.

  5. To use focus centering when anchoring a Pre-SMO or Post-SMO stage, choose either the manual or automated option.

    • Choose MANUAL (relative) to specify a value for focus center.

    • Choose Perform Focus Centering to automate focus centering. It is recommended to manually select focus-sensitive gauges for the centering analysis.

      To adjust the beam focus and defocus start using the target CD, activate and set the Defocus Start Adjustment. To adjust the parabola fitting for focus centering, adjust the maximum defocus value and the number of points used for fitting.

  6. (Optional) You may deactivate using the anchor clip in OPC and Verify by unchecking OPC. The default behavior is to use the anchor clip in the OPC and Verify stages.
  7. Click Set to accept the settings and close the dialog box.

Results

When you are done, the Anchor/Thres/Dose setting displays the current settings in the pxSMO Setup pane.
