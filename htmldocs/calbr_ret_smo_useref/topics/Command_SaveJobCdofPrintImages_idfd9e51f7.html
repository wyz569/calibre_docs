save_job_cdof_print_images

PxSMO Setup File Commands

Specifies the input layers and process window contours to save for CDOF jobs. Only valid for CDOF jobs.

Usage

save_job_cdof_print_images layout_filename [focus from cdof_focus_min to cdof_focus_max step cdof_focus_step] [dose from cdof_dose_min to cdof_dose_max step cdof_dose_step]

Arguments

layout_filename

A required argument that specifies the name of an OASIS file of contours saved at the end of a job.

focus from cdof_focus_min to cdof_focus_max step cdof_focus_step

An optional argument set that specifies the focus range and step size for saving contours. The specified focus range should be within the focus range specified for the CDOF job or within the ±100nm range default of the CDOF job. By default, the values are obtained using 20 steps across a focus range determined at the end of optimization. The values are in nanometers.

dose from cdof_dose_min to cdof_dose_max step cdof_dose_step

An optional argument set that specifies the dose range and step size for saving contours. If values are not specified, then the values are obtained using 10 steps across the dose latitude specified by the user.

Description

An optional command that specifies the name of a file of contours of all clips to save at the end of a CDOF job when the option is set. The contours are saved for specified focus-dose pairs. Focus-dose sampling greater than the PW Table values can be used. The command is added to a setup file for job-level debugging. The option applies only to the job where it is specified.

The following contours are saved:

  • All input layers to the pxSMO stage using a layer statement (target, smooth target, and so on).

  • All print images for the process window conditions of the run. Each print image contains contours of all clips. The contours are named “ff_ddose” where f is the focus in nanometers and dose is the exposure dose. Each focus-dose pair is on a separate layer in the layout file.

The file is saved in the session directory.

Examples

Example 1

save_job_cdof_print_images smo_cdof.oas focus from -50 to 50 step 25 dose from 0.96 to 1.04 step 0.02
