initial_source

PxSMO Setup File Commands

Sets the initial source type, overriding any optical sources passed in the litho model.

Usage

initial_source type

Arguments

type

A required keyword specifying the type of the source. The default is conventional. Specify one of the following:

  • conventional — A single circular illuminated area with sigma 1.0.

  • annular — A single ring with sigma from 0.75 to 1.0.

  • quasar — Four illuminated spots on the outer rim. Sigma is from 0.75 to 1.0. Phi extends 0.1π around the points 1/4π, 3/4π, 5/4π, and 7/4π.

  • quadrapole — Four illuminated spots on the outer rim. Sigma is from 0.75 to 1.0. Phi extends 0.1π around the points 0, 1/2π, π, and 3/2π.

  • horiz_dipole — Two illuminated spots on the outer rim. Sigma is from 0.75 to 1.0. Phi extends 0.1π around the points 0 and π.

  • vert_dipole — Two illuminated spots on the outer rim. Sigma is from 0.75 to 1.0. Phi extends 0.1π around the points 0.5π and 1.5π.

  • random — All pixels within the normal conventional area receive a random value for intensity. The integral over the source shape equals 1.

  • filename — A filename of a file containing a source map (.src).

In the descriptions above, sigma is measured from the center of the source out to the edge, and phi is measured around the circumference, with 0 at the rightmost point.
