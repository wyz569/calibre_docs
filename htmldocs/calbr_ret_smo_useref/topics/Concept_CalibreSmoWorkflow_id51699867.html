Calibre SMO Workflow

The Calibre SMO solution is integrated with the Calibre® WORKbench™ engine. It produces both a manufacturable mask and an optimized source. Generally process engineers run SMO early in the process setup phase to define the achievable technology possibilities.

Using the RET Selection window, accessed from within the Calibre WORKbench GUI, you can quickly set up an entire suite of experiments and compare the results. Each experiment uses the parameters you configure to do the following:

  1. Runs Calibre® pxSMO™ to optimize the source.

  2. Runs Calibre® pxOPC™ to optimize the mask with the source.

  3. Runs Calibre® OPCverify™ to check final results.

After the experiments complete, you can view the source and analyze the results within the RET Selection window.

This manual only covers the Calibre pxSMO and RET Selection interfaces.
