pw_tolerance_bound

PxSMO Setup File Commands

Requires the process window conditions for a process window condition group to produce contours within a specified area. Used when optimizing common depth of focus.

Usage

pw_tolerance_bound pw_group_name {outside | inside} layername

Arguments

pw_group_name

A required name specifying a process window condition group. The name must also be used as the pw_group_name in pw_condition_group and pw_select statements.

outside | inside

A required argument specifying the type of contour.

layername

A required name specifying a layer with the inner or outer contour (typically, either the inner or outer contour of a tolerance band). The name must also be used as the layername in a layer statement.

Description

An optional command that requires the process window conditions for a process window condition group produce print image contours within a specified area. The print image contours must either be outside of an inner contour or inside an outer contour. This command can be used two times per job - once for the inner contour and once for the outer contour.



Examples

pw_tolerance_bound PWG1 inside tolband_outer
pw_tolerance_bound PWG1 outside tolband_inner
