Glossary

active session

The session that will receive any changes you make in the setup panes. There can be only one active session at a time.

clip

A rectangular area of the layout used for simulation. Each clip should be distinct and separate from other clips in the layout.

global litho model

A litho model for extreme ultraviolet (EUV) processes. Global litho models support variant optical models for different x-coordinates (slits) in the EUV field. See “EUV Through-Slit Litho Model Extension” in the Calibre OPCverify User’s and Reference Manual for details.

job

A phase in the optimization. It solves for a particular combination of objective functions.

pane

A defined area within a larger window. The main RET Selection panes are able to be pulled out of the RET Selection window, so are sometimes also referred to as windows. These child windows also contain panes.

source mask optimization (SMO)

The process of optimizing an optical source and a mask layer in tandem in order to most accurately manufacture a desired layout.

tolerance bands (tolbands)

A pair of boundaries around a target indicating acceptable edge placement for a resist contour. Tolerance bands comprise the inner and outer boundaries for resist contour edges across a range of dose and focus conditions.
