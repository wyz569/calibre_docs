fixed_source_optimization_region

PxSMO Setup File Commands

Optimizes source pixel intensities in a specified part of a source while other pixels remain fixed.

Usage

fixed_source_optimization_region type sigma_min sigma_min_val
sigma_max sigma_max_val illumangle illumangle_val rotangle rotangle_val
{inside|outside}

Arguments

type

A required keyword specifying the type of the source. Specify one of the following:

  • standard — A single circular illuminated area defined by a sigma_max value.

  • annular — A single ring defined by sigma_max and sigma_min values.

  • dipolex — Two illuminated spots on the horizontal axis. Defined by illumangle, sigma_max, and sigma_min values.

  • dipoley — Two illuminated spots on the vertical axis. Defined by illumangle, sigma_max, and sigma_min values.

  • mirrored pole — Two illuminated spots on a rotated axis. Defined by rotangle, illumangle, sigma_max, and sigma_min values.

  • quasar — Four illuminated spots on axes rotated 45 degrees from horizontal and vertical. Defined by illumangle, sigma_max, and sigma_min values.

  • cquad — Four illuminated spots on the horizontal and vertical axes. Defined by illumangle, sigma_max and sigma_min values.

In the descriptions above, sigma is measured from the center of the source out to the edge.

sigma_min sigma_min_val

An argument set specifying the inner sigma of the source optimization region. Required for annular, dipolex, dipoley, mirrored_pole, quasar, and cquad source regions.

sigma_max sigma_max_val

A required argument set specifying the outer sigma of the source optimization region.

illumangle illumangle_val

An argument set specifying the illumination angle of the source optimization region. Required for dipolex, dipoley, mirrored_pole, quasar, and cquad source regions.

rotangle rotangle_val

An argument set specifying a custom rotation angle for the source optimization region. Required for mirrored_pole source regions.

inside | outside

An optional argument specifying what part of the source is optimized. The inside setting optimizes pixels inside the defined source shape; the outside setting optimizes outside. The default behavior optimizes pixels inside the defined source shape.

Description

An optional command that optimizes the pixels in a specified region of the source. By restricting optimization to specific regions and fixing pixel intensities in the remaining regions, a runtime benefit may be realized.
