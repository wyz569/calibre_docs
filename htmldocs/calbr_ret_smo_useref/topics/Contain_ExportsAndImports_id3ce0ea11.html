Exports and Imports

The RET Selection interface offers four types of relocatable containers for sessions and results.
Archive

Archives are the most complete. They include the layout, session, results, and Tcl scripts to re-run the session.

Layout

Layouts are the layout/gauge pair, without any sessions. This can be useful for creating slight variants on a layout/gauge pair for constructing experiments.

Session

Sessions include process window information, settings, and results, but cannot be run outside the RET Selection interface. They can also include the layout/gauge pair.

Testcase

Testcases are set up to be run outside the GUI. They include the layout/gauge pair information, all setup files, and a Tcl runscript. Testcases do not include results. Also, testcases cannot be imported to another RET Selection database.

Use archives, layouts, and sessions for moving data between different RET Selection users. Use testcases when contacting customer support, or as the basis in creating your own command-line scripts.
