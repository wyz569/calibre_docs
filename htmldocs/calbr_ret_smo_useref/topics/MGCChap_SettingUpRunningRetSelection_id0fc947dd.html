Setting Up and Running

Calibre pxSMO source-map optimization is most easily set up using the RET Selection interface. It prompts you for required settings and makes it easy to select critical areas.

Open the RET Selection window through the Calibre WORKbench menu Litho > RET Selection.

Calibre pxSMO can also be run without the RET Selection interface. This is referred to as batch mode. It can be useful when you have a large number of variations to run. Results can be retrieved for later analysis using the RET Selection tools. Batch mode still requires the Calibre WORKbench engine for processing.
