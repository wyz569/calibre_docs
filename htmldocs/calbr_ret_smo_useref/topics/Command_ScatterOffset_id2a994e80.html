scatter_offset

PxSMO Setup File Commands

Specifies the minimum distance between the main feature and positive SRAFs.

Usage

scatter_offset value

Arguments

value

A required argument in microns specifying the distance in microns from the main feature edge to the closest SRAFs. The distance must be greater than 0.

As of 2018.4, the units are in microns. Previously the units were equal to 0.125*lambda/NA, where NA is numerical aperture and lambda is the wavelength. The default is based on the optical model. For DUV optical models, the default value is 0.1875*lambda/NA. For EUV optical models, the default value is lambda/((1+sigma_max)*NA), where sigma_max is the maximum sigma value of the optical tool.

Description

This optional command specifies how close SRAFs may be to the main feature edge. The maximum distance is set by scatter_belt. The offset value affects only positive SRAFs. To set an offset for negative SRAFs, set scatter_offset_negative.

By default, the offset is calculated between the main feature and SRAF. In some cases, the difference between mask and target is large enough to allow SRAFs to be placed in undesired locations. To calculate the distance instead using the target layer and the output mask SRAF, set “scatter_offset_type target”.

Examples

Example 1

The following example sets the scatter offset to 1.3 microns:

scatter_offset 1.3
