Session Browser Window

To access: RET Selection Window View > Session Browser

The Session Browser window offers quick access to setup and results. It is only drawn when a session is selected.

Figure 1. Session Browser Pane

Description

In the default layout, the Session Browser shares space with the Gauge Analysis Pane. The tabs to switch between them are at the bottom. The Setup Panes area changes based on the stage selected in Setup Items.

Objects

The following table describes the action buttons at the top of the window.

Table 1. Session Browser Tool and Status Bar

Object

Description

Lock/Unlock Session

Indicates whether a session’s settings can be modified. By default, sessions that have completed are protected from edits (closed position). Click the lock icon to toggle the state. (If the session is currently running, it will not unlock.)

Session Mode

Indicates the mode of the session. This release supports Free-Form pxSMO and Parametric Explorer. See “Session Mode” for detailed descriptions of the session mode options.

Activate

Makes the selected session the active session. (If the icon is dimmed, the session is already active.) Changes made in the setup panes apply to the selected session, but commands typed in the command window or changes made in other windows apply to the active session.

Clone

Duplicates the current session. After a brief pause while the database updates, the cloned session appears in the Run Control/Layout Session View area.

Spec

Opens the Gauge Property Spec Editor. See “Scoring”.

Tags

Opens the Session Tags editor for creating, coloring, and managing session tags.

State

Displays the current state of the session. See “RET Selection Window” Table 3 for a complete list and meanings.

NAME fields

The layout and session ID and any name you may have given them. Names are recommended as memory aids. The name fields are editable.

STATUS area

Displays the progress for the session.
