Opening the RET Selection Interface

The RET Selection interface is available in the standard Calibre WORKbench menus.

The RET Selection interface as well as Calibre pxSMO can also be run without a GUI (batch mode). It requires saved configurations, so is not covered in the introductory material.

Video:

Training video is available on Support Center in KB article MG599114.

Prerequisites

  • The CALIBRE_HOME environment variable must be set. See “Setting the CALIBRE_HOME Environment Variable” in the Calibre Administrator’s Guide for details.

  • Your PATH environment variable includes $CALIBRE_HOME/bin.

Procedure

  1. In a Linux® shell, enter the following command to invoke Calibre WORKbench:
    	calibrewb
  2. In the Calibre WORKbench main window, choose Litho > RET Selection. This opens the RET Selection window.
