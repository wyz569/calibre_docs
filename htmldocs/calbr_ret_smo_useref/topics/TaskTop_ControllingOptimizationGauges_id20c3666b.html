Controlling Optimization With Gauges

Although you are required to use a single gauge file for all sessions within a layout/gauge pair, you do not have to run the same gauges for all sessions. You can selectively enable and weight gauges for different SMO runs.

Restrictions and Limitations

  • Disabled gauges are still used for the OPC and Verify runs. Enabling and disabling gauges only affects the pxSMO portion of a session.

  • Do not change gauges on a running or queued session.

  • Changes in the Gauge Analysis pane affect the active session, which might not be the currently selected session.

Prerequisites

  • The RET Selection window has a layout/gauge pair loaded.

  • The session has not yet been run.

Procedure

  1. Display the Gauge Analysis pane.

    If you do not see the Gauge Analysis pane, check the bottom of the Session Browser for tabs. In the default layout, the Gauge Analysis and Session Browser panes share the same area.

  2. To see which clip a gauge is associated with:
    1. Select the gauge row.
    2. Click the Highlight button in the Gauges toolbar.
    3. In the Calibre WORKbench window, look for the clip with a marker in the center.
    Tip

    To quickly view the clip, click on the thumbnail image and toggle the zoom level.

  3. Before making any changes to gauge settings, verify the session you intend the changes for is active.

    The active session is shown at the top of the RET Selection window on the right, and if it has a name, the Run Control/Layout Session View window shows it in a heavier font.

    If the session you want to modify is not active, use the Active Session dropdown list to make it active.

  4. To use only a subset of gauges for the SMO run, gauges can be selected automatically using Gauge Selection or manually.

    • To select gauges using Gauge Selection:

      1. Open the Gauge Selection tool by clicking the Select button in the Gauge Analysis pane.

      2. Specify the number of representative gauges using a range and step size by selecting Even Spacing or using specific numbers by selecting Custom Clusters.



      3. To run Gauge Selection on all available gauges, leave the Use All Gauges box checked. To run Gauge Selection on specific gauges, uncheck the Use All Gauges box and disable undesired gauges in the SMO Enable column.

      4. Typically, the default settings for Filter 0th Spectrum and Gaussian Sigma are sufficient.

      5. Click the Analyze button and results appear in the Analysis View pane.

      6. To apply the Gauge Selection results, choose the number of gauges in the Analysis View pane and click the Update ‘SMO Enable’ button. The SMO Enable column updates with the representative gauges selected.

        • Gauge Selection classifies each cluster of gauges as CD, Space, or Mixed (that is, a mixture of lines and spaces). It is recommended to avoid numbers of gauges with mixed clusters of gauges.

    • To manually select gauges:

      • Select the rows to turn on or off using Shift-click or Ctrl-click, then click Enable or Disable.

      • Use the On/Off button in the SMO Enable column.

      • Right-click and select Disable SMO or Enable SMO.

        By default, a new session uses gauges selected in the Gauge Analysis Pane for SMO. If you customize clips using the Clips and Gauges Editor, SMO ignores SMO Enable settings and applies the customized clips settings.

    Tip

    In many cases, an optimized source can be created using fewer than 20 gauges. Using additional gauges may produce a similar source while increasing run time.

  5. To change the weight of the gauges, right-click and select Set Weight. The clip weights are applied during pxSMO stages but not during OPC stages.
    Tip

    Gauges may also be weighted using weight layers in the layout and setup file, or weighted for individual jobs in the GUI. The layer weights in OPC stages are applied the same as in pxSMO.

    Starting from the 2019.1 release, weight layers and gauge weights are unavailable when job type CDOF or EL is specified.

    If you increase the weighting of a gauge, that weight is set across the entire clip. The objective for that clip is calculated and then multiplied by the gauge’s weight. Setting this weight therefore emphasizes the value of the entire clip. Weight layers emphasize only the portion of the clip that they cover. When both the gauge weight and the weight layer are set, the gauge weight is applied using the weight layer coverage.

  6. To change the CD tolerance of the gauges, right-click and select Set CD Tolerance.

    The CD tolerance is used in calculating the common process window. By default, the tolerance is 10%, but each gauge can be set individually. CD tolerances can be expressed as a percentage, an absolute value in nanometers, or a range.

  7. To further customize clips and gauges, click the Clips button to open the Clips and Gauges Editor.

    Customize clips using pattern symmetry and periodicity, or create custom gauges.
