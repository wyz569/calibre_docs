pw_variables

PxSMO Setup File Commands

Specifies the variables for process window optimization.

Usage

pw_variables pw_group_name var_list

Arguments

pw_group_name

A required name specifying a process window condition group. The name must also be used as the pw_group_name in pw_condition_group and pw_select statements.

var_list

A required argument set of one or more variables to use for process window optimization. The following var_list values are supported:

  • focus_range — Maximizes common depth of focus. For a CDOF job, this is the default.

  • dose_latitude — Maximizes common exposure latitude. Used in an EL job.

  • focus_center — Enables optimization of the focus center from its initial value.

  • dose_center — Enables optimization of the dose center from its initial value.

Description

An optional command to specify variables to use for process window optimization. For a CDOF job, the default behavior is maximizing common depth of focus. The command is applied per job.

Examples

pw_variables PWG1 focus_range
