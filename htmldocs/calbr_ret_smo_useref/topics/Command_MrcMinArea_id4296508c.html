mrc_min_area

PxSMO Setup File Commands

Constrains the minimum SRAF area during MRC.

Usage

mrc_min_area min_sraf_area_val

Arguments

min_sraf_area_val

A required argument specifying a minimum area for SRAFs in square microns during MRC. The default value is 0.

Description

An optional command that specifies the minimum SRAF size during MRC. For Manhhattan SRAFs, use the command in both improve and polish jobs. For curvilinear SRAFs, use the command in both the improve and polish jobs, and specify “mrc_enabled yes” in each job to activate MRC constraints for curvilinear output. For SRAFs that violate the minimum area check, the MRC increases their size to meet the constraint.

Examples

Example 1— Manhattan MRC

The following example ensures Manhattan SRAFs larger than 0.0015 square microns:

job 3 improve
	mrc_min_area 0.0015
	...
job 4 polish
	mrc_min_area 0.0015
	...

Example 2— Curvilinear MRC

The following example ensures curvilinear SRAFs larger than 0.0015 square microns:

job 3 improve
	mrc_enabled yes
	mrc_min_area 0.0015
	...
job 4 polish
	mrc_enabled yes
	mrc_min_area 0.0015
	...

The mrc_enabled statement is required to activate curvilinear MRC.
