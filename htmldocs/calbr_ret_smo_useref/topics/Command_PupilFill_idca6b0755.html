pupil_fill

PxSMO Setup File Commands

Specifies the minimum percentage of pixels to illuminate in the source.

Usage

pupil_fill pf

Arguments

pf

A required argument specifying the minimum percentage of pixels to illuminate in the source. The final fill may be different. The default value is 0.1 (10% of pixels).

Consult your scanner provider for an appropriate value. The minimum value is usually specified by the scanner provider for control of lens heating. The values from the RET Selection GUI can be used.
