mrc_max_internal_sraf

PxSMO Setup File Commands

Constrains maximum internal spacing during MRC for SRAFs only.

Usage

mrc_max_internal_sraf max_internal_val

Arguments

max_internal_val

A required argument specifying a maximum distance in microns between facing edges of a single SRAF polygon during MRC. The default behavior is to ignore maximum width.

Description

This optional command allows setting the maximum internal MRC constraint for Manhattan SRAFs explicitly. An SRAF passes the maximum internal width constraint if no square with max_internal_val sides fits inside the feature. MRC attempts to shrink features to meet the maximum width constraint.





For Manhattan SRAF shapes, specify the command in the improve and polish jobs.

Examples

Example 1 - Manhattan SRAFs

This combination constrains Manhattan SRAF shapes to allow only squares with sides smaller than 25 nm to fit inside the SRAF:

job 3 improve 
	mrc_max_internal_sraf 0.025
	...
job 4 polish 
	mrc_max_internal_sraf 0.025
	...
