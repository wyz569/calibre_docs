output_source_file

PxSMO Setup File Commands

Specifies the name of the file for the optimized source map. If this is not specified, the source is written to output_source_1.src or output_source_2.src for the first and second exposures, respectively.

Usage

output_source_file filename

Arguments

filename

A required argument that specifies the name of the file for the optimized source map. The convention is to use a .src suffix, but this is not applied automatically.

The source map is in text format. Each line consists of x and y coordinates and the filtered and smoothed intensity of a pixel.
