Setup File Format for RET Selection

Input for: Layout and Gauge Pair Loader (Setup File selection)

Generated by: Layout and Gauge Pair Loader (Lithomodel selection)

The RET Selection setup file can be used instead of a litho model when creating a new layout/gauge pair in the RET Selection GUI. If you use the other choice in the wizard, Lithomodel, at the end of the setup the wizard exports a RET Selection setup file.

When creating a new layout/gauge pair, the Lithomodel option is recommended.

Format

For your convenience, a template is provided in Figure 1. Replace the italic words with appropriate values for your model and layout. The models directory should be the parent of the litho_model_directory_name directory.

The setup file must have at least two layers defined outside the image_options block. The smoothed_target_layer can be created by SMO and does not need to exist in the clip layout file. The order of the layers does not matter, and the names do not need to match the layout because they are mapped when creating the layout/gauge pair.

Figure 1. Setup File for Layout and Gauge Pair Loader
	modelpath models 
	layer        smoothed_target_layer 
	layer        drawn_target 
	tilemicrons  100
	image_options MDF {
		layer drawn_target visible mask_layer 0
		litho_model litho_model_directory_name 
	}
	setlayer im0 = image MDF

Parameters

modelpath models

A required parameter indicating the full path to the litho model directory. The path can be absolute or relative.

layer layer_name

A required parameter that specifies a target layer, which will be read from the layout, and a smoothed layer created by setlayer curve or setlayer curve_target. The RET Selection session can be set up to create the curve for the smoothed layer.

This parameter must occur at least two times. The layer_names must be unique.

tilemicrons 100

A required parameter for parsing. The value is not used, as each point tool has its own setup file.

image_options MDF ‘{‘

    layer layer_name visible mask_layer 0

    litho_model litho_model_directory_name

‘}’

A required image options block that identifies the litho model and the target layer. Although this is based on the Calibre OPCverify image_options, the RET Selection GUI accepts only the two specifications shown. The braces ({ }) are required and must be in the positions shown.

setlayer im0 = image MDF

A required parameter that causes the image_options block to be read. The MDF value must match the MDF value of the image_options block.
