Migrating Session Settings to a New Layout/Gauge Pair

Conditions and settings from an existing session can be used in another session with a different layout/gauge pair.

In some cases, it may be desirable to re-run an optimization with additional gauges. This requires creating a new or modified layout/gauge pair and a new session. Rather than manually recreate all session information, the import tool saves time by migrating process window conditions, stage settings (for Pre-SMO, SMO, OPC, and Verify), and session command settings (accessed using Edit in the Run Control/Layout Session View).

The Import Session Settings tool can also import gauge settings (specifically, SMO Enable and weight settings) from another session if the same gauges are used in the source and target sessions.

Prerequisites

  • One existing layout/gauge pair and one new or modified layout/gauge pair.

Procedure

  1. Create a new session using the new or modified layout/gauge pair.
  2. In the Run Control / Layout Session View, right-click the new session and select Import Settings to Session-x.
  3. Select the existing session from which to copy conditions and settings. Select the conditions and settings to copy from the source session. Click Import.

Results

The specified conditions and settings are copied from the existing session to the new session.
