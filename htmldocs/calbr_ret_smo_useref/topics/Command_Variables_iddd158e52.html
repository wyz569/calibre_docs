variables

PxSMO Setup File Commands

Specifies a subset of variables to optimize.

Usage

variables set [set [set]]

Arguments

set

A required argument specifying which subset of variables to optimize. The following values are accepted:

  • dose

  • mask

  • source

Any combination can be specified. The default is to optimize source and mask.

Examples

To optimize only the dose, keeping the mask and source variables frozen:

variables dose
