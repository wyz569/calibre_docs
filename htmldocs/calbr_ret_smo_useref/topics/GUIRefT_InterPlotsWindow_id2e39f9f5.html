Inter-Plots Window

To access:

  • In the Session Browser Result tab, click Inter-Plots and select Sessions In this Layout.

  • In the Session Browser Result tab, click Inter-Plots and select Selected Sessions....

  • In the Run Control/Layout Session View, right-click in a session row and select Session Compare Plots .... In Session Compare Plots dialog box, choose the sessions to compare and the Spec. Layout. Click Open Plots.

  • In the Run Control/Layout Session View, right-click in a session row and select Layout ID > Inter-Plot.

The Inter-Plots window shows selected plots for groups of sessions within a layout/gauge pair.

Figure 1. Plots Tab

Note:

For a list of all plot types available in the window, see “Plot Types”. The Inter-Plots Window focuses on the controls common to all the plots.

Objects

Table 1. Plot Viewer Controls

Field

Description

1

Plot menu bar controls. Controls in this bar apply to all active plot views.

  • Zoom In, Zoom Out — Zooms all plots.

  • Prev, Next — Cycles through the list of plots.

  • Show All, Hide All — Displays or hides all plot views.

  • Unhighlight All — Removes all selection highlights.

  • Show Points Label, Hide Points Label — Displays or hides all plot views.

  • Plot Unit — Set plot units to nanometers (nm) or microns (um).

  • Report — Generates an HTML report. You will be prompted for a directory for the report; create a new directory for each one. (Directory contents are overwritten.)

  • Filter Traces — Similar to the legend filter, this text field filters all plot views for legends that contain a string. Remove the filter by deleting any text in the field and clicking the check button.

  • Update Plot — Click after changing session data in bottom of window.

2

Plot selector. Only plots that are checked in this list are shown in the Plot viewing area.

3

Plot viewing area. Plot views appear in the main area of the window unless deselected in field 2. Each plot has its own set of filtering and viewing controls that appear when you hover over the central area. (See next table.)

4

Session summary table. Displays the session results and aggregate “goodness” score (Layout-Spec Delta Components columns at far right of table). Above the session rows are these buttons:

  • Edit Layout Spec — Opens the Gauge Property Spec Editor, in which you can define per-gauge per-parameter target values. These are used to calculate each session’s goodness score. See “Scoring”.

  • Config Score — Opens the Score Configuration Editor, in which you can weight the six different properties that contribute to the goodness score and set whether the delta function is measured as a linear, polynomial, or exponential function.

  • Config Plot — Opens the Plot Configuration Editor, in which you can create custom plots and define the look of automatic plots.

  • Save Table to CSV — Writes all contents of the session summary table to a CSV file.

Usage Notes

You can toggle which sessions are plotted as a set, rather than using the individual On/Off control.

  1. In the session summary table, use Shift-click and Ctrl-click to select the session rows.

  2. Right-click and select Set ‘Display’ for selected rows.

  3. Toggle the On/Off control to the desired setting and click Set.

  4. Click Update Plot in the upper right of the Inter-Plot window.
