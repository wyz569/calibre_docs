sample_source

PxSMO Setup File Commands

Sets a source to use as a reference during optimization.

Usage

sample_source filename

Arguments

filename

A required argument that specifies the name of the reference source file.

Description

An optional global command that sets a source that is used as a reference during optimization. The similarity of the reference source to the current source during optimization is set with the source_difference_weight command.

Examples

The following example sets cquad.src as the reference source:

sample_source /user/opt_engr1/sources/cquad.src
