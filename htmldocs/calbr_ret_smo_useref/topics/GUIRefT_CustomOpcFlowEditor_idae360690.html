Custom OPC Flow Editor

To access: In the Session Browser, activate a session and click the OPC item in the Setup Items tree. Select Custom OPC as the OPC engine. Click Edit Custom OPC Flow.

The Custom OPC Flow Editor enables an advanced user to build custom OPC flows with different OPC, mask bias, and SRAF setups.

Description

The Custom OPC Flow Editor creates setup files for each setup used in a custom OPC flow.

Figure 1. Custom OPC Flow Editor

Objects

The following table describes the action buttons at the top of the window.

Object

Description

Add Setup

Adds a setup to the flow. Opens the Custom OPC Setup Configuration window to set the setup options.

Delete

Deletes the selected setup from the flow.

Load Setup Text

Loads a setup file and places the file contents in the Setup File Editor.
