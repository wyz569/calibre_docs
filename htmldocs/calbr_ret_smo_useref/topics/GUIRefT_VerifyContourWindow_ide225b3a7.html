Verify Contour Window

To access: In the Session Browser Result tab, click View Contour and select Open Verify Layout Viewer.

The Verify Contour window enables inspection of the post-run layout.

Figure 1. Verify Contour Window

Description

The Verify Contour window displays the post-run layout. The user can use the tool to inspect individual gauges at initial setup, intermediate stages, and final results.

Description

Select the gauge for inspection in the viewer using the gauge id or structure description.
