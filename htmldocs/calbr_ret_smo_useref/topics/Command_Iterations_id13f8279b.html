iterations

PxSMO Setup File Commands

Specifies how many iterations the job should run.

Usage

iterations number_of_iterations

Arguments

number_of_iterations

A required positive integer that specifies how many passes are performed. The default value depends on the job type:

Start

50

Decorate

15

Improve

25

Polish

500, or

With Polish-2 job, 300 each (when Image Model is set to CM1).

When a CDOF job is also specified, Polish defaults to 100.

Description

This command specifies the iteration count of optimization algorithms. It can be set either in the general section, in which case all jobs run through the same number of optimization passes, or on a per-job basis.

Examples

iterations 500
