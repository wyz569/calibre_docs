Comparing Process Window Performance

When refining an optical source, sometimes you will set up multiple layout/gauge pairs, each with many sessions in order to analyze sources across a wide variety of typical layer geometries. You can compare the source performance by looking at exposure latitude (EL) versus depth of focus (DOF) for multiple sessions within the RET Selection GUI.

Figure 1. EL vs. DOF Plot

Whereas process window plots display a single DOF that corresponds to a single EL value, this chart shows DOF for a range of EL values. The curve is useful for evaluating the maximum exposure latitude for different sources and the trade-offs between EL and DOF.

Prerequisites

  • All sessions to compare must have successfully completed execution.

  • All layout/gauge pairs must be in the same MDF database.

Procedure

  1. In the RET Selection main window, select Tools > EL vs. DOF Plots to open the selection dialog box.
  2. In the EL vs. DOF plot wizard window, select the sessions to compare. Only sessions that completed and had more than one process window condition are shown.
  3. Adjust other settings if necessary. The defaults are as follows:
    Table 1. EL vs DOF Wizard Defaults

    Type

    Elliptical

    Target CD

    Drawn

    CD Tolerance

    10%

    Dose Latitude

    5%
    Note:

    If you specify a dose latitude, the EL vs DOF plot shows a single point because the dose is fixed.

  4. Click Plot.

Results

The EL vs. DOF Plot window displays the graphs for the selected sessions.

  • As you move your mouse over the graph, the DOF and EL values appear in the lower right.

  • You can zoom in on an area by holding down the right mouse button and dragging right and down. Zoom back out by dragging left and up. Return to the original size by clicking the mouse in the plot area.
