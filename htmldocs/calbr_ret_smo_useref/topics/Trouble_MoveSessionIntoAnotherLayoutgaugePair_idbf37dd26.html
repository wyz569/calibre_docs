Move a Session Into Another Layout/Gauge Pair

Sessions in different layout/gauge pairs cannot be compared in the plots even if the layout and gauges are identical. You can use a short command-line script to move the session to another layout/gauge pair.

Solution

This procedure will change the layout id of a session.

Restrictions and Limitations

  • No sessions must be active.

  • You must know the layout ID and session ID of the session to move. You can get this information from the Session Browser.

Procedure

  1. If you have the RET Selection window open, close it. This will deactivate any sessions.

  2. In Calibre WORKbench, select LITHO > RET Selection. The RET Selection window opens with your last database open, but does not activate any sessions.

  3. In the Command Window pane of the RET Selection window, enter

    mdf smo start

    mdf smo session id layout id

    where id is the session ID and layout ID of the session you want to move.

  4. (Optional) Move the session directory to be part of the layout/gauge pair.

    cp -r session_dir new_dir

    where session_dir is the existing directory the session is stored in, and new_dir is where you would like it to be.

  5. (Optional) After moving all sessions out of the duplicate layout/gauge pair directory, delete it.

    rm -r old_dir

  6. If you moved the session, reset the session path.

    cd new_dir

    mdf smo session id workdir pwd

    Instead of “pwd”, you can enter the directory path.
