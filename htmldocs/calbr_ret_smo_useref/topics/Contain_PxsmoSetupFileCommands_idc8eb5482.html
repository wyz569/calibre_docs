PxSMO Setup File Commands

The Calibre pxSMO setup file commands can be added to an SMO setup file.

The list in this manual is not exhaustive. You may see some commands in generated files that are not included here. This list represents commands that are safe to add to custom setup files as they do not interfere with the flow of data between runs.

Table 1. Calibre pxSMO Setup File Commands

Command

Description
anchor_gidIdentifies an anchor gauge.
anchoring_enabledEnables optimization with anchoring to a gauge.
assist_inside_dose_multiplierScales aerial image from negative assist features.
assist_inside_threshold_multiplierScales print image from negative assist features.
assist_outside_dose_multiplierScales aerial image from positive assist features.
assist_outside_threshold_multiplierScales print image from positive assist features.
debug_levelWrites optimization rates to stdout.
delete_pixel_socs_dir_after_readDeletes the pixel SOCS directory after reading it.
enable_periodicityReads clip periodicity information from the clips.dat file when custom clips are used.
fixed_sourceSets parts of a source to fixed illumination levels to allow optimization on the remaining parts of the source.
fixed_source_optimization_regionOptimizes source pixel intensities in a specified part of a source while other pixels remain fixed.
initIdentifies the initial mask.
initial_sourceSets the initial source type, overriding any optical sources passed in the litho model.
iterationsSpecifies how many iterations the job should run.
jobCalls the various phases and allows local overrides of settings.
layerDefines layer characteristics such as name and type.
mrc_enabledActivates curvilinear MRC constraints.
mrc_max_internal_srafConstrains maximum internal spacing during MRC for SRAFs only.
mrc_min_areaConstrains the minimum SRAF area during MRC.
mrc_min_edgeSets a preferred edge length for Manhattan MRC geometries.
mrc_min_externalConstrains external spacing during MRC.
mrc_min_external_main2srafConstrains external spacing during MRC between a main feature and an SRAF.
mrc_min_external_srafConstrains external spacing during MRC for SRAFs only.
mrc_min_internalSets the minimum distance between two internal edges.
mrc_min_internal_srafConstrains internal spacing during MRC for SRAFs only.
mrc_min_lengthSets the minimum distance between two internal edges for Manhattan shapes.
mrc_min_rect_lengthSets the minimum distance between two internal edges for Manhattan shapes.
mrc_min_rect_widthSets the minimum distance between two internal edges for Manhattan shapes.
mrc_min_small_feature_area_excessSets the minimum area for features on the mask for Manhattan shapes.
mrc_min_squareSets the minimum distance between two internal edges for Manhattan shapes.
mrc_small_feature_sizeSets the minimum distance between two internal edges for Manhattan shapes.
objective_regionOptimizes a region using specified objectives.
optimize_doseOptimizes the dose threshold when anchoring to a gauge.
output_source_fileSpecifies the name of the file for the optimized source map. If this is not specified, the source is written to output_source_1.src or output_source_2.src for the first and second exposures, respectively.
output_unblurred_source_fileOutputs an illuminator-level source map file at the end of a session run.
parallelize_by_pw_conditionsRecommended. Enables distributed processing.
parameterizationDefines the mask shapes that represent features.
pixel_socs_dirSpecifies a new directory in which to cache optical models, potentially improving runtime.
pupil_fillSpecifies the minimum percentage of pixels to illuminate in the source.
pw_conditionCreates print image conditions for dose, focus, resist, and aerial threshold.
pw_condition_groupCreates a group of dose and focus conditions located along a defined process window shape before optimization.
pw_focus_base_pointsSpecifies defocus conditions for through-focus aerial image calculation. For job types other than CDOF, this command is required when focus_range or focus_center is specified by the pw_variables command.
pw_focus_range_factorControls the relative emphasis given to common depth of focus during process window optimization.
pw_selectRestricts simulation to only the specified processes.
pw_tolerance_boundRequires the process window conditions for a process window condition group to produce contours within a specified area. Used when optimizing common depth of focus.
pw_variablesSpecifies the variables for process window optimization.
sample_sourceSets a source to use as a reference during optimization.
save_job_cdof_print_imagesSpecifies the input layers and process window contours to save for CDOF jobs. Only valid for CDOF jobs.
save_job_maskSpecifies the name of an OASIS file of all clips to save at the end of a job.
save_job_print_imageSpecifies the name of a file of contours of all clips to save at the end of a job.
save_job_sourceOutputs a source map file at the end of a job using the specified name.
save_job_raw_sourceOutputs an illuminator-level source map file at the end of a job using a specified name.
scannerSets source parameters to values typical for different manufacturers.
scatter_assist_typeSpecifies where to place SRAFs relative to the target layer.
scatter_beltIdentifies how far from the target the SRAF rings should extend.
scatter_offsetSpecifies the minimum distance between the main feature and positive SRAFs.
scatter_offset_negativeSpecifies the minimum distance between the main feature edge and negative SRAFs.
scatter_offset_typeSets whether scatter_offset and scatter_offset_negative are calculated based on mask or target shapes.
set_on_targetImproves EPE for the nominal process window condition.
sigma_maxControls the source manufacturability constraint on maximum allowable sigma for the optimized source.
sigma_minControls the source manufacturability constraint on minimum allowable sigma for the optimized source.
source_difference_weightSets the importance of the source similarity when the sample_source command is used.
source_symmetrySpecifies the symmetry of the optimized source.
suppress_extra_printingSpecifies SRAF printing suppression for process window conditions or process window condition groups.
variablesSpecifies a subset of variables to optimize.
versionSpecifies the version to use.
weight_backgroundSets the default weight.
weight_layerIdentifies polygon layers which define more important areas for optimization.
