RET Selection Window

To access: Calibre WORKbench > Litho > RET Selection

The RET Selection window provides an easy way to set up Calibre pxSMO runs, analyze results, and create scripts to execute future runs from the command line.

Description

The RET Selection window consists of a common toolbar and five child windows.

  • To undock a child window, select the title bar and drag, or double-click.

  • To dock it where it was, click the icon to the left of the close control, or double-click.

  • To restore the windows to their default positions, select View > Default.

Objects

Table 1. Default RET Selection Window Contents

Objects

Description

Active Layout & Gauge Pair selector

This is a three-part control:

  • Use the yellow button to begin a new layout/gauge pair.

  • Use the dropdown to change which layout/gauge pair is active.

  • Use the text entry box to enter or view a name.

Active Session selector

This control is unavailable until you load a layout/gauge pair.

  • Use the yellow suitcase to begin a new session.

  • Use the dropdown to change which session is active.

  • Use the text entry box to give the session a new name.

Run Control/Layout Session View window

A window with two tabs:

  • Run Control — See “Run Control Pane”.

  • Layout Session View — A hierarchical listing of layouts and sessions that displays information on the session state (see Table 3) and when it was last active.

Session Browser window

Gauges window

These two windows share space in the RET Selection window, but can be undocked separately.

  • Session Browser — Controls for setting up the point tools and accessing plots.

  • Gauges — Also titled Gauge Analysis. Shows all the gauges for the active layout/gauge pair. When a session is active, additional controls and columns are displayed.

Other windows are hidden by default, as they are only needed for debugging or script creation.

Window

Description

Text fields for Gauge, Layout, Setup, and Litho

Select View > Active Files.

These fields are not editable. They show the files chosen when you created the layout/gauge pair that is currently active.

Command Window

Select the Command button at the bottom of the RET Selection Window.

The Command Window shows the equivalent mdf command for GUI interactions. It can be useful when developing a batch script, as described in “Converting GUI Sessions to Batch Mode Files”.

Log Viewer

Select the Log button at the bottom of the RET Selection Window.

The log viewer shows the transcript, and can be searched.

Some gauge measurements

In the Gauges window, right-click on a column heading to open the Column Manager.

Usage Notes

The session states appear in the Layout Session View tab. They are for information only and are updated by the run control.

Table 3. Session States

State

Description

Init (0)

Session has been created.

MDFLITHOMODEL (1)

Model has been created.

PRESMO (2)

The smoothed curve stage is set up.

Setup Done (3)

The setup command is done.

Executing (tool)

The session is currently executing. If the session run was started by the dispatcher, this is shown instead of states 4 – 8.

Clips Done (4)

The clip generation command is done.

SOCS Done (5)

The optical model cache is done.

SO Done (6)

Calibre pxSMO source optimization is done.

MO Done (7)

Calibre OPC mask optimization is done.

Verify Done (8)

Calibre OPCverify verification is done.

Complete (9)

All stages are complete.
