Verify Setup Pane

To access: In the Session Browser, activate a session and click Verify in the Setup Items tree

Enables quick setup of the Calibre OPCverify run before execution and configuration of the Process Window Analysis plot.

Figure 1. Verify Setup Pane

Description

The Verify Setup pane creates a file called verify.setup by default. The pane allows you to select different measurements to make, such as PV band and CD. It also lists the process window conditions in the PW Table. To change the conditions, use the Verify column in the PW Table Pane.

Note:

The PW Table conditions are shared across all sessions in a layout/gauge pair. Do not change the table if sessions are executing.

Some options appear duplicated between the Contour Outputs and Gauge-Based Checks, but are actually using different commands. You can see the commands being used by selecting the checks and clicking Preview Verify Setup. All commands in the Verify setup file are documented in the Calibre OPCverify User’s and Reference Manual.

The gauge-based Image Shift, MEEF, ILS/NILS, and Depth of Focus checks have additional settings that are accessed either through the Setup Items tree or the right-arrow icons after the check name. The Depth of Focus pane configures the Common Process Window plot.

PW Settings Tab

Settings affect the Common Process Window (cPW) calculations.

Model Form

The type of polynomial to use when modeling the common process window.

  • Poly13 — Default. Provides better fit for the DOF measurement.

  • Poly8 — A smaller polynomial provided for backwards compatibility. Poly8 is not recommended.

Type

The shape to fit inside the common area when measuring depth of focus (DOF). Select Elliptical (default) or Rectangular.

Target CD

Whether the DOF calculations should use the nominal print image CD or the drawn CD from the super gauge data (if available). Drawn is the default and recommended.

If you do not have a super gauge data file or if it does not include drawn CD measurements, the DOF calculation uses the nominal print image CD from simulation instead. Only set Nominal if you want to ignore the drawn CD.

If the super gauge data file contains target CD data in the “other” column, select Other. Choose this option when performing process window analyses where the mask CD is in the “drawn” column and the nominal CD is in the “simulated_CD” column.

Center

Forces the inscribed ellipse or rectangle in the common process window to be centered.

  • Dose — Default. This forces the dose variation to be symmetric around the dose value set in the Nominal field. It does not constrain the focus.

  • Focus — Similar to Dose, but forces the inscribed shape to be centered on the nominal focus.

  • None — Use for maximum common DOF. May be overly optimistic.

  • Both — Use to explore the common process window around nominal conditions.

CD tolerance

The allowable deviation between target and simulated CD on gauges. Gauges that cannot be resolved to within the tolerance are not included in the common process window plot. The CD tolerance upper and lower bounds are calculated using the Target CD selection (either Drawn or Nominal). The default is 10%.

Dose latitude

The extent of the common process window around the dose axis. The default is 5%.

RMSE Error Tol

Defines the threshold for outliers. The RET Selection common process window is calculated with a two-step fitting. Gauges whose CD at a given dose and focus vary greater than the RMSE value are not included in the second part of the calculation. The default is 5.

Nominal

Specify a focus or dose to use in conjunction with Center. Dose is in percentage. (100% is 1.) Focus is in microns.

Changing the nominal focus and dose affects the inscribed shape.

Max DOF/Max Latitude

Specify a tolerance for depth of focus and exposure latitude. Both default to 0, allowing the full range of latitude and focus to be explored.

Use fixed DOF

When set, DOF is held steady at the specified value.

Disabled GIDs

Gauges to be omitted when calculating common process window.

Set the values by clicking Open Linked PW Plot and setting the gauges to Off in the cPW Analyze column.

DOF Tab

Settings affect the gauge-based DOF calculations.

DOF PW Conditions

Select the Simple, Custom, or Multiplier mode to adjust focus and dose latitude for the depth of focus process window.

  • Simple mode permits setting only the maximum and minimum exploration distance.

  • Custom shows the exact values to be used. Click SET to edit them.

  • Multiplier displays the minimum and maximum values, which you adjust indirectly via a multiplier.

Note:

The Dose and Focus values set in the tab are not related to the Dose and Focus values set in the PW Settings tab. The initial values are drawn from the PW Table. Fewer than 16 defocus conditions should be specified.

Checks

Use the Configuration controls to the left to add checks to the Checks list. Initially the list is populated with only ldof.ellipse, the default Depth of Focus check.

To remove checks. right-click the check name and select Delete.

DOF

LDOF

Specifies the dose latitude of the otherwise-identical check.

  • DOF — Uses a minimum dose latitude; use for focus sensitivity or for comparing to older Calibre OPCverify DOF checks.

  • LDOF — Varies the dose latitude according to EL Criteria. Produces standard DOF data.

Rect

Ellipse

Specifies the shape to be inscribed in the process window for calculating DOF.

Property name

Specify a unique name for the check. Duplicate names cause the session to fail with an error.

CD Tol(%)

EL Criteria(%)

Specifies the CD tolerance and EL criteria (dose latitude). Larger values give larger DOF and focus ranges. Varying these may cause the best focus and best dose to shift.

EL Criteria is only available with LDOF.

Additional properties

Select properties to add to the gauge output.

Usage Notes

The PV Band contour is constructed from only the outer and inner contour, and if you have a complicated set of process window conditions may not represent the complete PV band. You can add more contours using the Options field in the selection’s pop-up menu, accessed from the button to the right of the label.
