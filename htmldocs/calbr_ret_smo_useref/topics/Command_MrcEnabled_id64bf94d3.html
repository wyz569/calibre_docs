mrc_enabled

PxSMO Setup File Commands

Activates curvilinear MRC constraints.

Usage

mrc_enabled [ no | yes ]

Arguments

no | yes

An optional argument specifying activation of curvilinear MRC constraints. The default is no (curvilinear MRC constraints off).

Description

An optional command specifying activation of curvilinear MRC constraints in improve or polish jobs.

Examples

Example 1

The following example activates curvilinear MRC constraints and permits curvilinear shapes as close as 12 nanometers and as narrow as 12 nanometers wide.

job 3 improve
	mrc_enabled yes
	mrc_min_external 0.012
	mrc_min_internal 0.012
	...
job 4 polish
	mrc_enabled yes
	mrc_min_external 0.012
	mrc_min_internal 0.012
