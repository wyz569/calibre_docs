mdf gauges commands

Commands that begin “mdf gauges” control operations on the gauge file.

computeils

Calculates image log slope (ILS) at polygon edges in a layer for all gauges.

computeils ‑layer layer_num [‑simtype aerial | full | both]

 

‑layer layer_num

A required argument set specifying the layer containing the polygon edges.

‑simtype aerial | full | both

An optional argument set specifying the image type for simulation.

  • aerial simulates the aerial image. This is the default image for ILS calculation.

  • full simulates the full resist image.

  • both simulates both aerial and full resist images.

After simulation, the tool adds following columns to the gauge table:

  • XL and YL — Left-side coordinates in dbu where the gauge crosses the polygon edge.

  • IlsLCtr — Left-side ILS calculation for the full resist image.

  • IlsL — Left-side ILS calculation for the aerial image.

  • XR and YR — Right-side coordinates where the gauge crosses the polygon edge.

  • IlsRCtr — Right-side ILS calculation for the full resist image.

  • IlsR — Right-side ILS calculation for the aerial image.

pw

Creates process window conditions for gauges. Subcommands “dose” and “focus” accept a list of values or a range using a start:step:end format. The following example adds three defocus conditions to the litho model file and builds optical models from the nominal:

mdf gauges pw focus "0 ‑0.050 0.050"

The next example simplifies a focus list by using a range format:

mdf gauges pw focus "‑0.15:0.03: 0.15"
