Setting Up the nmOPC Engine

As an alternative to the pxOPC engine, you can optimize the mask using the nmOPC engine.

Prerequisites

  • Complete the pxSMO setup of “Setting Up the Point Tools”.

Procedure

  1. Select nmOPC in the OPC Engine menu. In the Setup Items tree, the jobs change to Pre-Sraf, Pos-Sraf, and Neg-Sraf.
  2. The Pre-Sraf job is checked by default and should always be run regardless of SRAF tone.
  3. Pos-Sraf and Neg-Sraf cause positive and negative SRAFs respectively to be simulated. Select the checkboxes to indicate which types of SRAFs to simulate. Simulating both tones (or no SRAFs) is supported. The nmOPC engine is run regardless of the SRAF job settings.
  4. For the selected jobs, assign process window conditions in the PW Table.


  5. For Pos SRAF and Neg SRAF, use the nominal and through-focus conditions.
  6. To use PVBand OPC, click the PVBand checkboxes for the corresponding PW conditions. For more information, see the “PV Band PWOPC” section in the Calibre nmOPC User’s and Reference Manual.
  7. To change nmOPC setup file options, click Open nmOPC Setup Editor. Red text can be modified. Blue text cannot be modified, but displays a tooltip when a cursor hovers over it. SRAF print avoidance (SPA) is available in the nmOPC setup file.


  8. To use a setup file for the nmOPC stage or its jobs, turn off the Generate nmOPC Setup or Generate Jobname Setup switch and load the setup file.
  9. Complete the Verify job setup in “Setting Up the Point Tools”.
