Exporting an Optimized Source

The optimized source is saved in the session output directory in the source_sopxopc.src file by default. (You can specify a different name with the output_source command.) This file is the data shown in the RET Selection GUI source thumbnails.

The RET Selection GUI uses the Source View tool in Calibre WORKbench. Source View not only allows you to analyze the source map but to also export it in various formats.

Prerequisites

  • The RET Selection GUI is open.

  • At least one completed session.

    The Source View tool opens any source map and does not check that optimization is complete.

Procedure

  1. Click a source thumbnail.

    Source thumbnails are available from the Result tab or in the SRC column of the Run Control pane.

  2. In the Source View window, optionally enter a description. The description is saved with the source map, unlike the settings in the Map Options.
  3. In the Source View window, select one of the following:

    Format

    Menu Item

    Comma-separated values

    File > Export to CSV

    MATLAB™1

    File > Export to Matlab

    Nikon

    File > Export to Nikon

    PROLITH™2

    File > Save Source Map

    ASML

    Options > Save Zeros (set)

    File > Save Source Map
  4. In the Save Source dialog box, verify that you are not in a Calibre-created directory such as session_1. Enter a filename and click Save.

    The Save Source dialog box automatically appends a format-specific file extension if you do not supply one.

Results

The source data is written in the format you specify. The default extension for a source map saved from Source View is .smf.

Examples

This is the text of a source map saved from Source View:
