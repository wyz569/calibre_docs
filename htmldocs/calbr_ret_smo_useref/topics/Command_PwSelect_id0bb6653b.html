pw_select

PxSMO Setup File Commands

Restricts simulation to only the specified processes.

Usage

Standard Syntax

pw_select  {cond_name | pw_group_name} [cond_name...] [pw_group_name...]

Gauge-Based Syntax

pw_select  {cond_name | pw_group_name}  [cond_name ...] [pw_group_name...]
gid gid [gid...]

Arguments

cond_name | pw_group_name

A required name of at least one process window condition or process window condition group. Names are not case sensitive.

  • A process window condition name must also be used as the cond_name in a pw_condition statement.

  • A process window condition group name must as the pw_group_name in a pw_condition_group statement.

gid

A required argument specifying at least one gauge id to simulate using the specified process window condition or condition group names. If a pw_select statement specifies process window condition groups with gauge ids, any later pw_select statement with the process window condition groups must specify the same process set of groups (See the Examples for correct and incorrect usage). A gauge id may be specified in only one pw_select statement.

Description

An optional statement that restricts simulation to the listed processes. The process window conditions are defined in the pw_condition statements; process window condition groups are defined in the pw_condition_group statements. Only the specified process window conditions and groups are simulated. If one or more gauges are specified, the gauges are simulated only at the specified process window conditions.

This statement can be used to restrict conditions and groups for the entire run by placing it in the general section. It can also be used to override the conditions and groups on a per-job basis by placing it in a job section. In that case, only the job where the statement appears is affected; the next job runs with the general setting.

Examples

The following example correctly specifies process window condition groups by using the same set of groups in each statement:

pw_select PWG1 PWG2 nominal
pw_select PWG1 PWG2 gid 2 3 4

In the example above, nominal is a single process window condition and not a process window condition group.

The following example incorrectly specifies process window condition groups by using different sets of groups in each statement:

pw_select PWG1 PWG2 nominal
pw_select PWG1 gid 2 3 4

The second pw_select statement is incorrect because it specifies one of the process window condition groups from the first statement, but does not specify the other group.
