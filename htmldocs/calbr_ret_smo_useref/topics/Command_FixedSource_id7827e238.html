fixed_source

PxSMO Setup File Commands

Sets parts of a source to fixed illumination levels to allow optimization on the remaining parts of the source.

Usage

fixed_source filename

Arguments

filename

A required argument that specifies the name of the input source map. The input source map specifies parts of the source and their illumination intensities that are fixed during source optimization. Only the parts of the input source having an intensity of zero are optimized.

Description

An optional global command that specifies parts of the source that are fixed during source optimization. The bright pixels of the input source are considered fixed if fixed_source_optimization_region is not specified.
