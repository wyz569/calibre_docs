scanner

PxSMO Setup File Commands

Sets source parameters to values typical for different manufacturers.

Usage

scanner {asml | nikon | {euv 3300 | 3400}}

Arguments

asml

A required argument that sets internal parameters to values typical for an ASML DUV scanner. This is the default setting.

nikon

A required argument that sets internal parameters to values typical for a Nikon scanner.

euv 3300 | 3400

A required argument that sets internal parameters to values typical of an ASML NXE 3300 or NXE 3400 EUV scanner.

Description

The scanner keyword lets you conveniently set several parameters at once as listed in the following tables. Specify one of “asml”, “nikon”, “euv 3300”, or “euv 3400”. To specify the minimum pupil fill for the scanner, add the pupil_fill argument and corresponding value to the setup file.

The SMO setup file is processed linearly. This means that any of the parameters in the table that are set before scanner are ignored. Any of the parameters in the table that are specified after scanner take priority over the scanner setting’s value.

Examples

This example demonstrates the prioritization:

sigma_min 0.01
...
scanner asml
...
sigma_min 0.03

The sigma_min value used by the pxSMO run will be 0.03 The first sigma_min of 0.01 is overwritten by the scanner asml value of 0.0. The 0.0 value is then overwritten by the second sigma_min, 0.03.
