Plot Access

The first step of comparing optimized sources is to check that the session results fit your needs. The RET Selection window provides many ways of accessing single-session plots.

Source thumbnails appear in many parts of the RET Selection GUI. Any place they appear, you can click the thumbnail to display the Calibre WORKbench source view.

From Run Control/Layout Session View

Right-click on a session and select either Session Plots or Layout-ID > Inter-Plot.

From the Result Tab

In the default configuration, the Result tab is at the far right of the Session Browser window. To access the plots, click the Session Plots or Inter-Plots button.

Figure 1. Session Plots Button

Other Plots

Common Process Window

The Result tab also gives easy access to the session’s Common Process Window plot from Calibre WORKbench. Click the thumbnail of the process window plot in the Verify Summary or the SGD file link to open the Sgd Process Window gauge selector. In the dialog box, click Plot.

Bossung Plots

From the Common Process Window plot, right-click on a gauge in the Gauge Table area and select Bossung Plot for GID. The Bossung window offers eight CD plots.

To display the residual plots, select “Model CD Residual Plots” at the top of the window. By default, it is not selected.

EL vs DOF

The RET Selection window also helps you set up another type of process window analysis, exposure latitude versus depth of focus (EL vs. DOF).

For a single session, click the thumbnail of the process window plot in the Verify Summary in the Result tab. In the Process Window, select the EL vs DOF tab.

For multiple sessions, use the EL vs. DOF plot wizard, accessed from the RET Selection window Tools > EL vs. DOF Plots.

pxSMO Plot

When the Free-Form pxSMO session mode is used, results are displayed in the pxSMO Plot window. To review the pxSMO objective plot and transcript, click the pxSMO Plot button in the Result tab. The plot and transcript are described in “PxSMO Objective Plot”.

Parametric Plot

When the Parametric Explorer session mode is used, results are displayed in the Parametric Plot window. To view the results, click the Parametric Plot button in the Result tab. Review the tested sources in the table using the evaluation criteria or scoring that are critical to the design.

Figure 2. Parametric Plot Button

The “Demonstration: Parametric Plot for Reviewing Results” video in the “Getting Started with Parametric Explorer” video series demonstrates results review using the Parametric Plot. The plot is described in “Parametric Plot”.
