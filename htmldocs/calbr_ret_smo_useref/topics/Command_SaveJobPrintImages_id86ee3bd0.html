save_job_print_image

PxSMO Setup File Commands

Specifies the name of a file of contours of all clips to save at the end of a job.

Usage

save_job_print_image layout_filename

Arguments

layout_filename

A required argument that specifies the name of an OASIS file of contours to save at the end of a job.

Description

An optional command that specifies the name of a file of contours of all clips to save at the end of the job where the option is set. The command is added to a setup file for job-level debugging. The option applies only to the job where it is specified.

The following layers are saved:

  • All input layers to the pxSMO job using a ‘layer’ statement (target, smooth target, and so on).

  • All print images for the process window conditions of the job. Each print image contains contours of all clips. The contours are named “gidN_cond_name” where N is the clip number and cond_name is the name of the process window condition.

  • Three summary layers calculated for the specified process window conditions:

    • “Inside_contour” — The combined innermost edges of the contours.

    • “Outside_contour” — The combined outermost edges of the contours.

    • “PV” — The PV band.

The file is saved in the session directory.
