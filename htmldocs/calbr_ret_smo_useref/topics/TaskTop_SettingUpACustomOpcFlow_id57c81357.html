Setting Up a Custom OPC Flow

The Custom OPC Flow Editor lets an advanced user build new OPC flows.

Custom flows can be built by sequencing the following setup types:

  • pxopc and denseopc for OPC.

  • opcbias and nmbias for mask bias.

  • retsbar and mbsraf for scattering bars.

  • opcverify for layer manipulation using setlayer commands.

Prerequisites

  • Advanced knowledge of setup file structure and syntax. Also, advanced knowledge of setlayer statement structure and syntax.

  • Complete the pxSMO setup of “Setting Up the Point Tools”.

Procedure

  1. Select Custom OPC in the OPC Engine menu. Click Edit Custom OPC Flow.

    The Custom OPC Flow Editor appears and displays the input and output layer.

  2. To add the first setup, click Add Setup.
  3. In the Custom OPC Setup Configuration dialog box, select the setup type. Name the setup, choose the input layers, and click Add.
  4. In the flow editor, click on the new setup icon in the flow viewer. Customize the flow manually or by loading a setup file.

    • To build a custom flow manually:

      1. If needed, map layers such as the input layers to the setup input.

      2. Add process window conditions by clicking on them in the PW table and dragging them to the correct section in the setup file editor.

      3. In the setup file editor, add the job types and details.

      4. Add the setlayer statements for the setup output layers.

    • To build a custom flow from a setup file:

      1. Click Load Setup Text, select the setup file, and click Open.

      2. Map the input layers to the setup input.

      3. If needed, change the loaded process window conditions to match the conditions defined for the session.

      4. If needed, modify the job types and details in the setup file editor.

  5. Add more setups to complete the flow. Map the output of the previous setup(s) to the input of the added setup(s).
  6. Map the output from the last setup to the Output Layers. The retsel_mask layer is a required output and is passed to the Verify step for simulation.
  7. Check the setup file syntax.
    Note:

    Setup files are not syntax checked in the Custom OPC Flow Editor. Syntax checking must be performed by the user.

  8. Click Close and confirm there are no error messages.
  9. Complete the Verify job setup in “Setting Up the Point Tools”.
