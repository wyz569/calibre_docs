source_symmetry

PxSMO Setup File Commands

Specifies the symmetry of the optimized source.

Usage

source_symmetry {‘F’ | ‘|’ | ‘-’ | ‘+’ | ‘*’}

Arguments

‘F’ | ‘|’ | ‘-’ | ‘+’ | ‘*’

A required argument that indicates whether the output source is forced to be symmetric. Legal values are as follows:

  • F — No symmetry.

  • | (pipe) — Vertical symmetry (X-symmetric).

  • - (hyphen) — Horizontal symmetry (Y-symmetric).

  • + — Both horizontal and vertical symmetry (XY-symmetric). This is the default.

  • * — Horizontal, vertical, and diagonal symmetry.

The default value is XY-symmetric source, +.

Examples

The following example specifies the optimized source should be asymmetric:

source_symmetry F
