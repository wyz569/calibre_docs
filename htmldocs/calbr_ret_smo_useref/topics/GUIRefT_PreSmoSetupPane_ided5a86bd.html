Pre-SMO Setup Pane

To access: In the Session Browser, activate a session and click Pre-SMO in the Setup Items tree.

The Pre-SMO setup pane enables setup of the curve target and tolerance bands before run execution, or viewing of settings when the run is complete.

Description

The Pre-SMO Setup pane creates a file called smoo.setup by default.

Curve Target Tab

Max Clip Size

Controls the maximum size used to generate a clip layout from a gauge.

Curve Command

Adjusts the target.

  • curve_target — Creates a target that mimics the final image. This is the default.

  • curve — Creates a target using a smoothing effect.

The options for setlayer curve_target and setlayer curve are described in the Calibre nmOPC User’s and Reference Manual.

Tolerance Bands Tab

Objects common to Tolband-1 (cDOF) and Tolband-2 (EL/ILS)

Clip Based

Biases all edges in a clip using the CD Tolerance (%) value.

Opens the Custom Marker Editor for drawing markers and setting the CD tolerance.

CD Tolerance (%)

Specifies the CD tolerance applied to the target. By default, the CD tolerance value is set equal to the Process Window Analysis CD Tolerance.

To change the CD Tolerance, click the lock icon and adjust the value.

Bias Type

Specifies the type of CD measurements that the CD tolerance is applied to:

  • space — Applies CD tolerance to external CD measurements.

  • width — Applies CD tolerance to internal CD measurements.

  • space_width — Applies CD tolerance to the minimum of the internal and external CD measurements. This is the default for CDOF jobs.

  • fix — Applies a constant CD tolerance in nanometers. This is the default for EL jobs.

Use Gauge Type

Uses the gauge type to determine the type of tolerance band marker. Default is off.

Minimum Width

Maximum Width

Minimum Space

Maximum Space

Applies the CD tolerance value to lines and spaces within the minimum and maximum values.

  • If the target CD is between the minimum and maximum, then the CD Tolerance is applied to the target CD.

  • If the target CD is below the minimum, the CD Tolerance is applied to the minimum value.

  • If the target CD is above the maximum, the CD Tolerance is applied to the maximum value.

When Minimum Space and Maximum Space are set to zero, they use the Minimum Width and Maximum Width values. All values are in microns.

Copy Tolband Marker Options

Specifies which gauges have tolerance markers copied to periodic features.

If a gauge is placed on a symmetrical repeated feature (for example, a contact in a 2D periodic array), the tolerance band marker for that gauge feature can be copied to its repeated instances. The Copy Tolband Marker options controls how the tolerance band marker is copied to the repeated instances.

  • All — Copies the tolerance band markers for all gauges to the repeated instances. This is the default.

  • Custom CD Tol. — Copies the tolerance band markers only for gauges that have a custom CD tolerance.

  • Select — Copies the tolerance band markers for specified gauge IDs.

If Use Gauge Type is selected, then All is recommended.
