Invoking Synopsys Laker with Calibre RealTime

Invoking Synopsys Laker with Calibre RealTime adds the Calibre RealTime integrated toolbar to the design window.

Prerequisites

  • “Enabling the Calibre RealTime Integration in Synopsys Laker” completed.

  • System and license requirements described in “Calibre RealTime Custom Requirements” are satisfied.

  • (Recommended) “Database Read Setup for Calibre RealTime with Synopsys Laker” completed.

Procedure

  1. Start Synopsys Laker with the following command:
    $LAKER_HOME/bin/laker <options>

    A view of the design window in Synopsys Laker3 is shown in the following figure. The Calibre RealTime Integrated Toolbar (Laker) is loaded automatically in the Synopsys Laker window.

    Figure 1. Calibre RealTime Integration with Synopsys Laker

  2. Proceed to “Specifying and Using Multiple Run Configurations in Calibre RealTime (Laker)”.
