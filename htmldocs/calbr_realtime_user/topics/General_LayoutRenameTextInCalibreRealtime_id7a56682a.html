Layout Rename Text in Calibre RealTime

Calibre RealTime provides the following support for the Layout Rename Text statement.

Integration Support

Supported for: All integrations (Synopsys Laker3, Calibre DESIGNrev, Cadence Virtuoso, Pyxis Layout)

Supported Syntax in Calibre RealTime

LAYOUT RENAME TEXT 
“delimiter regular_expression
delimiter replacement_pattern delimiter 
[n | g] [e | b] [i | m] [Mc] 
[delimiter regular_expression

[n | g] [e | b] [i | m] [Mc] …]” 
[CELL LIST name]
