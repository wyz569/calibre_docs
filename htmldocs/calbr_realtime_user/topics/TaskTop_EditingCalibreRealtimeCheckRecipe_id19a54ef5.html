Editing the Calibre RealTime Check Recipe

You can edit existing check selection recipes or create new user-defined check recipes. You can create new recipes from scratch or you can copy existing user-defined or built-in recipes to create new ones. Custom recipes can be deleted when no longer needed. Built-in recipes may not be deleted.

Prerequisites

  • You have completed “Invoking Pyxis Layout with Calibre RealTime”.

  • You have a Calibre nmDRC rule file available.

  • You have invoked the Calibre RealTime Options Dialog Box in Pyxis Layout.

Procedure

  1. Click the button on the Calibre Options dialog box to open the Calibre RealTime Recipe Editor Dialog Box in Pyxis Layout.
  2. Do one of the following to open a check recipe for editing:

    • Create a new recipe:

      1. Click the New Recipe button on the Edit Recipe toolbar.

    • Copy an existing check recipe:

      1. Use the Recipe combo box to scroll through all the existing recipes and select one.

      2. Click the Copy Recipe button on the Edit Recipe toolbar.

    See Table 1 for button descriptions.

  3. Select checks to include or exclude in your recipe. The basic editing controls are shown in Figure 1. See Table 2 for descriptions of the basic controls.
  4. Check the Basic/Advanced button to display advanced controls. The advanced editing controls are shown in Figure 2. See Table 3 for descriptions of the advanced controls.
  5. Click OK in the Edit Recipe dialog.
  6. Click OK in the Calibre Options dialog.
