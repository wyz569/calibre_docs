Defining and Loading Keyboard Shortcuts (Hotkeys) in Calibre RealTime for Synopsys Laker

You can define custom keyboard shortcuts for Calibre RealTime. Several default keyboard shortcuts are defined, along with option user-defined keyboard shortcuts which can be enabled.

Prerequisites

  • Locate the pre-defined keyboard shortcuts file corresponding to the Laker version you are using. The directory path is:

    $CALIBRE_HOME/shared/pkgs/icv/tools/realtime/laker

    The filenames are:

    • Laker3 — laker_3_realtime_bindkeys.tcl

    • Laker OA — calibre.key

Procedure

  1. Copy the appropriate keyboard shortcuts file to a file in your home or project directory. For example:
    cp shortcuts_file ~/my_calibre.key
  2. Edit the shortcut definitions as desired. You can enable shortcuts listed as user-enabled in “Keyboard Shortcuts (Hotkeys) in Calibre RealTime for Synopsys Laker” by un-commenting the definition.
  3. Load the new shortcut definitions into Synopsys Laker.

    For example, for Laker3, use the following command:

    source ~/my_calibre.key

    For Laker OA, use the following command:

    gtSetBindKey -windowType LakerDsgWnd -file ~/my_calibre.key
