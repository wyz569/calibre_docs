Specifying a Check Recipe

The check recipe selects which rule checks are executed during the Calibre RealTime run.

Prerequisites

  • A supported design tool is open with Calibre RealTime enabled.

Procedure

  1. Click the Options toolbar button () to open the Calibre RealTime Options Dialog Box.
  2. Specify the rule file or a Calibre Interactive runset. This is required before selecting a check recipe.
  3. Select the check recipe in the “Check selection recipe” dropdown list.
    Tip

    If the rule file is already loaded you can select the check recipe from the toolbar without opening the Calibre RealTime Options dialog box.
