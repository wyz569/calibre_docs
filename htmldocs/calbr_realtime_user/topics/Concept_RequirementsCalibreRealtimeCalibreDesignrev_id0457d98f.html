Requirements for Calibre RealTime with Calibre DESIGNrev

Several requirements must be met in order to run Calibre RealTime with Calibre DESIGNrev.

The following list gives the detailed requirements:

  • Using Calibre DESIGNrev. The other Calibre layout viewers, such as Calibre WORKbench and Calibre MDPview, are not supported.

  • Database layers are properly set up in Calibre DESIGNrev to work correctly with the Calibre rule file. This requirement is typically already met when using a GDSII or OASIS®1 format layout database. When using other database formats, use the layer mapping options available with Calibre DESIGNrev. See the section “Database Read Options in Calibre RealTime with Calibre DESIGNrev” for more information.

  • Not using an overlay with the layout.

  • Using Calibre 2012.3 or later.

  • Met the system and licensing requirements described in “Calibre RealTime Custom Requirements”.
