Keyboard Shortcuts in Calibre RealTime for Cadence Virtuoso

A set of standard keyboard shortcuts are provided for Calibre RealTime with Cadence Virtuoso.

The keyboard shortcuts are defined in the following file:

$CALIBRE_HOME/shared/pkgs/icv/tools/queryskl/mgc_bindkeys.skl

This file is not loaded by default to avoid possible conflicts with existing keyboard shortcuts. The definition of the bindkey functions is given in table Table 1. The section “Defining and Loading Keyboard Shortcuts in Calibre RealTime for Cadence Virtuoso” describes how to define and load your own keyboard shortcuts.
