Setting the Layer Map File for Calibre RealTime with Pyxis Layout

The Calibre RealTime client in Pyxis Layout requires the use of a technology layer map to translate Pyxis Layout processes to GDS layers and datatypes.

The procedure you use to specify a layer map depends on whether a technology layer map is included with the design kit.
