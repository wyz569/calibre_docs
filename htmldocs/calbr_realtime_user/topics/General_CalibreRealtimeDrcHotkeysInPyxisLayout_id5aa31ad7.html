Calibre RealTime DRC Hotkeys in Pyxis Layout

Keyboard shortcuts, or hotkeys, are available for many functions in Calibre RealTime for Pyxis Layout.

The following table shows the hotkeys available in Calibre RealTime. F12 and Shift+F12 are always active within Calibre RealTime. The remaining keys can be activated by turning on an option in the Pyxis Layout preferences. The default option is off. To activate these keys, use Setup > Preferences > Behavior > Enable Calibre RealTime hotkeys.
