GDS Layer Map for Calibre RealTime in Synopsys Custom Compiler

A layer map file is required to map the Synopsys Custom Compiler layer objects to GDS layers and datatypes for layout export. The layer map file is specified in the Calibre RealTime Options dialog box.

File Format

Consult the Synopsys Custom Compiler documentation. This is the general format: 

OA_layer OA_purpose GDS_layer_num GDS_datatype
