Halos in Calibre RealTime Custom

Halos ensure that new errors caused by changes in the geometry are reported, and false errors caused by edge effects are not reported. A halo expands the area around a region selected for a DRC run so that surrounding geometries are included. Halos are used for all Calibre RealTime Custom runs.
