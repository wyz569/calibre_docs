Calibre nmOPC Operations

Calibre WORKbench and Calibre LITHOview support a number of core Calibre nmOPC operations. For instance, there are different types of analysis that can be performed on the results of an OPC run, as well as options to adjust how data is displayed.

The operations and where their controls are located are summarized in the following table.

Table 1. Calibre nmOPC Operations

Operation

Control

Location

Description

“Generating a Print Image”

PI

RET Flow Tool toolbar

Simulate OPC effects on geometries on opc layers that lie partially or fully within a selected area.

“Configuring an ILT Run”

ILT

RET Flow Tool toolbar

Use inverse lithographic transforms (ILT) to perform OPC.

“Fragment Information”

FInfo

Calibre WORKbench toolbar

Output marker layers and inspect data for individual fragments, including EPE, length, position, biasing movement, and so on.

“Mapping EPE from Correction Layers”

OPC Mapping

Fragment Information dialog box, (after using FInfo)

(Used only for multi-mask exposures) Display how EPEs are re-mapped from the correction layer to fragments on the opc layer.

“Fragment Generation”

Frag

RET Flow Tool toolbar

Calculate and show how edges are fragmented for OPC on the layout. Adjust the display properties of fragmentation points, mapping and measurement locations, and sites in a user-selected area.

“Cutline Operations”

Cutline

on the Calibre WORKbench icon bar

Calibre WORKbench toolbar

Compare one or more simulations along cutlines.

“Setting Cutline Properties”

Re-Cutline

RET Flow Tool toolbar

Change the display properties of cutlines.

“Creating an Aerial Image Contour Map”

MAP

RET Flow Tool toolbar

Create a contour map of the light intensity at the surface of the resist.

“Running Tests on Multiple Values With the Sweeping Tool”

Sweep

RET Flow Tool toolbar

Examine multiple values for specified command parameters.
