Setup File Operations

Calibre WORKbench and Calibre LITHOview use text files to configure important values and flags as input for multiple tools. Several tools in the RET Flow Tool have their own version of a setup file (.setup and .in extensions are typical). Each Calibre tool may have a setup file that is not cross-compatible with other Calibre tools.

The setup file editor is functionally the same for each tool that uses one. To create a new setup file, enter text into the main text entry field.
