Specifying Models

Many sessions require various types of models that can be specified in a setup file.

Procedure

  1. In your setup file, make sure you have one of the following:

    • A path to a valid litho model

    • Valid paths to an optical model using the following:

      • modelpath — Specify the directory containing your optical model.

      • optical_model_load — Specify the directory containing your optical model file.

  2. Make sure that you have mapped your layers from the setup file to the layers displayed in WORKbench.
