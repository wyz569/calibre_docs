Viewing Associated Tag Groups for a Fragment

You can find all the tag groups that your selected fragment belongs to through the Fragment Information dialog box.

Restrictions and Limitations

  • Calibre OPCpro setups are not supported. View EPE information for Calibre OPCpro runs in the Fragment Info area of the FInfo dialog box.

Prerequisites

  • Configured the run for your session as described in the “Using Sessions” chapter.

  • Loaded a layout file in the viewer main window.

  • Completed the task “Fragment Generation”.

Procedure

  1. In the layout viewer, select a layer containing a fragment iteration.
  2. In the viewer toolbar, click FInfo.


  3. The cursor changes to a hand; click a fragment to open the Fragment Information dialog box.
  4. In the Fragment Information dialog box, click the Tags icon.
    Note:

    Annotated tags, as defined by the -annotated or -aout command options, can also be displayed in the Calibre nmBIAS and Calibre nmOPC Fragment Information windows. As an example, the annotated NEWTAG statement highlighted in Figure 1 (see the red box) can be viewed.

    Figure 1. Annotated Tag Support

    The Tags view of the Fragment Information dialog box details the tag information for the selected fragment. Figure 2 shows the four main sections:

    Figure 2. Fragment Information Dialog Box (Tags)

    Table 1 describes the contents of this dialog box.

    Table 1. Fragment Information (Tags) Contents
     

    Item

    Description

    A

    Tags Info

    Shows which tag groups the selected fragment belongs to, if any. Any tags you define appear only if you select the Fragment Markers for All iterations prior to running OPC.

    B

    Executed Tag Script

    Highlights the tagging commands executed in the setup file. You can highlight and traverse the different fragments by clicking the Highlight Frags arrow keys, as well as clear the highlighting by clicking Clear Highlights.

    C

    Display Units

    Select display units in database units (dbus), microns (um), or nanometers (nm). You can view the information of the selected fragment for all iterations by enabling the View all iterations button.

    D

    Markers Display

    You can select either of the following options:

    • OPC Mapping is for double-exposure masks, indicating where EPEs are remapped from a correction layer to the opc layer.

    • Measurement Locations indicate where the EPE measurements are taken.
