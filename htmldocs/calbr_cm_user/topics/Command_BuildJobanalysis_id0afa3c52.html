build_jobanalysis

Related application: calcm_jobanalysis_app.tcl

Launches the categorization process immediately (instead of waiting for the next DBSTARTTIME) to rebuild the info_analysis table.

Usage

build_jobanalysis

Arguments

None.

Examples

% calcm_send_message build_jobanalysis
Analyzing info_jobs table (17749)...
