adjust_minmax

Related application: calcm_rmanager_app.tcl

Adjusts the minimum and maximum number of remote hosts for the specified job operation.

Note:

You can also access this command from the CalCM dashboard web interface with the required authentication.

Usage

adjust_minmax  job_id minmax_string

Arguments

job_id

Specifies the ID of the job whose minimum and maximum number of remote hosts you want to change.

minmax_string

A string specifying the number of minimum and maximum remote hosts for a list of job operations for the specified job ID.

Description

Use adjust_minmax with the calcm_send_message script to adjust the MINMAXOP string for the specified job ID. This information is passed to the calcm_rmanager_app.tcl application which adjusts the number of minimum and maximum remote hosts for the specified job operation list. The wildcard (*) can be specified for the job operation type.

Examples

calcm_send_message adjust_minmax job1 DRC:*=256/256,LITHO:*=128/1024
