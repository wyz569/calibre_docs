reset_quota

Related application: calcm_rmanager_app.tcl

Reads the quota configuration and re-calculates the quota.

Usage

reset_quota quota_configuration

Arguments

quota_configuration

Specifies the path to the quota configuration table. The default name is calcm_quota.conf.

Description

Use reset_quota with the calcm_send_message script in order to read the specified quota configuration and re-calculate the quota. This information is passed to the calcm_rmanager_app.tcl application which resets the quota hierarchy.

Examples

calcm_send_message reset_quota calcm_quota.conf
