reset_jobanalysis

Related application: calcm_jobanalysis_app.tcl

Updates the configuration for calcm_jobanalysis.tcl without restarting the CalCM daemon. To be used with the CONFPATH parameter.

Usage

reset_jobanalysis [configuration]

Arguments

configuration

Optional argument to specify the configuration for calcm_jobanalysis.tcl.

Examples

% more conf/calcmd.conf
…
APPLICATION TCL $CALCM_APP/calcm_jobanalysis_app.tcl FILE [
    CONFPATH = $CALCM_HOME/conf/jobanalysis.conf
]
…

% ./jobqueue/calcm_send_message reset_jobanalysis

Reading job analysis configuration from \
  <CALCM_HOME>/conf/jobanalysis.conf...
// -------------------------------------------------------------------

// This file is called by calcmd.conf -> calcm_jobanalysis_app.tcl ->
