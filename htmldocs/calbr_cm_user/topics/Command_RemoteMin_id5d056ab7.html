REMOTE MIN

Specifies a minimum number of remotes.

Usage

REMOTE MIN number

Arguments

number

Specifies a minimum number of remotes.

Description

This statement specifies a minimum number of remotes. The specified number is used in calcm_rmanager_app.tcl to keep the minimum number of remotes. This value can be overridden by calcm_rmanager_app.tcl. If ADJUSTMINMAXOP is enabled in calcm_rmanager_app.tcl, the minimum number of remotes specified by the REMOTE MINMAX statement overrides the REMOTE MIN value during a specified job operation.

Examples

REMOTE MIN 2
