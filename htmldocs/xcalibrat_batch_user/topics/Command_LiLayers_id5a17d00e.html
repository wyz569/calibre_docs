li_layers

Parameter for layer(s): device_li

An optional device_li layer parameter that specifies a space delimited list of li layer names used to associate the device_li layer with other li layers. Use this parameter to maintain proper physical connectivity between coplanar layers after applying bias.

Syntax

li_layers = {li_layer_name1  li_layer_name2 ... li_layer_namen}

Parameters

{li_layer_name1 li_layer_name2 ... li_layer_namen}

A space delimited list of li layer names enclosed in braces ({ }).

Examples

device_li = M0_OD1 {
	thickness = 0.09
	min_width = 0.045
	min_spacing = 0.045
	resistivity = 0.672
	layer_bias = -0.0070
	measured_from = ILD_nm
	zbottom = 0
	li_layers = {M0_ST1}
}
li = M0_ST1 {
	min_width = 0.02600
	min_spacing = 0.05800
	resistivity = 0.67
	measured_from = FOX
	zbottom = 0
}
