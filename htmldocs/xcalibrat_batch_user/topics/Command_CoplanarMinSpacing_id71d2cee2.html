coplanar_min_spacing

Type: conductor, device_li, diffusion, ground, li, pcaux, poly, resistor

An optional parameter for conducting layer definitions conductor, device_li, diffusion, ground, li, pcaux, poly, and resistor that specifies the minimum spacing used between coplanar layers.

Syntax

coplanar_min_spacing = value

Parameters

value

A positive floating point value that represents the minimum spacing used between coplanar layers.

Examples

coplanar_min_spacing = 0.04
