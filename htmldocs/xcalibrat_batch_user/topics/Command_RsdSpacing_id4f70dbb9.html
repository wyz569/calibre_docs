rsd_spacing

Parameter for layer(s): diffusion, pcaux, seed, src_drn, multigate

An optional parameter used in diffusion, pcaux, seed, src_drn, multigate layer definitions that specifies the raised source/drain to gate spacing for the device.

Syntax

rsd_spacing = value

Parameters

value

A floating point value used to specify the raised source/drain to gate spacing.

Examples

rsd_spacing = 0.044
