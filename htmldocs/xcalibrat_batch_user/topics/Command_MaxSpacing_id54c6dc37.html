max_spacing

Parameter for layer(s): conductor, device_li, diffusion, ground, li, pcaux, poly, resistor, seed, src_drn

Parameter for Table(s): RPV_VS_COUNT Table

When specified in a layer definition, max_spacing is an optional conducting layer parameter that specifies the maximum allowed metal spacing. When specified in the rpv_vs_count table, max_spacing specifies the maximum spacing between via arrays.

Syntax

max_spacing = value

Parameters

value

A floating point value used to specify the maximum allowed metal spacing or the maximum spacing between via arrays in a rpv_vs_count table specification.

Examples

Example 1

In a layer definition:

conductor = poly {
	thickness = 0.1
	min_width = 0.046
	min_spacing = 0.048
	max_width = 0.2
	max_spacing = 5
	r_sheet = 9
}

Example 2

Defined within an rpv_vs_count table:

via = via1 {
	...
	table = my_res_per_via {
		property = rpv_vs_count
		table_type = R
		value_type = absolute
		max_spacing = 3
		via_size = {0.026 0.026}
		count = { 4 6 8}
		ratio = { 0.2 0.3 0.4}
	}
	...
}
