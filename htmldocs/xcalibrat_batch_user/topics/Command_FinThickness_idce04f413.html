fin_thickness

Parameter for layer(s): multigate

A required multigate layer parameter used to specify the thickness of the fin.

Syntax

fin_thickness = value

Parameters

value

A floating point value that specifies the thickness of the fin.

Description

A required multigate layer parameter that specifies the thickness of the fin. This parameter is used only in multigate layer definitions. The fin_thickness is shown in Figure 1.

Figure 1. fin_thickness of the multigate layer

For more information on the multigate layer definition, see Multigate.

Examples

fin_thickness = 0.032
