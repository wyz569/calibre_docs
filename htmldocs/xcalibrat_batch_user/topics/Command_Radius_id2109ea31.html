radius

Parameter for layer(s): TSV

A required TSV layer parameter used to specify the radius of the through silicon via.

Syntax

radius = value

Parameters

value

A floating point value used to specify the radius of the TSV.

Description

The radius measurement for the TSV is shown in Figure 1.

Figure 1. radius for TSV Layers

For more information on TSV layer definition, see TSV.

Examples

Example 1

In Figure 1, Example 1 is an example of a solid TSV. In this case radius is 9. 

radius = 9

Example 2

In Figure 1, Example 2 is an example of a hollow TSV. In this case the radius is 9.
