gate_oxide_er

Parameter for layer(s): multigate

A required multigate layer parameter used to specify the permittivity of the gate oxide.

Syntax

gate_oxide_er = value

Parameters

value

A floating point value that specifies the permittivity of the gate oxide.

Description

A required multigate layer parameter that specifies the permittivity of the gate oxide. This parameter is used only in multigate layer definitions. For more information on the multigate layer definition, see Multigate.

Examples

gate_oxide_er = 6.0
