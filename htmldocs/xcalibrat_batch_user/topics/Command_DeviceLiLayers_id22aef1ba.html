device_li_layers

Parameter for layer(s): li

An optional li layer parameter that specifies a space delimited list of device_li layer names identifying the device local interconnect layers associated with it.

Syntax

device_li_layers = {li_layer_name1 li_layer_name2 ... li_layer_namen}

Parameters

{li_layer_name1 li_layer_name2 ... li_layer_namen}

A space delimited list of device_li layer names enclosed in braces ({ }).

Examples

li = M0_STI1 {
	min_width = 0.02600
	min_spacing = 0.05800
	resistivity = 0.67
	device_li_layers = {M0_OD1}
}
device_li = M0_OD1 {
	min_spacing = 0.05800
	resistivity = 0.672
	layer_bias = -0.0070
	devices = { ngate_mac ngate_15_mac
					pgate_mac pgate_15_mac}
}
