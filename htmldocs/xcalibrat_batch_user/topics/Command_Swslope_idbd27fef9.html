swslope

Parameter for layer(s): dielectric

An optional conformal or trench dielectric layer parameter used to specify the angle of the side wall in degrees.

Syntax

swslope = value

Parameters

value

An integer valued used to specify the angle of the side wall in degrees. It should be a positive value less than 180. The default is 90 degrees.

Description

The angle of the side wall, swslope, is shown in Figure 1. If this parameter has not been specified the parser issues an informational message that notifies you that the parameter was not found and the default of 90 degrees is used. The xCalibrate tool also verifies that the actual slope angle entered intersects the top plane for this material thickness. This parameter may only be used when the diel_type parameter is set to conformal or trench.

Figure 1. Sidewall Slope

For more information on dielectric layer definition, see Dielectric.

Examples

swslope = 90
