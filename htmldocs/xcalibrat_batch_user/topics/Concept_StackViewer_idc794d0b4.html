Stack Viewer

The stack viewer shows a graphical representation of a MIPT file. You can customize the representation to display only certain measurements, as well as defined devices. It only reads MIPT format files.

Stack Viewer Invocation

To invoke the stack viewer, enter the following at a shell prompt:

xcalibrate -view filename.mipt

The filename is required; you can load a different MIPT file later using the Load File button (see Figure 1).

For details on the command line syntax for the stack viewer, see “xcalibrate -view”.
