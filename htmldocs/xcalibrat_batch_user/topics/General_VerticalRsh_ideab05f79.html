VERTICAL_RSH Table

Use the vertical_rsh table to specify vertical resistance from the bottom of a gate to the middle of a horizontal gate.

Table Syntax

table = my_vertical_rsh_table {
	property = vertical_rsh
	table_type = R
	dim_type = actual
	value_type = absolute
	width = {w1 w2 ... wn} #variable1
	length = {l1 l2 ... ln} #variable2
	value = { v(w1,l1) v(w1,l2) ... v(w1,ln),
				 	...,
			v(wn,l1) v(wn,l2) ... v(wn,ln) }
}

Description

The vertical gate resistance model connects the gate pin to the horizontal gate resistors as shown in Figure 1. This creates a “T” resistor model with the gate pin at the bottom of the “T” and the poly endcaps at the two ends of the top of the “T”.
