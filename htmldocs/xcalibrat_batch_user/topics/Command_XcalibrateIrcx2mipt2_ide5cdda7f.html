xcalibrate -ircx2mipt2

Command used to convert an iRCX file (filename.ircx) to an MIPT file.

Usage

xcalibrate ‑ircx2mipt2 ircx_filename

Arguments

‑ircx2mipt2 ircx_filename

A required argument set used to convert an iRCX file to an MIPT file. The IRCX file can be encrypted or unencrypted. Translation of an encrypted iRCX file produces an encrypted MIPT file, and translation of an unencrypted iRCX file produces an unencrypted MIPT file.

Examples

The following command line converts an iRCX file called input.ircx to an MIPT file called input.mipt:

xcalibrate -ircx2mipt2 input.ircx
