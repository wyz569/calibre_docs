Keywords

Keywords are assigned a value by using the equals sign (=).

All keywords must have:

  • A value assigned to them or a parsing error is generated.

  • Values specified on same line as the keyword.

  • A single value assigned as a single one line string without spaces. Multiple string values may be assigned, but must be enclosed within quotes.

Keywords cannot be used as reference names. For more information on available MIPT keywords see MIPT File Elements.
