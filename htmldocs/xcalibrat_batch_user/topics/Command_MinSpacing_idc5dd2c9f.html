min_spacing

Parameter for layer(s): base_via, conductor, contact, device_li, diffusion, ground, li, pcaux, poly, resistor, seed, src_drn, via

A required parameter for conducting layer definitions conductor, device_li, diffusion, ground, li, pcaux, poly, and resistor that specifies the minimum drawn spacing of metal objects on this layer. This parameter is optional for seed and src_drn layer definitions. For contact, via, and base_via layer definitions, this parameter is required to specify the spacing between vias (or contacts) and conductors on their respective layers.

Syntax

min_spacing = value

Parameters

value

A required floating point value used to specify the minimum drawn spacing of metal objects on this layer.

Examples

min_spacing = 0.125
