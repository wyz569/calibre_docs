depletion_width

Parameter for layer(s): TSV

A required TSV layer parameter that specifies the depletion region width of a Through-Silicon-Via (TSV).

Syntax

depletion_width = value

Parameters

value

A floating point value that specifies the depletion region width.

Description

The depletion_width measurement is shown in Figure 1.

Figure 1. depletion_width for TSV Layers

For more information on TSV layers, see TSV.

Examples

depletion_width = 10.4
