count

Parameter for Table(s): RPV_VS_COUNT Table

An optional parameter whose values specify the number of vias in the array. Can only be used in the one-dimensional rpv_vs_count property table.

Syntax

count = { c1 c2 ... cn}

Parameters

c1 c2 ... cn

A list of one or more values that represent the number of vias in an array enclosed in required braces ({ }).

Examples

count = { 12 16 32 }
