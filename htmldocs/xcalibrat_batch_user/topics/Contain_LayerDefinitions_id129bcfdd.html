Layer Definitions

The process technology is principally defined by the layers.

The layer definition syntax is:

layer_type = layer_name {
   list_of_parameters
}

A layer_type keyword is followed by an equal sign, a unique user-specified layer_name, and a list of parameters enclosed in required braces, ({ }). The layer parameters describe the properties of the layer. Each parameter must appear on its own line. The parameters within the braces are order independent.

There are three categories of required layers:

  • Dielectric layers

  • Conducting layers:

    • Conductor

    • Ground

    • Poly

    • Diffusion

    • li (Metal or Interconnect layers)

  • Contact and Via layers

The following layer types are optional:

  • Base

  • Base_via

  • Device_li

  • Derived

  • Multigate

  • Pad

  • PCaux

  • Resistor

  • Src_drn

  • Seed

  • TSV

  • uBump
