gate_extension

Parameter for layer(s): multigate, seed

An optional multigate and seed layer parameter that specifies the distance the gate layer extends outside the diffusion area.

Syntax

gate_extension = value

Parameters

value

A floating point value in microns that specifies the distance the gate layer extends outside the diffusion area.

Description

Use this parameter to control gate resistance for more accurate modeling. The gate_extension parameter specifies the amount of the device poly layer that extends past the diffusion region for resistance calculation as shown in Figure 1.

Figure 1. gate_extension for FinFET

A warning is generated during calibration if this parameter is not specified in the multigate or seed layer definition.

Examples

gate_extension = 0.01
