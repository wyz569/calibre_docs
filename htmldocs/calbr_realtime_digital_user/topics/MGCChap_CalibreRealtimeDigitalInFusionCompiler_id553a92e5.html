Calibre RealTime Digital in Fusion Compiler

The Calibre RealTime Digital interface is integrated with the Fusion Compiler design tool from Synopsys. Calibre RealTime Digital integrates flat Calibre nmDRC with supported place and route tools so that Calibre nmDRC can be run directly from the design tools.

Refer to “Calibre RealTime Digital Introduction” for an overview and general system requirements.
