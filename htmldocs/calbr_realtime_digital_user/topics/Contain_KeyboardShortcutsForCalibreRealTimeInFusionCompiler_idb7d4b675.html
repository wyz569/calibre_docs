Keyboard Shortcuts for Calibre RealTime in Fusion Compiler

A set of standard keyboard shortcuts are provided for Calibre RealTime Digital in Fusion Compiler.

The keyboard shortcuts are defined in the following file:

$CALIBRE_HOME/shared/pkgs/icv/tools/realtime/icc2/icc2_realtime_bindkeys.tcl

This file is not loaded by default to avoid possible conflicts with existing keyboard shortcuts. The definition of the bindkey functions is given in the following table. The section “Defining and Loading Keyboard Shortcuts for Calibre RealTime in Fusion Compiler” describes how to define and load your own keyboard shortcuts.
