Multilevel Session Configuration File Support in Calibre RealTime Digital

Calibre RealTime Digital supports reading the session configuration file from a specified directory or multiple directories. If configuration files are located in multiple directories, the order of precedence is set with the environment variable MGC_REALTIME_CONFIG_DIRS. This feature allows system and project administrators to define settings for a group or project while still allowing settings to be adjusted by individual users.

You can also save the session configuration file in a directory other than your home directory.
