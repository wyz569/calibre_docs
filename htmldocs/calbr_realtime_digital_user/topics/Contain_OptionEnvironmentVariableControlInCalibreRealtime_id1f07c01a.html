Option and Environment Variable Control in Calibre RealTime Digital

Calibre RealTime Digital provides commands to control options and environment variables from the command interface of supported design tools. You can also set environment variables to specify the path to certain input files.

For more information on these commands, see the “Calibre RealTime Digital API Reference”.
