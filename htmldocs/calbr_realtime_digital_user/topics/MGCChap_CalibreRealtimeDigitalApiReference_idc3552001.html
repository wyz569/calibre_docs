Calibre RealTime Digital API Reference

Use the Calibre RealTime Digital API commands to get and set Calibre RealTime Digital environment variables and options within the design environment. The Calibre RealTime Digital API also offers script-level access to Calibre RealTime operations.

The commands that control environment variables and options can be used in all design environments. The commands that run DRC, highlight results, and run configuration managements are only supported in Cadence Innovus and Synopsys IC Compiler II.
