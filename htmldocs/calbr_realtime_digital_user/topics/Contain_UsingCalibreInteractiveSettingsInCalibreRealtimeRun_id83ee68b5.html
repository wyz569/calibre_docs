Using Calibre Interactive Settings in a Calibre RealTime Digital Run

If you use Calibre® Interactive™ nmDRC and want to use the same options for your Calibre RealTime Digital run, you can specify the Calibre Interactive runset in the Calibre RealTime Options dialog box. Calibre RealTime Digital calls Calibre Interactive to generate the Calibre Interactive control file, and the control file is used for the Calibre RealTime Digital run. If the runset includes a customization file, the Customization GUI is opened and the settings are applied to the control file.
