SPG DRC Rules

The SPG flow provides you with the ability to define minimum width and space constraints for applying basic DRC rules to the generated layout clips.

To apply minimum width and space constraints in the SPG flow, specify the following options in your SPG options file:

MinWidthInPixels=integer
MinSpaceInPixels=integer
Note:

The minimum width and space DRC rules for the SPG flow are not applied to the border cells in the generated patterns.

See “SPG Flow Options” and “Running Calibre LSG With SPG” for more information on the SPG options and flow.

The following is an example of how the SPG flow applies a minimum width constraint for a 4 x 4 generated layout clip with MinWidthInPixels=2. During pattern generation, the flow sets prefill flags designated as “FILL” (pattern 5.1), “EMPTY” (pattern 1.1) or “NA” (unassigned). The prefill flags determine the pattern for the next cells to fill in the layout clip in order to meet the constraint. The flow performs the following steps:

  1. Checks the fill status of the current cell (target) for pattern 5.1 (filled) and minimum width > 1 pixel.

    1. Checks the cell to the left of the target for pattern 1.1 (empty), if true, it marks the next cell to the right of the target to be filled with pattern 5.1 (fill) and sets the prefill flag to FILL.

    2. Checks the cell above the target for pattern 1.1 (empty), if true, it marks the next cell below the target to be filled with pattern 5.1 (fill) and sets the prefill flag to FILL.

  2. Continues with the normal flow, checking the prefill flag (either FILL, EMPTY, or NA) for the next cell to fill in the layout clip. If the prefill flag for the next cell is set to FILL, only pattern 5.1 (fill) is used.

Figure 1. Apply Minimum Width Constraint

For the minimum space constraint, the same steps are performed, but for this case, the flow checks the target for pattern 1.1 (empty) instead of 5.1 (fill). The next cell to the left and the next cell above the target are checked for pattern 5.1 (fill) instead of pattern 1.1 (empty). If the prefill flag for the next cell is set to EMPTY, only pattern 1.1 (empty) is used.

Figure 2. Apply Minimum Space Constraint

Note:

You can also define your own DRC rules with the format used for Calibre LSG. See “Calibre LSG Rules” and “Calibre LSG GUI Reference”.
