Template Tab

To access: From a Calibre WORKbench menu bar, Litho > Calibre LSG Layout Generation, or from a Calibre WORKbench Tcl shell call—see “Invoking Calibre LSG from a Tcl Shell Call”.

Click the Template tab.

Use this tab to predefine Calibre LSG unit pattern positions in a layout clip template file.

Description

Figure 1. Template Tab

Objects

Object

Description

A

Template File pane — Use to perform template file and pattern actions.

Definitions (left to right):



  • Create New Template File — Open (+) a new template file.

  • Close Template File — Close (X) current template file.

  • Import Template File — Import a saved template file.

  • Save Template File — Save the current template file.

  • Save as Template File — Save the current template file in a specified location.

  • Reset Template — Reset the template pattern in the Template Grid.

  • Description — Add a description for the template file.

B

Template Grid pane — Use to perform unit cell placement on a grid and create and display a template.

Definitions (left to right):



  • Rows: — Specify the number of rows in the template.

  • Cols: — Specify the number of columns in the template.

C

Unit Cells pane — Use to define and drag‑select unit cells for placement in the Template Grid.

Definitions (top to bottom):

  • Options — Select unit cell rotation options from the dropdown list.



  • Patterning Level — Display unit cell combinations for each mask layer (up to four levels).

D

Position pane — Display position, unit pattern definition, and delete (clear) icon.

E

Reference pane — Show the active template information.

Usage Notes

For a description of the file formats and argument options used for imported files see “calibre -lsg” and “Calibre LSG File Formats”.
