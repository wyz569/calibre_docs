Rule Editor Tab

To access: From a Calibre WORKbench menu bar, Litho> LSG Layout Generation, or from a Calibre WORKbench Tcl shell call—see “Invoking Calibre LSG from a Tcl Shell Call”.

Click the Rule Editor tab.

Use this tab to input Calibre LSG rules, priority (weight) settings, and unit pattern information. This window mode is the same as the Calibre LSG main GUI.

Description

Figure 1. Rule Editor Tab

Objects

Object

Description

A

Rule File pane — Use to perform rule file actions and display rule information.

Definitions (left to right):



  • Create New Rule File — Open (+) a new rule file.

  • Close Rule File — Close (X) current rule file.

  • Import Rule File — Import a saved rule file.

  • Save Rule File — Save the current rule file.

  • Save as Rule File — Save the current rule file in a specified location.

  • Add New Rule — Add (+) a new rule to the rule file.

  • Columns — Use the dropdown list to choose the columns to display.

  • Filter Rules — Filter rule information by name or comment.

B

Unit Cells pane — Use to define and drag‑select unit cells for placement in the Rule Grid and Composite Grid.

Definitions (top to bottom):

  • Use Default — Select default or custom unit pattern options.

  • Options — Select unit cell rotation options.



  • Border Cell — Drag‑select Border Cell to Rule Grid location.



  • Patterning Level — Display unit cell combinations for each mask layer (up to four levels).

C

Rule Grid pane — Use to perform unit cell placement on a grid and display relative placement information.

Definitions (left to right in pane):



  • Targets — Display target unit cell information.

  • Rule Grid — Choose (+) or (-) to change grid size.

  • Neighbor Combinations/Combinations — Display neighbor unit cell information.

D

Priority File pane — Use to create, import, and save priority (weights) for unit cells and rotations to a file.

Definitions (left to right):



  • Create New Priority File — Open (+) a new priority file.

  • Close Rule File — Close (X) current priority file.

  • Import Priority File — Import a saved priority file.

  • Save Priority File — Save the current priority file.

  • Save as Priority File — Save the current priority file in a specified location.

  • Add/Remove cells in Priority file — Add and remove unit cells in the priority file.

E

Composite File pane — Use to create, import, and save composite patterns to a file.

Definitions (left to right):



  • Create New Composite File — Open (+) a new composite file.

  • Close Composite File — Close (X) current composite file.

  • Import Composite File — Import a saved composite file.

  • Save Composite File — Save the current composite file.

  • Save as Composite File — Save the current composite file in specified location.

  • Add New Pattern — Add (+) a new composite Id and pattern.

F

Composite Grid pane — Use to perform unit cell placement on a grid and create and display composite patterns.

Definitions (left to right):



  • Rows: — Specify the number of rows in the composite pattern.

  • Cols: — Specify the number of columns in the composite pattern.

Definitions (left to right):



  • Composite File tab —Create and display composite patterns (default pane).

  • Borders tab — Create and display composite pattern borders.

Usage Notes

For a description of the argument options and the format of imported files, see “calibre -lsg” and “Calibre LSG File Formats”.
