SPG Generated Output

The SPG flow generates all possible combinations for small layout clips. For larger layout clips with more pattern combination, there is an option for generating incremental output.

The following figure shows an SPG flow generated layout with 64 (2^6) combinations of a 2 x 3 unit cell pattern. In this case, the output layout clips are merged (the default behavior). See “Running Calibre LSG With SPG” for information on how to run the flow.
