Filtering False Defects

Performing aerial simulation for false defects is optional during a Mask Pattern Classification run.

Procedure

In the Miscellaneous pane of the Calibre DefectClassify window, click Filter out False Defects before simulation.
Note:

This option is effective only if Inspection mode is set to High Resolution Mode and the Scanner option is enabled.

Figure 1. Filter False Defects

If this option is enabled, Calibre DefectClassify first determines whether the defect classification is Real or False and performs aerial simulation only if the defect is Real. If this option is not enabled, Calibre DefectClassify runs aerial simulation for all defects. The list of False defect categories are specified under the FalseClassifications node for their respective inspection types in the dat-ini.xml file for Calibre DefectReview.
