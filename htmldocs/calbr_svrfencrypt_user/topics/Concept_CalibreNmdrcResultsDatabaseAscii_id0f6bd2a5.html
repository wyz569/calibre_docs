Calibre nmDRC Results Database (ASCII)

Encrypted rules generate protected output in ASCII nmDRC results databases. Partially encrypted rules will generate non-protected output in ASCII for Calibre nmDRC results databases.
Table 1. ASCII Calibre nmDRC Results Database (Complete Encryption)

LAYOUT PATH 'layout.gds'

LAYOUT SYSTEM GDSII

LAYOUT PRIMARY '*'

DRC RESULTS DATABASE 'drc_results'

DRC CHECK TEXT ALL

LAYER V 0

#ENCRYPT

1_8_sc_vias { @ this is a comment

WITH NEIGHBOR V == 1

SPACE == .008 SQUARE CENTERS

}

#ENDCRYPT
