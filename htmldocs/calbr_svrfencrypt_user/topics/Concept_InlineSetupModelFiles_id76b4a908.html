Inline Setup and Model Files

When the setup or model files are used inline within rules and the rules are encrypted, the inline setup or model file transcript output is protected.

Examples of this are shown in Table 1 and Table 3. When encrypted rules that use external setup or model files are run, the setup or model file transcript output is not protected (see Table 2 and Table 4).
