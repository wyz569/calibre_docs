Hierarchical LVS

Calibre nmLVS-H is a hierarchical LVS application. Calibre nmLVS-H maintains design hierarchy and exploits this hierarchy to reduce processing time, memory use, and LVS discrepancy counts.

The command line options for Calibre nmLVS are discussed under “Calibre nmLVS and Calibre nmLVS-H Command Line”. The -hier, -hcell, and -spice options are of particular interest. You can also use Calibre Interactive to specify a hierarchical LVS run.
