ASCII DRC Results Database Format

Calibre nmDRC generates an ASCII DRC results database by default. This format is used by Calibre RVE and ICrules for viewing geometric results.

No blank lines appear in the ASCII DRC results database, and data always start at the beginning of the line. Figure 1 shows an example ASCII DRC results database. Depending upon the operations that write to the database, properties and other elements can also appear.
