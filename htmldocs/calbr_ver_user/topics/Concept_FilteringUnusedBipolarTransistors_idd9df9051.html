Filtering Unused Bipolar Transistors

Calibre nmLVS can internally filter unused bipolar transistors from the source and layout circuits. Bipolar filtering is useful for verifying gate-array layouts.

The LVS Filter Unused Bipolar specification statement controls unused bipolar transistor filtering. When YES is specified, filtering is performed only for properly-formed bipolar transistors, specifically, instances with component type Q or equivalent device types specified in an LVS Device Type statement. The devices must have at least three pins with the standard names as specified in “Table 1.” Q transistors with base and emitter tied together are filtered out.
