Warning Messages

Warning messages should be reviewed to determine if they indicate a real problem in your design.
Table 1. Calibre xRC Warning Messages

Message

Possible Causes

Layer “layername” is not mapped to a calibrated layer.

This warning occurs during the PDB stage. The layername is not mapped to a calibrated layer. See PEX Map statement.

CAPACITANCE ORDER statement is incomplete.

This warning occurs during the PDB stage. The Capacitance Order statement is missing a mapped layer name.

Bad source pin map number. Layout cellname.

This warning occurs during the formatter stage. A filename.stl file has been created for the cellname/hierarchy in a previous run and has missing or added pins compared to the new run.

Detected ideal (non-resistive) connection through layer layername.

The warning occurs for each layer that is found to have no assigned or inherited resistivity, but is specified in a Connect statement. It does not list all the derived layers, only layers found while looking at resistance connectivity.

No density window defined for layer layername - switching to nominal thickness.

The warning occurs when the PEX Density Window statement has not been specified. In this case, Calibre xRC uses nominal thickness during extraction.

Detected High Resistance Value on Cell: “cellname”, Layer: “layername”.

The warning occurs when resistors with values greater than or equal to 1e6 ohms (1Megaohm) are extracted on a layer. To eliminate this warning, make sure that the rules that reference layername in your rule deck are correct.
