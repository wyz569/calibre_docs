About the Example Design

The example design is a simple SRAM memory block which uses design hierarchy. The layout shows some of the block-level routing and major sub-circuits in the design, representing a simple design and how routing interacts with the different sub-circuits.

Figure 1 shows the layout used to illustrate the output of the extraction types. There are more cells and intentional devices at each level. For example, the word block is composed of bit cells. Each bit cell is composed of NMOS and PMOS transistors. These details are not shown in the layout.
