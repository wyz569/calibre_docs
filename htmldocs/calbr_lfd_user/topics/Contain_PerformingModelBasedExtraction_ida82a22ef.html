Performing Model Based Extraction

The typical timing analysis flow with Model Based Extraction includes Calibre LFD, Calibre nmLVS, and parasitic extraction.

Figure 1 shows a typical timing analysis flow using Model Based Extraction. Table 1 provides a description of each extraction type.

Figure 1. Timing Analysis Flow Using MBE

Table 1. The Timing Analysis Flow Using MBE

Type of Extraction

Description

A

Run Calibre LFD to generate the CSI layers and the CSG files for each poly condition/active condition combinations.

B

Run parasitic extraction on the simplified interconnect (CSI) layers.

C

Run discrete component extraction on each based on a CSG file containing the measurements that represent the adjusted dimensions of the transistor.
