Viewing MBH Results

This procedure shows you how to view the suggested hotspot fixes from running Calibre LFD with MBH. You display the design database in a layout viewer or editor and use Calibre RVE to view the hint sites.

Prerequisites

  • The Calibre DESIGNrev or Calibre WORKbench layout viewer. From other viewers, invoke Calibre Interactive from the Calibre menu.

  • The layout database in GDS or OASIS format.

  • The RDB hint file from “Running Calibre LFD with MBH”.

Procedure

  1. Load the MBH RDB hint file in Calibre RVE:
    1. Open the layout database in your layout viewer/editor.
    2. In Calibre DESIGNrev or Calibre WORKbench, choose Verification > Start RVE.
    3. In the Calibre RVE dialog box, select DRC/ERC database type, supply the path to the MBH RDB hint file, and then click Open.
  2. View the suggested layout fixes:
    1. In the upper right of the Calibre RVE window, click the result number for a specific check to select it.
    2. Highlight the selected site with the highlight function of Calibre RVE in one of the following ways:

      • Click the right mouse button and select Highlight from the pop-up menu.

      • Click the Highlight Selected Result icon in the toolbar.

    3. Scroll down in the Calibre RVE window directly below the numbered check sites to view the hint text. There may be more than one hint per site.
    4. In the graphic window of the layout viewer, ensure that the Layers browser selection shows the targeted MBH layers.
    5. Click on the numbered Hint text in the Calibre RVE window to view the potential fix in the layout viewer.
    6. Toggle the views of the suggested modifications to the layout shapes by clicking on the hints text in the MBH Hints Window. For example in Figure 1, click on add Poly or drop Poly.
