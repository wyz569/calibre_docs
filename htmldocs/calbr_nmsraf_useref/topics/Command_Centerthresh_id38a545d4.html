centerthresh

Conflict Keywords

Specifies the threshold of a conflict region. Used by RET SBAR only.

Usage

centerthresh threshold [sbwidth width]

Arguments

threshold

A positive floating point number that specifies a threshold value for a conflict region.

sbwidth width

An optional keyword and positive floating point number that specifies the width of secondary scattering bars invoked by centerthresh. The minimum value you can specify is 2 database units. If sbwidth is not specified, no width-based secondary SRAFs are generated. The sbwidth keyword can be set by priority alone for all cnsraf templates.

Description

An optional keyword that specifies a threshold for the length and width of a conflict region determined by setlayer cnsraf. The centerthresh keyword is used to consolidate SRAFs initially generated that violate MRC constraints.

If centerthresh is not specified, one square SRAF is placed in the center of the conflict region. For more detail regarding the size of the SRAF placed in the conflict region, see the sraf and oversizesraf keywords.

The centerthresh keyword can be set globally and by priority for all cnsraf templates.

Results of processing the conflict region depend on the value of centerthresh compared to the conflict region extents as shown in the table (where maxconflict is greater than or equal to the conflict region):

Table 1. Conflict Region Processing

Conflict Region Extent Evaluation

Conflict Region Resolution

Both extents are less than centerthresh

A square SRAF is placed in the center of the conflict region; see Figure 1.

Only one extent is less than or equal to centerthresh

A rectangular SRAF is centered along the longer extent exceeding centerthresh; see Figure 2.

Neither extent is less than centerthresh

The conflict region is removed; see Figure 3.

Examples

Example 1

A conflict region is shown where X and Y extents are less than centerthresh, so a square SRAF is placed at the center of the conflict region. The size of the square SRAF is specified by the sraf keyword.

Example 2

A conflict region is shown where only the X extent is less than centerthresh. The initial SRAFs forming the conflict region are resolved to a single SRAF placed along the longest length of the conflict region. The width of the SRAF is set by the sbwidth keyword.
