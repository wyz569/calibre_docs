maxoffset

Constraint Keywords

Specifies the maximum distance from an SRAF to a target shape. Used by RET MBSRAF only.

Usage

maxoffset value

Arguments

value

A positive floating point number specified in microns.

Description

An optional keyword that specifies the maximum distance from target shapes that SRAFs can be placed. The maxoffset keyword may be specified only once in a LITHO FILE, for either priority or global assignment, and must be specified before all cnsraf or mbsraf templates.

Note:

For recipes requiring placement of varying SRAF widths by multiple localmaxgradient templates, Siemens EDA recommends against using the maxoffset keyword.
