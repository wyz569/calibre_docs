Importing an Existing OPC Session to RFT

You import the Calibre nmOPC session into the RET Flow Tool.

Prerequisites

  • An nmOPC session file containing Calibre nmOPC simulation commands

Procedure

  1. In the RET Flow Tool, add an existing nmOPC session file by choosing File > Add Existing Session.
  2. In the dialog box, choose the .lsf file of interest and click OK.

    The session is loaded into RFT with some default settings.

    Figure 1. nmOPC Session

  3. Click the OPC button in RFT to generate OPC only on the small area in which you previously zoomed in.
    Figure 2. Run nmOPC

    OPC is generated and layers are added to the design.

    Figure 3. OPC Output on Design
