modelpath

LITHO FILE Statement and Commands

Specifies the directories to search for optical and resist models. Used by RET MBSRAF only.

Usage

modelpath directory [:directory ...]

Arguments

directory [:directory ...]

A required argument that specifies a colon-separated list of directories to search. The directories can be specified as either absolute or relative paths.

Description

A required keyword that specifies the directories to search for optical and resist models.

Examples

Example 1 — Absolute Path

modelpath /export/home/calibre_wrk/models

Example 2 — Relative Path

modelpath ./models

Example 3 — Multiple Paths Require a Colon Delimiter

modelpath ./models:../test_models:/usr/bin/opt/models
