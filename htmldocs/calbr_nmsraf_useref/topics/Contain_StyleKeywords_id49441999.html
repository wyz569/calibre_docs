Style Keywords

Provides management of SRAF output style.

For organizational keyword information, refer to “sraf_options”.

Table 1. Style Keywords

Keyword

Description
algorithmModifies geometric consistency of generated SRAF candidates. Used by RET MBSRAF only.
centerslitDisables slit interpolation. Used by RET EUV MBSRAF only.
centerwidthEnables the generation of SRAFs equidistant from target edges. Used by RET MBSRAF.
collecttypeCollects seed pixels near target corners to generate square SRAFs. Used by RET MBSRAF only.
cornerprefSpecifies the SRAF generation method at target shape corners. Used by RET MBSRAF only.
cornerprefnegativeSpecifies the negative SRAF generation method at target shape corners. Used by RET MBSRAF only.
gradientfactornegativeSpecifies a gradient factor value for negative SRAFs. Used by RET MBSRAF only.
gradientshiftSpecifies a gradient shift value. Used by RET MBSRAF only.
image_optionsA LITHO FILE command referencing the Lithomodel file and an sraf_options command referencing the litho_model name. Used by RET MBSRAF only.
intersectionEnables valid SRAFs to be merged at various junctions. Used by RET SBAR and RET MBSRAF.
intersectionnegativeEnables valid negative SRAFs to be merged at various junctions. Used by RET SBAR and RET MBSRAF.
jogignoreOverrides jogs in generated SRAFs. Used by RET MBSRAF only.
modeSpecifies a generated SRAF orientation. Used by RET MBSRAF only.
negativeEnables negative SRAF output. Used by RET SBAR and RET MBSRAF.
peaktypeSpecifies the method of SRAF propagation. Used by RET MBSRAF only.
pw_conditionSpecifies a process window condition. Used by RET MBSRAF only.
slopeweightSpecifies the gradient slope weight. Used by RET MBSRAF only.
symmetricEnsures all templates generate SRAFs with symmetry. Used by RET SBAR only.
