Performing Gradient Mapping

You can display gradient maps superimposed on your layout using Calibre WORKbench.

Prerequisites

  • A previously performed SRAF and gradient map run

Procedure

  1. In the Calibre WORKbench layout window double-click the visible gradient map (or select Edit > Maps/Contours). This invokes the Edit Map/Contour form. (see Figure 1).
    Figure 1. Edit Map/Contour Form

  2. Turn off the gradient map and leave the sdf map on.

  3. Click Apply.

    The sdf map appears as you have changed it.

    Figure 2. SDF and Gradient Maps



  4. Turn the gradient map back on for the next procedure, “Drawing Intensity Cutlines”.
  5. Click Apply or OK.

Results

You have modified the display of the gradient map for a metal layer.
