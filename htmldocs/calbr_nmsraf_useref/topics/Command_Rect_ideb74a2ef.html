rect

Templates

Places a rectangular SRAF relative to the center of the target shape. Used by RET SBAR only.

Usage

rect
[tag name ...]
{length value width value}
{pitchx value pitchy value}
[conflictonly]
[orient {horizontal | vertical}]
[prior value]

Arguments

tag name ...

An optional keyword specifying one or more tag names used in the template. The tag name identifies a group of edges or shapes to be used with the template. If specified, tags must appear as the first arguments in the template.

length value width value

Two required keywords, length and width, that specify the longer and shorter edges of a rectangular SRAF, respectively. The length and width values can be identical.

pitchx value pitchy value

A required keyword pair that specifies the horizontal and vertical distance from the center of the target shape to the center of the SRAF, respectively.

conflictonly

An optional keyword that outputs SRAFs within conflict regions only; no other SRAFs of this template are generated. Conflict regions are formed where neighboring SRAFs generated by this template violate the MRC rules you specify in the template. The conflictonly argument and dynamicsizing keyword cannot be used within the same priority. Siemens EDA recommends using the conflictonly argument only when conflict resolution shapes are relatively small and not using conflictonly for larger shapes such as line layer designs.

orient horizontal | vertical

An optional keyword that enables asymmetric templates to be placed and oriented based on the orientation of non‑square rectangles. Specifying orient horizontal applies SRAFs to target shapes as if the SRAFs were oriented horizontally (the long edge of the rectangle is parallel to the x-axis). Specifying orient vertical applies templates to rectangular target shapes as if they were oriented vertically (the long edge of the rectangle is parallel to the y-axis).

prior value

An optional keyword that specifies a numeric identifier for placed SRAFs. Identifiers are applied sequentially, so a value of 1 is applied first, then 2 is applied, and so on. The default priority is 1. See “priority” for more information.

Description

An optional template and argument set that places a single rectangular SRAF the specified distance from the center of a target shape. The center of the rectangular SRAF is placed (pitchx, pitchy) from the center of the target shape. This is an asymmetric template that is disabled with the symmetric keyword. This template cannot generate negative SRAFs.
