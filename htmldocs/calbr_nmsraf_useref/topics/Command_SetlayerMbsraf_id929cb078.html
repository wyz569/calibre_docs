setlayer mbsraf

LITHO FILE Statement and Commands

Generates SRAFs using a model-based methodology. Used by RET MBSRAF only.

Usage

setlayer output_layer = mbsraf target_layer OPTIONS statement_name

Arguments

output_layer

The required name of the new output layer. The new layer is created and the results of the operation are written to the layer. This layer name must be used in subsequent operations to output layer information.

target_layer

A required input layer that specifies the target layer.

OPTIONS statement_name

A required keyword and name pair that calls the previously defined sraf_options statement. For more information with mbsraf options, see sraf_options statement in this manual.

Description

This command is used inside of a LITHO FILE statement to generate SRAFs as defined by keywords in the sraf_options statement.

Examples

The following example creates one row of SRAFs that are parallel to the edges and another row of SRAFs that are rotated 45-degrees at the corners of the target layer.

Example 1. Use of setlayer mbsraf
LITHO FILE sample_setup [  

# -------------------- Model path and names --------------------
	modelpath /user/opt/models/model1:/user/opt/models/mbsraf_model
	optical_model_load opticsfile e1_f+0.0000
	resist_model_load resistfile model1.mod
# -------------------- Layer info --------------------
	background 0.0215 -0.0085
	layer sraf_target hidden 0.9785 0.0085
	layer dummy_opc hidden 0.9785 0.0085
	layer sraf_ref_layer hidden 0.9785 0.0085
	layer no_asst_marker hidden 0.9785 0.0085

# ------ specify max tile size to control memory use ------
	tilemicrons 25
# ------ sraf_options statement ------
	sraf_options mbsraf.opt {
		correctionlayer dummy_opc 
		offsetlayer no_asst_marker 
		optical opticsfile dose 1.0 
		resist resistfile
		algorithm 0
		peaktype 4dir
		mode nearest 
		localmaxgradient width 0.030 prior 1 
		priority 1 dynamicsizing 0.03 minoffset 0.010 minfeaturespace 0.035
	}

# ------ setlayer mbsraf call referencing sraf_option statement ------
	setlayer sb_sample = mbsraf sraf_target OPTIONS mbsraf.opt 
]
