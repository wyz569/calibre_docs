minareaextension

Constraint Keywords

Specifies the minimum area for 45-degree SRAF extensions. Used by RET SBAR and RET MBSRAF.

Usage

minareaextension value

Arguments

value

A floating point number specified in square microns. The minareaextension keyword may be specified globally or by priority. The default value is 0.

Description

An optional keyword that specifies the minimum area allowed for any 45-degree SRAF extension. 45-degree SRAF extensions with an area smaller than minareaextension are eliminated during cleanup.

For information regarding minareaextension interactions with extension45, see “rectedge”.
