newtag boolean

Tag Keywords

Specifies a tag name representing boolean combinations of layers or edges. Used by RET SBAR and RET MBSRAF.

Usage

newtag name tagname1 boolean tagname2 [fulledge]

newtag name boolean tagname1 tagname2 [tagnameN...] [fulledge]

Arguments

name

Specifies the name of the tag.

tagname0, tagname1, tagname2

Previously derived tags generated by taglayer or newtag commands.

boolean

Specifies a boolean operator used to derive tags:

  • AND — Outputs overlapping portions of the input tags.

  • NOT — Outputs portions of the first input tag not covered by the second tag. If NOT is used in unary form, the output consists of portions of the target layer not covered by the second tag.

  • OR — Outputs a combination of both input tags.

  • XOR — Outputs mutually exclusive portions of the input tags.

fulledge

An optional keyword forcing the entire length of a tagged edge to be used even when a shorter portion of the edge is originally tagged. This option is compatible with all constraint types.

Description

Selects edges processed by boolean operations to combine the results of other tags. The first usage syntax is for operations between two tags. The next usage defines boolean operations for two or more tags. All boolean operations are performed on sets of partial edges defined by previously defined tags, and select coinciding edges or merge the sets of edges.

Only manhattan- and 45-degree input layer geometries are supported. No skew-edged geometries are valid target shapes for SRAFs.

CAUTION:

Using constraint values that exceed cell or tile boundaries may cause flattening and increase run times. Siemens EDA recommends using the recta and rectb newtag types on relatively shorter geometries such as contacts, vias and local interconnect.

Examples

Example 1 — Newtag Boolean Usage

taglayer M1 
newtag sq square <= 0.05 
newtag sh length < 0.010 
newtag small_square sq AND sh 
newtag long NOT sh 
newtag or2  t1 OR t2 
newtag or4  OR t1 t2 t3 t4
newtag not1 NOT t1
newtag xor3 XOR t1 t2 t3 
newtag fuzzy (t1 AND t2) OR (t3 XOR (t4 AND t5))
