newtag square

Tag Keywords

Specifies a tag name representing the edges of square target shapes matching the constraint. Used by RET SBAR and RET MBSRAF.

Usage

newtag name square constraint

Arguments

name

Specifies the name of the tag.

square constraint

A keyword that specifies the constraint. For edge-based templates, the constraint may be a range.

Note:

When applying the square keyword to any pitch-based template, you must provide an upper bound to the constraint to enable limits during edge collection.

Description

The square tag selects all edges of square target shapes matching the specified edge length constraint.

CAUTION:

Using constraint values that exceed cell or tile boundaries may cause flattening and increase run times. Siemens EDA recommends using the recta and rectb newtag types on relatively shorter geometries such as contacts, vias and local interconnect.

Examples

Example 1 — recta and rectb Usage

newtag sq square == 0.2
newtag smallSquares square >= 0.090 < 0.120
