minsquarelength

Constraint Keywords

Specifies the minimum edge length for square SRAFs. Used by RET SBAR and RET MBSRAF.

Usage

minsquarelength value

Arguments

value

A positive floating point number specified in microns.

Description

An optional keyword that specifies the minimum edge length allowed for any square SRAF. Square SRAFs with an edge length smaller than minsquarelength are deleted. The minsquarelength keyword does not apply to non-square SRAFs. Although minarea applies to all SRAFs, if minsquarearea or minsquarelength are specified, all square SRAFs are checked against these two parameters but not minarea.

You can specify minsquarelength globally or by priority.
