cnsrafpromotion

Cleanup Keywords

Disables default promotion for cnsraf. Used by RET SBAR only.

Usage

cnsrafpromotion {0 | 1}

Arguments

0 | 1

Disables or enables default promotion, respectively.

Description

An optional keyword that disables hierarchical promotion for cnsraf. Disabling promotion changes SRAF generation results. This keyword adopts the standard Calibre promotion scheme. A performance impact may occur depending on your design hierarchy. This keyword can only be set globally and is enabled by default.

Note:

Prior to Calibre release 2016.2, cnsrafpromotion was disabled by default.
