jogignore

Style Keywords

Overrides jogs in generated SRAFs. Used by RET MBSRAF only.

Usage

jogignore value

Arguments

value

An optional argument specified as a positive, floating-point number in microns.

Description

An optional keyword that enables mbsraf to override jogs in generated SRAFs that are less than the specified value. Because algorithm 1 typically generates long, unbroken SRAFs when jogs are present, it does not require jogignore.

Note:

Siemens EDA recommends always using jogignore with mode collectno45 to achieve expected results.
