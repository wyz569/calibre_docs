setlayer sraf_options Keywords

Specify keywords within the sraf_options statement to control setlayer cnsraf and setlayer mbsraf commands.

The groups shown in Table 1 are comprised of keywords specified within the sraf_options statement residing in the LITHO FILE statement. Some keywords are used by both setlayer cnsraf and setlayer mbsraf, some are exclusive to either. The keyword short descriptions state their respective application.

Table 1. setlayer sraf_options Statement Keyword Groups

Group

Description
Layer KeywordsCalibre nmSRAF requires input layers for SRAF generation.
Tag KeywordsTags direct templates where to generate SRAFs. The newtag family of keywords select edges and refine locations of SRAF placement.
Constraint KeywordsCalibre nmSRAF requires the specification of some minimum SRAF dimensions for application of MRC rules.
TemplatesCalibre nmSRAF requires dimensional- or model-based templates to pattern and prioritize SRAF generation.
Style KeywordsProvides management of SRAF output style.
Conflict KeywordsProvides commands for managing conflicts formed during SRAF generation.
Cleanup KeywordsCalibre nmSRAF invokes SRAF cleanup after generation.
Priority KeywordsCalibre nmSRAF provides many keywords to be called by priority.
