centerwidth

Style Keywords

Enables the generation of SRAFs equidistant from target edges. Used by RET MBSRAF.

Usage

centerwidth value

Arguments

value

A positive, floating-point number specified in microns. The value you specify must be greater than the width parameter value.

Description

An optional keyword which generates SRAFs equidistant from input target shapes. The centerwidth keyword may be specified globally or in the localmaxgradient template. The centerwidth value must be greater than or equal to width. All localmaxgradient templates of the same priority must specify the same centerwidth value.
