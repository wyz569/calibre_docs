minlength

Constraint Keywords

Specifies the minimum length allowed for a rectangular SRAF. Used by RET SBAR and RET MBSRAF.

Usage

minlength value

Arguments

value

A positive floating point number specified in microns.

Description

An optional keyword that specifies the minimum length allowed for a rectangular SRAF. SRAFs with any side of a length smaller than minlength are eliminated after cleanup is performed. You can specify minlength globally or by priority.

For information regarding interactions between minwidth, minarea and minlength, see “minarea, minlength, and minwidth Keyword Interaction”.
