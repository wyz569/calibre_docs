sraf

Conflict Keywords

Enables and specifies the size of SRAF output in conflict regions. Used by RET SBAR only.

Usage

sraf value

Arguments

value

A positive floating point number. The minimum value is two database units.

Description

An optional keyword that enables secondary SRAF output in conflict regions. The sraf keyword also specifies the square size of the SRAFs added to the conflict region if the length of the region edges is less than or equal to maxconflict and less than centerthresh.

The SRAF is output at the size specified by the sraf keyword up to a maximum of the conflict region extent. If the sraf keyword is not specified, no SRAFs are generated in the conflict region. The minimum sraf keyword value specified must be two database units (DBU).

By default, during data promotion where conflict regions are split between tiles, conflict region‑placed SRAFs may be generated less than the sraf value when trimmed to a smaller conflict region boundary. Set oversizesraf to yes to override this behavior.

The sraf keyword can be set globally or by priority for any setlayer cnsraf template.
