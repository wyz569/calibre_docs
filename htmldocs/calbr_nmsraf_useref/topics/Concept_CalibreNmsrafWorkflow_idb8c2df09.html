Calibre nmSRAF Workflow

The Calibre nmSRAF workflow enables successful generation of SRAFs.

  1. Begin with an isolated target shape. If the target layer contains a variety of shapes, develop initial templates for each shape independently.

  2. Run Calibre pxOPC on each isolated shape.

  3. Create the template(s) for duplicating the Calibre pxOPC shapes of the inner ring (those SRAFs closer to the target). The templates can be created manually and optimized using the SVRF Optimizer tool.

  4. Next, create the templates for duplicating the Calibre pxOPC shapes of the next outer SRAF ring.

  5. Run Calibre pxOPC on other training layouts (pitch patterns, real layout clips, targeted test patterns) and add templates as needed. Alternatively, run Calibre pxOPC to improve process metrics such as PV band.

    A PV band polygon shows how edge locations respond to process variations. It represents the area within which a feature prints as the process conditions vary. Generating PV bands involves simulating the printed image at various process conditions and combining the resulting printed images. PV band widths alter due to process variation.
