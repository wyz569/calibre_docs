Calibre nmSRAF Requirements

Calibre nmSRAF requires executable paths, licenses and variables to be set prior to running the tools.

Before invoking the Calibre hierarchical engine, you must set your environment variables. Calibre tools require the CALIBRE_HOME environment variable to be set. See “Setting the CALIBRE_HOME Environment Variable” in the Calibre Administrator’s Guide for details.

If you intend to implement Calibre nmSRAF, you should be familiar with Calibre® nmDRC™, Calibre nmOPC, Calibre® RVE™, and Calibre® OPCverify™ procedures. An understanding of programming SVRF and TVF rule files is highly useful. To use Calibre nmSRAF, you must have the following items:

  • Calibre version 2014.1 or higher

  • The Calibre nmSRAF license. For more information on licensing, refer to the Calibre Administrator’s Guide.

  • An SVRF file

    This rule file contains specification and operation statements with verification rules that are applied to your designs.

  • A LITHO FILE

    The LITHO FILE statement contains commands used to configure Calibre nmSRAF for simulation. The LITHO FILE statement can be a separate file or it can be written within an SVRF file.

  • A GDS or OASIS® layout

  • Model information

    • Optical model

    • Resist model

For help executing Calibre nmSRAF, refer to “Running Calibre nmSRAF” in Appendix A.
