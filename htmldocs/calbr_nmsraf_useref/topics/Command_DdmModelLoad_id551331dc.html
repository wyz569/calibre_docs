ddm_model_load

LITHO FILE Statement and Commands

Loads DDM models. Used by RET MBSRAF only.

Usage

ddm_model_load name {filename | ‘{’inline_modelname‘}’}

Description

Loads the Domain Decomposition Method (DDM) for Hybrid Hopkins Abbe (HHA) models contained in the filename, or defined by inline model text within braces ({ }). The DDM model is assigned the user-defined name, which you may later reference in the image command. Calibre mbSRAF supports crosstalk and corner chopping with DDM models.

The ddm keyword specified in the sraf_options statement calls the declared symbolic model name.

A DDM or HHA model is optional and only required if you use the image command with the ddm modifier. This command is ignored if a litho model is present; the DDM model file is set inside the Lithomodel file instead.

Note:

Siemens EDA recommends using the litho model format for all model-based SRAF recipes. This non-litho model command is not supported and is obsoleted as of release 2019.1.

Note:

For more information on DDM and HHA models, see the Calibre WORKbench User’s and Reference Manual.

Arguments

name

An argument specifying the symbolic name of the DDM model referred to by subsequent DDM-related commands.

filename | ‘{’ inline_modelname‘}’

A required argument that specifies either a filename containing the DDM model or an inline file defining the DDM model. The inline DDM model definition must be enclosed in braces ({ }). One of the arguments must be specified.

Examples

LITHO FILE mbsraf_setup [
	tilemicrons 25
	modelpath ./models
	optical_model_load optical_ddm optical_ddm
	resist_model_load  RESIST cm1.mod
	ddm_model_load DDM_model ddm_model_named.ddm
 
	background -0.0259 0.0095
	layer target visible -0.9741 -0.0095
	layer dummy_opc visible -0.9741 -0.0095
 
	sraf_options opts {
		algorithm 1
		correctionLayer dummy_opc
		optical optical_ddm resist RESIST 
		ddm DDM_model
		peaktype 3Dir
		...
	} 
	setlayer mb.sraf = mbsraf target OPTIONS opts 
] 
 
mb_sraf = RET MBSRAF target dummy_opc MAP mb.sraf FILE mbsraf_setup
