rectdiag

Templates

Places four or eight rectangular SRAFs diagonally offset relative to the center of the target shape. Used by RET SBAR only.

Usage

rectdiag
[tag name ...]
{length value width value}
{pitchx value pitchy value}
[breaksym]
[conflictonly]
[orient {horizontal | vertical}]
[prior value]

Arguments

tag name ...

An optional keyword specifying one or more tag names used in the template. The tag name identifies a group of edges or shapes to be used with the template. If specified, tags must appear as the first arguments in the template.

length value width value

Two required keywords, length and width, that specify the longer and shorter edges of a rectangular SRAF, respectively. The length and width values can be identical.

pitchx value pitchy value

A required keyword pair that specifies the horizontal and vertical distance from the center of the target shape to the center of the SRAF, respectively.

breaksym

An optional keyword that prevents the template from generating symmetric target shapes. By default, the following rectdiag generates symmetric SRAFs:

rectdiag width 0.030 length 0.100 pitchx X pitchy Y

but also generates SRAFs at these symmetric locations (where X and Y are reversed):

rectdiag width 0.030 length 0.100 pitchx Y pitchy X

To prevent this symmetric generation, use breaksym (which also works for reversed X and Y pitches): 

rectdiag width 0.030 length 0.100 pitchx X pitchy Y breaksym
conflictonly

An optional keyword that outputs SRAFs within conflict regions only; no other SRAFs of this template are generated. Conflict regions are formed where neighboring SRAFs generated by this template violate MRC rules you specify in the template. The conflictonly argument and dynamicsizing keyword cannot be used within the same priority. Siemens EDA recommends using the conflictonly argument only when conflict resolution shapes are relatively small and not for larger shapes such as line layer designs.

orient horizontal | vertical

An optional keyword that enables the rectdiag template to place SRAFs oriented to the aspect of rectangular target shapes. Specifying orient horizontal applies templates to rectangular target shapes as if they were oriented horizontally (the long edge of the rectangle is parallel to the x-axis). Specifying orient vertical applies templates to rectangular target shapes as if they were oriented vertically (the long edge of the rectangle is parallel to the y‑axis).

Note:

If a template is symmetric or symmetry is not broken by the breaksym keyword, then there is no need to specify the orientation. For additional details regarding the usage of the orient keyword, see Figure 6.

prior value

An optional keyword that specifies a numeric identifier for placed SRAFs. Identifiers are applied sequentially, so a value of 1 is applied first, then 2 is applied, and so on. The default priority is 1. See “priority” for more information.

Description

An optional template and argument set that symmetrically places four or eight rectangular SRAFs, depending on pitch value, located diagonally from the center of the target shape. This template cannot generate negative SRAFs.

Example 3

Setting identical pitchx and pitchy values results in SRAF conflicts. Subsequent square SRAFs can be output in those conflict regions by the specification of the sraf keyword. Generating four SRAFs with rectdiag requires either identical pitches in conjunction with the sraf keyword for conflict cleanup, or the use of the rectdiag breaksym option. Resulting SRAFs differ between these two approaches. These differences are shown in Figure 3 and Figure 5, respectively.

Example 6

As depicted in Figure 6, the orient horizontal option places SRAFs perpendicular relative to the aspect ratio of the target shape, and orient vertical places SRAFs parallel relative to the aspect ratio of the target shape. These relative SRAF orientations remain consistent regardless of the orientation of the target shapes. If the target shape is square, orient places SRAFs either horizontally or vertically relative to the target shape.
