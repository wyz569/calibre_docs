Calibre pxOPC Generation

The Calibre pxOPC SRAF results can facilitate the generation of Calibre nmSRAF templates. In simple cases, this is accomplished by matching Calibre pxOPC SRAFs with Calibre nmSRAF templates.

Start by generating Calibre pxOPC results for an isolated contact, identifying the SRAF configurations, and defining Calibre nmSRAF templates that generate matching SRAFs. As the process of defining templates continues, Calibre pxOPC can be used to capture the best SRAF placement.

3D Mask models are supported by Calibre pxOPC for Calibre nmSRAF template generation. To aid you with Calibre pxOPC setup, an application note is available on Support Center, entitled “Using pxOPC for generating reference SRAFs in the model-based nmSRAF Template Generation Flow when using 3D Mask Models” (application note number MG587480).

For more information on using Calibre pxOPC, see the Calibre pxOPC User’s and Reference Manual.
