shielding

Constraint Keywords

Uses edge tag shielding to characterize SRAF generation. Used by RET MBSRAF only.

Usage

shielding {distance | no | seedpixel}

Arguments

distance

This option only generates SRAFs in regions closer to tagged edges or corners than to other target edges.

no

This option turns shielding of tagged edges off. This reverts behavior back to Calibre release 2015.4.

seedpixel

This option determines whether a tagged edge or corner is visible by projection from any seed pixel. It ignores other target edges and corners that are not tagged, even if they are closer to the seed pixel.

Description

An optional keyword that specifies shielding of tagged edges to alter SRAF generation relative only to the tagged edges. Shielding may be specified globally or by priority. Shielding is compatible with positive and negative SRAF generation.
