reducedbarsplit

Cleanup Keywords

Splits or merges SRAFs in conflict. Used by RET SBAR only.

Usage

reducedbarsplit mode

Arguments

mode

Four different modes can be specified:

  • 0 — Processes SRAF candidates perpendicular to one another. Mode 0 splits the top and prioritizes the stem of the T formed between perpendicular SRAF candidates. Siemens EDA recommends setting a value of 0.

  • 1 — Processes SRAF candidates perpendicular to one another. Mode 1 splits the stem and prioritizes the top of the T formed between perpendicular SRAF candidates. This is the default value.

  • 2 — Processes all non-parallel SRAFs. Mode 2 splits the top and prioritizes the stem of the T formed between non-parallel SRAF candidates.

  • 3 — Disables bar-splitting of SRAFs.

Description

The reducedbarsplit keyword removes conflicts between perpendicular SRAFs. Bar splitting is the only cleanup process that breaks existing SRAF candidates into multiple segments. The reducedbarsplit keyword controls which SRAFs are selected for bar splitting.
