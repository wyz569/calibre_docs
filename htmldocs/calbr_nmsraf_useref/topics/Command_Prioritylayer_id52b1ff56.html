prioritylayer

Layer Keywords

Specifies shapes or edges prioritized as targets during SRAF generation. Used by RET MBSRAF only.

Usage

prioritylayer name [prioritylayer name...]

Arguments

name

The name of a shape or edge.

Description

An optional keyword and layer name specifying original or derived polygons or edges that take priority during SRAF generation. Where possible, SRAFs are placed first near shapes in the specified prioritylayer within the mbsraf minfeaturespace distance from those shapes. You may specify multiple prioritylayer keywords.
