Calibre cnSRAF and Calibre OPCsbar Integration

Calibre OPCsbar and Calibre cnSRAF can be used together. You can use output from Calibre OPCsbar as input to Calibre cnSRAF or conversely, Calibre cnSRAF as input to Calibre OPCsbar.

To pass the output from Calibre OPCsbar into Calibre cnSRAF, you use the layer command to import SRAF data into Calibre cnSRAF. To pass the output from Calibre cnSRAF into Calibre OPCsbar, you use the SBLAYER command to import the SRAF data into Calibre OPCsbar. Refer to the OPCSBAR command in the Calibre OPCsbar User’s and Reference Manual and setlayer cnsraf in this manual for more information.

The following example uses Calibre OPCsbar first and then passes its output into Calibre cnSRAF. This example only shows the relevant setup instructions and not the entire recipe. Refer to the Calibre OPCsbar User’s and Reference Manual for more information.

// Create SRAFs using OPCSBAR.
// The output (assist_sbar) is used in setlayer cnsraf.
assist_sbar = OPCSBAR target 
 
assist_CN = RET SBAR V1_target assist_sbar FILE cn_setup MAP assist_CN  
 
LITHO FILE cn_setup [ 
 
	layer V1_target 
	layer assist_sbar 
 
	// Use output of OPCSBAR (assist_sbar) as sblayer input to cnsraf. 
	setlayer assist_CN = cnsraf V1_target sblayer assist_sbar newtag x1 ...
 
	// output layers
	setlayer 1_field     = cnsraf debug assist_CN field        1 
	setlayer 1_hard      = cnsraf debug assist_CN hardconflict 1 
	setlayer 1_conflict  = cnsraf debug assist_CN conflict     1 
	setlayer 1_iso       = cnsraf debug assist_CN isolate      1 
	setlayer 1_processed = cnsraf debug assist_CN resolution   1 
 
]
