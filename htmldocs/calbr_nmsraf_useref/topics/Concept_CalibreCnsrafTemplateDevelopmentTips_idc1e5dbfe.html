Calibre cnSRAF Template Development Tips

Careful setup during cnsraf template development insures optimal SRAF generation and best image fidelity.

Template Indexing

Using an indexing method with the priority keyword in conjunction with the prior keyword of any template facilitates the addition of templates during the development of your command file. For example, when priorities numbered in tens are first assigned you permit more development flexibility by retaining nine additional spots to add subsequent templates without modifying past priority values. Substituting recthv and rectdiag for srafringhv

The following code snippets represent the coding efficiency of using srafringhv in place of both recthv and rectdiag. As the accompanying illustrations show, results are nearly identical.

Figure 2 uses recthv and rectdiag only. The code colors represent the SRAF generated by that specific line of code shown in Figure 3.

Figure 2. SRAF Generation With recthv and rectdiag
setlayer sraf_out = \
	cnsraf sraf_target newtag e1 recta <= 0.076 rectb < 0.1 \
	recthv tag e1 pitchx 0.130 pitchy 0.130 \
		width 0.030 length 0.116 prior 1 \
	rectdiag tag e1 pitchx 0.090 pitchy 0.090 \
		width 0.035 length 0.035 prior 2 \
	recthv tag e1 pitchx 0.210 pitchy 0.210 \
		width 0.035 length 0.216 prior 3 \
	rectdiag tag e1 pitchx 0.190 pitchy 0.190 \
		width 0.035 length 0.035 prior 4 \
Figure 3. SRAF Results With recthv and rectdiag

Figure 4 uses srafringhv alone. The code colors match the SRAF colors generated by the respective lines of code, as shown in Figure 5.

Figure 4. SRAF Generation With srafringhv
setlayer ass_CN_0 = 
	cnsraf sraf_target newtag e1 recta <= 0.076 rectb < 0.1 \
	srafringhv tag e1 width 0.035 pitch 0.130 srafcount 1 \
		rectlength 0.116 rectwidth 0.030 prior 1 \
	srafringhv tag e1 width 0.040 pitch 0.210 srafcount 1 \
		rectlength 0.216 rectwidth 0.035 prior 2 \
Figure 5. SRAF Results Using srafringhv Alone

Use RET Flow Tool for cnsraf Template Development

The RET Flow Tool greatly improves efficiency and ease-of-use for debugging and optimizing cnsraf templates. The RET Flow Tool interface has several parts: Textual extension and layer mapping window, Options forms with debug layer selection, and Calibre WORKbench layout editor and viewer.

You access the RET Flow Tool for applications such as OPC and SRAF generation tools from Calibre WORKbench.

For detailed information regarding the use of Calibre nmSRAF with RET Flow Tool, see “RET Flow Tool and Calibre nmSRAF”.
