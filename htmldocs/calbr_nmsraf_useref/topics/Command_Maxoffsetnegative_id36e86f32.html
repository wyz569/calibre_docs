maxoffsetnegative

Style Keywords

Specifies the maximum distance from a negative SRAF to the enclosing target shape edge. Used by RET MBSRAF only.

Usage

maxoffsetnegative value

Arguments

value

A positive floating point number specified in microns.

Description

An optional keyword that specifies the maximum distance allowed to generate negative SRAFs (slots) within a target shape. The maxoffsetnegative keyword is used only with the negative keyword. The maxoffsetnegative keyword may be specified only once in a LITHO FILE, for either priority or global assignment.
