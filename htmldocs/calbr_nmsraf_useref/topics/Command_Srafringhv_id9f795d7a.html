srafringhv

Templates

Places rectangular and square SRAFs radially around a target shape. Used by RET SBAR only.

Usage

srafringhv
[tag name ...]
pitch value {rectlength value rectwidth value}
srafcount value
width value
[prior value]

Arguments

tag name ...

An optional keyword specifying one or more tag names used in the template. The tag name identifies a group of edges or shapes to be used with the template. If specified, tags must appear as the first arguments in the template.

pitch value

A required keyword that specifies the distance from the center of the target shape to the center of the rectangular SRAF.

rectlength value rectwidth value

A required keyword pair that specifies the length and width, respectively, of rectangular SRAFs.

srafcount value

A required keyword that specifies the number of square SRAFs placed between rectangular SRAFs.

width value

A required keyword that specifies the width of the square SRAFs placed between rectangular SRAFs.

prior value

An optional keyword that specifies a numeric identifier for placed SRAFs. Identifiers are applied sequentially, so a value of 1 is applied first, then 2 is applied, and so on. The default priority is 1. See “priority” for more information.

Description

An optional template and argument set that places rectangular and square SRAFs radially around a target shape. The center of the SRAFs are placed the specified distance (pitch) from the center of the target shape. This template cannot generate negative SRAFs.
