taglayer

Layer Keywords

Specifies a tag name representing imported layers or edges from SVRF. Used by RET SBAR and RET MBSRAF.

Usage

taglayer name [taglayer name...]

Arguments

name

Specifies the name of the taglayer.

Description

An optional keyword used to import polygons or edges into templates from SVRF, derived as needed and named for subsequent tag use. With specific exceptions, all cnsraf templates may reference taglayers by name. Multiple taglayers can be referenced in a single template. The input taglayer is internally ANDed with the target layer, and the resulting edges are then used as the operational taglayer. The taglayer must not also be used as an offsetlayer.

Only manhattan- and 45-degree input layer geometries are supported. No skew-edged geometries are valid target shapes for SRAFs.

Examples

This example demonstrates one possible layer flow using the taglayer keyword.

LAYER target 0 
x1 = EXPAND EDGE by 0.001 (EXT [target] > 0.1 < 0.5 opposite) 
 
LITHO FILE cn_setup_1 [
	layer target
	layer x1
	...
	sraf_options template {
		cleanupversion 1
		taglayer x1
		rectEdge tag x1 extension 0.01 width 0.03 offset 0.04 prior 1  
	}
	setlayer cnsb1 = cnsraf target OPTIONS template 
] 
sraf = RET SBAR target x1 FILE cn_setup_1 MAP cnsb1
