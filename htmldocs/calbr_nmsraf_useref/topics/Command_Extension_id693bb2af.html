extension

Constraint Keywords

Specifies SRAF extension length. Used by RET MBSRAF only.

Usage

extension value

Arguments

value

A positive or negative floating point number specified in microns.

Description

An optional keyword that adds to or subtracts from the length of a rectangular SRAF end by the specified value based on the neighboring target edge. The extension keyword works only with the localmaxgradient modes nearest and nearestno45. A negative extension reduces SRAF length if the extension is less than half of the target edge length and the resulting SRAF does not violate MRC constraints. The extension keyword may be specified globally or by priority.
