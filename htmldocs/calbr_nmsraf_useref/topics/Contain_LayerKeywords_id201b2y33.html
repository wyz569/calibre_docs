Layer Keywords

Calibre nmSRAF requires input layers for SRAF generation.

For organizational keyword information, refer to “sraf_options”.

Table 1. Layer Keywords

Keyword

Description
correctionlayerSpecifies a polygon layer for gradient calculation. Used by MBSRAF only.
offsetlayerSpecifies a layer name representing areas within which to prohibit positive and negative SRAF generation. Used by RET SBAR and RET MBSRAF.
prioritylayerSpecifies shapes or edges prioritized as targets during SRAF generation. Used by RET MBSRAF only.
sblayerSpecifies a layer name containing preexisting SRAFs. Used by RET SBAR and RET MBSRAF.
taglayerSpecifies a tag name representing imported layers or edges from SVRF. Used by RET SBAR and RET MBSRAF.
