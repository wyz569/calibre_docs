RET Flow Tool and Calibre nmSRAF

The RET Flow Tool is a productive GUI-based tool used with Calibre nmSRAF.
Table 1. Usage of the Text nmOPC RET Flow Tool

Step

Description
Invoking the RET Flow ToolThe RET Flow Tool provides a convenient, productive method for executing a number of GUI-based applications with Calibre nmSRAF.
Importing an Existing OPC Session to RFTYou import the Calibre nmOPC session into the RET Flow Tool.
Extracting an mbsraf Session From an SVRF FileYou extract the Calibre nmSRAF portions of your SVRF file, and import it into the RET Flow Tool. This session is added to the existing nmOPC session already in place.
Setting RFT Application OptionsYou can set a variety of options within the RET Flow Tool for analysis.
Performing Gradient MappingYou can display gradient maps superimposed on your layout using Calibre WORKbench.
Drawing Intensity CutlinesYou can draw cutlines and display resulting aerial intensity plots using Calibre WORKbench.
