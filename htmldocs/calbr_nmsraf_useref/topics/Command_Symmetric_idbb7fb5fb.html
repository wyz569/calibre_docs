symmetric

Style Keywords

Ensures all templates generate SRAFs with symmetry. Used by RET SBAR only.

Usage

symmetric

Arguments

None.

Description

An optional keyword used to ensure all templates generate SRAFs symmetrically. If any template does not specify identical pitch, SRAF width and length values, or offsetx and offsety values, a parse error results.

Examples

These code portions show both non-identical offsetx and offsety, and width and length values. When the symmetric keyword is specified, and the recipe has asymmetrical SRAF output specified, the run fails with a parsing error. The keywords in red constitute the parse failures when symmetric is specified.

symmetric 
rectedge 
	width 0.050 length 0.53
	offsetx 0.2 offsety 0.1
