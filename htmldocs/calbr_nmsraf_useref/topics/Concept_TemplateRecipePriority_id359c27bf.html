Template Recipe Priority

Calibre nmSRAF relies on proper ordering of templates within SRAF generation recipes.

Templates should be prioritized from regions closer to the target shape to regions farther from the target shape, having higher priorities through lower priorities, respectively.

Region Definitions

Isolated features are those placed at a distance greater from other features than any considered optical proximity where feature resolution would suffer through the optical interference of a neighboring shape. Dense features are those which interact with multiple features within their optical proximity.

Dense Regions

Dense SRAF regions are defined by spatial frequencies between features where SRAF directly overlaps the SRAFs of a neighboring shape. This is the starting point for determining the range of a conflict region for a template when the conflict resolution keywords are used. See “Calibre cnSRAF Conflict Regions” for more information.

Set the value of the maxconflict and centerthresh parameters for this template to be large enough to allow conflict resolution to occur for a wide range of conflicts:

maxconflict = centerthresh

Where:

centerthresh <= SRAF_length

Alternatively as:

centerthresh <= SRAF_width*2+MRC

Use Calibre pxOPC to determine the valid conflict resolution range for the template by adjusting the target shape separation and checking that the primary feature remains covered by the Calibre nmSRAF output.

The target shape separation where conflict regions exist and generated SRAFs are no longer valid indicates that a new template is required. The point at which the first template is no longer valid defines the maxconflict parameter value (the point at which the SRAFs are removed).

Use Calibre pxOPC for horizontal, vertical, and off-axis features. Determine the centerthresh values for each template. Begin moving target shape positions laterally and determine applicable ranges of replacement templates.

Without Calibre pxOPC, you review the best PV band (or other chosen metric) for the conflict regions of interest.

Isolated Regions

Because no optically competing shapes cause optical interference with isolated shapes, isolated shapes provide solitary test vehicles for generic SRAF generation. They should remain as a lower priority during SRAF generation.
