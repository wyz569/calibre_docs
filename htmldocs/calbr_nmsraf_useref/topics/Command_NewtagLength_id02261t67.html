newtag length

Tag Keywords

Specifies a tag name representing all target edges qualified by the length specified. Used for RET SBAR and RET MBSRAF.

Usage

newtag name [predefined_tag] {length constraint} [fulledge]

Arguments

name

Specifies the name of the tag.

predefined_tag

Previously derived tags generated by taglayer or newtag commands.

length constraint

A constraint applied to the edges of target shapes and existing tags used for classifying edge length and projecting edge run length. When applied to target shapes, length defaults to inputting the entire target layer. The length type can also accept up to one existing tag. Use the opposite metric with length to find the projecting edge pair run length. To measure lengths of existing tags, specify the existing tag immediately preceding the length type.

Note:

When applying the length keyword to any pitch-based template, you must provide an upper bound to the constraint to enable limits during edge collection.

fulledge

An optional keyword forcing the entire length of a tagged edge to be used even when a shorter portion of the edge is originally tagged. This option is compatible with all constraint types.

Note:

Partial edge tagging with Calibre mbSRAF may lead to unexpected results because of differences between intended SRAF and MRC enforcement. Siemens EDA recommends supplying the entire edge to mbsraf when using tagging.

Description

An optional keyword used to input tags by name to Calibre nmSRAF templates. Multiple tags can be referenced in a single template.

Only manhattan- and 45-degree input layer geometries are supported. No skew-edged geometries are valid target shapes for SRAFs.

When using tagging, positive SRAFs may be generated parallel and perpendicular (given template constraints) relative to tagged edges and negative SRAFs are generated only parallel to tagged edges.

CAUTION:

Using constraint values that exceed cell or tile boundaries may cause flattening and increase run times. Siemens EDA recommends using the recta and rectb newtag types on relatively shorter geometries such as contacts, vias and local interconnect.

Examples

Example 1 — Newtag length Usage

newtag medEdge      length >  0.050 <= 0.065 
newtag longEdge     length >  0.065

Example 2 — Newtag length Used with Boolean Keywords

taglayer M1 
newtag sq square <= 0.05 
newtag sh length < 0.010 
newtag small_square sq AND sh 
newtag long NOT sh

Example 3 — Specifying a Portion of an Existing Tag with length

taglayer M1_tag 
newtag long_tag M1_tag length > 0.10

Example 4 — Compounding length Tags

newTag tag1 l1 length > 0.10
newTag tag2 l2 length < 0.05
newTag tag3 external <0.13
newTag final tag3 and (tag1 or tag2)
