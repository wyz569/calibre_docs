collecttype

Style Keywords

Collects seed pixels near target corners to generate square SRAFs. Used by RET MBSRAF only.

Usage

collecttype {0 | 13}

Arguments

0

No seed pixels are collected at corners. This is the default setting for standard SRAF generation.

13

Seed pixels are collected at target corners for square SRAF generation.

Description

An optional keyword, enabled by mode collectno45, which performs seed pixel collection for SRAF generation at corners.

Note:

The collecttype keyword is deprecated as of Calibre release 2018.4 and will be obsoleted as of release 2019.2.

If a seed pixel is found near a target corner (not a target edge), an arc of seed pixels is collected around that corner and axis-aligned SRAFs are generated for the entire arc. The collection process for collecttype 13 has several steps demonstrated in Figure 1:

  1. A 45 degree angle from a target corner is chosen as a reference for the initial SRAF placement.

  2. Within this quarter ring, the minimum and maximum angle of all pixels with a local maximum facing the corner are determined.

  3. The angular range of the arc (Ra) is then covered by SRAFs maintaining the established minoffset spacing from each other, and the integral of the gradient over each SRAF is computed.
