negative

Style Keywords

Enables negative SRAF output. Used by RET SBAR and RET MBSRAF.

Usage

negative

Arguments

None.

Description

An optional keyword that specifies the generation of negative SRAFs, also referred to as slots in target shapes. The negative keyword cannot be used with pitch-based templates.
