image_options

LITHO FILE Statement and Commands and Style Keywords

A LITHO FILE command referencing the Lithomodel file and an sraf_options command referencing the litho_model name. Used by RET MBSRAF only.

Usage

Usage within the LITHO FILE statement:

image_options name ‘{’
layer name visible [mask_layer number] ...
litho_model litho_model_dir ...
[layer name filter]
‘}’

Usage within the sraf_options statement:

image_options name

Arguments

name

Declares a required, unique user-defined name for the image_options statement. This name is also used the image_options statement from within the sraf_options statement or setlayer mbsraf.

layer name visible [mask_layer number]

Specifies a visible layer. At least one visible layer is required. The mask_layer parameter maps this layer to a corresponding mask_layer template inside the Lithomodel file. The mask_layer number defaults to 0 if not specified (in the case of single template litho models, for example). For multi-patterning masks, specify a mask_layer argument to additional layer commands.

litho_model litho_model_dir

A required argument that specifies a directory name. This directory must contain optical and resist models, a Lithomodel file, and optionally contain an etch model. For more detail see “Lithomodel Format Usage for Calibre mbSRAF Execution”.

This is an abbreviated description of the image_options command. For detailed information, consult the Calibre OPCverify User’s and Reference Manual.

layer name filter

An optional filter layer limiting contour generation to the area of the layer. Only one filter layer may be specified.

Description

Specifies a statement name containing a reference to the Lithomodel file and subsequently called by the setlayer mbsraf command. The image_options statement within sraf_options must be specified prior to any pw_condition keyword.

Examples

Example 1

The keyword is first defined within the LITHO FILE statement and then called within the sraf_options statement:

LITHO FILE M1_model [
	image_options MBS { 
		litho_model 20nm_lithomodel
		layer target visible
	}
	...
	sraf_options mb_OPT {
		algorithm 1
		correctionlayer correction_layer
		image_options MBS
		...
		localmaxgradient width 0.026 prior 1
		pw_condition f40 focus -40nm dose 1.05 size -0.00025 weight 40
	}
	setlayer mbsb1 = mbsraf target OPTIONS mb_OPT
]
