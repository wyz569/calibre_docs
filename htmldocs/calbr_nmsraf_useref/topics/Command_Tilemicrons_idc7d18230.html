tilemicrons

LITHO FILE Statement and Commands

Specifies the size of a processing tile. Used by RET SBAR and RET MBSRAF.

Usage

tilemicrons value [adjust | exact]

Arguments

value

A number specifying the tile size in microns. The default value is 100 um.

adjust

An option adjusting the tilemicrons setting slightly to make it closer to a full multiple of simulation frames if possible. A simulation frame is a fast Fourier transform (FFT) frame, which is roughly six to ten microns in size. This is the default setting.

exact

An option using the value as specified.

Description

Specifies the size of a processing tile. Large tiles are broken into tiles of the value specified.

Examples

tilemicrons 35

Recommended Settings for tilemicrons per Node

The selection of tile size is critical for optimum processing. If the tile size is too large or too small, primary or remote memory consumption may go into swap and slow performance. The empirical, optimum tile size settings correlate to the technology node and balance the size of data per tile. Siemens EDA recommends that you set tilemicrons to the following values based on the node listed in Table 1. For double patterning, use the smaller values listed in the table.
