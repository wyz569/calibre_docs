centerslit

Style Keywords

Disables slit interpolation. Used by RET EUV MBSRAF only.

Usage

centerslit value

Arguments

0 | 1

Either 0 or 1, where 0 enables and 1 disables slit interpolation, respectively.

Description

The centerslit keyword disables slit interpolation for an EUV global litho model. When centerslit is set to 1, it allows only the center slit to be used during the gradient calculation. The default setting is 0, enabling slit interpolation and through-slit shadowing effects. The centerslit parameter allows use of a litho model that does not use multiple slits, and does not require preparation of a special litho model for Calibre mbSRAF.
