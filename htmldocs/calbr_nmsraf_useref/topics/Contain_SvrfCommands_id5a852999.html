SVRF Commands

Calibre nmSRAF commands adhere to SVRF standards. A typical SVRF rule file containing cnsraf-related commands may also contain other SVRF commands that specify input layers, perform preliminary layer manipulation, and save output layers.

These commands are documented in the Standard Verification Rule Format (SVRF) Manual.

Table 1. SVRF Commands

Command

Description
RET [EUV] MBSRAFInvokes Calibre mbSRAF.
RET SBARInvokes Calibre cnSRAF.
