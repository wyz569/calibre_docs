Calibre nmSRAF Recipe LINT Messages

Understanding informational messages and correcting warning messages aid in improving results generated by Calibre nmSRAF.

During compilation and validation of the setlayer input, a series of LINT messages are generated. These include warnings. Some layout-dependent warnings can only be found at runtime and not during compile-time. In addition to warnings, a brief overview of cleanup parameters is provided. Some messages inform the user of specified or default parameters that are in compliance with Siemens EDA recommended settings. If a parameter deviates from Siemens EDA-recommended settings, the specified value is reported as a warning.

Note:

Some specifications listed in Table 1 contain formatted output values of the parameters and do not appear the same in the log file generated at runtime. The messages shown are not listed alphabetically.
