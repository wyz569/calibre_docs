setlayer tilegen

LITHO FILE Statement and Commands

Creates a layer to display tile and cell boundaries. Used by RET SBAR and RET MBSRAF.

Usage

setlayer layer = tilegen value

Arguments

layer

A required output layer name containing the generated representation of the tiles.

value

If the specified value is negative, the actual tiles are sized by the value. If the value is positive, the tile boundaries between valid tile regions are sized, not the actual tiles. The value specified must be between -0.1 and 0.1 microns.

Description

When performing optical process correction, RET SBAR processes the data arrayed in tiles. This command generates a layer that displays the tile boundaries used when processing your layout.

The output layer contains polygons representing every other tile. Therefore, when you display the output layer using a fill pattern, you see a checkerboard pattern.

For a detailed description of tilegen, see the Calibre OPCVerify User’s and Reference Manual.

Examples

setlayer t1 = tilegen -0.05
