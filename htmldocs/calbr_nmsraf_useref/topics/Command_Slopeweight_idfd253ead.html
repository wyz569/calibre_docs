slopeweight

Style Keywords

Specifies the gradient slope weight. Used by RET MBSRAF only.

Usage

slopeweight 1 | {verticalweight value narrowweight value slopeweight value}

Arguments

verticalweight value, narrowweight value, slopeweight value

Alternative keywords specifying the vertical, narrow and slope weight values, respectively, of the gradient slope. Each value you specify must be a whole number between 0 and 1, inclusive.

Description

Optional keywords that specify the weight associated with each gradient type. The default usage is slopeweight 1.

If the alternative usage in which all weight keywords are specified, the values of slopeweight, narrowweight and verticalweight must be summed to equal 1. Setting narrowweight or verticalweight to 1 is discouraged. The summed values for verticalweight, narrowweight, and slopeweight must be set between 0 and 1, respectively, such that:

vweight + nweight + sweight = 1

If you use DDM models with HHA approximation for 3D mask modeling, mbSRAF does not use narrowweight. Only slopeweight and verticalweight are applied to the final gradient value. For more information regarding the pxopc gradient computing use of mbsraf slope values, see the Release Notes for Calibre Release 2015.4.

Note:

Siemens EDA recommends relying on the default gradient slope weight, slopeweight 1, whenever possible.
