LITHO FILE Statements for EUV

You write statements within the LITHO FILE block to control RET EUV MBSRAF commands.

The statements shown in Table 1 are called by RET [EUV] MBSRAF only, and are specified one statement per line in the LITHO FILE block. They do not appear in any sraf_options statement.

Table 1. LITHO FILE Statements for EUV

Command

Description

bbm_options

Sets parameters for a black border model, overriding the values in a loaded model.

euv_field_center

Specifies the center of an EUV field.

euv_slit_x_center

Sets the X-coordinate for the center of an EUV through-slit litho model extension.

flare_longrange

Loads long range flare information from a litho model.

flare_model_load

Loads a flare model through an inline model.

shadow_bias_model_load

Loads a shadow bias through an inline model.
