mode

Style Keywords

Specifies a generated SRAF orientation. Used by RET MBSRAF only.

Usage

mode {collectno45 | nearest | nearestno45}

Arguments

collectno45

This mode begins with seed pixels and follows the gradient to define the final SRAF. The width is then defined by mapping nearby target shapes, and length is established in an orthogonal direction to width but limited by the gradient. This mode does not generate 45 degree SRAFs.

nearest

For each local maximum, this mode obtains the nearest edge(s) and corner(s) of the target layer and then adjusts the length and orientation of the SRAF accordingly. SRAFs tilted 45 degrees are only constructed near tilted target edges or near corners.

nearestno45

For each local maximum, SRAF length and orientation is determined by the length and orientation of the nearest target edge or corner. This mode does not create SRAFs tilted 45 degrees. If algorithm 0 is set, nearestno45 is the default setting.

Description

An optional keyword that specifies which SRAF orientation is used for all templates. Only one mode must be set for an mbsraf run. The mode keyword can only be specified globally.

Note:

Regarding algorithm interactions:

  • If algorithm 0 is set, nearestno45 is the default mode setting.
  • If algorithm 1 is set, mode is set internally and must not be set explicitly.
