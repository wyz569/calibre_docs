rect45corner

Templates

Places four rectangular SRAFs rotated 45 degrees at target shape corners. Used by RET SBAR only.

Usage

rect45corner
[tag name ...]
{length value width value}
{offsetx value offsety value}
[angledalso]
[prior value]

Arguments

tag name ...

An optional keyword specifying one or more tag names used in the template. The tag name identifies a group of edges or shapes to be used with the template. If specified, tags must appear as the first arguments in the template.

length value width value

Two required keywords, length and width, that specify the longer and shorter edges of a rectangular SRAF, respectively. The length and width values can be identical.

offsetx value offsety value

A required keyword pair that specifies the horizontal and vertical distance away from the corner of the target shape to the corner of the SRAF, respectively. By default, offsetx is measured from the Y‑axis aligned target edge and offsety is measured from the X‑axis aligned target edge. The offsetx and offsety values can be identical.

angledalso

An optional keyword that additionally generates axis-aligned SRAFs at corners of 45-degree oriented target shapes only for the rect45corner template.

prior value

An optional keyword that specifies a numeric identifier for placed SRAFs. Identifiers are applied sequentially, so a value of 1 is applied first, then 2 is applied, and so on. The default priority is 1. See “priority” for more information.

Description

An optional template and argument set that specifies the placement of four SRAFs rotated 45 degrees relative to target shape corners. The SRAF is placed by the center point of its edge closest to the target corner at the corresponding values of offsetx and offsety.
