setlayer mbsraf debug

LITHO FILE Statement and Commands

Generates and saves in-process debugging layers. Used by RET MBSRAF only.

Usage

setlayer output_layer = mbsraf debug layer debugkey priority

Arguments

output_layer

A required output layer name. The results of the debug operation are written to this layer. This layer name must be used in subsequent operations to output debug layer information.

layer

A required mbsraf layer from which the debug information is extracted.

debugkey

One of the following keywords must be specified:

  • candidates — Outputs the SRAF candidate shapes prior to cleanup. These SRAF candidate shapes are subsequently processed by any cleanup keywords specified. All candidate shapes are output as merged data by mbsraf debug.

  • lineend — Outputs the line-end keepout areas derived by the minfeaturespace lineend option. If the offsetlayer keyword is specified, all shapes derived for offsetlayer are merged with the lineend layer.

  • negative — Outputs the negative SRAF shapes generated for all priorities.

  • positive — Outputs the positive SRAF shapes generated for all priorities.

  • seedpixels — All pixels with the minimum number of directions with local maxima in the gradient are displayed by the seedpixels debug layer.

priority

A number required to specify the priority of the mbsraf debug operation.

Description

The debug command receives output data from the setlayer mbsraf command and, during mbsraf processing, generates layers for analysis that are viewable in Calibre WORKbench. This intermediate data helps you develop rules and debug mbsraf processing.

You must first specify setlayer mbsraf with a target layer for debug use to have any meaning. If not, mbsraf stops with an error message. Additionally, generated SRAF layers must be output to the database for any associated debug layers to be saved and subsequently viewed. No error message is reported when generated SRAF layers are not saved.

Examples

LITHO FILE ca [
	sraf_options mbsraf.opt {
		algorithm 1
		minoffset 0.025 ...
	}
	setlayer ca_sbar = mbsraf debug CA OPTIONS mbsraf.opt
	setlayer seedpx = mbsraf debug ca_sbar seedpixels 1
	setlayer candid = mbsraf debug ca_sbar candidates 1
]
ca_sbar = RET SBAR target FILE ca MAP ca_sbar
seedpx = RET SBAR target FILE ca MAP seedpx
candid = RET SBAR target FILE ca MAP candid

ca_sbar {COPY ca_sbar} DRC CHECK MAP ca_sbar 3
seedpx {COPY seedpx} DRC CHECK MAP seedpx 5
candid {COPY candid} DRC CHECK MAP candid 6
