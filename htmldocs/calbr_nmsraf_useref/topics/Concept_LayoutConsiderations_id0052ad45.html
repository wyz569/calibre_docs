Layout Considerations

The type of layout dictates the optimal Calibre nmSRAF template.

Template Applications

Apply pitch-based templates to square features only, and apply edge-based templates to all features meeting the edge length constraint on at least one edge.

Line-end Descriptions

Line-end criteria are specified by the minfeaturespace lineend keyword. Line-ends can be varied depending on their user definition. The line-end length, line-end leg length and adjacent vertices all become criteria for the selection of line-ends for SRAF generation. For more information, see the minfeaturespace keyword.
