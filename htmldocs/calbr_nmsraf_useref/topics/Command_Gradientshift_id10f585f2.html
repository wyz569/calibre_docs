gradientshift

Style Keywords

Specifies a gradient shift value. Used by RET MBSRAF only.

Usage

gradientshift value [autobright]

Arguments

value

A positive or negative floating point number.

autobright

An optional keyword to invoke gradientshift for bright field masks or negative tone resists for obtaining gradientshift values during SRAF recipe development.

Description

An optional keyword that specifies the amount to shift the gradient during SRAF generation. Siemens EDA recommends using gradientshift only to identify the necessary shift value and the explicitly specify this value in subsequent batch runs.

The autobright option calculates the shift value needed to move the gradient under all of the seed pixels to be positive in the first processed frame. It applies this shift in the current run and then prints the value in the transcript for the user to specify during subsequent runs. The autobright option is intended to assist with early stages of SRAF recipe development.
