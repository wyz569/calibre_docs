LITHO FILE Statement and Commands

You write commands within the LITHO FILE statement to control setlayer cnsraf and setlayer mbsraf.

The commands shown in Table 1 are called by RET SBAR or RET [EUV] MBSRAF and are specified one command per line in the LITHO FILE statement. They do not appear in the sraf_options statement.

Table 1. LITHO FILE Statement and Commands

Command

Description
LITHO FILESpecifies Calibre nmSRAF commands within the SVRF rule file. Used by RET SBAR and RET MBSRAF.
backgroundDefines background imaging characteristics. Used by RET MBSRAF only.
clone_transformed_cellsCopies cells with nonzero transforms so that simulation of asymmetric optical models is correct. Used by RET MBSRAF only.
ddmSpecifies the DDM model name. Used by RET MBSRAF only.
ddm_model_loadLoads DDM models. Used by RET MBSRAF only.
flare_longrangeAssociates a precomputed long range flare convolution with a flare model contained inside a litho model. Used by RET EUV MBSRAF only.
image_optionsA LITHO FILE command referencing the Lithomodel file and an sraf_options command referencing the litho_model name. Used by RET MBSRAF only.
layerAn input layer called by setlayer for generating SRAFs. Used by RET SBAR and RET MBSRAF.
litho_modelSpecifies the directory name that contains files for the litho model format. Used by RET MBSRAF only.
modelpathSpecifies the directories to search for optical and resist models. Used by RET MBSRAF only.
opticalSpecifies the optical model name. Used by RET MBSRAF only.
optical_model_loadSpecifies an optical model. Alternate to litho model format. Used by RET MBSRAF only.
planeSpecifies the depth of the z-axis within the resist film where the image is computed. Used by RET MBSRAF only.
processing_modeEnables various processing modes. Used by RET SBAR and RET MBSRAF.
progress_meterEstimates the progress of the LITHO operation. Used by RET SBAR and RET MBSRAF.
resist | thresholdSpecifies the resist model name or aerial threshold value. Used by RET MBSRAF only.
resist_model_loadSpecifies a resist model. Alternate to litho model format. Used by RET MBSRAF only.
setlayer cnsrafGenerates SRAFs using dimensional templates. Used by RET SBAR only.
setlayer cnsraf debugGenerates and saves in-process debugging layers. Used by RET SBAR only.
setlayer mbsrafGenerates SRAFs using a model-based methodology. Used by RET MBSRAF only.
setlayer mbsraf debugGenerates and saves in-process debugging layers. Used by RET MBSRAF only.
setlayer tilegenCreates a layer to display tile and cell boundaries. Used by RET SBAR and RET MBSRAF.
sraf_optionsAllows Calibre nmSRAF keywords to be written in free format. The sraf_options statement is called by the setlayer command for SRAF generation. Used by RET SBAR and RET MBSRAF.
svrf_var_importImports variables from the SVRF into the LITHO FILE command. Used by RET SBAR and RET MBSRAF.
tilemicronsSpecifies the size of a processing tile. Used by RET SBAR and RET MBSRAF.
