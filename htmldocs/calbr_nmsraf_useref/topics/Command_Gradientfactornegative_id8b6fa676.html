gradientfactornegative

Style Keywords

Specifies a gradient factor value for negative SRAFs. Used by RET MBSRAF only.

Usage

gradientfactornegative {-1 | 1}

Arguments

-1 | 1

Either negative or positive 1.

Description

An optional keyword that specifies the placement of seeds on valleys or peaks for negative SRAFs. Setting the value to 1 places the SRAF seeds on gradient peaks and setting it to ‑1 places the SRAF seeds on gradient valleys. Siemens EDA recommends setting gradientfactornegative to 1 only for DDL simulations.
