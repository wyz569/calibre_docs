Requirements and Assumptions

Specific tools and data are required for successful nmSRAF generation.
Optical models

Required for running Calibre pxOPC and defining SRAFs.

Layout specifics

Feature size, presence of multiple feature sizes, feature shapes, and typical feature pitches are required. An actual layout is not required.

MRC rules

Required to generate MRC-clean SRAFs.

Calibre® WORKbench™

Use to create test suites with an array of varying pitches and offsets. Calibre WORKbench also measures shapes and displays processing results. The Calibre® RVE™ tool coupled with Calibre OPCverify is useful for OPC validation.

OPC rules

Used to prepare layouts and enable validation of the original target.

PV band and area ratio calculation rules

Used as validation and performance indicators.
