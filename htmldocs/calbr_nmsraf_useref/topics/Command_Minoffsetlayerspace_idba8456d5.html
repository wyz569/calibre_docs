minoffsetlayerspace

Constraint Keywords

Specifies the minimum distance to SRAF shapes from the offsetlayer. Used by RET SBAR and RET MBSRAF.

Usage

minoffsetlayerspace value
[lineend value]
[projecting value]
[lineend2long value]
[long2lineend value]
[corner2sqcorner value]
[lineend2sqedge value]
[long2sqedge value]
[sqcorner2corner value]
[sqcorner2sqcorner value]
[sqedge2lineend value]
[sqedge2long value]
[sqedge2sqedge value]

Arguments

value

A required positive floating point number specified in microns.

lineend value

An optional keyword and positive floating point number that specifies the minimum distance an SRAF is placed from another SRAF. The projection is measured from the short edge of the rectangular SRAF. Line-end distance is measured between SRAFs. The lineend value must be greater than or equal to minoffset. Square SRAFs have no line-end projection.

projecting value

An optional keyword and positive floating point number that specifies the minimum distance allowed between projections of SRAFs. The projecting distance is measured orthogonally to the edges of the offsetlayer shapes. SRAF to SRAF projecting distances are enforced symmetrically. The projecting value may be greater than or equal to the specified lineend value, and must be greater than or equal to minoffset.

lineend2long value

An optional keyword and positive floating point number that specifies the minimum distance from the long side of the offsetlayer shape to the line-end of an SRAF. The measurement is performed during SRAF cleanup. The default is the specified value of projecting. The behavior is depicted in Figure 3.

long2lineend value

An optional keyword and positive floating point number that specifies the minimum distance from the offsetlayer shape line-end to the long side of the an SRAF. The measurement is performed during SRAF cleanup. The default is the specified value of lineend. The behavior is depicted in Figure 3.

corner2sqcorner value

An optional keyword and positive floating point number that specifies the minimum distance from the corner of any SRAF to the corner of a square offsetlayer shape (non-projecting cases only). All corner-to-corner parameters must be a smaller value than any edge-related parameter. The measurement is performed during SRAF cleanup. The default is the specified minoffset value. The behavior is depicted in Figure 2.

lineend2sqedge value

An optional keyword and positive floating point number that specifies the minimum distance from the line-end of an SRAF to the edge of a square offsetlayer shape. The measurement is performed during SRAF cleanup. The default is the specified projecting value. The behavior is depicted in Figure 2.

long2sqedge value

An optional keyword and positive floating point number that specifies the minimum distance from the longer edge of an SRAF to the edge of a square offsetlayer shape. The measurement is performed during SRAF cleanup. The default is the specified projecting value. The behavior is depicted in Figure 2.

sqcorner2corner value

An optional keyword and positive floating point number that specifies the minimum distance from the corner of a square SRAF to the corner of any offsetlayer shape (non-projecting cases only). All corner-to-corner parameters must be a smaller value than any edge-related parameter. The measurement is performed during SRAF cleanup. The default is the specified minoffset value. The behavior is depicted in Figure 3.

sqcorner2sqcorner value

An optional keyword and positive floating point number that specifies the minimum distance from the corner of a square SRAF to the corner of a square offsetlayer shape (non-projecting cases only). All corner-to-corner parameters must be a smaller value than any edge-related parameter. The measurement is performed during SRAF cleanup. The default is the specified minoffset value. The behavior is depicted in Figure 2.

sqedge2lineend value

An optional keyword and positive floating point number that specifies the minimum distance from the edge of a square SRAF to the line-end of an offsetlayer shape. The measurement is performed during SRAF cleanup. The default is the specified projecting value. The behavior is depicted in Figure 3.

sqedge2long value

An optional keyword and positive floating point number that specifies the minimum distance from the edge of a square SRAF to the longer edge of an offsetlayer shape. The measurement is performed during SRAF cleanup. The default is the specified projecting value. The behavior is depicted in Figure 3.

sqedge2sqedge value

An optional keyword and positive floating point number that specifies the minimum distance from the edge of a square SRAF to the edge of a square offsetlayer shape. The measurement is performed during SRAF cleanup. The default is the specified minoffset value. The behavior is depicted in Figure 2.

Note:

Conditional MRC parameters such as lineend2long, long2lineend, square2long, etc., must be specified by priority only for minoffsetlayerspace. Do not define these conditional parameters globally. Although parsed as valid, global specification of these parameters degrades performance.

Description

An optional keyword that specifies the minimum distance from any SRAF shape to the offsetlayer. You derive the offsetlayer for additional line-end keepout areas. This keyword can also be applied as a priority. For extended application information, see offsetlayer and minfeaturespace.
