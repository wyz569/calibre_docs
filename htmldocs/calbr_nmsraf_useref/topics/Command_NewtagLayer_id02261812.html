newtag layer

Tag Keywords

Specifies a tag name representing all target edges interacting with the specified tag layer. Used by RET SBAR and RET MBSRAF.

Usage

newtag name {layer taglayer_name} [asis] [fulledge]

Arguments

name

Specifies the name of the tag.

layer taglayer_name

A required keyword and layer pair that selects all target edges interacting with the specified tag layer. A taglayer is defined in the setlayer command using the taglayer keyword. The layer may be a polygon or an edge.

asis

The asis keyword specifies edges of the taglayer used in the tag. The fulledge and asis keywords are mutually exclusive. Layer tags using the asis keyword must not be mixed with other tags, and they cannot be used as boolean tags. If templates specify multiple tags, asis layer tags and other tags must not be mixed. You must perform appropriate boolean operations outside of Calibre nmSRAF.

fulledge

An optional keyword forcing the entire length of a tagged edge to be used even when a shorter portion of the edge is originally tagged. This option is compatible with all constraint types.

Note:

Partial edge tagging with Calibre mbSRAF may lead to unexpected results because of differences between intended SRAF and MRC enforcement. Siemens EDA recommends supplying the entire edge to mbsraf when using tagging.

Description

An optional keyword used to input tags by name to Calibre nmSRAF templates. Multiple tags can be referenced in a single template.

When using tagging, positive SRAFs may be generated parallel and perpendicular (given template constraints) relative to tagged edges and negative SRAFs are generated only parallel to tagged edges. See the example in Figure 2 for more information.

CAUTION:

Using constraint values that exceed cell or tile boundaries may cause flattening and increase run times. Siemens EDA recommends using the recta and rectb newtag types on relatively shorter geometries such as contacts, vias and local interconnect.

Examples

Example 1 — Managing Tag Names

taglayer tagl1
newtag tagl2 layer tagl1 fulledge 
newtag tagl3 layer tagl1 asis

Example 2 — Compounding length Tags

newTag tag1 l1 length > 0.10
newTag tag2 l2 length < 0.05
newTag tag3 external <0.13
newTag final tag3 and (tag1 or tag2)
