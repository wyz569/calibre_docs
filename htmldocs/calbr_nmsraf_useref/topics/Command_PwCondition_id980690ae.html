pw_condition

Style Keywords

Specifies a process window condition. Used by RET MBSRAF only.

Usage

pw_condition name
{[optical name]
  [dose value]
  [plane value]
  [weight value]
  [size value]
  [defocus value]
}

Arguments

name

The name of the process window condition.

optical name

An optional keyword that specifies the name of the optical model for this process window condition.

dose value

An optional keyword that specifies the dose for the specified process window condition.

plane value

An optional keyword that specifies the value of the z-position within the resist film.

weight value

An optional keyword that specifies the weight for the EPE for the specified process window condition.

size value

An optional keyword that specifies the mask size, in microns, for this process window condition.

defocus value

An optional keyword that specifies the defocus for the specified process window condition.

Description

An optional keyword that specifies the name and various attribute arguments of a process window condition. At least one attribute argument must be specified.

Note:

Siemens EDA recommends using the litho model format for all model-based SRAF recipes. These non-litho model arguments to pw_condition are not supported and are obsoleted as of release 2019.1:

  • optical
  • plane
  • size
Note:

You can specify multiple process window condition statements. All process window condition statements must be specified after the nominal optical model and any correctionlayer specifications.
