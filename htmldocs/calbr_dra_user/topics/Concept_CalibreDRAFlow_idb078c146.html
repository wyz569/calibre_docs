Calibre DRA Flow

The Calibre DRA flow encompasses the job’s run. While the job is running, you can use Calibre DRA to change the job’s resources by initiating a DRA event. The Calibre DRA flow ends upon completion of the Calibre job.

The following figure illustrates how Calibre DRA is used to adjust job resource allocations. Calibre DRA includes a set of command line statements that can be issued during the Calibre job to update CPU resource allocations and get real‑time status for remotes and job operations.
