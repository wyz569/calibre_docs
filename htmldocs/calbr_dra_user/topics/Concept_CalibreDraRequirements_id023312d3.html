Calibre DRA Requirements

There are certain requirements for running Calibre DRA in your job environment.

Hardware and Networking

All remote hosts must be able to communicate with all primary hosts in your hardware system and operating environment. All other requirements for running Calibre must also be supported. Refer to your Siemens representative for information specific to your environment.

Licensing

A specific license is required for running Calibre DRA in addition to the necessary product licenses for your Calibre jobs. The standard Calibre license consumption formula applies to Calibre DRA and all product licenses. Licenses are acquired and released according to the number of CPUs added or removed during the Calibre run. See also “Calibre DRA Licensing Behavior”. For complete information on all Calibre product licensing, refer to the Calibre Administrator’s Guide.

Calibre Version Support

Calibre DRA is supported as of Calibre version 2018.4.

Operating Systems

Calibre tools run on various operating system combinations and hardware configurations. For complete information, see “Supported Operating Systems and Hardware” in the Calibre Administrator’s Guide.

Environment Variables

If you do not use any Siemens EDA products other than Calibre, you can set the MGC_HOME environment variable to the location of your Calibre software tree. If you do use other Siemens EDA products in addition to Calibre, then set the CALIBRE_HOME environment variable to the location of your Calibre software tree and use the MGC_HOME environment variable for other Siemens EDA products. See “CALIBRE_HOME Environment Variable” in the Calibre Administrator’s Guide for more information.

Configuration Files

A Calibre MTflex configuration file is required with your remote host names and CPU numbers for the Calibre run with Calibre DRA. When invoked, Calibre DRA uses certain required Calibre MTflex file configuration statements for the run. See “Configuring the Calibre MTflex File for Calibre DRA” for detailed information.

Job Input Files

The input data and files for your Calibre run includes a GDS or OASIS®1 database, model files, and a Standard Verification Rule Format (SVRF) file.
