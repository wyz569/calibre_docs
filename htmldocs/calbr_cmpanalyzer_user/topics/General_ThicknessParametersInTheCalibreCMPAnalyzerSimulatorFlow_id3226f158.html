Thickness Parameters in the Calibre CMPAnalyzer Simulator Flow

The Calibre CMPAnalyzer simulator flow reports results on certain thickness parameters. All thickness data are reported in whole angstroms.
Table 1. Thickness Parameters

Parameter

Name

Description

Z1

Thickness_Z1

The oxide height used for physical hotspot calculations.

Z2

Thickness_Z2

The metal height used for physical hotspot calculations.

Z3

Thickness_Z3

The height of the trench bottom. This is the distance from the zero surface to the bottom of the metal.

Z4

Thickness_Z4

The height of the oxide surface before trench patterning.

Z2 - Z3

Thickness_Metal

The metal thickness.

topology

Thickness_Topology

The pattern‑density averaged surface height defined as follows:

Z1 *(1-D) + Z2*D, where D is the pattern density.
