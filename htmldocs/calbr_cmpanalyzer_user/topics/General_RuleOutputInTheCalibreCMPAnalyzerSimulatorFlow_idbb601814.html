Rule Output in the Calibre CMPAnalyzer Simulator Flow

Calibre CMPAnalyzer flow reports results in Calibre RVE for DFM for a number of rules.

The rules in the simulator flow are defined as follows.

  • Rule_Depth_Of_Focus — Z4max - Z4min (across the entire die).

  • Rule_Dishing — Z1 - Z2 (for the local tile).

  • Rule_Erosion — Z1field - Z1tile, where Z1field is defined as AVG(Z1*(1-D) + Z2*D) for all tiles in the interval [Dmin, Dmin*(1 + x)], where D is the pattern density and x <= 0.1.

    Note:

    If the minimum pattern density (Dmin) is zero, then Z1field = AVG(Z1) over all zero density frames.

Depth of focus, dishing, and erosion data are reported in whole angstroms.
