CMP Batch Reporting

Batch reporting can be used to output charts, colormaps, histograms and tables from Calibre RVE for DFM to a specified directory.

It is used with the Calibre CMPAnalyzer batch flow. Refer to “Using the Calibre CMPAnalyzer Batch Flow”.

The configuration file follows the format described in “DFM HTML Reporting” in the Calibre RVE User’s Manual.

The following is a sample configuration file that is designed to output a CMP hotspot report, and histograms and colormaps for all metrics.

To generate the reports, run Calibre as follows:

calibre -rve [-turbo] -dfm dfmdb -report config_file  
-outputdir output_dir -cmp

where:

  • dfmdb — The name of the DFM database directory.

  • config_file — The name of the configuration file.

  • output_dir — The desired output directory for the reports.
