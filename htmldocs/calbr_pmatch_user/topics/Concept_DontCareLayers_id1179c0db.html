Don’t Care Pattern Layers

Geometries on Don’t Care pattern layers are ignored during the matching process. Don’t Care layers can be added to a library so that two or more libraries have the same number and order of layers, which is required by the merge utility.

Don’t Care pattern layers are specified with these methods:

Method

Summary

With the GUI

Select the layer in the Layers tab and click the Set/unset Don’t Care button ().

See “Adding Pattern Layers to a Library” for instructions on adding a Don’t Care layer to a library.

With DFM Pattern Capture

Specify the keyword DC with LAYER_TARGET to specify the layer as a Don’t Care pattern layer.

With pdl_lib_mgr convert

Use the option add_layer to add a Don’t Care or empty pattern layer to a library.

Note:

Don’t Care pattern layers are not the same as Don’t Care regions. See “Don’t Care Regions”.
