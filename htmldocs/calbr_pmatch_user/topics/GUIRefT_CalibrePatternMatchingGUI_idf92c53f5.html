Calibre Pattern Matching GUI

Access with:

  • “calibrepm [pmdb]” from the command line

  • Verification > Pattern Matching > Open GUI from a Calibre layout viewer

The Calibre Pattern Matching GUI enables you to view and edit patterns in a library. You can also run pattern library utilities, such as merge and compile.

Figure 1. Calibre Pattern Matching GUI Main Window

Objects

GUI Object

Description

Main toolbar

Includes quick access buttons for common actions. Hover over a button for a tooltip.

Pattern Filter

Filters the pattern list based on pattern names, properties, and keys.

“Filtering a Pattern Library”

Pattern Canvas

Displays the pattern. The drawing and constraint toolbars include tools for creating and editing patterns. Choose Options > Preferences to set editor and ruler options.

“Drawing Shapes with the Pattern Matching GUI”

Pattern Data Pane

Displays the layer, constraint, or attribute data for a pattern, depending on the tab selected at the bottom of the pane.

“Working with Layers”

“Working with Constraints”

“Working with Properties and Attributes”

Show/hide toolbar

Includes buttons to show and hide GUI panes.

The Windows menu also includes settings for pane display.

Resize

Hover the mouse between GUI panes to resize the panes.
