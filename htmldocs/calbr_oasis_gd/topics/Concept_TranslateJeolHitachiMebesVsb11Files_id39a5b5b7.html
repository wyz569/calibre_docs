Translate JEOL, Hitachi, MEBES, and VSB11 Files

When translating JEOL, Hitachi, MEBES, or VSB11 files to OASIS, use the Calibre MDPview conversion batch commands.

You can also translate JEOL, Hitachi, MEBES, and VSB11 job decks using the jobToOasis conversion batch command in Calibre MDPview. For complete information, refer to the Calibre MDPview User’s Manual.
