Specify the OASIS Format for Input and Output

Most of the specification statements that support GDSII also support the OASIS layout format. You can specify the OASIS format for input or output using SVRF statements in your rule file.

The statements in the following table are used for layout input and output.

Table 1. Supported SVRF Rule File Statements for OASIS

The following MDPview SVRF statements support the OASIS format:

  • Fracture OASIS_MASK

  • MDP Checkmap

  • MDP Embed

  • MDP_OASIS2OASIS

  • MDPverify
