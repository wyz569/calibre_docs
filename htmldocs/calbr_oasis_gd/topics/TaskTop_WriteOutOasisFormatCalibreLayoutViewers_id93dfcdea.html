Write Out the OASIS Format with Calibre Layout Viewers

You can write out an OASIS file from the Calibre® WORKbench™, Calibre® LITHOview™, Calibre® DESIGNrev, or Calibre® MDPview™ tools.

Procedure

Use either of the following methods when a GDS file is loaded in the layout editor.
  1. Choose File > Export Layout and select Oasis as the format.
  2. In the shell where the viewer was invoked, use one of these commands:
    % $L oasisout filename 
 
% $L oasisout filename primary_cell_name

    where $L is the handle of the layout file loaded in Calibre layout viewer. For a complete list of command options, see $L oasisout in the Calibre DESIGNrev Reference Manual.

    Note:

    You can also specify layout exception severities with any of these Calibre layout viewers. Consult “Specify Layout Input Exceptions with Calibre Viewers”.
