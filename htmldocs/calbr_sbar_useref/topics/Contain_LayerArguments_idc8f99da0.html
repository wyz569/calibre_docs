Layer Arguments

Calibre OPCsbar requires an input layer and allows various optional layers.

For organizational keyword usage information, refer to “OPCSBAR”.

Table 1. Layer Arguments

Argument

Description
in_layerSpecifies an input layer name for which to generate scattering bars.
OFFSETLAYEREnables shapes to act as keepout areas for scattering bar placement.
ref_layerReference layer name for measuring violations.
SBLAYERPreexisting scattering bar layer name.
