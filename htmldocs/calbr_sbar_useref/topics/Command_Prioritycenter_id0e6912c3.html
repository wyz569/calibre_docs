PRIORITYCENTER

OPCSBAR keyword: Cleanup Control Arguments

Prioritizes the generation of scattering bars.

Usage

          PRIORITYCENTER

Arguments

None.

Description

An optional keyword that gives centered scattering bars the highest priority during the cleanup process instead of scattering bars closest to any valid edge.

By default, Calibre performs an ordered prioritization where the highest priority scattering bars are those closest to the valid edge (see “Scattering Bar Cleanup Order” for more detail). The next highest priority scattering bars are those that are second closest to the valid edge. When using PRIORITYCENTER, Calibre OPCsbar maintains center scattering bars at the expense of non-centered scattering bars as illustrated in Figure 1.
