Manufacturing Constraint Arguments

Calibre OPCsbar adheres to process parameters to output MRC-clean scattering bars.

For organizational keyword information, refer to “OPCSBAR”.

Table 1. Manufacturing Constraint Arguments

Argument

Description
LINEENDOFFSETDefines minimum line end offset.
LINEENDSPACEDefines minimum line end spacing.
LINEENDTOLONGSPACEDefines minimum line end to side spacing.
MAXSBLENGTHDefines maximum length for positive and negative scattering bars.
MAXSBWIDTHDefines maximum width of scattering bars.
MINEDGELENGTHDefines minimum edge lengths.
MINSBLENGTHDefines minimum length of scattering bars.
MINSBOFFSETDefines a minimum offset.
MINSBSPACEDefines minimum scattering bar spacing.
MINSBWIDTHDefines minimum scattering bar width.
