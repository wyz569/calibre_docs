dfm::list_properties

Lists all property names defined on a layer or geometry.

Usage

dfm::list_properties {layer_name | iterator} [-min_max | -show_prop_target]

Arguments

layer_name

An argument that specifies a layer name. The command reports the properties on this layer. May not be specified with iterator.

iterator

An argument specifying a layer or geometry iterator. The command reports the properties on the current object for this iterator. May not be specified with layer_name.

-min_max

Optional argument that returns the minimum and maximum values for numerical properties, along with the property target type.

-show_prop_target

Optional argument that shows the target of the property when DFM Property originally assigned the property. The target can be POLY_PROPERTY (shape property), NET_PROPERTY, or CELL_PROPERTY.

Return Values

Tcl list.

Examples

Example 1

> dfm::list_properties metal1c -min_max
{AREA POLY_PROPERTY 40.0 415.5}
# property name is AREA, originally assigned to polygons, 
# with minimum being 40 and maximum being 425.5 square microns
