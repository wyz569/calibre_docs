Introduction to Calibre YieldServer

Calibre® YieldServer processes layout design and rule file data after a calibre -dfm run.YieldServer scripts post process design data in a DFM database. Specifically, YieldServer allows retrieval of layer, polygon, edge, connectivity, property, device, and other information in a DFM database. Annotations and DFM properties facilitate detailed data analysis.

The DFM database used by Calibre YieldServer supports revisions, so multiple versions of a database can be stored and accessed independently. YieldServer also allows modification of this stored data for output to mask or ASCII RDB layers.

Certain commands are supported in the Calibre Query Server Tcl shell. These are of interest in LVS flows.
