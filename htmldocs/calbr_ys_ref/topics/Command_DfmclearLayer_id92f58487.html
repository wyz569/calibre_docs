dfm::clear_layer

Deletes all geometries from a layer and makes it an empty layer.

Usage

dfm::clear_layer layer_name

Arguments

layer_name

Required argument that specifies the layer to be emptied.

Return Values

None.

Examples

# makes layer1 empty
dfm::clear_layer layer1
