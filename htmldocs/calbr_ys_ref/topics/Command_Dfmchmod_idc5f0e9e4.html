dfm::chmod

Changes the permissions of a DFM database.

Usage

dfm::chmod permissions dfmdb_path

Arguments

permissions

Required argument that specifies the permissions mode. This argument accepts either numeric (octal) or alphabetic permissions values, similar to the Unix chmod command.

dfmdb_path

Required argument that specifies the path to the DFM database to which permissions are applied.

Return Values

None.

Examples

# change permissions using octal values 
dfm::chmod 777 dfmdb 
 
# change permissions using letters 
dfm::chmod u+wr dfmdb
