dfm::count

Returns the number of polygons in a geometry iterator or the number of placements of the current cell in a cell iterator.

Usage

dfm::count iterator

Arguments

iterator

A required argument specifying a geometry iterator, such as from dfm::get_device_geometries, dfm::get_flat_geometries, dfm::get_geometries, or dfm::get_net_shapes, or a cell iterator from dfm::get_cells. This command gives an error when passed an iterator other than for polygons or cells.

Return Values

Integer.

Examples

# get the count of M1 shapes
set m1_iter [dfm::get_geometries M1]  
set m1_count [dfm::count $m1_iter]
