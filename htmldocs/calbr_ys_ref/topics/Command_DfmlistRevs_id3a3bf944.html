dfm::list_revs

Returns a list of revisions for a DFM database.

Usage

dfm::list_revs [-db db_name] [-name]

Arguments

-db db_name

Optional argument that specifies the path to a different DFM database than the currently loaded one. By default, the currently loaded one is queried.

-name

Optional argument that displays the names of the revisions, if they exist. By default, numbers are returned.

Return Values

Tcl list.

Examples

> dfm::list_revs 
0 
# current database only has the initial version
