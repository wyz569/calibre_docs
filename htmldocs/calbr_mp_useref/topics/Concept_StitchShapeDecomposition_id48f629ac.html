Stitch Shape Decomposition

The dpStitchCandidates command generates stitches over regions of overlapping shapes on previously decomposed masks.

Some foundries permit polygons to be split across masks. The dpStitchCandidates LITHO FILE shown here is one possible way to automate locating stitch placements. It returns two layers, the stitch candidates and the separators. Example Figure 1 shows the setup file, which would be placed inside an SVRF file and called by RET DPSTITCH.

Figure 1. Stitched Shape Decomposition
