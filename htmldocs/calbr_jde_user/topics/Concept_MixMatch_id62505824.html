Mix and Match

For this flow, the wafer map and the optimized chip placement parameters are provided as an input file. The geometry to be placed is also given in MEBES pattern format.

The work flow for the Mix and Match process is summarized in Table 1:

Table 1. Mix and Match Process

Stage

Description

Stage 1 - Input Wafer Map

Input a 1x job deck containing the product layout and a wafer map.

Stage 2 - Reading Chip Placement

The chip placement is already done and provided by the input wafer map file.

Stage 3 - Exclude Chips From Exposure

Chip placement can be controlled by specifying the allowed placements for this chip.

Stage 4 - Manual Editing

Manually correct any problems by editing the OASIS file.

Stage 5 - Generate Job Deck

Either save the work as an OASIS layout or fracture to generate a MEBES job deck.

The full process is documented in the section “Mix and Match Flow”.
