Topo Model Development Flow in Calibre nmModelflow

Implementing a topographical model in Calibre WORKbench nmModelflow uses a specific development flow.
Note:

The instructions in this manual assume a general knowledge of the concepts and tasks described in the Calibre nmModelflow User’s and Reference Manual.

Table 1. Topographical Modeling Flow (Calibre nmModelflow)

Step

Task

Described In

1

Create a litho model and add required model files and configuration information.

“Creating Litho Models for Topography Modeling”

2

Add gauge files to the database.

Described in the Calibre nmModelflow User’s and Reference Manual

3

Add stages to the database for each model in the litho model.

“Creating Calibration Stages for Topo Models”

4

Set up calibration jobs for the stages and run them.

“Creating and Running Calibration Jobs”

5

Set up verification jobs for the stages and run them.

“Creating and Running Calibration Jobs”

6

Use a calibrated model in Calibre nmOPC for simulation.

“Using the Topographical Model in Calibre nmOPC”

7

Use a calibrated model in Calibre OPCverify for design verification.

“Using the Topographical Models in Calibre OPCverify”
