toposignal

Tcl batch command

Outputs the toposignal data for a specified signal from the input topo model.

Usage

toposignal -model modelfilename -signal signaltype [-out outputfilename]

Arguments

-model modelfilename

A required argument specifying a file containing a topo model in topo_model Format. However, model-layer files are not read by this command; in order to use this command, you must copy only the contents of the topo_model block to a separate file.

-signal signaltype

A required argument specifying the type of signal to generate. Accepted values are corner, corner2, finfet, finfet_sw, gate, interconnect, sidewall, sidewall2, sidewallo2, trench, trench2, trench4, trencho2, trench2o2, trench4o2, and vcorner.

-out outputfilename

An optional argument specifying the name of an output file. If one is not specified, then standard output (stdout) is used.

The data has two columns. The first column is the distance from the source point in microns. The second column is the value of the topo signal. The topo signal is axially symmetrical.

You can visualize the data by plotting in a spreadsheet program or MathWorks MATLAB1. For MATLAB, use the following code: 

s = load('outputfilename'); 
plot(s(:,1), s(:,2));

Examples

This example outputs the sidewall signal data for the topo model in the file mytopo.txt. Output is written to the file toposignal.txt.

toposignal -model mytopo.txt -signal sidewall -out toposignal.txt
