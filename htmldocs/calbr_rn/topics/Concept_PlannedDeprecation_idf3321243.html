Planned Deprecation

Deprecated functionality should be removed from setup files before the functionality becomes obsolete. Obsolete functions can prevent your run from completing.

Table 1 lists all functionality for which support will be removed in a future release.

  • Deprecated functionality may continue to operate, however it will no longer be supported by the Calibre team.

  • All deprecated functionality that is currently documented may remain in the documentation, with a deprecation notice reminding customers that it is no longer recommended.

  • Deprecated functionality may continue to exist for a subsequent release after first notification through the Calibre release notes or documentation. After that time it may become obsolete.
