Obsolete Keywords

Obsolete keywords and variables may cause a run to halt with an error. The functionality once referred to by the keyword or variable has been removed from Calibre, which means even if the run completes the simulation and optimization may not have occurred as designed.

Table 1 lists all previous functionality that has been disabled or removed from the release.

  • Obsoletion will occur no sooner than six months after first deprecation of the functionality.

  • The functionality description will be removed from documentation and will no longer be supported by the Calibre team.
