Updates to SEMSuite Tools

Adds the SEMSuite toolset that includes:

  • The Raw Data Filtering (RDF) tool for processing CD SEM image and data files to produce a gauge file.

  • The Contour Data Flow (CDF) tool for extracting contours from CD SEM images to produce a Sites CSV file.
