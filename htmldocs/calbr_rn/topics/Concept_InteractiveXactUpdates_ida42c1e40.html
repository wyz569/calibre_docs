Calibre Interactive xACT Updates

The following updates to Calibre Interactive xACT were made in the 2021.2 release.

New Probes Page Added

A Probes page has been added to Calibre Interactive xACT for specifying probe points, which are used to verify timing from specific points on each net. On this page, you can add and load probe points, set their locations, and configure other options.

New PEX Netlist Keywords Added

The CALIBRATED and RES_EMIR keywords are now supported in Calibre Interactive xACT and can be enabled in the Options page when working with DSPF netlists.

The CALIBRATED keyword is only available if either LAYERMAP is selected or the netlist compatibility mode is set to TOTEM. When used with LAYERMAP, the CALIBRATED keyword specifies to only list calibrated layers in the LAYER_MAP section of the extracted netlist. When used with TOTEM, it specifies to only netlist calibrated layers in the LAYER_MAP section of the netlist.

The RES_EMIR keyword specifies to include the PCDEF and PEX ignore internal resistance dimensions in the extracted DSPF netlist for the EM/IR analysis flow.
