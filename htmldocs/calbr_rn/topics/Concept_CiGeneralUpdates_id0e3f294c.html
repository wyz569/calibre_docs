Calibre Interactive General Updates

The following general update was made to Calibre Interactive in the 2021.2 release.

New Batch Command Switch for Converting Classic Runsets

When running the Calibre command line in the new Calibre Interactive GUI, you can now convert classic runsets using the switch “‑output_runset” with a runset name argument. You can also use this switch to combine multiple runsets; if multiple are specified, they are combined into the new runset, with the last runset taking precedence.

File Viewing Enhancements Added

In the new Calibre Interactive GUI, you can now visualize files in the Include Rules Files and Waiver Criteria Files sections of the Options and Waivers pages, respectively.

Added Distributed MTflex Support

In the new Calibre Interactive GUI, you can now run Calibre using Distributed (MTflex) when running on Other Cluster, which was previously called Cluster Command. Specify this option in the Run Control page.
