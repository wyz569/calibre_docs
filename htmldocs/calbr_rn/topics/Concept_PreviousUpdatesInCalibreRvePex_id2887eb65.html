Calibre RVE for PEX Previous Updates

Several updates were made to Calibre RVE for PEX in previous releases.

Find Nets and Coupling Capacitance Control Bar Enhancements

(2020.3) Under the Parasitics tab, the Find Nets and Coupling to: Specified Nets search fields now have a dropdown menu populated with all of the layout nets. You can select a net or enter a space-separate list of nets as before.

BSPEF API Support Added

(2020.2) Calibre RVE for PEX can now read BSPEF netlist files produced by Calibre xACT runs. Point-to-point inductance calculation is not supported.
