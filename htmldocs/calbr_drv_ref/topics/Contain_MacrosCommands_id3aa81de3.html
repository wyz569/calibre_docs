Macros Commands

Macros are application extensions you write to add new functionality to your application. In order to use macros, you must first add them to the Macros menu.

For a complete discussion of creating macros, refer to Chapter Writing Macros, “Writing Macros”.

Table 1. Macros Command Summary

Macros Command

Description
Macros createCreates a macro.
Macros deleteRemoves (un-registers) the specified macro.
Macros menuRe-creates the Macros menu with all registered macros.
Macros showReturns information about macros currently registered with the tool.
