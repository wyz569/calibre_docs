$cwb setDepth

Sets the depth for the Calibre layout viewer object.

Usage

$cwb setDepth startDepth endDepth

Arguments

startDepth

Required arguments that specifies the starting depth to use for the specified Calibre layout viewer object.

endDepth

Required argument that specifies the ending depth to use for the specified Calibre layout viewer object.

Return Values

None.

Description

Sets the depth for the Calibre layout viewer object. You need to follow this command with a $cwb updateDisplay command to see the effects of the command in GUI mode.

Examples

set wb [find objects -class cwbWorkBench] 
$wb setDepth 0 3 
$wb updateDisplay
