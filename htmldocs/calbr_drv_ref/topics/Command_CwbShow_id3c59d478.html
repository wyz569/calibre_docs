$cwb show

Redisplays the layout.

Usage

$cwb show

Arguments

None.

Return Values

Redisplays the layout.

Description

Redisplays the layout. It raises the window to the front if it is hidden by other windows.

Examples

set wb [find objects -class cwbWorkBench] 
$wb show
