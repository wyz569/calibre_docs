$L clips

Returns clips for a particular cell.

Usage

$L clips cellName

Arguments

cellName

A required argument specifying the name of the cell.

Return Values

A Tcl list of lists of the attributes for the constituent clips. Each list contains:

  • Layer number.

  • Name of the clip.

  • Clip coordinates, starting with the lower left X and Y and followed by the upper right X and Y.

  • Boolean stating whether to incrementally load the area of the clip.

  • Boolean stating whether to place a marker dot in the center of the clip when the clip is drawn.

Description

Retrieve the set of clips for a particular cell. Clips are areas of interest you mark on a layout, allowing you to represent rectangular regions of a layout with four coordinates. Created clips are independent of the layout.

Examples

set wb [find objects -class cwbWorkBench] 
::cwbWorkBench::cwbWorkBench0 
set lay [$wb cget -_layout] 
layout0 
$lay clips a1220 
{1 {Clip 1} 1716 52549 16921 20622 false true}
