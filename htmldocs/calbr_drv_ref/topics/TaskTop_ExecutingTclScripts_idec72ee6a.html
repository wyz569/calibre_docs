Executing Tcl Scripts

You can execute Tcl scripts in batch or interactive mode.

Procedure

Use one of the following methods to set to execute a Tcl script:

If you want to...

Do the following:

Execute a Tcl script in batch mode, by including the pathname for the script when invoking the layout viewer

 
calibredrv my_script.tcl 
calibredrv my_script.tcl -gui

Execute a Tcl script in interactive mode, by sourcing the script from the command line

 
% source my_script.tcl
