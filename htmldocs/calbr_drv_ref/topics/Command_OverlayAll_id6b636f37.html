overlay all

Outputs the handles of all overlay objects existing within the application database.

Usage

overlay all

Note:

Use this function to see which overlays are open in memory.

Arguments

None.

Return Values

A list of all overlay object handles. The overlay handles are of the form “overlayN” where N is an increasing integer starting with zero.

Description

Outputs the handles of all overlay objects existing within the application database.

Examples

% overlay all 
olobj1 olobj2 olobj5
