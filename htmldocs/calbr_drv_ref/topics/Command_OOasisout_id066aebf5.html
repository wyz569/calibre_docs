$O oasisout

Saves the overlay to an OASIS file.

Usage

$O oasisout filename [topcell]

Arguments

filename

Required argument specifying the pathname for the OASIS database file to which the layout is written.

topcell

Optional argument specifying the name of the topcell to use in saving the overlay.

Note:

If topcell is not specified, tool uses the overlay cell name as the topcell.

Return Values

None.

Description

For overlays only, saves the overlay to an OASIS database file. Only overlays composed of OASIS files can be exported as an OASIS layout.

When exporting an overlay to an OASIS layout file, first save your latest changes for each layout handle of the overlay’s layout files, as this command saves each layout handle’s original file on disk.

Examples

% set wb [find objects -class cwbWorkBench] 
::cwbWorkBench::cwbWorkBench0 
% set O [$wb cget -_layout] 
overlay0 
% $O oasisout mixoas
