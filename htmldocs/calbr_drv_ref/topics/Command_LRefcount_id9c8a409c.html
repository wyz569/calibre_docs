$L refcount

Reports the number of instances of a specified cell or a total count of the instances in the specified parent cell.

Usage

$L refcount {[-parent cellName] cellName} | [cellName]

Arguments

-parent cellName

An optional argument used to return the number of instances of the specified cellName found in the parent cell.

cellName

An optional argument specifying the name of a cell. This argument returns different results depending on how it is used.

  • To return the total number of instances of a cell in a layout, specify cellName without any other arguments:

    $L refcount cellName

  • To return the number of instances contained in a cell, specify cellName with the ‑parent argument:

    $L refcount -parent cellName

  • To return the number of instances of a child cell found in a parent cell, specify the ‑parent argument, name of the parent cell, and name of the child cell:

$L refcount -parent parent_cell child_cell

Return Values

None.

Examples

Example 1

This example returns the number of instances found in “cellb40”.

$L refcount -parent cellb40

Example 2

This example returns the total number of instances of “celli01” in the design.

$L refcount cellio1

Example 3

This example returns the number of instances of “cellchild” in “topcell”.

$L refcount -parent topcell cellchild
