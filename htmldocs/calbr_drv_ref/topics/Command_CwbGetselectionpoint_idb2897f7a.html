$cwb getSelectionPoint

Gets the selection point.

Usage

$cwb getSelectionPoint [microns | dbu]

Arguments

microns | dbu

Optional argument indicating the units to use for the return data. The default is microns. One micron (um) times the precision equals one dbu.

Return Values

Returns the selection point.

Description

Returns a list containing the coordinates of the last selection point used.

Examples

% set cwb [find objects -class cwbWorkBench] 
% $cwb hasSelectionPoint 
1 
% $cwb getSelectionPoint dbu 
3500 360000
