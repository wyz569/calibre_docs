layout delete

Deletes the specified layout.

Usage

layout delete handle

Arguments

handle

A required argument specifying the handle of the layout to be deleted.

Return Values

None.

Description

Deletes the layout specified by handle.

CAUTION:

This command should only be used for deleting layouts when running scripts in batch mode without a GUI. If the script is working with a displayed layout in the GUI, use the $cwb deleteLayoutClbk command to avoid generating errors.

Examples

% layout all 
layout2 layout3 layout4 layout5 
% layout delete layout3 
% layout all 
layout2 layout4 layout5
