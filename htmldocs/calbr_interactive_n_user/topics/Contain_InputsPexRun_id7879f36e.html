Inputs for a PEX Run

The Inputs pane includes the tabs for each type of input. Use these tabs to specify the inputs needed for your PEX run.
Table 1. Inputs Pane for PEX

Tab

Purpose

Required

Layout

Specify the layout design database and format.

Yes

Netlist

Specify the schematic netlist. This is used when you want the extracted netlist to use source schematic names. Netlist data is also needed when controlling cell extraction options for an ADMS run.

No

H-Cells

Specify cell lists for performing non-flat extraction.

No

Blocks

Specify blocks for ADMS extraction. See “Setting Cell Extraction Options for Blocks” and the Calibre xRC User’s Manual.

No

Probes

Specify probe points. See “Setting Probe Points” and PEX Probe File in the SVRF Manual.

No
