Database Page in Calibre Interactive

To access: Choose Settings > Show Pages > Database

The Database page has settings for precision, resolution, and additional layout and source input files.

Objects

Object

Description

Units

Precision

Specifies the ratio of database units to user units.

Enter one floating-point value specifying the precision, or two positive integers, where int_1/int_2 is the precision.

Rule file statement: Precision

Override layout value

Specifies whether to override the layout database precision. This option is only visible if the Precision option is checked.

Rule file statement: Layout Input Exception Severity PRECISION_RULE_FILE 1

Resolution

Specifies the layout grid step-size (the number of database units on which original geometry is required to be aligned).

Enter a positive integer to specify an equal grid step size in the x and y directions.

Enter two positive integers to specify the x and y grid step size, respectively.

Rule file statement: Resolution

Library

Additional Layout Files

Specify layout input files in addition to what is specified on the Inputs page.

The additional layout files are added to the Layout Path statement.

Additional Golden Layout Files

Specify additional layout files that are to specified as golden.

See the GOLDEN keyword with the Layout Path statement.

Additional SPICE Files

Specify SPICE input files in addition to what is specified on the Inputs page.

The additional source files are added to the Source Path statement.

Additional VERILOG Files

Specify VERILOG input files in addition to what is specified on the Inputs page.

The additional source files are added to the Source Path statement. This option is only available if the source format is VERILOG.

DFM RDB Defaults

(DRC only)

Specify options for the DFM Defaults RDB statement.
