Edit Parameters for Device Dialog Box in Calibre Interactive

To access: On the Signatures page with a run mode of Signature Creation, expand the Device Statements area and do one of the following:

  • Click the button to add a device to the Device Statements table.

  • Select an existing device in the Device Statements table, and click the (Edit Device) button.

Use the Edit Parameters for Device dialog box to specify the layers, pin names, and options used for device signature creation in Calibre Interactive.

Description

Note:

You can specify and load a Layers File to pre-populate the dropdown lists and tables in the dialog box; see “Layers File for Device Signature Creation With Calibre Interactive”.

However, you can add table rows, and type in any entry field or table row to enter an arbitrary parameter, so loading a Layers File is not necessary.

Objects

Table 1. Device Parameter Fields for Create Device Signature

Device Parameter Field

Description

Device Name

The component name.

Seed Layer

The layer the device body is on; also known as the device layer.

Reference Layer

The reference layer defines the area around the seed shape used for deriving a signature.

Once a reference layer is used with a specific set of context layers, it may only be used with that set of context layers in the rule file.

Device Pins

The Device Pins table has the pin name and pin layer pairs necessary for the Device statement.

Use the right-click menu or button bar to add, delete, and select pins.

Swappable With — Enter the pin names that are interchangeable with the pin in the row. If this entry is left blank, the default pin swappability rule applies: pins on the same layer are interchangeable.

Context Layers

Shapes on the context layer that intersect the reference layer are considered in the signature.

You can specify multiple context layers. The order of the context layers matters, since the generated signature reflects the given order. You can change the order of context layers by dragging the layer entry—grab the row at either end and drag.

Use the right-click menu or button bar to select or unselect all layers and to toggle the layer selection.

You can use the layers file to map a list of context layers to a list name; see the *CONTEXT_LAYER entry in “Layers File for Device Signature Creation With Calibre Interactive”.

Note: The seed layer and pin layers are not part of the signature calculation unless the layers are included in the context layer list.

Use only seed layers with attached text label

Text Label

Text Layer

When enabled, only device seed layers with the specified text label on the given text layer are used in the Device statement. The text label string is appended to the seed layer name and a new layer is created which is used in the Device statement. These statements are used:

<seed_layer>_<label> = <seed_layer> WITH TEXT
    "label" <text_layer>
DEV <device> <seed_layer>_<label> ...

Model Name

Use the specified model name (or component subtype).

Text Model Layer

Use the TEXT MODEL LAYER keyword and parameter,

Netlist Model

Use the NETLIST MODEL keyword and parameter

Netlist Element

Use the NETLIST ELEMENT keyword and parameter.
