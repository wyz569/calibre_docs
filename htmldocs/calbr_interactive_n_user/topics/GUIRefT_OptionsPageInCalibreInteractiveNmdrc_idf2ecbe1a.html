Options Page in Calibre Interactive nmDRC

To access: Click Options in the left panel

The Options page includes settings that control the Calibre nmDRC run.

Note:

“Options Page in Calibre Interactive: Common Options” discusses these options that are common to all applications: “Virtual Connect,” “Include Rules Files,” “Include Rule Statements,” and “Include Layout Text File.”

Objects

Object

Description

DRC Maximum Results

Specifies the maximum results count for any given rule check.

Rule file statement: DRC Maximum Results

DRC Maximum Vertex

Specifies the maximum vertex count of any polygon written to the DRC results database.

Rule file statement: DRC Maximum Vertex

Preserve Cells

Specifies layout cells to protect from most hierarchy modifications, such as automatic expansion. Only used in hierarchical runs.

To read cell names from an existing waiver file, click the Import from RVE Waiver File button and selected a .waived file. All the cells that contain waived results are entered into the cell list. You can then delete or add cell names as needed.

Rule file statements: Layout Preserve Cell List and Layout Cell List
