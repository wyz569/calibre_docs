Preferences Page in Calibre Interactive

To access: Choose Settings > Show Pages > Preferences

The Preferences page has settings for loading and saving the runset, saving the transcript, loading a custom configuration file, and other settings.

Objects

Object

Description

Startup

Specify the startup behavior for loading a runset.

Recipes

(DRC and LVS/ERC)

Specify check recipe files to load at startup. The check recipes are listed on the Recipe Editor page, in the USER RECIPES section of the Recipe dropdown list.

Tip

Check recipes are saved to file with the Export button in the Recipe Editor.

Runset

See “Runset Preferences”.

Rules

See “Rules Preferences”.

Run

(PEX)

The following option is included for Calibre Interactive PEX:

  • Stop parasitic extraction if LVS indicates mismatch

Transcript

Specify whether to save the transcript to file.

Errors and Warnings

Specify whether to save errors and warnings to file, the file name, and the save mode (replace or append).

Configuration

  • Configuration — (checkbox) Specify whether to load configuration files on invocation.

  • Configuration files — Specify one or more configuration files.

  • Load Configuration Files — (button) Click to load the specified configuration file(s) in the current GUI session.

The configuration files are saved to the runset. If the “Configuration” checkbox is checked, the configuration files are loaded the next time you load the runset.

Note:

Configuration files specified on the command line or with an environment variable have higher precedence. If either is specified, configuration files specified in the runset are not loaded and the Load Configuration Files is not present.

Disable

Disables certain GUI options. See “Disable Preferences”.

Misc

See “Miscellaneous Preferences”.
