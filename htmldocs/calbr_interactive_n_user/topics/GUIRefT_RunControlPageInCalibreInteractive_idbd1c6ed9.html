Run Control Page in Calibre Interactive

To access: Click Run Control in the left panel of Calibre Interactive (shortcut: Alt-c)

You set the host processor, run, and licensing options on the Run Control page. You can also set options to connect to a design tool and specify whether to open Calibre RVE after the run.

Objects

Category

Description

Run Calibre On

Settings for the host processor or remote cluster.

  • Local Host — Calibre runs on the host currently running Calibre Interactive.

  • Remote Host — Calibre runs on the specified remote host.

  • Other Cluster — Calibre runs on a custom computer cluster. Enter the appropriate command in the Queue Command field. You can use the replaceable parameters listed in the “Parameter Substitution Information” area.

    Synchronous checkbox:

    • Checked — Calibre Interactive waits for the Calibre run to complete. Calibre RVE and reports are displayed when the run is finished if appropriate options are set.

    • Not checked — Calibre Interactive executes the queue command as a background process. The run transcript, Calibre RVE, and reports are not displayed when the run finishes.

  • Platform LSF — Calibre runs on the IBM® Spectrum™ LSF® (Load Sharing Facility) distributed computing solution. Specify the command in the “LSF Command” field.

Also see the Calibre Licensing and Remote Calibre Environment options when Remote Host or Other Cluster is selected.

Run Calibre

Settings for run mode (single-threaded, multithreaded, or Calibre MTflex) and behavior if licenses are not available for all requested CPUs.

  • Single Thread — Single-threaded execution.

  • Multiple Threads — Multithreaded execution on a single host computer. Specify the number of CPUs to use in the entry field. (‑turbo num_processors)

  • All Threads — Multithreaded execution on a single host computer with all available processors. (‑turbo)

  • Distributed (MTflex) — Use your existing computer network as a distributed computing platform with Calibre® MTflex™.

    Specify the remote hosts file, which is used as the parameter for the ‑remotefile command line argument.

    If desired, specify Remote Data Server (RDS) and the number of servers. See “Remote Data Server” in the Calibre Administrator’s Guide.

Run Calibre LVS Compare On Remote Host

(LVS only)

Settings for running LVS compare on a remote host. This option is available when the run mode is not set to Single Thread. Requires a Calibre nmLVS Advanced license.

  • Specify Host — Specify the host with the host name

  • Specify File — Specify the host with the remote host file

RVE Options

  • Show Results in RVE — Automatically open the results database in Calibre RVE when the run finishes.

  • Start RVE as soon as results are available — As described. (DRC only)

  • Do not open empty main results database in RVE — (DRC only) When enabled, an empty main results database (RDB) is not opened if at least one side (auxiliary) RDB will open automatically. If no side RDBs are specified to open automatically, or if all side RDBs are empty, then an empty main RDB is opened.

    Side RDBs can be specified to open automatically with the Calibre Interactive setting “Side RDBs” on the Outputs page; see “Outputs Page in Calibre Interactive nmDRC”.

    This option is not available if “Start RVE as soon as results are available” is checked.

  • Start RVE after batch run — Automatically open Calibre RVE after a batch run finishes.

  • Run RVE on local host — When checked, Calibre RVE runs on the local host. When unchecked, Calibre RVE runs on the host specified for “Run Calibre On”.

  • Close RVE before running Calibre app — As described.

  • Close RVE when Calibre Interactive closes — As described.

Design Tool Settings

Settings for connecting to a layout or schematic design tool, including the socket number. See “Connecting to a Design Tool in Calibre Interactive”.

PEX Steps

(PEX only)

  • Create the PHDB — Creates the PHDB with Calibre xRC if PEX Netlist specifies the use of layout names (LAYOUTNAMES). Creates the PHDB with Calibre nmLVS if PEX Netlist specifies the use of source names (SOURCENAMES, SOURCEBASED, or SCHEMATICONLY).

  • Create the PDB — Runs the PDB step.

  • Create the parasitic netlist — Runs the formatter step.

Calibre Licensing

Settings for the licensing mode. See “Setting Licensing Options in Calibre Interactive”.

The “Remote Calibre” section is available to set remote licensing options when “Run Calibre on” is set to Remote Host, Platform LSF, or Other Cluster.

Calibre Interactive License Timeout

Settings the for license timeout.

The license timeout can also be set with the environment variable MGC_CALGUI_RELEASE_LICENSE_TIME; the default unit is hours. See “License Timeout in Calibre Interactive” for instructions on using other units.

Remote Calibre Environment

Specify the value of MGC_HOME, MGC_LIB_PATH, the remote shell, and the remote username. See “Configuring the Remote Environment”.

This section is available when “Run Calibre on” is set to Remote Host, Platform LSF, or Other Cluster.
