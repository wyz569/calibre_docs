A list of lists defining the contents of the table for the option “LVS Black Box Port”. This table defines parameters for the for the LVS Black Box Port statement.

Each sublist defines parameters for one statement. The sublist syntax is:

[ bool, "val1 val2 val3"]

  • bool — true or false. Indicates whether the statement is used.

  • val1 — The original layer that forms the port.

  • val2 — The text layer.

  • val3 — The interconnect layer.

Both perc.lvsBlackBoxPort.specified and perc.lvsBox.specified must be true for this option to be used.

perc.traceProperty.parameters

A list of lists defining the contents of the table for the option “Trace Property”. This table defines parameters for the for the Trace Property statement.

Each sublist defines parameters for one statement. The sublist syntax is:

[ bool, "params"]

  • bool — true or false. Indicates whether the statement is used.

  • params — The parameters for the statement.

perc.traceProperty.specified must be true for this option to be used.
