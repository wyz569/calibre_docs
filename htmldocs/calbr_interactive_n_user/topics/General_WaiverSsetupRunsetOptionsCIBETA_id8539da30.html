cmn.waiverShapes.mergeWaivers.value

Specifies the source of waivers for a waiver run. The option is the keyword used for the MERGE statement in the waiver setup file.

  • NO — Use waivers both from the waiver database and from the input layout.

  • YES — Use only waivers embedded in the input layout.

  • IGNORE_DESIGN_WAIVERS — Use waivers only from the waiver database.

Waivers page, Waiver Shapes, “Waiver Database”

[NO YES IGNORE_DESIGN_WAIVERS]

{NO}
