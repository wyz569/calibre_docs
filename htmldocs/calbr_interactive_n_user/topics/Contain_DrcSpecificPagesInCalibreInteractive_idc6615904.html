DRC-Specific Pages in Calibre Interactive

Some GUI pages are specific to Calibre Interactive nmDRC.
Table 1. DRC-Specific Pages in Calibre Interactive

Page

Description

Outputs

Specify settings for the DRC results database, opening side RDBs, and for creating a CTO file, DRC HTML report, and DRC summary report

Fast XOR

Specify options for a Calibre FastXOR layout comparison run. See “Fast XOR Page”.

Recipe Editor

Create and edit check selection recipes. The Recipe Editor page is displayed when you click Edit next to the Recipe entry on the Rules page. See “Recipe Editor Page”.

Waivers

Specify information for the Calibre Auto-Waivers Flow.

This page is available if “Use Waivers when running DRC” is checked on the Inputs page or a run type of “Autowaiver creation” is specified.

See “Calibre Auto-Waivers in Calibre Interactive”.
