Calibre DESIGNrev and WORKbench

Calibre DESIGNrev and Calibre WORKbench include a Verification menu for starting Calibre Interactive, Calibre RVE, and some Calibre Pattern Matching applications.

The Verification menu includes the following selections:

Table 1. Verification Menu in Calibre DESIGNrev and WORKbench

Item

Description

Run nmDRC

Run DFM

Run nmLVS

Run PERC

Run PEX

Run xACT

Run 3DSTACK

Opens the corresponding Calibre Interactive application.

Run PM Capture

Run Pattern Matching

Opens Calibre Pattern Matching applications. See these topics in the Calibre Pattern Matching User’s Manual:

“Capturing Patterns From a Layout”

“Running Pattern Matching From a Calibre Layout Viewer”

Start RVE

Starts Calibre RVE.

RVE/CI Setup

Opens a dialog for setting Calibre RVE and Calibre Interactive setup parameters, such as socket number, highlighting layers, highlight line width, and cell mapping.

RealTime

See the Calibre RealTime Custom User’s Manual.

You can add custom menu items to the Verification menu; see “Custom Menu Items in Design Tools”.
