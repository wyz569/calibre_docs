Calibre nmModelflow: Calibration Stages

Calibration stages are a core component of modeling in Calibre® nmModelflow™. A stage holds the calibration settings for a model type; you must create stages in order to calibrate them. This video shows how to create a stage in the Calibre nmModelflow Flow Stage Wizard.

This video covers the following topics:

  • How to set up configurations

  • How to set calibration objectives
