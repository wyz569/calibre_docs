Open an SVRF Debugging Tool Session

Create a new SVRF Debugging Tool session by using an existing SVRF rule file, or load a previously saved session.

Prerequisites

  • A layout open in Calibre WORKbench

Procedure

  1. Open the Calibre RFT v2.0 GUI.
  2. Click the modal selection button in the upper right corner (red box in the figure). Choose SVRF Debugging Tool as the session type.
    Figure 1. Open SVRF Debugging Tool Session

  3. Hover your cursor over Add Project. Choose one of the following options:

    SVRF File — Opens an existing SVRF file.

    Load Project — Loads a previously-saved SVRF Debugging Tool project file.
