Introduction to the Calibre RFT v2.0 Environment

The Calibre® RET Flow Tool (RFT) v2.0 is available for Calibre® WORKbench™ releases 2019.3 or higher. The Calibre RFT v2.0 environment is a central access point for selected Calibre RET lithography tools, and is designed for the recipe developer to assist in debugging.
