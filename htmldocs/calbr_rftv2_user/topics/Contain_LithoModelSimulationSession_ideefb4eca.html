Litho Model Simulation Session

The Litho Model session type is designed to work with the current generation of Calibre tools. It supports normal litho models and EUV type (global) litho models.

The following operations are available for a litho model session once you configure it:

Table 1. Litho Model Simulation Operations

Operation

Control

Location

Description

“Generating a Print Image”

Print Image

Process Window Conditions pane

Simulate OPC effects on geometries on opc layers that lie partially or fully within a selected area.

“Running an Intensity Cutline Simulation”

Cutline

Process Window Conditions pane

Compare one or more simulations along cutlines.

“Creating a Contour Map”

Map

Process Window Conditions pane

Create a contour map of the light intensity at the surface of the resist.

“Running a Process Window Analysis”

Process Window Analysis

Process Window Analysis pane

Generate a super gauge data (.sgd) file of the simulated gauges.

The sessions you create in LM mode also appear with relevant tool buttons in Calibre WORKbench in the Cells pane on the left.
