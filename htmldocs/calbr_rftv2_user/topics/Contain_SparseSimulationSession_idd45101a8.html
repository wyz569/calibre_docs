Sparse Simulation Session

The Sparse simulation session enables you to individually load optical and resist models and define process conditions. This session corresponds to the legacy sparse OPC tool operations.

Currently, only one operation is supported in Sparse sessions.

Table 1. Sparse Simulation Operations

Operation

Control

Location

Description

“Generating a Print Image”

Print Image

Process Window Conditions pane

Simulate OPC effects on geometries on opc layers that lie partially or fully within a selected area.

The Sessions you create in Sparse mode also appear with relevant tool buttons in Calibre WORKbench in the Cells pane on the left.
