MT and Calibre MTflex Environment Variables

There are some environment variables you can use to control MT (multithreaded), Calibre MTflex, hyperscaling, and simultaneous multithreaded (SMT) operations.
Table 1. MT and Calibre MTflex Environment Variables

Variables listed in the Admin Guide

Definition

CALIBRE_HDBFLEX_MIN

Specifies the behavior for the MINCOUNT argument (in LAUNCH AUTOMATIC, LAUNCH CLUSTER, or LAUNCH MANUAL) when HDBFLEX is enabled.

CALIBRE_HYPER_SMTFACTOR

Controls the “-hyper” capability for OPC operations on HDB0 when run on hyperscaled systems.

CALIBRE_HYPERFLEX_SAVE_LOGS

Saves PHDB log files at the end of a Calibre run.

CALIBRE_MT_MONITOR_LOAD

Prints the LOAD associated MON statements to the transcript at a specified interval.

CALIBRE_MT_MONITOR_MEMORY

Prints the MEMORY associated MON statements to the transcript at a specified interval.

CALIBRE_MT_SMT_FACTOR

Overrides the BIOS setting for simultaneous multithreading.

CALIBRE_MTFLEX_HYPER

Specifies the same information as that defined by the HYPER configuration statement.

CALIBRE_MTFLEX_LAUNCH

Sets the parameters to the set of LAUNCH configuration file commands.

CALIBRE_MTFLEX_LOCAL_HOST_DIR

Performs the function of the LOCAL HOST DIR configuration file command when running without a configuration file.

CALIBRE_MTFLEX_MONITOR_LOCAL

Provides the same information as the MONITOR LOCAL command when running calibre -remote.

CALIBRE_MTFLEX_MONITOR_REMOTE

Provides the same information as the MONITOR REMOTE command.

CALIBRE_MTFLEX_REMOTE_DIR

Specifies a location on the remote host for writing log files.

CALIBRE_MTFLEX_REMOTE_MAXMEM

Sets the MAXMEM parameter for the REMOTE HOST command.

CALIBRE_MTFLEX_SAVE_LOGS

Saves Remote Compute Server (RCS) log files at the end of a Calibre run.

CALIBRE_REMOTE_CONNECTION

Specifies the host and port number for each connection to a remote host when using the rcalibre or rxcalibre utility.

Execution of the REMOTE COMMAND automatically sets the host:port values for this environment variable in the child process.

CALIBRE_REMOTE_COUNT

An environment variable set by the tool in a child process from REMOTE COMMAND, and equal to the COUNT parameter for the LAUNCH CLUSTER command.

CALIBRE_REMOTE_MINCOUNT

An environment variable set by the tool in a child process from REMOTE COMMAND, and equal to the MINCOUNT parameter for the LAUNCH CLUSTER command.

LITHO_SOCS_KERNELS_SHARED

Enables shared memory in post-tapeout runs.
