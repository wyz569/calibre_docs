Configuration File Reference Dictionary

You can use configuration file statements in a configuration file to control launching and connecting to Calibre MTflex servers.

Refer to “About the Configuration File” for information on creating a configuration file.

  • HYPER
  • LAUNCH AUTOMATIC
  • LAUNCH CLUSTER
  • LAUNCH MANUAL
  • LOCAL HOST DIR
  • LOCAL HOST COMPUTE
  • MONITOR LOCAL
  • MONITOR REMOTE
  • MONITOR SYSTEM
  • MONITOR UTILITY
  • REMOTE COMMAND
  • REMOTE HOST
