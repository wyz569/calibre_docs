Calibre Query Server

Licensing for Calibre Query Server.

Required License

  • calibreqdb or caldrclvseve

Substitute License

  • calibrepvs_s

Also Enables

  • None.

Considerations and Exceptions

  • When running Query Server without -cb, only the calibreqdb license is required.

  • When running Query Server with -cb, Calibre attempts to acquire a caldrclvseve license. If a caldrclvseve is not available, Calibre attempts to acquire a calibreqdb license.
