Log Settings for Licensing

Calibre Classic Licensing and Calibre Loop Licensing both produce log information that you can configure using the CALIBRE_LM_LOG_LEVEL environment variable. This information is written to the Calibre transcript.

Table 1 lists the valid environment variable values and information about the error resulting from each value. For information on the log files associated with the license server, refer to Licensing Mentor Graphics Software.

Table 1. CALIBRE_LM_LOG_LEVEL Environment Variable Settings

Environment Setting

Prefix

Description

Example

FATAL

// Error:

Fatal errors only. Fatal errors have no possibility for recovery and generally cause Calibre to exit immediately.

Unable to launch mgls_asynch, unable to authorize a product for any CPUs.

WARN

// Warning:

Warnings only. Warnings are errors with the possibility that execution could still succeed in at least a degraded state. Fatal messages are also included when using this setting.

Unable to secure a license. License has been queued.

INFO (default)

//

Informational messages. Fatal and warning messages are also included when using this setting.

CPU counts, products authorized, licenses acquired, licenses released.
