Potential Network Errors

Running a job in Calibre MTflex and Hyperscaling modes (‑turbo ‑hyper) may occasionally generate a “PTHREAD INIT Failure” or “Hyper data storage failure” error message.

You can prevent these errors by increasing the user limits (maxproc). The maxproc setting limits the total number of threads that can be used for a job. Table 1 provides some guidelines for the maxproc setting based on the number of physical remote CPUs that are to be used for a Calibre job.

Table 1. Guidelines for Calculating the maxproc Setting

Physical Remote CPUs (N)

Total Threads on Primary Host

< 200

N + 4N = 5N

>= 200 < 400

N + 2N = 3N

>= 400

N + N = 2N
