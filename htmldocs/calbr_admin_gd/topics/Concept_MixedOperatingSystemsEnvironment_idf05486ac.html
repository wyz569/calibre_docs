Mixed Operating Systems Environment

The different executables for the AOI, AOJ, and AOK trees are optimized for different operating system and hardware environments. A mixed operating system environment is not supported when using these trees.

If your network includes different distributions of Linux as listed in “Supported Operating Systems”, you can download and install the executables to a target directory. You then set the CALIBRE_HOME environment variable to the appropriate trees (see “CALIBRE_HOME Environment Variable”).
