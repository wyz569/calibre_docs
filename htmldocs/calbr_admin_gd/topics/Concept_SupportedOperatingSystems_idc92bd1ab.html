Note:

Calibre support for Red Hat® Enterprise Linux® (RHEL) is determined by the operating system version, for example, RHEL 7.2. Red Hat uses different terminology for marketing RHEL, including “Server,” “Workstation,” and “Desktop.” These different options are supported as long as a supported operating system version is loaded. RHEL for IBM POWER systems is not supported. Also, Red Hat Fedora® Linux is maintained, distributed, and supported differently than RHEL, and is not supported for use with the Calibre product line. Any use of the Red Hat Fedora operating system is governed by the “Linux Distribution Compatibility Statement”.
