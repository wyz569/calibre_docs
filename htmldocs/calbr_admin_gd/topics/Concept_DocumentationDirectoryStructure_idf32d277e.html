Documentation Directory Structure

The Calibre documentation resides in a self-contained directory structure. All links between documents are relative to each other. This enables you to make a copy of this directory and store it in any location and still have it function as a complete tree.

The installation process automatically installs the documentation at the same level as the Calibre software tree as shown in Figure 1. The Calibre software tree (install_directory/aoi_cal_201x.x_xx) contains a symbolic link named docs that points to the default install location for the documentation.

Note:

As of the 2013.1 release, the Calibre documentation is packaged with the Calibre software tree for both the DVD and software download file.

This structure allows you to easily move the documentation directory to a different location that has network access and reset the docs link to point to the new location. Maintaining this structure allows the documentation to be automatically accessed by the Calibre software applications.

Table 1 lists the naming conventions used for the common docs_cal_201x.x_xx directories and files.

Table 1. Naming Conventions for the Calibre Doc Tree Directories and Files

Name

Description

htmldocs/<handle>

A unique directory name for a manual in HTML format. Most handles consist of <prod>_<doctype>. For example, the handle for the Standard Verification Rule Format (SVRF) Manual is svrf_ur. The handles are the same for the HTML and PDF version of a manual.

index.html

The initial file used to open the InfoHub.

infohubs/<Calibre_scopes>

The scope definitions for the Calibre InfoHub.

legal/<handle>.pdf or legal/<handle>.txt

A unique filename for all Open Source Software (OSS) redistribution licenses used in the Calibre software.

pdfdocs/<handle>.pdf

A unique filename for a PDF manual. Most handles consist of <prod>_<doctype>. For example, the handle for the Standard Verification Rule Format (SVRF) Manual is svrf_ur.pdf. The handles are the same for the HTML and PDF versions of a manual.

videos/<video>

A unique directory name used for a video. The Calibre InfoHub provides links to the available getting started and how-to videos, as well as videos available on Support Center and siemens.com.
