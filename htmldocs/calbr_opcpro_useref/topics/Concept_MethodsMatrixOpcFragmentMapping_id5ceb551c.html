Methods of Matrix OPC Fragment Mapping

Matrix OPC provides several different methods for mapping correction layer fragments to the associated fragment on the opc layer.

  • One-to-One — Maps each correction layer fragment to the closest target fragment, as identified using the specified metric. Figure 1 shows one-to-one mapping on a dipole mask.

    Figure 1. One-to-One Mapping With Euclidean Metric

  • Many-to-One — Maps many correction layer fragments to the site on a single target fragment. Figure 2 shows many-to-one mapping on an Alt PSM mask. The image on the right shows the mapping of trim and phase layer fragments to the same site on the target layer fragment.

    Figure 2. Many-to-One Mapping With Euclidean Metric

  • One-to-Many — Maps one correction layer to many target fragments. Figure 3 shows how you can use one-to-many fragmentation to resolve areal image ripple problems.

    Figure 3. One-to-Many Mapping With Euclidean Metric
