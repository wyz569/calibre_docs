Ruler

Ruler Mouse Mode is available when you are in one of the 2D views. Use this tool to measure your structure in microns (um).

Figure 1 shows how the ruler measurements are displayed in both the 3D View display window and the status bar.

Figure 1. Ruler Measurement

The Ruler tab in the Preferences dialog box controls the appearance and behavior of the ruler (see Figure 8). Use the Number of Rulers option in this dialog to specify the number of rulers that can be drawn in the display; the minimum setting is 1 and the maximum setting is 10. The following dialog appears when the maximum number of rulers have been drawn:

Figure 2. Info Dialog for Ruler Limit

Figure 3 shows how multiple ruler measurements are displayed in both the 3D View display window and the status bar. In this example, Number of Rulers is set to 5, and 4 out of 5 rulers have been drawn in the display. Note that Ruler 5 in the status bar has no value yet.

Figure 3. Multiple Ruler Measurements

Select the Ruler toolbar button or change to 3D view to clear the drawn rulers from the display.
