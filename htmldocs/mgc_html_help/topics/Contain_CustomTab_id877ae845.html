Custom Tab

By default, the InfoHub displays the Custom Info tab. You can change the name of this tab and define links to your company's process and design documents. The information on the custom tab is visible in all InfoHub scopes within the same installed Siemens Software and Mentor software tree release or flow.
Note:

Neither the content on the actual Custom Info tab nor anything that it links to is searchable from the InfoHub.
