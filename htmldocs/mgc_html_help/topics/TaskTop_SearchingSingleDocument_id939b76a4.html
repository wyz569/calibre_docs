Searching a Single Document

If you know what document should have the answer, search within that document.

Procedure

  1. Open a document from within the InfoHub by clicking
  2. Open the search box using one of the following methods: Open HTML or Open PDF from a list box. You can also click the PDF icon in an open HTML document.

    If you want to...

    Do the following:

    Search HTML

    Click the Search tab from within the desired HTML manual.

    Search PDF

    Click the Search icon in the banner within Adobe Reader.
  3. Enter the text string to search for and click the Search icon. Your search results show in the sidebar.
