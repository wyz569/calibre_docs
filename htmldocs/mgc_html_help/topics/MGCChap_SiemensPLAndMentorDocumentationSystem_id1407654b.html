Siemens Software and Mentor Documentation System

Welcome to the Siemens Software and Mentor Documentation System. At the center of the system is the InfoHub that supports both PDF and HTML content. From the InfoHub, you can access all locally installed product documentation, system administration documentation, videos, and tutorials.

Many of the Siemens Software and Mentor software trees contain a large number of documents. To help you find information, the InfoHub provide scopes in the left, vertical navigation bar. These scopes are logical groupings of content based on related products within the software tree. You can use these scopes to constrain content when you search locally.

The InfoHub gives you control over the level of search you perform. A basic search that you can use to search: all installed documents, only documents within a scope, or custom list of documents (My Document List).

This search is constrained to locally installed HTML content. Local content is defined as HTML documents installed in your Siemens Software and Mentor software tree in the docs/htmldocs directory on your PC or on the network.
