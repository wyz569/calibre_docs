Calibre nmModelflow Workflow

Calibre nmModelflow has a number of different entry points, depending on whether you are migrating from a modelflow_v2 environment or continuing previous Calibre nmModelflow work.

Migration from modelflow_v2 Workflow

Although modelflow_v2 and Calibre nmModelflow share some common items, Calibre nmModelflow may use a newer version of them. In order to use the modelflow_v2 items, they must be converted or regenerated to fit their respective Calibre nmModelflow format.

Table 1. modelflow_v2 Migration Items

Item

Action

Task

Gauge files (.gg)

Super gauge files (.sgd)

Convert using the gauges convert command

  • Converting modelflow_v2 Gauge Files

  • Loading Gauge Files

Individual model files

Use the files to create a litho model

  • Creating Litho Models

Lithomodel file

Import an existing litho model to the database

  • Importing an External Litho Model

Existing Calibre nmModelflow Source Files Workflow

Calibre nmModelflow Modeling Tasks

Once you have your Calibre nmModelflow database set up, you can perform modeling tasks.

Table 3. Calibre nmModelflow Modeling Tasks

Action

Notes

Task

Load a gauge file

Gauge files are required to view, measure, simulate, and calculate information.

  • Loading Gauge Files

  • Manipulating Gauges in the Gauge Analysis Tab

Set up a litho model

Whether using an existing litho model, or creating a new one, the litho model must be imported into the database.

  • Creating Litho Models

  • Importing an External Litho Model

Start the Flow Wizard for stage setup

Each stage contains all the commands executed in a calibration job.

  • Creating a Stage With the Flow Stage Wizard

Set up a job

A job contains all the data and commands needed to run the job.

  • Setting up a Calibration Job

Set up a plot

Plots are created separately from an optimization run and can be requested on any job or on the active data.

  • Creating a Custom Plot

Set up a metric

Metrics assess the quality of calibrated model compared to one or more criteria.

  • Creating a Custom Metric

Set up a filter

Filters can remove gauges from the calibration candidates or filter for specific gauges that meet certain criteria after a calibration is run.

  • Creating a Filter

Saving model results

Calibrated models can be exported for use with other Calibre tools.

  • Saving the Models After Calibration

Add process window conditions

Calibration is run across all process window configurations, and will often give different results at varying dose and defocus values.

  • Adding Process Window Conditions to a Litho Model
