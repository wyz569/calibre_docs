Support for Akaike Information Criteria Computation

Use the Akaike Information Criterion (AIC) as a way to measure the quality of model results relative to other model results.

Calibre nmModelflow supports using the AIC starting with the 2015.4 release. To use the AIC features, the gauge data must be split into a calibration set and verification set. Multiple calibration runs must be performed.
