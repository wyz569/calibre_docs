Flow Stage Wizard, N2R/N2E Calibration

Stage Wizard dialog box

This wizard appears when you choose the N2R or N2E checkboxes in the Calibrate section of the Stage Wizard dialog.

Figure 1. Flow Stage Wizard, Neural Network Assisted Resist (N2R) Screen

Figure 2. Flow Stage Wizard, Neural Network Assisted Etch (N2E) Screen

Objects

Field

Description

Select Modelform

Sets the type of optical model used with this stage.

Epochs

Sets the number of iterations to run. Default is 600 for N2R and 6000 for N2E.

Batch Size

Sets the number of data points before a model is updated.

Grid Shift Num

Sets a data consistency adjustment value.

Mode

Changes the loss function for the optimizer to use a relative loss or absolute loss.

Max Norm

Sets the kernel constraint of the dense layers in the model.

Regularization

Sets the kernel regularizer for the dense layers in the model.

Model-Based Retargeting

Calibrates the stage using the etch-only retargeting algorithm.
