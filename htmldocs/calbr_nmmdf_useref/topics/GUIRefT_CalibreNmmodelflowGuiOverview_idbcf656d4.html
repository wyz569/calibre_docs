Calibre nmModelflow GUI Overview

To access: Litho > nmModelflow in the Calibre WORKbench main window

The Calibre nmModelflow GUI contains multiple tabs and sub-windows to display and interact with data.

Figure 1. Calibre nmModelflow GUI

Description

The Calibre nmModelflow GUI is a multiple document interface (MDI) with multiple windows docked by default. The windows are labeled in the order of importance. Your GUI may be arranged differently because all windows can be moved around and undocked.

Objects

Table 1. Calibre nmModelflow GUI Main Window Contents

Field

Description

1

The display panel.

Most of the GUI actions are performed in this window, including selecting items and displaying output data.

The display panel contains multiple display tabs (A) that change the mode of the panel.

The display tabs are:

  • Gauge Analysis — Displays the currently active gauge object.

  • Contour Analysis — Displays the active contours for analysis.

  • Plots — Displays any active plots.

  • Metrics Viewer — Displays any active metrics.

  • Database Browser — Lists all items in the active database.

  • Calibration Job Manager — Lists all defined jobs for calibration.

2

The menu bar.

The menu bar contains multiple tabs (B) that change the buttons available in the menu bar.

The menu bar tabs are:

  • Gauges — Buttons to control the gauges in the Gauge Analysis primary display tab.

  • Setup — Buttons to write the current configuration as a Calibre nmModelflow setup file.

  • Process Window — Buttons to calculate aspects of a process window gauge data set.

  • Cross Probe & Filter — Buttons to highlight and display gauges in the main Calibre WORKbench layout viewer window. The gauge analysis view will automatically scroll to the highlighted gauges when cross probing or selecting a region in the layout.

  • Plot — Buttons to generate built-in plot functions.

The menu bar also contains simulation buttons for one-click image and CD calculation.

3

The command window.

You can enter Command Line Interface (CLI) commands directly in this panel.

Some CLI commands output information to this window.

Additionally, every command executed by the GUI is also displayed in the Command Window. You can search in the window by pressing Ctrl-F when the Command Window is active, or by selecting Search from the right mouse menu.

Tip

The full CLI reference is available by selecting Help from the menu bar. (For more information, see “Calibre nmModelflow Online Help”.)

4

The application log window.

Calibre nmModelflow sends application status information from commands to this window.

5

The process window view.

Shows a matrix of dose versus focus conditions and the number of gauges in each condition.

6

Active files.

A read-only display on the bottom of the MDF window showing which gauge, setup, Lithomodel file, and layout files are currently active.

Usage Notes

Use the Calibre nmModelflow GUI by selecting a mode using the display tabs in the display panel. You interact with the data in that window using any controls in that window as well as the controls in the various menu tabs. Each display tab represents one data group in Calibre nmModelflow.

Tip

The Settings menu contains controls that adjust the behavior of Calibre nmModelflow, including:

  • Loading a different Qt style sheet
  • Changing the font size
  • Changing the base directory for the SEM Image Viewer
Note:

If the keyboard is not working properly (typed keys do not match displayed symbols) on starting Calibre nmModelflow, you may need to set the XKB_DEFAULT_RULES environment variable to “base”.
