Plot Wizard, Bossung-Type Plots

Plot Wizard screen

This wizard appears when you select the Thrufocus, Thrudose, or Thruplanes buttons from the Plot button bar or the first screen of the Plot Wizard.

Figure 1. Plot Wizard, Bossung-Type Plot

Description

Creating a plot with the Bossung Plot wizard a creates Bossung plot for a gauge.

Objects

Table 1. Plot Wizard, Bossung Plot Contents

Field

Description

Plot Type

Specifies the variable plotted on the x axis:

  • Focus — CDs are plotted versus focus.

  • Dose — CDs are plotted versus dose.

  • Plane — CDs are plotted versus the depth in the photoresist for which gauge measurements were taken.

Note:

This control does not appear when the wizard is opened by the Thrufocus, Thrudose, or Thruplanes buttons.

Gauge IDs

Specifies the gauge(s) to be plotted.

CD Type

Specifies the type of data to be used to plot the bossung curve.

  • Sim — Uses the simulated CDs.

  • CtrSim — Uses simulated aerial image CDs.

  • Meas — Uses the measured CDs.

  • Shift — Uses the image shift difference between simulated and drawn CDs.

  • CtrShift — Uses the image shift difference from the simulated aerial image.

  • Meas Sim — Uses both simulated and measured CDs.

  • Meas CtrSim — Uses both measured CDs and simulated aerial image CDs.

  • Sim CtrSim — Uses both simulated CDs and simulated aerial image CDs.

Show Fit Function

Toggles whether or not the parabola fit curve is shown on the plot with the data.

PW Condition

Optionally selects a process window condition to plot in the form of (focus, dose).

Format

Optionally sets the format for the legends, using the following variables:

  • $xLabel — The x-axis label.

  • $xVal — The x-axis value, in situations where IDs are used (such as GIDs).

  • $yLabel — The y-axis label.

  • $yVal — The y-axis label, in situations where IDs are used.

  • $T — The type.

  • $P — The plane.

  • $D — The dose.

  • $F — The focus.

Only items in the selected plot data that match a variable type in the string are printed; the rest are considered “”. For example, if the legend format specified for GIDs is $xlabel($gid)T($type)D($dose)_$ylabel, the output is GID(12)T(CD)D(1)_Meas, with focus and plane being ignored (both for being empty and not in the legend string).

Single Plot

Specifies to create a single plot for all gauge data.

Metric

Selecting this checkbox opens an additional set of controls to add a metric to the plot results. The selected (or created) metric is listed as a subtitle to the plot.
