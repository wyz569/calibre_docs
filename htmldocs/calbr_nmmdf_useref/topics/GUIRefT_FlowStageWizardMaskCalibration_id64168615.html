Flow Stage Wizard, Mask Calibration

Stage Wizard dialog box

This wizard appears when you choose the Calibrate option of the Stage Wizard with the Mask check box selected.

Figure 1. Flow Stage Wizard, Mask Calibration

Description

Use the options in the Mask Calibration screen of the Flow Stage Wizard to specify mask parameters to calibrate. Options appear in the Independent Parameters panel depending on what you choose. See “Flow Stage Wizard, Optimizer Settings” for help with mask calibration-related optimization settings.

Objects

Table 1. Flow Stage Wizard, Mask Calibration Contents

Field

Description

Specify Number of Lithomodel Mask Layers

Increasing the number adds more sets of MASK_MODEL_LAYERx parameters to the list. Each set can be calibrated individually.

Up to 5 layers can be used. The number of mask layers must correspond to the number of mask layers used in calibration, for which the layer properties can be set independently.

Select Mask Parameters for Calibration

  • Use Defaults — Uses current best practices for the calibration items. De-select this checkbox to enable the other options.

  • CX=CC — Toggles whether convex (CX) and concave (CC) corners are calibrated equally. De-select this checkbox to add separate CX and CC calibration parameters to the list.

  • Bias Type — Sets the bias calibration types between Bias (both X and Y directions have their calibration bias performed together), Bias X and Bias Y.

  • DDM Group — Adds the DDM group calibration parameters to the list. Note that you only need to enable the DDMGROUPPARAM parameters for the number of DDM groups you have; if you only have one group, uncheck everything but DDMGROUPPARAM0.

Note:

Not all mask parameters are selectable from the GUI. For a full list of currently supported mask calibration parameters, use the command:

   mdf param get mask
