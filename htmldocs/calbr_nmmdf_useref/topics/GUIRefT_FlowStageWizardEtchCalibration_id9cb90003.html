Flow Stage Wizard, Etch Calibration

Stage Wizard dialog box

This wizard appears when you choose the Calibrate option of the Stage Wizard with the Etch check box selected.

Figure 1. Flow Stage Wizard, Etch Calibration

Objects

Table 1. Flow Stage Wizard, Etch Calibration Screen Contents

Field

Description

Select Etch Model Form

Sets the modelform for the etch model. The modelform you choose sets the contents of the Etch Parameters list.

  • If you activated a litho model prior to opening this screen, you are also given the choice to use the Active Input Lithomodel as the modelform, which tunes all available parameters.

  • Clicking the Preview Etch Modelform button to the right of the Select Etch Model Form field brings up a preview viewer for available modelforms. This viewer is also available from the Help menu.

  • Clicking the Launch VEB Composition button to the far right of the Select Etch Model Form field opens up the custom etch modelform wizard.

Etch Parameters

Lists the parameters that are calibrated for the etch model. Note that the Operation field only activates if a Constraint is specified as a value other than the default (Independent). The Operation field is used to create a constraint-type dependency between two selected parameters.

Note:

Not all etch parameters are selectable from the GUI. For a full list of currently supported resist etch parameters, use the command:

mdf param get etch
Tip

To set a limit on the absolute value of BTERMs during calibration, add the command:

   mdf optimize set_etch max_linear_coeff val

Select VEB Fit Type

Sets the fitting type used for VEB calibration to relative or absolute.
