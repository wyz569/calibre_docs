Plots Tab

Calibre nmModelflow Main window primary display tab

The Plots tab shows selected data traces for gauge information columns in a visual format. Multiple plots can be displayed.

Figure 1. Plots Tab

Description

Plots are either created automatically by calibration jobs (from a successful run in the Calibration Job Manager) or by being activated from the Plots group in the Database Browser.

To create a plot, see the task “Creating a Custom Plot.”

Objects

Table 1. Plot Viewer Controls

Field

Description

A

Plot viewer. Plot views appear in the main area of the window once activated, or after a successful calibration run.

  • Each plot has its own set of filtering and viewing controls.

  • Drawing a box around points in the plot selects the corresponding gauges in the Gauges tab.

  • Right-clicking on gauges in the plot viewer displays options to manipulate gauge data directly from the viewer.

B

Plot selector. Only plots that are checked in this list are shown in the Plot viewer.

C

Traces selector. Only data sets that are checked in this list are shown in the related plot view.

D

Filter. Entering a string in this field retains only plot traces that contain the string in their legend name. Clearing the field restores the list of traces to the view.

Note that you can use the trace filter to either show or exclude a set of filters by first selecting only filters with a specific string, then toggling the checkbox next to the Legend title.

E

Plot level controls. These buttons appear in every plot view, and control only items in that view. The controls appear in the following order:

  • Plot/Table

  • Activate

  • Edit Plot

  • Add Plot to Database

  • Save Plot

  • Copy Plot to Clipboard

  • Remove Connecting Lines

  • Change Visibility / Position of Legends

  • Change Markers Style

    Tip

    Setting the Marker Size to zero will hide a marker, if you need to see other markers occupying the same space.

  • Apply/Remove Gauge Sorting

  • Apply Plot Grouping

    Tip

    Applying Plot Grouping for a SimErr scatter plot when the Group column is defined in the active gauge object can display the groups in a single consolidated graph.

    Tip

    The Settings > Plot Settings menu option allows you to group gauges with similar structure names together (matching up to the specified number of starting letters). Any gauges that have less than 1% of the gauge count are put into a “Misc” group.

  • Apply / Remove Filters

  • Refresh the Plot

  • Hold On/Off Active Plot

  • Format Axis Options

  • Reset Axis Range

  • Show Points Label

  • Delete Plot

  • Delete All Plots

F

Plot menu bar controls. Controls in this bar apply to all active plot views.

  • Zoom In, Zoom Out — Zooms all plots.

  • Prev, Next — Cycles through the list of plots.

  • Show All, Hide All — Displays or hides all plot views.

  • Delete All — Deletes all plots.

  • Hold On — Keeps the current plot active; new plots are added to the held plot.

  • Unhighlight All — Removes all selection highlights.

  • Hide/Show Legends — Hides/ Shows legends in all plots.

  • Hide/Show Toolbar — Hides/Shows the toolbar across the top of all plots.

  • Format Axis Options — Controls to customize the range of both the x and y axes for all plots. Plots can be graphed on a user-specified spacing or logarithmic scale basis.

  • Reset Axis — Resets all axis back to the default settings.

  • Show Points Label — Labels each point in the graph with the X and Y data values of that point.

  • Report — Exports the plots to an HTML report.

  • Filter Traces — Similar to the legend filter, this text field filters all plot views for legends that contain a string. Remove the filter by deleting any text in the field and clicking the check button.

G

Plot window controls. These buttons appear at the top right of every plot view, and control the plot window. The icon buttons appear in the following order:

  • Dock-in All Docked out Plot Widgets

  • Dock/Undock Widget

  • Maximize/Restore

  • Close

Usage Notes

A variant version of the Plots tab appears when you click Compare in the Calibration Job tab for two or more completed jobs:

Figure 2. Plots Tab (Calibration Jobs Compare Version)

Table 2. Plots Tab (Calibration Jobs Compare Version

Field

Description

A

Mode control tabs. The following modes are available:

  • Plots — Select and view plots by picking gauge items to use as the X and Y axes.

  • Contour Plots — Select and view plots by picking contours to use as the X and Y axes.

  • Parameters — A list of parameters that can be compared, shown by value.

  • Parameters Plots — Select and view plots by picking parameters to use as the X and Y axes.

Note:

The Parameters (as a right-click menu option in a column) and Parameters Plots (as a button on the bottom left corner) tabs have a Cluster Plot option that allows you to consolidate data from multiple calibration jobs into a single cluster plot.

  • Akaike Information Criteria — See the section “Comparing Akaike Information Criterion From Calibration Jobs”.

B

Plot item Selector. A single factor can be chosen for the X-axis; multiple factors can be chosen for the Y-axis.

C

Plot Viewer. Functions as described in the previous table.

D

Bossung Plot Compare switch. Selecting this checkbox enables options to select process window plots for selected gauges.
