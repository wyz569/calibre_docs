Litho Model Information for Toploss Calibration

Toploss calibration in Calibre nmModelflow has the following additional information specific to configuring a litho model.

The Lithomodel file should include an optical model and multiple mask layers for each plane sampled for toploss. A DDM model is recommended. It should look similar to the following code: 

version 1
resist resist.mod
mask 0 {  optical e1_f+0.0000  
	background atten 0.06  
	mask_layer 0 TRANS clear DDM x1.ddm
	mask_layer 1 TRANS clear DDM x1.ddm
	mask_layer 2 TRANS clear DDM x1.ddm
}
zplanes zpl.mod
}

The ZPlanes model should have toploss parameters in it, similar to the following code:

version 1
modeltype ZPLANES
toploss_plane 0.0
toploss_dose 1.0
toploss_mincd 0.0
