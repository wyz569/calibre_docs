Flow Stage Wizard, Etch Bias Table Generation

An etch bias table is used by the Calibre nmBIAS BIASRULE command to move edge fragments based on their critical dimension measurement during OPC. Calibre nmModelflow uses the Etch Bias Table Generation wizard to create tables based on your input.

Figure 1. Etch Bias Table Generation Wizard (veb Mode)

Figure 2. Etch Bias Table Generation Wizard (lsf Mode)

Description

Two types of etch bias tables can be generated by the Etch Bias Table Generation Wizard. The mode is specified using the Method option.

Rule Table Parameters

Selecting Autobin instructs the generator to automatically create the bins based on the input gauges. Enabling Set Boundary Limits lets you set the width and space to values smaller than the boundaries of the gauge set.
