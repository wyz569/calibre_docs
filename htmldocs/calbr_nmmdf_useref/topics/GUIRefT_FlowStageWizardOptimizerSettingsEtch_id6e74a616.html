Flow Stage Wizard, Optimizer Settings With Etch

Flow Stage Wizard screen

This wizard appears as part of the wizard sequence when you have selected the Calibrate Etch option for a stage. It sets the optimization search algorithm for the etch model.

Table 1. Flow Stage Wizard, Optimizer Settings With Etch Contents

Field

Description

Cost Function

Sets the objective function for the etch optimization. It is handled separately from optics and resist calibration.

Optimization Parameters

Controls general optimization parameters.

Changing the Anchor and Optimizer Settings opens up additional option groups.

  • Activating the Anchor parameter requires you to specify an anchor gauge, whose simulated etch CD is to be kept equal to the measured one while calibrating or building models.

    You can also specify which process condition to use for the anchor gauge.

  • Switching the Optimizer Settings option to “modify defaults” allows you to change any of the following values:

    • Biasing_Type — Select 0 for speedier, less accurate calibration, 1 for more accurate calculation, “auto” to detect gauges that need special attention, and “auto active_model” to run auto mode with the active VEB model.

      Tip

      If you run with Biasing_Type “auto”, gauges that need further inspection are listed in the Job transcript as being set for biasing_type 1. Search for the string “BT1” to find the relevant section.

    • Selective_biasing_type1:layer number — Used with Biasing_type 0 to filter certain gauges as biasing_type_1 (more accurate calculations).

    • Structs_biasing_type1:string list — Used with Biasing_type 0 to include specific structures in more accurate calculations. When this option is active, you can select multiple structures using the checkboxes in the list, filter by name using the Filter field at the top, and add specific names using the text field at the bottom of the list.

    • CD Tolerance — Used to set the tolerance threshold in nm for gauges that require additional calibration time.

    • Inputlayer and its related fields (Layer Number and Contourcal Mode) — Used for Direct VEB optimization.

    • Thresholdtolerance and Thresholdstability — Two mutually exclusive tuning parameters, used to set threshold where gauges are considered resolved (by a tolerance distance and range percent, respectively).

    • Ftolerance — Optionally limits the front algorithm tolerance.

    • Frontparallel — Optionally toggles parallel processing for front search mode when activated.

Tip

In most cases, using the defaults represents the best practices for the current release.

SEM Box

Selecting SEM Box opens a subsection where you can set the box width and measurements used if you are using a SEM CD measurement algorithm to compute simulated CD values.

Regularization Parameters

Selecting Average Bias and setting a Tolerance causes the calibrated VEB model to use a computed average bias before selecting a final VEB model.

Usage Notes

In most cases, the “use defaults” option Optimizer Settings should suffice, because it represents the best practices for the current release.
