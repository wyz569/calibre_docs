Plot Wizard, Cutline Plot

Plot Wizard dialog box

This wizard appears when you select the Cutline Plot item in the Plot Wizard.

Figure 1. Plot Wizard, Cutline Plot

Description

The Cutline Plot wizard creates a plot for the cutline intensity of the specified gauge. It requires a gauge data or super gauge data object to be active in the Gauge Analysis tab, as well as an active lithomodel.

Objects

Table 1. Plot Wizard, Cutline Plot Contents

Field

Description

Gauge ID

Specifies the gauge that appears in this plot.

All PW Cutlines

Toggles how many process window conditions are plotted; selecting “no” only plots the active condition; selecting “yes” plots all conditions.

Sim Type

Sets the simulation type to be either “full” (uses full litho model) or “aerial” (uses the aerial image only).

Format

Adds a custom format string for the labels.
