Edit Lithomodel Dialog Box

Database browser tool

A litho model contains process information, and also contains other models with their own separate definition files. Use the Edit Lithomodel dialog box to navigate through the parts of a litho model and make changes as needed.

Figure 1. Edit Lithomodel Dialog Box

Objects

Field

Description

A

Navigation tree. Shows the hierarchy starting with the Calibre nmMDF setup file as the parent.

B

File Viewer. Shows the contents of the selected file.

  • Clicking an .smf file brings up the Source View tool.

  • Zernike files appear as .xml files underneath the optical model.

C

Keywords and Commands tabs.

  • The Keywords tab shows individual legal parameters for the currently selected file. You can type in the search field to find keywords matching the string.

  • The Commands tab shows actions that can be taken for the selected file, such as changing the modelform or loading a new model. Click Submit in this tab to make the change.

D

Example pane. Shows the syntax of the various litho-model related files.

E

Litho Model Attributes pane. Shows the information for the selected litho model entry in the Calibre nmMDF database.
