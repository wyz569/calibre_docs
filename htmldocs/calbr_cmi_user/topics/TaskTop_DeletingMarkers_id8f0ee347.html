Deleting Markers

You can delete existing markers using CMi.

Prerequisites

Before starting this procedure, you should have the following:

  • Invoked Calibre WORKbench and the Calibre Metrology Interface

  • A mask layout

  • A marker list loaded into CMi

Procedure

  1. Select the desired marker row in the table. The selection is indicated by darker color background in the row. Multiple rows can be selected together. To select a range of rows, click on the beginning row, and Shift-click the end row. Add or remove selections with a Ctrl-click on individual rows.
  2. Right-click on the highlighted row(s) and select Delete Markers. A pop-up dialog box will appear for confirmation. The selected markers disappear from the table and layout views.
