Adjusting Marker Parameters

You can use the Calibre Metrology Interface to adjust the parameters for your markers.

Prerequisites

Before starting this procedure, you should have the following:

  • Invoked Calibre WORKbench and the Calibre Metrology Interface

  • A mask layout

  • A marker list loaded into CMi

Procedure

  1. Select the desired marker row in the CMi Sites table (see “Sites Table and Properties”). The selection is indicated by lighter color background in the row.
  2. Click on any of the columns to adjust the value. The changes will be made in the layout and table.
