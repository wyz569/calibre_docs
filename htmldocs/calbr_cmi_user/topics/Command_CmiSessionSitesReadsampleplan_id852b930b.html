sites readSamplePlan

CMi Tcl Commands

Loads a sample plan from different sources by defining the file format and the file path.

Usage

$cmi_session sites readSamplePlan -type {csv | ss | gg | rdb | xylist} -file file_path

Arguments

cmi_session

A required argument that references CMi session name created from the CMiSession create command.

-type {csv | ss | gg | rdb | xylist}

A required argument that specifies the file format type: csv (CSV), ss (super spreadsheet), gg (gauge), rdb (RDB), or xylist (list of X Y marker coordinates).

-file file_path

A required argument that specifies the path name to the sample plan file.

Description

This command loads a sample plan file into CMi. Sample plan files contain marker information that can be loaded into a CMi session. Several different file formats are supported, including CSV, super spreadsheet, gauge, RDB, and an X Y list.

Examples

$cmi_job sites readSamplePlan -type csv -file test_sepeed.csv
