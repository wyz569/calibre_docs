Reloading Markers

You can reload preexisting markers in CMi as part of an error recovery function. For example, you accidentally imported markers to an incorrect layout file and you need to import the same marker list to another layout file instead.

Prerequisites

Before starting this procedure, you should have:

  • Invoked Calibre WORKbench and the Calibre Metrology Interface

  • A mask layout

  • A marker list that was previously loaded into CMi

Procedure

In CMi, select Tools > Reload Markers. The markers listed in the Sites table (see “Sites Table and Properties”) are reloaded from your original marker source.

Examples

For example, you have two layouts (layout1 and layout 2) loaded into Calibre WORKbench, and you are importing markers. You intended to import markers to layout1, but you accidentally imported them to layout2 instead, as it was the one currently displayed in Calibre WORKbench.

In this case, you can select layout to display in Calibre WORKbench from the Layouts drop-down list, then select Tools > Reload Markers in CMi to reload the imported marker list to layout1.
