openSession

CMi Tcl Commands

Loads a previously-defined session file into CMi.

Usage

$cmi_session openSession -session session_path

Arguments

$cmi_session

A required argument that references CMi session name created from the CMiSession create command.

-session session_path

A required argument that specifies the path name to the session file.

Description

This command instructs CMi to load the settings from a previously-defined session file. A CMi session file defines the paths of the inputs to CMi design layout, job specification template file, and site sources.

Examples

$cmi_session Opensession -session test_session_working
