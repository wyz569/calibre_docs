Managing Chiplets

You can use CMi to generate metrology recipes for sites in different chiplet placements. Chiplets are designs that utilize far smaller microscopic circuitry than currently used to create electronic components such as microprocessors and memory.

Use the CMi Chiplet Manager to specify different placements and transformations of chiplets to the input layout, enabling the generation of metrology recipes for the same site in different transformations.

Figure 1. Chiplet Manager Work Flow

CMi accepts a user specification of the chiplets transformation either through a specific UI or through extended job deck format. According to the specification, the transformation matrix is applied on the input sites and generates the metrology recipe accordingly.

Figure 2. Generating a Recipe With Chiplets

Prerequisites

  • A layout file (containing the design to create chiplet transformations from) loaded into Calibre WORKbench.

  • An extended job deck where the placement of the chiplets are defined. This is used for saving a loading purposes.

Procedure

  1. Perform the steps described in “CMi Workflow Overview”. You should have a list of markers defined (or imported from a previous session) and run options in the Job Configuration area specified.
  2. In CMi, select Tools > Chiplet Manager. The CMi Chiplets Manager appears.
    Figure 3. Chiplets Manager

  3. In the Chiplets Manager, you can perform the following:

    • Load chiplet transformations — Click the Open File button to load an extended job deck containing chiplet transformation settings. See “Chiplets Manager” for information on the extended job deck.

      Figure 4. Open File

    • Add a new chiplet transformation settings — Click the Add (+) button to add a new entry field representing a chiplet, then manually enter settings for shift x, shift y, magnification, rotation, and mirror transforms.

      Figure 5. Add Chiplet Transforms

    • Delete an entry — Select an entry and click the Delete (-) button.

      Figure 6. Delete Chiplet Transforms

    • Modify an existing entry — Click on an entry and you can manually edit the shift X, shift Y, magnification, rotation, and mirror transforms.

    You can also load a CMi session file, and if the session includes the job deck file (chiplets session) the Chiplets Manager appears automatically.

  4. Save your transformation changes to an output file by clicking the Save icon.
    Figure 7. Save Transforms

    Saving the CMi session automatically saves the settings in the Chiplets Manager as well.

Results

The metrology recipes created by this process output the sites defined by the chiplets specification using the transformation matrix. CMi outputs the site IDs in a recipe using the input site ID as a prefix, then appending “_ placement#” to each entry. The generated layout clips (GDS or OASIS) are captured for each chiplet placement. The following figure illustrates how the input sites are generated in the output.
